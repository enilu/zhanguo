# 战国小说项目

这是一个以中国战国时期为背景的长篇历史演义小说项目。

项目当前目标是：

- 以 `秦国东出，终成一统` 为主线
- 采用 `天下共视角`
- 起笔于 `苏秦张仪时代`
- 风格定位为 `史实主干 + 演义化适中`
- 强化人物能力辨识度与讨论度，使读者在读完后愿意讨论：
  - 名将排名
  - 统帅排名
  - 纵横家排名
  - 悲剧人物排名

## 写作原则

- 不把秦写成天然正义的一方
- 不把六国写成整齐划一的昏庸群像
- 不把战国写成三国翻版
- 不靠夸张单挑支撑武将魅力
- 重点写出：
  - 纵横
  - 统兵
  - 国策
  - 国运
  - 悲剧英雄
