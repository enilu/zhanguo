# 项目约束详细说明

## 项目核心不变量

### 主线与结构
- **主线**：`秦国东出，终成一统` - 所有情节围绕这一核心展开
- **起点**：`苏秦张仪时代` - 从合纵连横开始，展现战国时代的政治格局
- **篇幅**：`90 回左右` - 每回控制在合理篇幅，保持叙事节奏
- **结构**：`纵横家开局，名将收束，帝王定局` - 第一卷纵横，第二卷连横，第三卷血战，第四卷长平，第五卷统一

### 立场与视角
- **立场**：`天下共视角` - 不偏帮任何一方，客观展现各国立场与利益
- **秦的塑造**：秦国要强，但强在法、兵、吏、地利与代代积累
- **六国的塑造**：六国要败，但败得各有缘由，也都必须有可敬人物

### 风格与语言
- **风格**：`史实主干 + 演义化适中` - 在尊重历史的基础上进行合理的艺术加工
- **语言**：`现代白话为主，关键处带古意` - 兼顾可读性与历史感
- **演义化程度**：不进行过度夸张，保持人物和事件的合理性

## 人物能力体系
本书核心人物按照五条能力线塑造：
- `武勇`：个人战斗能力
- `统兵`：指挥军队作战的能力
- `机谋`：计谋策略能力
- `纵横`：外交游说能力
- `国策`：国家治理与战略规划能力

写作目标不是让所有人都“厉害”，而是让他们“厉害得不同”。

## 必须坚持的写法
1. 秦国要强，但强在法、兵、吏、地利与代代积累
2. 六国要败，但败得各有缘由，也都必须有可敬人物
3. 大人物要有鲜明标签、代表名场面、致命短板
4. 虚构人物可以串联叙事，但不能改变历史主轴
5. 战争戏要突出统帅差异，不依赖过量单挑描写

## 必须避免的写法
1. 把战国写成三国翻版
2. 把武将塑造成脱离时代逻辑的纯猛将
3. 把谋士全部写成只会嘴上斗法的“同类角色”
4. 把六国写得过弱，导致统一缺乏史诗重量
5. 让秦国胜利显得轻易
