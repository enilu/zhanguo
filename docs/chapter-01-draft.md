# 第一回 周室衰微诸侯竞势 布衣立荒坡先看六国山河

周室东迁已久，王命早轻。

洛邑还在，天子也还在，可天下谁都知道，如今真正压得住人心与甲兵的，不在那一座旧都，而在函谷以西的秦，在河山之间的六国，在一国国朝堂上的兵、粮、城与法。

齐富，楚广，赵险，魏韩据中原，燕守北地。

人人都自称强国。

可若真论法令整齐、兵甲精锐、关塞险固、国政一体，诸侯心里又都暗暗顾忌着同一个字。

秦。

这念头人人知道，偏又没人肯先认。近秦者怕兵，远秦者观风；强国不肯先低头，小国不肯先担险。于是馆驿之间，游士往来如织，日日都有人说天下，说强弱，说合散，说谁可救谁可并。有人佩剑求人，有人负书求仕，有人只图一席富贵，也有人真想着拿一张嘴去拨整个乱世的方向。

这年冬末，赵地官道上正有这样一人独骑而行。

他衣袍旧，马也瘦，鞍旁只悬一囊书简。若只看形貌，不过寻常寒士；若看那双眼，却总不肯低，只一路望着山、望着路、望着远处城郭。

此人姓苏名秦，洛阳人。

他一路北行，心里算的不是一城一地，而是赵在此，燕在北，魏韩在中原，齐楚分东西，若西边秦再逼一步，东方诸侯究竟会先往一处靠，还是先各自关门。

午后，他行到一处馆舍前。

墙外停着两辆漆车，门前又拴着几匹高头大马，一望便知里头不是寻常商旅。苏秦心想，若能借机见一见诸侯门下宾客，也许能先试一试口。于是翻身下马，整了整衣冠，取了书简，走到门前通名。

守门仆役只拿眼角扫他一眼：“何人？”

“洛阳苏秦。”他说，“闻馆中有贵人门下宾客在此，愿求一见，陈说天下之势。”

那仆役听完便笑了。

“这几日来，说天下的人多了。你也是？”

“正是。”

仆役又把他从头到脚看了一遍，懒懒道：“我家主人一路北来，见的都是列国使者、名门高士，哪有工夫听你一个路上寒儒说势。去罢，莫站在门前冲了车驾。”

苏秦没有立刻走。

他拱了拱手：“天下之势，不因言者衣敝而轻重有别。烦请通报一声，若主人听后无用，我自退去。”

仆役嗤地一笑：“天下之势若真在你囊中，怎至于独骑瘦马，立在我家门前吃这北风？”

旁边几个小厮听了，都低低笑出声来。

馆中这时又正好走出一名年轻门客，听见苏秦求见，只随手理了理袖口，道：“如今各国都说秦强，人人都能说几句危言。可真要用你，凭什么？凭这身旧衣，还是这几卷旧简？”

苏秦听到这里，手中书简微微一紧。

他看着那门客，沉声道：“若但凭衣冠取人，今日所失，未必只是一个布衣。”

那门客像听见一件可笑的事，笑着摇头，转身便入内去了。

馆门半掩，里头灯火渐起，外头风也更紧。苏秦在门外站了片刻，终于收了书简，转身离去。

天将晚时，他没有再进城，只在城外寻了一处高坡坐下。

坡下旧驿里已有灯火，远处城头也有昏黄一点。再往西望，山势层叠，暮色沉沉；再往东看，道路蜿蜒，不知通向多少国都与关津。风从旷野深处吹来，把袍角吹得猎猎直响。

苏秦把书简放在膝上，久久不语。

白日里那些轻慢的话还在耳边。他不是不怒，只是怒过之后，心里反倒更冷了。那仆役、那门客轻的，当然不止他一人，而是天下所有尚无名势的寒士。你言得再高，只要手里无势，旁人便先把你的话轻去三分。

想到这里，他慢慢抬起头，望着远处那一片层层隐去的山河。

赵虽强，独当秦锋也难；燕远而弱，更不能自全；魏韩在中原，最先知痛；齐楚看似远些，也不过是把祸往后延一点。若真有人能把六国各自心里的惧意，暂时织成一股向西的势，今日看似无门之局，未必便永远无门。

风又吹来，坡下驿火晃了一下。

苏秦两手按在书简上，指尖已冻得发白，心里那一团火却一点点亮了起来。
