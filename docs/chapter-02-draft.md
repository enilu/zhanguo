# 第二回 说赵未成空怀策论 入秦受辱始识炎凉

次日天未大明，荒坡上残霜未化，坡下旧驿旁几匹旅马还在低头啮草。苏秦已先起身，将昨夜反复摩挲过的书简一一理齐，以旧布裹好，重收入囊中。他立在坡前，向东望了片刻，又向西望了片刻，眼底并无踌躇，只有一股冷静逼出来的锐气。

昨夜那一点念头，经一夜风寒，非但不曾熄灭，反而越发明白了些。若六国仍各守门户，各自盘算，便等于把山河一寸寸让给西土。此念若果真不错，便不能只留在坡头风里，总要拿去试一试诸侯肯不肯听。

他翻身上马，沿官道南下。一路上天色渐开，田畴间已有农夫驱牛，村口也见樵夫荷担。赵地城邑虽不若大国都门那般繁盛，然道路平整，行旅不少，间或可见运甲送粮的车队，轮痕深深压在冻土之上。苏秦看在眼里，心中愈发肯定：赵并非弱国，若能早知秦患，未必没有争衡之力。只可惜，国虽可争，未必肯轻信陌路寒士。

午后，他到了邯郸近郊。

邯郸本是赵国重地，城郭宏阔，市廛繁盛，车马之声隔得老远便能听见。城门前人流不绝，有商旅携货入市，有小吏骑驴出城，也有佩剑门客成群结伴，意态从容。苏秦牵马徐行，衣袍旧素，混在众人之间，愈发显得不起眼。可他眼睛却亮，一路看城门守卒、看往来宾客、看那些车上所悬家纹，心里已将赵国气象暗暗量过一遍。

入城之后，他并不先寻客店，而是先打听赵国几处权门宾馆所在。一路问去，旁人听他说是洛阳来的游士，多半只是点头敷衍，并无多少惊异。战国之世，负书求仕者何止千百，谁都自称胸中有策，真正能走入高门的却少之又少。苏秦明白这个道理，故也不多辩，只循着所问消息，到了赵国一位近臣门下。

那门第不算最显赫，却也朱门高墙，车马杂沓。门外已有数人等候求见，有的衣冠楚楚，有的神色局促。苏秦立在一旁，听见有人低声议论近来秦使往来、边地军报、朝中议兵之事，便知自己没有走错地方。过不多时，门中出来一名执事，依次问众人来意。轮到苏秦时，他拱手道：“洛阳苏秦，有策愿陈，欲言赵国西患。”

那执事听见“西患”二字，倒多看了他一眼，问道：“你也知秦？”

苏秦道：“天下人谁不知秦？只是知之有浅深，言之有缓急。苏秦所言，不止于秦强，更在赵当如何处之。”

执事见他言辞尚整，便没有立时赶走，只道：“可将所欲陈之意先略说一二。我家君子近来事务繁多，不是人人都能入见。”

苏秦也不迟疑，便立在阶下，压低声音道：“赵据山河之险，士卒敢战，本可与秦争衡。然赵若只恃本国之强，坐看韩、魏受逼，燕、齐、楚各怀其心，则秦可以次第困东方。到那时，赵纵强，亦不过独木支危墙。赵今之上策，不在独固边防，而在先定东方诸侯共同之惧，使天下知秦不可独当，唯有并势，方可争衡。”

他这一番话，说得不算长，却把赵与诸侯、秦与东方的轻重先提了出来。门前几个候见之士原本漫不经心，听到这里，也都不由侧目。那执事沉吟片刻，道：“你这话倒不像寻常游谈。且在此稍候。”

说罢，转身入内。

苏秦在门外立着，心中虽仍沉稳，终究也生出一丝微弱希望。自出洛阳以来，他见多的是门吏冷笑、宾客轻慢，像今日这般能把话递进去，已算难得。若门内那位近臣肯拨一瞬工夫听他陈说，也许赵国未必全无入手之机。

等了约有半炷香，门内终于有人召他进去。

苏秦随那执事穿过前院，只见院中宾客来往，堂下侍从持简而行，气象果然不俗。到偏厅时，厅中已坐着两人：上首是一名四十余岁的赵臣，眉目清峻，衣冠整肃；下首陪坐者却是一位年轻门客，面白须短，眼神锐利，正在翻看一卷军报。那赵臣抬眼看了苏秦一眼，不曾让座，只淡淡道：“闻你言赵国西患，有何见教？”

苏秦再拜，随即答道：“赵之患，不在秦一朝动兵，而在天下久不能成共势。今秦以法令整其内，以甲兵逼其外，韩、魏当冲，最先受其侵削；韩、魏若困，赵便西门洞开。故赵欲自安，不能只看赵境，当先看诸侯之心。”

那年轻门客听到这里，便插口道：“诸侯之心，谁不知道？各国都怕秦。可怕归怕，凭什么就肯听赵调度？又凭什么肯为赵先出力？”

苏秦看了他一眼，答道：“不必先问谁肯为赵出力，要先问谁不愿自家国门先临秦兵。惧秦者，六国皆然；所差者，不过是远近轻重。若能把这份共同之惧，化作共同之利，则纵不能一日成盟，也能先成同势。此事若有人先言，赵最可先应。”

那赵臣闻言，神色稍动，却仍不置可否，只问：“你说诸侯并势制秦。此策若成，自是天下大计；若不成，赵先举其议，岂不先招秦疑？再者，六国谁无自尊，岂肯因你一介布衣几句话，便甘受牵合？”

这话问得极实。苏秦知道，对方并非全不信，只是不肯轻易押注。他略一沉吟，答道：“大策本就难在可成与未成之间。赵若畏秦疑而不敢先动，则终日守边亦不过待敌来攻；赵若见诸侯各私便自收门户，则看似自保，实则是坐看天下之势一一坏尽。至于臣之身分寒微，固不足重。然策之可否，当论利害，不当先论衣冠。”

“说得好听。”那年轻门客微微一笑，把军报卷起，轻轻敲在案上，“只是天下朝廷，不是路边说书场。你说纵则可纵，你说共势便可共势，拿什么叫人信你？你可有国中援手？可有边郡佐证？可有一国已然应诺？若都没有，不过是把六国最难办的事，说成了嘴边一张图。”

苏秦听得出这是有意逼问，却也知道，对方所问，正是他此刻最痛之处。他胸中有大势判断，却无寸进之资；他看见列国共患，却拿不出任何一国已先点头的凭据。于是这一刻，厅中忽然静了些，连上首那赵臣都不再开口，只望着他，看他如何作答。

苏秦缓缓道：“今日苏秦来此，本就是为求一个使人肯信的起点。若天下大计必待势位既成后方可说，那便永远只轮到已在局中之人说话；而局外之见，纵能救局，也只能死在门外。赵君若问苏秦今日凭什么，不过凭一点：秦患非虚，东方终不能久散。今日不听，明日未必不验。”

那赵臣手指在案上一顿，像是被这几句话触到一点心事。然而片刻之后，他还是轻轻叹了一声，道：“你之言，不是全无所见。只是此事太大，非一朝一夕可定。赵国自有赵国眼前之务，边事、朝议、邦交，哪一样都不能空放。你今日所论，我可以记下；至于起不用你，却不是凭你这几句便能定的。”

这话并不算苛刻，甚至可说留了体面。可苏秦一听，便知此路终究未开。所谓“记下”，多半也不过是留在案头，随诸多文牍一同沉去罢了。他仍再拜道：“愿君上熟思。”

那年轻门客见状，忽又添了一句：“先生若真有心，不妨先在赵地多走几处，多结识几位知名之士。待声名稍著，再来言天下，或许更有人肯听。如今这样，太早了。”

太早了。

苏秦退出偏厅时，这三个字一直在耳边盘旋。他知道，对方这话未必全是讥讽，甚至可以算一句实话。只是正因实话，才更刺人。天下不是不许人言大计，而是不许无名之人先言大计。你若布衣，便该先求名；你若求名，便多半已被无数琐碎所磨，未必还能守住那一口初起的天下气。

离开那处门第时，邯郸日色已偏。长街之上，酒肆喧哗，贵游车骑鲜衣怒马，自他身边穿行而过。有人高谈兵事，有人争说朝局，也有人在楼头临窗看他这等寒士牵马独行，眼里并无半点停留。苏秦走到街角，回头看了一眼方才出来的高门，忽觉得方才那一席问答，好似已被门内车马声立时淹没，连余波都无。

他在城中歇了一夜。夜里灯下重整书简，把白日里那几句问难又逐一想过。越想越知，赵臣并非不能识他言中之理，只是大策太大，而他说话之人太轻。世间门槛，往往不是论你说得对不对，而是先论你有无资格把这话说出口。想到后来，他心中反而生出一种近乎执拗的念头：既然赵尚如此，那么不如索性往西去，去看一看天下最强之国的门庭，到底高到什么地步。

若连秦也不肯听他一言，那他便该彻底认清，自己如今究竟站在什么地方。

于是数日之后，苏秦离赵西行。

越往西走，山川形势越显险峻。道路渐渐收束，关隘与戍卒也日见森严。行旅之中，说起秦地，多半声音不自觉低下来。有人夸其法令整肃，有人怨其关卡苛严，也有人只摇头道一句：“强国之地，容不得闲人。”苏秦听在耳中，并不作声，只把这些细处都记在心里。

待他到了秦地一处权贵宾舍之外，已是薄暮时分。

那宾舍建在大道旁，外有高墙，内有重门，门前立着数名持戟守卒。阶下停着两乘华车，车后又有随从十余人，显然主人不是寻常人物。苏秦站在门外，看见一名衣饰华贵的门客正与守卒谈笑，言辞间提及咸阳、朝中、上卿，旁边人无不低首相从。只这一眼，便知这里比赵地那处门第又高出了一层。

他整了整衣冠，趋前通名，道：“洛阳苏秦，自东方来，有策愿陈，求见贵人。”

守卒先看他衣袍，再看他马匹，神色里便有了几分懒散。其一人道：“求见何人？”

苏秦道：“愿见门内主事之人，言天下西东之势。”

那守卒听了，笑意更明显了些，回头与旁边同伴对了一眼，道：“近来东方来的说士倒不少，个个都说自己知天下。你要言天下，也该先有个引见名帖。没有名帖，没有荐书，空口一句‘有策愿陈’，便想入此门？”

苏秦道：“策若可用，何必先问名帖。”

话音刚落，旁边那华服门客已失笑出声。他走下两级台阶，上下打量苏秦，像看什么新鲜物事一般，说道：“东方游士都这般大口气么？你既说策可用，不妨先说来听听，也好叫我等知道，今日这寒风里拦门的，究竟是个狂士，还是个真有见识的人。”

苏秦知道此时若退，连一线机会都没有，便直视着那门客道：“秦之强，天下共知。然秦虽强，若使东方惊惧而共谋，则秦亦不能恣意东下。今东方诸侯各怀私心，故秦得以步步进逼。若秦上层真欲久制天下，不可只恃兵锋，也当知东方共惧一旦成势，将成何等局面。”

那门客听到“东方共惧”四字，脸上笑意未收，眼底却掠过一丝轻蔑：“原来又是纵横并势这一套空话。我还当你有什么新鲜见识。东方若真能同心，何至今日？六国若真有胆共谋，又何至让秦兵压到这般地步？你一个布衣，倒替天下诸侯先把梦做圆了。”

守卒们听了，都跟着低笑。有人甚至把戟柄往地上一顿，故意发出一声闷响，像在提醒他莫要多缠。苏秦并未被这些笑声逼退，仍道：“天下未并，不等于永不能并；诸侯未合，不等于不可因势而合。贵人若但以今日之散，断他日之不能合，未免看轻了乱世逼人之力。”

那门客闻言，神色微冷，语气也淡了下来：“你倒会强辩。只是秦门之前，不缺会说的人，缺的是值得一听的人。你如今名不著，位不立，连一纸荐书都无，纵说得天花乱坠，也不过是途路客口中狂言。你想见贵人，先问问自己配不配。”

最后一句，说得并不高，却像一根细针，极稳地刺进人心里。

苏秦站在阶下，手指按着书囊边缘，掌心已微微发紧。他不是不想再争一句，只是忽然明白，再争下去，也不过是让这些人多添几分取笑罢了。高门之外，真正压人的往往不是震怒，不是喝斥，而是这样轻轻一拨，便把你整个人连同你的策论一道拨进尘土里，连发作都显得可笑。

那门客见他不语，便转身欲上阶。走了两步，又像忽然想起什么，回头淡淡道：“你若真想求仕，不如先学会一件事。天下先看人，再看言。人若不值，言虽有理，也无人肯替你认真。”

说完，袖手而入。

重门在眼前缓缓合上。门内灯火亮起，映得门外暮色更冷。守卒们已不再看他，仿佛方才这几句讥诮，不过是闲时取乐的小事。大道上车声辘辘，风从关中平野卷来，带着土腥与寒意。苏秦在门外立了许久，才慢慢转身。

这一回，他没有像在赵地那样立时生出不平，反倒只觉得胸口像压着一块冷铁。那块铁不烫，也不烈，却沉得厉害。

天下确是只认势。

不是无人知道他的话有几分道理，而是这些道理，若出自一个没有门地、没有名声、没有凭附的人口中，便先天轻了大半。你尚未入局，旁人便已替你把话贬成了空谈。

他沿着大道独自西行一段，眼见城舍渐稀，暮色全沉，才在一处枯水沟旁停下。四野无人，只有远远近近几处戍火，在夜色中时明时暗。天上无月，风却极硬，刮过枯草，响声簌簌，仿佛有无数细碎冷语，从四面八方一齐逼来。

苏秦把书囊解下，放在身侧，独自坐在土坡上。

白日里赵臣那句“此事太大”，秦门客那句“先看人，再看言”，此刻都像被夜风重新吹回耳边。前者是顾虑，后者是轻贱；可说到底，指向的却是同一件事：他看见了天下，却还没有资格叫天下听见自己。

想到这里，苏秦低头看着膝前那囊书简，忽觉自己从前虽也知寒士难进，却到底还存着几分侥幸。如今才知，这世道并非先给你说话之地，再看你说得如何；它往往先问你凭什么站在这里，之后才轮得到言辞。

这一层关节既被看破，许多先前模糊的想法，反倒因此渐渐聚拢起来。若只为求一官半职，他大可像无数游士一般，揣摩一国君臣眼前所喜，收起那些太大的话，只求先换一席食禄。然而真走到那一步，也不过沦为门下清客，再难谈什么天下共势。

若要使诸侯听从，先得使诸侯不能不闻其名。

若要使自己所见不死于门外，便不能再只凭一腔孤愤、一卷旧简，逢门便叩，逢人便说。

他想到这里，缓缓抬头，望向东方无边夜色。那夜色里隔着赵、燕、魏、韩、齐、楚，也隔着无数尚未开启的门庭、尚未开口的朝堂。路仍旧远，门槛仍旧高，可他心里却有一处，比初到赵门、初立秦阶之时，更冷，也更定了。

人已受挫，志却不能就此折断。

既然天下先看人，再看言，那便先去铸一个让天下不能轻看的自己；既然大计不能凭空落地，那便先回头，把那些零散念头打磨成真能惊动诸侯的锋芒。

风愈发紧了。苏秦把书囊重新背起，起身时，膝上沾着一层薄土。他拍也不拍，只向来路看了一眼，随即转身向东。

这一去，不是认输。

这一去，是回头磨剑。
