# 第二回 说赵未成又西行入秦 两重门前受辱方知世只先看名势

次日天未亮，苏秦便离了那处荒坡。

他没有回洛阳，先南下邯郸。一路上，他把昨夜那点念头在心里翻来覆去地磨：若六国仍各守门户，迟早要一国一国地被秦压过去。可这念头若要拿去见人，总不能还像白日里那样，逢门便叩、逢人便试。

到了邯郸近郊，城门前车马正盛。

他先打听赵国几处近臣宾馆所在，寻到一处门第，照旧通名求见。这一回门下执事倒没有立时赶他，只让他略说一二。苏秦立在阶下，压低声音道：“赵据山河之险，本可与秦争一时之势。然若赵只恃自强，坐看韩魏受逼，则纵有甲兵，终成独木支危墙。”

这几句终于叫门里人多看了他一眼。

他被召进偏厅，见着一位赵臣与一名年轻门客。厅中案上铺着军报和边图，显见赵廷这些日子议的也不是太平事。赵臣听完他的策，沉吟许久，末了只道：“此事太大，非一朝一夕可成。”

那年轻门客更把话说得直些：“你说得虽高，可凭什么叫人信你？你有哪国已点头？有哪郡佐证？若都没有，不过是把六国最难办的事，说成嘴边一张图。”

这话刺得极准。

苏秦心里明白，对方不是全不信，只是不肯拿一国之议来押一个无名寒士。厅中再往下问，问到最后，仍还是落回那一句看不见却处处都在的话：

你是谁？

你凭什么？

从赵门出来，苏秦没有再逗留。

他忽然生出另一层心思。赵疑的是局太大，人太轻；那若去更西边、更强的秦，又会如何？几日后，他竟真的西行入秦。

越往西，道路越险，关卡也越严。行旅之中一提秦地，人人声音都压低几分。待他到了秦地一处权贵宾舍之外，天已将暮。高墙重门，持戟守卒，阶前又停着华车，气象比赵门还高出一层。

苏秦上前通名：“洛阳苏秦，有策愿陈，求见贵人。”

守卒先问荐书。

他答没有。

旁边一名华服门客闻声走下台阶，打量他一眼，便笑了：“东方游士都这般大口气么？既说有策，不妨先讲来听听。”

苏秦于是道：“秦虽强，若使东方惊惧而并势，秦亦不能恣意东下。今日六国未合，不等于来日不可因势而合。”

那门客听到“东方并势”四字，脸上笑意一点没收：“原来又是这一套。六国若真能同心，何至今日？你一个布衣，倒先替天下把梦做圆了。”

苏秦仍不退：“天下未并，不等于永不能并。诸侯未合，也不等于不能因势而合。”

那门客听罢，神色冷了下来：“秦门之前，不缺会说的人，缺的是值得一听的人。你如今名不著，位不立，连一纸荐书都无，纵说得再好，也不过途路客口中狂言。”

这句话，比赵臣那句“此事太大”，伤得更深。

因为它把问题说穿了。

不是他说得全无道理，是他说这些话的人太轻。天下不是先听你说什么，而是先看你是谁。人若不值，言便先轻；名若不立，策便先死在门外。

苏秦在门前站了很久，终于一句话也没再争。

夜里，他独自坐在城外枯沟边，风从关中平野上卷来，硬得像铁。他把书囊解下，放在膝前，先想赵门的疑，又想秦门的辱，到最后忽然明白了一件事。

若只为一官半职，他大可学无数游士那样，收起这些太大的话，先去迎合一国一时之需。可若要让诸侯真听见，便不能再拿一身散乱念头去撞门。

既然天下先看人，再看言，那便先去磨出一个让天下不能轻看的自己。

既然大计不能凭空落地，那便先回头，把这些零散判断磨成一套真能惊动君臣心胆的锋。

想到这里，他缓缓起身，重新背起书囊，转身向东。

这一去，不是认输。

这一去，是回头磨剑。
