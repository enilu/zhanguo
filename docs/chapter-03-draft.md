# 第三回 败归洛阳先受家门轻看 长夜摊图之后苏秦始把六国织成一线

苏秦东返时，脚程比来时慢了许多。

来时他胸中还有一股试天下的锐气，如今回洛阳，锐气未尽，身上的风尘却先重了几层。几日后，洛阳旧城已在眼前，城门、街市、旧巷都还和离家时差不太多，只有他自己像被外头风沙磨薄了一层。

入城之后，旧识远远看见他，多半先笑着问一句：“苏子回来了？这一趟可曾得哪国赏识？”

他只答：“尚未有成。”

那笑意便总会淡去几分。

回到家中，兄长、嫂氏都出来迎他。兄长问得还算克制：“这回可曾寻着门路？”嫂氏看了看他旧袍、瘦马，叹气道：“家里原还指望着你出去能得些起色，如今人是回来了，倒比走时更清减了。”

话不算刻薄。

也正因不刻薄，才更伤人。

晚饭时，席上无多言。兄长后来把他叫到廊下，低声道：“我知你心高，也知你这些年不是胡混。只是人总要先有个落脚处。你若还能寻个稳妥差事，不妨先做着；那些诸侯大计，等日后有了凭依，再慢慢图也不迟。”

苏秦听完，什么都没辩，只道：“兄长放心，我心里有数。”

那一夜，他没有睡。

屋里灯火如豆，风从窗缝里钻进来，把火苗吹得一晃一晃。他把赵门、秦门前听来的每一句轻慢都在心里过了一遍，先觉羞，继而觉愤，最后却把这股愤意自己压了下去。

怨旁人容易。

真看自己难。

赵门问他“谁肯先应”，秦门问他“凭什么叫人信你”，这些话刺耳，刺的却都在实处。他能看见秦之强，也能看见六国各自的惧意，却还不能把这些零散判断织成一套足以压住诸侯疑心的完整说辞。

想到这里，苏秦把书囊拉到膝前，一卷卷旧简都取了出来。

这一夜，他没再想怎么受辱，只开始想六国究竟该从哪里下针。

秦据关中，法令整，国势长。

韩魏当其冲，最先知痛。

赵有山河与甲兵，可当锋，却不能独挡。

燕偏北而势单，最怕到最后无势可借。

齐远而富，楚广而大，看似离祸稍远，实则越到后头越难独保。

他把这些一点点排开，再一点点往回拢。

若真要制秦，不能先求人人尽信，只能先择一国最痛之处破门，再逐次牵动旁国，使其利害暂时归到一路。

鸡鸣时分，东方微白时，苏秦终于把最后一卷旧简放下。

这一刻，他心里不再只是一个模糊念头，而是一条渐渐成形的路。不是再去逢门便叩、逢人便试，而是拿着一套能逼近国运、震动君臣的说辞，按着次序，一国一国地去走。

又过数日，他终日闭门，少出门外。

家里人只道他受挫太过，不愿见人；乡里旧识偶尔上门，也只从门缝里看见他伏案不出，地上散着写满删改痕迹的帛片，便都摇头，只当他还在做不切实际的大梦。

只有苏秦自己知道，这一回不是空想。

又一夜将尽时，他终于整束停当，把删定过的策简重新编好，以麻绳束紧。天色将明未明，他推门而出，晨风吹在脸上，比前些时日更冷，却也更清。

这一次再上路时，他身上带着的，不再只是几卷旧简，也不只是一口受辱后的不平。

他带着的，是一套已经渐渐成形、要去叩动诸侯心胆的天下策。
