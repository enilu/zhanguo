# 第三回 寒门苦读长夜磨剑 败士回身立说纵横

苏秦自秦地折返，一路东行，脚程却比来时慢了许多。

来时他虽寒，胸中到底还有一股试天下的锐气；此番归去，锐气未尽，衣上风尘却先重了几层。旧冠失了光泽，袍角也多了两道磨裂。那匹瘦马本就不壮，经过连番奔波，更显得肋骨嶙峋。官道之上，春寒虽已比冬末略缓，吹在人面上，仍带着一股不肯轻饶人的硬意。

一路之上，苏秦话极少。偶有同路行旅与他搭话，问他是往何处去，他也只淡淡答一句“归洛阳”。旁人听了，多半也不再追问。战国年间，游士来去本多，成与不成，全写在衣冠气色里。像他这般带书而返、神色沉静的，十之八九便是碰了一鼻子灰，回乡再寻出路的。别人看得出，苏秦自己自然更知道。

几日后，洛阳城外旧道已在眼前。

洛阳毕竟是周旧都，虽王命不行，城廓气象却仍在。只是这份旧都气象，落在如今苏秦眼里，非但未能添几分宽慰，反倒更叫人心里发涩。昔日离乡时，他也曾在这条路上勒马回望，自觉胸中有策，迟早能凭一身口舌在诸侯门下挣出个名字来。那时城郭依旧，柳树依旧，道旁酒肆依旧；如今再回来，一切都像未动，偏偏只有自己像被外头风沙磨过了一层。

他牵马入城，街上也有旧识远远看见，先是一怔，随即便笑着招呼道：“苏子回来了？这一趟游说，可曾得了哪国赏识？”

这话问得不算恶，甚至还带几分寻常人情。可苏秦听在耳中，却像衣上尘土忽又重了些。他只拱手道：“路远劳顿，尚未有成。”

那人“哦”了一声，脸上笑意便淡了几分，口里仍道“无妨无妨，来日方长”，眼神里却已带出些不以为然来。苏秦看得分明，也不多说，只牵马继续往前。一路上又遇见两三个旧识，问的都差不多。有的人还肯安慰一句，有的人则干脆只扫一眼他那匹瘦马和旧书囊，便已把结果猜了七八分。

等他走到自家旧巷时，日头已偏西。巷中砖墙斑驳，檐角低窄，几个孩童在土墙边追逐打闹，见外人牵马进来，先都停住脚步，待认出是苏家的人，便又嬉笑着跑开了。门前一株老槐还在，只是枝叶未盛，干瘦枝条横在暮色里，像一只只瘦硬的手。

苏秦在门前停了片刻，方才抬手叩门。

开门的是家中小侄，先愣了一下，随即朝里头喊道：“叔父回来了！”

这一声喊出去，院中便有脚步声杂乱起来。须臾间，兄长、嫂氏都迎了出来。兄长见他风尘满面，先是怔住，继而皱眉道：“怎回得这样快？”话一出口，似觉不妥，又改口道：“我是说，你这一去时日不长，可曾在外头寻着门路？”

苏秦听了，心里微微一沉，却仍道：“赵、秦都走过了，暂未得用。”

兄长“嗯”了一声，脸上失望之色虽极力压着，到底还是露了出来。嫂氏站在一旁，看了看他身上旧袍，又看了看那匹瘦马，不由叹了口气，道：“你这一去，家里原还指望着能得些起色。如今人是回来了，倒比走时还更清减些。”

这话仍不算刻薄，只是太实。实话最伤人，正如赵门与秦门前那些话一样，不一定声高，却句句都往最难受处落。

苏秦没有辩白，只牵马入院，把书囊卸下。院中器物皆旧，灶边冷灰未净，堂前木案也已有些掉漆。这本是他自小看惯的家景，可此时再看，竟比从前更觉逼仄。人若在外尚有一口心气，回到这等狭窄院落里，才知所谓“无成而归”四字，究竟沉到何等地步。

晚饭时，家中气氛并不热闹。兄长问了几句外头道路与列国情形，本是想探听是否尚有可走的门路；待听见苏秦说赵国疑其策太大、秦地嫌其名太轻，便渐渐不再多问，只低头吃饭。嫂氏替众人添羹时，嘴里轻轻叹了一句：“人家同样是读书，有的能做小吏，有的能入豪门幕中，再不济也能给人教书换口饭吃。你倒好，胸中尽是天下大事，家里这几口人的生计，却先顾不得了。”

兄长皱了皱眉，像嫌她说得太直，可终究没有喝止。苏秦握箸的手微微一停，片刻后才又放下，只道：“是我无能，累家中挂心。”

嫂氏见他这样答，倒也不好再说什么，只把碗放下，转身去了灶间。可那一句“胸中尽是天下大事”，却像一根细刺，留在席间，谁也不再碰它，谁也都绕不过它。

饭后，兄长把他叫到廊下，低声道：“我知你心高，也知你这些年并非胡混。只是人活在世，总要先有个落脚处。你若还能寻个稳妥差事，不妨先做着；至于那些诸侯大计、天下纵横，等日后有了凭依，再慢慢图也不迟。”

这番话说得已极克制，分明是出于家人的担忧，并无半分恶意。可苏秦听完，心里却忽地生出一阵说不出的疲惫来。外头权门之前，人家嫌他名轻；回到家门之内，连兄长也劝他先把那些大话收起来，换一条稳妥小路。仿佛天下上下，都在逼他承认：你看见的那些东西，并不值当为之折腾这一身穷困。

他良久才道：“兄长放心，我心里有数。”

兄长看了他一眼，像还想再说，却终究只是叹了口气，道：“有数便好。你既回来，便先歇一歇。”

那一夜，苏秦并未早睡。

屋中灯火如豆，风从窗缝里钻进来，把火苗吹得一晃一晃。旧席铺在地上，书囊就搁在手边。他独自坐着，白日所见、一路所闻、赵门与秦门前那一张张脸，又都在灯影里浮了上来。嫂氏那句带怨的埋汰，兄长那句“先寻个稳妥差事”，也都掺进其中，与那些门客守卒的话混作一处。

苏秦先是觉羞，继而觉愤。可愤意刚起，便又被他自己压了下去。

他已经受够了把一切都推给旁人。赵臣不敢轻用他，秦门不肯正眼看他，家中亲人不信那些虚远大计，这些当然都叫人难堪。可若只把这些都算成“世人不识我”，那便与路上那些只会怨天尤人的狂士又有何异？

想到这里，苏秦伸手把书囊拉到膝前，慢慢解开，取出一卷卷旧简来。

竹简仍是那些竹简，上头字迹却有新旧深浅，哪里是途中添改，哪里是旧日成稿，一眼都能分明。他把自己这些时日说赵的话、在秦门前被打断的话、一路上心中翻涌过的念头，全都重新摆在眼前，像把一个失败之人散落一地的骨头，又一寸寸拾起来看。

看得越久，心里越冷，也越亮。

他终于承认，自己先前不是没有见识，而是见识尚散；不是没有胆气，而是胆气先行，分量未足。他能看出秦之强，也能看出六国各自的惧意，却还不能把这些零散判断织成一套足以压住诸侯疑心、逼他们不得不正眼看来的严密说辞。赵臣问他“谁肯先应”，秦门客问他“凭什么叫人信你”，这些话固然刺耳，却都刺在了实处。

列国不缺说客。

缺的是一个能把山河利害、国运轻重、远近缓急一齐摆到案上，让君臣听完之后再不能只把他当作途路游谈的人。

苏秦想到这里，忽然起身，从墙角取来一块旧木板，又把布囊中的一幅残旧地图摊开压在其上。图上山川关塞画得简略，许多地方还是他从前自己补记上去的。他伸手自西而东慢慢划过，口中不出声，心里却把秦、韩、魏、赵、燕、齐、楚一一放回原位。

秦据关中，地利最厚，法令最整，能并力东出。

韩、魏当其冲，最先受压，也最先知痛。

赵有山河甲兵，可以当锋，却不能独挡。

燕偏北而势单，若无外援，终难自安。

齐远而富，楚广而大，看似离祸稍远，实则越到后头越难独保。

这一夜，苏秦几乎不曾合眼。

灯花一再爆开，又一再被他挑去。案上竹简时开时合，木板上被他以指代笔，反复划出一道道无形的线。哪国先动，哪国后应；谁最惧秦，谁最爱惜羽毛；谁可先警以亡国之祸，谁可重诱以并势之利；何处该说近患，何处该说远略；哪些话不能说满，哪些话必须一开口便压住满堂心神。他把这些反复推演，推到后来，连先前所受那些轻慢与羞辱，也像被投入炉中的杂质，一点点烧尽，只剩下更硬的东西留在心里。

鸡鸣时分，东方微白，他才放下最后一卷竹简。

这一刻，他心中第一次不再只是一个模糊的念头，而是一条渐渐成形的路。

秦之强，非一国可当。

六国虽异心，却同惧秦。

既同惧秦，便可借其惧而并其势；既可并其势，便不必先求一国尽信，只消先择其最痛处破门而入，再逐次牵动诸侯，使其利害暂时归于一路。

想到这里，苏秦只觉胸中原先许多散乱之气，忽然有了筋骨。

他并非一夜之间便成了什么天下无双的纵横士，可至少到此时，他已不再是前些日子那个逢门便叩、逢人便试的寒士了。他终于明白，自己接下来要做的，不是去求谁怜惜，不是去碰谁一时兴起的善意，而是拿着一套真正能逼近国运、震动君臣的说辞，按着次序，一国一国地走进去。

此后数日，苏秦少出门，终日闭在屋中。

家里人只道他受挫过甚，心气折了，一时不愿见人；乡里旧识偶有上门，也只从门缝里见他伏案不出，案旁竹简堆叠，地上散着些写满删改痕迹的帛片，便都摇头叹息，只当他仍在做那等不切实际的大梦。

唯有苏秦自己知道，这一回不是空想。

白日里他读旧书，校旧策；夜里则对着地图与竹简，将列国利害再推一遍。推到艰深处，便停下来重写，写了又删，删了又改。窗外春意渐浓，院中老槐一点点抽出新芽，他却像全然不觉。人是瘦了些，眼里的光却一日日定下来。那光不再像前两回那般只是少年寒士的不服之气，而更像刀锋在石上来回磨过之后，渐渐逼出的冷亮。

又过了些时日，一日午后，兄长推门进来，见他仍伏案而坐，不由皱眉道：“你这些日子闭门不出，到底在图什么？莫非还要再去游说诸侯？”

苏秦闻言，抬起头来。

这一抬头，倒叫兄长微微一怔。眼前人还是那个人，衣袍仍旧，面容仍瘦，可神气却与前番败归时大不相同。若说那时他身上还有风尘未定的狼狈，此刻那份狼狈已沉进骨里，反倒把目光磨得更定。

苏秦缓缓道：“前番出去，我是见了门便叩，见了人便试，所以虽知天下之病，却无医病之方。如今不同了。”

兄长问道：“有何不同？”

苏秦把手按在案上那幅残图之上，道：“秦之强，不在一将一战，而在国势。欲制其东出，也不能只凭一国逞勇。六国虽各有私算，却都有惧秦之心。若能先择一国最痛之处言之，使其知独守必危，再推及诸侯共同利害，则合之虽难，未必不可为。”

他说这几句时，声音并不高，却句句都比从前稳。兄长虽未尽懂那套国势纵横之理，却也听得出，这已不是先前那种一腔热气便想去撞天下门槛的言语了。他沉默片刻，道：“你还要再去？”

苏秦道：“要去。只是这一回，不再乱撞了。”

兄长望着他，许久没有说话。院中风过，新抽的槐叶微微作响。末了，他只低低叹道：“你既决意如此，我也拦不住。只是此番若再去，便当真要拿出个样子来。”

苏秦道：“正为此，才有这些日夜。”

兄长听了，点点头，转身出去。走到门口，又回头看了他一眼，眼里虽仍有忧色，却已少了几分先前那种把他当作失路之人的无奈。

又过一夜，天将明时，苏秦已整束停当。

他把删定过的策简重新编好，以麻绳束紧；旧冠依旧，旧袍依旧，只是在衣带之间，多了一股说不出的整肃之气。院中尚无人起，他独自推门而出，只见东方天际微微发白，巷口薄雾浮动，晨风吹在面上，已不再像前些日子那般只有寒硬，倒像把人胸中那团久熬的火，吹得更清了一层。

苏秦回头看了一眼自家旧门，没有多停。

这一次，他再上路时，心中所负的，已不只是几卷旧简，也不是一腔受辱之后的不平。

他负着的，是一套已经渐渐成形、要去叩动诸侯心胆的天下策。

说罢，他整了整衣襟，束简上肩，踏着将明未明的天色，径自出巷去了。
