# 第四回 再入邯郸苏秦不先言大义 一言先刺赵国最怕独受秦锋

天还没亮，苏秦已在洛阳北门外。

这一回他先入赵。

赵近秦，知秦之逼最深；赵自强，也最不甘坐守困局。若合势之议要打开第一道真正的门，赵最可先试。只是这次他不再乱撞门路，而是先寻最能听懂痛处的人。

入邯郸后，他先住一夜，把沿途所闻、赵国近来边事和门下宾客往来又在灯下细细过了一遍。次日清晨，才择一处赵国重臣宾馆而去。

门下执事照例先问荐书。

苏秦说没有，又添了一句：“上回我入赵，只言西患。此番再来，却是欲言赵国未必亡于秦兵之锐，倒可能先困于自强而不得其势。”

执事听到这句，果然不敢立刻打发，只转身入内。

不多时，一名青衣门客出来看他。先上下打量一遍，才道：“你这句话，不像寻常危言。既如此，便进去说。”

苏秦被引入一间议事小室。室中坐着一位年长赵臣和一位常议军国之务的年轻议者。上首那人看着他道：“赵国何以会困于自强？”

苏秦答：“赵诚强国也。士卒敢战，山河可守。可若只恃本国之强，坐看韩魏受逼，则纵有甲兵，终要独受秦锋。”

那年轻议者立刻问：“诸侯各怀鬼胎，凭什么肯随赵而动？”

苏秦道：“不必先问谁肯为赵出力，要先问谁愿不愿各自去死。韩魏最先知痛，燕惧孤弱，齐楚虽远，也知若赵亡，则东土再无屏障。诸侯未必同心，却都不愿见赵先倒。”

两人听到这里，都不再像先前那般只拿他当寻常游士。

次日，他便被引入赵宫。

赵王端坐上首，殿侧列着数名近臣。赵王先看了他一眼，便道：“闻卿言赵国自强将成孤。此语不小。细说。”

苏秦行礼后道：“今之秦，不可畏于一将一战，而可畏于国势绵长。赵若独恃甲兵之强，韩魏迟早尽削，赵终成秦东出所先攻者。赵欲自安，不在独战，而在先使天下之惧暂时并成一线。”

一位老臣出列反问：“诸侯异心积久，便算一时可并，又凭什么不转眼便散？”

苏秦答：“正因未必永固，所以更须先成。能暂同，已足以改一段大势；能改一段大势，已足以使秦不敢轻举。”

赵王听罢，又问得更实：“若赵先倡其议，于赵何利？”

苏秦道：“其利有三。其一，可分秦压。其二，可居中势。其三，可使天下之先机不尽落在秦手。”

说完之后，殿中静了很久。

赵王终于开口：“若寡人令卿先往燕国一试，卿当何说？”

苏秦答得不慢：“说赵，重其锋；说燕，当重其路。燕地偏远而势单，若赵受压而北方无势可借，燕终难自全。燕不必先争主，只须先入局。”

赵王听到这里，脸色才真正动了一下。

末了，他道：“卿且留邯郸，馆于外舍。待寡人与诸臣再议。”

这不是起用，也不是拒绝。

可对苏秦而言，已是他真正推开的第一道门。
