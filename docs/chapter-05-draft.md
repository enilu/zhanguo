# 第五回 赵廷再问本国得失何在 苏秦转辞入燕先说远国最怕失势

苏秦留在邯郸外舍，没有因赵王一句“待再议”便生出半分松气。

他知道，门既开了一道缝，便要趁赵国君臣心里那点震动尚热，再往深处压一层。果然，不出两日，赵廷之中便已分成两股议论：有人说此策太大，一旦高倡合从，秦必先怒于赵；也有人说赵正因近秦，才不能把这话只当危言。

再入赵宫时，赵王开口便问：“前回卿言赵若独自强，终将独受秦锋。今日寡人只问一件事：赵若先倡合从，于赵究竟得什么？”

这一问已不再是试他会不会说，而是要听本国所得。

苏秦答：“赵若不动，其利不过守眼前强国之名，其害却是韩魏愈削，赵愈先当秦锋。赵若先倡其议，其利有三：可分秦压，可居东方共势之中，可使诸侯先按赵的痛去看天下之痛。”

殿上一位武臣出列：“说得热闹。若赵先倡其议，最后流血出兵的，岂不还是赵？”

苏秦转向那人：“将军所忧，正是赵若不先图之，迟早也要独担的事。今赵先倡其议，便是逼诸侯不得只拿赵来试秦锋；若赵不动，后面便连逼人共应的名也没有了。”

这一番答完，殿中气口已与前回不同。

赵王最后没有当场定局，只道：“卿先往燕。一则试北方远国之意，二则也让寡人看看，你究竟能不能因国设辞。”

数日后，苏秦北入燕境。

这一路上，他已不再拿“赵将独受秦锋”那一套去压人。燕远于秦，也弱于赵，燕最怕的不是“我太强而将独当”，而是“我离刀远，所以总想着慢半步，慢到最后连可借之势也没了”。

入燕廷后，主事之臣一开口就道：“若还是说赵那套，先生便不必费舌。”

苏秦拱手：“臣若拿说赵的话来说燕，便是误燕，也误臣自己。”

燕臣道：“既如此，燕之病何在？”

苏秦答：“燕之病，不在争先，而在太肯自宽。赵若受压，韩魏若削，东方大势一坏，燕虽一时暂无秦兵临城之急，也会渐渐失去一切可借之路。到那时，不争亦不得安。”

又有燕臣问：“便算燕愿附其议，所得何在？跟在强国后头应声，岂不徒壮他人门面？”

苏秦道：“弱国最怕的，不是不能居首，而是始终不在局中。燕若先附此议，未必争名，却先占一席；往后北方有势可借，燕便不是局外自守之国。”

这几句一出，燕廷神色已变。

末了，主位那人缓缓道：“赵人说赵，说得重其锋；你说燕，说得重其路。看来先生不是一篇旧策走遍天下的人。”

又过数日，燕廷终于回话：愿与赵相通其议，先观北方诸国如何应之。

这答复还不算定。

可苏秦出燕都时，心里已知赵、燕这两根线，总算真搭上了。
