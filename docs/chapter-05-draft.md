# 第五回 危言警赵陈唇亡势 远略惊燕借唇舌锋

苏秦留居邯郸外舍，未曾因赵王一句“待再议”而稍露松气。

他很清楚，朝堂上那一席话，不过是把门真正推开了一道缝。门既已开，便不能只等旁人来定，须趁赵国君臣心里那点震动尚热，再往深处压上一层。若让它冷下去，合势之议便仍可能停在“说得有理”四字上，再难化作真正在列国间传动的次序。

果然，未过两日，赵廷之中便起了两股议论。有人道苏秦之言虽大，却未免过险，一旦高倡合从，秦必先怒于赵；也有人道正因秦强而赵近，才更不能把这话只当游士危言。两派争持不下，连外舍里来传话的侍吏，神色都一日一变。苏秦听完，只淡淡点头，并不追问。他知道，争便说明已入其心；若连争都不争，那才是真正无望。

这一日，前回殿中那位年轻议者奉命来见，语气较前已和缓许多，却仍带几分试探：“君上与诸臣近日所议，多在于合势固然可图，可赵若先出头，究竟得多还是失多。先生若还能言赵之利害，明日可再入议。”

苏秦看了他一眼，道：“前回我说的是赵若不借众势，将先困于自强；这一回，我说的是赵若肯先倡其议，不只可免独受秦锋，更可借此定东方之位。前者是避祸，后者才是取势。赵若只看避祸，便仍嫌其策太大；若看取势，方知此事不可后人。”

那年轻议者心中一震，面上却只道：“先生明日可当众说这句话么？”

苏秦道：“正为此而留。”

次日再入殿时，赵廷比上回更肃。除了赵王与近臣之外，又添了几位掌兵议政之臣。显然，这一次不只是看他会不会说，而是真要问：赵若动，当如何动；赵若不动，又将失什么。

赵王开口便道：“前回卿言赵若独恃其强，终将独受秦锋。今日寡人要问另一层。若赵先倡合从，于赵何利？寡人不能只听‘天下大义’，总要先知本国所得。”

此问一出，殿中数人都微微侧目。若答得空，苏秦此前所立便会立时发虚。苏秦却不疾不徐，先行一礼，方道：

“君上所问，正是赵今日最该先问之处。赵若不倡合从，其利不过保眼前自强之名，其害却是韩、魏愈削，赵愈先当秦锋。赵若倡合从，其利有三。

其一，可分秦压。使天下共知西患非赵一家之患，则秦再东向，不得不先顾诸侯并应。

其二，可居中势。赵地近秦而兵强，最知其痛，故最有资格先言。此议若出于远国，则近秦者未必信；若出于弱国，则强国未必服。今赵开此口，恰居轻重之中。

其三，可正名分。天下若从赵之议而暂并其势，则赵所得，不只是一时避兵之安，更是主导东方先机之势。”

殿中一位武臣闻言，立时出列道：“说得热闹。可赵若居中主议，岂不正把自己置于最前？到头来诸侯口头附和，真正流血出兵的，还不是赵？”

苏秦转向那人，答得极快：“将军所忧，正是赵若不先为之图，迟早也要独担之事。今赵若不动，韩、魏受压，秦兵仍是要到赵前；那时赵独战，诸侯更有借口观望。今赵若先倡合势，则将军所担之兵，不再只是赵兵，而多了诸侯不可全然袖手的名义。此两者，孰轻孰重，将军岂不自知？”

那武臣一时语塞，只得退下。苏秦又顺势道：

“赵之强，在能战；赵之真正大利，却不在独战，而在使天下先替赵分秦之力。君上若但见合从需分名于诸侯，不见其能替赵分祸于诸侯，便只看见一半。”

赵王听到这里，神色已和先前明显不同，便又问：“若赵先倡其议，诸侯未必尽服。卿又何以使他们各就其位？”

苏秦道：“服不服，不在先讲同心，而在先击其痛。对近秦之国，当言其亡国之迫；对偏远之国，当言其唇亡齿寒之危；对强国，当言其若不先共制秦，虽强亦终将独受其敌；对弱国，则言其若不借众势，连自保都无从谈起。诸侯不同，辞亦不可同。只要各自都从自己最深之惧处动心，合从便有可成之机。”

这一番话，说的是赵国利害，落的却已是列国次第。殿中原本还把他当作赵国一时之客的人，至此也都听出，这人胸中已不是一国之策，而是一张从赵出发、牵向诸侯的网。

赵王沉思良久，忽道：“若寡人令卿先往燕国一试，卿当何说？”

殿中几位近臣都抬起头来。此问一出，已不再是考他会不会说，而是试他能不能真的因国设辞。

苏秦不假思索，答道：“臣若说燕，不会言其兵强可主诸侯，亦不会言其足以独当西患。臣只会言三件事：燕地偏远而势单，若天下秩序坏尽，燕最难自全；赵若先受秦压，燕北处孤角，终究无大国可倚；今若先附合从，既可借强国自护，又可因先从大义而抬自身于北方诸国之列。赵之患在太强而或自恃，燕之患在不强而或自缓。其病不同，药亦当异。”

殿中一时寂然。

赵王看了他片刻，方道：“好一个‘病不同，药亦当异’。卿且先赴燕。若能使燕心动，赵国自当更进一步议其后事。”

苏秦再拜领命。退朝之时，昨日那位年轻议者随在一侧，低声道：“上回你让赵廷不能轻看，这一回你让赵廷知道，你不只是会说赵。”

苏秦淡淡道：“要成其局，本就不能只会说一国。”

数日之后，苏秦北入燕境。

去赵时，他辞锋似刀，专挑强国自恃处落下；入燕之后，那股锋气却明显收住了几分。燕地较赵更偏，城邑也更疏，沿途山风虽急，商旅与甲车却少。苏秦一路所见，心里更知对燕不可只用危赵之辞。燕所怕的，不是“我太强而将独当”，而是“我太远太弱，终有一日会在大势崩坏时无处借力”。

到了燕都之外，他并未先以“合从主议”自居，只托人传言：洛阳士苏秦，有言燕国可以不先争天下，却不可不先争自保之位。门下听见这一句，果然先起了疑意。燕国主事之臣召他入见时，开口便带几分审慎：“赵人近来好言合从，先生又自赵来。燕地偏北，向来不争天下中枢。先生若拿说赵的话来说燕，那便不必费舌。”

苏秦闻言，先拱手道：“臣若拿说赵的话来说燕，便是误国，也误臣自己。”

那燕臣目光微敛，道：“既如此，卿便说说看，燕之病何在？”

苏秦道：“燕之病，不在争先，而在太肯自宽。地偏而国小，最易想着‘秦祸先不到我’，可正因如此，一旦天下中原之势坏尽，燕便再无可借之大势。赵若倒，魏韩若弱，齐楚若各自自保，则燕虽暂无秦兵临城之急，也会渐渐失去一切可依可借之路。到那时，不争亦不得安。”

一名燕国老臣摇头道：“燕与秦之间尚隔诸国，何至说得这般迫？”

苏秦看向他，声调比在赵廷时更缓，却更深：“大夫所见，是今日疆界；臣所见，是明日次序。秦若一步步东逼，先削韩魏，再压赵，则北方之势立变。燕最不该等到赵已独困、东方已乱时，才想起借势。彼时再附，不过是被裹挟；今日先附，才叫自谋。”

另一位燕臣又问：“便算燕愿附其议，所得何在？若只是跟在强国之后应声，岂不徒为人壮门面？”

苏秦答道：“燕若今日附义，不是自降，而是先占一席。合从若真成，燕得以依众势自安，不必独惧秦锋；合从若未必长久，燕也已藉此进入北方大局，不再只是边隅自守之国。弱国最怕的，不是不能居首，而是始终不在局中。”

这几句一出，燕廷之中神色俱变。因为赵国所重者，是不愿独受其锋；燕国所重者，却正是“不能被排除在局外”。苏秦这一转辞锋，竟像顺着燕国心里那层最深的忧虑，慢慢捅了进去。

那主事之臣沉吟片刻，终于问道：“若燕欲从其后，先当如何？”

苏秦道：“先不必争主，不必争名，只先表其意。使赵知北方并非孤倡，使诸侯知燕虽远亦知唇亡齿寒。燕只需先把自己放进这条线里，后面的位次与所获，方有可图。”

燕臣闻言，互相顾视。有人仍觉得此议过大，有人却已明显动心。主位那人终于缓缓道：“先生从赵来，所说赵者重其锋，说燕者重其路。看来先生确非一篇旧策行遍天下之人。”

苏秦道：“臣若只会一策一辞，便只配在诸侯门外吃风，不配言天下。”

这一句说得平，殿中却无一人再觉得它狂。

数日之后，燕廷传出话来，愿与赵相通其议，先观北方诸国如何应之。这个答复虽不算尽定，却已足够。苏秦出燕时，北风正起，城外旌旗猎猎。他回望燕都一眼，知道自己这一步并未白走。

赵已动其强心，燕已醒其远虑。北方两线，至此总算真正连起了一段脉络。

而这，还只是合从之网最初的两根线头。
