# 第六回 赵燕两头先起合从之风 魏韩因痛不敢退齐楚因大不肯轻应

燕都归来之后，苏秦并未在一国长住。

赵、燕两处既已先后动心，合从之议便不再只是他口中一篇策论，而开始变成馆驿之间、使者往来、诸侯私议中的一股真风。可风起不等于势成，苏秦比谁都更清楚这一点。

赵肯动，是因近秦而自知不能独战。

燕肯应，是因偏远而不愿日后无势可借。

两国各有算盘，能暂时同向，已是极难。

接下来最痛的，自然是魏、韩。

这两国离秦最近，理当最该与合从一气；可越是近秦，越知道惹怒秦国的后果。苏秦还未正式入其朝，魏韩内部的犹疑便先传出来了。有人说“若诸侯果能并势，自当附之”，更多的人却说“万一合从只是虚名，秦怒先至，谁替我挡”。

这份迟疑，并不奇怪。

苏秦先入魏。

魏廷接他不算失礼，却始终带着一种既盼其言能救局、又怕其言真把局推到自己头上的犹疑。席间一位魏臣直问：“若我先应其议，秦怒先加于魏，诸侯盟名未成，魏岂不先成试刀之石？”

苏秦并不与他争意气，只道：“正因魏近秦，才最不能等别人替魏做主。魏今日最怕的，不是先应合从，而是人人看出魏最怕，所以人人都等着魏先吃那一刀。”

这话逼得魏廷不能再只拿“谨慎”二字遮过去。

入韩时，情形又不同。韩地更狭，朝臣心思也更细。韩国臣子问他：“韩若一入此局，是不是反倒成了大国之间最先要拿来安置的一块地？”

苏秦答得更直：“若韩不入此局，韩便不是‘最先被安置’，而是最先被取。小国最怕的，不是在局中居下位，而是根本不在局中。”

魏、韩都未立时拍板。

可两国都已不敢再把“合从”当作一阵游士声浪。

与此同时，更远的齐、楚也在观望。

齐自恃富强，先算的是若合从真成，齐居何位；若不成，齐又要先担什么。楚疆域广大，自重更深，忧的则是若顺着旁人起的旗走，楚是不是先失了自己的位。于是齐更偏冷观，楚更偏摇摆，都不曾轻动。

苏秦对此并不焦躁。

他对身边门客道：“齐楚若一闻赵燕有动便即尽附，那反倒不是战国。大国之所以为大国，正在它必先问自己得什么、失什么。”

于是他往来诸国，不急于逼齐楚立刻表态，只先一再把北方已有动静、近秦诸国已有惧意、若坐视北方先结势则大国日后反受其制这一层意思，慢慢送进两国耳中。

齐、楚仍旧沉着。

可也终于不敢再把合从只当作一句空话。

西边的咸阳，对这股风也不再完全无视了。

边报一份份送入关中，赵与燕使节较往常频密，魏韩私议增多，齐楚虽未尽动，却都明显开始留意“合从”二字。秦廷主事之臣看完，低声道：“若只是六国旧例相疑相结，不足为奇。可如今竟有一人能使近秦者因惧而不敢退，远秦者因势而不敢全不理，此事便不可只当作说士虚声。”

秦这才第一次，把东方正在生长的这张网，当成了一个值得细看的东西。
