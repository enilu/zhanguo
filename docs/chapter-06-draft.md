# 第六回 列国各怀疑惧心思 布衣独运合从权策

自燕都归来之后，苏秦并未在一国长住。

赵、燕两处既已先后动心，合从之议便不再只是他口中一篇策论，而开始变成馆驿之间、使者往来、诸侯私议中的一股真风。可苏秦也比谁都明白，风起并不等于势成。赵肯动，是因近秦而自知不能独战；燕肯应，是因偏远而不愿日后无势可借。两国各自有各自的算盘，能暂时同向，已是极难。至于旁的国家，绝不会因为赵、燕已有松动，便一齐跟上。

所以他接下来做的，不再是逢国便陈一篇完整大论，而更像在几国之间穿针引线，试其冷暖，调其缓急。

先热的是赵、燕。

邯郸那边，赵王既见燕国不曾一口拒之，心里便更添了几分分量。几位原本还疑其“太大”的朝臣，也渐渐改口，转而议论若真要合势，应先从哪些国家起手，怎样既不显赵独逞其强，又不失居中之位。燕那边则更多的是私下盘算：若北方真能借赵起线，燕早些附上，不但可求自保，也可借机抬高本国在诸侯间的位置。两边虽出发不同，却都已不再把合从当作空话。

这便够了。

有了赵燕为首线，苏秦接下来所要看的，便是哪几处最痛，哪几处最疑。

最痛的，自然是魏、韩。

这两国近秦最甚，按理也最该与合从一气。可越是近秦，越知道惹怒秦国的后果。苏秦还未正式入其朝廷，魏、韩内部议论便先一步传了出来。有人道“若诸侯果能并势，自当附之”；更多的人却道“万一合从只是虚名，秦怒先至，谁替我挡？”这份迟疑并不稀奇，甚至可以说正合情势。因为离刀锋最近的人，既最怕被割，也最不敢先站出来。

苏秦先入魏。

魏廷主事之臣待他不算失礼，却始终带着一种既盼其言能救局、又怕其言真把局势推到自己头上的犹疑。席间一位魏臣直问：“先生言合从，听来是条生路。可魏比不得赵，近秦太甚。若我先应其议，秦怒先加于魏，诸侯盟名未成，魏岂不先成试刀之石？”

苏秦并不与他争意气，只道：“正因魏近秦，才最不能等别人替魏做主。若魏因惧而不敢先言，秦便更知魏可逼。魏今日最怕的，不是先应合从，而是人人看出魏最怕，所以人人都等着魏先吃那一刀。魏若能表其意，不必立刻争先出兵，却至少让天下知道：近秦之国也知不可各自求活。如此，则赵、燕、韩皆须重新看魏，秦亦不能再把魏当作可先独压之国。”

那魏臣听了，沉默许久，终究没有立时驳回。因为这番话并未逼魏去做第一个担尽风险的人，却逼魏承认：若连表意都不敢，那才是真把自己交给秦来拿捏。

入韩时，情形又略有不同。

韩国朝臣中，有人更偏向求近安，心思比魏还细密几分。韩臣对苏秦道：“韩地狭而势促，最受秦压，也最受赵、魏牵连。先生劝韩附合从，韩自然知其理。可韩若一入此局，是不是反倒成了大国之间相争时最先要拿来安置的一块地？”

苏秦答得更直：“若韩不入此局，韩便不是‘最先被安置’，而是最先被取。小国最怕的，不是在局中居下位，而是根本不在局中。只要韩仍被诸侯共同看作近秦之要地，赵不敢轻忽，魏不得不顾，燕齐楚也都要算上一层，韩便尚有可借之势。若只闭门求近安，那才真是把自己变成秦与诸侯都可随手处置之地。”

这番话说得韩国君臣无不心惊。因为韩之所惧，正是“国小而常作大国缓冲”；苏秦却把这恐惧一翻，变成“你若不借合从自抬，便更只是旁人缓冲”。他说的仍旧是同一个天下大局，落在韩身上，却已成了另一种逼命之辞。

魏、韩都未立时拍板，却都不敢像从前那样把苏秦的话随手置于馆外。两国朝中开始有了同一种声音：合从未必可靠，可若众国都在往那里挪，自己绝不能全然置身其外。

与此同时，更远处的齐、楚也在观望。

齐国自恃富强，原不肯轻易把自己摆入别人起头的秩序中。齐臣听闻赵、燕、魏、韩皆有其议，多半先算的是：若合从真成，齐居何位？若不成，齐又要先担什么？楚国疆域广大，气象自重，所忧却与齐不同。楚不是不知秦强，而是不愿轻易承认自己也需跟在别人倡起的旗帜下行事。于是齐更偏冷观，楚更偏摇摆，皆未真正拒绝，也都不肯轻动。

苏秦对此并不焦躁。

他对身边门客道：“齐楚若一闻赵燕有动便即尽附，那反倒不是战国。大国之所以为大国，正在它必先问自己得什么、失什么。如今不必先逼成，只需先令其知：若他日局成而自己不在局中，所得之位只会更低。”

于是，他遣人往来，不急于求齐、楚立刻表态，只先一再把北方已有动静、近秦之国已有惧意、若坐视北方先结势则大国日后反受其制这一层意思，慢慢送入两国耳中。齐、楚仍旧沉着，却也终于不敢再把合从只当作一阵游士声浪。

这一番操作，便是苏秦真正不同于寻常说客处。

寻常说客，往往只求一时折服一人；苏秦此时想的，却是哪一国先热，哪一国须缓，哪一国只能逼其不逆势，哪一国暂不必求其尽信。他知道赵可做头线，燕可做应线，魏韩须以近秦之惧迫之不敢后退，齐楚则不可逼得太紧，须先让它们明白自己若缺席，便终会在大局里失位。

他心中早有一张天下轻重图。每一国不必都立时尽服，却要一国一国放到该有的位置上。

就在东方这张网一点点被他拉起时，西边秦廷也终于不再只把此事当作几国使者的寻常往来。

函谷关内，边报与探报一份份送入咸阳。赵与燕使节较往常频密，魏韩私议增多，齐楚虽未尽动，却都明显开始留意“合从”二字。更重要的是，报告中反复出现一个名字：苏秦。

秦廷主事之臣看完诸报，低声道：“若只是六国旧例相疑相结，不足为奇。可如今竟有一人能使近秦者因惧而不敢退，远秦者因势而不敢全不理，此事便不可只当作说士虚声。”

旁边另一人却道：“六国异心积久，纵有一人暂把它们扯到一处，也未必真能同骨。与其先自惊，不如看其后如何运转。”

前者点头：“正因未必成骨，才更值得看清。若真要防它，不在防它名盛之日，而在看它哪几处最易自裂。”

这番议论并未传出咸阳宫墙，却正是第一回以来，秦第一次把东方正在生长的这张网，当作一个值得细看的变量。

而东方这边，苏秦仍在不停奔走。

他往赵时，压的是强国自恃；往燕时，激的是弱国远虑；对魏韩，重在逼其不敢完全退开；对齐楚，则先让其知局外亦有失位之险。说到末了，六国没有哪一国是因真正信任别国而动，也没有哪一国是因信服某种空泛大义而动。它们动，皆因各自的惧、各自的利、各自不得不动的一层现实。

正因为如此，合从才第一次真有其势。

这一势还很松，还远未成骨，甚至一国一国之间都还隔着无数旧疑旧算。可它毕竟不再只是苏秦一人的话，而是同时在赵、燕、魏、韩、齐、楚心中各自发了不同的芽。

回到邯郸时，夜色正深。苏秦在馆舍灯下摊开地图，看着北方已连起的线、近秦诸国已动的心、齐楚虽远却终究未敢完全置身事外，心中第一次生出一种稍近“局正在成”的感觉。

可他也同样知道，这张网之所以能起，不过因为人人都怕秦，却还没有哪一国真愿把自己的旧算盘放下。

能让它同时发芽，已是难事。

至于能否让它真正长成，那便是更难的下一步了。
