# 第七回 秦廷闻变深宫议策 东方乘势合议盟书

咸阳宫中，春日的风吹不过重重殿门。

边报、使报、探报一层层送入内廷时，秦廷诸臣已不再把“东方近来往来频密”当作寻常外交走动。赵、燕使节相通，魏、韩虽未明言尽附，却都显出不敢逆势之态，齐、楚也开始留意北方议论。更叫秦人记住的，还是那个不断在诸国之间穿行的名字。

苏秦。

朝议之时，一位主边臣子先道：“东方六国素来疑忌甚深，今忽然往来如此频，非但因秦强，亦因有人在替它们结线。若只看诸侯，不足为异；若看那个能把诸侯暂时拉到一条线上的人，便不可不察。”

殿中随即有人应道：“臣以为不必太重其人。六国纵有共惧之心，也各怀私计。名义上的合从，未必抵得过利益上的相疑。今日看似忙乱，未必不是一阵风。”

另一位年长近臣却缓缓道：“风虽未必久，却能鼓人心。六国若先得合从之名，即便骨不成，也足以使东方士气一振。秦自不必惊，但也不可视之如无物。”

几人所论，各有其理。上首秦王始终静听，待众议稍歇，方淡淡道：“六国若果真同骨，自是异数；若只是同名，也不足使秦先乱。寡人所要的，不是争他们此刻热不热，而是看这热里哪几处最虚。记下便是。”

这句话一落，殿中诸臣都知道，秦并非轻此局，而是已把它放进了更冷的一层判断里。西土没有因东方热而躁，反倒更显出一种法度深宫里的沉稳。

与此同时，东方却果真忙了起来。

赵、燕之间使者往返渐密，魏、韩虽仍各自保留，却也不敢再对“合从”二字含糊其辞。齐、楚尚未彻底入局，却都派人来探北方真意。馆驿之间，车马昼夜不绝；文书往来，常常一日数至。往日各国使者相见，先问私利，后说大义；如今却常常先问苏秦何在、赵国何议、燕国何意、魏韩愿到何步。

苏秦就在这阵忙乱里，成了众人绕不过的一根主线。

盟议初开之日，赵地一处大馆之中，使者分国而坐，礼数周全，气氛却远比礼器更紧。表面上，人人都说“共制西秦”；真正坐下来后，问的却全是另外一些事。

谁先发起，谁居主位？

若秦试探东向，谁先应之？

盟成之后，若有一国失约，又当如何？

强国不肯白白分名，小国不肯平白担险。近秦者最怕先被拿来试刀，远秦者则更爱先看旁人能否真守约。场上众使一开口，便像丝线缠丝线，看似都朝一处，实际却各自使着劲。

一位赵国代表道：“赵近秦，先知其迫，故有倡议之名，并非欲夺诸侯之位。然若事事都让远国只居其后、近国独受其锋，此盟又将何以久持？”

齐国来使立刻应声：“倡议是一回事，主位又是一回事。齐若与诸侯同向，自当尽其应有之力；可若一开盟议，便处处由近秦之国定轻重，那远国何以安其心？”

话音未落，韩使已冷冷道：“齐地远而富，自可从容谈位。韩若今日只争坐次，不争生死，那倒真配不上来坐这一席了。”

几句话一交，满堂气息便越发绷紧。有人争名，有人争责，有人争远近缓急，有人争守约轻重。若只听一方，似乎都各有道理；可若人人都守着自己那一点道理不退，这盟议便要立时散在堂中。

苏秦先前一直未急着插话，待众人说得差不多，方抬眼道：“诸君今日来此，究竟是来争谁先坐上位，还是来争谁的国能不先亡？”

堂中一静。

苏秦又道：“名位固然要紧，可再要紧，也重不过国在与国亡。今日若只争坐次，则秦明日便坐看诸侯自散；今日若只争谁先担责，则秦更知只须逼一国，众国必先互相推让。诸君以为自己在争细处，其实争的是让秦看清诸侯何处最不齐。”

魏使皱眉道：“先生此言太高。非我等不知大体，而是盟若要行，终须先有定法。谁先出力，谁后出力，总不能一句‘同心’便都糊过去。”

苏秦点头：“正因如此，才更不能一开头便把最难之处压死。今日先求一件事：诸国共认西患非一国之患。此意一成，则出力先后、轻重缓急，方可分步议定。若连这一层都未立，便先把兵责、尊卑、罚则一并压到底，诸君今日在此，便不是成盟，而是先替秦拆盟。”

赵国那位代表顺势问道：“若有国口中肯附，心里却迟疑，守约何凭？”

苏秦答道：“诸侯之间，本就不能尽凭私信。可今日也不必求其尽信，只需使人人都知：若坏此盟，便先失天下对秦共同之势；若守此盟，虽未必人人得尽其利，至少都不至先独受其祸。人心难齐，利害可牵。守约之本，不在空言，而在谁都知道逆此势者先吃其亏。”

这番话说得并不热，却正中盟议最实的一处。堂中众使互相看了一眼，原本绷得最紧的那股气，终于稍稍松了一线。

苏秦见机，又进一步道：“强国不必急着把主位尽拿，先取其共势，则名自随势而来；弱国亦不必只求把风险尽推出去，若众国都如此，最先被推出去的必是最弱者。今日诸君退一步，不是让位于人，而是留生路于己。若连这一退都不肯，明日秦进一尺，诸君便都无从再退。”

这一席话，既不单护赵，也不单安小国，反倒像在一锅滚沸的水上，慢慢压下一块石。众使未必因此尽服其心，却都知道，若无这人居中转圜，今日之议十有八九已经散了。

及至傍晚，盟书草议终于得以继续往下写。名位之争暂缓，出力之责改作分步而议，失约之惩也不再强求一口定死。众人表面上仍各守矜持，可谁都知道，这一轮之所以未崩，全靠苏秦把众口从各争私算，重新拉回了“共同惧秦”这一条主线上。

然而，盟议虽暂平，暗流却并未尽息。

齐使临去时仍私下道：“今日不争，不等于日后不争。”韩使则低声道：“只怕日后真到担兵之时，说得漂亮的人，未必都还坐得住。”赵使听见，只淡淡一笑，不与相争。众人都明白，这盟是被局势压成的，不是被真心捏成的。

夜里，苏秦独坐灯下，看着案上那份未尽定稿的盟书，久久不语。他今日压住了一场大散，却也更清楚地看见：诸侯来到一席，不难；真要在一席上坐得不散，却比说服任何一国都更费力。

而西边秦宫之中，另一份边报也已送到。

报中写得极简：东方盟议已开，众国争执颇多，然终未散；苏秦其人，能于众争之中暂平其议。

秦王看毕，只把竹简轻轻合上，道：“记下。名势可成，众心未齐。”

殿中无人再多言。因为这一句，已把东方刚刚热起的同盟，看成了既有其势、又有其缝的一团火。

火已点着。

至于烧得多旺、烧得多久，秦仍在冷冷观它的风向。
