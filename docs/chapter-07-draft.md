# 第七回 邯郸初议六国先争坐次 秦廷得报时却只记苏秦能把众争压住

赵、燕既动，魏韩不敢全退，齐楚也都开始留意北方议论。

苏秦于是把线往前再扯一步，请诸国先派重臣入邯郸一议。人果然都来了，馆驿里车马不绝，礼数也齐，可真正坐到一张案前时，争的第一件事不是如何制秦，而是谁居主位，谁先担责。

赵使说近秦者先知痛，不可反被当成理所当然的挡锋之国。

齐使则说大国体面不可轻让，不能一开头就只剩出力而无定局之权。

韩使更冷：“小国若一入盟便只剩先受刀的命，这盟立来何用？”

燕使坐在稍后，话最少，却句句都绕着一件事打转。燕不愿再做局外之国，可也不愿一进局便仍只坐边上听人分派。

殿中你一句我一句，越说越紧。

苏秦等到众人都把话说出来，才缓缓开口：“诸君今日来，是来争谁先坐上位，还是来争谁的国能不先亡？”

一席之上，顿时先静了半息。

他又道：“强国怕白白分权，小国怕白白担险，近秦者怕先吃刀，远秦者怕局成后自己无位。这些都不算错。可若人人都只守自己这一层对，秦便不必动兵，先坐看诸侯自散了。”

魏使皱眉：“说大义容易，真到兵责与先后，终究要有人先站出来。”

苏秦点头：“正因如此，今日先不该压死谁先出多少兵、担多少责，而该先定一条最要紧的共识：近秦之警，不得只算近秦之国之事；秦若试一国，众国不得先借口观望。”

齐使听后，仍不服气：“齐若共制西秦，自会尽其应有之力。只是总不能叫大国一开始就让位于人。”

苏秦转头看他：“主导之位，从来不在口头先争，而在众人危疑时谁能让局不散。齐若真重其位，今日便不该只争坐次。”

这一轮压下来，众人终究都往后退了半步。

盟书草案得以继续往下写：先立共抗西秦之义，名位与兵责暂缓深争，先把“共同惧秦”这一层先立稳。谁都没有真服，谁也都不敢先散。

同一夜，咸阳也得了这份探报。

秦王听完，只淡淡问了一句：“苏秦此人，何如？”

边臣答：“能于众争之中，暂把众人压到一线。”

秦王把竹简轻轻合上：“记下。名势可成，众心未齐。”
