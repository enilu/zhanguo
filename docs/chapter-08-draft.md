# 第八回 一席雄谈先把众账压住 同盟初有其文却仍处处带着勉强

盟议初开之后，真正难的便不再是把人请来，而是让人坐定。

次日再议，齐使仍先提主位，赵使仍不肯只吃近秦之苦，韩使则把“小国附之反更危险”这句话又翻出来说了一遍。人人讲的都不是废话，人人也都不肯先松自己手里那一寸绳。

苏秦先不说大义，只把众人的算盘一一挑明：

强国怕白白分权，

小国怕白白担险，

近秦者怕先受其祸，

远秦者怕局成而自己无位。

他把这些都说穿后，席上反而安静了些。因为人人忽然都知道，对面并不是不懂自己在算什么。

苏秦便顺势再压一步：“既然如此，今日便不该先问谁得名最大，而该先问若盟不成，谁失得最多。”

赵使答得最快：“自然是近秦诸国。”

“近秦诸国先失，强国真能独安么？”苏秦问。

这一下，连齐使也没立刻接上。

于是这场议从午前磨到日暮，终究磨出一份能写下去的草案：强国暂缓争名，弱国不再只求尽退，兵责分步而议，先立“共向西秦”之文，再把难处一层层往下压。

这份草案并不漂亮。

上头处处都看得见折中与勉强。

也正因如此，它反倒更像真的。因为强国并未真忘其矜，小国也未真放其疑，人人都是被秦逼着、被局势推着，勉强挪到同一面旗底下。

夜里，苏秦独坐灯下，把新成的草案摊开。

他看得最久的，不是写成的那些字，而是字缝里还没死透的那些旧账。他心里比谁都更清楚：今日能把众人压住，靠的不是谁真心服了谁，而是人人都知道，再争下去，最先得利的只会是秦。

这便够了。

至少到这一刻，合从不再只是苏秦口中的一张图，而已经有了可以摆到案上的第一份文书。
