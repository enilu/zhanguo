# 第八回 一席雄谈暂平众议 数方权计始见同盟

盟议既开，真正难的便不再是把人请来，而是让人坐定。

第七回那一轮争执虽被苏秦暂时压住，到了次日再议时，众使心中所藏的那一本本账册，仍旧一页不差地带到了席上。礼官才刚唱毕入坐，齐使便先提到“主位不可轻定”，赵使随即应以“近秦者不当徒受其锋”，韩使则坚持“盟约若无切实护持，小国附之反更危险”。人人说的都不是一句废话，人人也都不肯先松自己手中的那一寸绳。

看似只是坐次、名分、先后之争，实则争的是谁在未来局面里多担一层风险，谁又能多握一分主导。

齐国所算，是大国之位不可轻让；楚国虽尚未尽到，却也通过使者把话递来，意思无非是“合从可以，楚不愿受人节制”。赵国所忧，是自己若先倡其议，不能反被众国当作理所当然的挡锋之国；魏韩所惧，则是名义上的共同一旦落到现实，最先吃痛的必然还是近秦之地。燕国最弱，却又最明白自己若不趁此时附上一线，日后便更难挤进大局。

这一番人心，各有其理，也各有其窄处。

议到午前，赵使终于忍不住道：“今日若仍只讲各自不肯吃亏，那还议什么同盟？赵既先知其患，先开其口，难道反倒要坐看诸国口口声声说共制西秦，手里却把一切先推到赵头上？”

齐使冷笑道：“赵若不愿先担，又何必先争倡议之名？天下诸侯皆有山河，不会因赵一句先言，便都成了赵之后从。”

话一出口，席间气息陡紧。魏使在旁插言：“两位争的是名，我等近秦之国怕的是命。若盟一立，秦先试边，谁来当之？诸位总不能要魏韩先担其怒，而强国只守其位罢？”

这一句又把矛头挑回了责任上。燕使原本一直少言，此刻却也开口：“若诸位都只怕先担，那弱国岂不永远只能等着旁人先把我们推到最外头？”

一席之中，强国、小国、近国、远国，各自所怕所求俱在其内。此等争端，不是靠一句“大义”便可抹平，也不是靠压谁一头便能成局。因为谁都不是来行善，而都是来算自己那一份能不能保全、能不能借势、能不能不先吃亏。

苏秦仍旧等到众声都发过一轮，才缓缓起身。

他先不说大道，只看着席上众人，道：“诸君今日所争，臣大致听明白了。争名位者，怕日后失其主导；争兵责者，怕今日先受其祸；强国怕白白分权，弱国怕白白作盾。说到底，人人都不是不想成盟，而是都想先看清自己会不会在盟成之后，比盟成之前更吃亏。”

这番归纳一出，众人反倒都静了些。因为苏秦并未先责谁短视，反先把每个人的算盘都点得极准。

他便顺势道：“既然如此，今日便不该先问谁得名最大，而该先问若盟不成，谁失得最多。”

赵使皱眉道：“自然是近秦诸国。”

苏秦点头：“近秦诸国先失，强国岂能独安？赵若独受其锋，齐楚真能长保其远？韩魏若先坏，则诸侯之间哪还有缓冲可借？燕若失其北势，又凭何在将来的大局里保其自处？诸君今日争的是自己别吃亏，可若盟不成，谁都不会真占便宜。区别只在早晚而已。”

齐使道：“说来说去，还是要众国先退一步。可为何偏要我等先退？赵既近秦，难道不该先尽其兵？”

苏秦看向齐使，语气并不重：“强国若只问别人先尽什么，自己便永远得不到真正的主导。主导之权，从来不在口头先争，而在众人危疑时，谁能让局不散。齐若真重其位，便不当只守其尊，而应守其可为天下所倚之势。今日先退一步于名，不是失位，乃是养位。”

齐使听到这里，神色虽仍倨傲，目中却已少了几分先前那种只以坐次论高下的轻慢。

苏秦随即又转向魏韩：“近秦之国所忧最实，臣不敢以空言压之。故今日之盟，不必一开始便把兵责一口压死。可先定共向，再分缓急。凡秦若有试逼之举，则近国先报，诸国共议应之；强国不得借口观望，远国亦不得永远只居其后。今日先把应秦之意合成一条，日后兵与责方可按势而定。若一开头便逼诸位把一切都押死，臣亦知此盟必散。”

韩使闻言，慢慢点了一下头。因为这话至少给了近秦者一线喘息，不再让他们觉得所谓合从，只是把自己名字写进一张日后先受刀锋的纸里。

至于燕使，苏秦则只道：“弱国最怕的，不是盟中位低，而是盟散时先无人问。今日燕既已在此，便已不在局外。只要诸国共意尚存，燕便不是孤角之地。若诸位因争而散，最先被遗在风里的，正是燕这等国。”

这几句话又把燕那点最深的忧意重新按回了盟席之中。

说到此处，堂中诸使已无一人还能把苏秦当作只会高谈空论之辩士。因为他并未求谁真心服谁，而是在一条条争执里，各给一分下得来的台阶，又同时把所有人都拽回同一层更大的利害上。

真正的高光，不在他把话说得多漂亮，而在于满席本来各自拉绳的人，到最后都不得不承认：若再照原先那样争下去，最先得利的只会是秦。

于是苏秦最后才把那一锤落下：

“诸君今日所失，不过是一点先后、几分名面、暂时未能尽遂己意的私算。今日若盟不成，明日所失的便不止这些。强国失其主势，小国失其自保，近国先亡，远国后危。既知如此，便不该再问‘我如何才全得’，而该问‘我如何才不先尽失’。

天下之势，本已不许诸君人人都按旧算盘从容而行。若还想像承平时候那般，一步不让、一责不担、一名不分，那便不是议盟，而是议亡。臣今日不求诸君尽服臣，但求诸君都看明白一件事：只要秦在西，诸侯便不可能各自尽善；既不能尽善，就只能先求一同不败。此便是今日合从之所以不可不成。”

这一席话说完，堂中再无先前那种句句顶死的对撞。有人仍旧不甘，有人心里仍另有保留，可更多的人已经明白，今日这一局若没有苏秦在此居中拆结、压场、调轻重，便真过不去。

当天下午，盟书轮廓终于初成。

诸国先定共抗西秦之义，名位次序暂缓深争，兵责以缓急先后分议，守约不全凭空言，而以共同利害相互牵住。说穿了，这份同盟并不完美，甚至处处都还能看出勉强拼接的痕迹；可也正因如此，它反倒更像真的。

因为强国并未真忘其矜，小国也未真放其疑；有人点头，是被说动于势；有人点头，是权衡于利；也有人并非心服，只是不敢在众势初成时独自逆之。

这便是“暂平众议”的真意。

盟成之后，众使出席时神色各异。赵使面上虽仍自持，眼底却已有了真正把北方线头握住的稳意；燕使则像终于替本国占得一席位置，虽不显山露水，心气已全不同；魏韩两使神色仍重，却到底少了几分先前那种随时欲退的惶疑。至于齐、楚一线，虽未至于尽服，却也再无人敢把这盟当作几国一时热议。

夜里，馆舍中酒食虽开，苏秦却未多留席间。他独自回房，灯下展开那份盟书草案，手指慢慢按过几处仍带着明显折中痕迹的条文，心里没有多少得意，反倒先起了几分冷清的明白。

同盟终于第一次像个样子了。

可也正因为终于像个样子，他比谁都更清楚地看见：这东西并非天衣无缝，而是被局势硬生生压成的一体。只要外力稍变、私算再起，那些今日勉强退下去的争执，迟早还会回来。

然而，能让六国在此刻先坐到一处、先承认同一面旗，已是难得。

至于这面旗能撑多久，便要看往后风从哪一面吹了。
