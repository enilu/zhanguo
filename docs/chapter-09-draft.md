# 第九回 佩印将成天下侧目 秦关未动西土观风

盟书初定之后，最先变化的不是兵，不是地，而是人心看人的眼色。

馆驿之间，苏秦的名字像顺着春风一夜散开。昔日门下执事看他，多半先看衣冠旧不旧、马瘦不瘦；如今再见，先问的却是“苏子今在何处”“赵燕之议又到哪一步”。来往使者递帖时，称呼也与前大异。过去不过叫一声“先生”，眼下却多半先加“持盟之客”“主合从之士”一类敬辞。

这变化来得极快，却也并不全是趋炎附势。

有的人是真服其口舌与心力，知道若无此人，前番盟议必散；有的人敬的则不是苏秦本身，而是他如今手中那股能把数国暂时连成一线的现实分量；还有的人心里虽未必服，却也知道，这等时候若还把他当作寻常说士，便是不识局势。

人情冷暖，本就从来不只认苦，不认势。

苏秦对此看得分明，所以神色反倒比旁人更稳。馆舍里添了来访之客，他只是见该见之人；送来的礼帖渐多，他也不曾因此多露半分喜色。因为他知道，今日众人所敬的，是“局已见形”的苏秦，而不是第一回里那个在门前吃风的寒士。若哪一日局又松了，这份热礼自然也可能转冷。

然而，天下对这一局的眼光，毕竟已经真切地变了。

赵地士人议论道：“此人一舌已不止动一国，竟像要替六国定次序。”燕地有人叹道：“昔日不过馆门外布衣，今却能使北方两国同声。”魏韩之间的私议则更复杂些，有人庆幸近秦之国总算不再各自求活，有人仍忧这盟只是表面好看，可无论赞与疑，都再无人能把合从只当作几句馆舍高谈。

齐、楚更是不能不侧目。

大国看苏秦，并不单看他会不会说，而是看这个人是否真把一股原本各自零散的惧秦之心，渐渐织成了某种次序。齐臣私下道：“若任其这样下去，纵使齐未尽从，其势也终究会逼到齐来表态。”楚中也有人低声说：“此士之可畏，不在空言，而在他能让别人先觉得不入其局便会失位。”

这种“侧目”，比一时赞叹更有分量。因为这说明苏秦已经从一名说士，变成了政治秩序里不可忽略的中枢。

外面人心愈热，门庭冷暖翻得愈快。

从前邯郸那些未必愿正眼看他的人，如今见面多半先让半步；有几位昔日把他当作一时客的门下人物，也开始主动递话、探意、问局。连驿舍中的小吏都知道，这位洛阳来的布衣先生，往后只怕真要以一人而系六国之名。

可苏秦最深处却未因此轻飘。

他偶尔夜里独坐，仍会想起第一回那处馆门前的冷风、第二回赵秦之间的轻慢、第三回回到旧巷时席间那一句“胸中尽是天下大事”。正因都记得，他此刻更清楚：今日所得，不是凭命，是凭他一层层把自己和这张局都硬生生推到这里来。也正因如此，他更不敢把这股骤起的名势看得太稳。

就在东方热议苏秦与合从之际，西边函谷关内却仍旧没有什么额外的动静。

秦军照旧操练，边报照旧呈进，咸阳宫中议事的节奏也并未因东方风声更高而乱上半分。秦廷诸臣对近来盟议草成、苏秦声名大起一事，自然不是不知；只是知道归知道，态度仍旧冷得很。

一位边臣看着新送来的探报，道：“东方使者往来愈勤，苏秦之名也已压过寻常说客。六国若真借此同声，对秦总归是个麻烦。”

旁边老臣却道：“麻烦是真麻烦，可未必便是大祸。名可以聚众，骨未必便成。今日他们看似同向，明日若真到分兵分责、担轻担重处，只怕各自旧心又会起来。”

边臣点头：“然若任其名势日盛，东方士气终究不同。”

老臣答道：“士气可振，秩序难造。先看着。越到盛处，越易见其真薄厚。”

这便是秦之“观风”。它并非无视东方，而是把东方此刻的高热，当成一团仍须观其成色的火。

而东方这边，随着北方线头渐连、魏韩已不敢全退、齐楚亦已不能假作不闻，关于“六国之印将归于苏秦”一事，也终于不再只是零碎传闻。

起初还只是馆驿之间低声相告，到后来，连诸侯近臣也都开始明着提起。礼官往来，条目渐定，使臣进退，称谓愈隆。人人看得出，这件事离真正落成，已经不远了。

当一个曾在诸侯门外吃风的寒士，竟真走到“六印将定”的边上时，天下哪有不震动的道理。

邯郸城中，士人夜谈时已有人慨叹：“布衣而持六国之势，自古未尝多见。”也有人低声冷笑：“敬的未必是其人，更多的是其印。”说这话的人原想压一压苏秦名势，可旁边人听了，却反问：“若没有其人，印又怎会到他手里？”

一言之下，席间便无人再笑。

因为人人都明白，此时的苏秦，已不只是一个人是否显贵的问题，而是东方在秦压之下，终于由他身上看见了一线可以共同发声的秩序。

数日后，又有使者自外归来，馆舍内外礼数再增一层。那位曾在赵门里与苏秦交锋的年轻议者见了，不由低声道：“天下看你的眼神，已和数月前全然不同了。”

苏秦听了，只淡淡道：“天下看重的，从来不是我如何困顿，而是我手里这股势是否真能系住。”

说完，他抬眼向西，目光越过城墙与暮色，像要穿过千里山川，望到函谷关内那一片依旧静着的西土。

东方的风，已起得很高。

可西边那道关，仍旧静静立着，没有因此先乱半分。

这份静，反倒让即将落到苏秦身上的六国相印，显得更重了。
