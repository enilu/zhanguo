# 第九回 盟书初定人心看苏秦已变 咸阳闻六国渐连仍只冷看火色薄厚

盟书初定之后，最先变化的不是兵，不是地。

是人心看苏秦的眼色。

馆驿之间，他的名字开始顺着春风一夜散开。昔日门下执事见他，先看衣冠旧不旧、马瘦不瘦；如今再见，先问的却是“苏子今在何处”“赵燕之议又到了哪一步”。来往使者递帖时，称呼也与从前不同。

有的人是真服他能把众争压到一席。

更多的人敬的，则是他手里那股已经能把数国暂时连成一线的势。

苏秦自己反倒更冷了些。

他知道，众人今日敬的，是“局已见形”的苏秦，而不是第一回里那个在馆门前吃风的布衣。若哪一日局又松了，这些热礼自然也会转冷。

可不管众人心里各有几分真敬，这一轮盟议毕竟已经把他的名字推到了六国案前。齐、楚两国不能再把他当作北地一阵游士风声；魏韩也不能再拿“不过几句馆驿高谈”来搪塞。

与此同时，咸阳仍旧很静。

秦廷听见邯郸盟议草成、苏秦名势大起，并没有因此先露慌色。边臣看完新送来的探报，只道：“东方士气必振。”

老臣却回得更冷：“士气能振，秩序未必便成。先看它这团火，烧得是名，还是骨。”

秦王闻言，只淡淡道：“记之。火已起，骨未见。”

这份静，反倒让东方那股新热更显得只是较量的开始，不是终局。

数日之后，关于“六国之印将归苏秦”的话，也终于不再只是低声传闻。诸使来往，称呼渐隆，礼官开始排次，各国都在按自己的分寸往这一场大礼里放东西。

谁都知道，真正震动人的并不只是苏秦这个名字，而是东方数国在西秦重压之下，第一次真要把共同之名、共同之势，公开托到一个人身上。
