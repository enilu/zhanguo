# 第一百回 临淄闻口开满城失色 田文见齐先病在争快慢

临淄这一夜，灯火一直没有真正熄过。

宋后以来，城中人已习惯深夜也有军书来回。可这一回不同，北门外先是急骑一拨接一拨，后又有值夜小吏一路小跑进宫，脸色白得像刚从寒水里捞出来。

到天将亮时，宫中终于传出一句极短的话：

“北外中线开了。”

这七个字，比先前所有“失半口坡脚”“外线不整”都更重。

因为这不再是小失。

这是齐边真正被人撕出了能继续往里做事的口。

齐湣王接报时，整个人竟先静了一下。

静得连旁边侍立的近臣都不敢喘大气。

不是不怒。

而是他到这一刻才真正感觉到，田文先前那些“不要只信第一眼”“不要总以为它们自己先乱”的话，原来不是为了扫兴。

它们是在替他挡一道本来可以更早便看见的风。

可这念头只闪了一瞬，便很快又被另一层更硬的东西压回去：

现在不是认谁早说对了的时候。

是齐还能不能把这口开出来的线，先堵住的时候。

于是他立刻问：“丢到何处？”

边吏低头：“只开中线，不深。韩前军扩口而止，魏中军侧托，赵粮火已接。”

又是那几个词。

不深，

而止，

侧托，

已接。

这些词合在一处，比一句“敌大胜”更叫人发寒。

因为它们说明，对面每一步都像是提前知道自己只该拿哪一层、不该贪哪一层。

殿中终于有人先开口，请立刻重兵反扑中线。

也有人说，不如弃外护里，先把齐自己几处最要紧的骨再并死。

两路话一出，齐湣王脸色更沉。

因为他忽然发现，自己如今最难的，不是选哪一条。

而是不论选哪一条，都已是被对面先把节带到了这里。

田文站在一旁，终于还是出班。

他没有再说“臣早知如此”那种废话。

到这一步，说那个，已只配让人看轻。

他只道：“大王，臣以为，齐眼下先病的不只是那一道口。”

齐湣王抬眼：“那先病的是什么？”

田文答：“是快慢。”

满殿一时都没听明白。

田文便道：“前些日子我齐处处都慢它半拍。如今口一开，若大王一怒之下处处都要改快，前线便会先争谁快、谁先补、谁先出、谁先立功。快若先乱，便比慢还坏。故臣请大王此刻先只定一件事：谁说了算。先把快慢归到一处，再议反扑与收线。”

这句话一出，连原本主战最急的几人都不禁停了停。

因为他们忽然听懂了：

齐如今不是不想打。

是若人人都急着先去把那口口子堵上，反会把剩下本还没乱透的地方也一起拖乱。

齐湣王盯着田文看了许久，终于道：“好。传前线，自今日起，北外诸口归一将一令。先不许各垒各自争回面子。违者军前问。”

这道令比前几次都更硬。

可田文心里却仍冷。

因为他知道，这一道令虽对，却终究又是被乐毅那边逼着下出来的。

齐每慢一层，便输一层；

齐每急一层，又可能再乱一层。

如今齐最难的，已不是选战还是守。

而是怎么在别人先定下来的快慢里，重新找回自己的节。

散朝之后，老门客又来。

田文看着他，苦笑道：“齐如今像个被逼着在别人鼓点上换步的人。”

老人叹道：“能换过来，尚可。最怕的是脚还没换稳，心就先乱着去抢拍。”

田文没有再接。

他只望向北面。

城上晨色已起，照着临淄高墙，照着宫中未散的夜灯，也照着那些一夜未睡、此刻仍在来回跑的军书小吏。

这一城的气，还没有散。

可谁都已知道，它正在被逼着从最熟的那种从容里，往另一种更苦、更急、也更容易见出真骨头的活法里去换。
