# 第一百回 临淄忽闻北口已开 田文回看病根原在争快慢

临淄这一夜，灯火一直没有真正熄过。

到天将亮时，宫中终于传出一句极短的话：

“北外中线开了。”

这七个字，比先前所有“失半口坡脚”“外线不整”都更重。

齐湣王接报时，先静了一下，随即便问：“丢到何处？”

边吏低头：“只开中线，不深。韩前军扩口而止，魏中军侧托，赵粮火已接。”

又是那几个词：

不深，

而止，

侧托，

已接。

这些词合在一处，比一句“敌大胜”更叫人发寒。

殿中终于有人先开口，请立刻重兵反扑中线；也有人说，不如弃外护里，先把齐自己几处最要紧的骨再并死。

田文这时出班：“大王，齐眼下先病的不只是那一道口，是快慢。前些日子我齐处处都慢它半拍。如今口一开，若一怒之下处处都要改快，前线便会先争谁快、谁先补、谁先出。快若先乱，便比慢还坏。”

齐湣王盯着他看了许久，终于道：“好。传前线，自今日起，北外诸口归一将一令。先不许各垒各自争回面子。违者军前问。”

令出之后，殿中仍无人敢先退。

因为谁都听得出来，这一道令虽是对着北外诸口发的，实则是在替整个临淄压一股乱气。前几日城中议论纷纷，各署各垒都还想着“先把自己这一口补平”，如今中线一开，若再人人只争自己那点快，便不只是前线失口，连城里也要先乱出次第来。

田文退出殿门时，天边刚有一点青白。

他站在阶下回看宫门，忽然听见城北方向传来很远的一阵梆声。那声音平日里原是守夜常响之物，这一夜却听得人格外发紧。因为从今往后，临淄再听北面诸口报上来，便不能只问“失了哪一口”，还得问“这一口一开，城里究竟会不会跟着一起抢快”。
