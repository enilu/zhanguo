# 第一百零一回 乐毅得口却仍收锋 齐边回整只待再试轻重

北外中线一开，联军诸将里最急的那股气，反倒被乐毅先压住了。

韩前军想借新口再探深垒；魏中军也觉得，此时若再往里半里，齐边几处横应会更难接。

乐毅只道：“再往里，便不是今日之胜，是明日之累。”

于是联军这日整整一日都反常地稳。

韩前军只修外线、补盾、深埋新得坡脚；

魏中军只压节、不抢前；

赵后线把原先三站再往前补实半站，却故意不让粮车过分拥到新口前。

齐边这时已换了新令：北外诸口归一将一令。按理说，该比昨日整。可问题恰恰在这“整”字上。

新统总的将领一到，先要看诸垒旧损、新线、昨夜新并之处、今日联军所处，再要压各垒那股“不能让我这口先难看”的旧性。口头上人人都应得是，真正传到垒上，却又难免带出各自旧气。

午后，魏节应将站在高处看了半日，忽然低声道：“主将，它归一了，却像还没归心。”

乐毅点头：“这便是我们今日不急的值。”

说完，他只命前线做一件小事：韩前军在傍晚前，再轻逼一回已开那口的边缘，不求再入，只求让齐新归一的那位将领不得不把今日压了一天的几股旧气，全拿来再应一次。

这一下小逼，比昨日大战轻得多。可齐边反倒应得更难。

站在高处看去，那已不太像昨日那种被联军逼出来的乱，更像齐自己在学着换步时，脚下先打了一绊。

乐毅看完，只道：“好。它开始自己绊自己了。”

韩老将听见这话，先是一愣，随即才慢慢咂出味来。

前些日子他总觉得，得口之后就该趁热再压。可到了这一刻，他也看见了另一层东西。齐若还乱着，联军顺势多打几下便是；齐若开始学着收，真正值钱的便不是多取半口坡脚，而是看它为这一个“收”字，要把哪些旧气、旧法、旧路一并先压下去。

这一压，便总会有人不服，总会有旧账冒头。

联军今日等的，正是这股不服。
