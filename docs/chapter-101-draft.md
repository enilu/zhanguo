# 第一百零一回 乐毅得口反缓不深贪 齐边归一先看谁压谁

北外中线一开，联军诸将里最急的那股气，反倒被乐毅先压住了。

韩前军想借新口再探深垒；

魏中军也觉得，此时若再往里半里，齐边几处横应会更难接；

连燕本部诸将都有人请战，说既已开口，何不趁势先把这条线一举压穿。

乐毅听完，只道了一句：“再往里，便不是今日之胜，是明日之累。”

众将都静了一下。

他随即把图摊开，指着已开那口与齐后头几道垒、几处应路、一条临时新并出来的中线，道：“齐今日必归一令。它若归得住，我们猛进，正好撞在它最想用来立新节的那一刻；它若归不住，我们不必多打，它自己便会把口越撑越裂。”

韩老将听到这里，忽然彻底没了再争那口“我还能多咬一步”的硬气。

因为他此刻才真正看清，主将眼里每一口胜，不是单看当时能不能多拿几步。

而是看多拿那几步，会不会反叫对面最快把最该并的地方先并出来。

于是联军这日整整一日都反常地稳。

韩前军只修外线、补盾、深埋新得坡脚；

魏中军只压节、不抢前，专盯齐边诸垒号令有没有真正归一；

赵后线把原先三站再往前补实半站，却故意不让粮车过分拥到新口前，免得反把联军自己那道线先堵坏。

这种稳，连己方营中都有人看得发闷。

可齐边看着，却比昨日更烦。

因为他们已换了新令：

北外诸口归一将一令。

按理说，该比昨日整。

可问题恰恰在这“整”字上。

新统总的将领一到，先要看诸垒旧损、新线、昨夜新并之处、今日联军所处，再要压各垒那股“不能让我这口先难看”的旧性。口头上人人都应得是，真正传到垒上，却又难免带出各自旧气。

有垒想先把昨日丢掉的面子抢回；

有垒则怕新统总一来便先让自己挨军法，故处处都求稳；

还有人见联军今晨不深逼，心里那点旧侥幸又悄悄抬头，觉得也许对面其实已气尽。

于是齐归一之后的第一日，竟比前两日更别扭。

号是统一了。

可号底下的人心，未必齐。

午后，魏节应将站在高处看了半日，忽然低声道：“主将，它归一了，却像还没归心。”

乐毅点头：“这便是我们今日不急的值。”

说完，他只命前线做一件小事：

韩前军在傍晚前，再轻逼一回已开那口的边缘，不求再入，只求让齐新归一的那位将领不得不立刻把今日压了一天的几股旧气，全拿来再应一次。

这一下小逼，比昨日大战轻得多。

可齐边反倒应得更难。

因为昨日众人还都在大惊里，顾不得那么多。

今日要照新令应，又都想证明自己不是最拖后腿的那一个，结果几口号一起起，竟还是有人快，有人慢，有人抢着往前补，有人又怕补过了反犯新令。

站在高处看去，那不再像昨日那种被联军逼出来的乱。

更像齐自己在学着换步时，不免脚下先打了一绊。

乐毅看完，只道：“好。它开始自己绊自己了。”

这句话听在诸将耳里，比再夺半口地还更重。

因为他们都知道，若一支军到了“自己绊自己”这一步，后面真正能开的大口，便不远了。
