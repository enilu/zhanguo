# 第一百零二回 临淄再易将令乱意更深 乐毅看透齐边误处将成

北外归一的第一日应得不顺，消息很快又飞回临淄。

齐湣王看完，脸色阴得几乎滴下水来。不是因为前线又大失，恰恰相反，是因为没有大失，只有一些“号有快慢”“补有先后”“诸垒相顾”“未尽一节”的碎字。

齐湣王忍了又忍，终究还是拍案：“换。”

“换北外主将。既归一而仍不齐，寡人还留他作甚？”

田文听到这里，竟一时无话。因为他知道，大王已被逼到一种最难的局里：慢，是错；急，也未必不是错；换，也许会更乱；不换，又像坐等旧乱长大。

于是他最终只道：“既换，便请大王只换一层，不要连号、连法、连诸垒旧责都一并再翻。”

齐湣王沉默片刻，终于点头：“便依此。换将，不换大令。”

而在联军前线，探报也立刻到了：

“齐北外换将。”

“换人，不换大令。”

韩老将听完，眼里先亮：“它急了。”

魏节应将却道：“急，但未全乱。新将若够硬，反可能借旧令压出一口新齐。”

乐毅把“换将”“不换大令”这两句压在图上，静看许久，最后才伸指一点北外那条几日来被齐一层层收、一层层并、一层层又并不齐的中线：

“它最易误的时候，不在今天。”

韩老将问：“那在何时？”

“在新将刚想证明自己比旧将更快、更齐、更能把这条线压住的时候。”

乐毅随即下令：“今夜诸军不动。明日先看齐新将第一道令。”

这道令一下，韩老将反倒先把嘴闭上了。

前几日他还总嫌主将不趁热追，如今却也慢慢看懂了一层。齐每次最易出错的，不是在旧法将尽时，而是在新法刚立、主将又急着证明“我来便能比旧人更快、更齐”的那一刻。那一刻若联军太急，齐反倒会借着外压把上下都逼成一股；联军只静看它自己先出第一道令，才更看得见这位新将究竟会先护哪里、先急哪里、又会在何处把那口“我比旧将更能整齐”露出来。
