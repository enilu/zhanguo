# 第一百零二回 临淄再换将令气愈乱 乐毅点破只待齐先误

北外归一的第一日应得不顺，消息很快又飞回临淄。

齐湣王看完，脸色阴得几乎滴下水来。

不是因为前线又大失。

恰恰相反，是因为没有大失。

只有一些“号有快慢”“补有先后”“诸垒相顾”“未尽一节”的碎字。

这些字拆开来都不算很重。

可合在一起，却像一层看不见的湿气，慢慢往人骨里钻。

因为它们说明，齐如今最大的问题，已不只是敌怎么逼。

而是自己每换一层新法，都还来不及把人心、军心与旧习一并换过去。

齐湣王忍了又忍，终究还是拍案：“换。”

殿中众臣一惊：“换谁？”

“换北外主将。”齐湣王声音极硬，“既归一而仍不齐，寡人还留他作甚？”

这句话一出，田文心里先是一沉，随即更冷。

因为他知道，大王这不是全错。

可也未必全对。

眼下北外那层乱，未必只是一个主将能不能压得住。

它更像是齐这几日被逼着一层层换法之后，从上到下那股“总想着还能有一层更顺、更快、更体面的补法”的急，一起在往外冒。

你换将，也许能压住一口。

也可能把“人人都怕自己再慢、再错、再被换”的那层更急，一下压得更重。

他正要出班，齐湣王却已先转头看他：“孟尝君，寡人知道你又要说慢。可如今不换，前线自己便要先散。”

田文听到这里，竟一时无话。

因为他知道，大王已被逼到一种最难的局里：

慢，是错；

急，也未必不是错。

换，也许会更乱；

不换，又像坐等旧乱长大。

这时再说一句“都不宜”，已是废话。

于是他最终只道：“既换，便请大王只换一层，不要连号、连法、连诸垒旧责都一并再翻。”

齐湣王沉默片刻，终于点了头：“便依此。换将，不换大令。”

这已经是他此刻能给出的最稳一层了。

而在联军前线，探报也立刻到了。

“齐北外换将。”

“换人，不换大令。”

韩老将听完，眼里先亮：“它急了。”

魏节应将却比他更冷，只道：“急，但未全乱。新将若够硬，反可能借旧令压出一口新齐。”

乐毅听完，没有立刻表态。

他只是把“换将”“不换大令”这两句压在图上，静看许久，最后才伸指一点北外那条几日来被齐一层层收、一层层并、一层层又并不齐的中线。

“它最易误的时候，不在今天。”

韩老将一怔：“那在何时？”

“在新将刚想证明自己比旧将更快、更齐、更能把这条线压住的时候。”乐毅道，“他既不能全翻旧令，又想立新威，最易在半日一日之间，把一条本来就绷得极紧的线，再往前压过半寸。那半寸，便可能是他的口，也是我们的口。”

帐中众将听到这里，竟都静了。

因为他们忽然觉得，齐边眼下那种似乱未乱、似齐未齐的状态，在主将眼里，根本不是烦。

是鱼刚咬钩前，那一线最细最轻、却最不能催的颤。

乐毅随即下令：“今夜诸军不动。”

韩老将这回竟没有立刻急。

他只问：“那明日？”

乐毅望着图上那条中线，缓缓道：“明日先看齐新将第一道令。”

帐外夜风吹得旗角轻响。

营中各处灯火照常，并不因齐换将而骤盛。

因为到这一层，联军最重的已不再是怎么把自己再逼紧一分。

而是怎么忍着不先去碰，把齐那道新换出来、又必然想自证明的硬劲，先逼成自己的误。

乐毅站在图前，手一直按着那条线，没有松。

他知道，这一口若再看准了，后面被撕开的，就不会再只是半口坡脚、一截外线、一段中道。

而会是一整层，齐自己刚想并紧、却还没来得及把气养稳的骨。
