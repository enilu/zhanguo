# 第一百零三回 齐新将晨令只求一口整 乐毅守着半寸终等其失

齐北外新将到营时，天还没亮透。

一路看过几垒旧损、新并之线与昨夜诸报，脸色先沉，后又慢慢绷成了一种很硬的静。站到图前许久，他终于咬着牙下令：

“中线前并半步。”

“右垒车前提一层。”

“左垒不许再留旧缓，依新号同应。”

“诸垒今日先整成一线，不许各自惜力。”

这道令一出，帐下几名旧将心里都同时一沉。因为他们听得出来，新将虽未全翻旧法，却已经把那条绷得最紧的中线，再往前压了半寸。

而在联军高处，探骑几乎同刻回报：

“齐中线前并半步。”

“右垒车前提。”

“左垒旧缓尽去。”

韩老将眼里那层久压不发的战意猛地跳了起来：“它果然压过头了！”

乐毅把齐新令与眼前线势重新压在一起，过了片刻，才抬头道：“就是这半寸。”

韩前军几乎贴着晨雾出去。

不朝昨日之口，

不向最厚之垒，

而是直逼齐新并中线那一截刚往前压出的半步。

齐新将本想以此半步先把北外一线整起来，谁知联军第一口便卡在这里来咬。他脸色当场一变：“右垒合上！左垒跟压！”

令一下去，问题便全出来了。

右垒车已前提，转身不便；

左垒旧缓昨夜方去，号次还未真熟；

中线新并那半步本就是生挤出来的，一见韩前军逼近，前后都想先替新将把这一寸脸面撑住，几口力便一起挤在最窄的那一截上。

挤得越急，越乱。

齐新将眼看不好，厉声再压：“中垒不许退！”

可就在这时，魏中军的号到了。不早不晚，正卡在齐右垒要往里合而未合、往外压又未压住的那一下。魏中军一切进侧应路，齐右垒那口原本勉强想替中线兜住的气，顿时被截薄了一层。

韩前军便在这一层薄里狠狠干了进去。

这回不再是前几日那种啃半口，是一口真撕。

新并出来的中线半步先断，右垒外提之车被韩军逼得横转不过，左垒又因号次未熟，补上来时整整慢了一号。

就这一号，北外那条几日来一直像快要绷住、却始终未真绷住的整层骨，终于被联军一口撕开。
