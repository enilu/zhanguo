# 第一百零三回 齐新将晨令先求一口整 乐毅看准半寸终等到

齐北外新将到营时，天还没亮透。

他年纪不算老，甲也整，步子更急。一路看过几垒旧损、新并之线与昨夜诸报，脸色先沉，后又慢慢绷成了一种很硬的静。

因为他一眼就看见，北外真正最难的不是哪一垒失了半口，或哪一道应路已露。

而是整条线都在等他这一道令，来证明新将一到，齐就能比旧将更齐、更快、更硬。

这正是最容易出错的时候。

若太缓，军心先疑“新将也不过如此”；

若太猛，半日内便可能把本就没养稳的那根线一下压断。

可人在这种当口，最难的恰恰就是拿得住那半寸。

新将立在图前，看了很久，终于咬着牙下令：

“中线前并半步。”

“右垒车前提一层。”

“左垒不许再留旧缓，依新号同应。”

“诸垒今日先整成一线，不许各自惜力。”

这道令一出，帐下几名旧将心里都同时一沉。

因为他们听得出来，新将虽未全翻旧法，却已经把那条绷得最紧的中线，再往前压了半寸。

而这半寸，正是他想拿来立威的一寸。

若成，北外整线便能在新将手里先站住一回；

若不成，前几日所有“归一未齐”的病，便会被一下子同时逼出来。

而在联军高处，探骑几乎同刻回报。

“齐中线前并半步。”

“右垒车前提。”

“左垒旧缓尽去。”

三句一落，韩老将眼里那层久压不发的战意猛地跳了起来：“它果然压过头了！”

魏节应将也沉声道：“这半步是新将自己的气，不是北外本该有的气。”

乐毅没有接他们的话。

他只看着图，把齐新令与眼前线势重新压在一起，过了片刻，才抬头道：“就是这半寸。”

说完，他一口气下了五道令：

“韩前军，不打旧口，直逼新并处。”

“魏中军，看齐右垒一动便切它应路，不许早一号。”

“赵粮火外明内收，叫它看见后稳，不许前拥。”

“燕本部压住侧线，断其想借旧缓回身的一口路。”

“诸军记住，今日不是争垒，是争它新令刚落未稳这一刻。”

这几道令一下，帐中竟没有人再像前几日那样先急着问“可追几步”。

因为所有人都知道，主将等的，正是眼前这一刻。

韩前军几乎是贴着晨雾出去的。

不朝昨日之口，

不向最厚之垒，

而是直逼齐新并中线那一截刚往前压出的半步。

齐新将本想以此半步先把北外一线整起来，谁知联军第一口便卡在这里来咬。

他脸色当场一变：“右垒合上！左垒跟压！”

令一下去，问题便全出来了。

右垒车已前提，转身不便；

左垒旧缓昨夜方去，号次还未真熟；

中线新并那半步本就是生挤出来的，一见韩前军逼近，前后都想先替新将把这一寸脸面撑住，结果谁也不肯先让，谁也不敢先退，几口力便一起挤在最窄的那一截上。

挤得越急，越乱。

乱得不是兵先逃。

而是号先相压，车先相碍，人先相顾。

齐新将眼看不好，厉声再压：“中垒不许退！”

可就在这时，魏中军的号到了。

仍旧不早不晚，像算准了齐右垒正要往里合而未合、往外压又未压住的那一下。魏中军一切进侧应路，齐右垒那口原本勉强想替中线兜住的气，顿时被生生截薄了一层。

韩前军便在这一层薄里狠狠干了进去。

这回不再是前几日那种啃半口。

是一口真撕。

新并出来的中线半步先断，右垒外提之车被韩军逼得横转不过，反把后头应路堵死；左垒又因旧缓新去、号次未熟，补上来时整整慢了一号。

就这一号。

北外那条几日来一直像快要绷住、却始终未真绷住的整层骨，终于被联军一口撕开。

齐军这回真乱了。

不是一垒乱，

不是一处坡脚乱，

是中线与侧应一齐乱。

新将想压，压不住；

旧将想补，补不上；

后头原本还想靠“收诸垒、锁中线”慢慢再养回来的那一点气，到这里终于像被谁从当中狠狠干断。

而联军仍旧不乱。

韩前军杀进去，却不自散；

魏中军切侧，却不抢功；

赵后线火稳，车稳，人稳；

燕本部侧压得恰恰好，不让齐边仍存的那一点旧缓回身把口再缝住。

高处看去，像不是一场热血忽起的大战。

而像一把磨了许久、试了许久、压了许久的刀，到这一刻，终于顺着它早看准的那条骨缝，稳稳切下去了。
