# 第一百零四回 北外整层忽裂临淄震 乐毅得势仍不肯轻取

齐北外那条线，是在辰时前后真正裂开的。

不是哪一垒被单独撞穿，而是中线一断，右应不及，左补迟号，整条原本还想勉强并住的北外骨一下子都跟着发空。

前线急报入临淄时，齐湣王正在宫中强逼自己坐定。可当“北外整线裂”“新将压不住”“联军韩魏并进而赵后不断”这几句报出来时，他还是猛地站了起来。

殿中很快有人失声请调更重之兵，要不惜一切先把北外再并回去；也有人主张速收更内一层，把外骨先让。

齐湣王脸色来回变了几次，终于道：“诸君闭口。”

过了片刻，他才转头看向田文：“你说。”

田文深吸一口气，最终只道：“北外这口，已不再是如何再缝一针，而是我齐接下来，要用哪一层骨，去换回城外这整口胆。”

而在前线，韩老将这一回是真的杀出了气：“主将！齐整层已裂，再不给我一口，我便夜里都睡不着！”

乐毅先看回报：

韩前军已入深一层；

魏中军侧路已立；

赵后线一日未断；

齐边新将仍在试图收口，却已顾左失右。

看完之后，他只道：“可再进，但不许散追。”

韩老将一愣。

乐毅道：“今日我们开的是齐一整层骨，不是一口赌命之门。门既已开，它后头自然会自己生慌。此时若诸军各凭血气先冲，明日乱的便不是齐，是我们。”

这话一出，帐中一时都静了。

诸将这才明白，乐毅看到的已不是北外今日还能再吃几寸，而是北外整层一裂之后，临淄明日、后日、再后日会怎么一层层自己生出慌来。整层骨一断，齐王今日便要在“补回外面这一层”与“收住更里那一层”之间先选；他一选，底下诸垒、诸县、诸署便都得跟着改活法。

联军不必替它改得太多。

只要看着它自己先去改。
