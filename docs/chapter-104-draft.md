# 第一百零四回 北外整层先裂临淄震 乐毅得势仍不肯轻快贪

齐北外那条线，是在辰时前后真正裂开的。

不是哪一垒被单独撞穿。

而是中线一断，右应不及，左补迟号，整条原本还想勉强并住的北外骨一下子都跟着发空。

最先白了脸的，反而不是寻常小卒。

是那些最知道这条线这些日子怎么一寸寸被逼着改、被逼着并、又怎么一寸寸都还没养熟的人。

他们一看见这回不是旧口再失、不是小处再误，而是新将新令落下才半日，整层骨便真叫人顺着那半寸之误撕开，心里那口原还想再硬撑一撑的气，便先塌了一层。

前线急报入临淄时，齐湣王正在宫中强逼自己坐定。

他已经知道，今日若再慌着立起身来、或再用那些最顺手的旧词先压一压心口，便只会叫满殿都看见大王先乱。

可当“北外整线裂”“新将压不住”“联军韩魏并进而赵后不断”这几句报出来时，他还是猛地站了起来。

案前几名近臣同时低下头去。

因为他们知道，这不是再能靠一句“不过外线小失”遮过去的时候了。

田文站在下首，只觉得胸口像被一块更冷更沉的铁压住。

他不是没想过会有这一日。

只是没想到，会来得这样快。

北外整层一裂，齐接下来最难的，已不再是守某一垒、堵某一口。

而是如何在一条整线都被人看透、带透、撕透之后，重新把城外那口胆养回来。

殿中很快有人失声请调更重之兵，要不惜一切先把北外再并回去。

也有人主张速收更内一层，把外骨先让。

齐湣王脸色来回变了几次，终于道：“诸君闭口。”

这一声极硬。

可硬归硬，谁都听得出，其中已带出了一层前些日子还没有的急。

过了片刻，他才转头看向田文：“你说。”

田文心里微微一震。

因为这已不是大王要他补一句逆耳之言。

这是真把那口最难听、也最不想听的实话，推到了他眼前。

他深吸一口气，最终只道：“臣请大王先认一件事。”

“什么事？”

“北外这口，已不再是‘如何再缝一针’。”田文答，“而是我齐接下来，是要用哪一层骨，去换回城外这整口胆。”

满殿静得能听见铜漏滴水。

因为这话太重。

它等于在说：此刻再只想着把刚裂开的那层草草缝回去，已未必够。

齐湣王眼神沉得发黑：“你的意思，是要寡人认让？”

田文俯身：“臣的意思，是要大王先认轻重。若为了争今日之面，明日把整城外更要紧那层骨也一并逼裂，那便不是让与不让之争，是齐自己替乐毅把后面的路开了。”

这几句话一出，殿中竟没人能立刻驳。

因为谁都知道，到这一步，再说“立刻夺回”“一鼓并回”固然听着热，可若一热便再踩错拍，后面失的便可能不止北外。

齐湣王闭眼良久，终究还是没有立刻下“死夺”之令。

这一停，比任何一句话都更说明问题。

因为齐到了这里，终于也开始被逼着学一件它本最不愿学的事：

不是处处都把面子先顶在最前。

而是要先认，哪一层骨还值得拼，哪一层又不得不忍着疼先让半寸。

而在前线，韩老将这一回是真的杀出了气。

“主将！齐整层已裂，再不给我一口，我便夜里都睡不着！”

帐中竟有人被他这句硬话惹得笑了一下。

可乐毅仍旧没有被这股气带着走。

他先看回报：

韩前军已入深一层；

魏中军侧路已立；

赵后线一日未断；

齐边新将仍在试图收口，却已顾左失右。

看完之后，他只道：“可再进，但不许散追。”

韩老将一愣：“还不许？”

乐毅看着他，声音极稳：“今日我们开的是齐一整层骨，不是一口赌命之门。门既已开，它后头自然会自己生慌。此时若诸军各凭血气先冲，明日乱的便不是齐，是我们。”

韩老将站在那里，胸口还剧烈起伏着，可这回却没再顶一句。

他终于从心里真认了：

乐毅之所以能把齐逼到今日，不是因为他每一口都比别人咬得更狠。

恰恰是因为他每一口都知道，只该咬到哪儿。

帐外风过，带着血气、泥气、火气，一股脑卷进来。

乐毅抬眼望向南面。

他知道，今日这一整层骨一裂，临淄后面必有更重的令、更难的取舍、更急的换法。

而他要做的，仍不是为眼前多吃几步地就先乱了自己的节。

他要等。

等齐下一层更苦、更急、也更容易自己做错的应法。

因为大战到这一步，真正会决定后面走向的，已经不只是谁今日多夺一垒。

而是谁能在对面整层骨裂之后，仍旧不被眼前那点热血和眼前那点便宜先拖偏。
