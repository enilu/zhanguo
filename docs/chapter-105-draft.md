# 第一百零五回 临淄艰定弃外保内议 田文一言先保齐城气

北外整层一裂之后，临淄宫中连着两日都像压着一口未能吐尽的闷血。

主战者仍不甘。

他们一遍遍盯着图，想从那条已被撕开的线里再找出一口“也许还能立刻并回”的缝。

守重者却越来越多。

不是他们不想争。

而是到了这一层，谁都看得出来，若再为了城外已裂那一层骨硬去争快，极可能把城前更要紧那层气也一并赔进去。

齐湣王这回没有急着当殿拍板。

他把众人留下，自己反而先退进内殿。

田文也被单独召入。

内殿比外头更静，只听得见窗外风一阵阵掠过檐角。

齐湣王立在图前，背对着他，许久才道：“孟尝君，寡人若现在收北外更内一层，外间会怎么说？”

田文沉默了一下，答得极慢：“会说齐退了。”

“然后呢？”

“若退得住，后头会有人说，大王总算认了轻重。”田文道，“若退不住，只图遮眼前一句难听，后头外间便不会只说‘退’，会说齐连自己还值不值得守哪一层骨，都已先看不清了。”

这话极难听。

可齐湣王这回没有立刻变色。

因为他自己心里也清楚，到这一步，最伤人的已不是外头一句“齐退了”。

最伤人的，是自己明知该退半寸，还因咽不下这句难听，硬把一整层更要紧的东西也陪进去。

他终于转过身来，看着田文：“你以为，寡人该如何？”

田文一拜：“臣请大王弃北外外层之面，先保临淄前线之气。”

“说清。”

“外层骨既已裂，今日最要紧的，不是把那一层缝得像旧日一样好看。”田文道，“是先叫城中、军中都看明白：齐不是乱了，是在收。能收得整，后面才有再争之本。若为了‘不能让天下说齐退’，反逼前线明知不值也去争，那才是真教天下看见齐慌。”

这几句话，终于真正落到了齐湣王最难受的那一点上。

他缓缓闭了闭眼。

再睁开时，声音已比先前更低，也更硬：“传令。北外弃外保内。诸垒不许各自争面，只守新定中层。城前号令自今日起，不许一人一垒为自己求快。”

这道令发出去的时候，田文心里反倒轻了一寸。

不是因为齐局自此便稳。

而是因为大王终于肯在最伤面子的时候，先替齐保那口还没全散的城气。

若连这一步都不肯走，那后面许多事便再也无从谈起。

散出内殿时，田文抬头望见天色已近黄昏，宫外回廊长长一线，被日色压得发暗。

他忽然明白，齐到如今，终于也被逼着学会了一种过去最不爱学的本事：

不是只会高处压人。

而是在真被人撕开一层之后，还能不能忍着疼，先把余下那层更要紧的气收住。
