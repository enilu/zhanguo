# 第一百零五回 临淄艰定弃外保内议 田文一言只求城中气不断

北外整层一裂之后，临淄宫中连着两日都像压着一口未能吐尽的闷血。

齐湣王这回没有急着当殿拍板。他把众人留下，自己反而先退进内殿，田文也被单独召入。

内殿比外头更静，只听得见窗外风一阵阵掠过檐角。

齐湣王立在图前，背对着他，许久才道：“孟尝君，寡人若现在收北外更内一层，外间会怎么说？”

田文答得极慢：“会说齐退了。可若为了咽下这句难听，硬把一整层更要紧的东西也陪进去，那便不是退与不退之争，是齐自己先看不清轻重。”

齐湣王终于转过身来：“你以为，寡人该如何？”

田文一拜：“臣请大王弃北外外层之面，先保临淄前线之气。外层骨既已裂，今日最要紧的，不是把那一层缝得像旧日一样好看，是先叫城中、军中都看明白：齐不是乱了，是在收。”

这几句话，终于真正落到了齐湣王最难受的那一点上。

再睁眼时，他声音已比先前更低，也更硬：“传令。北外弃外保内。诸垒不许各自争面，只守新定中层。城前号令自今日起，不许一人一垒为自己求快。”

这道令写下去时，田文看着齐湣王的手，分明见那一笔“弃外”落得比旁边几字都重。

因为这不是寻常军令。

这是临淄头一回在还未见城门大动之前，就先承认有些看着还像齐国门面的东西，眼下已经保不住，也不该再硬保。大王要写下这两个字，难过的不只是失一层垒，更是失那口“我还能把旧样子撑回去”的气。

可也正因这两个字终于落了纸，临淄才第一次像是开始照着新局活，而不是只照着旧面补。
