# 第一百零六回 乐毅见齐能收便转锋 联军不骄再逼第二层

齐“弃外保内”的新令，很快也传到了联军帐中。

韩老将听完，先是一怔：“它竟真肯退这一步？”

赵奢看完回报，低声道：“齐若真收得整，后头会比前几日更难啃。”

魏节应将也点头：“乱而不收，最易打；裂而能收，才最磨人。”

乐毅把那道新令压在图上，看了很久，才道：“齐总算活过来半口。”

韩老将忍不住道：“那我们是否该趁它新收未稳，再狠狠干它？”

乐毅却摇头：“不抢它刚收住的那一层。因为它此刻最醒的，便是那一层。今日该逼的，不是它刚忍痛保下的地方，而是它为保那层，不得不先让出来、也最易暂时照顾不到的第二层。”

说完，他把手移到更侧一处连着齐新内线与后续应车的偏口上。

赵奢一看，眼神立刻沉了：“若齐把前线中层收得太死，这一偏口的后应与补转反会先薄。”

于是夜里，联军果然不再照着北外最热那条中层去逼。

韩前军稳住原口；

魏中军仍像昨日那样把眼压在正中；

赵后线照旧不乱，火却刻意把轻重做得更像主力仍在此处；

真正往外滑出去的燕、韩轻股，却已沿着那道偏口外的旧缓地，一寸寸摸过去。

这一步走得极轻。

轻到前线许多老卒起初都没看出主将为何忽然改锋。可赵奢站在坡上，望着那几处火色与骑影，心里已经明白：齐若真把北外中层收住，它自己营中上下这两日最醒的、最紧的，便都在那里。越是刚忍痛保下来的地方，人越会盯得死；而为保那一层新整出来的体面，旁边一些本来能慢慢顾上的偏口、转路、续应，便总会先薄一点。

乐毅这一夜不碰最热处，偏取那一线刚被齐自己让出来的凉处，正是要叫齐知道：

它会收，

联军也会跟着换手。
