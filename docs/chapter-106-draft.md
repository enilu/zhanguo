# 第一百零六回 乐毅见齐能收先转锋 联军不骄再逼第二层

齐“弃外保内”的新令，很快也传到了联军帐中。

韩老将听完，先是一怔，随即皱起眉：“它竟真肯退这一步？”

这一步，在他这样的军中宿将眼里，其实并不陌生。

可放到齐这几年一路高压诸侯、处处爱把脸面顶在最前的气口上，便显得极少，也极重。

赵奢看完回报，低声道：“齐若真收得整，后头会比前几日更难啃。”

魏节应将也点头：“乱而不收，最易打；裂而能收，才最磨人。”

帐中几名年轻将校一时都不吭声了。

因为他们原先都以为，齐既整层骨裂，后面多半便要一路乱下去。

如今却忽然发现，对面并非只有一味骄与一味乱。

它居然还能在最难看的时候，先忍下一层退。

所有人这才更真切地明白，乐毅此前为何总是一再压着不让联军因小利先乱：

因为他打的，本就不是一国木头人。

而是一头会痛、会惊、会乱，也还会学着收住自己的大国。

乐毅把那道新令压在图上，看了很久，才道：“齐总算活过来半口。”

韩老将忍不住道：“那我们是否该趁它新收未稳，再狠狠干它？”

乐毅却摇头：“不抢它刚收住的那一层。”

“为何？”

“因为它此刻最醒的，便是那一层。”乐毅道，“我们若照着它刚收住的地方硬撞，便是在替它试新骨够不够硬。今日该逼的，不是它刚忍痛保下的那层，而是它为保那层，不得不先让出来、也最易暂时照顾不到的第二层。”

说完，他把手移到更侧一处并不最重、却连着齐新内线与后续应车的偏口上。

那不是前几日众人眼里最显眼的地方。

可赵奢一看，眼神立刻沉了：“若齐把前线中层收得太死，这一偏口的后应与补转反会先薄。”

魏节应将也道：“而且它刚换法、刚换将、刚换心，越想把一层收整，别处越易暂时留白。”

韩老将这回终于没有再说“可惜”。

他只盯着图，过了片刻才低声道：“原来主将从来不是等它全乱。主将是在等它每次想把自己救回来时，先露哪一层新白。”

乐毅点了点头：“正是。”

他随即下令：

“前军休一线，不争最热之口。”

“魏中军向侧略移，假作仍盯中层。”

“赵后线火一明一暗，让它以为我军仍重昨日之口。”

“燕本部小股先压偏口外探，不与其正应早见。”

这道令一下，帐中竟有一种比前几日更沉的静。

因为到这一步，诸将都已慢慢习惯了主将的打法：

不是敌哪里裂得最大，便去扑哪里；

而是敌每一次自救时，哪里最不得不新露出一层白，便先压哪里。

夜里，联军果然不再照着北外最热那条中层去逼。

韩前军稳住原口，不躁；

魏中军仍像昨日那样把眼压在正中；

赵后线照旧不乱，火却刻意把轻重做得更像“主力仍在此处”；

而真正往外滑出去的燕、韩轻股，却已沿着那道偏口外的旧缓地，一寸寸摸过去。

齐边那边，新将今夜也不敢懈。

他已吃过“只快半寸便误半寸”的亏，这一回把自己压得极狠，专盯中层与已裂旧口，生怕联军再从正处狠狠干来。

可人越是把一口气全压在最痛那处，别处便越容易生出自以为“尚可缓半刻”的空。

而这半刻，正是乐毅最爱要的东西。

高处夜风里，乐毅远远望着齐边火色，神情比前几日更静。

因为他知道，这一场伐齐最难的，不是把齐打成只会乱的敌。

而是把它每一次想重新站稳的力，都逼成联军下一次可下手的口。

到这里，他与齐争的已不再只是锋。

更是在争：

谁更会在自己疼的时候，还能忍住不走错下一步。
