# 第一百零七回 偏口忽明齐边又失措 乐毅再取半层仍按诸军

齐边新内线收住之后的第二夜，本该是最能喘上一口的夜。

可乐毅偏偏不去碰那一口。

夜到三更，偏口外忽然起了很轻的一阵火。不是大火，只是一线线火点，忽明忽暗，像有小股骑卒在借地试路。

齐新将得报时，手指微微顿了一下。副将低声道：“将军，可否仍照旧守？”

新将没有立刻答。最终还是下了令：“偏口加骑，不动中层主线。若有实兵，先逼退再报。”

这道令看着稳。可也正因为他要稳，派去偏口的那支人马便不得不从本就刚刚才收整好的一条补应线上硬挤过去。

联军等的，便是这一勒。

偏口外那点忽明忽暗的火光里，真正藏着的韩、燕轻股这时终于动了。不多，却快。他们不去硬撞齐新派来的骑，只专往那条被齐自己挤薄了的补应线侧腹切。与此同时，魏中军那边也忽然起了侧号，像仍在正中等着，却又恰恰把齐新将最不敢抽空的那一层心气再压了一下。

新派去偏口的骑被韩、燕轻股一下咬住半腰，想退，后路已窄；想进，又被旧缓地形与自己方才硬挤出来的车队绊住；中层那边则因魏号骤起，不敢真抽空去救，只能眼睁睁看着偏口那条边补线，先乱成了一团。

到了天将亮时，齐偏口虽未尽失，却已被联军一口咬开半层。

韩老将看着偏口那层新得之地，眼里又是一亮：“主将，这回总该追了！”

乐毅却仍旧不许：“偏口得半层，已经够齐心再乱一层。因为眼前每多拿一步，便可能少逼出它后面一整层的错。”

说完，他把偏口与中层之间那条刚被齐自己挤薄的补应线又看了一遍。

诸将这才慢慢明白，主将眼里最值钱的并不是这半层地，而是齐新将为了守“稳”字，竟也被逼得自己去挤自己那条线。挤出这一回，后头齐再说“我已学稳”“我已换法”，临淄城里也没人能再全信得那么整了。
