# 第一百零七回 偏口忽明齐边又失措 乐毅再取半层不许诸军快

齐边新内线收住之后的第二夜，本该是最能喘上一口的夜。

中层已并，

外口已让，

新将也总算把前几日那股“各垒各自争面”的散气压下去一层。

连不少齐军将校都在心里暗暗发愿，只要联军今夜仍照着中层与旧口来逼，自己这一回便绝不再乱。

可乐毅偏偏不去碰那一口。

夜到三更，偏口外忽然起了很轻的一阵火。

不是大火。

只是一线线火点，忽明忽暗，像有小股骑卒在借地试路。齐边瞭望卒最初看见，还以为不过又是前几日那种“放影试眼”的旧手。

可这一回不一样。

那几处火点压得极低，却偏偏正亮在偏口外那条旧缓地与补转道之间最叫人难受的地方。看着像虚，偏又不像全虚；看着不重，偏又刚好亮在你最不愿它亮的那一线上。

齐新将得报时，手指微微顿了一下。

副将低声道：“将军，可否仍照旧守？”

新将没有立刻答。

因为他心里清楚，这一线若真只是联军放影，自己一动，便又会重蹈前几日“因急自误”的旧辙；可若这里头真藏着一口准备借旧缓而入的实兵，偏口一旦失手，自己这些日子好不容易才收住的那半层新内线，又得被逼着回头。

这正是主将最难的时候。

不动，怕失；

一动，又怕正中对面之计。

他终究还是下了令：“偏口加骑，不动中层主线。若有实兵，先逼退再报。”

这道令看着稳。

比前几日那些半寸半寸压过头的令稳得多。

可也正因为他要稳，派去偏口的那支人马便不得不从本就刚刚才收整好的一条补应线上硬挤过去。

问题，就出在这“硬挤”二字上。

偏口不比正中。

路窄，地斜，原本就是先前收线时被有意让出来、又一直只靠旧缓吊着的一截。齐军这一压骑过去，等于是在自己最没空余的筋骨上，再往前猛勒一把。

联军等的，便是这一勒。

偏口外那点忽明忽暗的火光里，真正藏着的韩、燕轻股这时终于动了。

不多，

却快。

他们不去硬撞齐新派来的骑，只专往那条被齐自己挤薄了的补应线侧腹切。与此同时，魏中军那边也忽然起了侧号，像仍在正中等着，却又恰恰把齐新将最不敢抽空的那一层心气再压了一下。

齐新将脸色当场变了。

“回补！”

“中层不得动！”

“偏口先锁！”

三道令几乎同时出口，结果便是谁也没真先动成。

新派去偏口的骑被韩、燕轻股一下咬住半腰，想退，后路已窄；想进，又被旧缓地形与自己方才硬挤出来的车队绊住；中层那边则因魏号骤起，不敢真抽空去救，只能眼睁睁看着偏口那条本来还算“可缓”的边补线，先乱成了一团。

到了天将亮时，齐偏口虽未尽失，却已被联军一口咬开半层。

更要命的，不是这半层地。

而是齐新将这一回已经没有“半寸压过头”那样好解释了。

他明明是想稳，

想先加骑再看，

可最后仍是自己在最紧那条线里，再挤出了一处乱。

这比单纯的急，更伤心。

因为它让你连“若我更稳一些便好了”这句自安的话，都不好再说得那么完整。

而在联军营中，韩老将看着偏口那层新得之地，眼里又是一亮：“主将，这回总该追了！”

乐毅却仍旧不许。

“偏口得半层，已经够齐心再乱一层。”

“为何总是够？”

乐毅看了他一眼：“因为眼前每多拿一步，便可能少逼出它后面一整层的错。我们今日不是来抢小快的。是来逼它自己发现，连想稳着救、都还会救出新口来。”

韩老将这回没再争。

他站在高处，看着齐边偏口那一团还未完全散尽的乱烟，心里第一次真正生出一种近乎发寒的服。

原来主将最厉害处，不是他能看见敌人哪里乱。

是他总能比敌人更早看见：

敌人下一次想不乱时，会从哪里再乱出来。
