# 第一百零八回 临淄方知稳中也能生误 齐王胸口第一次发空

偏口半层再失的报文送入临淄时，齐湣王沉默得比前几次更久。

若说前几回他还能把错处归到“轻敌”“改得不够快”“换将不及”上去，这一回，连他自己都很难再顺着这些旧路去安自己。因为新将这回并不是不稳，恰恰相反，是太想稳，太想先不乱、先看准、先把中层守死。

齐湣王把报文放下，问了一句连他自己都未必想听懂的话：

“它到底是怎么做到的？”

殿中无人敢立刻答。

田文最终只道：“大王，齐眼下不能再只盯一口一口的得失。要盯节。若只盯哪口失、哪口要夺回来，后面便总要被乐毅引着一处处救火。”

齐湣王望着图，许久没有说话。到最后，他只缓缓吐出一句：

“寡人如今连想稳，也像都稳不进它那一步里。”

说完这句话后，他没有立刻起身，只把那卷报文压在图角上，目光从北外一路往中道挪。

殿中几名近臣这时才真正听出一层更冷的意思来。若说前几回失口，还能归在轻敌、躁进、换将不及，这一回却连“想稳”也成了乐毅能用的地方。那便说明齐如今不是哪一法没学会，而是乐毅已经总能比齐早半拍看见“这一法学会之后会露出什么新缝”。

田文站在一旁，也没有再多说。

他知道，大王这句“稳不进它那一步里”，比发怒更重。一个强国之主若第一次说出这样的话，伤的便不只是眼前一处失得，而是胸口那股“快慢终究由我来定”的旧气。
