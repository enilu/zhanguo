# 第一百零八回 临淄知稳中也会生误 齐王胸口第一次发空

偏口半层再失的报文送入临淄时，齐湣王整个人竟沉默得比前几次更久。

若说前几回他还能把错处归到“轻敌”“改得不够快”“换将不及”上去，这一回，连他自己都很难再顺着这些旧路去安自己。

因为新将这回并不是不稳。

恰恰相反，是太想稳，太想先不乱、先看准、先把中层守死。

可偏偏就在这种“稳着救”的时候，偏口又失了半层。

这就像一盆更冷的水，直接浇在齐廷这几日好不容易才攒出来的那点“也许我们只要稳下来便还能慢慢扳”的心上。

齐湣王把报文放下，问了一句连他自己都未必想听懂的话：

“它到底是怎么做到的？”

殿中无人敢立刻答。

因为这句话，问的已不只是某一战某一口。

问的是：

为什么齐每次刚想学会一点东西，对面便已在等它下一层怎么学错。

田文站在下首，心里也在发沉。

他很清楚，到了这一刻，问题已经不只是齐该战还是该守、该换将还是不该换将。

更深一层的病，已经露了出来：

齐这些年太习惯自己逼别人跟着自己换步。

如今第一次轮到自己被别人牵着节来学，就处处都显得慢、显得硬、也显得不肯先信“对面已经早看着我这一手会怎么下了”。

可这层话太重。

重到一旦说得太直，便不像救局，反像在当众宣齐旧气已尽。

他最终只道：“大王，臣以为，齐眼下不能再只盯一口一口的得失。”

齐湣王抬眼：“那该盯什么？”

“盯节。”田文答，“若只盯哪口失、哪口要夺回来，后面便总要被乐毅引着一处处救火。大王若还想把城前这口气稳住，便得先问：我齐现在还有没有一条自己的节，能不跟着它每一层新逼法去换。”

这句话一落，连几名原本最急着请战的近臣都静了。

因为他们也终于察觉到，齐这几日最怕的，确实已不只是“又失了哪半口”。

而是每一次想补，都会像被对面提前等在那层补法之后。

齐湣王望着图，许久没有说话。

到最后，他只缓缓吐出一句：“寡人如今连想稳，也像都稳不进它那一步里。”

这句话说得极轻。

可殿中几人听见时，竟都下意识把头压得更低。

因为他们知道，大王这是真切地摸到了最冷的一层：

齐不只是被攻。

而是自己的快、慢、惊、稳，都开始落在对方更前一步的算里。

这一层若不能扳回来，后面许多仗，便会越打越像是帮着乐毅把他的局做全。

散朝之后，田文独自走出宫门。

风不大，城中人也还照旧出入买卖。

可他看着这些熟得不能再熟的临淄街巷，心里第一次真生出一种极深的空。

不是因为齐已亡。

恰恰是因为它还强、还盛、还大、还没到真亡那一步。

可就是这样一个仍旧盛着的大国，如今竟已被逼到了一个极险的境地：

它每一回都想学着收、学着稳、学着改，

却总像慢在那个人半步之后。

这种慢，比一败更耗国。

因为它耗的，正是一个大国最贵的那层“我自己能定快慢”的骨。
