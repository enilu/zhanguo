# 第一百零九回 乐毅再压诸军心更定 魏将看齐终知非偶然

偏口半失之后，联军诸营里反倒比前些日子更静。

不是不振。

而是那种原本最容易因小胜而浮起来的热，到了这里，竟都被乐毅一口口压成了更沉的东西。

韩前军知道，自己再快，也得等总节；

赵后线知道，自己越稳，前面越能放手去压；

魏中军则在这几日里第一次真正感觉到，自己那口“节应第一责”不是苦差。

它是整支联军能不能真像一口兵往前走的脊。

这一夜，魏节应将独自站在营后高处，看着对面齐边那些比前几日更密、却也更乱的火，心里忽然生出一种说不清的寒。

副将走到他身后，低声道：“将军，这几日看下来，乐毅真有这般可怕么？”

魏将没有立刻答。

过了片刻，才道：“你以为这几日齐边这样，是偶然？”

副将一怔。

魏将缓缓道：“若只一次看偏，可说偶然。若只一次急了半寸，也可说偶然。可它几日来，每次想改、想并、想稳、想收，都恰好在新改那一层又露出新白。这便不是偶然。是主将一直看着它下一步要往哪儿走。”

这句话一出，副将背后竟隐隐生出一层寒意。

因为他这才真正明白，联军这几日之所以能越来越不乱，不只是因为四国终于咽下了那些苦差。

更因为前头真有一个人，能让四口不同的兵，都大致知道自己该等哪一下、该停哪一下、该忍哪一下。

而这，才是比几场小胜更可怕的东西。

同一时刻，乐毅再开中帐，看的仍不是今日得了多少地，而是：

齐偏口失后，哪一垒先添车，

哪一处火骤盛，

哪一路探骑忽多又忽少。

韩老将这回已经学会了先不问“何时再打”，反问：“主将，你看它下一回会先学什么？”

乐毅抬头看了他一眼，竟露出一点极淡的认同：“将军这句，问对了。”

韩老将自己都愣了一下，随后胸口却像被什么轻轻一撞，沉沉地稳了下来。

他知道，自己终于学会了不再只问“我们能不能再咬一口”，而开始问“它下一回会怎么改”。

而这，正是这位主将这几日一直逼着他们所有人学会的东西。

乐毅把几处新报一并压到图上，最后道：“齐下一回，怕会想先稳一处，再故意放一处，好骗我们以为它还在照旧急救。”

赵奢闻言，立刻道：“若它真学到这一步，便说明齐也开始在学主将的法。”

乐毅点头：“所以后面更不可急。我们打到今日，最怕的不是齐会乱。最怕的是齐终于在乱里学会一层真正能用的稳。”

帐中一时都静了。

因为这句话让所有人都明白，局势虽在联军这一边，可主将眼里从来没有把齐看成一个会只往下掉的敌。

他打一层，便预备着对方可能学一层；

他看一层，便在等对方下一层要怎么用错、或用对。

这已不是寻常意义上的“乘胜追击”了。

是拿大局、大国、大军，彼此一步步去磨谁更能稳住那根最要紧的节。
