# 第一百零九回 乐毅再压诸军心更定 魏将看齐方知败势非偶然

偏口半失之后，联军诸营里反倒比前些日子更静。

韩前军知道，自己再快，也得等总节；

赵后线知道，自己越稳，前面越能放手去压；

魏中军则在这几日里第一次真正感觉到，自己那口“节应第一责”不是苦差，它是整支联军能不能真像一口兵往前走的脊。

这一夜，魏节应将独自站在营后高处，看着对面齐边那些比前几日更密、却也更乱的火。副将低声道：“将军，这几日看下来，乐毅真有这般可怕么？”

魏将缓缓道：“若只一次看偏，可说偶然。若只一次急了半寸，也可说偶然。可它几日来，每次想改、想并、想稳、想收，都恰好在新改那一层又露出新白。这便不是偶然。是主将一直看着它下一步要往哪儿走。”

同一时刻，乐毅再开中帐，看的仍不是今日得了多少地，而是齐偏口失后，哪一垒先添车，哪一处火骤盛，哪一路探骑忽多又忽少。

韩老将这回已经学会了先不问“何时再打”，反问：“主将，你看它下一回会先学什么？”

乐毅抬头看了他一眼，竟露出一点极淡的认同：“将军这句，问对了。”

乐毅把几处新报一并压到图上，最后道：“齐下一回，怕会想先稳一处，再故意放一处，好骗我们以为它还在照旧急救。”

帐中几人听到这里，神色都跟着一变。

因为这话意味着，从此往后，联军要看的便不只是齐哪一处真乱、哪一处真弱，还得看它哪一处是故意放出来给人看的。齐若真学到这一步，战场便会比前几回更难打，也更像一场谁先把对方下一层心思看穿的较量。

韩老将沉默片刻，终于低声道：“原来它一学会，我们反倒更要慢。”

乐毅听了，没有再多解释，只把那几条新报往图上一压。

那一压，帐中诸将心里便都明白了。乐毅这几日压的不只是前线血气，也是在逼联军自己跟着学：齐会变，联军便也不能还用前两日的旧眼去看它。
