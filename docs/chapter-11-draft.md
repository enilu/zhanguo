# 第十一回 合从既定诸侯异志 名势方高说士知难

六印既归，礼声渐歇，真正压上苏秦肩头的，却不是赞辞，而是一件件细碎而沉的实事。

先是赵使来问，盟成之后若秦边有动，近秦诸国的报应次序当如何定；继而燕使又来探，北方若有共议，燕国的话究竟能占几分轻重；魏韩那边更急，几封文书措辞各异，说的却都是一件事：既然合从已立，总要先见些实在依凭，不能只让近秦之国担着名义先吃风险。便连齐楚两边，虽未曾急迫逼问，也都开始在礼数之外，慢慢试苏秦对后续议事的轻重取舍。

这便是盛礼之后的回落。

前十回里，众人看见的是印，是势，是一个寒士如何被推到天下中枢；第十一回一开始，压上来的却是同盟一旦真要运转，谁先动、谁后动、谁担重、谁担轻这样绕不过去的具体难处。

苏秦这才真正尝到，成势易，持势难。

一日，赵、魏、韩三边代表同坐议事。赵人道：“赵先倡其议，所图正在不独受秦锋。若盟成之后一切都仍先压近秦与赵地，那岂不是名虽成，局未改？”魏使立时接道：“赵至少兵强地固，魏却是秦逼最急处。若凡事都叫近秦之国先试，魏要这合从何用？”韩使则更直：“韩地狭小，最经不起‘先顶一阵再议后援’这种说法。”

若只听三家，各家都有不得不说的理。赵怕自己倡议到头来只换来更早受锋，魏怕盟名未真护己反先惹秦怒，韩则怕小国终究只是大局里最先被拿来安置的一块地方。

苏秦没有急着判谁是谁非，只道：“诸位若只从自家最急之处看，自然都觉得自己最该先被照看。可合从若要不散，不能人人都先问‘别人替我做什么’，而要先定一条最要紧的秩序：秦若试边，不得有国借口观望；近秦之警，不得只算近秦之国之事。此条若立，强国才不能全推，弱国也不致白白作盾。”

赵人皱眉：“话虽如此，具体到事，还是要有人先站出来。”

苏秦道：“站出来不等于孤自承担。今日最怕的，不是有人先站，而是先站者仍被诸侯当作他自己家的事。若众国先在文书与使节上把此理立定，秦再试边时，谁都无法装作没看见。合从第一步，要压的就是这层推诿之心。”

这一轮议论，从午前磨到日暮，终究只是把“近秦有变，则众国共议应之”的条文又往实里压了一寸，却已耗去了满室心力。

晚上退席之后，燕使私下又来见苏秦，低声道：“北方合势，燕是愿附的。可若往后议事总由近秦与强国先定，燕岂不永远只是尾随其后？”

苏秦看着他，道：“燕今日最要紧的，不是急争主位，而是先让所有人都认燕不在局外。位次以后可争，若现在只争位，不争在局之名，燕便仍只是边隅自守之国。你回去告诉你家主君，先守住‘燕亦共此患’这层身份，便已比从前前进一步。”

燕使沉默半晌，终究点头。他走后，苏秦独坐灯前，才觉出一阵极淡的疲惫来。

人人都敬他手中之印，人人也都把各自解不开的绳结递到他手里。可这些绳结背后，并没有哪一国真肯把自己的一整颗心都交给他。它们敬其位，借其威，必要时也愿把责任与抱怨一道推到他这里来。

高位之难，正在于此。

次日又有齐楚使者至，虽不似赵魏韩那样急迫，却都在轻描淡写间问起“若诸国议有异同，该听谁先说”“若盟中有一国迟缓，是否便算失约”之类的话。表面上仍是温礼，骨子里却都在试探苏秦到底更偏强国，还是更护近国；更重名分，还是更重生死缓急。

苏秦心里比谁都清楚，自己只要明显偏向一边，另一边便会立刻起疑。于是这一回，他已不再像前几回那样多用大段陈词，而更多是短、准、急地分配轻重。

他对齐楚说：“大国若只在意自己不受约束，这盟便只剩空名。”又对赵道：“强而先言，不是为多得名，正是为多担一点不使局散之责。”再对魏韩说：“近秦之苦最急，正因此更不可只求别人立时替你尽担；要先把众国都拖进一条不得不应的线里。”

这些话都不漂亮，却都很有用。

因为同盟走到这一层，最怕的已不是谁言辞不够壮，而是谁处理轻重缓急失了平衡。苏秦要做的，也不再是让所有人满意，而是让所有人暂时都退不到掀桌子的那一步。

数日之后，这一轮争议总算压了下去。

压下去，不是因为谁真心服谁，而是因为人人都看见，若继续争，盟未必立刻坏，信却一定先松。强国不愿见此，小国更不敢见此，于是众人各退一点，表面秩序总算仍维持住。

可苏秦心里也更明白了一层。

前十回里，六印归身看似荣耀到了极处；第十一回之后，他却知道，真正难走的路，其实才从印归之后开始。印能聚礼，印未必聚心；势能压一时，势未必压得住日日要来的新麻烦。

这一夜，他把各国文书重新分作数摞，一摞写“强国之矜”，一摞写“近秦之惧”，一摞写“弱国之疑”，看了许久，忽低声道：“原来持局之难，不在一场雄谈，而在日日都要替众人把旧心思拽回到同一个方向上。”

灯花爆了一下，又暗了下去。

局还在，印还在，势也还在。

可说士到此，已先知道了高位之难。
