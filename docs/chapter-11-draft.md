# 第十一回 六印既定诸使立刻追着问责 苏秦夜分文书始知持局比成势更难

六印既归，真正压上苏秦肩头的，却不是赞辞。

是实事。

赵使先来问，若秦边有动，近秦诸国报应次序当如何定；燕使随后探口风，问北方共议之中燕国到底能占几分轻重；魏韩那边更急，几封书措辞各异，说的却都是同一件事：既然合从已立，总得先见些实在依凭，不能只让近秦之国担着名义先吃风险。

苏秦这才真正尝到，成势易，持势难。

一日，赵、魏、韩三边代表同坐议事。赵人道：“赵先倡其议，原为不独受秦锋。若盟成之后，一切仍先压近秦与赵地，那局又有什么改动？”

魏使接声：“赵至少兵强地固。魏却是秦逼最急处。若凡事都叫近秦之国先试，魏要这合从何用？”

韩使更直：“韩地最狭，最经不起‘先顶一阵，再议后援’这种好听话。”

若只听一家，家家都有理。

苏秦便把话重新压回最要紧的一处：“诸位若只从自家最急之处看，自然都觉得自己最该先被照看。可合从若要不散，不能人人都先问‘别人替我做什么’，而要先定一条秩序：秦若试边，不得有国借口观望；近秦之警，不得只算近秦之国之事。”

赵人皱眉：“话虽如此，终究还是要有人先站出去。”

“站出去不等于孤自去死。”苏秦道，“今日最怕的，不是有人先站，是先站者仍被众国当作他自己家的事。”

这一轮议从午前磨到日暮，才把“近秦有变，则众国共议应之”又往实里压了一寸。

当夜，齐楚使者也来了。

两边话说得更圆，却都在轻描淡写间试探同一件事：往后议有异同，究竟谁先说了算；若盟中有一国迟缓，算不算失约；大国让了名面之后，还能不能保住它该有的主导。

苏秦这一回已不再大段陈辞，只短、准、急地给每一边压轻重。对齐楚说：“大国若只在意自己不受约束，这盟便只剩空名。”对赵说：“赵先言其议，不是为多得名，而是为不使局散。”对魏韩说：“近秦之国最苦，也正因此更不可只求别人立刻尽担；要先把众国都拖进一条不得不应的线里。”

夜深后，他把各国文书重新分成几摞。

一摞写“强国之矜”，

一摞写“近秦之惧”，

一摞写“弱国之疑”。

他看了许久，才低声道：“原来持局之难，不在一场雄谈，而在日日都要替众人把旧心思重新拽回同一个方向上。”

灯花爆了一下，又暗下去半寸。
