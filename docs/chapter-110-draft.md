# 第一百一十回 临淄终知乐毅先看人 乐毅望城不语待更大变

到这一步，临淄城中最叫人难受的，已不再只是边上某一口又失。

而是越来越多人开始隐隐感觉到：

城外那个人打的，像不是一条线、一座垒、一道口。

他打的，更像是齐每一回会怎么急、怎么稳、怎么怕失面子、又怎么想把自己从乱里救回来。

这层感觉，先前还只是田文心里的寒。

如今却一点点漫到了更多人的心上。

这一日，齐湣王独留田文。

他没有先问军，也没有先问垒，只问了一句：“孟尝君，乐毅最厉害处，究竟在何处？”

田文沉默极久，才缓缓答道：“在他先看人。”

齐湣王抬眼：“说下去。”

田文道：“若只看地、看垒、看兵多兵少，许多将领都能为。乐毅如今最压人的，不只是会看这些，而是他先看见我齐下一步会怎么想、怎么救、怎么忍不住、怎么想体面地把局收回来。故他每一回下手，都不止在眼前那口地上。”

这几句话出口时，连田文自己心里都微微发寒。

因为他知道，这已是把乐毅真正说到了骨里。

齐湣王没有驳。

他只是把目光慢慢移到图上，许久后才低声道：“如此说来，寡人如今不是在与他争一口垒。”

田文道：“是。大王如今是在与他争，我齐还能不能在被人看透几分之后，仍留住那几分不被他先算着走的骨。”

殿中久久无声。

齐湣王立在图前，背影比前些日子看着竟像更沉了些。

不是弱。

而是第一次有了真正要和一个能先看透自己的人对局时，那种大国之主才会有的重。

而在联军高处，乐毅这夜却没有再开小打。

他独自站在一处离前线不远的缓坡上，望着南面临淄方向。

夜里风不算大，远处城郭火色却仍看得见一层淡淡的红。

韩老将与魏节应将远远望见主将独站，谁也没有上前。

因为他们都知道，到了这里，主将看的已不再只是明日该压哪一口、哪一线。

这一大段仗打下来，齐已不是最初那个只会轻、只会骄、只会处处把自己放在高处的大国。

它也开始学着收，学着稳，甚至学着猜联军下一手。

这说明后面的仗，只会更重。

乐毅在坡上站了很久，终于只说了一句：

“更大的变，还在后头。”

随从不敢多问。

因为他知道，这位主将话里的“变”，未必只是一处边线、一口垒门。

它可能是齐终于学会一层真正难啃的稳；

也可能是齐在再多一层逼压之下，从内到外彻底换成另一种更乱的活法。

而无论哪一种，后面都不会再只是这几日这种“半口、半层”地磨。

风从临淄那头吹来，掠过联军营火，也掠过乐毅衣角。

他没有再动。

只是望着那片仍未真正被大军压近、却已先在几日来一层层逼压中渐渐发紧的齐都方向，神色比任何时候都更静。

因为他知道，真正的大局变化，往往不是出在一口最热的战阵里。

而是出在你终于把一个本来很强、很盛、很会压人的国，逼到了它不得不换一种活法的时候。
