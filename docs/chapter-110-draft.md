# 第一百一十回 临淄终知乐毅所攻在人 乐毅望城不语静候大变

到这一步，临淄城中最叫人难受的，已不再只是边上某一口又失。

而是越来越多人开始隐隐感觉到：城外那个人打的，像不是一条线、一座垒、一道口，更像是齐每一回会怎么急、怎么稳、怎么怕失面子、又怎么想把自己从乱里救回来。

这一日，齐湣王独留田文。

他没有先问军，也没有先问垒，只问了一句：“孟尝君，乐毅最厉害处，究竟在何处？”

田文沉默极久，才缓缓答道：“在他先看人。”

齐湣王抬眼：“说下去。”

田文道：“若只看地、看垒、看兵多兵少，许多将领都能为。乐毅如今最压人的，不只是会看这些，而是他先看见我齐下一步会怎么想、怎么救、怎么忍不住、怎么想体面地把局收回来。”

齐湣王没有驳，只把目光慢慢移到图上，许久后才低声道：“如此说来，寡人如今不是在与他争一口垒。”

田文道：“是。大王如今是在与他争，我齐还能不能在被人看透几分之后，仍留住那几分不被他先算着走的骨。”

而在联军高处，乐毅这夜却没有再开小打。

他独自站在一处离前线不远的缓坡上，望着南面临淄方向。夜里风不算大，远处城郭火色却仍看得见一层淡淡的红。

乐毅在坡上站了很久，终于只说了一句：

“更大的变，还在后头。”

这句说得很轻，跟在他身后的亲兵起初都没听全。

可话虽轻，意思却已很明白。北外、偏口、补道、中层，这些眼下看见的得失，都还只是齐在外面一层层换法、换气、换活路。真正要命的变化，还得等这些换法沿着县、署、城门、车道一路传进临淄心口去。到那时，齐便不是单失几口边垒，而是整国上下开始用另一种更短、更紧、更疑人的法子活。
