# 第一百一十一回 齐新练一节终要先试 乐毅按兵只看它能稳几分

齐既被逼到“换一种活法”，临淄朝中便再不能只靠旧日那种高处压人的气撑局了。

田文那句“先争还能不能自己定快慢”，终于在数日后落成了几件真事。

北外不再日日急调。

新换上去的将领也不敢再只凭一口“我来便能压住”往前硬顶，而是先按新的节重新整号、分岗、定应、试车。连最爱急着争面子的几名垒将，这几日都被压得老实了许多。

这不是齐忽然变强。

却是它在这场被乐毅逼出来的恶战里，第一次真正开始学一件自己原本最不擅长的事：

不先逞，

先练。

消息传到联军大帐时，韩老将第一个皱眉：“它若真叫这口气养顺了，后头岂不难打？”

魏节应将却比他更冷，只道：“难打，未必不是好事。若它始终只会乱，主将前面那些忍，反倒显得浅了。”

韩老将愣了一下，竟一时没接上。

因为他心里也明白，打到今日，谁都已不是在和一个只靠旧气硬撑的大国缠。

若齐真能从乱里立起一层新稳，这场仗才更像真正的伐齐之战。

乐毅听完回报，没有立刻命人去试。

他只问了一句：“它先试哪一节？”

探骑答：“先试中线联号。又试两垒同应之鼓，尚未外出。”

乐毅点头，把图上几处先前已烂熟于胸的旧口都压住，反而把手放到了更后一处看似不重的小道旁。

赵奢见状，先是一怔，随即道：“主将是要看它新节一旦真动，后头旧路还受不受得住。”

乐毅答：“正是。”

说完之后，他下了这几日来最静的一道令：

“诸军不逼北外。”

“只派轻骑远远试它新节，不给它热仗。”

“赵后线亦缓半日，不叫它误以为我军今夜必有大压。”

韩老将听得一脸不耐：“主将，敌人在那头练，我们在这头看？”

乐毅抬眼：“它若真想学稳，你越去逼，它越会把那层稳学在你刀口上。如今让它先练，我才看得清它这层稳究竟是骨，还是只是刚学会的一口气。”

这句话落下，帐中便再没人急着请战了。

因为几位主将都听出来了，这一回主将不是在等敌误。

而是在看敌会不会第一次，真的做对一步。

齐边那头，这一日也果然没见联军大逼。

新将站在高处，望着远处联军只是轻骑来回一晃、火明火暗，却不见大股压前，心里反倒更沉。

因为他知道，对面这不是怯。

而是也在看。

看自己这口新学来的稳，究竟能不能经得住没热仗时的那份难。

到了午后，齐两垒第一次照新节同应，虽仍有一处鼓慢半拍，却总算没有再像前几日那样一动就散、一换便乱。

副将看完，低声道：“将军，这一回……似乎成了些。”

新将没有露喜。

他只盯着远处联军那几道不深不浅的骑影，过了很久才道：“成不成，不在我们自己说。要看那边看完之后，会不会还照旧找得到口。”

风从齐境吹过，带着一点土气与新翻起来的木气。

那是这几日齐边重理号车、补整营线时留下的味。

联军高处，乐毅也正望着那边。

他听完今日各路探报，只说了一句：“它这回真学到一点了。”

韩老将听得胸口微微一紧。

不是怕。

而是一种更像战前压到极处的沉。

因为他知道，主将肯说这句话，便意味着：

后面的仗，才会真正更重。
