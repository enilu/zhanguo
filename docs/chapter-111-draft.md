# 第一百一十一回 齐新练一节终须先试 乐毅按兵只看它能稳几分

齐既被逼到“换一种活法”，北外便不再日日急调。

新换上去的将领也不敢再只凭一口“我来便能压住”往前硬顶，而是先按新的节重新整号、分岗、定应、试车。连最爱急着争面子的几名垒将，这几日都被压得老实了许多。

消息传到联军大帐时，韩老将第一个皱眉：“它若真叫这口气养顺了，后头岂不难打？”

乐毅听完回报，没有立刻命人去试，只问了一句：“它先试哪一节？”

探骑答：“先试中线联号。又试两垒同应之鼓，尚未外出。”

乐毅点头，把图上几处旧口都压住，反而把手放到了更后一处看似不重的小道旁。

“诸军不逼北外。”

“只派轻骑远远试它新节，不给它热仗。”

“赵后线亦缓半日。”

韩老将听得一脸不耐：“敌人在那头练，我们在这头看？”

乐毅抬眼：“它若真想学稳，你越去逼，它越会把那层稳学在你刀口上。如今让它先练，我才看得清它这层稳究竟是骨，还是只是刚学会的一口气。”

齐边那头，这一日也果然没见联军大逼。到了午后，齐两垒第一次照新节同应，虽仍有一处鼓慢半拍，却总算没有再像前几日那样一动就散、一换便乱。

新将没有露喜，只盯着远处联军那几道不深不浅的骑影：“成不成，不在我们自己说。要看那边看完之后，会不会还照旧找得到口。”

联军高处，乐毅也正望着那边。他听完今日各路探报，只说了一句：

“它这回真学到一点了。”

话音落下，韩老将反倒不似往日那样烦躁。

因为他也看见了，齐这回学到一点，联军眼前未必就更好打，可往后却更好看。学到了，才会露出“这一点学成之后，它还会怎么再学、又会在哪一步学过头”的新缝。敌若总是一味笨乱，胜它不过是一时；敌若开始真学，主将却还能总比它早看半步，那才说明乐毅如今压齐，已不只是压兵了。
