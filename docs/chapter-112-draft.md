# 第一百一十二回 乐毅不欺真稳先换刀 齐边一得新骨更难守

齐边那一层新稳既被看见，乐毅便没再照旧路去逼。

韩老将起初还以为主将会立刻改口，趁齐新节未熟狠狠干它一场。谁知次日一早，中军令先下到的，却不是前军。

而是燕本部与魏中军。

“燕轻股东侧并行，逼其转车。”

“魏中军不看前口，只看齐新节应车出与不出。”

“韩前军今日只立，不争第一声。”

这几道令一下，连魏节应将都微微抬了抬眼。

因为他看出来了，主将既已认定齐学出了一层真稳，便不再想拿旧口、旧误去硬拆它。

他要换刀。

齐那头的新将果然比前几日更能沉得住气。

燕轻股先沿东侧并行时，他并未急着调主垒车骑，只先让侧后应车挪半格，想看联军究竟是借影压人，还是另有实兵藏后。魏中军号起得也极轻，不逼前、不压正，像只是跟着看。

这种局面，最折磨人。

因为你明知对面没有乱来，偏又不知那一刀真正会从哪里切进来。

午前，新将终于下了第一道自己真正主的令：

“不争前口。”

“先护转车。”

“中层不动，侧后自整。”

这三句一出，齐边将校心里都微微一松。

因为这说明，新将并没有一见联军换刀便又急回旧路。

他开始真能压住那口“先抢眼前”的急了。

而联军高处，韩老将看着齐竟没再上钩，忍不住低声道：“主将，它这回倒真忍住了。”

乐毅点头：“正因它忍住了，才值得我们再换一层。”

说完，他目光从图上东侧那条转车道慢慢移到更后一点，那条原本并不起眼、却是齐新节若想长久站稳，后面最不能断的补木补车小道上。

赵奢一看便懂了：“它如今学的是前头能稳，可前头一稳，后头补给新节的那几条小道便更要日日顺着来。”

乐毅道：“它前面学出一层新骨，后面便必有新筋。前头难拆时，便先看它新筋养得紧不紧。”

于是当日下午，联军第一次不再把眼死压齐边最显眼那几口前线。

燕轻股转而去逼那条后补小道；

魏中军压着节，只在齐应车正想往后顺补时轻轻一顶；

韩前军则仍立原线，像故意叫齐新将心里更认定：“联军今日仍不在前口。”

新将果然一时犹疑了。

不是他看不见后补道。

而是他刚刚才靠“先护转车、不争前口”在诸垒心里立起一口新气，若立刻又把眼往后全移，前头那些刚学会一点稳的将校，多半又要慌着问“前口真不要紧么”。

这一迟，便是半刻。

而乐毅要的，仍旧只是这半刻。

燕轻股顺着那条后补小道一压，齐边后补车列先乱；魏中军再轻轻一顶，应车一时进退都不便；等新将终于回身再压时，前头几口虽未失，后头新养出来给这层稳供气的那几条小筋，已先被联军摸得明明白白。

夕阳压下来时，乐毅并没有下“追”字。

他只让探骑一路跟，看齐后补究竟会怎样补、补得多快、补得是否齐。

韩老将终于彻底服了。

“主将如今打的，已不是它哪一处还没学会。”

魏节应将接道：“是在等它哪一处刚学会，便又不得不为这层新学会的东西，露出一层新软。”

帐中众人一时都静了。

因为他们都感觉出来了，这场仗已越打越不像寻常攻守。

联军每逼齐学会一层，主将便立刻去找它为这层“新会”付出的代价。

而这种代价，恰恰就是下一层口。
