# 第一百一十二回 乐毅不欺真稳忽换刀 齐边一得新骨反更难守

齐边那一层新稳既被看见，乐毅便没再照旧路去逼。

次日一早，中军令先下到的，却不是前军，而是燕本部与魏中军：

“燕轻股东侧并行，逼其转车。”

“魏中军不看前口，只看齐新节应车出与不出。”

“韩前军今日只立，不争第一声。”

齐那头的新将果然比前几日更能沉得住气。

燕轻股先沿东侧并行时，他并未急着调主垒车骑，只先让侧后应车挪半格，想看联军究竟是借影压人，还是另有实兵藏后。午前，他终于下了第一道自己真正主的令：

“不争前口。”

“先护转车。”

“中层不动，侧后自整。”

这三句一出，齐边将校心里都微微一松。

乐毅却点了点头：“正因它忍住了，才值得我们再换一层。”

说完，他目光从图上东侧那条转车道慢慢移到更后一点，那条原本并不起眼、却是齐新节若想长久站稳，后面最不能断的补木补车小道上。

于是当日下午，联军第一次不再把眼死压齐边最显眼那几口前线。

燕轻股转而去逼那条后补小道；

魏中军压着节，只在齐应车正想往后顺补时轻轻一顶；

韩前军则仍立原线。

新将果然一时犹疑。不是他看不见后补道，而是他刚刚才靠“先护转车、不争前口”在诸垒心里立起一口新气，若立刻又把眼往后全移，前头那些刚学会一点稳的将校，多半又要慌起来。

这一迟，便是半刻。

而乐毅要的，仍旧只是这半刻。

燕轻股顺着那条后补小道一压，齐边后补车列先乱；魏中军再轻轻一顶，应车一时进退都不便。等新将终于回身再压时，前头几口虽未失，后头新养出来给这层稳供气的那几条小筋，已先被联军摸得明明白白。
