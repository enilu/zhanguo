# 第一百一十三回 临淄见齐能学更添惧 田文知强国最怕慢失真节

齐边新将这几日稳住了几口应号，又能压住前口不乱，按理说，临淄该稍稍松一口气。

可田文心里那股寒，反而更深了。

因为他比谁都清楚，最危险的时候，未必是齐只会乱。

更危险的，往往是齐好不容易学会一点稳，偏偏对面又立刻换刀，逼它为这点稳再露出别的新口。

这就像一个人刚学会忍痛走直，下一刻却发现自己为这一步“走直”，已把另一处筋先扯得发紧。

齐仍在学。

可每学一层，都像慢在对面半层之后。

这种慢，不像败那样响。

却比败更容易一点点耗空一个大国那口最贵的真节。

这一日，齐湣王再召田文。

他开口第一句便是：“孟尝君，齐若还能学、还能改，为何寡人心里反倒比前几日更不安？”

田文听见这句话，心里竟也微微一震。

因为这已不是大王在问对策。

是在问病根。

他思忖良久，才道：“因为大王如今最难受的，不是齐不会。是齐每会一层，都像晚在人一步之后。”

齐湣王盯着他：“晚一步，便如此可怕？”

田文缓缓点头：“小国晚一步，可以退，可以让，可以舍一地一城换后路。大国若总晚一步，伤的便不是一处得失，而是整国上下那口‘我自己定快慢’的真节。节若慢慢空了，国仍大、兵仍多、城仍盛，却会越来越像在别人节里做自己该做的事。”

这番话，说到这里，连田文自己都觉得背后发冷。

因为他知道，这几乎是在把齐眼下最难堪、也最不愿明说的那层局，赤裸裸摊在了大王面前。

齐湣王没有发作。

他只是缓缓坐下，望着图久久不动。

过了很久，才低声道：“如此说来，寡人如今最该保的，已不只是几口边垒。”

田文答：“是。大王如今最该保的，是齐还剩几分，能自己定节。”

殿中极静。

窗外有风掠过长廊，吹得帘角轻轻一动。

齐湣王抬头看着那一动，忽然生出一种极少有的倦。

不是疲兵之倦。

不是连夜军书之倦。

而是一个一向习惯自己替天下定轻重、定快慢、定谁高谁低的人，到这一刻才发现：

自己竟也会被别人一步步逼到不得不按着对方节来换步。

这种倦，比怒更深。

而在联军营里，乐毅这夜却没有议仗。

他只命诸将各自回营，把近三日齐边所有新应、新补、新换、新缓之处，一一写成小条，不许漏。

韩老将拿到这令时，忍不住笑了一声：“主将如今像不是在打齐，倒像在替齐记病案。”

赵奢在旁，罕见地也接了一句：“病案记得越细，刀下得越准。”

韩老将听完，竟没有再笑。

因为他知道，这不是戏言。

这些日子，主将正是这样一笔笔记、一步步看，才把齐从“轻敌”逼到“学稳”，又从“学稳”逼到“每稳一层便新露一层白”。

而后面这场仗往哪里走，怕也正压在这些看似最枯的小条上。
