# 第一百一十三回 临淄见齐能学更添惧 田文方知强国最怕慢失真节

齐边新将这几日稳住了几口应号，又能压住前口不乱，按理说，临淄该稍稍松一口气。

可田文心里那股寒，反而更深了。

这一日，齐湣王再召田文，开口第一句便是：“孟尝君，齐若还能学、还能改，为何寡人心里反倒比前几日更不安？”

田文思忖良久，才道：“因为大王如今最难受的，不是齐不会。是齐每会一层，都像晚在人一步之后。”

齐湣王盯着他：“晚一步，便如此可怕？”

田文缓缓点头：“小国晚一步，可以退，可以让。大国若总晚一步，伤的便不是一处得失，而是整国上下那口‘我自己定快慢’的真节。”

齐湣王没有发作，只是缓缓坐下，望着图久久不动。

而在联军营里，乐毅这夜却没有议仗，只命诸将各自回营，把近三日齐边所有新应、新补、新换、新缓之处，一一写成小条，不许漏。

军令传出去后，燕、赵、魏、韩诸将起初都还有些不解。

按常理，齐边既然终于学会应号、补口、压乱，联军这夜更该议的是明日从哪一面再逼一步。乐毅却偏不先议逼，只要各营把齐边新学会的东西抄成细条送上来，连几处小小调更、几次回火先后都不许漏。

到三更前后，各营小条都堆上案时，诸将才慢慢看明白。

齐每会一层，乐毅便要先看这一层会叫它哪一处跟着变慢、变重、变迟。也正因如此，田文在临淄所怕的那层“强国总晚一步”，到了联军案上，已不只是冷意，而是可以一条条写下来、明日便拿来用的东西。
