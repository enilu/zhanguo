# 第一百一十四回 乐毅点尽齐边新旧病 联军静候大崩前那口气散

次日一早，诸将所记的小条，便一卷卷送到了中帐。

韩前军记的是齐何处最怕被逼着连续换势；魏中军记的是齐哪几道应号表面已整、底下其实仍有快慢；赵后线记的是齐每回换法，哪几处补车、补木、补道最吃紧。

乐毅一卷卷看过去，几乎不说话。看到近午时，他终于把最后一卷放下，伸手在图上连点数处：

“齐如今有三层病。”

“其一，前线学得太急。每会一层，便想立刻拿那一层去救全局。”

“其二，后补跟得太苦。前线一换，后头便日日跟着换。”

“其三，城中之令越来越想兼顾。既想守面，又想守骨，既想快，又怕乱。”

魏节应将沉吟片刻，问：“那我们如今是再逼，还是先候？”

乐毅道：“逼，但不热逼。仍叫它日日不得安，日日要改，日日要补；却不给它一场可以靠死拼先把诸垒、诸将、诸心都逼成一股的大热战。”

赵奢看着图，缓缓道：“如此说来，我们眼下等的，已不是再开一口边线。”

乐毅点头：“是。我们在等齐自己那口更大的气，先换错、先吃重、先断一层。”

说到这里，他又把手往图上收了一寸，停在临淄与中城之间那几条已经被折弯过的路上。

“边线开口，只伤其皮。”乐毅道，“县路争轻重，署中争先后，城里号次开始短，这才是往骨里去。”

魏节应将听到“号次开始短”几个字，眼神微微一沉。

他终于明白，乐毅眼下已不只是在算外面这一圈围压，而是在等齐从县、署、驿、路一路改到城心去。等到那时，临淄纵还立着，旧齐也已不是旧齐。
