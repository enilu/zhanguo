# 第一百一十四回 乐毅点尽齐边新旧病 联军静候大崩前那口气

次日一早，诸将所记的小条，便一卷卷送到了中帐。

韩前军记的是：

齐何处最怕被逼着连续换势；

魏中军记的是：

齐哪几道应号表面已整、底下其实仍有快慢；

赵后线记的是：

齐每回换法，哪几处补车、补木、补道便最吃紧；

燕本部的探骑则记得更细，甚至连齐新将近两夜在哪一处垒停得最久、哪一次下令后先朝哪边看，都写了进去。

这些东西摊满案时，外人若进来，只会觉得这不像大军主帐。

更像什么医家在给一个大病之人望闻问切之后，摊开了一案子的脉案。

乐毅一卷卷看过去，几乎不说话。

帐中诸将也都屏着气。

他们如今已习惯了这位主将看似最慢、最枯的这一面。

因为这些日子，他们已经亲眼看见：主将看得越细，后面下刀便越准。

看到近午时，乐毅终于把最后一卷放下，伸手在图上连点数处。

“齐如今有三层病。”

帐中一下更静了。

这还是这些日子他第一次把对手的病，直接当着诸将一层层说出来。

“其一，前线学得太急。每会一层，便想立刻拿那一层去救全局。”

“其二，后补跟得太苦。前线一换，后头便日日跟着换，久则自重。”

“其三，城中之令越来越想兼顾。既想守面，又想守骨，既想快，又怕乱，故每道令都比该有的更重，也更难到底。”

韩老将听到这里，忍不住道：“那主将的意思，是它后面终究会有一层自己先撑断？”

乐毅抬眼：“会。只是要看，先断在前，还是先断在后；先断在将心，还是先断在城心。”

这句话，叫帐中众人都不由背后一寒。

因为他们听得出来，主将眼里此刻看的已不再只是边线一仗。

是在看整个齐局，接下来会先从哪一层开始更大地往下塌。

魏节应将沉吟片刻，问：“那我们如今是再逼，还是先候？”

乐毅道：“逼，但不热逼。”

韩老将立刻听懂了半句，却仍追问：“何谓不热逼？”

乐毅答：“仍叫它日日不得安，日日要改，日日要补；却不给它一场可以靠死拼先把诸垒、诸将、诸心都逼成一股的大热战。大热一开，反倒容易叫它把许多原本还在慢慢裂的东西，先并成一时之气。”

这番话一落，帐中许多人都沉默了。

因为他们直到这一刻才真正明白，主将这些日子为何总不许贪、不许躁、不许因小利热追。

他不是怕打。

他是怕一场太热的仗，反倒替齐把原本快要自己崩出来的那口气，先烧成一股。

赵奢看着图，缓缓道：“如此说来，我们眼下等的，已不是再开一口边线。”

乐毅点头：“是。我们在等齐自己那口更大的气，先换错、先吃重、先断一层。”

说完之后，他抬起头，看向帐外。

午后风正从南面吹来，吹得营中旗角一层层向北又向东偏。

那风不大。

却足够叫人知道，对面那座大城、那片边线、那一整国正在如何一日日被逼着换步、换法、换气。

乐毅最后只下了两道令：

“诸军照旧稳压，不许热追。”

“探骑加倍，城中消息一条不可漏。”

令下之后，帐中诸将都起身领命。

韩老将走到帐门口时，忽然回头看了主将一眼，低声道：“主将，你是在等齐自己先塌。”

乐毅没有否认，只淡淡道：“大国最难亡在外刀先重。它们往往先亡在自己不得不换法、又总慢半步的那口气上。”

帐中一时再无别声。

因为所有人都知道，这句话不是空。

他们眼前这场仗，正是在一寸寸把齐逼到那口气前去。

而更大的崩，不会太远了。
