# 第一百一十五回 齐边新稳却日日吃重 县道老马先闻国气沉

齐边这一层新稳，到了第四日，终于显出了它最沉的一面。

清晨未明，东南角三条补道上，车声便已连成一线。

守在后补道口的小校最先觉出不对。他昨日记下的车数，今日一比，竟多了近三成。

押车的主簿脸都熬青了，只把袖里小札一递：“城中改令。凡前线新节要守住，各处旧备，先挪来一半。”

而联军高处，这日仍未开一场热战。

乐毅只叫各营照旧立压，前口不猛争，后口不猛追。

赵奢站在图边，看着齐那三条补道，忽然伸手在更后一点又圈了数处：“若这些车再添三日，近边诸县便不只是供前线了。”

乐毅没有接话，只淡淡下了两道令：

“燕轻股明日改逼第二补道，不逼第一道。”

“魏中军节应照旧，只看它回车，不看它去车。”

到了午后，联军果然未向最显眼那条补道下手。燕轻股只往第二道的石桥前压了压，逼得齐边那边不得不多分半列车去应。

这一分，前头不见得立刻少什么。可第三道上，原本该在日中前回来的两列空车，却一直迟到申时还未见影。

探骑一路跟下去，到了傍晚回报：

“不是车坏，是县中换马换不出来。”

“原来备着的那十余匹，前日先抽了六匹，昨日又抽了四匹。今日回车到县，马槽只剩三匹能立着喘气的。”

而同一时刻，距前线百余里的小县里，一位年过五十的县丞也正立在仓前发怔。暮色降下来后，他独自走到后厩，看见一匹老马被卸了套，脊背上汗白未尽，腿却已经在轻轻打晃。

县丞站着看了很久，忽然道：“还能走几回？”

厩卒没有抬头，只低声回了一句：

“若照这几日的使法，走不到三回。”

县丞听完，没有再问。

他只是伸手在那匹老马汗白未尽的脊背上按了一下，掌心立刻沾了一层热湿。前线这几日还未再开大热战，城里看去也像总算稳住了几层，可百余里外这座小县里，马槽先空、役先疲、车次先乱，这些看不进临淄大图的小苦处，却已比任何军报更早告诉人：齐眼下这口新稳，是日日拿更深处的筋骨在换。
