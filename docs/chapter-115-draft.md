# 第一百一十五回 齐边新稳日日吃重 县道老马先闻国气沉

齐边这一层新稳，到了第四日，终于显出了它最沉的一面。

清晨未明，东南角三条补道上，车声便已连成一线。

先是补木车，后是转轮车，再后是装着干粮与箭束的小车。车车都不算重，单看一列，也不算吓人。可三条道连着走，走到日出之后仍不绝，那股压在人心上的沉，便慢慢出来了。

守在后补道口的小校最先觉出不对。

他昨日记下的车数，今日一比，竟多了近三成。

“怎么又添了这许多？”

押车的主簿脸都熬青了，听见这话，只把袖里小札一递：“城中改令。凡前线新节要守住，各处旧备，先挪来一半。”

小校低头一看，那札上密密写着的，不只是车数。

还有木、索、皮、钉、粮，甚至连驮车的替马都一一摊派到了近边诸县。

他看完没有作声，只抬头望向更远一点的道上。

那边正有一匹老马拉着空车慢慢往回挪。马背汗都干白了，肋下一收一放，像每走一步都在咬牙。

小校忽然明白，齐这几日所谓“稳住”，并不是白得来的。

它是把后头许多原本还藏着不动的力，一层层往前抽。

而联军高处，这日仍未开一场热战。

乐毅只叫各营照旧立压，前口不猛争，后口不猛追。

韩老将看着对面车声日重，忍不住道：“它这几日像真把骨先收住了。”

乐毅道：“骨未必没有收住。只是收骨的人，已开始吃气。”

赵奢站在图边，看着齐那三条补道，忽然伸手在更后一点又圈了数处。

“若这些车再添三日，近边诸县便不只是供前线了。”

韩老将转头：“还会如何？”

赵奢道：“会先空自己。仓会薄，马力会老，路会坏。县里主簿、亭长、押车人一层层都得跟着熬。前头得一层稳，后头便得三层人替它扛。”

这番话，乐毅没有接。

他只看着图，片刻之后，淡淡下了两道令：

“燕轻股明日改逼第二补道，不逼第一道。”

“魏中军节应照旧，只看它回车，不看它去车。”

韩老将起初没懂。

等想了一息，才猛地抬起头来。

不逼第一道，是叫齐那边更认定“这一道还能撑”；只看回车，是要看它哪条道看着还通，其实已开始回不来。

主将如今看的，已不只是前面供得上供不上。

是在看一国后头这些看似最细的筋，哪一条先吃不住。

到了午后，联军果然未向最显眼那条补道下手。

燕轻股只往第二道的石桥前压了压，逼得齐边那边不得不多分半列车去应。

这一分，前头不见得立刻少什么。

可第三道上，原本该在日中前回来的两列空车，却一直迟到申时还未见影。

探骑一路跟下去，到了傍晚回报：

“不是车坏，是县中换马换不出来。”

“原来备着的那十余匹，前日先抽了六匹，昨日又抽了四匹。今日回车到县，马槽只剩三匹能立着喘气的。”

帐中听到这里，韩老将半晌没说话。

这种事，比丢一口前垒更叫人发寒。

因为前垒失了，人人都看得见，也知道该急。

可这种县道、马槽、车轮、主簿眼下的黑青，表面看着都还撑着，等人真觉出它们的沉时，往往已积了许久。

而同一时刻，距前线百余里的小县里，一位年过五十的县丞也正立在仓前发怔。

他脚边是一封刚拆开的军札，札上要他次日再发车十五乘，替马十二匹，干粮二百石。

仓吏低声道：“再发，县里春末赈粮便要动了。”

县丞沉默良久，才道：“军前先紧。”

仓吏没敢顶，只低头应了一声。

可他退下去时，那句“军前先紧”却像一块湿石，沉沉压在了这座小县所有人的心上。

暮色降下来后，县丞独自走到后厩。

他看见一匹老马被卸了套，脊背上汗白未尽，腿却已经在轻轻打晃。

厩卒蹲在旁边，正用手一下下给它揉腿。

县丞站着看了很久，忽然道：“还能走几回？”

厩卒没有抬头，只低声回了一句：

“若照这几日的使法，走不到三回。”

风从厩门外卷进来，把草屑吹得乱了一地。

县丞没再说话。

他只是望着那匹老马一下一下喘气，忽然觉得，齐眼下这口国气，怕也正像这匹马一样。

表面还立着。

可每一回喘，已都比前一回更沉了。
