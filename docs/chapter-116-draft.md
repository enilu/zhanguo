# 第一百一十六回 田文劝齐且弃轻面 齐王忍痛收线保中仓

第六日，齐边连着送进临淄三封军报。

第一封报前口仍稳。

第二封报后补日重。

第三封最短，只写了一句：

“近边诸县，已有人求缓发。”

齐湣王把那句看了两遍，手指慢慢压在“求缓”二字上。

过了许久，田文才道：“大王，该收线了。”

齐湣王猛地抬眼：“收哪一线？”

田文答得很慢：“外面那几层看着还体面的轻面。把人、粮、车、马，往真正不能失的中仓、中道上收。”

“大王若还想样样守，后头求缓的便不会只是一县两县。等求缓之声真的连成片，再收，便晚了。”

齐湣王终于停下脚，声音压得极低：

“拟令。”

“外三层轻垒，可并者并，可撤者撤。”

“近边小仓，留壳不留实。”

“中道、中仓、主转车，不得断。”

“各县若再求缓，不许越报推来推去，由司仓、司马共签，一道回断。”

消息出宫时，天已擦黑。

出宫的不只是那道令。

还有一股谁都不愿明说、却已压到喉口的气。外三层轻垒可并，可撤者撤，近边小仓留壳不留实，这几句一旦写上简，便等于临淄头一回正式承认：有些看着还像齐国脸面的东西，眼下已不能再硬护着不放。

田文捧简退出时，风从长阶尽头灌上来，把袖角都吹得往后翻。

他知道，这一道令发得不算早，却总算发了。可他也知道，大王仍旧舍不得把病看尽。外面那些轻面虽已肯先收，中层真正该怎么并、怎么断、谁有权先在路上改次第，这令里还只写了一半。也就是说，齐不是还不肯动刀，而是只肯先割掉人人看得见的那层薄皮。
