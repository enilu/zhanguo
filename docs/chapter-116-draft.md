# 第一百一十六回 田文劝齐先弃轻面 齐王忍痛收线保中仓

第六日，齐边连着送进临淄三封军报。

第一封报前口仍稳。

第二封报后补日重。

第三封最短，只写了一句：

“近边诸县，已有人求缓发。”

齐湣王把那句看了两遍，手指慢慢压在“求缓”二字上，脸色一点点沉下去。

求缓两个字，放在平日，不过是小吏推责的常话。

可放在眼下，便像边线上最不该先露的一道口。

殿中无人先开口。

过了许久，田文才道：“大王，该收线了。”

齐湣王猛地抬眼：“收哪一线？”

田文答得很慢：“外面那几层看着还体面的轻面。把人、粮、车、马，往真正不能失的中仓、中道上收。”

这话一落，殿中立时便冷了。

因为谁都听得懂。

所谓“轻面”，并不是真轻。

那是齐这些年习惯挂在外头给天下看的气派，是那些不肯轻弃的边亭、侧垒、小仓、短道。

若一旦收回来，前线或许能更稳一层，可齐自己心里，也等于认了一句难听话：

有些原本想都不肯想的地方，如今只能先放。

齐湣王盯着田文，久久不语。

田文没有退。

他知道，这种话若说软了，便等于没说。

“大王若还想样样守，后头求缓的便不会只是一县两县。”他说，“等求缓之声真的连成片，再收，便晚了。”

殿下诸臣都不敢插话。

只有靠西那位老司仓低头看着地，像恨不得把自己看进砖缝里去。

因为他最清楚，近边几处仓的余数，这几日已被抽得有多薄。

齐湣王忽然站起身，在图前来回走了两步。

他年轻时最恨别人叫他舍。

舍名，像弱。

舍地，像败。

舍外面那层人人看得见的气，像先认输半句。

可这一回，图上那些密密的点、线、仓、道，却像一齐逼到他眼前，叫他第一次不得不承认：有些东西若不自己先狠心收，便会被别人替你打散。

他终于停下脚，声音压得极低：

“拟令。”

殿中众臣齐齐一震。

齐湣王没有再看任何人，只盯着图道：

“外三层轻垒，可并者并，可撤者撤。”

“近边小仓，留壳不留实。”

“中道、中仓、主转车，不得断。”

“各县若再求缓，不许越报推来推去，由司仓、司马共签，一道回断。”

这几句一出，殿中像忽然少了半层气。

因为人人都知道，这不是寻常调令。

这是齐第一次真正把“面”往回收，把所有还想撑着体面的地方，硬生生往“能活”那头压。

田文听完，只缓缓一揖。

他知道，大王这一笔，下得极难。

可若还想把局往后拖，这也是非下不可的一笔。

那位西边老司仓此时才终于抬起头，嘴唇微微动了动，像想谢恩，又像想叹气。

他心里清楚，这道令虽狠，却也真给后头那些快要绷断的仓、道、马槽，先让出了一口喘气。

消息出宫时，天已擦黑。

第一拨持令骑才出北门，城门下便已有人听见风声。

“外三层要收了。”

“连小仓也要空？”

“那不是把外头许多地方先放给联军看？”

这些声音起初都压得很低。

可越低，越显得沉。

因为临淄百姓不一定懂边线几道应号怎么换，却都听得懂“收线”“空仓”这几个字背后那股味。

那是齐第一次不再只想着怎么把外头重新撑圆。

而是开始算，怎样先把里面保住。

夜里，田文亲自送那道总令到北司仓。

司仓印泥已备好，却迟迟没有落印。

田文看着那位老司仓微微发抖的手，忽然问：“怕后人说你先空外仓？”

老司仓苦笑了一声：“臣怕的不是这句。臣怕的是这一印落下去，齐自己心里也知道，从今夜起，有些地方已不能再按旧年那样看了。”

田文听完，没有劝。

他只把总令往前又推了半寸。

印终于落下。

那一声很轻。

却像替齐把外头那层原本最不肯收的硬面，轻轻敲裂了一道缝。
