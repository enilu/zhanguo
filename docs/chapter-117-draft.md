# 第一百一十七回 乐毅不急争空垒先收人 郡城初见齐令未到心先偏

齐那道收线令出后第三日，联军前口果然看见了空。

不是大空。

只是原本侧翼三座互应的小垒里，有一座清晨起便再不见炊烟。

韩前军最先报到中军，语气里都带了热：

“主将，齐退了一口！”

“末将请乘势拿下，再逼旁边两垒！”

乐毅看完军报，只问了一句：“那垒外百姓，昨夜走了没有？”

来报的小校一愣，忙道：“大半没走。只听说垒中兵夜里撤得急，县门天亮才关。”

乐毅点了点头，把军报放下。

“此垒可取，不可抢。”

“先出告示：不入民舍，不动私仓，不扰井市。”

“再遣能说齐语者先入县，问县丞、父老，要不要按旧例开仓煮粥，稳住街市。”

帐中诸将一时都静了。

韩老将最先明白过来，低声道：“主将不是只要这一口空垒。”

乐毅道：“齐自己收线，最怕的从来不是丢一处土。是令还在路上，人心已先散半步。”

赵奢在旁接道：“若这县里今日乱上一乱，临淄还可说是被联军趁空抢乱。若县里今日反而稳了，外头诸城便都会想：原来齐退了，日子也未必立刻更坏。”

这句话一出，魏节应将都微微吸了一口气。

他到这时才觉出，主将这一刀，已不止往军线上下。

是在往齐那条“只要王令在，诸县自会咬牙先撑”的旧气上切。

午后，联军一队不过百人的小股先入空垒。

他们没有先立旗。

第一件事，是把垒门口散着的三辆弃车推到一旁，又叫军中书记把告示贴在县门外。

告示字不多，只四句：

“旧吏照旧理事。”

“民舍不扰。”

“井市照常。”

“有饥者可报。”

县门后本来挤着一群探头探脑的人。

他们原以为联军一进来，少不得先要喝骂、搜粮、闭市。

谁知半个时辰过去，门外先响起来的，竟是军士叫人把翻倒的井栏扶正的声音。

县中那位县丞原本还缩在署里不敢出来。

等听说联军贴了告示，又有人来请他照旧掌事时，他先是不信，后又不敢不出。

他走到门外时，正看见一名燕军书记蹲在地上，替一个跌破膝的小童把翻开的木盆扶起来。

那书记抬头见他，只道：“县君可认此地户口、仓簿？”

县丞怔了一下，下意识道：“认。”

书记便起身，把一卷空白小册递给他：“认便好。今日先记缺粮几户、病伤几人。主将有令，县中若稳，不必惊市。”

这几句说得平平。

却叫县丞站在原地，好半天没回过神。

因为他来之前，心里早已想好最坏的几层。

却独独没想过，对方入城之后，第一件事竟是叫他照旧理事。

傍晚，北门外粥锅真支了起来。

锅下火不大，粥也薄。

可当那股热气慢慢冒上来时，县门内外许多人看着它，心口都像被轻轻碰了一下。

这一碰，不是感恩。

更不是立刻变心。

而是一种极难说出口的动摇：

齐退了这一口，临淄的令还没来，外头的日子，竟也没有立刻塌。

而这层动摇，比丢一垒更难补。

夜色深下来后，探骑再报：

“旁边两县已都知道这边今日无乱。”

“有一县还特意遣人远远来看了两回。”

乐毅听完，只叫把明日的军旗再往后收半里，不许故意张声势。

韩老将问：“主将，既已叫人看见，为何反而收旗？”

乐毅道：“收旗，他们才更会自己去想。若我替他们把声势做满，他们反倒容易同仇。”

帐外夜风掠过，吹得灯影微微一晃。

乐毅没有再多说一句。

可诸将都明白，齐收线之后，真正先松的，也许还不是哪一道土垒。

而是那些原本靠着“王令终会到、齐终会全守”的心。
