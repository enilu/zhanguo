# 第一百一十七回 乐毅不急争空垒转先收人 郡城初见齐令未到心已偏

齐那道收线令出后第三日，联军前口果然看见了空。

不是大空，只是原本侧翼三座互应的小垒里，有一座清晨起便再不见炊烟。

韩前军最先报到中军：“主将，齐退了一口！”

乐毅看完军报，只问了一句：“那垒外百姓，昨夜走了没有？”

来报的小校一愣，忙道：“大半没走。只听说垒中兵夜里撤得急，县门天亮才关。”

乐毅点了点头：

“此垒可取，不可抢。”

“先出告示：不入民舍，不动私仓，不扰井市。”

“再遣能说齐语者先入县，问县丞、父老，要不要按旧例开仓煮粥，稳住街市。”

午后，联军一队不过百人的小股先入空垒。他们没有先立旗，第一件事，是把垒门口散着的三辆弃车推到一旁，又叫军中书记把告示贴在县门外。

告示字不多，只四句：

“旧吏照旧理事。”

“民舍不扰。”

“井市照常。”

“有饥者可报。”

傍晚，北门外粥锅真支了起来。锅下火不大，粥也薄。可当那股热气慢慢冒上来时，县门内外许多人看着它，心口都像被轻轻碰了一下。

夜色深下来后，探骑再报：

“旁边两县已都知道这边今日无乱。”

“有一县还特意遣人远远来看了两回。”

乐毅听完，只叫把明日的军旗再往后收半里，不许故意张声势。

这半里一收，看上去像是联军不欲趁空垒大作声势，实则比立满一城旗更叫旁边几县发寒。

因为它说明一件事：乐毅要的不是一日吓出满地逃人，而是叫齐边诸县一点点看见，城外这支兵既能取垒，又能守秩序；既会逼人失守，也不急着把失守之地立刻变成一片乱地。这样一来，旁边那些还在门里门外犹疑的人心，便会比被大旗大鼓猛压时更慢、更深地偏过去。
