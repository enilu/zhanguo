# 第一百一十八回 临淄连夜看诸县异动 齐王持未发急令到天明

第四日夜里，临淄北司马署的灯，一直亮到三更。

案上摊着六封急报。

前两封还只是报外垒既撤，百姓惊惶；

第三封便已写到：

“某县未得中朝新断，先自行将小仓并入主仓。”

第四封更难看：

“某亭长以护转车为先，擅借县门役卒，不待上命。”

田文坐在灯下，一封一封看过去，手一直没有离开案角。

这些事单拿出来，都不算大乱。

可连在一处，便叫人看见了另一种比败兵更冷的东西。

那就是诸县、诸亭、诸小吏开始自己先动。

他们未必是反。

也未必是不忠。

只是前线催得紧，城中回令又重又慢，于是人人都开始先替自己辖下那一小块地，抢一口先喘之气。

这种气一旦生出来，齐那道原本自上而下压到底的旧力，便会在极细的地方先碎。

田文看完最后一封，起身出署，直往宫中去。

夜已很深，宫门守者见他神色，连通传都快了几分。

齐湣王披衣起坐，听见是司马、司仓两署连夜并报，眉头便先压住了。

等田文把那六封急报一一呈上，他一开始还坐得住。

看到第四封时，手已微微攥紧。

看到第六封那句“县中父老问：若外垒已撤，后续所纳应输，是仍照旧，还是减半”时，他忽然把竹简重重按在案上。

“他们在问什么？”

“是在问，国朝一令未下，他们凭什么先算自己该出几分？”

殿中无人敢应。

田文也没有立刻开口。

他知道，大王怒的并不只是那句问话。

怒的是这一问背后那层味。

那是诸县开始不再只等中朝替他们定轻重，而是自己先掂量：这口气，我还该替齐顶到哪里。

良久，田文才道：“大王，诸县不是有意犯上。是令重，路远，事急。它们先求活路了。”

齐湣王猛地抬头：“求活路，便可先自改国令？”

田文迎着他的目光，答得极稳：“若中朝此时还只怪它们改，便会有更多地方先学着瞒。与其叫它们暗改，不如先给一条明路，叫它们知道，哪几件事可权变，哪几件事一步也不可擅。”

这话说得很硬。

殿中诸臣的背，都不自觉绷紧了。

因为这等于是在劝大王承认：

齐如今最要紧的，已不是把每一处都死死按回旧制。

而是先给这张越来越吃重的网，一点不至立时绷断的余地。

齐湣王站起身，走到殿前长窗边。

窗外夜色压得极深，宫墙外风过树梢，像有人一阵阵在远处低声说话。

他从前最信的，便是齐令一下，四境自应。

如今这些急报却逼着他看见：不是四境不应了。

是齐令太多、太重、太慢，压到县里亭里时，底下人已先被战事和车马逼得不得不自己动半步。

这半步若只是一县，还能斥。

若一县学一县，一亭学一亭，最后塌的，便不是哪条边线。

而是“齐命所至，天下皆知其重”的那口老气。

他站了许久，才回身道：“取笔。”

内侍忙把笔与空简捧上。

齐湣王落笔极快，先写了三条：

“外县并小仓者，事后报，不以擅论。”

“亭役可暂借护转车，但不得越县擅调。”

“中道、中仓、主应三项，不得权变。”

写到这里，他笔尖忽然停住。

因为他忽然发现，这道令只要发下去，便等于亲口承认：齐如今已有许多地方，须靠“权变”先续气。

这与他从前最不肯说、也最不愿天下听见的话，只隔着这最后一笔。

殿中静得连灯花一爆都听得见。

田文站在阶下，没有催。

他知道，大王今夜真正难下的，不是前头三条。

而是最后到底写不写那句：

“有权变之地，不必层层待报。”

若不写，诸县往后仍会暗里自改。

若写了，齐自己也就承认，这张大网已不得不松一松。

更漏一点点往前移。

殿外东方尚黑，离天明却已不远。

齐湣王执笔站着，手背青筋一寸寸浮起，又一点点压下去。

那最后一句，始终没有落。

直到宫墙外第一声晓角隐隐传来，他才把那道写了大半的急令慢慢卷起，没有加印。

田文看见那未发的竹简，心头忽然一沉。

因为他知道，齐今夜最怕的，并不是没有令。

而是有些必须先改的东西，大王自己也还没有狠下心，替天下把那半步认出来。
