# 第一百一十八回 临淄连夜看诸县异动 齐王捏着急令一直拖到天明

第四日夜里，临淄北司马署的灯，一直亮到三更。

案上摊着六封急报。

前两封还只是报外垒既撤，百姓惊惶；

第三封便已写到：

“某县未得中朝新断，先自行将小仓并入主仓。”

第四封更难看：

“某亭长以护转车为先，擅借县门役卒，不待上命。”

田文看完最后一封，起身出署，直往宫中去。

齐湣王披衣起坐，等田文把那六封急报一一呈上，一开始还坐得住。看到第六封那句“县中父老问：若外垒已撤，后续所纳应输，是仍照旧，还是减半”时，他忽然把竹简重重按在案上。

良久，田文才道：“大王，诸县不是有意犯上。是令重，路远，事急。它们先求活路了。”

齐湣王猛地抬头：“求活路，便可先自改国令？”

田文迎着他的目光，答得极稳：“若中朝此时还只怪它们改，便会有更多地方先学着瞒。与其叫它们暗改，不如先给一条明路，叫它们知道，哪几件事可权变，哪几件事一步也不可擅。”

齐湣王站起身，走到殿前长窗边。过了许久，才回身道：“取笔。”

内侍忙把笔与空简捧上。

齐湣王落笔极快，先写了三条：

“外县并小仓者，事后报，不以擅论。”

“亭役可暂借护转车，但不得越县擅调。”

“中道、中仓、主应三项，不得权变。”

写到这里，他笔尖忽然停住。

更漏一点点往前移，殿外东方尚黑，离天明却已不远。

直到宫墙外第一声晓角隐隐传来，他才把那道写了大半的急令慢慢卷起，没有加印。

田文站在一旁，没有催。

因为他知道，大王此刻卡住的，不只是几条字句，而是那一下最难落笔的承认。只要加印发出，便等于临淄正式告诉北面诸县：有些旧法、旧额、旧样子，眼下已经不能全照旧守了。大王肯写到一半，说明他看见了病；不肯加印，说明他还舍不得把病认到底。
