# 第一百一十九回 齐县各自护住一口气 北门远驿却递来两种回文

天明之后，临淄那道未发的急令，终究还是没有出宫。

可宫中不发，并不等于外头能停。

北门外驿舍这一日从早到午，连着收了七封县报。

前两封都还谨慎，只说“外垒既撤，已并小仓，候上命再定后续”；

第三封开始便有了别样口气：

“本县近边，车马已疲，愿照旧额七成先输，余待后断。”

到了第五封，更干脆：

“转车不断，民役已竭，县门守卒暂减四人。”

午后，田文亲自到了驿舍。

他接过回文，只看了三封，便已不再问别的。他问的是：

“第一个说七成先输的县，是不是前日那处老马先吃不住的地方？”

驿丞忙道：“正是。”

田文点了点头，目光落在第七封回文上。

那一封字写得最稳，也最难看：

“本县不敢减额。然本日后补二十车，只能到十六。”

这不是请示，是报成数。

田文看着那“十六”两个字，许久没有翻页。良久，才把那几封回文合上：“都抄一份，原件送司马、司仓，不必再压。”

与此同时，联军营里也收到了几条细碎消息：

“北面三县，回文不同。”

“有县先保车，有县先保仓，有县还在硬撑额数。”

乐毅听完，只道：“因为城中的一条总令，还没替他们把轻重定死。”

他说完，营中记事吏便把这几条县报一一抄上旁纸，另标出“先输七成”“守卒暂减”“二十止十六”几句。

这些句子乍看都像小县自保时难免露出的难看话，可并着看去，便已足够说明另一件更冷的事：齐王口里还没正式认下的那层病，底下诸县其实已经先各自认了，只是认法不同。有的先认车，有的先认卒，有的还想硬撑额数不松口。总令一日不下，这三种认法便会在北面一路并着跑，最后谁都说自己没违令，整线却先被拽成几股。
