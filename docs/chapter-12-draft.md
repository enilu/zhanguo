# 第十二回 强国守体面小国怕作盾 盟旗既起人心却各还带着旧账

邯郸城中，春深之后，馆驿往来更密了。

苏秦坐在案前看各国新送来的文书，越看越知合从之难，已不在把人请来，而在同一面旗下面，每一国其实都把另一笔旧账也带了进来。

赵、齐、楚这类有分量的国家，怕的不只是秦，也怕合从一成，自己的主导权被悄悄稀释。它们愿借同盟分秦压，却不愿平白把多年积下的兵势、山河和名位都拿来替别人担先手。

魏、韩、燕这一类更弱、更近祸或更需借势自保的国家，想法又全然不同。

它们当然盼同盟是真，因为不同盟，自己往往更快先受其祸；可也正因如此，它们最怕的便是“盟一成，自己反倒被推到最前”。

一日，几国代表同坐，席间并无高声。

齐使道：“齐自不避共制西秦之义，只是凡议定诸事，总该有个不失大国体面的安排。”

韩使便回：“小国体面原也不值什么，只是小国若总先担险，长此以往，谁还敢真信同盟之名？”

赵使又道：“赵不辞其责，但不可把赵之能战，看成理所当然多担。”

燕使最后才轻轻添一句：“燕愿附义，可也不愿盟中只有强国相议，弱国只在外头等传话。”

人人都说得很平。

也正因为太平，才更难。

苏秦回到馆舍，独坐至夜深。窗外月色照进来，落在案上那几份文书上，像把强国、小国、近国、远国各自不同的心思都摊在了一起。

他提笔写下两句：

“盟可立，心难一。”

“非盟之罪，乃人之常情。”

写完之后，他没有立刻收卷，只让那张竹简在灯下多放了一会儿。

他知道，往后的路只会比先前更难。

因为合从之旗虽已立起，旗下面的人，却都还按自己的旧账活着。
