# 第一百二十回 乐毅借静势迫齐自分轻重 联军收空道不急夺重城

又过两日，联军前线明明占了顺势，却还是没有大举前推。

韩前军里已有将校憋得难受。

“齐外垒既收，诸县又乱，何不趁势把北面那座中城也压一压？”

这种话传到中军时，韩老将自己都觉得耳熟。

因为这已经不是第一回了。

每逢齐露出新口，营里总有人想着一鼓再吃一层。

可这些日子，主将偏偏总在人人看见“可打”的时候，把手往回收半寸。

这日午后，乐毅召几位主将到图前，只问了一句：

“北面那座中城，眼下最怕什么？”

韩前军将校抢先道：“怕我军压城。”

魏节应将却摇头：“不止。它怕周边县道、转车、守卒、民役，这几日各有各的算法，最后没人替它把那一口死死供住。”

乐毅看了他一眼，算是应了。

随后他伸手在图上轻轻划了一道弧线。

那不是直逼中城的路。

而是绕着中城外三处空道、两处小仓、一处旧驿，像慢慢合起一只手。

“中城先不压。”

“先收它四周能叫人自己想重想轻的地方。”

韩老将看懂了，眼角都微微一动。

若大军直压中城，齐那边无论县里、仓里、城里心思多散，最后总会先被一股“大敌在前”的气逼成一团。

可若联军不急着打城，只去把周边那些看似轻、实则正叫齐各处自己算账的小地方一处处拿稳，那么中城里的人便会日日看见：

城还在。

可供城的四周，已不再像旧日那样都听一条线、一口气。

这比压城更磨人。

命令很快分下去。

燕本部去收北面旧驿，不许张旗喝道；

魏中军只压空道，不碰城外拒马；

韩前军则照旧立在人人看得见的前口，像一根钉子，叫那中城始终不得不看着它，却又摸不准它哪一日才真来。

最先被收住的是那处旧驿。

驿里本就无几名守卒，前一夜又被借走两人去护转车。

联军去时，门里的人甚至没有立刻关门。

他们只看见来者没有纵马冲进，也没有先砍驿旗，只是下马之后，先把驿前乱摆的两架破车推开，让路过的车能照常转出去。

驿长出来时，手还在抖。

可对面的燕将只问：“这两日从城里来的公文，哪一类最多？”

驿长怔了很久，才下意识回道：“催额的多，改额的少。”

燕将点点头，也不再问。

他只叫书记把“驿传照旧，行人勿惊”八个字写了出来，贴在驿门内。

这等做法，连跟来的魏将都看得皱眉。

“收一处驿，只问这个？”

燕将回头看他一眼：“主将要的是它知不知道，眼下齐城里催得急，改得却慢。”

魏将听完，没有再问。

因为他忽然明白，这场仗打到这里，联军收的已不只是道、驿、仓。

是在一处处验：齐里外之间，那口旧日传得又快又重的命气，如今到底还剩几分。

傍晚，北面那座中城城头也看见了旧驿上换了人。

守城司马当即要下令试射。

城中校吏却迟疑了一下，低声道：“驿还在传行，路上车也没断。若此时先射，附近几县只会更不敢走这一线。”

司马转头便骂：“不射，难道看着它们这样贴脸坐下？”

校吏不敢再辩。

可那一瞬的迟疑，却像针一样扎进了司马心里。

从前这等地方，齐一声令下，哪还轮得到校吏先替几县的车道算轻重。

如今却连城头的第一箭，都要先想：射了之后，底下还会不会更乱。

这一夜，乐毅照旧没有追加压城令。

他只让各线把这一日各处“迟疑”“改口”“先算轻重”的细处都记下来。

夜深之后，图上又多了三处极细的墨点。

一处在旧驿。

一处在中城外空道。

还有一处，在城头那支直到夜半也没有放出的第一箭上。
