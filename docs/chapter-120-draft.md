# 第一百二十回 乐毅借静势迫齐自分轻重 联军收空道却不急夺重城

又过两日，联军前线明明占了顺势，却还是没有大举前推。

这日午后，乐毅召几位主将到图前，只问了一句：

“北面那座中城，眼下最怕什么？”

魏节应将答：“它怕周边县道、转车、守卒、民役，这几日各有各的算法，最后没人替它把那一口死死供住。”

乐毅伸手在图上轻轻划了一道弧线。

那不是直逼中城的路，而是绕着中城外三处空道、两处小仓、一处旧驿，像慢慢合起一只手。

“中城先不压。”

“先收它四周能叫人自己想重想轻的地方。”

最先被收住的是那处旧驿。

联军去时，门里的人甚至没有立刻关门。他们只看见来者没有纵马冲进，也没有先砍驿旗，只是下马之后，先把驿前乱摆的两架破车推开，让路过的车能照常转出去。

驿长出来时，手还在抖。对面的燕将只问：“这两日从城里来的公文，哪一类最多？”

驿长怔了很久，才下意识回道：“催额的多，改额的少。”

燕将点点头，也不再问，只叫书记把“驿传照旧，行人勿惊”八个字写了出来，贴在驿门内。

傍晚，北面那座中城城头也看见了旧驿上换了人。守城司马当即要下令试射，城中校吏却迟疑了一下，低声道：“驿还在传行，路上车也没断。若此时先射，附近几县只会更不敢走这一线。”

司马转头便骂，可那一瞬的迟疑，却像针一样扎进了他心里。

因为这一下迟疑已经说明，城里眼下最难的，不再只是“要不要守”，而是“还想不想让周边县道、旧驿、转车路继续像旧日那样替中城供气”。若一箭射出去，眼前是痛快，后头却可能让更多本就发紧的县路更不敢往前来；若不射，城里便得眼看着联军这样一点点把中城四周那些原本最寻常、最该照旧运转的地方，慢慢收成另一套活法。
