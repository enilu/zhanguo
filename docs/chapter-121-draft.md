# 第一百二十一回 田文逼宫请发权变令 齐王半准半拒诸署因之更乱

北面诸县回文送上后的第二夜，司马、司仓两署终于一同入宫。

司仓先奏的是数：“近边六县，近五日所输额数，已各不同。”

司马接着奏的是令：“亭役、守卒、驿卒，近四日已有三处互借，皆未越县，却都未及待中朝断。”

直到田文上前，双手举起前夜那卷未发的急令：

“大王，这令该定了。”

田文再往前半步：“诸县如今不是不要齐令，是等不起全令。大王若还不替它们把‘可变’与‘不可变’分明写下，底下只会各自去猜。”

良久，齐湣王才一把取过那卷竹简，展开重看。看到后面那行留白时，他的眼神重新沉了下去。

殿外风声一阵紧过一阵。

齐湣王忽然提笔，在那卷未发简上重重添了几句：

“外县小仓可并，亭役可借，驿传可缓次。”

“然一切改动，止于原县，不得越境相援。”

“中道、中仓、主应三项，仍照旧额，少一分者问罪。”

田文心里先是一松，随即又是一沉。松的是，大王终于发令了。沉的是，这道令只认了一半病。

出殿时，司马、司仓两人对视了一眼，谁都没有说话。

因为他们也都看明白了。大王肯认“外县小仓可并，亭役可借，驿传可缓次”，说明临淄终于允许底下在局部活一点；可那句“止于原县，不得越境相援”又把刀收了回来。也就是说，齐王还是怕一旦让底下真学会自己互补，后头连中朝也未必还能完全按旧法压住。

田文捧着那卷新令下阶时，忽然听见远处驿马过门，蹄声极急。

他没有回头去看，只把竹简在掌心又按紧了一寸。令是发了，可前线和诸县接到之后，最先学会的未必是“可变”，更可能是“哪些地方仍旧不能真替旁边多担一步”。这病若只认一半，底下拿到手里，也只会先活成半截。
