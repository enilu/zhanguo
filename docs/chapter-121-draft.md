# 第一百二十一回 田文逼宫请发权变令 齐王半准半拒使诸署更乱

北面诸县回文送上后的第二夜，司马、司仓两署终于一同入宫。

这回不只是田文。

连先前一向最不愿开口认“需改旧制”的几位老臣，也都到了。

殿上灯火压得很低，每个人脸上的影子都比平日深。

司仓先奏的是数。

“近边六县，近五日所输额数，已各不同。”

“照旧者两县，减一成者一县，减三成者一县，其余两县虽未明报减额，实到数已短。”

司马接着奏的是令。

“亭役、守卒、驿卒，近四日已有三处互借，皆未越县，却都未及待中朝断。”

这些都不是刺耳的大祸。

却句句都在说同一件事：

齐那口从上到下压到底的旧令，已经压不成旧样了。

齐湣王坐在高处，听完前两人，脸色一直没有变。

直到田文上前，双手举起前夜那卷未发的急令。

“大王，这令该定了。”

齐湣王的目光落在那竹简上，久久不动。

他当然认得。

因为那卷简，是他自己写到天明，又自己卷起来的。

田文没有退，反而再往前半步。

“诸县如今不是不要齐令，是等不起全令。”他说，“大王若还不替它们把‘可变’与‘不可变’分明写下，底下只会各自去猜。猜得越久，齐法越碎。”

殿中一时静得很。

那几位老臣这回没有像往常一样先拦。

因为连他们也看出来了，事情已过了“只守旧制便能过去”的时候。

齐湣王终于开口，声音却冷：

“寡人若今日许了权变，天下会如何看齐？”

田文答得极直：“天下会看见齐在改病。可若大王不许，天下便会看见齐诸县已自己先改。”

这句话像一柄钝刀，慢慢压进殿中每个人耳里。

因为谁都知道，后一种更难看。

良久，齐湣王才一把取过那卷竹简，展开重看。

他看得极慢。

看到“外县并小仓者，事后报，不以擅论”时，眉头微松；

看到“亭役可暂借护转车，但不得越县擅调”时，手指又紧了一下；

而看到后面那行留白，他的眼神便重新沉了下去。

那一行，本该写的是给诸县一条真正能走的明路。

可也正是那一句，一旦写明，便等于齐王亲口认：齐如今得靠“先变半步”来续住全局。

殿外风声一阵紧过一阵。

齐湣王忽然提笔，在那卷未发简上重重添了几句：

“外县小仓可并，亭役可借，驿传可缓次。”

“然一切改动，止于原县，不得越境相援。”

“中道、中仓、主应三项，仍照旧额，少一分者问罪。”

这几句落完，田文心里先是一松，随即又是一沉。

松的是，大王终于发令了。

沉的是，这道令只认了一半病。

它给了诸县一点喘气之机，却又把“不得越境相援”“主应少一分者问罪”钉得极死。

换言之，齐王肯让各县自己先活半步，却仍不肯承认：

有些地方之所以先动，恰恰是因为一县之力已经未必够。

田文抬头，想再说一句。

齐湣王却已把简往下一放：“加印，今夜便发。”

这一声落下，殿中再无人敢拦。

印泥递上来时，那位老司仓的手比上回更抖。

印是终于落下了。

可殿里几位最懂边上实情的人，心里却都明白：

这令不是没用。

它甚至会先止住几分乱。

可它也会立刻叫另一层新乱长出来。

因为从今夜起，齐等于亲口告诉诸县：

你们可以在自己那一小块地上先变。

可若真要跨出去，把彼此连成一口再活，仍不准。

夜半，急骑持印出宫。

田文站在阶下，看着那一串火把往北门去，胸口却并未松开。

他忽然觉得，齐今夜这道令，像是终于给将裂的大网补上了几根线。

可补线的人，又亲手把网眼钉得更死。

往后若再吃重，那些先得了半口喘气的县，恐怕只会更快看清：

他们能自己先活一点。

却仍很难替旁边那一县一驿，一起多活半步。
