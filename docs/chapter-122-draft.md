# 第一百二十二回 新令一下诸县竞争近利 乐毅看齐网自补却更绷

齐王的权变令发出去后，北面诸县头两日果然都像松了半口气。

先前不敢明写的，如今敢写了；先前暗里借的，如今也肯补报了。

可到了第二批、第三批回报进来时，那口刚松开的气，便又一点点僵住了。

因为同样是照新令行事，各县先争的，却都是离自己最近、最能立刻活命的那一小块利。

有县一得可并小仓之令，先把邻近两处粮都收入本县主仓；

有驿得了“缓次”二字，先缓的正是替别县转送的公文；

还有一处更难看，甲县为护本县转车，照令借了亭役，乙县当夜便来讨人。

这件事报到田文案前时，午日才过。

田文看完之后，竟没有立刻发怒，只是把竹简轻轻放下，闭了闭眼。

午后，他召司马、司仓两署共议，只问了两句话：

“甲县争人，乙县争路，后面还会生出什么？”

“若都照令不算违，谁来替他们判先后？”

而联军这边，这一日也收到了更细的回报：

“齐北面诸县，得新令后并未齐整。”

“有争人者，有争仓者，有争驿次者。”

乐毅把图卷起一半：“齐这张网，如今是在自补。可补网的人，先补的都是离自己最近那一格。越是这样，整张网看着像还在，实则会越绷。”

韩老将听得皱起眉：“它既都在补，绷些难道不是更紧？”

乐毅摇头：“绷得匀才是紧。如今是各补各的近处。近处先活了，远处便更薄；一格先松了，旁边几格便都要跟着吃重。”

赵后线的记事官这时也把新报补了一句：“几县新并小仓之后，往中城那条大路上倒像更安静了些。”

赵奢听见，反倒点头：“安静未必是好。越安静，越说明它们已先把不该上来的东西都截在县里自己消化了。”

这话一落，帐中诸将便都懂了。齐北面眼下最危险的，不是再闹出一场人人看得见的翻脸，而是人人看着都像学会了照新令求活，整张网却在这样一处处自救里越拉越紧，等真要扛大重时，便不知会从哪一格先崩。
