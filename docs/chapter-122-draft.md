# 第一百二十二回 新令一下诸县先争近利 乐毅看齐网自补却更绷

齐王的权变令发出去后，北面诸县头两日果然都像松了半口气。

先前不敢明写的，如今敢写了；

先前暗里借的，如今也肯补报了。

北司马署清晨收进来的第一批回报，看上去甚至比前几日整齐许多。

“某县已并小仓三处。”

“某亭已照令暂借役卒八人。”

“某驿按新断缓次，不致断传。”

这些话摆在案上，连司马署里几位熬了数夜的老吏都难得松了松肩。

可到了第二批、第三批回报进来时，那口刚松开的气，便又一点点僵住了。

因为同样是照新令行事，各县先争的，却都是离自己最近、最能立刻活命的那一小块利。

有县一得可并小仓之令，先把邻近两处粮都收入本县主仓，旁边县道上一时便空了一截；

有驿得了“缓次”二字，先缓的正是替别县转送的公文；

还有一处更难看，甲县为护本县转车，照令借了亭役，乙县当夜便来讨人，说自家旧额未减，守卒却已被借得不够。

两县使吏在驿前当众争了半刻，连押车的卒都围着听。

这件事报到田文案前时，午日才过。

他看完之后，竟没有立刻发怒，只是把竹简轻轻放下，闭了闭眼。

他等的就是这种事。

也最怕这种事。

因为它说明新令并非全无用处。

它真能叫各地先喘。

可它也同时把另一层藏得更深的病，推到了明处：

当诸县终于被许“先顾自己”之后，它们也会立刻先顾自己。

而齐眼下偏偏最缺的，是有人能替这些各自先活的县、驿、亭，再把彼此的轻重重新串成一线。

午后，田文召司马、司仓两署共议，只问了两句话：

“甲县争人，乙县争路，后面还会生出什么？”

“若都照令不算违，谁来替他们判先后？”

殿下诸吏面面相觑。

因为这两句话，恰好问中了新令最要命的一处空。

齐王把“各县可权变”写下了。

却没有来得及，也可能是不肯，再往下写“数县相撞时，谁轻谁重由谁来断”。

于是各地一得了喘气，立刻便会拿这口喘气先救自己。

老司仓低头良久，才哑声道：“除非城中另设一署，专断交界诸事。”

司马署主簿也接了一句：“便是设，也要有快马快令，不然断文还没到，底下已吵完两轮了。”

田文没有说话。

他只是看着案上摊开的新令，忽然觉得那上面每一个“可”字，此刻都像带着倒刺。

它们原是为了救急。

可若后面没有更快、更细的一层新断跟上，这些“可”字便会叫底下无数人各拿一半，越抓越紧。

而联军这边，这一日也收到了更细的回报。

探骑报：

“齐北面诸县，得新令后并未齐整。”

“有争人者，有争仓者，有争驿次者。”

“甚至有县里主簿说：‘王上既许先变，我先保本县何错之有。’”

韩老将听得直摇头：“它总算发了令，怎的反又更杂？”

赵奢道：“发的是救一县一驿的令，不是统诸县诸驿的令。”

乐毅正在看一幅新送来的小图。

图上已不再只记空垒与道，还多了几处细小标记，都是这两日齐北面诸县争人、争仓、争驿次的地方。

他看了许久，才把图卷起一半。

“齐这张网，如今是在自补。”

“可补网的人，先补的都是离自己最近那一格。越是这样，整张网看着像还在，实则会越绷。”

韩老将问：“那我们眼下仍不压那座中城？”

乐毅摇头：“先不。”

“待它们自己先把‘本县可活’与‘诸县如何同活’这两层轻重打上一打。”

他说完，提笔在图上那几处争人争仓的地方各点了一下。

“探骑加一层。”

“谁先争赢，谁先争输，都记。”

“尤其看，哪一处争过之后，旁边县驿会不会立刻学。”

帐中诸将一时都静了。

因为他们都听出来，主将眼下等的，已不是一座城头先露空。

而是齐这道刚发出去的新令，究竟会先在哪些地方，教人把“自救”学得更快，把“共救”忘得更早。

夜里，北司马署又收进一封急报。

写的是：

“甲县争得亭役后，当夜先护本县车道。乙县见状，已拟明日自并邻亭。”

田文看完，许久没有动。

窗外细雨正落在檐角，一点一点，极轻。

可他心里却明白，齐北面这些县、亭、驿，从今夜起，怕是要一处看着一处学了。

而这种学，比前些日子学着先求缓、先并仓，更难收。
