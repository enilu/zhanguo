# 第一百二十三回 两县争卒当驿前翻脸 田文亲断次第已伤旧气

权变令发下后的第三日，北面终于闹出了第一场压不住的当面争执。

争的不是城，不是垒，也不是哪一路前军，争的是十二名亭卒。

甲县说这十二人前日已奉新令先借去护本县转车，眼下车道正紧，一人也不能放。乙县则拿出旧额文书，说自家守卒本就被抽过一轮，若再不给回四人，今夜北门便连轮值都凑不齐。

两边使吏在北门外那处旧驿前先是争纸，后是争声。驿长眼看压不住，只得飞骑入城。

田文赶到时，日头已偏西。

他没有先问谁对谁错，只叫人把两县这七日的输额、借卒、并仓、转车小札一齐取来，铺在驿案上。最后，他先点了甲县那几张：“你县前日先并两仓，又借十二卒护车，此举照令，不算错。”

田文又点向乙县那叠小札：“你县旧额未减，守卒又连抽两轮，今夜北门若真空了，也不算你虚报。”

可下一句，他的声音却更沉了：

“两边都不算错，所以今日最错的，不在你们。”

说完，他提笔便在驿案上写了三行断文：

“甲县留卒八人，放还四人。”

“乙县今夜北门先足轮值。”

“三日内，司马署另调邻亭四卒补甲县车道。”

断文写毕，两县使吏谁都没有立刻接。

不是因为看不懂，而是因为这一断虽然把眼前这十二名亭卒分出了次第，也等于把另一层更难听的话摆到了驿案上：从今往后，北面这些县不可能都照旧额、旧法、旧面子一起活，只能一件一件争出谁更该先活。

田文把断文推过去时，目光在两县使吏脸上各停了一停。

他知道，这一断未必能叫两边都服，却至少得让两边都记住：以后再争，不许只拿自己那张县簿来争。谁先、谁后，已不是各县自己能关起门定的事了。
