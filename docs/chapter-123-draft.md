# 第一百二十三回 两县争卒当驿前翻脸 田文亲断先后已伤旧气

权变令发下后的第三日，北面终于闹出了第一场压不住的当面争执。

争的不是城，不是垒，也不是哪一路前军。

争的是十二名亭卒。

甲县说这十二人前日已奉新令先借去护本县转车，眼下车道正紧，一人也不能放。

乙县则拿出旧额文书，说自家守卒本就被抽过一轮，若再不给回四人，今夜北门便连轮值都凑不齐。

两边使吏在北门外那处旧驿前先是争纸，后是争声。

一开始还都守着官样。

“此乃本县先借。”

“贵县先借，也不能借到我县门薄。”

“王上新令在前，不得越县相援，岂能说借便借？”

“既不得越县相援，那也不当拿我县守卒去护你县车。”

说到后面，连“贵县”“本县”都顾不得了。

一人指着对面鼻尖冷笑：“你们不过是见自家粮先少了，便想拿旁人来补。”

另一人立刻回骂：“你们若不是前日先把两处小仓都并进自己主仓，哪至于今日装出这副紧相！”

驿前本来只围着几个押车卒。

等这两句一出，路边卖水的、看车的、借宿的行人都停了脚。

因为这已不再是两名使吏在争纸面上那十二个人。

是在把这些日子每一县心里都不肯明说的那点私苦、私急、私算，当众翻了出来。

驿长眼看压不住，只得飞骑入城。

田文赶到时，日头已偏西。

那两边的人见他来了，总算都跪下。

可跪是跪了，眼里那股火却没灭。

田文没有先问谁对谁错。

他只叫人把两县这七日的输额、借卒、并仓、转车小札一齐取来，铺在驿案上。

风从破窗里灌进来，吹得那几卷小札边角一阵阵乱跳。

他低头一张张看过去，足足看了半盏茶。

众人都不敢出声。

最后，他先点了甲县那几张。

“你县前日先并两仓，又借十二卒护车，此举照令，不算错。”

甲县使吏刚松了半口气。

田文又点向乙县那叠小札。

“你县旧额未减，守卒又连抽两轮，今夜北门若真空了，也不算你虚报。”

乙县使吏也抬起了头。

可下一句，田文的声音却更沉了。

“两边都不算错，所以今日最错的，不在你们。”

这句话一出，驿前忽然静了。

连旁边围着看的几个行人，都不由把呼吸压低了。

田文抬眼，看着甲乙两县的人，一字一句道：

“错在城中这道令，只许你们各自先活，却还没替你们把撞上时谁先谁后写明白。”

驿前众人脸色都变了。

因为谁也没想到，孟尝君竟会当着两县使吏、押车卒和一堆行人的面，把这层最难堪的话直接说出来。

可他说完，也没有让这句话悬着。

他提笔便在驿案上写了三行断文：

“甲县留卒八人，放还四人。”

“乙县今夜北门先足轮值。”

“三日内，司马署另调邻亭四卒补甲县车道。”

写完之后，他把竹简往案上一压，抬头道：“今日先这样断。往后再撞，不许在驿前先翻脸，先报司马署。”

甲县那人嘴唇动了动，终究没敢多说。

乙县使吏也是一脸不甘，却也只得叩首领断。

事情似乎就此平了。

可田文心里很清楚，这种“平”，只是平下了今日十二名亭卒的去处。

并没有平掉底下那层越来越清楚的东西：

齐如今已不是靠一条旧令，便能叫诸县自己知轻知重。

有些先后，已经得由城中一处处、一件件重新替它们断。

而这种断，本身就说明旧气在伤。

傍晚，他回城时，没有坐车，只骑了一匹瘦马。

北门外路上，正好遇见两列相错而过的转车。

前一列往甲县去。

后一列往乙县去。

两边赶车人擦肩时，谁也没有说话。

只是都各自把车往自己这一边死死贴了贴，像生怕多让半寸，回去便要少活一口气。

田文骑在马上看着这一幕，胸口忽然一阵发沉。

他知道，从今往后，齐要补的，已不只是边线。

还得补这些同在一线、同为一国，却开始在一条路上彼此防着、彼此挤着走的人心。
