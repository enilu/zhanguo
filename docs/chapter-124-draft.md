# 第一百二十四回 乐毅坐看齐县互嫁重 苦守中城先失一条直路

齐北面两县在驿前翻脸的消息，第二日午前便递到了联军中帐。

乐毅没有立刻接话，只起身走到大图前，顺着北面那座中城外的几条线，一点点往两边看去。

中城还立着，城头旗也还整。可供它的那几条直路，这几日已因诸县各争轻重，被折成了许多弯。

乐毅终于开口：“中城暂不碰城门。先把它东南那条最直的转车路截成两段。”

命令一下，燕本部当日便动了。只是一队轻骑先沿林边并走，逼得那条直路上的前后两处小驿都不敢再明火传夜。魏中军随后又在那路中段一处浅桥外立了拒马，不多，却正好叫大车过时非得慢下半截。

入夜前，中城里果然先乱了一阵。

守城司马先是下令：“今夜直路火把不许灭，驿传照旧。”

可下令不到一刻，城下便回报：

“火把一亮，林边便有骑影。”

“大车过桥太慢，后头已堵了四列。”

司马脸色一变，最后还是咬着牙改了三道小令：

“前列先行。”

“后列分两道绕。”

“今夜驿火减半，不许照旧直传。”

中城这还是头一回，被逼得亲手把自己城外那条最直的路，掰弯了一截。

这条路一弯，城里的声音也跟着乱了一层。

守卒先怨驿火减半，转车的小吏又怨前列先行，后头等着过桥的车队则在夜里一列列堵开。若在平时，这些都只是城外一条路上的小烦小怨；可在眼下，中城上下人人都知道，这正是城里最不能失的那一条直路。连它都要让火、让桥、让车次，城里那股“虽困犹能照旧”的气，便也跟着弯了一分。

联军营中，乐毅听完回报后，只问了一句：“它自己改的，还是我们替它改的？”

回报之人答：“令皆由城中自下。”

乐毅点了点头，没有再追。

诸将却已听懂了。若是联军一把打断那条路，不过是外力能压；如今却是中城自己先把最直的一条线改弯，这才说明齐在乐毅连日不碰城门、专取县路的逼法之下，已经开始自己替自己改活法了。
