# 第一百二十四回 乐毅收看齐县互嫁重 苦守中城先失一条直路

齐北面两县在驿前翻脸的消息，第二日午前便递到了联军中帐。

来报的探骑说得很细，连那十二名亭卒最后如何分、驿前围了多少看客、田文怎样当众写下断文，都记得一清二楚。

韩老将听完，先是冷笑了一声。

“它们如今连十二个人都要当众争成这样？”

赵奢却没有笑。

他看着图，缓缓道：“争的不只是十二人。争的是各县都知自己先顾自己不算错，可一旦撞上，谁肯先让那半步。”

乐毅把那份探报折起，放到案角。

他没有立刻接话，只起身走到大图前，顺着北面那座中城外的几条线，一点点往两边看去。

中城还立着。

城头旗也还整。

可供它的那几条直路，这几日已因诸县各争轻重，被折成了许多弯。

有的道上车还走，却绕得更远；

有的驿还传，却先传本县后传邻县；

有的守卒人数未减，却夜夜先顾自己那道门。

这些东西单看一处都还不算塌。

可合在一起，中城那口原该最直、最快、最不容被分心的供气，便开始悄悄变得不直了。

乐毅终于开口：“中城暂不碰城门。”

“先把它东南那条最直的转车路截成两段。”

韩老将抬头：“那条路离城最近，若一动，城头必有反应。”

乐毅道：“要的便是它反应。”

“城中若还想照旧护那条直路，便得逼周边几县与数处驿舍先替它让；若它不护，那中城便会头一回知道，自己也有一条最直的气，开始不是想保就能保。”

命令一下，燕本部当日便动了。

却不是大股压上。

只是一队轻骑先沿林边并走，逼得那条直路上的前后两处小驿都不敢再明火传夜。

魏中军随后又在那路中段一处浅桥外立了拒马，不多，不足以死断，却正好叫大车过时非得慢下半截。

这半截，平日看不出什么。

放到如今中城外那口已开始不直的供气上，便像一根细刺，刚好扎在最烦人的地方。

入夜前，中城里果然先乱了一阵。

守城司马先是下令：“今夜直路火把不许灭，驿传照旧。”

可下令不到一刻，城下便回报：

“火把一亮，林边便有骑影。”

“大车过桥太慢，后头已堵了四列。”

司马脸色一变，立刻又要改成“前列先行，后列暂待”。

他话才出口，旁边一名校吏便低声提醒：“若后列今夜暂待，明日便要占邻县道。”

司马猛地转头瞪他。

可那校吏虽低了头，话却没有错。

如今北面这些道，已不是从前只凭一座中城一句话，便能叫邻县邻驿都先替它让。

司马在城楼上站了半晌，最后还是咬着牙改了三道小令：

“前列先行。”

“后列分两道绕。”

“今夜驿火减半，不许照旧直传。”

三道令发下去时，城楼上的人都沉默了。

因为每个人都知道，这不是大败。

甚至不算丢什么。

可中城这还是头一回，被逼得亲手把自己城外那条最直的路，掰弯了一截。

夜深之后，探骑把这一切一条条送回中帐。

韩老将听到“驿火减半，不许照旧直传”时，竟一时说不出话来。

他打了这么多年仗，知道一座城最要紧的，未必是墙有多高。

而是它通向外头的那几口气，还直不直，快不快，别人还肯不肯先替它让。

而今夜，齐这座中城，分明还未被联军真正围住。

却已先自己把最直那口气，折弯了。

乐毅听完，只在图上那条直路中段轻轻画了一笔。

那一笔不长。

像刀还没下去，只先在皮上留了一道浅浅的印。

可帐中诸将都明白，从今往后，这座中城再想照旧把那条直路立回来，怕不会容易了。
