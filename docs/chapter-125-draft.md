# 第一百二十五回 齐宫新设断事案仍嫌慢 三署同灯先争谁该先活

田文在驿前写下那道断文后，第三日，临淄宫中总算又添了一张新案。

案设在北司马署与北司仓之间的小偏厅里，不设大匾，只挂一块黑木牌，上书两个字：

“断事。”

按齐王口谕，今后凡北面诸县、诸驿、诸亭相撞之事，先由此案速断，不必层层再转。

这本是为了快。

可案子一设起来，头一日送到偏厅里的，便不是一两件。

而是整整二十三封。

有争卒的，有争仓的，有争车道先后的，还有两县同时来问同一处空亭究竟算谁辖。

小偏厅原本只放得下三张长案。

这一日却被竹简堆得连地上都快没有下脚处。

司马署来的人觉得该先断守卒与车道；

司仓署来的老吏却咬定先断仓与输额；

驿传那边的主簿拍着简说：“你们都慢。驿次若今日不断，明日所有断文都要晚半日。”

三边本都是来救急的。

谁也不是故意推诿。

可一旦坐进同一间小厅里，人人先看见的，便都是自己案上那一叠最急的事。

田文进来时，看见的正是这副样子。

三署同灯，灯下三种声音谁也压不住谁。

他没有立刻发怒，只走到中间那张长案前，伸手把最上面三封简抽了出来。

一封是“乙县北门今夜守卒仍短两人”；

一封是“丙县主仓已满，再并粮须先腾车”；

还有一封是“旧驿缓次后，军报与断文同道，先后难分”。

他把三封简排在一起，问厅中诸人：“你们说，哪一封先断？”

屋里一时静了。

因为无论先答哪一封，都像在承认另外两封可以往后再挨半日。

司马署的人先咬牙：“守卒不够，门今夜便空，先断这个。”

司仓署那位老吏立刻回道：“仓若满了不腾，明日车便无处落，后头三封守卒断文也白写。”

驿主簿更急：“你们断得再好，若驿次不清，文书不到，一样都成空话。”

这几句一起，厅里那股焦火又要起来。

田文抬手往案上一按。

不重。

却让所有声音都停了一瞬。

“你们都没错。”他说。

“可若日日都只说自己这一封先断，齐今日便是另设了十张断事案，也一样慢。”

说完，他提笔在空简上飞快写下四行：

“今后断事案先分三等。”

“涉今夜开闭者为一等。”

“涉明日输转者为二等。”

“涉文报缓急者为三等，然一等二等断文须配驿卒专递，不得与常报混。”

写到这里，他笔顿了一下，又添了一句：

“若同属一等，两署主事当面共断，不得各持回厅。”

这道小法一下，厅中诸人脸色都变了变。

因为它看着只是再添几行细则。

可这几行，实则是在逼三署承认：

从今往后，不能再只替自己那一摊急事争第一。

得先在同一盏灯下，把“谁该先活”当场断出来。

司仓老吏先是皱眉，后又慢慢点了点头。

驿主簿也低头看着那句“专递”，像终于抓住了一根能使力的绳。

只有司马署那名年轻主簿，仍忍不住低声道：“可若日日都有一等，三署岂不是日日都要坐在一起争？”

田文看了他一眼。

“是。”他说，“如今齐北面要活，先得学会在争里断先后。”

这话一出，偏厅里再没人接声。

外头天色已暗，宫人又送进来两盏新灯。

灯一添，竹简上的字便更清了。

可屋里众人心里却都明白，眼下这些字越清，便越说明齐已经过了只靠一纸总令便能撑住全局的时候。

夜半，第一批照新法重排的急简终于送出偏厅。

田文没有回宫，也没有回府。

他就在断事案边坐着，看一名名书记把“一等”“二等”“三等”的木签别上不同竹简。

别到后半夜时，一名年轻书记忽然手一滑，把一封“一等”简掉到了地上。

他忙俯身去捡，脸都白了。

田文却没有斥责。

他只是看着那竹简落地时滚出的半圈，心里忽然生出一种说不清的凉意。

因为他知道，齐如今能不能活，竟已要靠这些木签、灯盏、偏厅里谁先开口、谁先让一步，来一点点往回缝。
