# 第一百二十五回 齐宫新设断事案仍嫌慢 三署同灯争论谁该先活

田文在驿前写下那道断文后，第三日，临淄宫中总算又添了一张新案。

案设在北司马署与北司仓之间的小偏厅里，不设大匾，只挂一块黑木牌，上书两个字：

“断事。”

按齐王口谕，今后凡北面诸县、诸驿、诸亭相撞之事，先由此案速断，不必层层再转。

可案子一设起来，头一日送到偏厅里的，便不是一两件，而是整整二十三封。

司马署来的人觉得该先断守卒与车道；

司仓署来的老吏却咬定先断仓与输额；

驿传那边的主簿拍着简说：“你们都慢。驿次若今日不断，明日所有断文都要晚半日。”

田文进来时，看见的正是这副样子。

他没有立刻发怒，只走到中间那张长案前，伸手把最上面三封简抽了出来，问厅中诸人：“你们说，哪一封先断？”

屋里一时静了。

田文提笔在空简上飞快写下四行：

“今后断事案先分三等。”

“涉今夜开闭者为一等。”

“涉明日输转者为二等。”

“涉文报缓急者为三等，然一等二等断文须配驿卒专递，不得与常报混。”

写到这里，他又添了一句：

“若同属一等，两署主事当面共断，不得各持回厅。”

这道小法一下，厅中诸人脸色都变了变。因为它逼三署承认：从今往后，不能再只替自己那一摊急事争第一，得先在同一盏灯下，把“谁该先活”当场断出来。

偏厅里那晚的灯因此一直亮到很深。

司马署的人起初还想把守卒与车道都压成一等，司仓署则死守仓与输额不能后，驿传主簿更是一口咬定断文若慢半日，前头所有“一等”“二等”都要跟着乱。三边争着争着，终于都被田文按回那张长案前，一封一封照着新法重排。

这便是临淄眼下最难也最值钱的一步。

不是终于想出什么妙法，而是齐王、田文与三署都开始承认：旧法已经断不完新事，所以只能先在同一盏灯下，把旧日各署各守的一口气掰开来，重新排谁先、谁后。
