# 第一百二十六回 乐毅看齐先学争中求活 联军压住不战坐等城心弯

断事案设下后的第四日，齐北面的乱，果然没有更大地炸开。

至少表面上，没有。

驿前争吵少了；

县报上那些彼此推诿的字眼也收敛了些；

就连中城外那条被掰弯过一截的直路，这两日看上去也像又顺了一点。

若只看这些，营里已有将校私下道：

“齐到底是大国，真被它这么缝回来了，也未可知。”

这话传到韩老将耳里，他自己都没急着驳。

因为他这两日站高处看，也确实看见了一点“像又稳了”的样子。

可等他把探骑送来的细报一条条再看下去，眉头却慢慢又锁了起来。

中城外直路虽像顺了些，可过桥的大车多了两次换马；

断事案的急简虽送得快了，却是靠抽走常报驿卒硬挪出来的；

北面两县不再当驿前争卒，可当夜甲县便悄悄把邻亭原该南送的两车木先截去了半车。

这些都不响。

甚至都不像前几日那么难看。

可每一条底下，都压着同一股味：

齐如今不是不学会在乱中求活。

是正学着把原先摊在明面上的争，改成更细、更隐、更会彼此吞一点的活法。

韩老将拿着这几条细报进中帐时，心里竟先有些发沉。

他原想的是，齐若继续大乱，联军便更好下手。

可如今看着对面这副样子，他才忽然明白，最磨人的，也许不是它大乱。

而是它真能边乱边学，边学边把伤都往细处、暗处吞。

乐毅听完之后，却只问了一句：

“中城城头这两夜，换岗时还喊得像前些日子那样整么？”

探骑答：“不差。”

“只是更短。”

“短？”

“是。号声还齐，可每一段都收得比前几日快，像不愿多耗那口气。”

乐毅点了点头，没有再问。

他转身走到帐外，望了望北面。

暮色里，那座中城还立着。

远远看去，城上旗影、门下火色，都不算乱。

像一个人把衣冠又重新整了整，勉强还能见人。

可真正懂的人，往往正是从这种“还能见人”的地方，看出它把多少气先省进了骨里。

赵奢跟出来，低声道：“主将，齐是在学会把乱藏起来。”

乐毅道：“能藏，便说明它还没到死乱的时候。”

赵奢又问：“那我们仍旧不打？”

乐毅看着远处城影，片刻之后才道：“打，但不替它把这层新学会的活法打成英雄气。”

“它如今最怕的，不是我们看见它会缝。”

“是我们偏不去撕那缝口，只叫它日日这样缝、夜夜这样省、处处这样先吞邻边半口，最后连城里的人自己都慢慢习惯：凡事先想少耗一点，少让一点，少担一点。”

这番话一出，韩老将背后竟有些发凉。

因为他听得出来，主将眼里如今要等的，已不只是城外路弯、县里争卒。

而是等这整座中城、乃至更后头那层城中之心，也慢慢学会一种更紧、更小、更不敢先多出半口气的活法。

若真学成了，齐未必立刻塌。

可那股旧日大国该有的直气、重气、敢替外头先担的气，便会越收越短。

夜里，联军照旧只动两处。

一处是城东外那条浅桥，魏军又往前添了半列拒马。

另一处是旧驿北口，燕军把原本照常亮着的两盏大灯悄悄撤去一盏。

韩老将看见这一改，先是怔了一下，随后便懂了。

少一盏灯，路上行车便更慢一点；

可又不至于慢到逼齐当夜狠狠干一场来抢。

这仍是主将如今最拿手的那一刀：

不替对手把难堪一下子打成血勇。

只叫它在看似还撑得住的地方，日日多耗半分，夜夜多省半口。

更深夜时，探骑再回：

“中城今夜三更换岗，号仍整，只是第三声明显短了。”

乐毅听完，提笔在图上城旁写了两个极小的字：

“气短。”

写完之后，他把笔搁下，没有再添第二个判断。

帐中灯火静静照着那两个字。

谁都没有说话。

因为诸将都知道，主将今夜等到的，不是一处新空垒，也不是一条新断路。

而是齐这座城，终于在最不该先短的地方，先短了一口气。
