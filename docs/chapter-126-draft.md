# 第一百二十六回 乐毅看齐已学争中求活 联军压住不战坐等城心弯

断事案设下后的第四日，齐北面的乱，果然没有更大地炸开。

至少表面上，没有。

驿前争吵少了；

县报上那些彼此推诿的字眼也收敛了些；

就连中城外那条被掰弯过一截的直路，这两日看上去也像又顺了一点。

可等韩老将把探骑送来的细报一条条再看下去，眉头却慢慢又锁了起来。

中城外直路虽像顺了些，可过桥的大车多了两次换马；

断事案的急简虽送得快了，却是靠抽走常报驿卒硬挪出来的；

北面两县不再当驿前争卒，可当夜甲县便悄悄把邻亭原该南送的两车木先截去了半车。

乐毅听完之后，却只问了一句：

“中城城头这两夜，换岗时还喊得像前些日子那样整么？”

探骑答：“不差。只是更短。”

“短？”

“是。号声还齐，可每一段都收得比前几日快，像不愿多耗那口气。”

乐毅点了点头，没有再问。

夜里，联军照旧只动两处。

一处是城东外那条浅桥，魏军又往前添了半列拒马。

另一处是旧驿北口，燕军把原本照常亮着的两盏大灯悄悄撤去一盏。

更深夜时，探骑再回：

“中城今夜三更换岗，号仍整，只是第三声明显短了。”

乐毅听完，提笔在图上城旁写了两个极小的字：

“气短。”

写完之后，他把笔搁下，却没有立刻收图。

帐中诸将一时也没人说话。因为“气短”两个字写出来，事情便忽然从县、驿、桥、车、灯这些散处，收回到了更要紧的一处上。中城眼下还能守，城头号声也还整，可若连换岗时那一口原该挺直喊完的气都开始自己往回收，便说明城里上下已经在不自觉地学着省、学着短、学着把旧日那种“照足、守足、喊足”的做派一点点往里缩。

乐毅等的，正是这种缩。
