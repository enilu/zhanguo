# 第一百二十七回 中城主仓夜减一列灯 箭车到门前先听木轮涩响

第五夜，城西主仓照例开门验车。

仓门外原挂四盏大灯，这夜只亮了三盏。

看门老卒先没觉出异样，等第一列木车推到门前，才发现北边那半幅车辙照不清。押车人把缰绳一紧，后头两列车也跟着慢了下来。门里管验的仓吏提着灯出来，只说了一句：“今夜省半列照灯，后半夜再添。”

老卒望了他一眼，没问。

门外却已堵住。

东门送来的箭车本该先入东角，偏偏被两列木车斜斜压在后头。年轻校尉站在门边，手一直按着腰间木尺，听见押箭卒在身后低声催：“再晚些，晨更前就进不了东角了。”他没有应，只看着门里那三盏灯下的人影来回挪动。

一盏灯没亮，仓门前就短了半截亮处。

前头验一车，后头就得停一车。木轮碾过石边，发出一阵干涩的磨声。那声音不大，却一声接一声，把门前的人都磨得不敢催。

押箭卒又上前半步：“校尉，箭车若再压着，明晨谁担？”

校尉回头，嘴唇动了动，最后只吐出两个字：“少说。”

这两个字出口，押箭卒便退了。

副司仓赶到时，门前已堵出三层。主簿低声请他添回第四盏灯，副司仓立在门槛边看了一阵，抬手指向南道：“木车先分两辆，绕后。”

命令一下，赶车老卒就把缰绳往侧面拽。车头擦着石边过去，硬把原本一条直路掰出一个弯。箭车仍没能先入，只得继续贴着仓墙等。

天将亮时，最后一辆箭车才慢慢进了东角。

年轻校尉站了一夜，直到车尾过去，才低头松开拳头。两只手握得太久，指节全白了。他抬头看仓门上那盏始终没有点起的灯，又听见门外木轮最后一声涩响，什么话也没说。

仓门重新关上时，门缝里只漏出三道灯影。
