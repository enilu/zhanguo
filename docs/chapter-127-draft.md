# 第一百二十七回 中城夜点仓火先省半列 将校看短粮话到嘴边又吞

中城那口“气短”，先在仓门口露了出来。

第五日夜里，城西主仓照例要点灯验车。

往常这道程序最是分明。

东门来的粮车先靠北列，南道进来的木车靠南，验过一列，仓门前的灯便添一盏，叫后头的人知道哪一列可前、哪一列当停。

可这一夜，仓门上原该亮着的四盏大灯，只点了三盏。

看门的老卒起初以为是灯油未到，扯着嗓子问了一句。

仓吏从门里出来，低声回道：“不是未到。是司仓那边说，今夜先省半列照灯，后半夜再添。”

老卒怔了怔，忍不住往城上望了一眼。

城头火把也还整。

可一座主仓的灯，竟先要“省半列照灯”，这在从前，是他想都没想过的话。

第一拨车到门前时，果然慢了。

不是车不肯走。

是前头那点灯光只够看清半边轱辘，押车的人谁也不敢抢。

后头的木车头一停，南道又跟着梗住，连带着本该直入东仓角的两辆箭车都只得斜斜靠在旁边等。

守在门侧的年轻校尉看着这阵子一点点堵出来，脸色越来越硬。

他原本是来催箭车先入的。

可真到仓门口，话却迟迟没有说出口。

因为他看得出来，门里门外每一处都像已先把能省的、省得不至太难看的地方，都悄悄省过了。

若他此刻再硬开口要箭车先入，便等于叫别的车再往后挪半刻。

这半刻落在平日，不算什么。

落在如今，便像是逼另一个地方的人，再替你多咽一口。

校尉站着不动，身后那两名押箭卒却急了。

“校尉，再迟，明日晨更前入不了东角。”

“前头那几辆木车又不是要紧军需，凭什么压着路？”

校尉听见这句，终于回头瞪了他一眼。

他本想喝斥。

可看见那两人满脸疲色，嘴角刚动，终究只压低声音道：“少说。”

这一个“少说”，像是连他自己也一并按住了。

因为他心里最清楚，押箭卒这句并非全错。

可木车若真被喝退，后头仓中那些本已少了半列灯的人，今夜便更不好交差。

于是他只能站在仓门口，眼看着那两辆箭车迟了一刻又一刻。

半夜时，城西守仓的副司仓也到了。

他走到门前，只看了一眼，便知道堵在哪里。

旁边主簿低声道：“要不要先恢复第四盏灯？”

副司仓的眉头极轻地跳了一下。

恢复一盏灯，不难。

可那一盏灯一亮，便等于承认今晚这点“省半列”的省法，省到了不该省的地方。

他沉默片刻，只道：“不添。”

“把南道木车先分两辆绕后。”

这法子看着圆。

可一绕后，原本还算直的一条入仓路，又被弯出了一截。

两名赶木车的老卒听令时，谁也没多话，只闷着头把缰绳往侧边一扯。

车轮碾过石边，发出一阵发涩的声音。

那声音不大。

却叫守在旁边的年轻校尉心里猛地一沉。

因为他忽然听出来了，齐今夜这点看似只是仓门前的小改小省，底下压着的，其实和前几日城头那一声变短的号，是同一口气。

它们都不响。

可都在说明，这座城如今要撑住，越来越不是靠“大开大合”。

而是靠各处都先收一寸、先让半刻、先把想说的话吞回去。

天快亮时，那两辆箭车总算进了东角。

年轻校尉站在仓门边，看着最后一辆车轮慢慢碾过去，忽然发觉自己两只手握得发麻。

他低头一看，指节都已压得发白。

他其实什么也没做。

只是从头到尾，站在那儿看了一夜。

可也正因如此，他比谁都更清楚看见：

中城如今连一座主仓前的灯、车、道、话，都已经在学着先短半口气了。
