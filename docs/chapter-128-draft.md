# 第一百二十八回 田文持仓报入宫请分轻重 齐王看见缺灯仍嫌话难出口

次日午后，城西主仓夜报送到临淄。

夜报写得很平，只两行字：

“暂省照灯一列，改分车绕后，晨更前箭车已入东角。”

田文看完，把竹简平放在案上，指尖停在“省照灯一列”五字上，半晌没有移开。

他当日就入了宫。

齐湣王正在案前看北面折道图，见他进来，先问的是桥，不是仓。田文把那卷夜报展开，压在图边：“大王，中城主仓昨夜先少了一盏灯。”

齐湣王抬眼：“少了一盏，车还是进去了。”

“进去了。”田文道，“只是这盏灯若也能先省，下面以后见了桥、仓、路、门，都会想着先从最不难看的地方省起。”

齐湣王把折道图往前推了一寸：“不省，难道照旧点灯、照旧走车，等着一夜全堵死？”

殿里很静，连案角铜兽口里那一点烟都看得见。

田文没有绕，说得更直：“臣不是请大王不许省。臣是请大王先把话说清楚，哪些地方能省，哪些地方一步不能退。”

齐湣王听到这里，脸色就沉了。

他走下座，负手在案前来回踱了两步，停住时正好看见夜报上的那五个字。那五个字写得很小，却把话都逼到了眼前。

“再发一令，专说哪处可省、哪处不可省？”他冷冷问。

“要。”田文答。

齐湣王看着他，忽然笑了一下，笑意却半分也没有。

“寡人若真把灯、车、仓、门都一条条替他们划出来，天下便都知道，齐已到了连一盏灯该不该省，也要王上亲口替它说的地步。”

田文垂首，没有再接。

殿外北廊有人提灯油走过，脚步极快。齐湣王听见那一串急步声，转身回到案后，把夜报卷起，放到一旁。

“这道令，眼下不发。”他说。

田文拱手退下。

下长阶时，他正遇见那名提油的小内侍。油壶不大，小侍却抱得很紧，像怕漏了一滴。田文停了一停，让人过去，才继续往外走。

长廊尽头，风把灯影吹得一晃一晃。

他想起中城主仓昨夜没亮的那一盏灯，脚下没有停，袖中的夜报却一直没有松手。
