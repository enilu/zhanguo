# 第一百二十九回 北门主记改一字夜报平了 中城守将缓一列拒木先求不断

齐王没有再发细令。

北司马署里的人便照旧收简、抄简、汇简，只是案上的眼色一天比一天重。

第七日夜里，北门送来一条门报：

“二更末，西侧木车误占东道，堵一刻。”

主记小吏提笔抄录，写到“堵一刻”时，笔尖停了一停。外头值更的铜梆刚敲过两下，旁边抄录的同僚已把别的门报推了过来。他低头再看一遍原札，提笔把那四个字改成了“稍滞，已解”。

写完之后，他把原札压到最下头，晨报却先送了出去。

天亮时，北司马署收报的老吏果然没追问。

同一时刻，中城西外那段弯路上，守将也做了个小改。

那处本该再补两列拒木，木材、人手都到了。副将照旧请令，守将站在坡上看了看路口，又看了看后头正排着的补车，只说：“今夜先补一列，另一列明夜再说。”

副将一愣：“木都齐了。”

“车也齐了。”守将道，“两列齐补，今夜路先堵。”

副将听完没再争，挥手把人分开。原本要并立的两列拒木只先栽下了一列，剩下一列沿路平码在旁，像一排尚未合拢的牙。

午后，这两件细事前后到了联军帐中。

探骑把北门晨报和原札一并送来，又把中城夜补拒木的事记在后头。韩老将先看门报，瞥见“稍滞，已解”四字，皱起眉：“这字太圆。”

赵奢把原札抽出来，按到那卷晨报旁边，没说话。

乐毅坐在图前，听完探骑口述，只问了一句：“守将说先补一列时，旁边人怎么应？”

探骑答：“没有人驳。都照令搬木去了。”

乐毅点了点头，把两卷简放在图上中城之北，指腹轻轻压住，没有再翻。

帐里一时只听得见灯花轻爆。

韩老将顺着他的手看过去，也不再说了。

那一夜，北门主记抄完最后一卷夜报，抬头时天色已明。他把笔在砚边搁好，手指上还沾着半点墨。那点墨怎么擦都擦不净，只能一路带着，带到晨光里去。
