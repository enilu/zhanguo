# 第一百二十九回 中城将校先学少开一口 门下小吏已懂迟报半更

齐王不肯再发“分省令”的第七日，中城果然没有立刻出大错。

相反，它看上去甚至比前几日又整了一些。

仓门前堵车少了；

城头换岗的号虽短，却越来越齐；

连断事案送来的那些急简，也像渐渐摸出了先后。

若只看表面，便连城中几位老将都私下说：“这座城，倒真被它自己磨住了。”

可磨住的另一面，很快也露了出来。

最先露出来的，不是大将。

是门下一名主记小吏。

那人年纪不过二十余，平日最不出眼，只管抄录各门夜报，晨起再汇成简送上。

这一夜，北门口报来一条：

“二更末，西侧木车误占东道，堵一刻。”

若照旧例，这种事该立刻入夜报。

可那小吏提笔时，手却微微一顿。

他知道，这条一旦写进去，明晨城司马看了，多半又要追问“何以占道”“为何未早报”“值门何人”。

而眼下北门这几日好不容易才显得“平”些。

若因为这一刻堵道，又翻起一轮责问，最后只怕整夜值门的人都要跟着遭殃。

他这么一想，笔尖竟先挪开了半寸。

等到三更又有一条别的夜报送来，他才把那“堵一刻”轻轻并进后头一句，写成了：

“二三更间，门下车行稍滞，已解。”

这不是假报。

可已经不是原来的那条报法了。

到了天亮，这句“稍滞，已解”果然没有惹出什么追问。

门下那几名昨夜差点被责的小吏，甚至还都松了口气。

其中一个拍了拍那主记的肩，低声笑道：“你倒会写。”

那主记也勉强笑了笑。

可等他转过身去，心口却忽然一阵发空。

因为他很清楚，自己这一下“会写”，救的不是哪条大道，也不是哪处仓门。

救的是一夜清静。

而这口清静一旦变得比那一刻堵道本身更紧要，齐这座城里许多本该直着往上送的东西，怕就会开始一层层变得更圆、更软、更晚。

同一日午后，中城守将也做了一件看似不大的改动。

城西外那段弯路上，原该再补两列拒木。

木已到了，人也在。

可守将看了一眼天色，又看了看外头那条本就已被折过一回的直路，忽然摆手道：“今日先补一列，另一列明夜再说。”

旁边副将一愣：“为何？”

守将只答：“今夜若两列齐补，车又要再停。眼下不求更严，先求不断。”

副将听见这句，嘴唇动了动，终究没有再驳。

因为这句话，也不是没道理。

可他心里却有一处沉沉往下坠。

从前齐的将校遇事，先问的往往是“这样够不够严”“够不够整”“够不够像一座该守得住的大城”。

如今他们先问的，却越来越是：

“今夜会不会又停”“会不会再堵”“会不会惹出多一层乱。”

这不是怯。

甚至可以说，很实。

可这份实一旦越长越熟，齐那口原本该先向外、向前、向重处用的气，便会越来越先拿去保“不要再乱一点”。

入夜之后，探骑把这两件细事都报到了联军帐中。

韩老将听完，半晌才吐出一句：“城里的人，竟已学会替自己把话写圆了。”

赵奢道：“不只写圆。还学会把该直着上的东西，先绕半圈再上。”

乐毅看着图上城门那几处标记，没有说别的，只问：“那守将说‘今夜不求更严，先求不断’时，旁边的人是什么脸色？”

探骑想了想，道：“像都听懂了。也像都觉得，这话不难听了。”

帐中一下静了。

这才是最冷的地方。

若连“先求不断”都开始不难听，便说明中城里许多人的心，已在不知不觉间，把原本更高一层的求法往下放了。

乐毅抬手，在图上中城门下轻轻圈了一道极小的圆。

“它如今不只气短。”他说。

“连上来的话，也开始短了。”

说完，他没有再添一句。

只是把那几条细报都压在图旁，没有折去。

仿佛这种由主记小吏、守门副将、值夜校尉一点点学出来的“圆”“缓”“少担一点”，比一条新断路更值得他看。
