# 第十三回 六印既重苏秦日日补局 秦廷分条记下六国各自不同的散法

六印归身之后，苏秦几乎没再得过一日清闲。

今日在赵，要压一压强国自矜；明日到魏，要稳一稳近秦诸国的疑；后日又得遣人入燕，免得燕国因自己附势既早，反生出被边缘化之惧。齐楚虽不日日催迫，却也时时放话试他，看他究竟更重北方急势，还是更重远国体面。

他做的，已不是偶尔救火，而是天天补局。

有时清晨还在邯郸与赵臣议“近秦有动，当谁先报”，午后便要转去馆中见魏使，解释“共议应之”不是空话；到了夜里，还要在灯下改文书，把一句伤强国面子的字挪开，再把一句能安小国疑惧的话添重。

同是补缝，每一国裂法却都不同。

赵强，怕多担；

魏韩近秦，怕先死；

燕弱，怕自己又被自然挤到局边；

齐楚大，怕入局之后反失主位。

苏秦若拿压赵的锋辞去逼魏韩，近秦之国立刻便要生出“果然还是叫我先吃刀”的疑心；若拿劝燕的话去说齐，齐只会发笑。

所以他越来越像个医者，对什么病下什么药。

对赵说：“居中者若只要名，不肯稍担其责，众国嘴上敬你，心里却只等你失势。”

对魏韩说：“近秦之国若一味只求别人先替你挡，旁国看见的便不是你苦，而是你怯。”

对燕说：“附势既早，更要自己时时发声，不然久而久之，众人便真把你当作可有可无。”

对齐楚，则换另一副说法：“大国最大的失位，不是在盟中分了别人一点体面，而是在局真成之后，自己反成迟到者。”

这些话没有一句是重复的。

而在咸阳，秦廷也把这些不同裂法一条条记下了。

探报不再只是“某国有使，某国有议”，而开始按国分目：赵之裂，在强而自矜；魏韩之裂，在近祸先惧；燕之裂，在弱而求附；齐楚之裂，在大而观势。

一位年长近臣看完后，只低声道：“若日后真要破这盟，不必先向它最热处撞，只须找到每一种裂法最怕被碰的地方。”

灯下竹简一卷卷平码，谁都没有再多说一句。
