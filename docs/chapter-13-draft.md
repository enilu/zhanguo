# 第十三回 苏子持衡频调众口 秦人观衅暗记诸邦

自六印归身以来，苏秦几乎没有过真正清闲的一日。

今日在赵，要压一压强国自矜之气；明日到魏，要稳一稳近秦诸国之疑；后日又须遣人入燕，免得燕国因自己附势既早，反生出被边缘化之惧。齐楚两边虽不日日催迫，却也时时放话试他，看他究竟更重北方急势，还是更重远国体面。

第十三回里，苏秦的日子已不是偶尔救火，而是几乎天天都在补局。

他有时晨起还在邯郸与赵臣议“近秦有动，当谁先报”；午后便须转入馆中，见魏使解释“共议应之”不是空话；夜里又常常灯下改动文书，把一句容易伤强国面子的话稍稍挪开，再把一句能安小国疑惧的话悄悄添重。

同是补缝，每一国的裂法却都不同。

赵这边，问题在强而不愿多分主导，也不愿无端多担；魏韩的问题，则是明知最该附盟，却总怕自己先成试锋之地；燕国要的是“在局中”，生怕北方线虽起，自己终究仍只是跟随其后；齐楚则更高一层，它们不是怕不在局里，而是怕在局里却不占重心。

苏秦若对赵说韩的话，赵只会不服；若对齐说燕的话，齐只会轻笑；若拿压赵的锋辞去逼魏韩，近秦之国立刻便要生出“果然还是叫我先吃刀”的疑心。

所以第十三回里的苏秦，已越来越像一个医者，对什么病下什么药。

他对赵说：“居中者若只要名，不肯稍担其责，则众国嘴上敬你，心里却只等你失势。”对魏韩则说：“近秦之国若一味求别人先替自己挡，众国只会看出你怯，而不是看出你苦。”对燕说：“附势早者，反更要自己记得时时发声，不然久而久之，众国便真把你当作可有可无。”至于齐楚，他又换了一副说法：“大国最大的失位，不是在盟中分了别人一点体面，而是在局真成之后，自己反成迟到者。”

这些话没有一处是重复的。

而秦人在关中，也正在一点点把这些不同的裂法记下来。

咸阳宫中，近来送来的探报愈发细了。不再只是“某国有使某国有议”，而开始按国分条，记赵国所忧、魏韩所惧、燕国所盼、齐楚所算。秦廷主事之臣看着那一卷卷竹简，忽道：“从前只觉六国异心，如今看来，异得还不一样。”

旁边一人问：“何谓不一样？”

那人便一条条说道：“赵之裂，在强国自矜；魏韩之裂，在近祸先惧；燕之裂，在弱而求附；齐楚之裂，在大而观势。若只笼统说‘六国不齐’，还太粗了。如今当知道，它们各有各的散法。”

另一位老臣听完，轻轻点头：“记清这些，不是为今日立刻下手，而是为日后真要动它们时，知道该先拨哪一根线。”

这便是秦的可怕处。

东方此刻还在热中持局，靠苏秦的口舌、威望、分寸，一回回把将散之气重新压回去；西边秦廷却已不再满足于看“六国会不会散”，而是开始按类别去看：它们若散，会先从哪一种心思散起。

于是第十三回最耐看的地方，便成了一种冷与热的并行。

一边是苏秦日日奔走。今日才替赵魏把一条文书次序排停，明日又要因燕使一封措辞不平的书信，赶紧补一层安抚；午后才说服一位齐使不必急争一口面子，夜里回舍时却又看见韩人送来新简，言辞间仍满是“若秦先逼，谁为实援”的老问题。

另一边则是秦人在灯下记账。它们不急着笑东方，也不急着撞上去，只是把这张合从之网看成一块正在自己用力显出纹路的木板：哪处是虚的，哪处最吃力，哪处一敲便易裂，慢慢都显出来了。

苏秦当然不是白忙。

他每补一次，局便真的多稳一分；每压一轮，诸侯便真的多习惯一层“有事需回到合从之名下共议”。换了旁人，只怕这些裂缝早已让盟名散乱。正因有他在中间日日往返，强国与小国、近国与远国，才暂时还都肯在同一面旗下面坐下来。

可他越是这样补，秦人也就越能看清：这盟能借名成势，却仍远未成骨。

一夜，苏秦自魏地返赵，车过半道，天已微明。他在驿亭中短歇，随手把一路上收来的各国来简分列案前，竟忽然生出一种极细的寒意。

这些简牍表面说的都是盟中事务，背后却无一处不是旧心思。自己今日尚能凭威望与次序把它们缝在一起，可若有朝一日外压稍缓、利害轻重再换，眼前这些被缝住的边角，会不会立刻各自翻起来？

他看着灯下散开的文书，没有说话，只慢慢把它们重新收拢。

而同一夜的咸阳，秦廷也刚合上最后一份探报。

那位年长近臣低声道：“此盟可以借名成势，未必经久成骨。”

众人都明白，这一句并不是轻慢苏秦，而是看得更深。

因为真正可怕的，不是秦已经出手，而是秦已知道，日后若真要破这同盟，不必先向它最热处撞，只需找到它每一种裂法各自最怕被碰的地方。
