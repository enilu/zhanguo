# 第一百三十回 临淄连收三日好报先起疑 田文召原札入署见十七处磨平

又过三日，北司马署案上的晨报忽然整了许多。

“车道稍滞，已解。”

“仓门夜验无误。”

“角楼照旧。”

一卷接一卷，字都平，口气也平。偏厅里连熬数夜的老吏看到这些字，脸色都松了些，只有田文把第三卷晨报放下后，反把前两卷又抽了出来。

他让人去传北门主记。

小吏进署时，还算稳。田文没有喝问，只把三卷晨报摊开在案上，又叫人把各门原札一并取来。原札到得很快，竹片边上还带着门署夜露。

第一条就不同。

原札写“堵一刻”，晨报写“稍滞，已解”。

第二条，原札写“值门争语”，晨报只剩“门下已整”。

第三条最短，原札写“北角灯偏暗”，晨报改成“角楼照旧”。

田文不说话，旁边老吏一条条往下对。一直对到日偏西，案上已单独挑出了十七条。

北门主记跪在阶下，额上的汗一层层往下落。田文这时才问他：“这些字，谁教你的？”

小吏伏地答道：“无人教。臣只是怕细处太多，案上又起烦乱。”

田文看了他一眼，没有当场发作，只命人把十七条原札与晨报并起，封成一卷，送入宫中。

齐湣王入夜才看到这卷对照。

第一片竹简上写的就是“十七处”。他往后翻了几条，面色越来越冷，翻到“北角灯偏暗”那一条时，手背青筋都起了。

“它们如今连晨报也会自己修了。”他说。

田文站在阶下，道：“北面不是无事，是下面人先替案头把字磨平了。”

齐湣王把竹简合上，没有立刻下令拿人。

殿里很安静，安静得能听见风从窗缝里擦过去。隔了好一阵，他才把那卷对照简重新搁回案上，声音比先前更低：“往后晨报若太整，附原札一并送来。”

田文应了。

他退出去时，宫门外夜色已沉。送卷来的小黄门还候在廊下，怀里抱着一摞未拆的北面简，最上头那卷封泥刚裂开一角。田文经过时，目光在那道裂口上停了片刻，随后径直下阶。

风从北面吹来，把廊下一盏灯吹得细细摇了一下。
