# 第一百三十一回 临淄疑报之后先添复核 北门急简到案头已迟半更

临淄既见北面晨报被磨平十七处，北司马署第二日便立了一条新规。

凡中城与北面诸门送来的晨报，若字样过整、语意太圆，须附原札一并上送。

若原札未到，晨报先压半更，不得立刻入总案。

这规矩一出，署中众吏都知道，是冲着北门那名主记去的。

可真正先被咬住的，却不是那小吏。

是所有原本靠“晨报先入、总案先转”来赶那一点点时辰的人。

第三日清晨，北面中城送来一封急简。

简上只四字：

“东桥复堵。”

照旧例，这种字一到，北司马署先抄一份送司仓，再转断事案，半刻之内便能见人。

可这回，值案老吏先把竹简翻来覆去看了一遍，皱眉问：“原札呢？”

送简的小卒跪在阶下，额上还带着晨雾里的湿气。

“回老吏，原札在后。”

“后到多久？”

“约……半更。”

这一句一落，堂中几位正等着看北面晨简的主簿，都不由抬了头。

半更。

放在平日，不过是煎一壶水的工夫。

放到如今，便足够叫一封原本该立刻去司仓、去断事案、去催桥上清路的急简，先躺在案边发冷。

老吏没有作主。

他把那简压在案角，叫人记了时辰，只道：“待原札并到，再入总案。”

那送简小卒脸色一白，像想说什么，终究又咽了回去。

因为他也知道，这规矩是新立的。

谁若先开口说“急事便先别等原札”，便像是在拿前两日磨平晨报的事当耳旁风。

于是那封“东桥复堵”的急简，便真的在案边躺了半更。

等原札到了，里头写得比晨简更细：

“二更后换车两列，第三列木重，轱辘偏折，压了东桥半幅。守桥卒已分人拖解，然桥东车后又堆三列。”

司仓那边一看，立刻便知道，这不是只派两名小吏去桥上喝一声就能解的。

至少得先截后路，再把南侧两列木车改道。

可等这道断文拟好，天色已大亮。

中城那头的桥口，则已经自己先用老法子把后头三列车往北侧小道挤开了半截。

这一挤，当下是通了。

可北侧小道原本只供轻车过，此刻被大车硬碾过去，两处石边立刻塌了一角。

到中午，这一角塌路的消息，又得重新报回临淄。

断事案里的年轻主簿看着前后两封简，一时说不出话。

因为大家都看得清楚：

第一封“东桥复堵”，本不该拖成第二封“北侧石边塌角”。

可只因如今临淄先不敢全信北面来报，北面那边又不敢坐等临淄回断，于是最该快的那一步，便先慢了半更。

这半更，最后就变成了一处塌角。

田文下午进署时，案上正摆着这两封前后相咬的简。

他看完之后，什么都没说，只把新立的“附原札”那条规矩又看了一遍。

旁边老吏低声道：“孟尝君，若不立此规，北面还会继续把话磨圆。可规矩一立，急处又慢了。”

田文听见这句，胸口像被什么东西轻轻抵了一下。

他当然知道。

这正是现在最难的地方。

不复核，怕又看不见真相。

先复核，怕真相到了手里，事情已多走了半步。

而这半步，往往恰是最咬人的那半步。

傍晚，他把那两封简带进宫。

齐湣王看完，脸色很沉，第一句话便是：“所以如今寡人既不能全信它们的好报，也不能先用它们的急报？”

田文答：“不是不能用。是凡要用，便得先多问一句真假。”

齐湣王冷笑：“多问一句，桥便先塌半角。”

这话说得并不重。

可殿里几位老臣都听得背上一凉。

因为谁都听出来了，大王此刻骂的已不只是北面小吏磨字。

是在骂齐自己竟走到了这一步：

连最该立刻相信、立刻发令的急报，都得先隔着一道疑心，再看半更。

出宫时，天色将黑。

田文立在长阶下，忽然望见北廊那边一名小内侍正捧着两卷新到的北面简，脚下走得极快，却又不敢跑。

他看着那少年一步快、一步收，忽然觉得，齐眼下许多最要紧的事，也正像这双脚一样。

想快。

却已不敢真快了。
