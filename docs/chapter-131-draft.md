# 第一百三十一回 北司马署添规要附原札 东桥急简压案角先迟半更

北司马署次日就把新规立下了。

凡北面晨报，若字样太整，须附原札。原札未到，晨报先压案角，不入总案。

规矩写出来，悬在偏厅木柱上。来往小吏抬头都能看见，真正着急的却是送简的人。

第三日清晨，中城先送到一封急简，只有四个字：

“东桥复堵。”

值案老吏接了简，没有往司仓转，先问送简小卒：“原札呢？”

小卒还带着晨雾，跪在阶下答：“在后。”

“多久到？”

“半更。”

老吏听完，把那卷急简压到案角，另叫人记时辰。偏厅里等简的主簿都抬起头来，谁也没出声。那卷写着“东桥复堵”的竹简就静静躺在案边，竹片被晨光照得发白。

半更后，原札到了。

原札写得细得多：

“二更后换车两列，第三列木重，轱辘偏折，压东桥半幅。守桥卒分人拖解，桥东后车又堆三列。”

司仓看完，立刻知道不能只派两名小吏去桥上喝斥，便命人截后路，又让南侧两列木车改从北侧小道走。

桥是通了。

可到中午，第二封简又送回临淄：

“北侧石边塌角。”

原来重车碾上小道，把靠外那层石边压塌了半角。断事案把前后两封简并排摆在案上，年轻主簿盯着那两道封泥看了很久，才伸手把后一卷往前推了推。

田文入署时，正看见这一幕。

老吏把新规和前后两封简一并呈上。田文看完，没有责人，也没有改规，只把那张写着“原札未到，不入总案”的木牌重新扶正。

傍晚，他把两封简带进宫。

齐湣王翻过第一封“东桥复堵”，又翻第二封“北侧石边塌角”，眉间一直没有松开。末了，他把竹简按回案上：“所以如今急报到了，先不能信；等你们信了，桥边已塌。”

田文答道：“不验，看不见；一验，就迟半更。”

齐湣王没有再说话。

殿外有小黄门捧着新简疾步穿廊，跑到门边又硬生生收住，只敢快走，不敢真跑。田文听着那脚步由快转稳，直到人影过去，才低头告退。

长阶下，风把袖口吹得贴在手背上，凉得很。
