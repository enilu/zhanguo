# 第一百三十二回 乐毅见东桥急报先压案角 韩魏夜压桥外只添一层问

“东桥复堵，压案半更。”

这条消息当夜就送进联军中帐。

韩老将先看见探骑记下的时辰，忍不住用指节敲了一下案边：“急简到了不转，倒先在案上躺着？”

赵奢把图拉近了些，看着东桥和北侧小道那两处新添的墨点，没有接话。

乐毅问得更细：“桥后来怎么通的？”

探骑答：“中城自把南侧两列重车挪去北侧小道，桥通了，小道塌了半角。”

乐毅听完，只在图上东桥外轻轻一点：“今夜魏中军压桥外。”

韩老将抬头。

“不夺桥。”乐毅又道，“换骑勤些，让桥头不得松手。韩前军去北侧小道尽头，多立一列暗桩。”

韩老将听明白了，咧了咧嘴，却没笑出来：“还是叫它们自己先问。”

夜里，魏骑先到东桥外。

他们不擂鼓，只在桥外雾里一轮一轮地换马。桥头齐卒站了一夜，眼睛始终不敢离开雾中那几处若有若无的马影。桥东补车一来，守桥卒就回头；桥外蹄声一响，守桥卒又扭头去看雾里。

北侧小道那头，韩卒把暗桩一根根打进土里，动作不快，木锤却一下一下敲得很稳。值道校尉站在道口，看见那一列新桩，先把本要分过去的重车停住了。

副吏问：“还挤不挤？”

校尉抹了一把脸，只道：“先报中城。”

来报的人半夜叫醒守将。守将披衣出来，先问的不是小道能不能走，而是桥外魏骑压得多不多。听说对面未真逼近，他就把那卷请断的简搁回案上：“等天亮，再看回文。”

命令传回桥头时，天还没亮。

桥上换岗的齐卒踩着湿木板，一队一队从雾里走过去。桥下水声很闷，雾又厚，桥头每个人都把手里兵器握得很紧，却谁也没往前多走一步。

黎明前，探骑把桥外换骑、小道停车、中城回问这几条细报一并送回中帐。

乐毅看完，在东桥与中城之间添了一笔短线，线头微微往上折。

帐中诸将都看见了，也都没有问。

灯火照着那一笔折线，像一道刚刚抬头又先收回去的手势。
