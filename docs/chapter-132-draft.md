# 第一百三十二回 乐毅见齐最该快处先慢 韩魏夜压浅桥只逼再问

“东桥复堵，塌了半角”的消息，当夜便到了联军帐里。

韩老将看到探骑所记的时辰，先愣了一下。

“它临淄那头，竟叫一封急简先在案边躺了半更？”

探骑答：“正是。因新规要附原札，原札未至，晨简不入总案。”

韩老将一拍膝头：“这不就成了自己先拴自己手脚？”

赵奢没有立刻接这句。

他看着图上东桥与北侧小道那两处新添的墨点，片刻后才道：“不是它们不知道急。是如今它们先怕看错，再怕下错，最后便连最该先断的地方，也得再问一句。”

乐毅点头，目光却仍落在图上。

东桥半角虽塌，不算大。

北侧小道虽伤，也还未断。

可这件事里真正值得他看的，从来就不只是桥与道。

而是齐如今已开始在“急报能不能信”“信了会不会又被证明看错”之间，自己先生出了一道迟疑。

这道迟疑，比塌半角桥更值钱。

因为桥还能补。

迟疑一旦坐进上头人的手里、下头人的腿里，再想补回来，便难了。

这夜，乐毅只下了两道极轻的令。

“魏中军夜压东桥外，不夺桥，只叫它守桥卒不得轻松换手。”

“韩前军别动城门，只在北侧小道尽头多立一列暗桩，逼它明日若再改道，先得多问一层。”

韩老将听完，心里都微微一震。

主将要的，竟不是趁塌桥之时狠狠干它一下。

而是偏在它最该快的地方，再添一点“不得不问”“不得不想”的东西。

到了三更，魏军先到了东桥外。

他们没有擂鼓，也没有张旗。

只是在桥外雾里换了两轮轻骑，让桥上守卒始终不敢把那口提着的气放下。

守桥的齐卒一开始还咬牙顶着。

可到后半夜，桥下雾重，远处又时不时有马蹄擦石声传来，他们便不得不多添一轮值、更频地回头看桥东那几列补车有没有又挤上来。

这种夜，不像大仗。

却最耗人。

因为你明知对面未必真来，可偏又不敢当它不来。

而中城里，北侧小道那头也果然先起了第二层问。

值道校尉本想照昨日上午老法子，再把几列重车往那边挤。

可走到半道，看见道尽头新插起来的那一列暗桩，脚步便先顿了一顿。

副吏在旁低声道：“还走不走？”

校尉没有立刻答。

他心里其实已知道，多半还能走。

可问题不在“能不能走”。

而在如今只要那一头也稍有变数，这一挤，会不会又变成第二封、第三封难看的急报。

他最终还是没敢自己断，只说：“先报中城。”

这三个字一出，旁边副吏脸色就变了。

因为这又是一问。

而眼下北面最怕的，偏偏就是这些原本能当场断、如今却都要先往上问一层的小事。

中城守将半夜被叫醒时，听完这事，第一句便问：“对面压得紧么？”

来报的人道：“未真抢，只桥外不净。”

守将沉默了一会儿，才道：“那先别挤北侧。等天亮，再看临淄今日回文。”

这句话传回桥头时，天离亮还早。

守桥齐卒谁也没说什么。

可每个人心里都清楚，这又是一口气先被收了回来。

不是因为不能打。

而是因为如今一切只要稍有不准，后头便可能再多生一串追问。

黎明前，探骑把这一夜的桥外换骑、桥头回问、小道停挤一条条记下，快马送回中帐。

韩老将看完，良久才道：“它们如今真是怕一步走快了。”

乐毅没有否认。

他只在图上东桥与中城之间，添了一个极小的问号似的弯钩。

“怕快，未必立刻败。”他说，“可一座城若在最该快的地方，先养成了再问一层的手脚，后面许多能自己撑住的事，也会慢慢都变成要等。”

帐中灯火照着那一笔小小弯钩。

谁都知道，这一夜联军没夺下一桥一道。

可齐却又多学会了一件更要命的事：

在最急的时候，先把手收回来，再往上问。
