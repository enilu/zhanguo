# 第一百三十三回 齐王口谕桥仓路三事先断 中城偏厅新立牌影子却缠住

东桥再出事后，齐湣王终于下了口谕。

“桥道再堵者，守将先断。”

“仓门再滞者，副司仓先挪车次。”

“夜报若涉敌骑近桥，先行一更，后补原札。”

三条口谕一出，北司马署的气像松了半分。值案老吏把口谕抄成数份，当日就发往中城、桥头、司仓。

午后，两封简前后脚送回来了。

第一封是中城守将报来的：

“东桥午后再阻，已照口谕，先放南列重车改西。”

第二封是副司仓报来的：

“南列重车本备夜补西仓，今先改西，西仓车次遂缓。”

两封简一左一右摆在案上，纸面都干净，口气也都平。偏厅里几名老吏对着看了半晌，没有谁先伸手去拿第三卷空简。

田文入宫时，把这两封简一起带上。

齐湣王看完，先把守将那封放下，再去看副司仓那封，来回两遍，才冷冷道：“寡人既叫他们先断，断了又各来报苦。”

田文道：“桥先通了，仓就慢了。若不先排轻重，下面各自守各自那一头，今日救桥，明日就得补仓。”

齐湣王在殿中站了片刻，终于没有再往下细排，只让人传出口谕：

“桥、仓、路、门，凡先断者，当日并报所误何处。守将、副司仓、断事案同日互校，不得隔夜。”

口谕传到北面，中城偏厅里立刻添了一张长案。

案上立四块木牌：桥、仓、路、门。

每来一封“先断后并报”的简，主记就把对应木牌立到前头，再在后头添一小片木签，写上“所误”二字。守将那封一来，“桥”字牌前先站了起来；副司仓那封一摆，“仓”字牌又挨了过去。到了傍晚，长案上四块木牌已互相贴近，谁也没法离谁太远。

年轻主簿站在案后，看着那几块牌影在灯下挤在一起，忍不住伸手把“桥”扶正。桥牌刚立稳，边上的“仓”又斜过来，碰了它一下。

主簿手指停在半空，没有再动。

夜更深些，偏厅里只剩守牌的两名老吏。外头传来轮车过门的声音，一声比一声沉。老吏把灯芯掐短了一点，影子便随之压低，恰好把四块木牌全罩在一块影里。
