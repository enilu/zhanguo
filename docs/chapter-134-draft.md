# 第一百三十四回 乐毅不逼塌桥先逼失断 中城晨会第一次无人敢先开口

齐王“先断三事、当日互校”的口谕传出后，北面头两日竟真的像安稳了些。

东桥未再塌角。

中城晨报也不再拖到日上三竿。

偏厅里那几块“桥”“仓”“路”“门”的木牌，虽日日要摆，却毕竟还站着。

可到了第三日清晨，中城第一次在晨会上卡住了。

卡住的，不是因为敌军忽然大举压来。

也不是因为哪一处昨夜真出了大祸。

恰恰相反，昨夜送到案上的，是三件都不算大的事。

一件是东桥外雾重，魏骑换了三轮；

一件是西仓副门夜半短堵一刻；

还有一件，是北侧小道边有一辆重车陷轮，天亮前已拉起。

按新口谕，这三件都各可先断，也都要并报所误。

所以晨会一开，守将先看桥；

副司仓先看仓；

断事案主簿则盯着那辆陷轮车后头究竟误了哪条小道。

三人都拿着自己的简。

也都不是没道理。

可当城司马照例问出那句“今日先紧何处”时，屋里却忽然静了。

从前这种静，多半是一瞬。

总有人会先站出来，说城门、说桥、说仓，哪怕说得不全对，起码先给这座城一口向前用的气。

可今日这间屋里，人人手里都拿着简，人人心里也都先算到了：自己若先说，后头另一块若因此误了，今日这口“先断后互校”的账，便多半要先记到自己头上。

于是守将先看了副司仓一眼。

副司仓又去看断事主簿。

断事主簿握着简，喉结轻轻动了一下，竟也没有先开口。

窗外晨光一点点亮进来，照在三人案前的竹简边上，亮得极薄。

城司马的脸色，也在这片越来越亮的光里，一点点沉了下去。

他当然明白屋里这些人为什么都不抢那第一句。

不是他们都怯。

更不是都偷。

而是齐走到现在，每一处“先断”背后都像绑着另一处“所误”。

谁先说，便等于谁先认那后头可能跟上的一笔。

最终，还是城司马自己把案一拍。

“桥先紧。”

“仓按旧次后让一轮。”

“小道陷轮，今日午前必须补边。”

这三句一下，屋里众人才像一口气重新回来，纷纷应声。

可那一瞬先静下去、又靠城司马自己把气扳回来的空，终究还是留在了每个人心里。

因为人人都知道，这还是中城第一次在晨会开头，竟没有一个人敢先把今日第一句先说出来。

同日午后，这一条“晨会先静一刻”的细报便送到了联军中帐。

韩老将读到这里，指尖都在竹简边上顿了一下。

“它们终于连自己人坐在一屋里时，也得先看一圈别人脸色了。”

赵奢缓缓道：“先断太久，又日日要并报所误，最怕的便是这一步。不是没人懂。是都懂，也都怕先替别人扛后头那一笔。”

乐毅听完之后，没有立刻开口。

他只看着图上中城那处被圈过数回的地方，片刻之后，才提笔落下一道极轻的令：

“明夜，韩前军在城门外多起半段土烟。”

“魏中军不进，只在东桥外换骑更频。”

韩老将一听便懂了。

主将这回仍不是要狠狠干出一场攻门仗。

是要在它们已开始“无人敢先开口”的地方，再添一点更像今日先紧何处、明日又误何处的难题。

到了第二夜，城外土烟一起，中城果然先乱的不是城门。

而是晨会前半个时辰里，各署递进来的三份小札。

桥上说：“魏骑更频，今夜是否先加守？”

城门说：“烟起西南，是否先添门拒？”

仓司则问：“若两处皆先加，夜补木车还走不走？”

三札并至。

送札的人脚步都不算急。

可那三卷竹简一齐落到晨会案上时，屋里众人的眼神却都明显比昨日更沉了一寸。

因为他们知道，昨晨那一刻静，只是开头。

若今日还是人人都先看别人脸色，最后便不只是“静一刻”。

而是真会把该断的一步，拖成失断。

城司马望着那三卷简，手指在案边轻轻敲了两下。

这一回，他也没有立刻开口。

窗外天色尚灰，屋里每个人都听得见那两下极轻的敲木声。

一下一下，像在替整座中城算着：

它究竟还能靠这口越来越慢、越来越不敢先担的气，往前再撑几日。
