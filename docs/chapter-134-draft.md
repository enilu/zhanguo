# 第一百三十四回 中城晨会首度相看无人先言 乐毅令桥外换骑城门外添烟

“先断后互校”的第三日，中城晨会第一次慢了下来。

昨夜送到案上的事并不大。

一是东桥外雾重，魏骑换了三轮。

一是西仓副门短堵一刻。

一是北侧小道陷了一辆重车，天亮前已拖起。

三卷竹简摆在案上，各有各的去处。守将拿桥报，副司仓拿仓报，断事案主簿拿小道那卷。城司马坐在上首，照旧问：“今日先紧何处？”

屋里却静了。

守将先看副司仓，副司仓又看向主簿。主簿两只手都压着竹简，眼睛落在案边，不肯先抬起来。晨光从窗纸后透进来，一寸寸爬上众人案前，竹简边沿被照得发白。

隔了片刻，城司马才一掌拍在案上。

“桥先紧。”

“仓让一轮。”

“小道午前补边。”

话一出口，底下人立刻应声，誊札、取印、传令，晨会总算动了起来。可方才那一下空，已经留在屋里了。几名属吏出门时脚步都快，谁也没回头。

午后，这条“晨会先静”的细报送到了联军帐中。

韩老将看完，抬眼道：“它们一屋坐着，第一句话也要互相等了。”

赵奢道：“等久了，手就更慢。”

乐毅没有接话，只看着中城外那几处已被圈过数次的桥、仓、门、路。灯影在图上轻轻晃，他抬手点了两处：

“韩前军，明夜城门外起半段土烟。”

“魏中军，东桥外换骑再勤一轮。”

两道令都不重，传令校尉却记得极稳。

第二夜，城门西南先起了一条淡烟。风不大，烟也不高，只沿着地皮慢慢爬。东桥外的魏骑换得更勤，桥头齐卒才刚松了半口气，雾里又有新马影替上来。

天未亮，中城晨会案上又多了三卷小札。

一卷问桥外换骑该不该加守。

一卷问城门外起烟该不该添拒。

一卷问若两处都先动，今夜补木车还走不走。

城司马坐在上首，手指在案边轻轻敲了两下。

屋里的人全听见了。

窗外天色还是灰的，门外小吏抱着空简候着，不敢催。案上三卷竹简并排放着，封泥颜色各不相同，偏都没有人先伸手去拿。
