# 第一百三十五回 三札并至中城终失断 东桥先让门拒仓车俱慢半步

那一晨，城司马敲了两下案，终究还是没能像前一日那样把第一句立刻拍下来。

不是他不想断。

而是桥、门、仓三卷小札摊在案上，哪一卷都不像能先轻轻放过去。

桥上说魏骑更频，要不要先加守；

城门说西南土烟起，要不要先添门拒；

仓司则问，若两处都先加，夜补木车还走不走。

三问都不大。

可三问偏偏是同一口气里长出来的。

城司马看桥札时，心里先想到：桥若真被换骑压得守卒不稳，前几日塌角之事便可能再来一回。

他看门札时，又立刻想到：西南土烟若只是虚势倒罢，若真有一股试门，今日不先加拒，后头问罪便直落门下。

再看仓司那札，他更清楚：若桥、门都先加，夜补木车一停，今夜看着没事，明晨仓门与西道便都要跟着受。

这三层轻重在他心里绕了一圈，外头晨光便又亮了一寸。

守将终于先开口，声音却不大：

“桥可先加一轮守。”

副司仓立刻接道：“门若也加，夜补木车便要迟。”

断事案主簿下意识插了一句：“若门不加，土烟一起，值门今夜便不敢轻。”

三人一句接一句，谁也不是在争声势。

可也正因如此，屋里那股气反而越来越散。

因为人人都在补一句“那另一头怎么办”。

城司马听了几句，忽然道：“桥先加守，门先不动。仓车照旧，午后再看。”

这话一落，屋里竟没人立刻应。

不是不懂。

而是每个人都在脑子里先算这三句后面会咬到哪里。

最终，还是守将先拱手：“诺。”

这一声一出，其他人才跟着应下。

命令随即发出去。

东桥加了一轮守卒；

西南门拒暂不添；

夜补木车照旧从西道走。

若一切照城司马所想，这本也未必是坏断。

可偏偏午前风一起，西南那半段土烟竟比昨夜更久不散。

值门校尉站在门楼上，一面看烟，一面看城下远处并不逼近的韩军影子，心里那口昨夜便未尽消的疑，越压越重。

他不敢擅加门拒，只得先遣人往上再问。

问札刚出门，西道那头又传来第二条报：

“夜补木车照旧走西，今早回车与入车对撞，西仓副门前已停四列。”

这一下，桥、门、仓三处原本被强拧成一口的气，立刻开始各自打架。

守将听见西道对撞，先想的是若仓车堵死，桥上新加那轮守卒今日晚些时候也未必供得上。

副司仓听见门上又来问是否添拒，脸都沉了。

因为这等于他一早那句“门若也加，仓车便迟”还没到午时，就已先咬了回来。

而最要命的，是此时谁也不敢说自己早上那句就一定对。

于是第二轮问，又来了。

“西仓副门先清？”

“西南门拒是否补？”

“东桥那轮加守要不要先抽回半列，改去压车？”

三问再到案上时，城司马脸上终于第一次露出了一点压不住的烦躁。

他知道，这不是因为下面的人笨。

而是今晨那一下没断爽，后头每一处便都开始找补。

找补越多，原先那一下失断便越明显。

正午刚过，西仓副门那四列车终于因一辆重木车转弯不及，狠狠干翻了半排拒木。

车没翻。

人也没死。

可那一声木断之响，几乎半城都听见了。

门楼上的值门校尉听见这声，脸色一下发白。

因为就在这之前不久，他刚又把“是否添拒”那道问札往上催了一回。

而那札，还在路上。

到这一刻，城司马终于再拍案：

“西南门立添半拒。”

“西仓副门先断回车，入车北绕。”

“东桥加守不抽，午后桥东补卒改从北门借。”

这几句终究还是断下来了。

可它们落地时，整座中城的人都知道，今日最该在天亮前断妥的那一步，终究还是失了。

不是失在不会。

而是失在人人都先看见了后头那一笔，于是第一句便慢了。

午后，这一日的细报送到联军帐中。

韩老将看到“西仓副门拒木断响半城闻”时，呼吸都沉了一下。

赵奢则只看着时辰，从晨会到第一道再问、再到第二轮找补，一笔一笔往下数。

数到最后，他低声道：“这不是桥失，也不是门失。是中城第一次把一个本能天亮前扛过去的晨头，拖成了全日补漏。”

乐毅没有说话。

他只在图上西仓副门与西南门之间，缓缓连了一道细线。

那线不像路。

更像一处本不该先被连起来的疼处，终于在今日，被这一城自己慢慢扯成了一处。
