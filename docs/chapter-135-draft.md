# 第一百三十五回 三札并到晨会终难快断 西仓午后拒木断时半城皆闻

第二日天未亮，中城晨会案上又摆了三卷小札。

一卷来自东桥，说魏骑换得更勤，要不要先加守。

一卷来自西门，说夜里土烟未散，要不要先添门拒。

一卷来自西仓，说若桥、门都先动，夜补木车今晨还走不走。

城司马坐在上首，先把三卷都看了一遍，才抬头。屋里的人也都抬头，却没人先开口。守将知道桥要紧，副司仓知道木车不能停，断事主簿知道门外土烟不能当看不见，偏偏三卷都压在同一个晨头。

晨光又亮了一点，城司马才道：“桥先加一轮守。门先不动。仓车照旧，午后再看。”

守将先应了。

副司仓也只得应。

命令传下去，桥上添了一轮守卒，西门没加门拒，西仓木车照原路进出。若只看这一时，也不算错。

偏偏午前风转，西南那股土烟比昨夜久得多。值门校尉站在门楼上，连看了三回壕外，终于差人再问。问札刚出门，西道那边又传来急报，说回车与入车对撞，西仓副门前已停了四列木车。

一时桥、门、仓三处都来要人。

守将要保桥，副司仓要通西仓，门楼上还催着问是否添拒。第二轮小札送回晨会案上时，城司马脸色已沉得很难看。正午前后，西仓副门终于出事，一辆重木车转弯不及，车尾把半排拒木狠狠干断。

那一声脆响从副门里撞出来，传得极远。

门楼上的值门校尉先听见了，脸色刷地白了。副司仓站在断木边，脚下全是新裂的木刺，开口第一句却不是骂人，而是朝旁边主簿道：“记时辰。”

主簿低头就记。

城司马这时才把新令一口气拍下来：“西南门立添半拒。西仓先断回车，入车北绕。东桥加守不抽，桥东缺卒从北门借。”

三道令一发，城里人跑的跑，搬的搬，半日才把局扳住。桥还在，门也还在，副门那四列车终究挪开了。只是等一切都稳下来，已过正午。

黄昏前，这一日的细报送到联军帐中。

韩老将翻到“西仓副门拒木断半排”那一行，眉头立刻拧住。赵奢没去看结语，只顺着时辰往下数，从晨会到再问，到对撞，到断木，一笔一笔都落得清清楚楚。

乐毅把那卷细报压在图上西仓与西门之间，没有说话。

帐中灯火晃了一下，恰好照见图上那处新压出来的折痕。
