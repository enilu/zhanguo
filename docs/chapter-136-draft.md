# 第一百三十六回 田文指午后补漏请定总重 齐王看着时辰仍不肯先写

西仓副门断木的急简到临淄时，天色还未黑。

简上只列了三段时辰：

“晨断桥先，门缓，仓照旧。”

“午前西道对撞。”

“正午副门断木，今已北绕，并补西南半拒。”

田文看完，立刻入宫。

齐湣王刚拆开那封简，还没往下看完，田文已开口：“大王，该定总重了。”

齐湣王抬头，看了他一眼，又把目光落回简上：“桥、门、仓、路，你又要寡人替它们排一遍？”

“不是细排。”田文道，“是把最重那一层先写出来。今日这半日，不是在守，是在补。”

他说着，伸手点在简上三处时辰之间。齐湣王顺着那只手看过去，沉默了好一阵，才问：“寡人若写了，下面往后遇别的局，就只会照条走，不会自己看势，怎么办？”

田文没有退：“不写，今日这种半日补漏，往后只会更熟。大王总得替北面定一句，真撞在一处时，哪一头宁可后补，也不能先缓。”

殿里没有一个人敢出声。

齐湣王站起身，在案前来回走了两步，最后停在窗边。窗外天色将暗，宫墙外不知谁家在修栅，隔着风送来一声一声木响，极像午后那排拒木断开的声气。

他听了一阵，回身时脸色还是沉的。

“先不写。”他说。

“传北面一句：今后晨会不许再静。城司马以下，若再拖成半日补漏，一并问。”

田文拱手领命，袖中的竹简边角却被捏得发紧。

他知道齐王看见了，也明白了，却仍旧不肯把那一句先落在纸上。

出宫时，天已经黑透。廊下有人抱着新送来的北面简快步过去，怀里的竹卷互相轻轻碰撞，发出细碎的响。田文站在长阶下听了一会儿，才把手里那封午后急简重新收入袖中。
