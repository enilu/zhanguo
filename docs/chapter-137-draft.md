# 第一百三十七回 乐毅见齐已熟半日补漏法 联军轻探西门只收回门迟应

午后那一城补漏的消息传来之后，联军帐里难得静了好一阵。

韩老将先前还以为，中城这口慢气总要再积几日，才会真正咬出一处大的失手。

谁知不过一晨，便已把桥、门、仓三处扯成了一整日的补漏。

他忍不住道：“它们如今倒真学会了另一种活法。”

赵奢接道：“不是不会补。是越来越像只会补。”

这句话一落，帐中众人都没出声。

因为他们都知道，“会补”原本不是坏事。

可一座大城若把最要紧的半日、整日都耗在“哪处先补、哪处后补”上，那口原该向外去顶、向前去压的气，便会越来越少。

乐毅看着图，许久之后，才在西南门外一点轻轻敲了敲。

“明夜不动桥。”

“只在西门外做一件小事。”

韩老将抬眼：“如何做？”

乐毅道：“韩前军拨十骑，入夜后在西门外旧壕边试探一回。不要逼近，不要撞拒。只要叫门上知道，今夜若它再把门后那口气先拿去补别处，外头有人看得见。”

这令一出，韩老将便懂了。

主将这回仍不是要取西门。

而是要逼中城再做一次取舍：

既然今日午后它把西门拒补上了，那今夜若门外忽又有一点小动静，它是会先拿出一口真气来守门，还是又要想着“别急，先看一看，莫再多生一处补漏”。

入夜后，十骑果然到了西门旧壕边。

他们没有擂鼓，只趁着夜风起时，在壕外策马走了一个来回。

西门门楼上的值门校尉听见下头那阵短短的铁响，心里立刻一紧。

他白日里刚因“门缓”之事被城司马点过一次名，今夜本该比平日更敢先断。

可他手刚搭到传令木上，脑中却又先闪过午前那一整串因“门先不动”而惹出的后补乱象。

若今夜外头只是试一试，他先把整座西门都喊醒，再把门后两列守卒、一列补木全拽到门边，明日若桥或仓再起问，他便又得解释，自己何以先耗了这口门气。

这一犹疑，便是数息。

数息之后，他终究还是先低声道：“先照火，不敲大梆。”

门楼上火盆立刻多亮了两个。

可本该一声传下去的那面木梆，却没有立刻响。

门下两列守卒看见火亮，个个都先紧了半身。

可梆声不下，他们也不敢自己动。

直到壕外那十骑已绕了半圈，值门校尉才终于咬牙：“小梆一通，莫惊全门。”

这一通小梆落下去时，夜已经深了。

门后仓里原本替别处备着的一列补木车，这才被临时抽来半列，横在西门内侧。

整件事到头来并未出大祸。

外头十骑也只是探了一回便退。

可值门校尉站在门楼上，看着那半列临时抽来的补木车，额上竟全是冷汗。

因为他知道，自己今夜还是慢了。

不是慢在不懂门要紧。

而是慢在下意识里先替白日那一城补漏想了一遍。

他先想到“别再多生一处乱”，才想到“此刻门上该不该先响大梆”。

这份先后，才最叫他心里发虚。

黎明前，这一条“西门迟应半通梆”的细报便送到了联军中帐。

韩老将看完，长长吐出一口气。

“它们如今连门上第一声该敲多大，也要先想会不会拖别处后腿了。”

赵奢道：“这就是补漏法学熟了。先算后头会不会又咬回来，再断眼前。”

乐毅没有多说，只在图上西门内侧那半列临时抽来的补木车处，点了一点。

“它已不只是桥、仓、路相咬。”

“门，也开始学会迟应了。”

帐中众将都明白，这一夜联军只试了西门一回。

可这一回试出来的，不是西门有多弱。

而是齐如今凡遇一点外来之逼，先在心里算的，已越来越不是“当下何者最重”。

而是“若先应这一处，别处会不会又要我去补”。
