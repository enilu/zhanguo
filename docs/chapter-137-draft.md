# 第一百三十七回 韩骑夜探西门只走旧壕 值门见火先亮梆声却慢下来

午后那场补漏传到联军帐中后，乐毅隔了一整夜都没有动桥。

第二夜，他只给韩前军一条小令：“十骑入西门旧壕外走一回，不撞拒，不逼门。”

韩老将领令时问了一句：“只走一回？”

乐毅点头：“叫它看见。”

入夜后，十骑沿旧壕边策马过去，蹄声不急，只在夜风里撞出几串短促铁响。西门门楼上的值门校尉一听见声，就把手搭到了传令木上。

他白日里刚因“门缓”被点过名，照理今夜该先敲大梆。

可手搭上去之后，他先想到的是午前那一整串桥、仓、门互相咬住的问札。若这一声梆敲大了，门后守卒、补木、车次都得跟着动；若外头只是试探，明日案上又要多一封谁先抽了哪一列的简。

门楼上先亮了火。

两只火盆一齐提上墙垛，底下守卒看见火亮，人人都先握紧兵器，偏偏等不来梆声。壕外十骑又从风里转了一回，校尉才终于咬牙下令：“小梆一通。”

梆声落下去时，门后才匆匆抽来半列补木车，横在内侧。

壕外十骑已转头退了。

整夜下来，西门并无大险。值门校尉下楼时，后背却全湿了。他从半列补木车旁经过，车辕还热，木头上全是新抹的泥。他伸手按了一下，没说话，继续往里走。

黎明前，门楼夜报先到了中城，又在午前送进联军帐中。

韩老将看见“先照火，后下小梆”那一行，冷笑了一声，笑过之后又把简摊平了。赵奢问探骑：“门后抽的是哪一列车？”

探骑答：“原备西仓的补木车。”

乐毅听完，把图上西门内侧轻轻点了一点，指头离开时，墨痕旁边多了一小粒黑点。

帐外风不大，灯却被吹得连晃了两下。
