# 第一百三十八回 田文看见门报也学迟应 齐王对图久立终问桥门孰先

西门夜报次日一早就摆上了临淄案头。

“照火先，梆后半通，未致外扰。”

字不多，口气也平。田文看完，却把竹简扣在案上，立刻入宫。

齐湣王这几日被北面桥、仓、门、路一连串简报磨得火气暗长，见他进来，先问：“又是哪处？”

“门上。”田文把简呈过去。

齐湣王看完那一行字，竟笑了一下，笑得殿里人都不敢抬头：“桥要问，仓要补，门如今连梆声也只敢敲半通。”

田文没有顺着这句骂下去，只道：“大王，门上不是不知门重。它是先怕动了这一处，后头又多三处要补。”

齐湣王把那卷简扔回案上，起身走到地图前。北面桥、门、仓、路几处，都已被近几日的朱笔点过一遍又一遍。地图摊在灯下，边角都起了卷。

他站在图前看了很久，忽然问：“若真要写，桥门谁在前？”

殿里一时更静。

田文抬头，看着那张图：“桥与门都不可再慢。”

齐湣王没有立刻答，手指只在图上桥门两处之间来回移了两次，末了停在桥边，停了一会儿，又移到门上。风从侧窗进来，把地图吹得轻轻一翘，正好把两处朱点都掀亮了些。

“今夜再议。”他说。

田文领命退出。

下阶时，天边已近黄昏。宫门外一辆送简小车刚停下，车上竹卷堆得很高，最上头那卷封泥被风吹得轻轻转了一下。田文站在阶下看了片刻，抬步便走，没有再回头。

同夜，西门迟梆的细报也进了联军帐。

韩老将看完，道：“门上也开始怕了。”

乐毅没有看他，只看着案上的图。桥与门之间那条旧线已经画过数次，今夜他没再添线，只把两处朱点中间空出来的那一截，压了一条更深的折痕。
