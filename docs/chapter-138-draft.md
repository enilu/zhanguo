# 第一百三十八回 齐宫连看桥门皆慢半拍 田文知北面已从失断走向失胆

西门那一通迟下的小梆，第二日一早便摆上了临淄案头。

报得极省，只一句：

“西门夜照火先，梆后半通，未致外扰。”

若按从前，这样的夜报，最多也只算值门校尉谨慎过头。

可放到眼下，田文看着“照火先，梆后半通”这六个字，脸色却一点点冷了下去。

因为他看得出来，这已不是单一门上一名校尉的胆小。

而是这几日桥失、仓补、晨会先静之后，北面许多人心里都开始同样先想一件事：

若我先重打一处，后头会不会又多出三处要补。

这种念头一旦先于“眼下何者最重”站到前头，齐北面的病，便不只是失断。

而是在慢慢往另一层更深的地方走。

田文捧着那封夜报，入宫时脚步都比平日更急。

齐湣王这几日也已被北面一连串“先静”“再问”“补漏”“迟应”磨得心火暗长。

见他进来，先问的便是：“又是哪一处慢了？”

田文把简呈上，道：“门上。”

齐湣王看完，忽地笑了一声。

那笑里没有半分轻松，反而冷得让殿中人都不敢抬头。

“桥要问。”

“仓要补。”

“门如今连梆都只敢敲半通。”

“寡人的北面，何时竟活成了这副样子？”

田文没有立刻答。

因为他知道，大王这句问的已不是值门校尉，不是城司马，甚至不只是北面那座中城。

而是在问齐自己，何时从那个敢先压、敢先担、敢让外头诸县先照着自己来活的大国，走到了今天这种凡事先想“别再多生一处乱”的地步。

良久，他才低声道：“大王，北面如今最怕的，已不是一时失断。”

“而是失断多了，人人都开始怕先断。”

“怕先断久了，后面便会失胆。”

这最后两个字一出，殿中空气都像轻轻一紧。

因为“失胆”二字，比前些日子说“气短”“言路弯”更重。

前二者还像是在说病象。

这一句，已近乎是在说骨里那层最不该先软的东西。

齐湣王猛地抬头，眼神一时厉得几乎逼人。

“孟尝君，你是在说寡人北面守将，已开始失胆？”

田文迎着那目光，没有退。

“臣是在说，这口活法若再往下走，失胆便只是早晚。”

“今夜门上之人并非不知门重。可他先想到的是，若梆一响、全门尽动，明日桥、仓、路那边会不会又多一串补漏。”

“这不是单个校尉怯。这是整座中城，已开始拿‘少惹新乱’压过‘先断当前最重’。”

齐湣王听完，竟一时没有再驳。

他站起身，在殿中走到地图前。

北面那几处近来频频上简的桥、门、仓、路，如今看在他眼里，已不再是一张旧日随手可拨、可断、可令诸县照行的图。

而像一张一处处都被人暗里先留了半分余地、先省了半口气、先想着别再多生新乱的网。

这张网还在。

却已不再那样硬，也不再那样直。

他看了许久，忽然问：“你要寡人如何？”

田文答：“至少得让北面先知道，有一处，一步也不可再慢。”

齐湣王回头：“哪一处？”

田文沉声道：“门与桥。”

“仓可补，路可绕，报可核。可门与桥若也日日先想后补，北面守人的心，便真要先被磨没。”

这一次，齐湣王没有立刻摇头。

可他也没有点头。

殿中只剩风从侧窗灌进来的声音，吹得地图角轻轻起伏。

许久之后，他才道：“此事，今夜再议。”

田文听见这句，心里却并未轻松。

因为他知道，大王肯再议，已比前几日多走了一步。

可这一步，离真正把“有一处一步也不可再慢”写下去，仍还有一层最难割开的脸面。

同夜，联军帐中也收到了那条“西门梆后半通”的细报。

韩老将看完，低声道：“主将，它们门上也开始怕了。”

乐毅没有立刻接。

他看着图上桥、门两处这几日连起来的细线，片刻之后，才道：“怕还不够。”

“得等它上头也看见，这不是一夜之误，是整个北面已在学一种先别担重的活法。”

赵奢听完，没有作声。

因为他知道，主将如今等的，已不只是再试哪一门、哪一桥。

而是在等临淄自己把那句最难说出口的话，慢慢逼到案头上来。

灯火下，乐毅伸手把图上桥与门之间那条细线又加深了一分。

这一次，那线已不像先前那样几乎看不见。

它像一根被日日磨出来的细裂，终于开始在图上显出真正的形来。
