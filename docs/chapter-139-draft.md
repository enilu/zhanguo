# 第一百三十九回 齐王终于划门桥为死线 乐毅知其最难一句已出口

西门半通梆、东桥再三失断之后，临淄终于在第三夜把那句最难写的话落了下来。

齐王亲批北面急令，只两条：

“门与桥，自今以后，遇警先断，不得再以后补为先辞。”

“仓与路可缓半刻，不得使门桥再问第二次。”

这两句一出，北司马署里几位老吏竟半晌都没有说话。

他们等这句话，已等了许久。

可等它真的落下来，心里却并不轻。

因为谁都明白，这不只是两条新令。

这是齐王终于亲口承认：北面那口旧日不言自明、人人本该知道桥门最重的国气，已经不够了。

如今要靠王命再替他们把最重那一层写出来。

田文看着那两行字，先是长长出了一口气，随即又慢慢把气收了回去。

他知道，这一步非走不可。

可也知道，这一步一走，北面纵然能少一些再问与迟疑，齐自己那层最不肯认的难看，也就再遮不住了。

命令连夜出宫，天未亮便到中城。

城司马接简时，手指在“门与桥”四字上压得极紧。

昨夜西门那一通迟梆还在耳边。

如今王命亲下，像是终于替这座城把最沉那口气从乱网里拽了出来。

他当即召守将、副司仓、断事主簿入厅，只说了一句：

“今后先断，不必再互看脸色。桥门在先。”

这句说得不高。

可屋里几人都听出了另一层意思。

从今往后，若桥门再慢，便不只是失断。

而是违了王命。

这一层压下来，城中那股日日先算“后头会不会又多一处补漏”的气，竟真的被硬压住了半分。

可联军帐里，乐毅听见探骑报来“齐王亲划门桥为死线”，却没有半点惊意。

他只把图上桥门之间那条细线又看了一遍，淡淡道：

“它最难的一句，终于自己说出来了。”

韩老将先是一怔，随即明白过来。

齐若还不肯写这句，说明它还想靠旧气扛。

如今既已写下，便等于自己承认：旧气已经扛不住，北面要靠一道明令，才能把最重那层重新扶正。

赵奢站在旁边，低声道：“那我们要不要立刻逼桥门？”

乐毅摇头：“不必替它证明这句是对的。”

“它如今最怕的，不是桥门再慢。”

“是桥门既被王命扶正，仓路便更要自己去吞那半刻，那半刻吞得久了，整座城会更清楚看见：自己如今能守住门桥，靠的是别处日日往后让。”

说完，他只下了两道更轻的令：

“东桥外骑影照旧，不再加频。”

“西门外土烟今夜全止。”

韩老将听完，竟有片刻说不出话。

主将打到这一步，已不只是看对手哪里会慢。

是在等对手终于明白什么最重之后，再看它用什么代价把这层最重托起来。

同日午后，中城果然显出不同。

东桥再有小堵，守将一句话便断：“桥先净，仓后让。”

西门夜里再闻壕外马声，值门校尉也不再先照火后想，而是先下大梆，再回身调车补拒。

桥与门都快了。

可西仓那头的副司仓站在仓门边，看着今日又被后压了一轮的木车，眼神却越来越沉。

因为他知道，从今往后，只要桥门一有动静，仓路便都要先替它们让。

这不是说不该让。

而是这座城终于要日日正面看见：

自己能守住最重那两处，靠的是别的地方一次次先吞下去。

傍晚，田文在临淄案头收到北面回报：

“桥门今日皆快。”

“仓路后让，然尚可撑。”

他看完之后，把那“尚可撑”三个字看了许久。

终于还是缓缓把简合上。

他知道，齐今日是把最要紧的一句说出来了。

可真正的代价，才刚刚开始往纸上显影。
