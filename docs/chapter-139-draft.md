# 第一百三十九回 齐王亲批桥门自此居先 中城得令后梆声先下车次后移

第三夜，临淄终于把那道最难写的令发下来了。

急令只两条：

“门与桥，遇警先断，不得再以后补为辞。”

“仓与路可缓半刻，不得使门桥再问第二次。”

北司马署收令时，几名老吏看着那两行字，谁也没先开口。最后还是值案老吏亲自把急令拆成数份，连夜送往中城、东桥、西门。

天未亮，城司马已把人召进偏厅。

守将、副司仓、断事主簿都到了，城司马把王命往案上一拍，只说：“今后桥门在先。”

声音不高，厅里却立刻静了。

这话一落，往后桥门若再慢，就不是晨会一时没断好，而是违了王命。守将先拱手应是，值门校尉领到抄令时，两手把竹简攥得极紧，几乎把边角都压弯了。

这一日，东桥果然先快了。

桥头再堵时，守将一句“桥先净”，后头的人便各自让开。西门入夜再闻壕外马声，值门校尉先下大梆，门后补木车立刻顶上来，连火盆都比梆声晚了一步。

桥门都稳住了。

副司仓却站在西仓门口，看着今日又后压了一轮的木车，没有说话。年轻主簿来报车次，他只问：“记了没有？”

主簿说记了。

“那就接着记。”

同日午后，北面回报送回临淄：

“桥门皆快。”

“仓路后让，尚可撑。”

田文拆到这里，手指在“尚可撑”三字上停住，停了许久，才把简往下翻。后面附的，是西仓后压车次、桥东折道迟车数。数字不大，却一笔一笔都在往后挪。

夜里，探骑把同样的消息送进联军帐。

赵奢看完，只看后附那几笔车次，没有先看桥门。韩老将问：“它既扶正了桥门，我们今夜还试不试？”

乐毅摇头：“让它自己先撑。”

说完，他把图上桥边那枚旧钉拔下，挪到西仓外，又把门边那枚挪到桥东折道，两个小木钉在图上落定时，都发出极轻的一声。
