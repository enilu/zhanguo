# 第十四回 盟书立下之后先露旧缝 苏秦夜记一行字知疑只能暂压

盟书在案，并不等于万事就顺。

条文既定，一到真正往下行的时候，先前藏在礼数和雄谈底下的那些难处，反而一件件露得更明。

今日魏使来问，有警共议究竟议到哪一步才算守约；明日韩人又逼问，近秦边报一起，究竟是先由邻国会商，还是立刻四方告急；齐国来简说得最圆，只问一句“既然众国同盟，大国体面何在”。

这些都不是空问。

赵、齐、楚这类强国，怕的是主导权被悄悄稀释；魏、韩、燕这类更弱或更近祸的国家，怕的是盟一成立，自己反倒被推到最前。

一日，几国代表同坐，席间竟无人高声。

齐使道：“齐自不避共制西秦之义，只是凡议定诸事，总该有个不失大国体面的安排。”

韩使便回：“小国体面原也不值什么，只是小国若总先担险，谁还敢真信同盟之名？”

赵使又道：“赵不辞其责，但不可把赵之能战，看成理所当然多担。”

燕使最后才添一句：“燕愿附义，可也不愿盟中只有强国相议，弱国只在外头等传话。”

人人都说得很平。

也正因太平，才更难。

夜里，苏秦回到馆舍，独坐灯下，把这几份文书都摊开，许久才提笔写下两句：

“盟可立，心难一。”

“非盟之罪，乃人之常情。”

写完之后，他没有立刻收卷，只让那张竹简在灯下多放了一会儿。

火光落在字上，一明一暗，像极了这张刚立起来的盟旗底下那些尚未死透的旧心思。
