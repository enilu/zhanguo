# 第十四回 名盟初立人心未一 共势虽成旧隙犹深

盟书在案，并不等于万事就顺。

第十四回一开头，最清楚的便是这一层。条文既定之后，一到真正往下执行，先前藏在礼数与雄谈底下的那些难处，反而一件件露得更明。

譬如守约。

纸上写着“有警共议”，可共议到哪一步才算守约？近秦之国一有边报，是立刻遣使四方，还是先由邻近数国会商？又譬如先后。谁先表明共守之意，谁又先担与秦对峙的实险？再譬如分责。强国若兵多地厚，是否便自然该多担；弱国若国小势窄，是否便永远可以以“自保不易”为由少尽其责？

这些问题，前几回也提过，可到了第十四回，已经不再只是议席上的口舌，而是制度一落纸、执行一落地就立刻变薄的地方。

有的国在文书里附和得极快，真正要遣使、备粮、定次序时却忽然拖延；有的国嘴上说不争名，私下却总要在礼仪和称序里添减一笔，不肯真把自己摆到与别国齐平的位置上。还有些国家，最善于“表面尊盟，手上留缝”：明着不逆，暗里却总给自己多留一条退路。

同盟并非虚假，只是人人都在按自己利益去重写它。

这便是第十四回真正要清账的地方。

一日，苏秦连着看了三份不同国家送来的文书。赵国来简，说近秦应变之议，不能总由赵先出例；魏使附文，则再问若秦试边时，强国援应是否应有更明白承诺；齐国那边措辞最圆，只说“盟既成，当求众国各安其位”，看似平和，实则仍在强调大国不愿轻受拘束。

他把三份简牍并排放在案上，忽然笑了一下。

不是得意的笑，而是看明白之后那种带冷意的笑。

原来六国真正难解的，并不是“为什么要合从”，而是“合从之后，每国都仍想按过去的旧账去活”。强国没忘自己曾居上位，小国没忘自己常作人盾，远国没忘自己仍可观望几分，近国更没忘自己每一步都离秦锋更近。

旧隙并没有因一面合从之旗而消失，只是暂时被旗面盖住了。

于是第十四回里的苏秦，已不再奢望尽去众疑。

若是前三回、前十回的苏秦，还会想着凭一番更大的说辞，把众人都压到“看见大势”的高处；到了这一回，他却更像一个早知木板本就带缝的人，不再幻想把缝磨没，而只是想办法让它不要在此时断开。

赵臣来时，他便不再只说“赵当居中”，而提醒对方“居中之国最要紧的是让众国仍愿意围着这中枢转，而不是只记得自己先倡之功”；魏韩来问时，他也不再许什么尽善之约，只反复把“近国若退，秦更知可逼；强国若不应，盟名即薄”这一层死死按回去。至于齐楚那边，他更明白，不能指望大国真心放下旧算，只能让它们知道：旧算可以留，盟名却不能先拆。

这些处理，听来不壮，却比壮语更难。

因为它承认了不完美，也承认了“持局”很多时候并不是把人心讲成一致，而是在不一致里仍维持一个还算能走的秩序。

有一晚，邯郸春雨细下，馆外檐声滴答。那位赵国年轻议者来见苏秦，看着案上堆成数摞的诸国文书，不由道：“如今我算明白了，印能压住礼，果然压不尽这些心里的旧数。”

苏秦正在改一份措辞，头也不抬，只道：“若真能压尽，倒不是战国了。”

对方苦笑：“那你还日日替他们修什么？”

苏秦笔下一顿，方道：“旧隙本就不可能因今日之盟尽除。可若连盖住它们的旗都不要，裂缝便不是裂缝，而是直接把天下分回旧局。如今能做的，不过是在旧隙仍深时，先让它们别迎风裂开。”

这几句话，把第十四回最深的一层说尽了。

合从不是洗去六国旧怨旧算，而只是让它们在更大的外压下，暂时认同一面共同对秦的旗。旗越高，旗下一道道未消失的纹路，反倒越看得清。

而这一回收束时，局面并未破。

众国仍在其名下行事，文书仍来往，盟中仍有秩序，秦也仍在外面。可读者若从高处看去，已经能看清：这联盟不牢，不是因为某一个人忽然背盟，而是因为它从生下来起，便是把六家各不相同的旧心思，暂时压到了一处。

未破之局，至此已经可以看见它内部每一道将来可裂的纹路。
