# 第一百四十回 桥门既快仓路日日后吞 乐毅不争门只等一城自见价重

齐王那道“门桥为死线”的急令发下去后，北面头两日果然稳了许多。

东桥不再复堵。

西门也再无半通梆之事。

中城晨会一开，守将与值门校尉都敢先说话了。

若只看表面，像是那条最难的边线终于被扶正了一寸。

可扶正一寸，别处便得往后退半寸。

第三日清晨，西仓副门外又排起了七列木车。

不是因车多。

而是昨夜西门一闻壕外马蹄，门上依王命先抽走了两列替补车木。

到天亮，这两列木还未全回，西仓外头便只能眼看着后车一层层顶上来。

副司仓立在门口，没有发火，也没有催骂。

他只是拿着小札，一笔笔记今晨哪一列车本该何时入，现又迟了多久。

旁边年轻主簿看了一眼那札，低声道：“依如今这般让法，西仓怕是日日都得后压。”

副司仓没有抬头，只道：“记下来。”

这句平得像水。

可那年轻主簿却听得心里发闷。

因为他知道，“记下来”三个字，已不是在记偶然之误。

是在记这座城如今为了让桥门先快，到底要拿多少仓路日子去填。

同样的事，桥东也在发生。

东桥这两日确实净了。

可桥东那条原本用来缓解仓车回转的小折道，却因连着两日被让给守桥卒与补桥木，渐渐压得连轻车都难并行。

到午时，一名押粮小吏蹲在道边看着前头堵死的两列空车，忽然低声骂了一句。

骂完之后，他自己都愣了。

因为他骂的不是联军，不是秦军。

骂的是这条本该为桥让路的折道，竟如今日日都像在替桥还债。

这句话没传远。

却还是被旁边老卒听见了。

那老卒只抬头看了他一眼，半晌才道：“少说。桥总归得先活。”

小吏没有再接。

可他心里明白，老卒说得对。

也正因如此，这句对才更压人。

因为它逼所有人承认：

你如今堵着、等着、后压着，不是因为谁乱断、谁失职。

而是因为桥门确实比你更重。

而联军这边，这两日当真也没有多动桥门。

韩老将站在高处看了一日，见东桥比前几日整，西门也比前几日稳，终于忍不住问：

“主将，就这么看着它把桥门扶起来？”

乐毅道：“扶得起来，便更要看它每日拿什么去换。”

赵奢接道：“桥门是被王命硬扶正的，仓路若日日替它们后吞，城里便迟早会有人先算这笔账。”

乐毅没有说话，只叫探骑改记新项：

“今日起，不只看桥门快不快。”

“还看西仓外平均后压几列，桥东折道日日迟多少车次。”

韩老将听到这里，心里忽然一动。

主将眼下要的，竟已不是一处破绽。

而是一张更清楚的账。

一张齐自己把桥门扶正之后，仓、路、车、木究竟日日往后吞了多少的账。

夜里，中城副司仓把白日那几页小札送入偏厅。

断事案老吏一行行看过去，眉头越皱越紧。

“西仓后压七列。”

“东折道迟轻车十一乘。”

“桥木补次先于仓木回次。”

这些字都不惊人。

也没有前些日子那样叫人听着发硬的断响与急梆。

可老吏把这几页札子并在一起时，手却有些沉。

因为他忽然明白，齐如今北面真正新长出来的那层难，已不是“哪里总在慢”。

而是“哪里必须先快”，这件事终于定死之后，别处便开始一日日替这份先快付账。

更深夜时，乐毅在图上桥门旁又添了两笔。

一笔落在西仓外。

一笔落在桥东折道。

韩老将站在旁边看着，忽然觉得，这两笔比先前画在桥门上的线，还更像刀。

因为桥门如今被扶得越正，城里别处那层“日日后吞”的疼，便越会自己显出来。
