# 第一百四十回 桥门扶正之后仓路渐吞 副司仓日日记账不言只看车次

齐王急令发下后的第三日，桥与门都稳住了。

东桥不再反复堵死，西门也再没有半通梆。晨会一开，守将和门上校尉都敢先说话，中城表面上比前几日整齐得多。

可西仓外头的车却一日多过一日。

这一晨，副司仓出门时，外头已排了七列木车。不是车骤然多了，而是昨夜门上一闻壕外马蹄，先抽走了两列替补车木；桥头又先挪了桥木回次，等到天亮，这两列空缺还没补回，西仓门前便只能往后压。

年轻主簿抱着车次簿跟在旁边，低声道：“再这么让，西仓日日都得后压。”

副司仓没有抬头，只伸手把最前头那卷车次簿翻开，在“迟”字旁又添了一笔。

桥东折道也一样。

原本那条小道是给仓车折返用的，这几日却先让给守桥卒与补桥木。午前，一名押粮小吏蹲在道边，看着前头十一乘轻车挤作一团，忍不住骂了一句。旁边老卒回头看他，只说：“桥总得先活。”

小吏没再出声，抬手把车辕往旁边又挪了半尺。

这些细事当天都入了偏厅账册。

“西仓后压七列。”

“桥东折道迟轻车十一乘。”

“桥木补次先于仓木回次。”

断事案老吏把账册一页页翻过去，越翻手越沉。纸上没有断木，没有大梆，也没有哪一处忽然告急，可每一笔都像把别处那口气往后又推了一寸。

同夜，联军帐里新添了一本簿子。

赵奢亲自把西仓车次、桥东迟车、门上抽木一项项记进去。韩老将站在旁边看了一会儿，终于明白主将这两日为何不再逼桥门。

乐毅坐在图前，只吩咐探骑：“桥门照旧看，仓路另记账。”

帐外更鼓响过，书记把新簿合上，放到旧图旁边。灯火照着两本簿子的边角，一新一旧，平码在一处。
