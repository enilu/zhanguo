# 第一百四十一回 齐边扶住桥门却失仓路 乐毅乘暮收北面重城外郭

第四日黄昏，中城外终于丢的不是桥，也不是门。

丢的是北面重城最不体面的那一层外郭。

事情起得极小。

西仓外那七列后压木车里，有一列因连着两日未能入仓，只得先借停在外郭西角一处空坪。

那空坪原本并不打紧。

可偏偏这几日门桥皆重，守外郭的那一队老卒又被抽去两轮护门。

外郭看似还在，却已只剩一层薄薄的巡守。

傍晚时分，燕军一小股不过百余人，顺着外坪边那条废沟慢慢摸上来。

他们没有直扑城门，也没有先抢仓车。

第一件事，是先把停在外角那列木车头尾截断。

值守外郭的齐卒听见动静，最初还只当是夜里押车人起了争语。

等喊声真正撞出来时，第一道报竟先绕去西门。

因为门上如今最重，谁都习惯了出了动静先问门边要不要添手。

西门值门校尉接报后一愣，立刻又想起王命“门桥先断”，只得先看壕外，再看门内，最后才咬牙分出半列人往外郭去。

这一分，已迟了一截。

等那半列人绕出去时，燕军已借着那列被截断的木车，在外郭西角立住了一小块脚。

守外郭的老卒不是不敢拼。

反而是太敢。

他们知道自己若此刻一退，这几日北面日日后吞的那层疼，便要一下子长出形来。

于是十余名老卒就在木车与残墙间狠狠干住，刀声撞木声混在一起，竟把半边暮色都撞得发闷。

可再敢也挡不住一个事实：

外郭如今本就只剩薄守。

它原先还能靠西门、西仓随时补人。

如今门桥为死线，西门那头一听壕外还有骑影，便不敢再多抽；西仓这两日又日日后压，更不肯再轻放车木与人手。

到天将黑尽时，外郭西角终究还是失了。

失得不大。

只是一角。

可这一角一失，中城里许多人心里都同时一沉。

因为人人都看得明白：

这不是联军突然神兵天降。

是他们自己这几日为了扶住桥门，把别处一点点抽薄，抽到最后，那层原本最不该先丢体面的外郭，终究先漏了一角。

夜报飞骑送到临淄时，齐湣王看完第一句话便问：

“桥可失了？”

“门可失了？”

田文答：“都没有。”

齐湣王沉默了片刻，才又问：“那失的是哪一处？”

“外郭西角。”

这四个字一落，齐湣王的手指在案上轻轻点了两下，像一时竟分不清该怒还是该沉。

桥门未失，按他先前所断，北面最重两层仍在。

可一城若要靠牺牲外郭、西仓外坪与日日后压，才能把桥门托住，这样的“未失”，便也已和旧日大不一样。

田文站在阶下，低声道：“大王，这就是臣前日所说的账。桥门扶得越正，别处若无人再替它们另找活路，终有一处先漏。”

齐湣王没有驳。

只把那封夜报缓缓合上。

而联军帐里，韩老将看见探骑记下“外郭西角失而桥门未失”，竟先是冷笑，后又不笑了。

冷笑，是因齐终于把自己那笔账付出了形。

不笑，是因他知道，这样的一角虽小，却比先前十次急梆、十次堵桥都更伤人。

因为它说明对面已不是乱。

而是被迫拿一处明确的外皮，去养住另一处更重的骨。

乐毅听完，只道：“今夜不再进。”

韩老将问：“为何？”

乐毅看着图上那一角新失的外郭，淡淡道：“它今日付了第一笔看得见的价。再往里逼太急，反倒容易叫它把门桥与外郭重新并成一口血气。”

“先让它自己看一夜：原来守住最重，也会先丢得见人的地方。”

这夜，中城无人能睡。

西门值门校尉站在门楼上，远远望着外郭西角那片暗下去的地方，忽然觉得胸口发闷。

他知道，今夜桥还在，门也在。

可正因桥门都在，他才更清楚看见：那一角外郭，是替他们留下来的。
