# 第一百四十一回 燕军暮取西角截断外坪木车 桥门未失中城却见外郭漏口

第四日黄昏，北面先丢的不是桥，也不是门。

是外郭西角。

西仓外那七列后压木车里，有一列连着两日没能入仓，只得先停到外郭西角的空坪上。那片空坪平日不算要处，这几日门桥为先，守外郭的老卒却已被抽去两轮，西角只剩薄薄一层巡守。

傍晚时，燕军一小股人顺着外坪边那条废沟摸上来。

他们不扑门，也不先抢仓，只先把停在外角那列木车头尾截断。外郭齐卒听见响动，最初只当押车人争吵，等真见刀光时，报信的人已经先往西门跑了。

西门值门校尉接报，第一反应仍是去看壕外。

王命写得明白，门桥在先。壕外那几处马影还在，他不敢一下抽空门后，只得勉强分半列人往外郭去。等那半列人绕出去，燕军已借木车和残墙，在西角站住了脚。

守外郭的老卒拼得很凶。

十几个人贴着木车和残墙狠狠干住，刀声、木裂声搅在一处，把暮色都撞得发闷。可西角本来就薄，西门不敢多抽，西仓也不肯再轻放人手，撑到天黑，外郭西角终究还是失了。

只失一角。

中城里却像一下静了许多。

人人都看见，桥还在，门也在，偏偏是外郭先漏了口。西门校尉站在门楼上，望着那片黑下去的角地，胸口闷得厉害。旁边守卒来请再加一轮灯，他看了看外头，又看了看门内，最后只说：“照旧。”

夜报飞骑入临淄时，齐湣王先问：“桥可失了？”

田文答：“未失。”

“门呢？”

“也未失。”

齐湣王这才停了一停，问出第三句：“那失的是哪处？”

“外郭西角。”

殿里一时没有别的声音。齐湣王把夜报看完，又重新折起，放回案上，折口压得很平。田文站在阶下，也没有再劝，只等着王上把那卷简放稳。

同夜，联军帐中得报。

韩老将先笑，笑意很快就没了。乐毅听完，只说：“今夜不再进。”

帐里诸将都看向他。

乐毅抬手指了指图上那一角新失之地：“让它自己看一夜。”

说完，他把图边那枚压角石往西角挪了半寸，正好压住新添的墨记。
