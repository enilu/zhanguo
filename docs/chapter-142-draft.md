# 第一百四十二回 乐毅收住外郭不使纵兵 临淄回望北面时已见旧法改形

外郭西角失后，北面并没有立刻大乱。

东桥照旧守着。

西门照旧守着。

临淄也没有连夜再发更重的新令。

次日一早，乐毅亲自出帐去看那片新得的角地。他不命诸军乘势再扑，只叫燕军先把残墙、木车、外坪理顺，不许纵兵，不许妄取。原先停在外坪上的木车也没尽数拖走，只挪开最挡路的三辆，其余仍按次平码在地。

韩老将站在旁边，看了一阵才道：“主将像不是在收一角，倒像在把它摆给人看。”

乐毅道：“叫它自己看清。”

这句话一落，众将都没再接。

他们都明白，这一角不是靠一阵猛攻硬砸下来的，是齐自己扶桥扶门，日日让仓让路，最后把外郭让薄了。

同日傍晚，齐湣王又召田文入殿。

这一次，他没有先问桥，也没有先问门，只把北面新到的几封简推到田文面前。简上既有外郭失角，也有西仓车次后压、桥东折道迟车的细数。

齐湣王看着案上那几卷简，问得很慢：“若桥门终守，而外郭、西仓、折道一处处这样往外让，齐守的，还是不是旧法？”

田文没有立刻回。

殿外暮色压下来，窗纸渐渐灰了。隔了好一阵，他才低声道：“能守住，便还是守。只是已不是旧年的守法。”

齐湣王听完，没有再说什么，只把最上头那卷失角夜报拿起来，又放下。

夜里，乐毅回到中帐，把这一路所用的齐北图慢慢卷起。桥、门、仓、路、外郭，都在图上留下了深浅不一的线。他把图卷好，放到案旁，手却没有离开。

案侧另摆着一卷新图，还未全展。

风从帐外吹进来，把那卷新图的一角轻轻掀开。赵奢站得近，看见图上山势更长，边线更硬，中间只隔着一条极窄的空白。

乐毅伸手把那卷新图压住，没有展开，只把旧图往旁边推开了半尺。
