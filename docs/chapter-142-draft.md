# 第一百四十二回 乐毅收住伐齐卷末高势 天下回看秦赵终成正冲

外郭西角失后，齐北面并没有立刻大乱。

中城照旧守住了桥。

西门照旧守住了门。

临淄那头也没有再连夜下出第二道更重的死令。

可天下看齐的眼光，从那一夜起，已全然不同了。

因为人人看得明白，齐这几月不是不会守。

甚至可以说，是守得比前些日子更有章法了。

可它守到最后，守出来的却是另一层更冷的事实：

这座大国如今每扶正一层最重的气，别处便得一日一日替它后吞，直到终于有一处先把账付成看得见的失。

乐毅次日亲自出帐，站在高处看了半日中城。

他没再下重攻令。

只命诸军照旧压住桥门，且把外郭西角那片新得来的地先稳稳理好，不许纵兵，不许妄取。

燕军书记在角下竖起新札，齐旧吏照旧记名，外坪停着的那列木车也没有被尽数掠走，只被挪开了最挡路的三辆。

韩老将见了，低声道：“主将这回不像收一角，倒像在替齐把那一角失出来的难看，摆得更清。”

乐毅答：“它已不是靠一场热战才会失血的局了。摆得越清，越叫它知道自己如今守成什么样。”

这句话，连赵奢都听得沉默了一息。

因为他明白，主将如今争的，已不只是一城一角。

而是把齐这几个月一路学出来的活法，最终推到一个谁都看得见的形上。

同日傍晚，临淄也终于不再只盯着北面桥门。

齐湣王召田文入殿，第一句话便是：

“若桥门终守，而外郭、西仓、折道、空坪一处处这样往外让，齐守的，还算不算齐原来的守法？”

田文听完，没有立刻答。

因为这句话，比前些日子任何一封急简都更沉。

它不再问哪一处该先断，哪一处该后补。

而是在问：

齐如今这套终于摸出来、却日日在付账的活法，到底还能不能算是那个旧日大国该有的守法。

良久，他才低声道：“大王，能守住，便还是守。只是已不是旧年的守。”

齐湣王没有再问。

他只是转头望着北面，像忽然有一瞬连自己都看见了：

齐还未亡。

可齐这一段被逼出来的新活法，已在天下人眼里留下了一道擦不掉的印。

而这道印，往后无论是他自己，还是后来者，都再难装作没看见。

同夜，乐毅在帐中收了这一卷最后一张图。

桥门、外郭、西仓、折道，都已在那图上留下了细细深深的线。

他把图卷起来时，韩老将忽然道：“主将，这一场到这里，齐还在。”

乐毅点头：“在。”

“可天下看它的方式，已不在旧处了。”

帐中一时都静了。

因为诸将知道，这不只是在说齐。

是在说这一整个时代。

自张仪破盟以来，天下先失的是同心；

自白起、赵奢、乐毅相继立名以来，天下又见识了什么叫名将能把国势打出新的形；

而到今日，齐这场伐与守终于叫所有人都更清楚地看见：

强国未必会一夜塌。

可一国若只能靠日日补漏、处处付账去守自己最重的骨，那它纵还站着，也已与旧日大不一样。

更深夜时，西风自北来，吹得帐角轻轻作响。

乐毅没有再看齐图。

他转身，拿起案旁另一卷尚未全展开的大图。

那图上更西一些的地方，山势更长，边线更硬。

赵字与秦字，中间只隔着一线。

韩老将顺着他的目光看去，心里忽然一沉。

因为他知道，乐毅这一卷到此，天下下一卷最重的那口气，已不在齐了。

而在那一线更长、更硬，也更难补漏的秦赵之间。
