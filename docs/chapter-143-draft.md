# 第一百四十三回 廉颇先把赵线坐成铁壁 白起回看长坡只等一国自急

乐毅伐齐的高势未散，天下的目光便已开始一点点转向西北。

赵与秦之间，那条本就越来越紧的边线，终于成了谁也绕不过去的正冲。

赵廷最先动的，不是豪言。

是廉颇。

他到边上后第一件事，不是求锐进，也不是立名阵。

而是亲自沿长坡、谷口、浅水、粮路一段段看过去，回城后只说一句：

“先坐死。”

于是赵边诸军这一年最常见的，不是奔袭与鼓噪。

而是搬土、立垒、深沟、连营、校粮、计月。

年轻将校起初都憋得难受。

“秦兵压境，何以日日只见垒高，不见兵锋？”

廉颇也不与他们争声，只带他们夜里去看边仓。

一夜风起，营外旌旗猎猎，仓中一盏盏灯却极稳。

廉颇站在灯下，只道：“能叫赵这口气先不乱，便比你们今夜出去砍一百个首级都重。”

这话许多年轻人一时都未必服。

可等他们再去前线看秦兵，才慢慢懂了。

对面压上来的，也不是乱军。

而是法齐、令整、路明、后劲极长的秦。

这种对手，最怕你先急。

赵这边在坐死。

秦那边却也并不躁。

白起还未亲临大阵时，秦军前线主将便已照着旧法，一寸寸试赵线。

试谷口多深；

试赵垒换防多密；

试夜里第三更的粮车，从后营到前线究竟要多久。

这些试法看似枯。

可枯到尽头，便是把一条线的气血都摸得差不多了。

等白起真正到边上时，第一夜甚至没有议战。

他只命诸将把近三月赵边所有“未动”“不追”“夜补如常”的细处，一一记上。

旁边年轻秦将看得发急，忍不住道：“赵线坐成这样，若再由它坐下去，岂不越来越难吃？”

白起只问：“它为何能坐？”

那将一时答不上来。

白起便自己把图上赵边那几处垒与粮线慢慢连起来，道：“因为赵还未自急。”

“廉颇守的，不只是地。是赵朝与赵军那口还肯忍、还肯认‘先不败便是胜一半’的气。”

他说完之后，众将都静了。

因为他们都听出来了，主将眼下看的，已不只是廉颇守法有多稳。

而是在看赵国上下，究竟能把这种“先坐死”的活法认多久。

这一看，便是数月。

赵边不出。

秦边不躁。

天下却开始先躁了。

赵国年轻将校觉得这般守下去，有气无名；

邯郸朝中有人也开始问：“赵兵如此强，何以对秦只坐不动？”

而秦廷里，范雎一听见这些风声，便先笑了。

“廉颇不难打。”他说，“难的是让赵先不许他这样打。”

这句话送到前线时，白起看完，只在图上廉颇几处主垒外轻轻划了几道极浅的线。

那线不像包围。

更像在一块已坐稳了的铁上，先慢慢找出哪一处，只要火再烧久一点，里头的人就会先嫌它太冷、太慢、太不长面子。

而这，才是真正更长、更硬、也更难补漏的那一卷，刚刚起笔的时候。
