# 第一百四十三回 廉颇先沿长坡坐死赵线 白起到营只看三月夜补旧札

乐毅压齐之势未尽，西北边上已另起一股更硬的冷气。

廉颇到边后，先走的不是前阵，是长坡、谷口、浅水、后仓和粮道。

他连着看了三日。

第三夜回营，众将都等着他说要不要先出一阵，他却只把木尺往图上一横：“先坐死。”

自那以后，赵营里最常见的不是请战。

是挖沟，立垒，平码拒木，给谷口量宽窄，给后仓记月耗。年轻将校看得难受，有人终于忍不住问：“秦兵压境，何以日日只见垒高，不见兵锋？”

廉颇没在帐里答，夜里把人都带去边仓。

风很大，营外旗子扑扑乱响，仓里的灯却一盏不歪。廉颇站在灯下，伸手按了按仓门木闩：“赵边先不乱，比你今夜斩几个首级更重。”

年轻将校没人再吭声。

秦那边也不躁。

白起到前线第一夜，没有议战，只叫诸将把近三月赵边所有旧札全搬来。帐中铺了满地竹简，记的不是大捷大败，而是“夜补如常”“三更不追”“换防未乱”这些小字。

年轻秦将看得心焦：“赵垒越坐越硬，再让它坐下去，只怕更难。”

白起蹲在图前，把赵边几处主垒和后仓慢慢连起来，问他：“它为何坐得住？”

那将答不上来。

白起也不催，只把最后一根木签插到粮道旁边：“因为赵还肯认这口慢气。”

帐里静了。

自此之后，赵边整日搬土连营，秦边整夜翻旧札看时辰。两边都不快，天下却先有了议论。邯郸有人嫌廉颇太缓，秦廷那头也已有人顺着这股嫌气，开始往赵都里送话。

风从帐口灌进来，把地上的一片旧札掀起半角。白起伸手按住，没有让它翻过去。
