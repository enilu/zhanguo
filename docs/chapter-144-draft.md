# 第一百四十四回 赵廷先嫌廉颇坐得太久 秦廷只把流言一层层送进来

廉颇在边上坐得越稳，邯郸朝中的气便越发不稳。

起初只是年轻言官在殿角轻声一句：

“廉将军善守，然赵岂可常守？”

后来，连贵戚里也有人开始说：

“赵兵之盛，天下称强。若对秦只筑垒计月，岂不叫诸侯笑我赵国有兵而不敢用？”

这些话单听一条，都不算毒。

可一条条叠起来，便像细沙日日往人鞋里进。

赵王起初还肯压。

可边上军费月月增，邯郸仓中账簿一卷卷往上送时，他心里的那口疑，也在慢慢长。

“廉颇所守，究竟是为国，还是怕战？”

这一问一出，赵廷上空气都静了。

因为谁都知道，这话一旦从王上心里生出来，后头许多看似只是议兵的事，便都不再只是议兵。

与此同时，秦廷并没有明发离间大书。

范雎做得极细。

先是让几路商旅在邯郸酒肆里慢慢传：

“秦军亦苦守耳，赵若敢一击，未必不得利。”

又叫旧日与赵人有往来的小臣在私下局里轻轻问一句：

“廉将军年高，持重固可敬，然天下局势这样拖下去，赵可有尽头？”

这些话没有人站出来拍胸说是秦使教的。

可也正因没有，才最难防。

它们像风，不从正门进。

只从每个人心里那一点本就已在发痒的地方，一层层吹进去。

邯郸朝中最先顶不住的，不是大臣。

是那些日日看军费、催粮道、算月耗的吏。

一名司仓老吏在夜里合账时，盯着那一串又一串押往边上的数字，忽然低声道：“若如此守到来年，赵纵不败，国库也先要喘。”

这话被旁边主簿听去，第二日便绕了半圈，进了贵戚耳中。

再过一日，殿上便有人顺势道：

“大王，廉将军能守，臣不疑。可守法若太久，亦伤国本。”

廉颇的表疏到此时仍旧只八个字：

“秦锐未疲，赵不可轻。”

字不多。

也不替自己辩。

可正因如此，邯郸里越来越多人开始觉得，这位老将像一块太沉的铁。

沉得有用。

可也沉得不肯顺着朝堂那口越来越急的气往前走。

白起在前线收到邯郸风声时，仍没动。

他只问探骑：“赵边诸垒，军心可变？”

探骑答：“未变。士卒多服廉颇。”

白起点头：“那便不是边军会先乱。”

“是邯郸会先嫌它太稳。”

他说完之后，把图往邯郸那头多移了半寸。

韩老将若在此，必会看懂。

这已不是单看边线了。

而是把一场大战真正推向决定处的，不再只是谷口与高坡。

而是赵王、赵廷、粮簿、虚名、国库与那句越来越多人想听见的：

“何时可以不再这样守下去。”
