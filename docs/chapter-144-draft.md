# 第一百四十四回 邯郸先从粮簿嫌廉颇太稳 秦使不立大书只把闲话吹进酒肆

廉颇坐得越稳，邯郸朝里的气便越急。

先是司仓官夜里对账，看着月月往边上出的粮、木、车马，低声说了一句：“若如此守到来年，国库先喘。”这句小话第二日绕进贵戚宅里，又过一日，就上了殿。

殿上开始有人说：“赵兵强而不出，诸侯岂不笑我？”

也有人说：“廉将军善守不假，可守法太久，终归伤财。”

赵王起初还能压，等边上军费一卷卷送来，眉头便再没舒开过。某夜退朝后，他把两卷军费簿并在一处，看了很久，只问近臣一句：“廉颇这是守国，还是怕战？”

近臣不敢答。

与此同时，秦廷也没发什么明白离间书。

范雎做得更细。他让商旅在邯郸酒肆里说：“秦军亦苦守耳，赵若敢一击，未必不得利。”又叫旧识赵人的小臣在私席上随口一问：“廉将军年高持重，自是老成，只是赵总不能年年这样守下去。”

这些话没有封泥，也没有使节身份，只在酒气和灯影里一遍遍地被人接过去。

邯郸最先被磨动的不是大臣，是算账的人。司仓、主簿、催粮的老吏、押车的中吏，一个个都在簿子边上多添了一笔月耗。那些笔迹到了朝上，就慢慢有了别的分量。

廉颇递回邯郸的表疏却一直很短。

“秦锐未疲，赵不可轻。”

八个字，不多求，也不自辩。

表疏越短，朝里看他越像一块不肯顺着朝气转身的老铁。赵王连着看了数封后，把其中一卷搁到案边，手指在“不可轻”三字上敲了两下，没有再翻。

同一时候，白起在前线收到邯郸风声，只问探骑：“边军如何？”

探骑答：“军中多服廉颇，垒未乱，粮道如旧。”

白起听完，把图往邯郸方向轻轻挪了半寸。

他这一挪，旁边几名秦将都看懂了。

边上那层铁，还没松。

先松的，会是邯郸案上的手。
