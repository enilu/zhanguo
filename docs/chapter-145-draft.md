# 第一百四十五回 赵括奉诏接军先问何处可争 廉颇出营时只替老旗再紧一扣

邯郸终于还是换将了。

诏书到边那日，风刮得很紧。廉颇站在主垒前，把那卷诏简从头看到尾，神色没动，旁边老部将的眼睛却先红了。

来接军的是赵括。

他到营时甲鲜马壮，目光亮得很。年轻将校看见他，脸上的气都变了。赵括进主帐后，第一句先问的不是后仓，也不是月耗，而是：“秦军哪一处最认定我赵不敢出？”

帐里许多人一下就精神起来。

廉颇看着他，道：“秦最想见的，不是你不敢出，是你急着出。”

赵括拱手很恭，话却不退：“老将军守法，晚生敬服。只是赵若再只守不争，人心、国力都要先被拖空。”

这句话正说在邯郸换将的根子上，也正说在营里许多年轻人的心口上。廉颇听完，没有再争，只把各营粮线、退道、谷口、忌轻出的坡地一处处指给赵括看。

赵括都认真听了。

只是听的时候，眼睛总会不自觉地往图前那片更开的地方去。

交军那夜，营里没有擂鼓。廉颇最后一次上主垒时，老旗在风里磨得发白。他站了一阵，伸手把旗绳又紧了一扣。

老部将在旁边哑着声问：“将军，还要留什么话？”

廉颇看着那面旗，过了片刻才道：“告诉旧部，别忘了秦最能吃什么。”

说完便下了垒。

沿途旧卒见他出营，都只是站直，没有人敢大声相送。火把一路往外去，照得土垒边上一截一截发黄，像一条慢慢退远的旧路。

赵括接手后的第一夜，就改了三条军令：

“前营探路加倍。”

“主垒三处前推。”

“旧日只守不出的谷口，逐一试秦。”

令一下去，营里少年将校眼里都有了火。

秦营得报时，白起把帐中的灯吹灭了一盏，图上却多点了几处谷口。
