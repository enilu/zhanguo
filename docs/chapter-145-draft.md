# 第一百四十五回 赵括受军时志气满身 廉颇去垒处无声只剩老旗

邯郸终于还是换将了。

诏书到边时，风正大。

廉颇站在主垒前，把那卷诏简从头到尾看完，脸上一点波澜也没有。

只有旁边几名老部将的眼睛，一下子都红了。

因为他们知道，这不是寻常调动。

是赵国终于嫌这位老将坐得太稳、太久、太不肯给朝堂一口痛快气了。

来接军的是赵括。

他到边上那一日，甲鲜、马壮、目光极亮，整个人像一支刚从鞘里拔出来的长锋。

年轻将校们先是惊，后是喜。

因为赵括一开口，便与廉颇全然不同。

他不先问粮，不先问月，不先问后垒。

他先问：

“秦军何处最以为我赵不敢出？”

这句话一出，营里许多人心口都热了一下。

像是憋了太久的一口气，终于有人替他们先说出来了。

廉颇却只是望着赵括，缓缓道：“秦最盼的，不是你不敢出。是你急着出。”

赵括听完，拱手很恭，却并不退：

“老将军守法，晚生敬服。然天下局势到此，赵若再只守不出，只怕人心、国力都先被拖空。”

这句话，说中了赵廷换将的根，也说中了无数年轻将校这几月心里最难压的那点火。

廉颇没有再争。

因为他知道，到了这一步，靠再说几句，是挡不住这卷诏书、这满营求战气、这邯郸已先嫌他太稳的局了。

交军那一日，赵边主垒前没有擂鼓。

廉颇只把各营粮路、备用退道、几处最忌轻出的谷口一一指给赵括看。

赵括也都认真听了。

只是听归听，他眼里的光却始终亮在更前面。

亮在那片他还未真正推开的秦阵上。

入夜前，廉颇最后一次站上主垒高处。

风吹老旗，旗角磨得发白。

那是他守赵这一段日子里最熟悉的一面旗。

他看了很久，忽然伸手把旗绳又紧了一扣。

旁边老部将声音发哑：“将军，还要叮嘱些什么？”

廉颇想了想，最后只道：

“告诉旧部，换将归换将，先别忘了秦最能吃什么。”

老部将眼眶一热，低头应了。

可他心里也明白，这一句话太轻。

轻到未必敌得过满营年轻人的热气，敌得过邯郸朝里那口终于等到换将的快意。

廉颇离营时，天已黑透。

没有万人相送。

也没有谁敢在这个时候把难过喊得太响。

只有沿路旧卒远远看见那支出营的火把时，都不由自主地站直了身子。

他们谁也没说话。

可许多人心里都很清楚，今夜走出去的，不只是一个老将。

还走出去了一种最沉、最耐、也最不讨人喜欢的守法。

赵括入主后，第一件事便是改前线军令。

“前营探路加倍。”

“主垒三处前推。”

“旧日只守不出的数处谷口，明日起逐一试秦。”

这三道令一下，营里少年将校几乎人人眼里都有了光。

他们终于不用再只看土垒与粮线。

终于能往前。

秦军那边，白起听见赵换将的消息时，反而把帐中灯吹灭了一盏。

旁边年轻秦将不解。

白起却只道：“灯太多，容易叫人看不清火真正在哪一处亮。”

说完，他提笔在图上赵边那几处原本死守的谷口外，轻轻点了数下。

众将都看懂了。

赵线那块沉铁，已经被挪开了。

接下来秦要等的，不再是它如何守。

而是新上来这把锋，究竟会先往哪一处最想立名、也最易被人借力的地方扎下去。
