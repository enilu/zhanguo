# 第一百四十六回 赵军由试成争先压东坡 白起等它把最热一口气尽放出去

赵括接军后的第六日，赵营已与先前判若两样。

旧垒还在，垒前却多了探骑、前出小股和日日试锋的年轻将。头两回试探，赵军还真占了点便宜：浅坡逼退秦前哨半里，东面谷口也抢回了一截高地。

胜不大，营里却越传越热。

“新将一来，秦线便松了。”

“旧日那套死守，果然太迟。”

这类话传到赵括耳里，他没有压，反倒亲自去前阵看了两回。第三次回营时，他站在图前，用木尺点着东坡：“这里再压一层，秦就得让。”

众将应声极快。

秦营这边，白起仍不动大令，只让人记三件事：赵前推之后哪一路最不肯退，哪几名年轻赵将最爱乘小利再请一步，赵括自己最常往哪一面看。

半月后，东坡外终于压出了一口新势。

那日赵括亲临前阵，见秦军再退，立刻下令：“今日争坡。”

这两个字一下，赵营整口气都往前涌了。赵军不是乱冲，可这半月来试出来的那点小胜，已把所有人都推到只想再多争一寸的地方。东坡之上，赵军一度真把秦线压得往后松。

营里少年将校个个眼睛发亮。

白起这时才把图卷起。

旁边秦将一怔：“主将，不看了？”

“够了。”白起道。

他说完，重新展开另一张图，把原先铺好的两翼伏势和后截线一处处压实。帐里诸将看见那几道线，呼吸都不由放轻了。

第二日清晨，赵军果然沿东坡再进。

再进一层，秦线还退。赵括站在高处，看着前头那道口子越张越开，终于把原本还留着的一层收势也放了出去：“前军再压，左右并展。”

午后，秦两翼先合。

傍晚，后截线闭上。

赵军前锋直到这时，才发现先前那一路似退非退的秦阵并不是被打散，而是在把他们往更深处一寸寸引。营中先乱的不是胆，而是号与路。前头还在要坡，后头已在找退道，军令撞在一起，谁都听不全。

白起立在高处，望着那片开始自相挤压的赵阵，没有追着喊令，只抬手把东坡那枚木钉轻轻按到底。
