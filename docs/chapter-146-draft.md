# 第一百四十六回 长平由试变争终入决口 白起见赵新锋满营便知可围

赵括接军后的第六日，赵边前线已与先前判若两营。

旧日那些最稳的垒，仍在。

可垒前多了更多探骑、试锋、前出小股。

年轻赵将们像终于得了一口久憋的气，日日都想在秦阵前先争一寸。

头两回试锋，赵军甚至真占了点便宜。

有一处浅坡，秦小股后退半里；

又有一谷口，赵骑夜里突然压上，逼得秦前哨换位。

这些利都不大。

可传回营里，却足够把那股“老将只会守、新将一来便有起色”的气越吹越高。

赵括站在图前时，眼里那层光也越来越亮。

“秦并非不可逼。”他对诸将说，“只要我军不再拘旧法，被它拖住。”

这句话满营都爱听。

因为它替许多人把几个月来最想说、又一直没能说出口的话，先说了。

可秦军那头，白起看着这些前推、夜试与浅胜，却几乎没有任何动色。

他只命诸将记三样：

“赵前推后，哪一路最不肯再退。”

“哪几名年轻赵将一得小利，便最爱请更进一步。”

“赵括本人，是更重左谷，还是更重东坡。”

这些东西看似细。

可细到尽头，便是把一支新锋最喜欢往哪边压、最不甘在哪边收，一点点全摸出来。

半月之后，赵军终于真在东坡外压出一口新势。

那一日，赵括亲临前阵，见秦前线再退，终于下令：

“前军不再只试。”

“今日争坡。”

这“争坡”二字一落，赵军上下都像被点着了一把火。

他们不是无脑冲阵。

可自换将以来越积越满的那股“终于可以把旧守法翻过去”的热，终于在这一刻全往前涌了。

东坡之上，赵军一度真压得秦线往后松。

营中少年将校简直要喊出声。

可也正是在这一刻，白起把图卷了起来。

旁边秦将一怔：“主将，不再看了？”

白起道：“够了。”

“它如今不是来试我哪一处虚。是已认定，这一处争下去，便能把秦线真正扯开。”

这句话一出，众将心里都同时一沉。

因为他们知道，主将等的从来不是赵军什么时候小胜。

而是等它什么时候开始相信，自己抓住了一处可以一路压下去的决口。

到了夜里，秦营议战第一次不再只谈试。

白起把数路伏兵与后截线在图上徐徐铺开时，帐中连最老的将领都不由屏住了气。

那不是一场寻常反扑。

而是把赵军新锋最爱往前去的那条路，直接往更深处引。

第二日清晨，赵军果然沿东坡再进。

再进一层时，秦线仍退。

赵括看着那片似乎越来越松的秦前阵，终于第一次把原本还留着的一层收势，也放开了。

“前军再压。”

“左右并展。”

“今日要把这一口彻底争成我赵之势。”

这三句，便是长平真正从“试”走进“争”的那一刻。

也是白起等了许久的那一刻。

午后，秦两翼伏势先合。

傍晚，后截线再闭。

赵括直到这时，才猛地看见先前那一路路似退非退的秦军，原来并不是被自己打散。

而是正把他一路最想压下去的新锋，慢慢引到了一个再难轻退的位置上。

夜风扑营，赵营第一次真正乱了。

不是士卒不敢战。

而是他们直到这一刻，才发现自己原以为终于争出来的那一道新口，竟忽然成了把全营往更深处拖下去的决口。

而白起站在高处，望着那片已开始自乱的赵阵，脸上一点波动也没有。

因为他知道，长平这一卷真正最重的那一步，不在赵括接军。

也不在赵军得小胜。

而在这一刻：

赵终于把自己最年轻、最热、最想证明新法比旧守更强的那口气，整整压进了秦为它留好的深处。
