# 第一百四十七回 白起合围长平断赵元气 赵括死阵前举国闻哭如风

长平一入深处，后面的事便快得像刀。

秦两翼一合，赵军最先失的不是勇。

是回头的路。

赵括连下数道军令，要前军回收、中军稳住、后路先开。

可阵一旦被拖进那样的深口里，许多令还未传遍，谷与坡、坡与沟之间便已不是原先那样一气相通。

有人还在往前撞。

有人已在回头找路。

更有一批年轻将校，直到此刻都还不肯信眼前这一下会真把全军拉成死局。

他们昨日还在东坡上见过秦退。

怎么一夜之间，便像整个天地都把口收紧了。

赵括骑在阵前，脸上最后那点少年锋气并未立刻散。

他先是怒，再是不甘。

怒自己竟真中了这一步。

不甘的是，若此刻立刻全退，岂不等于承认自己接军以来所争、所压、所说的每一句，都在把赵往死口里带。

于是他没有第一时间全收，只命主力先顶，自己亲率一部去找西侧缺口。

这一找，便再也没有回来。

后来活下来的人说，赵括死时还在往前。

不是往胜里去。

是在往一条也许还未全闭的缝里去。

可那缝终究没有给他。

赵括一死，赵军那口最后还能自欺“主将必有后着”的气，便一下子空了。

后面几十日，赵营已不是在争胜。

是在断粮、断水、断望里，一日日把人心熬干。

白起没有急着狠狠干尽。

他只把所有能让赵再接外气的路一一收死。

收得比杀更冷。

因为他要断的，不只是一军。

是赵国往后许多年都再难补回来的元气。

到最后，赵营里连想骂的人都少了。

许多人只是抱着兵器坐着，看谷上天色一日日白、又一日日黑。

他们不知邯郸此刻在做什么。

也不知赵王究竟有没有在想办法。

他们只知道，自己从前跟着廉颇守时嫌太慢，后来跟着赵括争时嫌不够快，到如今却连慢与快都已没了，只剩下等。

等一口已知不会再来的活路。

当白起最终下令整收长平时，秦营里也没有多少欢声。

因为在场每一个老将都知道，这不是寻常大胜。

这是把一个大国最不该这样失掉的青年、兵心与可再起之气，狠狠干成了无数后年都难补的一道断口。

消息传到邯郸那日，赵宫外风极大。

宫门前先是静。

静到像城里的人都还不肯信。

等到“赵括死”“大军尽没”这几句终于确准下来后，哭声才像风一样，先在宫门外卷起来，再顺着街巷一层层卷开。

赵王在殿中坐了很久，像忽然一下连怒都发不出。

换将是他下的。

那一口嫌廉颇太稳、太慢、太不像大赵强兵该有之势的急，也是他心里先长出来的。

到这一刻，所有人都知道，赵未亡。

可赵那口最硬、最重、最让天下一直不敢轻看的军国元气，已经被长平狠狠干断了一大截。

白起立在长平高处，看着远处天光，脸色仍旧冷得像石。

旁边秦将低声道：“赵自此难复。”

白起没有答。

因为他知道，长平这一仗打出来的，从来不只是“赵难复”。

而是天下自此都要慢慢看见：

秦已不仅能胜强敌。

秦已能把一个最强敌国，整整打成往后许多年都带着这道断口活。
