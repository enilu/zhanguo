# 第一百四十七回 秦军合围先断赵阵归路 赵括西寻缺口战死坡前乱军中

东坡那口气一放尽，后头的口子就收得极快。

午后，秦两翼先合。

赵前锋还在往前压，后军已发现来路不再通。号角一阵一阵地吹，前头听见的是进，后头听见的却是收。坡、沟、谷口之间原本还算熟的路，忽然全像换了样。

赵括连下数令，要前军回收，中军稳住，西侧先开一路。

令传到一半就乱了。

有人还在往前找秦军退口，有人已掉头抢路，更多的人挤在坡下，不知道该往哪边听。昨日还在营里夸东坡得手的年轻将校，到这时才看见秦军那一路路似退非退的影子，原来是把人往里引。

赵括骑在阵前，脸色先白了一瞬，随即又硬了下来。

他不肯立刻全收。

若此刻全退，便等于承认接军以来所有前推、试锋、争坡，都把赵军一步步送进了这个口袋。他把身边亲骑一分，自己带一部人直往西侧去找缺口。

西侧山风大，土也松。

秦军早在那边埋了截线。赵括冲到半坡时，前头已见乱石和拒木，后头又被自己人挤住。他回身喝令再冲，话还没传尽，箭已先下来了。

后来逃出来的人说，赵括死前还在朝前指。

指的不是胜，是一条再也冲不开的缝。

主将一死，赵阵里最后那层还想“再撑一阵也许便有转机”的气，一下就空了。秦军却不急着尽扑，只把能出粮、能出水、能通外气的路一条条收死。赵军从东坡争势，一下跌进了断粮断水的长熬。

白起每日只问三件事：赵营还剩多少水，夜里还有没有人敢结队突口，谷里死马埋到了哪一层。

答上来的数字一天天短下去。

赵营里先还能骂，后来连骂的人都少了。许多人抱着兵器坐在壕边，看天色由白转黑，再由黑转白。有人忽然想起廉颇守时那一圈圈土垒，想起当时嫌它慢，如今却连一堵能退到后头再喘一口气的垒也没有了。

到最后秦军整收长平时，秦营里也没什么欢声。

都是老将了，谁都看得出这一回收住的，不只是一军。

消息传到邯郸那日，宫门外先是静得厉害。等“赵括死”“大军尽没”的话真正坐实，哭声才从宫门口卷出去，一街一巷地往外散。赵王坐在殿里，面前放着换将时的旧诏和今日新到的败报，半晌都没有抬手去碰。

长平坡上，白起站得很高。

风从谷里往上卷，把衣角吹得很直。他低头看了一会儿谷口新竖的秦旗，转身下坡，一次也没回头。
