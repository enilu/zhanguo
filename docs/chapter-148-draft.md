# 第一百四十八回 信陵君夜入侯嬴门下问策 魏营夺符之后拔军直赴邯郸

长平之后，邯郸城像病了一场大病。

秦兵压境，城里再没人嫌谁慢，也没人再提什么强兵锐气。朝上朝下都只剩一句话，援兵何在。

赵急书送到大梁时，夜已深。

信陵君拆简看完，外头更鼓刚过三下。简里没有多少哀词，只一句写得最重：“邯郸若破，赵后便轮诸国自守。”

他看了三遍，起身出门。

去见的不是魏王，也不是朝臣，是侯嬴。

老者坐在暗灯下，像早知道他会来。信陵君刚进门便道：“赵不可不救。”侯嬴抬眼先问：“魏王许么？”

这句话把屋里一下问静了。

信陵君没答。

魏王不是不知赵若亡、魏必受压，只是秦势方盛，谁都怕先替赵挨那一刀。侯嬴看着他，慢慢把案上一枚旧符推近了些：“公子若真要去，就别再等‘许’字了。”

灯花轻轻爆了一声。

信陵君低头盯着那枚兵符，过了片刻，伸手按住，没有再放开。

后面的事都在夜里。

窃符，入营，夺兵。晋鄙起初不肯从，无忌也没有多话，只把符放到灯下，再往前走了一步。帐里刀光一闪，晋鄙倒下那一刻，营中先乱了一阵，乱声刚起，无忌已提剑出帐。

“拔军。”他说。

魏卒一时不知该看符，还是该看倒地的主将。无忌把兵符高高举起，剑尖却指向西面：“今日赴赵，不独为赵一城。”

没人再敢迟疑。

夜里拔营最怕乱，偏这一夜魏军反而极整。前营先起，中军接马，后营卷火，辎车一路一路出营门。侯嬴没随军，只在城门前远远看着。大队开出后，他才把手里那根旧杖轻轻往地上一顿。

邯郸城头看见魏旗时，天正蒙亮。

城上先是没人敢喊，等“信陵君至”四字一层层传开，哭声和欢声才一起撞出来。那些声音太杂，连守城老卒都分不清自己究竟是在哭，还是在笑。

秦军自然也来拦。

可魏军这一股夜里拔出来的气太直，前头撞上去时，后头仍在往前送。无忌不在后营坐镇，就立在军前，甲上全是夜露。邯郸城下那一阵，杀声把长平后压在赵人胸口上的那块石头，硬生生顶起了一寸。

战毕入城，街上很多人只知道跪。

无忌从马背上看过去，谁也没扶，只叫后军先入粮、先接城门。他知道邯郸还远没到能哭尽的时候，城门口那两辆先推进去的粮车，才是眼下最要紧的东西。
