# 第一百四十八回 邯郸危城信陵君夜出 六国悲烈最后仍有人肯为赵赌命

长平之后，赵举国如病。

邯郸城下，秦兵之压一日重过一日。

赵宫里却不再有人喊快，也不再有人嫌廉颇太慢。

那些话都像被长平一夜之间埋了。

可埋了不等于还有路。

赵国要活，只能等外援。

六国之中，真正肯把这份“外援”两字当成自己命去扛的，最后竟只剩魏公子无忌。

信陵君收到赵急书时，夜已半。

书中没有多少哭语。

只一句：

“邯郸若破，赵之后便轮诸国自守。”

信陵君把那句反复看了三遍，终究起身出门。

他去见的，不是朝中重臣。

是侯嬴。

老者坐在暗灯下，像早知道他今夜必来。

信陵君一进门，第一句便是：“赵不可不救。”

侯嬴没有立刻答，只问：“魏王许么？”

信陵君沉默。

因为他心里最清楚，魏王不是全不知赵若亡、魏必危。

可秦如今之势太盛，朝中人人都怕先替赵担下那一刀，最后叫魏自己先成第二个口子。

这便是今日天下最深的病。

人人都懂大局。

可真到要先拿自己去给大局挡一步时，便都想等别人先动。

侯嬴看着他，忽然道：“公子若真要救，便莫再拿‘许不许’问自己了。”

这一句像火。

信陵君立在那里，胸口那股早已压了许久的气，终于一下子全燃了起来。

后面的事，便都是赌。

窃符。

夺晋鄙兵。

魏营夜动。

每一步都像踩在刀刃上。

可信陵君脸上那股神色，反而越到后面越平。

因为他知道，到了这一步，能让天下还记得“六国虽败，仍有人肯替别人先把命押出去”的，怕也只剩这一遭了。

晋鄙被杀那一夜，魏营大乱一瞬，随即又被无忌一把压住。

他立在军前，拔剑只说一句：

“今日不为魏一国救赵。”

“为天下尚知唇亡齿寒者救赵。”

这句话一出，许多原本心中仍惴惴的魏卒，竟都硬生生把那口气提了起来。

因为他们忽然明白，自己今夜往赵去，不只是替一座邯郸城。

也是替天下还剩下的那点不肯全散的人心。

邯郸城中，赵人望见远处魏救兵旗影时，许多人竟先不敢信。

直到“信陵君至”四字真正传遍城头，哭声与欢声才混在一起，几乎震得残城都发抖。

秦军那边亦不是无备。

可这股从死局里硬冲进来、带着天下最后一层烈气的援赵兵，终究还是把邯郸往回硬拽了一把。

战后，信陵君入城时，邯郸街上许多老幼都只是跪着，不知该说什么。

因为他们心里都懂，这不是赵已无事。

也不是六国又能回到旧日均势。

只是当天下一路坏到这里，仍有人肯在最不值、最危险的时候，替“还有人愿为别人死一步”这件事，狠狠干出一道光来。

而这道光，悲得很，也烈得很。

足够照亮长平之后那片最重的暗，却终究照不回从前去了。
