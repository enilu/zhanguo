# 第一百四十九回 邯郸虽解天下旧势已断 嬴政坐定之后先从宫中旧手开刀

信陵君把邯郸往回拽住了一把，赵总算没在那一口气上断掉。

可长平前的旧局，再没人当它还能回来。

赵伤了根，魏虽出兵，朝里怕秦的气却没散。楚还大，齐还在，六国彼此来往照旧，谁心里却都明白，再想像从前那样把诸国真连成一口久气，已很难了。

西边的秦，却在这道断口后头把目光收得更紧。

咸阳宫里，秦王政先收的不是山东诸侯，是宫中的旧手。

嫪毐之乱起时，城中很多人还当这是后宫私祸。等秦王政亲自下令闭门、封道、收印、逐署拿人时，老臣们才看见这位年轻君王的手有多冷。乱起于何门，兵出于何巷，哪处库钥曾被换过，哪名旧客夜里出过城，他都一条条往回收。

宫城那几夜，最忙的不是刀，是簿。

每抓一人，后头就跟着一卷旧案；每封一门，旁边便立起一块新牌，写清哪署改谁掌。乱压下去后，咸阳城中死寂了数日，等城门再开时，许多人忽然发现，先前那些靠着太后、旧客、私门转动的线，已经被一截截切没了。

吕不韦也退了。

他的车出咸阳时，街边看的人不多。旧相权最盛时，车过处人人避道；到这时，只有几名老吏站在路旁，把车尾消失的方向看了很久。

李斯正是在这段时日里一步步站到前面来。

他上殿时，不谈烈，不谈哀，也不拿旧辞动人，只把各署法、粮、路、兵、印如何归到一线，按次一条条排出来。秦王政坐在上首听着，脸上没什么神色，等李斯说到末尾，才把案上一枚旧印推到一边，换上了新制官印。

那一枚印落下去很轻。

殿中人却都听见了。

因为这一声以后，秦往后许多仗、许多令、许多取地灭国的手，便不会再像旧日那样各自生长，而是都要从同一套新收紧的法与器里出去。

夜里，咸阳宫的灯比平日少了许多。

值夜老宦经过旧署空院，看见门上新换的封条还未被风吹平，边角轻轻翻着。他站住抚平，转身再走时，脚步都比往常更轻。
