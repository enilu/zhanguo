# 第一百四十九回 长平邯郸卷终天下失均势 嬴政将起帝国机器先扫内患

信陵君救赵之后，邯郸是活下来了。

赵也还在。

可天下再没有人会真把长平之前的局，当成还能回来。

赵伤根已成。

魏虽出了一位信陵君，可魏王与朝中那口怕秦的气，也并未因此尽散。

楚仍大，齐仍在，可人人看得见，六国如今再难像从前那样真正把彼此连成一口能久撑的大气了。

这一卷至此，真正断掉的，不只是赵国一军。

是天下“尚可恢复均势”的那点底。

而秦国在这断口后头，终于把眼光慢慢从“如何再胜一国”抬到了更远的一处。

那便是如何把这一连串战争与胜势，真正熬成一座能吞诸国而不自乱的帝国机器。

其时秦王政已长。

咸阳宫里，最先要杀的，不是山东六国。

是秦自己身上的旧患。

嫪毐之乱起时，咸阳城中许多人都还当这只是宫闱之祸。

可嬴政亲手压下去时，秦廷老臣们却都看见了另一层。

这位年轻君王压乱时，脸上没有少年初掌大局该有的躁。

有的只是极冷的整。

乱起于何门；

兵出于何道；

牵连到哪一批旧客、哪一层宗室、哪一处仓与印，他都要一一收干净。

那不是一时泄怒。

而像一个人终于坐上整张图最中央之后，先把图里一切自己不能尽控的旧手、旧线、旧口，狠狠干断。

吕不韦退，旧势渐去。

李斯在这时候真正站到了更前面。

他不是白起那样的人。

也不是乐毅、廉颇、信陵君那样的人。

可若说这个时代后半最冷的一种力量，便正是他所代表的那一种。

那是把国不只当国。

而当成一架一层层咬紧、能把人、法、粮、路、兵、令全拧成一线的机器。

李斯第一次在廷上把“今天下之势，已不在诸侯各争短长，而在谁能把天下之争，变成一套不会自己先散的法与器”说出来时，许多老臣都沉默了。

因为他们听得明白，这已不是旧式列国眼光。

是在说：秦如今要做的，不只是再胜一仗，再取一地。

而是要让所有后面的胜仗、取地、灭国，都像从同一只冷手里，一件件排着次序拿出去。

嬴政坐在上首，看着李斯，脸上并无多余神色。

可殿中人都知道，从这一刻起，天下最后一卷最重的人与最重的力，正在慢慢坐实。

一边，是六国还剩下的名将、烈士、亡国者与不肯低头的人。

另一边，则是秦终于不再满足于“胜”，而是要把“胜到最后”也做成法度、做成程序、做成谁死谁存都按着一张更大图往前滚的帝国机器。

而那张图，才刚刚开始在咸阳真正展开。
