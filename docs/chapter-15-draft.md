# 第十五回 一身系印权重诸夏 四顾藏针势压群疑

同盟的裂纹既已一回回被看清，苏秦这“一身系印”的分量，反倒显得更重。

第十五回开始时，诸侯与群臣几乎都已习惯了一件事：凡盟中有疑，先问苏秦之意；凡几国轻重难分，先看苏秦如何排；凡有一处线头松动，也总是先想把消息送到苏秦案前，再看他会如何补这一缝。

这便是权重。

它不全来自六印，更来自前十四回里一回回压出来的事实：旁人能议局，真正能把局暂时维持住的人，眼下只有苏秦。

可权重越显，疑心也越跟着长。

赵国有人暗道：“此人持印日久，诸国往来皆绕其手，权是否太重了些？”齐楚那边也有人私语：“今日是借其持局，明日若诸侯事事都先问苏秦而后问君上，岂不又成另一层挟势？”便连平素最倚赖盟中声势的小国里，也偶有疑念：苏秦居中久了，会不会终究更看重大国之意，而把弱国之急放在后头？

这些疑，不是翻脸，却都像细针一样，慢慢藏进四顾目光之中。

苏秦对此并非不知。

他最清楚，自己今日之位，既令众国都需他，也令众国都不敢完全放心他。因为他越能持局，越不像只是某国一时门客；越不像一国门客，诸侯便越会防他是否借持局之名，坐大其权。

一日，盟中又起一场小危。

魏韩那边新得边报，说秦境兵马调动略有异样，于是急催赵国先明示援应之意；赵廷内部却有人觉得，边报未明，不宜轻易把自己先推到最前。魏使闻言，立时冷道：“合从若到此时还只顾自保，魏韩今日何必还守其盟？”赵人也硬声道：“赵倡其议，不等于赵要替诸国先把一切险都担尽。”

若换在前几回，这类争执还可借“大义”先压一压；到第十五回，众人彼此都已太熟，旧话再说，只会更显空。

苏秦于是只做两件事。

先稳其势。

他当着两边使者道：“边报既未尽实，今日最忌两件事：一是近秦之国先疑盟无用，二是倡议之国先让人看见其遇事便退。此二者一出，秦纵未真试边，也已先看出合从可动之处。”说到这里，他又转向赵人：“赵若在此刻只想着别先担，倡议之名便会立刻变轻。”再对魏韩道：“你们若在边报未尽实前便先以盟逼人，旁国也会只觉近秦者仍只是求把险全推出去。”

再定其分。

他当场提议，由赵先出使问魏韩边情，魏韩则先以实报相通，不得夸急以自重；齐楚两边亦各遣使列席，以明此事不止近秦数国之事。如此一来，赵未被逼着先担尽实责，魏韩也不致觉得盟义只是空名，而大国远国又都被扯回了“不得假装置身事外”的位置。

这一招并不华丽，却极有效。

因为苏秦心里明白，自己此时不能偏心，也不能怕人说自己权重便缩手。他若一缩，众国立即各退回旧位；他若乱压，则又真会坐实旁人对他“持印太重”的戒意。高位之难，恰在必须时时应答，却又无处可以放手偏向。

这一轮小危最终压了下去。魏韩虽仍心急，却也承认赵并未全退；赵人虽仍不肯多让，却也知道苏秦这番调度，是为守它倡议之势，不叫它一时显怯。齐楚来使则都默默把这一幕看在眼里，因为它们也更看清楚：苏秦的威望并非虚名，而是真能在群疑并起时，把局从散边上重新压回去。

可越是如此，四下藏着的针便越多。

有人借他之威办事，有人防他之权太盛；有人敬他，是因真服其能；有人敬他，却只因眼下离不开他。苏秦行于其中，像立在一张薄冰上持印而行，脚下每一步都还稳，冰面却并非没有细响。

这一晚，他独行于馆后长廊，风过庭树，叶影微摇。那位年轻议者同行片刻，忽道：“你今日又替众国把一场疑心压了下去。若换旁人，只怕早已各自翻脸。”

苏秦淡淡道：“局还能压，不是因为他们真肯同心，而是因为他们都知道，此时翻脸对自己更坏。”

那人又问：“既如此，你何以还这样尽力？”

苏秦望着远处烛火，缓缓道：“印既在我身，势既因我而聚，我便不能只享其名，不受其险。何况他们虽各有针，我也不能任其针都朝外去。外头还有秦。”

年轻议者闻言无语。

他直到此时才真正懂得，第十五回最该写出的，不是苏秦又压住了一回群疑，而是他越在高位上显出其能，就越像站在众目、众疑、众需之间的一处最险高台。

高处是真高。

寒意也是真寒。
