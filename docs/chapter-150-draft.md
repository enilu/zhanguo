# 第一百五十回 韩非入秦文章照殿如刃 李斯同门对坐终把人世让给帝国

韩非入秦那一年，咸阳许多人都在等着看一场奇景。

法家文章第一人，终于到了这座最用法、也最会用人的城。

嬴政见韩非时，殿上灯火极明。

韩非说话不快，甚至带着一点天生的吃力。

可他每一句一落，殿中人都像看见一柄极薄的刀，在纸上、在人心里、在诸侯旧习与君臣旧情上，一层层削过去。

他说势。

说法。

说君主如何不为臣蔽、不为私情乱。

说到深处时，连秦廷老臣们都不得不承认，这不是寻常纵横之士的巧言。

是把这个时代最冷的那一层理，几乎说到了尽头。

嬴政听得很久都没有打断。

因为他知道，眼前这个韩国公子说出来的，正是他要拿来压诸侯、压旧臣、压天下那一套帝国骨法里最锋利的部分。

可殿下另一个人，李斯，却在这明灯下越听越沉。

他与韩非同门。

懂得对方文章有多高，也更懂得：这样的人若真正被嬴政全收进心里，自己往后在秦廷所占的位置，便不只是遇见了一个同道。

而是遇见了一个更纯、更险、也更容易在君王案前显得像“法之本身”的人。

韩非后来被留在咸阳。

留得并不自由。

他日日在馆舍中写，写韩国可存不可速亡，写诸侯之势不能只靠硬吞，写法之外仍有人心与时机。

这些话未必不对。

可到了秦已把天下看成一张次第可收之图的时候，对的人，反也可能正因太对，显得不合时。

李斯最终还是下了手。

不是在殿上与韩非辩到脸红耳赤。

而是在最秦的方式里，把一个天生最懂法术之人，悄无声息地逼进了那座大国机器里最冷的一处死角。

韩非死于咸阳的消息传出来时，秦廷上下都很静。

因为没人不知道，这里面不只是旧国公子的难命。

也有同门、才名、君王之需与帝国之路不能容太多同样锋利之刃并立的一层冷。

李斯那夜独坐案前，灯极小。

案上压着的，既有韩非旧文，也有新拟的灭韩策。

他看了许久，最后还是伸手把旧文一点点卷起，换上了新策。

从这一刻起，秦对韩国再下手时，已不再只是取其地。

也是在向天下说明：

这个最后的时代，不再给最深的理与最好的人都留位置。

它最后要留下来的，只是那架能走到终局的帝国机器，与最适合替它转动的人。
