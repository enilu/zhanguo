# 第一百五十回 韩非入秦殿上陈法如削 李斯夜归卷旧文换上灭韩新策

韩非入秦那年，咸阳很多人都等着看这一场会面。

韩国公子，法家名士，终于走进了最重法、也最会用法的宫。

秦王政召见那日，殿上灯火极明。韩非说话不快，声音也不高，偏偏一层层说下去，殿里便越来越静。说势，说法，说君主不可为臣蔽，也不可为私恩乱，到深处时，像有人拿一柄极薄的刀，沿着旧日诸侯习气和君臣情面慢慢削过去。

秦王政听得很久，没有打断。

殿下一侧，李斯也在听。

他与韩非同出一门，听得比旁人更深。韩非每往前说一层，殿里便有一层人心往他那边沉过去。等会面散时，韩非留下，秦王政却没有立刻发话，只把人送来的那卷《孤愤》重新打开，又看了一遍。

韩非住进咸阳馆舍后，写的却不全是秦人爱听的话。

他写韩国不可骤亡，写并天下也要看时与势，写法之外仍有人心可用。竹简一卷卷送入宫中，也一卷卷从李斯案前过。

那几夜，李斯总是看得很慢。

旧同门的字他太熟了，熟到一看就知哪一句会让君王欣赏，哪一句又会在如今的咸阳显得太迟。到后来，他的案上便并排压了两卷东西：一边是韩非新文，一边是自己新拟的灭韩策。

馆舍里，韩非的出入越来越少。

守门小吏最先察觉不对，是因为送进去的灯油比从前省了。后来连传简的人也少了，原本日日要往馆舍跑一趟的中吏，隔几日才会去一次。等到韩非死讯传出时，咸阳城里没有谁大声议论，街上却静得很怪。

李斯那夜独坐案前，灯只留了一盏。

案上一边摊着韩非旧文，一边放着新拟的灭韩策。他看了很久，终于伸手把旧文一卷卷收起，换成了新策。竹简互换时，木片相擦，发出极轻的一串声。

门外值夜的小吏听见那声音，抬头望了一眼，又很快低下去。

到天快亮时，案上只剩灭韩策铺在最前。

李斯提笔补上最后几字，把笔搁回砚边时，灯芯也恰好烧到了尽头。
