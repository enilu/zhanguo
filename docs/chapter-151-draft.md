# 第一百五十一回 韩地先收咸阳按次翻图 赵北尚有李牧独把边风撑住

韩亡时，天下并没有多少惊声。

新郑城门一开，秦吏便带着簿册、印泥、尺度进城。旧韩官还未散尽，秦的新牌已一块块挂上门去。百姓站在街边看着，只听见靴声和木牌相碰的声响，竟比喊杀还冷。

咸阳收到韩国入秦的报文后，李斯先做的不是庆贺，是把天下图展开，挪去一枚写着“韩”的小木签，又把旁边几处官道和仓线重新连了一遍。

秦王政坐在上首看着，没有说“下一个是谁”，只用手指在图上往东轻轻滑了一段。

那一段后头，正是赵。

赵此时还能把边风撑住的人，只剩李牧。

他守北地，营中少有夸言。风雪一起，先点马，先看草，先让骑卒绕边跑过一圈，再回帐议兵。秦军几次东探西压，都被他挡了回去，挡得不奇，也不乱。

边上老卒最服他的一件事，是李牧从不让边军空耗。

能省一队夜巡，便把那一队留给次日风口；能少跑一回空骑，便把马力记到账上。赵国走到这一步，边军还能像军，不像散兵，全靠他一手压着。

咸阳那头看战报，看得也越来越烦。

不是烦败，是烦李牧这种守法。韩已先收，赵却因李牧这一根硬骨，迟迟不能照着图上的次第往下翻。

于是秦廷先动的，不只是兵。

重赂、离言、旧日老法，一样样往邯郸去。起初赵廷还能压，后来谣言多了，连朝会上也有人敢问：“李牧若真可用，何以秦军到此，总还在边上拖着？”

这话最蠢，偏偏最容易入君王耳。

李牧奉召入朝那日，天色发阴。来使立在营门外，神色不敢抬。李牧看见那张脸，已知道这回不是前线出了事。他回头看了一眼北地营门，风把营旗吹得斜过去，旗影扫过雪地，像一道长长的旧痕。

到邯郸，话并没有说太久。

李牧也没有辩。

他脱下佩剑时，旁边老臣的手都在抖。剑放到案上，只发出很轻的一声。隔日，赵北边营得报，许多老卒整整一夜没有睡，天亮后照旧出营换防，只是旗底下少了一个再也不会回来的人。

咸阳收到李牧死讯时，李斯把那卷早已备好的灭赵案默默推上前去。秦王政看了许久，提起朱笔，在案角写下一个“可”字。
