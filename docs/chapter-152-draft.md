# 第一百五十二回 太子丹寒夜置酒送荆轲 图穷匕见时秦殿第一次见热血

赵将亡，燕都更冷。

太子丹知道，燕已没有正面扛秦的本钱。朝上仍有人说守，有人说缓，有人说且再看齐，听到最后都像空话。真能拿出来的，只剩一条险路。

荆轲受命那夜，燕都天寒得很。

置酒的厅不大，高渐离坐在一旁击筑，酒也不烈。座中人都知道这一去多半无回，反倒谁也不肯哭。荆轲举杯时，太子丹的手一直按在案边，按得指节发白，直到杯空也没松开。

次日发时，城外风更硬。

送行的人不多，马蹄踩在冻土上，声音很脆。高渐离的筑声送到路口便断了，荆轲没有回头，只把袖口理了一下，继续往前。

入秦后，咸阳宫里接待诸侯使者的礼一切照旧。

嬴政坐在殿上，先看的是地图匣、督亢图和来使脚下的步子。秦廷礼官唱名，韩非死后，咸阳宫里对这类异国使者已多了一层更冷的防。

图穷匕见那一刻，殿上还是乱了一瞬。

匕首出匣太快，连最近的侍臣都慢了半步。荆轲扑得极狠，脚下带着整个人往前撞，袖中地图散开，竹轴滚出去很远。嬴政后退，撞在屏后，殿中兵器和脚步声一下全起。

那一阵极短。

秦廷侍卫冲上来时，血已溅到柱边。荆轲被逼到尽头，背抵着柱，仍死死盯着前头。他最后没能再近一步，匕首却钉在了殿柱上，钉得很深，木屑碎了一地。

事后很久，殿柱那处痕都没人敢先去削平。

嬴政站在柱前看了片刻，转身时一句多话都没有，只令：“燕，不必再留。”

消息传到燕都，太子丹听完，手里那盏酒竟没有端稳，酒沿着杯口淌下来，流了半案。旁边人想替他拭，他摆了摆手，自己用袖子抹去。袖上酒痕很快凉透了。

咸阳那边，值夜宦者路过那根殿柱时，总会不自觉看一眼柱上留下的那道小裂。
