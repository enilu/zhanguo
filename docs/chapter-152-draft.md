# 第一百五十二回 太子丹知燕只剩一柄险 荆轲入秦前酒冷歌长天下皆缩成一匕

赵既将亡，燕便更冷。

太子丹比谁都清楚，到了这一步，燕再谈正面抗秦，已近梦话。

可若什么都不做，燕也不过是等着那张并吞次第的图，终有一日翻到自己这一页。

于是天下最后最险、也最像绝路里磨出来的一柄尖锋，终于出鞘。

荆轲受命时，燕都天寒。

许多人都知道这一去大概无回。

可也正因如此，这一去才像把一国最后那点不肯只等死的心，全压进了一人袖中、一柄匕里。

高渐离击筑那夜，酒并不烈。

烈的是座中每个人心里都清楚：

荆轲此去，不是要真靠一匕改变天下大势。

而是要替已无多路的弱国，把“我知不成，仍要向那最不可近处直走一步”的人心，狠狠干出来一次。

荆轲入秦殿时，嬴政比许多人想的更静。

他早已不是少年。

也不是只靠勇气压乱的秦王了。

他如今坐在那架帝国机器的中枢上，看任何来人，先看的都不只是人。

还看他身后是一国、是一局、是一条会不会叫机器失节的线。

图穷匕见那一刻，殿上空气像猛地被人掐紧。

荆轲扑得极狠。

不是无谋。

是把一切都知而仍要扑下去的狠。

可那一扑终究还是扑空了天下。

血溅，柱寒，殿中兵响骤起。

荆轲最后倚柱而笑时，许多后来听过此事的人都忘不了那点笑。

不是胜者之笑。

更像一个人明知自己已不能改变最后大局，却仍在用最后一点力，让这场一统之路上最冷的那座宫，也终于见一次血热与人命直冲帝王案前的样子。

消息传出后，咸阳更冷。

嬴政不再对燕存半点余手。

因为他知道，这一匕刺来的，已不只是一国最后的绝望。

也是天下所有亡国者、待亡者心里最险最不甘的那一点，曾经真地靠近过自己。

而燕都那边，太子丹听见败讯，脸上反而像一下空了。

他知道，这局到此便彻底尽了。

燕所能拿出来的最后一柄险，已经断在秦殿上。

往后剩下的，便只有等那张图终于翻过来，把燕也收进去。
