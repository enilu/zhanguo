# 第一百五十三回 王翦先请田宅坐定君疑 项燕提楚兵骨一段段迎住秦锋

天下最后几场大仗里，最沉的一场在楚。

楚大，地阔，旧气又深，病了许多年，骨头却还在。秦廷议灭楚时，年轻将领纷纷请战，王翦站在殿下，等众人说尽了，才慢慢抬头：“非六十万，不可。”

满殿都静了一下。

有人觉得他老了，有人觉得他怯。秦王政眼里那点疑也一闪而过。王翦却像没看见，只接着请田、请宅、请赏，一样样都说得平平，仿佛只怕仗后少了自己的富贵。

殿上许多人暗里发笑。

秦王政没有笑。

他盯着王翦看了很久，最后把兵符推过去。王翦双手接过时，神色仍旧平，像方才请的不是田宅，只是出征前该领的军器。

大军东下，楚并没有像韩那样一碰就散。

项燕提兵来迎，楚军里老卒多，旧将也多，军中旗帜不算整齐，落到阵前却有一股别国少见的沉劲。秦楚第一次对垒，秦军并未立刻占尽便宜。年轻秦将心里先憋起气，王翦却按着不发，只让营垒一层层往前坐，辎车一队队平码进去。

楚军来撞，便守。

楚军退去，便仍守。

项燕知道这一仗求不了轻胜。他要的，是让楚这口病了很久的大气，还能再整整提起来一次。于是每退一段，就在下一处再顶住；每失一城，就在下一道水边再立住。

楚地的雨一下，营里泥就重。

秦军鞋底日日带泥，辎车压过的印痕第二天还在。王翦在泥里走营时，连甲边都沾着湿色，回帐后只问两件事：粮车今日晚了多少，楚营夜里还亮着多少火。

拖得久了，楚先老。

项燕再硬，也挡不住秦那六十万兵一层层把水陆粮路全磨过去。最后楚营崩时，没有哪一声像长平那样响，倒像一棵极老的树，先是里面空了，再被风从外头慢慢掀倒。

败讯传出那天，楚地很多地方的人并没有立刻哭。

他们站在原地，看着远处官道上那一串往西去的败兵影子，像一时还没认下“楚竟也会败到这里”。等晚些时候城门再开，哭声才从巷里慢慢翻上来，沉得很长。

王翦回到帐中，解甲时把腰间那块旧玉佩一并放到案上。

玉佩压住了新送来的楚地舆图，半张图都被挡了进去。
