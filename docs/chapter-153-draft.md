# 第一百五十三回 王翦索赏先把疑心坐实 项燕尽楚兵骨终知此战只许死撑

若说天下最后一卷真正最重的大战，不在燕。

在楚。

楚太大，也太能忍。

它病久，却从未真轻。

而项燕到这时，几乎已是楚最后能把这口沉国之气再硬提起来的人。

秦要灭楚，朝中先争的甚至不是出不出兵。

而是谁去。

年轻将领请战者众。

王翦却只在殿上慢慢抬头，道：“非六十万，不可。”

这句话一出，满殿皆惊。

有人觉得他老了、怯了、故作慎重。

嬴政眼里那点疑，也一闪而过。

可王翦不争。

他只是继续请田、请宅、请赏，像一个老将忽然只剩下惜身爱富那一点心。

许多人私下都笑。

可真正看懂的人都知道，他是在用这种最不英雄的方式，把君王那点最容易在大战前先生出来的疑，先坐实、先消掉。

你既觉得我求赏爱富，便该知道我无别图。

你既知我无别图，往后六十万兵尽付我时，便能少疑半层。

这便是王翦的狠。

不是冲阵前一声大喝的狠。

而是连君王心里那道最小、最不能不先算的疑，也提前替你算完。

王翦东下时，楚国并未像韩国那样一碰就倒。

项燕提兵迎之，楚军那股沉、硬、忍与久病不死的劲，在此战里全压了出来。

第一回合，秦军甚至并未立刻尽得便宜。

年轻秦将们心里都先憋起了一口气。

可王翦照旧不躁。

他比谁都清楚，楚最可怕的从来不是一时锋。

是这大国最后一层还能把残余国气拧成一支死战之兵的本事。

项燕也知道，自己这一仗不是求全胜。

是求楚还能不像那些已亡之国那样，被人按着次第一页页轻轻翻过去。

所以楚军顶得极狠。

每一处退，都像还想找下一处再撑。

每一处撑，又都像知道自己后头已无多少可再撑之地。

这便使整场灭楚之战，带上了一种别国少有的沉。

不是一口气冲到底的烈。

而是知道后面全是坠，也还要一段段踩着往前死撑的沉。

最终，王翦还是赢了。

赢在更大，更稳，更能等，也更能把一国最后那点死战之气一点点磨干。

项燕败讯传出那日，楚人许多都没有立刻哭。

他们只是站着，像忽然连“楚竟也会到这一天”的念头都还没完全认下。

可等这念头真正压到胸口，哭声一起，便比别国更沉，更长，也更像从极深处翻上来。

因为楚是最后那个让天下还肯相信“大国病久，未必就真会死得这样快”的旧影。

如今连这旧影也断了。

秦之一统，便再没有哪一国、哪一将、哪一口残余国气，真能把这条路从根上拖住了。
