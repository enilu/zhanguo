# 第一百五十四回 大梁水入宫墙先白一层 齐王临海东时只剩一国孤看天下

楚一亡，魏便更窄了。

秦军围大梁，用的不是一鼓作气的猛攻，是水。

水先漫到城外田里，再漫过坊墙，最后贴上宫墙根。墙脚那层旧灰被水泡得发白，剥下来一片片黏在地上。城里人起初还想着修、想着堵，到后来只剩抬着东西往高处挪。

魏王在宫中走廊上看水，一日要换三回鞋。

宫人把木屐垫得很高，水还是往里浸。等到最后一层门槛都守不住时，许多人竟先不是哭，而是忙着把案上竹简和祭器往更高的台阶搬。

大梁一陷，魏也就尽了。

燕那边则是另一种尽法。

荆轲之后，太子丹死，残城一座座失下去。燕地旧人还记得易水送别时那阵寒风，也记得匕首钉在秦殿柱上的消息，可这些记得到后来都救不了城门。秦军按着地图一路北推，能收的收，能并的并，燕最后那点靠一柄险撑起来的不甘，也跟着旧城一起沉下去了。

到这一步，天下只剩齐。

可齐已不再是当年那个能和秦、赵、楚并提的齐。海东路远，朝里还有人说“齐可自保”，还有人说“只要不逆秦，还能再晚几年”。这些话一日日说着，齐王建也一日日听着，听到后来，满朝竟没人再提别国了。

因为别国都没了。

秦兵真正东向时，齐国这才看清自己已是最后一个。

没有赵挡在西面，也没有楚在南边扯住半口秦锋，连魏、燕这样的旧邻都已不在。临淄城中，太史还照例记天象，司仓还照例点库存，城门却一天比一天静，静到连守卒都少了说话。

齐王建有一日登高，远远望见海。

海风很大，城中旗子却已不怎么动了。身边近臣还想说几句“海东可久”，话到嘴边，自己也先停住。齐王建看了很久，最后只把袖子往后拢了拢，下阶时一句话都没说。

齐降那日，血并不多。

城门开时，外头秦吏先入，带的仍是簿册、印泥和封条。临淄旧臣站在道旁，看着那些东西一件件搬进去，忽然有一人低头笑了一声，笑完便哭了。旁边人想扶，他却只是摆手，自己慢慢蹲了下去。

海风一路吹进城来，把门边新挂上的秦牌吹得轻轻摇动。
