# 第一百五十四回 魏城终陷齐王尚观海东 王翦回军时海内已只剩孤气

楚一亡，魏再难支。

大梁之水一漫，魏王最后那点宫中体面，也被一层层泡得发白。

魏亡没有楚那样沉，也没有赵那样裂。

更像一个早知自己终会到此、却直到最后那刻仍不肯把“真到此了”这件事彻底说出口的国，终于在水声里把那口拖了很久的气吐尽。

燕也在荆轲之后一路被压。

太子丹死。

残城失守。

那些曾经靠一柄险、一夜歌、一口不甘撑起来的尖意，终究敌不过帝国机器一层层往前滚的冷。

到这一步，天下真正还未覆名的，只剩齐。

可齐已不是当年那个还能与秦、赵、楚并看高低的齐。

它在乐毅伐齐之后一路自补、自守、自看伤。

到最后，竟活成了六国里最孤的一国。

齐王建在海东看天下时，诸国已次第亡尽。

朝中仍有人说：“齐地远，尚可自全。”

也有人说：“只要不先逆秦，未必不能晚存几年。”

这些话听着像谋。

其实都不过是在天下大局已死之后，给自己那点残喘找个稍不难听的名字。

当秦兵真正东向时，齐已没有谁可等。

没有赵可为屏。

没有楚可为阔。

没有魏、燕还在旁边替它分去半口秦锋。

这正是齐最后的悲处。

它并不是最先亡，也不是最烈亡。

它是看着别人一个个都倒下之后，才忽然发现自己原来早已被留成了最后那一个无人相援的孤国。

齐降那一日，并没有太多血。

可也正因少血，才更显得苍凉。

因为它让人明白，一统到最后，已不必每一次都靠长平、靠邯郸、靠灭楚那样的巨响。

有时只要把天下别的气都先收尽，最后剩下这一口最孤的气，自己便会知道，再往前也不过是空。

王翦回军过旧齐地时，曾远远望见海边。

风很大。

海色却平。

他看了一会儿，没有说什么。

因为到此为止，天下该打的最重几仗都已打过，该死的最硬几层人也都已死过。

海内剩下的，不再是“谁还能挡秦”。

而只是：这一切终于完了之后，活下来的人，要拿什么去记住那些已经倒下去的国与人。
