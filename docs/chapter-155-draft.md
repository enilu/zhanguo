# 第一百五十五回 咸阳钟动新制一条条下 旧国故臣立在风里听见山河改名

齐降之后，咸阳大钟连撞了数声。

钟声过城时，宫门、官署、坊市一齐安静下来。许多人停在原地，只抬头听。那声音极沉，像把这些年诸侯征伐留下的尘都一层层压回地里去。

第二日，新制便一条条下来了。

郡县名册先送三署，接着是车轨、度量、书字。李斯立在殿下，按着次序往下奏，每说一条，旁边小吏便换上一块新牌。旧印一枚枚收入木匣，新印蘸了朱泥，平码在案。

秦王政坐在上首，听得极静。

殿外站着新降诸国的故臣。

赵臣看见“郡”字牌时，想起邯郸宫门外那阵风；楚臣听见南郡归署，手指微微一缩；燕臣站得最靠后，目光却总往远处的殿柱去，那柱上还有当年匕首留下的一道浅痕；齐臣衣角带着海边的旧盐气，听到临淄改署时，低头闭了一下眼。

没有人喧哗。

礼官唱名，新制一卷卷往外发，脚步声在长廊里来回。宫外各署也跟着动起来，量器换新，旧车轮距一架架校正，城门值册上的国名慢慢被郡县名替过去。

到傍晚，咸阳城中最忙的是匠人和书记。

木匠在官门上换牌，书记在灯下重抄地志，驿骑一队队出城，把新的尺寸、新的法和新的名送向旧六国故地。城里一条旧街口，原先挂着山东货牌的商铺掌柜站在门边看了半日，最后自己搬来梯子，把旧牌摘了下来。

夜里，钟不再撞了。

咸阳城却比白日更亮。许多新降故臣仍留在外殿阶下，没有谁先走。风从高处下来，吹得众人衣袍轻轻作响。有人抬头望向北面，有人望向东面，眼里装的却都不是同一处旧国。

殿内最后一枚新印压下去时，发出极轻的一声闷响。

那声音并不大，外头的人却都像听见了。
