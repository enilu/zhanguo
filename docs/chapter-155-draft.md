# 第一百五十五回 海内归秦咸阳钟响震地 亡国故臣回望旧山河皆成帝国道里

齐降之后，咸阳大钟第一次连震数次。

那钟声极沉。

沉得像要把百年诸侯争战、合纵连横、名将辩士、强国亡国者的影子，一并压进同一座城里。

海内归秦。

这四个字，从此不再只是策论里会写的壮语。

而成了道里、郡县、律令、车轨、城门、印绶都要一一照着转起来的现实。

嬴政站在咸阳高处时，天下在他眼里终于不再是一张谁亡谁存未定的大图。

而成了一座完成了的器。

器成则震。

也冷。

因为他比谁都明白，这一路走到这里，死过多少人，断过多少国，也压碎过多少本也足够称一世英雄的才、勇、烈与不肯低头。

李斯在殿下奉上新制时，声音极稳。

郡县、书同文、车同轨、度量之齐，一条条往下铺。

这正是他与秦最擅长的地方。

既然天下已被打成一个，便要让它真能按一个天下转。

这其中没有太多儿女情。

也不容太多旧国之念继续盘踞在法度之上。

可咸阳殿外，那些新降诸国故臣站在风里，心里想的却未必是同一件事。

有人看见统一，先想到乱世终于止息。

有人却只看见自己故国的山川、宗庙、旧印与旧名，从此都要被压进另一套更大的名目里去。

赵人想起长平与邯郸。

楚人想起项燕最后那口不肯先散的沉气。

燕人想起荆轲入秦前那夜的筑声。

齐人想起海东最后那点平得几乎像梦的风。

而苏秦、张仪、白起、廉颇、乐毅、信陵君、李牧这些名字，到此也都像从不同的山头回看过来。

他们有的早胜，有的早败，有的功成，有的命苦。

可若没有他们，这一条路也不会长成今天这个样。

钟再响时，许多故臣都低了头。

不是都服。

也不是都恨。

而是到这一刻，连最不愿认的人也得认：

天下真的从此不再是那六国七雄各自争衡的天下了。

它已被一个更冷、更整、更不肯回头的帝国，整个抱住了。
