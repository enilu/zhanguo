# 第一百五十六回 一统终成而英雄尽散 史家后看乱世仍闻旧人长叹

海内归一之后，世上并没有立刻只剩欢声。

路上少了战车。

关隘少了反复易帜。

百姓终于能在一些地方把门栓得不那么紧。

可与此同时，许多人心里也都知道，自己这一生最熟、最恨、也最离不开的那个诸侯并立的时代，到这里终究是完了。

苏秦当年困顿起身时，大约不会想到，自己佩六国相印的高光，最终只是这条大路开头的一团最亮也最短的火。

张仪也未必会想到，自己一层层拆开的，不只是一个联盟。

更是在替后面所有更硬、更冷的战争，把门慢慢打开。

白起、廉颇、乐毅、赵奢、李牧、王翦、项燕，各自都曾把一国之势、一时之势，狠狠干出自己的样子。

可信也好，不信也好，到最后没有谁能只靠一人之兵、一人之智、一人之勇，把这个时代整个拽回别处去。

信陵君曾在最暗处替天下聚过一次还不肯尽死的人心。

荆轲曾把一柄匕首送到帝王眼前。

韩非把理说到极深，最终却仍死在更冷的人世之用里。

这些人并没有白活。

他们让后来人知道，这个时代不是一条平平顺顺走向秦的路。

它中间有过无数真想扳回、真能挡住半程、真敢在最不值的时候先把命押出去的人。

可也正因如此，秦最后的胜，才不轻。

它不是打倒几个无能之辈得来的容易胜。

而是在那么多聪明、刚烈、稳重、雄武、悲壮的人都先后出场之后，还一步步把天下整整收进手里的胜。

史家后来写战国，往往先写秦统一。

可若真懂这一百多年的人，便不会只写“终归于一”四字便了事。

还要写那些失掉的、断掉的、来不及的、明知不成仍要向前的。

要写苏秦夜读，张仪破盟。

写楚怀王的误、陈轸的药、屈原的直。

写白起之冷，廉颇之忍，乐毅之深，赵括之热，信陵君之烈，李牧之撑，荆轲之决，项燕之沉。

也要写嬴政与李斯。

因为若没有他们，这一切便不会真走到结束乱世那一步。

这一部天下共视角的乱世，到此终于收住。

最后留在纸上的，不该只是帝国完成时那一下钟声。

还该有钟声之后，那些散在故关、旧垒、空城、废桥、海边与山道上的回响。

风过时，像仍能叫人听见：

有人曾佩六印而起。

有人曾一言拆盟。

有人曾在长平高处把一国元气打断。

也有人曾在邯郸危城外，为天下最后那点不肯尽散的人心，赌上自己的一生。

而历史终究向前。

可那些名字，那些胜与败，那些本来就不该被轻轻翻过去的痛与光，也终究都还在。

这，才是战国真到尽头时，最该留下来的余声。
