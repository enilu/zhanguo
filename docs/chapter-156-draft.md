# 第一百五十六回 故关空垒旧桥海风仍在 后来史笔写尽一统先听乱世余声

海内既一，路上渐渐少了战车。

关隘不再日日易旗，市井也慢慢敢把门栓得早一些。可对许多亲历此世的人来说，天下安下来，并不等于心里那些旧声旧影立刻散了。

数年后，有史官出咸阳，往东去。

他先过旧韩地，看见城门牌额已全换成秦字，街边却还有老人管某条巷叫旧韩名。再往北，到赵地，长平旧坡草已长高，风从谷里卷上来时，仍带着说不清的凉。守坡老卒给他指过一处低地，只说：“当年人都陷在这下面。”

史官没有多问，记下了。

再往邯郸，危城旧垒只剩半截。城外有人卖酒，酒肆墙上挂着信陵君救赵的旧画，颜色早褪得发灰。店家给他添酒时，轻轻拍了拍那幅画，什么也没说。

他又往南，过楚地。

雨下在旧楚城上，比别处更绵长。有人领他看项燕旧营，泥里仍能翻出断箭头；又有人说屈子旧庙里香火从未断过。史官站在檐下避雨，听见庙里童子背诵旧辞，字句隔着雨声传出来，一句清，一句模糊。

到燕地时，天已很冷。

易水边有人击筑，手法并不好，旁边听的人也不多。史官驻马听了一会儿，想起咸阳殿柱上的旧裂，没有过去打断，只让从者把这一段也记上。

最后他到海边。

齐地风大，海色很平。旧临淄的城门改了名，海边老人却仍说“当年齐王曾在此望海”。史官顺着老人手指的方向看去，只见一线灰蓝，什么也看不出来，耳边却一直是风。

回咸阳后，他把这些一路所见摊在案上。

案上既有新定的郡县名，也有旧人口中不肯改尽的国名；既有秦法如何齐一天下，也有一处处旧垒、空城、废桥、海风和酒肆里没有散掉的旧声。

夜深时，他提笔写下卷首。

写到“秦并海内”时停了一停，又往下写。灯火照着竹简，照见他笔下时而是苏秦、张仪，时而是白起、廉颇、乐毅、信陵君、李牧、荆轲、项燕，最后又落回咸阳宫中的秦王政与李斯。

写到更尽，窗外风起。

案角压着的一卷旧地志被吹开半页，露出里头一条早已不用的旧国边线。史官伸手按住，没有立刻合上，只让它在灯下露着。
