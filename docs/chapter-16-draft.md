# 第十六回 东方尚借合从自壮 西土但凭法度观时

到第十六回时，东方与秦之间的根本差异，已经不必靠大战便能看得很清。

东方之所以还能把众国暂时拢在一处，靠的是“合从”这面旗。

诸侯借它自壮，士人借它谈希望，近秦之国借它求不至独受其祸，远秦之国也借它抬一抬自己在大局里的位置。哪怕人人都知道这旗下面并非铁板一块，可只要旗还在，众国便还有一层可以共同出声、共同自安的名义。

这名义是有用的。

礼可因它整，文书可因它通，诸侯之间那点本来各自散乱的惧秦之心，也可因它暂时汇成同一股声气。若无这面旗，六国便更快退回各自求活、彼此观望的旧路。

可第十六回也要补上一笔更冷的真相：名能壮心，未必壮骨。

因为旗一立，兵不会因此立刻更整，粮不会因此忽然更足，彼此多年不信也不会因此就真化为信。强国仍旧强在它本有之兵势，小国也仍旧弱在它本有之根底。合从之名可以让东方一时不至散得太快，却不能自动把六国底下那些本来不齐的骨头一夕接好。

苏秦对此越发明白。

他有时站在盟中高处看下去，能清楚感觉到：如今东方多的是“借名抬势”的人。赵借其名稳北方，燕借其名求不在局外，魏韩借其名不至立成弃子，齐楚也借其名，不至让自己在这场大势里显得太后知后觉。

人人都在借。

正因人人都借，这旗便一时更高。

可苏秦也知道，借来的势，若底下不够实，终究比不过另一种更稳的东西。

那便是秦。

秦在这一回里，仍未大举东动，也未见咸阳因东方合从先起什么额外喧声。可若把镜头转回函谷关内，便能看见另一种与东方全然不同的稳法。

秦军按期操练，吏治照例运转，边臣议事、文法行令、仓廪征发、关塞巡检，样样都有旧例可循。它不需借什么共同之名来给自己壮胆，因为它本就靠法度、军政与一层层积年累下来的秩序站在那里。

这才是秦最可怕处。

东方要时时重申“合从不可散”，因为稍一松口，人心便会向旧处退去；秦却不必日日对自己说“不可散”，因为它本身就是一架已磨得极熟的国家机器。东方靠名把众心托住，秦靠骨把自己立住。

有一日，苏秦与赵国那位年轻议者同登城楼。楼下使者车马往来，正是合从名义最盛时。那人望着城中风声，道：“今日东方士气，毕竟与一年前全不同了。”

苏秦点头：“这是实话。若无此旗，六国如今更不会还坐在一处。”

那人又道：“既如此，你为何仍常常忧色不去？”

苏秦望向西方，沉默片刻，才道：“因为旗可以让人抬头，不一定让人站稳。你看东方，人人借其名自壮；再看西边，秦不借名，也一样站得极稳。”

对方顺着他目光望去，只见天际山影沉沉，虽隔万里，也仿佛能看见那道函谷重关静静压着西土。

同日的咸阳，一位边臣向秦王回报：“东方合从之名近来更盛，士气与往来皆较往年不同。”

秦王问：“它因此便多了多少兵、多少粮、多少能一令而行的国？”

边臣一时无言。

旁边老臣便答：“名能壮声，未必壮骨。”

秦王点头：“正是。它壮其声，我修其内。待其声与骨相验之时，再看谁更重。”

这几句话，便把第十六回最深的一层写尽了。

东方靠联合止损，借一面共同之旗来把人心暂时拖住；秦靠法度养骨，就算一时不举旗不大动，也仍能凭自己的国家机器稳稳压在那里。前者真有其用，后者却更见根底。

到回末时，东方仍盛，苏秦持印而行，诸侯尚借合从之名彼此自壮；可读者已比前几回更清楚地看见：这场较量从来不只是口舌与盟书之争，更是“借势”与“养骨”两条路数在暗里早早分出了轻重苗头。
