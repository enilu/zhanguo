# 第十六回 合从旗高先压诸议 咸阳灯冷只记人心

邯郸入秋之后，馆驿间来往更密。

赵、燕、魏、韩的使者隔三差五便入相府，齐楚的书也一封接一封送来。封泥新破，案上又添一卷；堂下的人还未退净，门外又已有车声。六国相印悬在苏秦身侧，人人看见它时，脸上都要多出两分郑重。

这郑重是真的。

至少眼下，谁都不敢再把“合从”当成一句空话。赵要借它稳北，燕要借它入局，魏韩要借它不至独受秦锋，齐楚纵还未肯尽出全力，也不能再装作此事与自己无关。大势一旦被人叫出名字，许多原先散在地上的东西，便会自己向那名字底下聚。

那几日，邯郸城里最常见的，便是深夜馆门仍亮着灯。

苏秦一处处见人，一处处补话。

燕使嫌盟书里写赵太重，怕北面成事之后，燕又被挤回角上；魏使追着问一旦秦压河外，齐楚到底肯不肯先动；韩使最不放心的却是粮运与应援次序，嘴上不敢说得太明，翻来覆去问的全是“若边急，该由谁先发车、谁后发卒”。

苏秦整夜整夜地改简。

有时只是一行字先后换一换，便能让远国觉得自己没有白来；有时不过把“共击”改成“共应”，便能叫近秦之国肯把印再按下去一点。

一日夜里，赵国一名年轻议者送来新抄盟案，低声道：“相国，照这样修下去，似乎人人都还能安。”

苏秦正伏在灯下改字，闻言抬了抬眼。

“安是安，”他说，“可别把能安几日，看成长安。”

那议者没有接上。

苏秦把竹简翻过来，指了指末尾两处墨迹深浅不同的地方：“你看这两笔。上一笔是燕人要改，下一笔是魏人要添。改的时候，都说是小处。可小处若总在改，说明底下并不齐。”

那人看了半晌，才道：“但总算都还肯坐下来。”

苏秦点了点头。

“肯坐下来，已是不易。”

他说这句时，声音很轻，像是不愿惊着案上的灯火。

他当然知道，能把六国逼到一张案前，已是眼下最大的本事。可也正因为日日坐在这张案前，他比谁都更清楚，那面旗高是高了，旗下面却并不是铁板一块。有人借它求自壮，有人借它求自安，有人借它抬名位，有人借它躲在别人身后多喘几口气。

人人都在借。

借的人一多，旗便更高；可借来的势若落不到兵、粮、路、次上，到了见血的时候，未必顶得住。

同一时日，咸阳宫里却是另一番气象。

殿门开着，渭北的风穿堂而过，吹得案上几卷军簿边角微微卷起。秦王嬴驷没有先看东方盟书，先看的反倒是函谷守备、秋收仓数、边骑换班几件小事。

樗里疾立在阶下，等秦王把最后一卷放下，才道：“东方近来很热。”

秦王淡淡道：“热到什么地步？”

樗里疾答：“赵馆夜饮不散，六印已归苏秦，诸侯文书往来极密。看样子，都怕落在那面旗后头。”

秦王听罢，只嗯了一声。

他拿起另一卷细作抄来的副本，慢慢看过去。上面不是盟书正文，记的反倒是各国私下问过什么、改过什么、争过什么。燕问北线名位，魏问河外援次，韩问粮期，齐问何时当先动，楚问若真与秦绝好，后头由谁补南边空出来的利口。

看完之后，秦王把竹简递给樗里疾。

“你看见什么？”

樗里疾答：“看见他们都说共局，心里却先记自家。”

秦王道：“这便够了。”

他起身走到殿门前，远远望向函谷方向。

“他们能成，是因都怕秦。可怕同一件事，不等于就肯替彼此受同一件事。”

风从殿外灌进来，把王袍下摆吹得略动。

秦王又道：“先别碰它。让它再高一些。”

樗里疾抬头：“大王不趁此时先压一压？”

“压得太早，他们反而会更紧。”秦王道，“让他们先自己尝尝持局是什么滋味。等每个人都觉得那面旗已经压到自己肩上时，哪一处最先喊重，哪一处便是日后最好下手的地方。”

樗里疾领命。

殿中灯火不旺，只照亮了案上的几处地名。河外、上党、临淄、郢都、蓟城，各自隔着千里，却都被同一只手慢慢记进了竹简里。

那天夜深时，苏秦还在邯郸改盟文。

同一夜，咸阳也还亮着灯。

邯郸那盏灯下，改的是一句句如何把众人先拢在一处；

咸阳那盏灯下，记的却是一条条众人各自最舍不得放开的地方。

四更过后，赵相府门外起了风。

纸窗被吹得轻轻一响，苏秦抬手按住案上那枚相印。

另一边，秦王在卷尾写下两字：

“待裂。”
