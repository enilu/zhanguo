# 第十七回 众国相安只凭时势 一人持局未得长宁

第十七回里，合从局面看上去反倒比前几回更平。

文书照例往来，诸侯表面守约，馆驿间虽仍有轻重试探，却少了先前那些当场几欲顶死的激烈冲撞。边报未有大变，秦也未骤然东压，故众国都还肯维持这一层“同在一旗之下”的秩序。

若只看表面，似乎这一局已经渐渐稳住。

可真正看得深的人都明白，这份相安主要不是因为大家终于信了彼此，而是因为秦还在外面，外压仍旧重着。共同之患还没有轻到可以放手各算，所以那些旧隙、旧疑、旧账，才暂时还被压在下面。

也就是说，压住不安的，是时势，不是信任。

魏韩私下仍会反复盘算：若秦下一步更逼，强国是否真会如盟书所言那般应之？齐楚也仍旧在心里记着，自己究竟该把这盟当作可久持之局，还是当作眼前一段不得不借的名势。赵虽居其中，亦未曾真正忘记“自己不可被天下借得太过”。燕国则时不时仍要提醒众人，北方之势里不能总把燕看作附属陪衬。

这些都在，只是暂时未被外力拨到最响处。

苏秦仍在持局。

可到了第十七回，他做事的样子，已经越来越像一个不断修补边角的人。今日才把一份言辞偏重的赵国文书改平几分，免得远国生疑；明日又要劝魏韩不必因一点边上风声就立时疑盟；后日还得安抚燕使，让它知道自己并未在北方线里被自然挤开。

这些事都不大，却没有一件是可不做的。

正因局面表面还稳，修补便更要细。真等裂口大到人人看见时，再去补，多半就晚了。

一日傍晚，苏秦与数国使者共饮，席间大家都守着分寸，言笑也算平和。席散之后，赵国那位年轻议者送他出门，低声道：“这几月看来，诸国总算还能相安。”

苏秦望着馆外沉沉暮色，道：“相安是真相安，却未必是长安。”

那人一怔：“何意？”

苏秦道：“若是互信得来之安，便可稍宽些心；如今这安，多半是因为秦还在外头压着，谁都知道先乱对自己更坏。这样的安，一旦有人真懂得往旧隙里拨一拨，便未必还能这么安。”

这话说得极轻，却像把第十七回该有的寒意全点出来了。

风还没有来。

可真正懂局的人，已经能闻到远处隐隐有一股将来的折风味道。

同一时节，咸阳宫中有老臣与边臣夜议。边臣道：“东方如今表面甚稳，未见将散之象。”老臣却笑了笑：“稳，未必就是牢。若稳只是因外压未去，一旦有人善于拨动它们各自最怕那一点，稳便可能最先从最平静处起裂。”

边臣问：“拨哪里？”

老臣道：“强国拨其矜，弱国拨其惧，近国拨其急，远国拨其慢。诸侯若只是凭时势暂时相安，便总有可拨之处。”

这番话并未说破下一步是谁来做、何时去做，却已把“破局的方法”在逻辑上摆了出来。

可第十七回不能让同盟因此就显得空壳。

因为苏秦此时仍旧有效。他的威望尚在，分寸也还在，诸侯也都还知道眼下不是自乱的时候。正因如此，局面才越发呈现出一种最危险的状态：它还没破，它还在用，它甚至看上去还比从前更会维持表面平衡；可真正的寒意已经先于破局到来。

这一回末尾，苏秦深夜独坐，案上文书已比前些时少了些，庭外风声也不急。他看着烛火，心里却没有多少宽松。因为这份静不是终得安宁，而更像大风将至前林叶先暂时停住的那一瞬。

局面还在。

他自己也还在。

可他已比任何时候都更清楚：合从最危险的时候，未必是它明着破的时候，而往往是它表面仍稳、人人都说尚可维持，而那些真正看得深的人却已经从这份平稳里先闻到裂音的时候。
