# 第十七回 众国相安多凭秦在 一人持印只恐风移

入冬之后，东方反倒平了几分。

没有大战，没有谁忽然翻案，连馆驿间那些动辄顶到脸上的火气，也像被寒气压住了。赵国照例收北边急报，魏韩照例补河外防务，燕人照例把北线军簿誊得整齐，齐楚的使者来了去、去了又来，说的也多半还是那几句“共持合从”“不可自乱”。

旁人看着，难免要生出一点局面渐稳的错觉。

赵王也这样问过苏秦。

那日暮色压城，赵王留他在偏殿用茶。殿外宫人添炭，火盆里时有细响。赵王翻着近来各国回书，道：“近几月似乎都还守得住。秦不东动，诸侯也都肯照盟书行事。你这一步，总算站住了吧？”

苏秦没有立刻答。

他看着赵王案上的几份回书，最上头那封是魏国来的，说河外近来无警，只请仍照旧约；压在下面的是燕国的，请明岁春使共议北线军次；最末一封来自齐国，字写得最周正，口气也最软，只说“愿观后势，再议深浅”。

“守得住，”苏秦道，“可还不算站住。”

赵王抬眼看他：“为何？”

苏秦伸手，把那几封书简按成一叠。

“它们眼下肯安，不是因为彼此真信了，而是因为秦还在外面。”他说，“秦若骤压，诸侯会更怕；秦若久静，诸侯便会各自松口气。怕的时候会想并力，松的时候便要先算自家。如今这份安，多半是压出来的，不是长出来的。”

赵王沉吟片刻，道：“你是说，秦若不动，反倒更难？”

苏秦点头。

“秦大动，我还能借势催众国；秦久不动，大家便要问，何苦日日绷着这一根弦。”

赵王把茶盏放下，没有再说。

他当然明白这话。赵国居中，受益于合从，却也最怕被这张网反过来拽住脚。盟在时，可借名压秦；盟若要日日真照着做，便要日日拿赵的兵、粮、面子去填。

殿里安静了一阵，只听见炭火轻裂。

赵王末了道：“寡人会尽力撑着这杆旗。”

苏秦起身一礼：“臣知道。”

他出宫时，天已全黑。

宫门外的风比前几日更硬，车辕上积了一层薄霜。随行门客替他掀帘，低声笑道：“大王肯说这话，已算难得。如今诸国都还肯认相国，外头也都说合从已定。”

苏秦上车后，没有立即放帘。

他望着宫墙外一盏盏被风吹得忽明忽暗的灯，过了一会儿才道：“最怕的就是‘已定’这两个字。”

门客一怔。

苏秦道：“局若真定了，便不用日日补。如今大家只看见旗高，看不见旗杆下头有多少人是各自扶着自己那一边。”

车轮滚动，碾过街面冻硬的泥。

回到馆中后，苏秦又看了一遍各国近来的细报。哪一国添了新税，哪一国先修边仓，哪一国嘴上说共应、手里却先把自家近边壮丁多编了一层，他都看得很细。

看得越细，心里越沉。

因为这些动作并不直接坏盟，甚至多半都说得过去。可每一件都在告诉他，同盟眼下能安，靠的不是彼此之间忽然生出了多少真信，而是秦仍在那边，刀还悬着，所以谁都不敢先把那面旗扔下。

一旦风向真变，便要见底。

同夜，咸阳一处偏殿里，秦王也在看东方回报。

张仪的名字，第一次被正式写上了简首。

那人尚未入殿，先送来的，是他这些年辗转诸国的粗略履历。楚中受辱，赵门受拒，洛阳贫居，辞气锋利，好论天下利害，不甚顾廉名。

秦王看完，把竹简递给樗里疾。

“你觉得此人如何？”

樗里疾答：“若只看行迹，未得其时；若只看字里行间，像一把久藏而未出鞘的刀。”

秦王问：“刀最怕什么？”

樗里疾道：“怕磨坏了，却无处试。”

秦王笑了笑。

“那便给他一处。”

次日，函谷传骑西出。

再过数日，一骑瘦马从西路风雪里慢慢进关。守关甲士验过木牌，没有多问，只让开道路。马上的人衣袍旧，神情却很静，像是早知自己该往这里来。

咸阳宫里，秦王第一次见到张仪时，没有先问计，也没有先问怨。

他只问了一句：“你师弟如今佩六国相印，名震山东。你来秦，要拿什么见寡人？”

张仪抬起眼，答得很平。

“臣不拿名来见大王。臣拿他们那面旗底下各自不肯给人的地方，来见大王。”

秦王盯着他看了一会儿。

殿中无人出声，只有风把灯焰吹得细细偏了一下。

张仪又道：“苏秦能把六国先并到一处，已是大才。可并在一处的人，不必人人都肯往前走同一步。臣愿为大王做的，不是与六国同争一口气，而是叫他们自己去问：这口气，为何总要我先出。”

秦王听完，慢慢坐直了身。

“好。”他说，“你先去看舆图。”

那夜，张仪站在咸阳殿中，看了很久那幅天下图。

灯影落在图上，把楚、齐、赵、魏、韩、燕一一照亮，又一一沉下去。张仪的手没有先落在赵，也没有先落在齐，最后停在了楚地北缘。

他没有说话。

只是指尖在商於一带，轻轻叩了一下。
