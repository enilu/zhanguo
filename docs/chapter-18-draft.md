# 第十八回 合纵成名一时震世 连横伏脉暗处生风

第一卷至第十八回，若从高处回看，苏秦这一段路走得极险，也极高。

第一回里，他不过是赵地官道上一骑瘦马的寒士；第二回、第三回之间，他先后在赵门、秦门外吃尽了门槛的冷风，回洛阳时几乎已是“天下看不见他，他也还未有资格让天下看见自己”的人。可自第四回起，他再入诸侯之门，先让赵廷不能轻看，继而因国设辞、连起北方、推动盟议、压住众争、佩六国相印。到第一卷收束时，那个昔日不得其门而入的布衣，已真成了天下共知其名的纵横之士。

这一段人物弧线，是完整的。

合从之势，也并非白忙一场。

这一卷里，东方诸侯确实因苏秦与合从重新提起过一段气脉。赵不再只想着独当，燕不再甘于局外，魏韩也知道不能只是闭门求近安，齐楚纵未尽服，也再不能假作北方之势与己无关。士人风气因此一振，诸侯之间也终于有了一条可以暂时共声应秦的线索。

若说第一卷写的只是虚热，那便轻了苏秦，也轻了这个时代。

因为合从在这一阶段，确实让东方得了一时之气、缓了一时之压、提了一时之名。这一切都是真的。

可同样真实的，是它从来没有真正长成一副齐整的骨头。

强国之矜未去。赵要居中，却不愿多受；齐楚肯入局，却都不愿轻失自己的主位与从容。小邦之惧也未去。魏韩附盟最深，却也最怕自己先被推去挡锋；燕虽借此入局，心里仍常怕自己久而久之又被挤回边角。守约之难、分责之难、远近缓急之难、人人都想借盟而又不肯尽失旧算之难，第一卷后半早已一层层写了出来。

所以第一卷若要说透，便须承认：

合从的成功，不在于它真把六国变成了一家；

合从的成功，只在于它在秦压之下，真的把六家暂时压到了一面共同对西的旗下面。

而它的隐忧，也正藏在这成功里。

因为旗既立得高，旗下面那些原本就没消失的旧隙，便也在高处的风里显得更清。越是名震天下，越要日日靠苏秦去修补；越要靠他去修补，便越说明这同盟并不能自生其骨。

秦也正是这样看的。

这一卷里，秦没有急着乱撞上来，没有因东方佩印、盟议、声名大起就先失其静。它所做的，是看，是记，是等。看苏秦如何把诸侯拖到一处，记六国各自不同的裂法，等这张网在自己受力足够久之后，把最容易被拨动的地方显得更明。

这不是轻视。

恰恰相反，这是一种更可怕的重视。

真正强大的对手，往往不是在你刚盛时先扑上来，而是在你最盛、最耀、最容易让自己也信以为已经成骨的时候，仍旧静静站在远处，把你所有未缝合好的地方一一看清。

所以第一卷收束到此，最稳的一笔，不该写成“苏秦已败”，那样不公；也不该写成“合从已定，东方自此可安”，那样太浅。

最稳的一笔，应是：

苏秦与合从确实把天下的气往东方重新提了一次。

这一提，提得诸侯振作，提得士人动容，提得六国一时真像有了可与秦争衡的共同方向。

可也正因为提得太高，那些原本藏在六国之间的旧算、旧隙、旧疑，反倒在更高的风里更容易发出未来将裂的声响。

卷末时，邯郸城中仍有佩印后的盛名余波，苏秦的威望也仍在众国之上。可若极静时去听，已经能听见另一股更细、更冷的风，从看不见处穿林而过。

风还没有到。

林却已先动。

第一卷便收在这“盛势仍在眼前，而拆局之脉已在暗处生风”的一刻。
