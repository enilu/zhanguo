# 第十八回 佩印余声仍满邯郸 商於空信已出咸阳

年关将近，邯郸城里还留着佩印后的热气。

街市上酒肆仍有人传说苏秦一日佩六印的事，说到兴起处，连筷子都要敲桌。外郡来的士子也爱往相府门前站一站，像是只要多看两眼，便能把这场震动天下的事记得更牢一些。

苏秦却比先前更少出门。

他大半时间都在馆里见人、回书、改约。名一旦上去了，跟着压下来的，便不是喝彩，而是日日都要落笔的次序。哪国使先回，哪封书先答，哪一处边急须先请赵王发话，哪一处只是馆驿间的口舌，不值得把整盘局也拖进去，他全要亲自掂量。

一日夜里，燕使辞归，赵臣亦散。

相府前堂终于空下来，只剩案上几卷还未封口的回书。门客轻手轻脚收拾残灯，见苏秦还坐着，便道：“相国今日可算稍松一口气了。”

苏秦看着案上那几卷书，没有点头。

“气能松，”他说，“手不能松。”

门客笑道：“如今六国都认你，外头又都说合从已成，难道还怕它一夜散了不成？”

苏秦抬起眼，看了那门客一会儿。

“不会一夜散。”他说，“真要散，也不是先倒在大家都看得见的地方。”

他说完，拿过一卷楚书。

那是郢都新送来的，字里话外都在问南线若与秦绝好，诸国后续如何共应。苏秦看着那封书，指腹在封口已裂的泥痕上轻轻一抹，像是在想什么。

他当然知道楚怀王肯应盟，已算难得。可楚是大国，大国肯来，常常也要先问：我进来之后，别人拿什么补我放出去的那一截利。齐也是一样。越是强国，越不会只凭一句“大义”就把自己整个压进来。

所以合从成了，并不等于事事都成。

至多只算，先把六家逼到了同一面旗下面。

夜深之后，苏秦把最后一封回书封好，起身走到窗前。

邯郸城里还有灯。

那些灯在冬夜里看去很暖，也很远。它们照着赵国宫阙，也照着他这座相府；照着满城仍未散去的声名，也照着案上那些一天不能停的补缝。

他忽然想起自己初入赵时那匹瘦马，和官道旁被风吹斜的野草。那时候他只是想，六国若再各自自安，终有一日要被秦一国一国压过去。如今合从已起，自己也终于站到天下都看得见的地方上，可真正压在肩上的东西，却比那时想的还重。

门外忽有急足。

值夜门客快步入内，低声道：“咸阳新有一人得见秦王，连着数夜都在宫中。”

苏秦转过身：“谁？”

“张仪。”

屋里一时很静。

那门客不敢再往下说，只把自关中转来的简报双手奉上。纸极薄，字也不多，只记“张仪入秦，日夕见王，疑将出使东方”。

苏秦接过那卷书，良久未动。

窗外的风正从廊下穿过去，吹得檐铃细响。那声音并不大，却偏偏叫人觉得冷。

他没有问更多。

有些事，名字一出现，后头的路便已能看见大半。张仪既入秦，便不会只是去做个客卿闲人；咸阳肯连夜见他，也不会只是听他谈旧学旧识。

门客低声道：“相国要不要即刻再致书楚、齐，提醒诸国提防？”

苏秦把简报慢慢卷起。

“提醒是要提醒。”他说，“但不能写得像惊乱。越是这时候，越不能让各国觉得我先慌了。”

他走回案前，重新铺开一卷空简。

起笔前，他却停了停。

因为他很清楚，张仪若真要动，不会先从人人看得最紧的地方下手。他会去寻那种看似还在盟中、实则最容易被一句利害拨动的地方。

而这地方，多半不在边上，在人心里。

同夜，咸阳宫里，张仪第一次把话说到了明处。

秦王、樗里疾与他站在舆图前，灯火把商於一带照得发亮。张仪没有先说齐，也没有先说赵，只拿手指在楚北点了点。

“为何是楚？”樗里疾问。

张仪道：“赵近而多疑，魏韩近而先怕。齐虽可动，却太爱自重。唯楚，地大、气盛，既不肯久居人后，又最爱见现成之利。若能先叫楚离席，其余五国便都会回头看一眼自己脚下。”

秦王道：“怎么叫它离？”

张仪抬眼，脸上没有什么笑意。

“给它一块它想先拿在手里的地。”

樗里疾问：“真给？”

张仪道：“话先给得大些，字再写得小些。人只要先动了心，后头便会替自己把许多不该信的地方也信过去。”

殿中静了片刻。

秦王望着图，没有立刻点头，也没有摇头。

过了一会儿，他只道：“你去。”

张仪一礼而退。

他走出殿门时，咸阳夜色正深，宫道上结了一层薄冰。甲士的靴声从远处传来，一下一下，极稳。

另一边，邯郸仍有未灭的灯。

苏秦提笔，先给楚国写了一封极平的书，只说秦廷近日新用张仪，词锋甚利，请楚王慎看其辞，勿先为小利动心；写完之后，又给齐国补了一封，仍旧不提“破盟”两字，只请齐继续守定旧议，不为关中一时新辞所诱。

两封书封好时，东方已快见白。

院中老树被风吹得轻轻发响。

苏秦站在廊下，看着差役把急书带出门去。

邯郸城里，佩印后的余声还在。

可他已经听见，另一阵更细的风，正从商於那边起身。
