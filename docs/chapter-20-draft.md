# 第二十回 张仪出楚巧设六百里 怀王贪利初动合纵心

## 场景一：使楚

秋末冬初，江风已带寒意。

张仪一行数十骑，自咸阳出发，沿汉水南下，历经半月，终于抵达楚都郢城。

这是他第二次来楚国。

上一次，他是令尹昭阳门下的一个贫寒门客，因被疑窃玉而遭鞭笞数百，狼狈逃出。那日他浑身是血，被人扔在昭阳府外的泥地里，连爬带滚才捡回一条命。

如今，他是秦国的使臣，奉秦王之命，带着连横之策，来见楚怀王。

"先生，"随行的秦卒低声道，"前面就是郢城了。"

张仪勒住马，望着远处那座熟悉的城池，嘴角浮起一丝冷笑。

昭阳，你当年鞭笞我的时候，可曾想过会有今日？

"进城。"他淡淡地说道。

殿中群臣闻言，一片哗然。

楚怀王也是一怔。他没想到张仪会如此直白地提起这段往事。

"你……你不恨楚国？"楚怀王问道。

张仪微微一笑："张仪今日来，不是来谈旧怨的。张仪是来谈利的。"

"什么利？"

"大王可知，秦国最想要的是什么？"张仪反问道。

楚怀王沉吟片刻："东出？"

"正是。"张仪点头，"秦国想要东出，可函谷关外，六国合从，秦虽强，却也不敢轻举妄动。可大王想过没有，这合从之盟，对楚国又有什么好处？"

"合从乃六国共约，同力拒秦，楚国身为大国，岂能背盟？"楚怀王皱眉道。

"背盟？"张仪笑了，"张仪从未让大王背盟。张仪只是想让大王看清，这盟约之下，楚国得到了什么，又失去了什么。"

他上前一步，声音低沉而富有磁性："合从之后，楚国得到了什么？得到了一个与秦为敌的名分。可实际上呢？秦若东出，最先受锋的，是韩、魏，不是楚。秦若南侵，最先受锋的，是赵、燕，也不是楚。楚国地广人众，远在南方，本可坐观成败，为何要为了一个与己无关的盟约，把自己绑在六国的战车上？"

楚怀王沉默不语。

"再说，"张仪继续道，"合从之后，楚国失去了什么？失去了与秦通商之利，失去了与秦联姻之好，更失去了……"他顿了顿，"扩张之机。"

"扩张？"楚怀王眼中闪过一丝光芒。

"正是。"张仪察言观色，知道楚怀王已经动心，"大王可知，商於之地？"

"商於？"楚怀王想了想，"那是秦地，在丹水、析水之间，方圆六百里。"

"大王好记性。"张仪笑道，"秦王有言，若楚国愿与秦结盟，背弃合从，秦国愿割商於之地六百里，归楚国所有。"

"六百里？"楚怀王猛地站起身，眼中满是震惊与狂喜。

商於之地，六百里！那可是秦国腹地，沃野千里，得之则楚国北境可大大扩张，更可威胁韩、魏，称霸中原！

"此言当真？"楚怀王颤声问道。

"秦王亲口所言，张仪岂敢欺瞒大王？"张仪从怀中取出一块玉牌，"这是秦王信物，大王可验。"

楚怀王接过玉牌，仔细端详，确系秦王之物。

殿中群臣此时已是一片骚动。

## 场景三：朝议

"大王，"令尹昭阳出列奏道，"此事万万不可！"

张仪转头看向昭阳，眼中闪过一丝冷意。正是此人，当年将他鞭笞数百，几乎致死。

"为何不可？"楚怀王问道。

"合从之盟，乃六国共约，楚国若背之，则失信于天下。且苏秦佩六国相印，合从之势正盛，楚国若此时背盟，岂不与天下为敌？"昭阳急道。

"令尹此言差矣。"张仪淡淡地说道，"张仪从未让大王背盟。张仪只是让大王选择：是继续守着一个空有其名的盟约，还是得到一个实实在在的好处？"

"六百里之地，岂是空言可得？"昭阳冷笑，"张仪，你当年在我门下为客，因窃玉被逐，如今做了秦国的说客，便想以虚言欺我楚国？"

张仪看着昭阳，忽然笑了："令尹还记得当年之事？张仪也记得。当日令尹以张仪贫寒，便疑张仪窃玉，鞭笞数百。可令尹可曾想过，那块玉璧，究竟是张仪所窃，还是另有其人？"

昭阳一愣，脸色微变。

"当日之事，张仪不想再提。"张仪转向楚怀王，"大王，张仪今日来，不是来报私怨的。张仪是来谈利的。商於六百里，秦王诚意相赠，大王若不信，可遣使随张仪回秦，当面与秦王确认。"

楚怀王沉吟不决。

这时，大夫屈原出列奏道："大王，臣以为，令尹所言有理。合从之盟，乃六国共约，楚国若背之，则失信于天下。且秦乃虎狼之国，其言不可信。商於六百里，恐是诱饵，大王不可贪小利而失大义。"

张仪看向屈原，心中暗暗点头。此人倒是清醒，可惜……

"大夫所言，张仪不敢苟同。"张仪说道，"何谓大义？守一个空盟，让楚国错失扩张之机，是大义？还是得到一个实实在在的好处，让楚国百姓安居乐业，是大义？"

他转向楚怀王，声音诚恳："大王，张仪斗胆问一句：苏秦以合从联六国，可曾给楚国带来一寸土地？可曾给楚国带来一分实利？没有。他带来的，只是一个与秦为敌的名分。可如今，秦国愿意主动割地六百里，与楚结盟，这是实实在在的好处，大王为何要拒绝？"

楚怀王心动了。

六百里之地，那是实实在在的土地啊！

"张仪，"楚怀王说道，"若寡人与秦结盟，秦何时割地？"

"大王只需派一使者，随张仪回秦，与秦王歃血为盟，商於六百里，立刻交割。"张仪说道。

"好！"楚怀王拍案而起，"寡人便派使者随你回秦！"

"大王！"昭阳和屈原同时惊呼。

"不必多言！"楚怀王挥手，"六百里之地，岂能错过？"

## 场景四：消息

邯郸城中，苏秦正在馆舍中处理文书。

一个门客匆匆进来："苏相国，有急报从楚国来。"

苏秦抬起头："说。"

"张仪出使楚国，以商於之地六百里为饵，说服楚怀王背弃合纵，与秦结盟。楚怀王已派使者随张仪回秦，准备歃血为盟。"

苏秦的手指微微一颤。

张仪，你终于出手了。

"知道了。"苏秦的声音依然平静，"退下吧。"

门客退下后，苏秦独坐案前，久久未动。

他知道张仪会出手，只是没想到，出手如此之快，如此之狠。

商於六百里，那是秦国的腹地，秦王怎么可能真的割让？这分明是一个诱饵，可楚怀王却贪利上钩了。

"楚怀王啊楚怀王，"苏秦喃喃自语，"你这一动，合从便裂了。"

他提起笔，在竹简上写下几个字：

"连横既起，合从将裂。"

写完之后，他放下笔，望向窗外。

窗外，北风呼啸，卷起漫天黄沙。

第二卷的风，已经开始吹动了。
