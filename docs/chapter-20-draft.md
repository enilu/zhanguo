# 第二十回 咸阳殿上盟先成礼 商於案前字忽收锋

景鲤入咸阳时，先看见的是宫门前那两列甲士。

不多，也不喧。

甲叶冷得发青，人在风里站着，像一排钉死在地上的铁。景鲤从车中下来，忽然就觉得这座城不像是在等着交地，倒像是在等着把所有话都按着早排好的次序说完。

张仪在旁引路，神色平平。

自出郢都之后，他便少言。景鲤起初还问过两句商於地界，张仪都答得不急不缓，只说“入秦之后，当于大礼中定”。话说得有余地，却又总叫人不好追得太狠。

到了正殿，秦王高坐。

国书先呈，礼官唱名，秦楚两边依次进退，丝毫不错。景鲤原本想在奉书之前再把地界问死，可一层层礼走下来，鼓声、赞声、升席、献玉，全都挤在一处，竟把最该问清的那一句压到了后头。

待歃血器具摆上来，秦王才道：“楚王肯解旧疑，寡人甚喜。两国若和，于彼此皆利。”

景鲤忙称是。

张仪立在一侧，适时接了一句：“大礼既成，余事自可按书而行。”

景鲤心里一松。

这“按书而行”四字，在此刻听来极稳。既有盟书，既有大礼，商於之事便像已从口头落到了纸上。

他按下手印时，甚至已在想回郢都后该如何报喜。

可礼一成，殿中气氛便忽然冷了半分。

景鲤收手，拱道：“敢问大王，商於地界何日交吏？”

秦王没有立刻答，只把目光移向张仪。

张仪走上前，取过盟书，当众展开。

“今日即可交。”他说。

景鲤一喜：“以何为界？”

张仪把书卷略抬高了些，语气还是那样平。

“自某亭至某里，凡六里之地，皆归楚。”

景鲤整个人像被什么东西当胸打了一下，愣在原地。

“六里？”

殿中静得连帛角轻响都听得见。

张仪点头：“盟书在此，大夫可自验。”

景鲤一步上前，把书接过来，手指急得都有些发抖。墨迹新鲜，字却分明，写的正是“六里”。不是“六百”，没有漏笔，也没有余地。

他猛地抬头，脸色一下子白了。

“张仪！你在郢都明明说的是六百里！”

张仪并不退，只道：“臣说的是商於之地，今日书上所载，也是商於之地。大夫方才亲手按印，难道礼成之后，楚使反倒不认书了？”

景鲤几乎要把那卷书攥裂。

他终于明白，从郢都到咸阳，自己走的根本不是验地的路，而是当众按下这份书的路。礼数一层层压下来，压得人没法在最该开口的时候翻脸；等到礼成，再要翻脸，便成了楚使自己失仪。

他眼里发红，转向秦王：“秦以此欺楚，大王不怕后患？”

秦王这才开口，声音很淡。

“盟已成，书在此。楚若重书，便知今日该看什么；楚若只重怒，寡人也拦不住。”

这一句更狠。

它把景鲤想说的话都压成了意气。

殿边秦卒略向前移了半步，甲片相碰，响得并不大。景鲤手压剑柄，终究没有把剑拔出来。拔了，他先死；不拔，回郢都之后死的便是脸面。

他只能把书卷死死攥住，退后一步。

张仪这时又补了一句：“大夫回去后，不妨告诉楚王。若为地来，六里也是地；若为好来，盟已先成。至于要不要翻脸，在楚，不在秦。”

景鲤听见这话，反倒平静下来。

平静不是因为怒消了，而是因为怒已大到一时说不出来。

他收了盟书，向秦王一礼，动作僵得像木人。礼毕之后，他再不多看殿上一眼，转身便走。宫门外渭风扑面，冷得像把人从头到脚刮了一遍。

车队出城时，没有一人说话。

车中那卷盟书就搁在景鲤膝上，薄薄一卷，却沉得像压着一块石。

咸阳宫里，景鲤一退，殿中诸臣才重新抬头。

秦王看向张仪：“六里，太薄了些。”

张仪拱手：“正要它薄。薄到足以伤楚王脸面，却又不至让天下只骂秦舍不得地。楚若忍下，臣不过得一场巧；楚若忍不下，这纸书便会逼它自己往前多走一步。”

秦王问：“你料它会走哪一步？”

张仪道：“怀王爱名，也爱利。先动过利心的人，受辱之后最难回头。他若肯先认自己错在前头，楚或还能缓；他若只想先把这口气讨回来，诸侯便都会看见，楚是为自己那一口气起兵，不是为天下起兵。”

樗里疾站在一旁，没有作声。

他知道，这一刀最厉害的地方，不在骗地，而在借楚王自己的性情，把后头那半局一并推了出来。

郢都接到景鲤回报时，宫里正摆着北境旧图。

楚怀王连案上的木尺都已取来，方才还指着图与近臣说，商於若得，何处可设守，何处可添仓，何处又可叫韩魏不安。景鲤入殿后，先伏地不起。

楚怀王见状，心里先沉了半截，却还强按着问：“地界可验明了？”

景鲤把盟书双手举过头顶，嗓子发哑。

“商於非六百里，止六里。”

楚怀王像没听懂。

“六里？”

景鲤不敢抬头，只把书再往前送了一寸。

楚怀王接过来，低头看了三遍。每看一遍，脸色便更沉一分。殿上所有人都不敢出声，只听见那卷竹简在他手里被捏得咯咯轻响。

下一刻，楚怀王猛地把书掼在地上。

“张仪欺我！”

那声音震得殿梁都像在回响。

昭阳上前一步，还未开口，楚怀王已转身下阶：“传令整军！寡人若不问秦，天下便真以为楚可如此轻侮！”

屈原闭了闭眼。

他知道，这时候最该说的是“先缓”，可也知道这时候最难说出口的，偏偏也是“先缓”。

殿外北风直灌进来，把地上那卷盟书吹得微微卷边。

六百里没有来。

先来的，是一场谁都拦不住的怒。
