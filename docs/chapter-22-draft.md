# 第二十二回 诸侯观楚先收两手 昭阳临秦只见冷垒

楚军北上之后，沿路州县都把鼓打得很响。

谁都知道这是去雪耻的兵。军中过处，百姓夹道看得也多，话里多半带着一股“楚不可欺”的热气。将校们听得心更急，前营后营都盼着赶紧碰上秦军，先打一场像样的仗，把郢都那口气顶回去。

昭阳却越往北走，脸色越沉。

他每日看三样东西最多：粮车有没有跟上，传骑多久能来回一趟，前锋离后营究竟隔了几层山道。楚王与朝中主战之臣等的是“速见其效”，他在军中看见的却是另一件事：兵可以快，辎重和命令却快不了这么齐。若一支军只是怒得快，后头许多本该跟着一起到的东西便会慢上一截。

到汉北时，探骑终于回报，说秦军已据前方数处高垒，营火不多，出哨亦少，看上去像是在等楚先撞上去。

帐中年轻将校听了都说秦怯。

“怯？”昭阳抬起头，“若真怯，便不会只占高处，不占空地。”

他把木尺压在军图上，沿着几处垒口一一比过去。

“它不求大胜，只求你先失步。”

众将一时无话。

昭阳随即下令：“前锋可试，不可贪。中军跟粮车，左右翼各守山口。谁若见敌退便争功，我先斩谁。”

军令下得很硬。

可他心里清楚，等真正交上手，能把这道令守住几成，还得看后头郢都催得有多紧，看前头那一阵血热又有多盛。

同一时日，张仪已自大梁转去临淄。

临淄朝会上，齐王听完秦使来意，先笑了一声。

“张子是来劝寡人不救楚？”

张仪拱手：“臣不劝大王做不愿做的事。臣只是替大王把一笔账先摆开。楚王前日求地，今日求兵。大王若出兵，是为合从，还是为替楚先把它自己惹出来的后半截接上？”

齐王把玩着案上的玉镇，没有立刻答。

这话他心里本来就在算，只是平常不好说得太直。齐是大国，最爱听的从来不是谁来求义，而是谁来替它证明“持重”二字就是见识。

田文在旁慢慢开口：“秦先失信，这话总不能一笔带过。”

张仪看向他：“自然带不过。可大夫若是齐臣，最该先问的，难道不是齐兵一旦南下，耗的是谁的米、谁的卒、谁的春赋？”

田文听罢，没有立刻驳。

因为这话脏是脏，却正戳在要处。

齐王笑意更深，口里却说得很稳：“寡人非不忧楚，只是大兵一动，不能只看眼前一怒。”

张仪一礼：“大王持重，正是山东大国气象。”

这句话送出去，齐王心里便更顺了。

一旁田文看着张仪，心中却越发发冷。他看得很清，这人不是来替秦争一场辩口，而是来给每个不想出兵的人，送一层更好听的皮。

有了这层皮，观望便更稳，观望一稳，苏秦再来时就更难把人往前推。

果然，齐廷最后只回了楚一封“深以为憾、容观后势”的书。

魏、齐如此，韩国更紧。

韩国离秦太近，朝廷里人人都怕自己这边稍一轻动，秦锋便先压过来。张仪甚至不必亲自久留，只把一句“楚若再败，河外何处最先受秦意”送进韩廷，韩国便已把所有“援楚”之议压到了末座。

燕国远在北边，本就只是隔水观火。

短短半月，诸侯朝堂之上便都生出同一股冷气：楚是可怜，可可怜不等于值得自己先把兵推出去。

这股冷气，很快也吹到了邯郸。

苏秦连着几日见赵臣、魏使、齐使，听到的都是差不多的话。有人说楚先背盟，有人说楚先动怒，有人说大兵不可轻发，也有人什么都不直说，只把“需再观一阵”四字说得极圆。

那日夜里，他在馆中接见赵国数名议臣。

一人道：“相国常言共局。可楚先图商於，如今又先起兵。赵若立刻为它受兵，岂不等于告诉天下，谁先坏盟都无妨，只要回头喊一声，赵还得替他补？”

苏秦坐在灯下，听完之后，过了一会儿才道：“今日若人人都只记楚先有过，不记秦已试出这一手，往后秦便知道，拿一国君主的性情一拨，就能看见其余五国肯不肯真动。”

那人又道：“可这不还是楚自己先给了它把柄？”

“是。”苏秦没有否认，“可合从若只在人人都没犯错时才算盟，那便不是盟，是逢场作戏。”

话说出来，屋里静了片刻。

众人不是不觉得这话有理，只是谁都不想做第一个替“有理”付兵、付粮、付边险的人。

散议之后，门客送人出门。

回屋时见苏秦还未睡，便低声问：“相国还要不要再去齐？”

苏秦把案上一封齐书递给他。

“你看这句，”他道，“‘愿观后势，再定深浅。’写得多好。谁都挑不出错，谁也别想从里头拿出一句真应诺来。”

门客看完，苦笑不语。

苏秦道：“张仪不是叫他们立刻倒向秦。他只是先替他们把不肯出手的话说顺了。人一旦说顺了自己退半步的理，后头就会越退越自然。”

而前线上，秦军也确如昭阳所料，冷得厉害。

楚军先试了两处垒口，秦军都只是应了几轮箭石，随即退入后垒，既不追，也不乱。那样子不像怕，倒像是在一点点试楚军的躁处在哪一段、快处在何处。

申时以后，前锋将校请战愈急，都说再压一压便能逼秦露底。

昭阳亲自登高望阵，只见远处秦营灯火稀疏，垒线却收得极整，像一排伏在暮色里的冷牙。

他盯着看了很久，末了只道：“传令，今夜不进。”

将校们先是一愣，随即有人急道：“令尹，军中士气正盛，若又停下，只怕伤锐。”

昭阳回身看他：“士气是用来打阵的，不是用来替人发火的。”

那将领被堵得一时说不出话。

营外北风越刮越大，吹得旌角乱响。秦垒上没有多少灯，却偏偏叫人觉得那片黑里什么都备好了。

昭阳站在坡上，望着那一线冷垒，忽然想起郢都那卷写着“六里”的盟书。

他知道，张仪真正交到楚手里的，从来不只是那六里。

还有眼前这一整片，逼着楚不能慢、又最需要慢下来的战场。
