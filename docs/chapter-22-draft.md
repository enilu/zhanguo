# 第二十二回 楚师北上先争一口气 秦使东行再拆六国盟

郢都军令一下，楚国上下像骤然被火点着了。

前一日还在传看那份盟书的朝臣，后一日便都忙着议兵、点将、催粮。城中鼓角昼夜不歇，江汉之间的舟楫、车乘、甲仗都被一股脑往北线上推去。楚怀王要的不是一场缓缓筹算的边战，他要的是立刻让天下知道，楚国并没有被张仪一纸盟书打得抬不起头。

可真正懂兵的人，看着这股声势，心里却都不轻松。

怒兵可出，不易久持；重声势者，往往反更怕不能速胜。

昭阳被命总督此战时，领命是领了，心却沉得厉害。他很清楚，楚王此番发兵，打的与其说是秦，不如说是自己的那口脸面。脸面要得越急，朝廷就越不肯听慢话；而不肯听慢话的时候，最容易把一场本该算细的仗，打成只能硬撑的仗。

屈原送他出朝时，二人并行至殿外长廊。北风沿廊卷来，把檐下铜铃吹得细细作响。

屈原低声道：“令尹此去，先求不败。”

昭阳苦笑一声：“大王如今要的，不是不败，是立刻见胜。”

屈原道：“正因为如此，才更要先守其筋骨。楚这回若只是拼怒，便是在替张仪补完他后半局。”

昭阳停了停，回头看一眼高殿，沉声道：“我明白。可眼下朝中听得进去这话的人，已经不多了。”

楚军仍旧出发。

大纛北指，沿途郡县不敢怠慢，所过之处皆以大捷相祝。军中士气确也高涨。许多人并不真懂咸阳宫里那份盟书如何设局，他们只知道秦欺楚，王命雪耻，此战一开，人人都觉得自己是去替国争面。

这种气，可以催锋，却也最易伤身。

消息传入咸阳时，秦王嬴驷看完军报，只问了一句：“楚兵行得急不急？”

前来回报的边臣道：“急。楚军诸道并进，沿途催发甚速，似欲先声夺人。”

秦王把军报放下，笑意很淡：“那便是了。”

殿中群臣有主张乘其躁而迎头痛击者，也有主张先守关中、挫其锐气者。秦王不急着断，只看向张仪：“楚兵已起，诸侯何如？”

张仪答道：“赵不愿先出，齐乐见其耗，魏韩惧祸近身，燕隔得更远。此时若只看兵，楚尚有可战之力；若连人心一并看，它已先孤了一半。”

秦王道：“你还是要出使？”

张仪拱手：“臣正要趁楚兵未挫、诸侯未定之时，再去把他们心里那句不肯出口的话，说得更明些。”

秦王点头。

这一回，秦国对楚并不急着只靠战阵取胜。张仪前一刀试出的是楚王，后一刀要试的，便是诸侯究竟愿不愿为一个已先背盟、又怒而擅战的楚国，把自己真拖进战局。

于是咸阳一边整军，一边发使。

张仪先去的，是魏。

魏国最难受。

它离秦近，也离楚不算远；它既怕秦借楚兵之机再压自己，也怕若对楚太冷，日后被诸侯指为无义。所以魏廷这些日子议得最乱。有人主张借机重申合纵，多少给楚一点声援；也有人主张坚守封境，绝不被楚军北上拖出自己的盘算。

张仪入大梁时，魏王本不欲立见。

可张仪只递进一句话：“臣不是来劝魏背义，臣是来问魏，愿不愿替楚王之怒受秦之兵。”

这一句送进去，魏王终究还是召见了他。

殿上群臣环立，人人都知道张仪此来不安好意，可人人又都想听听他能把这件已很难堪的事说到多难堪。

张仪入殿后不多铺陈，只道：“楚王前日受臣之说，舍合纵而求私利；今日受臣之辱，又舍合议而先起私兵。魏若此时助楚，是助合纵，还是助楚王自己的喜怒？”

魏廷一时无人即答。

张仪便把话再往前推了一寸：“若诸侯共议伐秦，魏可以说是循盟；可如今楚先以私心坏盟，后以私怒起兵，魏若跟着动，便不是守盟，而是替一位失算的君主收拾后患。魏肯替楚收这后患么？”

有魏臣厉声道：“秦先失信，还有脸言义？”

张仪转头看他，答得极稳：“秦失信于楚，楚失信于诸侯。二者皆非美事。可魏今日要选的，不是谁更美，而是谁的祸更先落到魏头上。若为楚出头，秦兵便能名正言顺先压河外；若按兵不动，楚纵有怨，也终究隔着一层。利害如此，何必强说成别的？”

这番话恶毒处，不在狡，而在它把魏人最不愿承认的算盘当殿摊了出来。

魏王脸色很难看，却也没有发作。

因为张仪说的，正是魏廷多数人正在心里盘的账。

张仪离魏之后，又东向齐。

齐比魏更容易说。

临淄本就惯于远看天下起伏。齐王听闻张仪至，甚至带了几分玩味相见。他本想先看张仪如何替自己那场“六里之欺”圆场，不料张仪一开口，竟先谢齐王。

齐王失笑：“谢寡人什么？”

张仪道：“谢大王至今未轻动。天下事最怕被怒人拖着走。楚王若怒而战，齐王若亦怒而随，则山东诸侯皆失其静；如今齐不动，正显大国之定。”

齐王听到这里，心里便先受用了三分。

田文在旁冷眼看着，知道张仪又在顺着齐人最爱听的话往下走。

果然，张仪下一句便道：“楚若胜，于齐有何益？楚若败，于齐又有何损？可若齐出兵，不论胜败，耗的都是真正属于齐的兵与粮。大王以一国之实，去替楚王补一时之面，这笔账，齐人素来是不做的。”

田文终于开口：“先生这话，倒像比齐臣还懂齐。”

张仪笑道：“臣不懂齐，只懂天下人人都先顾自己。”

这句话太直，殿上反而静了一瞬。

齐王随后大笑，道：“张子之口，果然是刀。寡人虽未必尽信秦，却也不至于替楚怀王去受这股气。”

张仪这一趟临淄，不在于说服齐变向，而在于替齐的观望找一个更体面的说辞。说辞一有，朝堂便更容易安稳，齐也就更不可能在短期内被苏秦重新激动起来。

魏、齐如此，韩、燕更不待言。

韩国本就贴着秦锋，边境稍有风吹草动，最先紧张的便是韩。张仪根本不必多费唇舌，只把一句“楚兵若败，秦意或转，韩当先自固”送进韩廷，韩国朝议便立刻把“援楚”压回了最末一层。

燕国远在北方，听楚、秦相持，更多只是觉得天下又添一场南边乱局。燕自顾尚且勉强，哪还有心力为楚发兵。

于是短短时日之间，秦兵未必真胜，秦使却已先替秦国赢了一半局面。

邯郸那边，苏秦也在走。

只是这一回，他走得比从前更难。

从前他入诸侯之朝，是说“若不并力，终将各亡”；这话虽难，却站在一个大家都还愿意认的共同危险之上。如今他要说的却是“楚虽先失，诸侯仍不可坐视”，这便难得多了。因为每一国都能立刻找出一条比“共扶盟局”更贴近自家眼前利益的理由。

赵廷再议时，苏秦亲自入朝。

他没有先提楚，也没有先骂秦，只道：“诸侯今日都以为，楚先受祸，与己尚隔一层。可若人人都认这一层，合纵从今以后便再不是盟，而只是一张谁都能拿来遮一遮、真到见血时却立刻放手的皮。”

有赵臣答道：“相国此言高了。可楚先背盟求地，今又因怒发兵，赵若为它出头，岂不等于告诉天下，往后谁先坏盟都无妨，只要受秦一欺，诸侯仍得替他补锅？”

这话一出，殿中不少人都点头。

苏秦便知道，这正是眼下最硬的地方。

他沉声道：“不是替楚补锅，是替自己守底。今日若人人只论楚有过而不论秦得势，则来日诸侯与秦相接时，便都可被各个击破。诸位以为自己是在罚楚，实则是在教秦如何试下一个人。”

赵王坐在上首，听得无言。

这番话，他不是不懂。

可懂，并不等于就愿意立刻押上赵国去为楚受兵。赵国毕竟是近秦大国，它最怕的不是说错一句道义，而是走错一步全局。

于是那日朝议至终，也只落到一句“再观后势”。

苏秦退朝而出时，天色阴沉，宫门外风里尽是尘土。

门客低声问他：“赵仍不肯明应？”

苏秦点了点头，道：“不是赵不懂，是人人都懂，才人人都更会算。”

门客又问：“那这一局当真要看楚国独撑？”

苏秦沉默良久，才道：“若楚能先打一场不失其势的仗，也许诸侯还会重新算一遍；若楚一上来便挫，那张仪这条连横线，就真要一根一根穿起来了。”

而此时的前线，楚军确实已经与秦军接上了。

两边初战，都还在试。

楚军仗着兵多气盛，数路压前，想先逼出一场可夸于天下的快胜；秦军却并不贪应，只凭险据要，层层设拒，能退则退，能守则守，像是一团故意不与你硬撞、却又始终甩不开的冷铁。

楚军将校中有人不满，觉得秦军避而不战，是怯。

昭阳却知道，这不是怯，是等。

秦人在等楚军锐气先泄，等楚国内部先急，等诸侯那边的观望慢慢坐实。到了那时，楚就算还有兵，也会越来越像一支没人愿意真正靠过来的孤军。

他站在军前高处，望着远处秦垒，忽然想起屈原送他出朝时那句“先求不败”。

可这一战最难处，偏偏就在于楚王未必给他“不败”这样的时间。

军报一封封飞往郢都。

楚怀王等的不是“尚能相持”，他等的是“已破秦军”。每次看到前线只是写“攻某垒未下”“敌军退而复守”“我军推进若干”，脸色便一日比一日更沉。朝中主战者趁机鼓噪，都说楚师方盛，只差再添一把力气；屈原与少数稳重之臣却都看得清楚，越是在这种时候再催快，越可能把本就躁进的战局压成一场非胜不可的险赌。

楚怀王终于下诏，责前线“勿久持以沮军心”，催令诸将速求决胜。

诏书送到军中时，昭阳看完，许久没有说话。

他知道，这纸诏书一到，前线便再难守住分寸。楚军上下都会明白，大王已经不许他们再把这场仗打成一场稳扎稳打的雪耻之战，而是逼着他们尽快交出一个能昭告天下的结果。

他把诏书慢慢合上，只道：“传令诸营，谨慎再进。”

身旁将领听得发急：“大王已催，令尹还要慎？”

昭阳看着他，道：“正因大王催，才更要慎。你若只看眼前这一阵，觉得慢是误国；可若看后头十阵，便知道乱动一步，误的就不是一阵，是全军。”

那将领被他说得一时无言。

可昭阳自己也知道，他能压住一营两营，未必压得住整条战线；能压住今日，未必压得住明日。

更要命的是，他已经隐隐感觉到，真正向楚国逼近的东西，并不只在秦营里。

它还在赵、齐、魏、韩、燕那些尚未发兵的朝堂上；

还在一封封写满观望和推诿的回书里；

还在张仪那张看不见、却正顺着各国私心慢慢收紧的网里。

夜里，秦营灯火不多，风吹旌角。

张仪从东方诸国回到咸阳，再见秦王时，只说了一句：“楚可以再战，诸侯却已不会再轻易与它同心了。”

秦王问：“苏秦呢？”

张仪答道：“他还会走，还会说，还会一处处去补。但从今往后，他补的再不是一张正在张开的网，而是一张已经开始往下坠的网。”

秦王沉吟片刻，缓缓道：“如此，便先让楚再往前走两步。”

张仪低头称是。

他知道，大局至此，已经不是一回两回言辞能尽写的东西。可有一件事已十分分明：

楚国这一战若不能尽快打出声势，它失去的便不只是一场胜负，而是身后本来还可能勉强借到的天下之心。
