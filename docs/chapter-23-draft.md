# 第二十三回 丹阳阵前楚军先躁 土坡背后秦鼓忽翻

楚军到丹阳时，天已经连阴了三日。

道路泥重，辎车过处总要多费一层力，前锋将校却偏偏都催得紧。军中人人都知道，蓝田之前，丹阳这一战若能先拿出点样子，郢都那边便还有话可说，诸侯那边也未必就会一直冷着。

昭阳立在高处看了半日，只觉得心里越来越沉。

秦军垒得很巧。前垒不深，后垒却收得紧；左右山口都留着能退能转的窄道，看上去像是守得勉强，实则步步都在让楚军觉得再压一压便能穿过去。

帐中有将领请战。

“令尹，秦军一退再退，正该乘它立脚未稳，先撞出个口子。”

昭阳把军图上的木尺按住，没有抬头。

“它退得太肯了。”他说。

那将领不解：“退得肯，不正是怯？”

昭阳这才抬眼：“若真怯，便会乱。你看它乱了吗？”

众将一时无言。

昭阳随即发令：“前锋可进，不得脱中军。两翼只许跟，不许抢。见敌退，不得深追。”

军令传下去，字字都稳。

可战阵这种地方，最怕的不是将令不稳，而是人人都觉得自己抢的那一步，恰好还是“可控”的那一步。

次日一早，鼓声便起。

楚前军压出时，士气确实盛。甲列一开，前几回积在胸口的怒都像跟着这一阵鼓一起撞了出去。秦前垒接了几轮，果然往后退。退得不快，也不乱，偏偏更像是被楚军一下压住了气势。

最先乱的不是阵，是心。

前锋数营见秦退，口里已有人叫起“可破”。一营抢了一步，旁边营便不肯落后；前头将校一看别处旌旗已前压，也怕自己回去被说成怯，便又多催了半层。

到了午前，原本该与中军咬紧的几支前营，已慢慢拉开了缝。

昭阳在后阵望见，脸色立刻变了。

他刚命传骑前去收束，丹阳两侧土坡后忽然同时起尘。

先前一直退着守的秦军竟在此时回身。两路伏兵自侧后压下，中路鼓声也跟着变了拍子。那种声音一响，昭阳便知道秦军等的就是这一刻。

“收前锋！”他厉声下令。

传骑飞奔而出，泥点直溅到马腹。可前头已不是一句“收”便能收回来的样子。楚军这时候最难受的，正是人人看着都像还没败，只差再推一推就能压住眼前那一线秦兵。战场上最可怕的，往往便是这种“看着还像能赢”的时候。

秦军却不给它再看。

中路一合，侧翼一切，先前还像松口的垒线顿时全硬起来。楚前锋数营一下被夹进了尘里，鼓声、人声、车轴折断声混成一团。屈匄亲自领兵要去顶中路，才冲到一半，左翼又见秦旗翻上坡口。

那不是零碎应战，是早备好的收口。

昭阳连下三道令：中军顶住、右翼回护、后队开路。

每一道都发得极快，可战场上最难补的，就是前头抢出来的那半步。缝一旦裂开，再补时便要用整队整队的人去填，而楚军眼下最缺的，恰恰就是这种还没被前锋热气带乱的整队。

申时将近，丹阳阵势已坏。

屈匄所部先折，前锋几支也被逼得转不过身来。中军虽尚能勉强撑住，可两翼一乱，撑便只是在替撤退多争一点时辰。昭阳亲自勒马断后，几次把快散开的队伍往回压，最后总算保住了一截不至全崩。

保住不等于没败。

天擦黑时，丹阳坡下全是折断的戈、翻倒的辎车和没收拢净的败卒。楚军一路退下来，谁都说不出这场仗究竟是从哪一刻开始真坏的。像是人人都还觉得只多抢了一步时，坏就已经埋在那一步底下了。

昭阳下马立在坡顶，看着远处秦营重新收灯。

那边灯不多，却处处都稳。

他忽然想起屈原送他出郢都时说的那句“先求不败”。如今看来，不败这两个字，在楚王的催令、诸侯的冷眼、军中求速的火气下面，竟比求一胜还难。

败报入郢都时，楚怀王正看汉北军簿。

景鲤站在阶下，不敢抬头。殿上群臣听完“丹阳失利”“屈匄受创”“诸营退收”几句，谁都不敢先出声。楚怀王把那卷战报看了一遍，又看一遍，像总想从里面再翻出一句“尚可挽回”的话。

可战报是昭阳写的，字句都很硬。

最后只落一句：“秦已候我躁处，宜固后线，不宜再以速胜逼阵。”

楚怀王把书慢慢放下，面色沉得吓人。

“这是在劝寡人停兵？”

屈原出班，道：“这是劝大王先保楚兵。”

楚怀王霍然起身，殿中空气一下紧了。

“保兵？”他冷笑，“寡人受此欺辱，今又丹阳失利，若此时便只说保兵，天下将谓楚何如？”

屈原还欲再言，昭阳后续军报已随之入殿。

楚怀王没有再看人，只看着案上那两卷书。

一卷是六里盟书。

一卷是丹阳败报。

他盯着看了很久，忽然伸手，把两卷都压在掌下。

“传诏前线，”他说，“整军，再战。”
