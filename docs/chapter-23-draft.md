# 第二十三回 丹阳一战楚锋先挫 樗里设局秦势愈张

楚军北上之后，真正让人屏息的一战，终于落在丹阳。

此地并非关中腹心，却是两军都绕不过去的咽喉。楚若能在此打出一场快胜，前面几个月积着的怒气、后面诸侯犹疑不定的眼光，便都还有转圜；秦若能在此先折楚锋，那张仪在朝堂上、驿馆里、诸侯面前说过的那些话，便会一下子有了最硬的凭据。

所以这一战，表面上是前线攻守，实则早已牵住了关中与山东两头的人心。

楚营中，昭阳整整两夜没有安睡。

探报一封接一封送来，说秦军仍旧不与楚决大阵，只据险、设垒、退守、再引。许多楚将看得心头火起，都道秦人已怯，只差再压一压便可逼出大胜。尤其年轻将校更是请战不止，人人都想赶在第一场大捷里立名。

昭阳却始终觉得不对。

他对着地图站了很久，手指沿着丹阳一线慢慢划过去，末了只说了一句：“秦人不是在退，是在让。”

身旁楚将屈匄道：“令尹太过看重秦了。秦若真有把握，何必连退数处？我军兵锋正盛，士气亦高，再让它守下去，倒是我军锐气先耗。”

昭阳转头看他：“你看见的是秦退了几垒，我看见的是它每退一步，都退在我们最想它退的地方。我们越觉得快能追上它，它就越像在把路摆给我们走。”

屈匄仍不服：“可王命已催。若丹阳再不求一场见得着的胜，郢都那边如何交代？”

这句话一出，帐中便静了。

昭阳心里最沉的，恰恰就是这句。

秦军或许还可以慢，楚军却已经被朝中的催迫、军中的怒气、天下的目光，一层层逼得慢不得。一个本该先求稳的战局，到了这里，却被逼成了人人都想尽快见结果的战局。

昭阳终于下令：“可以战，但不可尽压。前锋试其虚实，中军固其后路，左右两翼不可脱节。凡见秦军深退，不得贪追。”

军令传下去时，每个字都很稳。

可他自己也知道，真正到了阵上，稳字最容易先被战场吞掉。

秦军那边，樗里疾立在高处土坡，看着远处楚营灯火，神色比夜色还静。

他不常亲临军前厮杀，可这并不妨碍他看得清这场仗真正该打在哪里。

有人问：“楚军来势甚盛，是否当先避其锋，再守数日？”

樗里疾摇头：“再避，楚人便要起疑了。今日要做的，不是再退，而是让它以为终于追上了。”

又有人问：“若楚不入呢？”

樗里疾道：“楚王在后面催，楚将在前面急，楚军中层层都想要一场能昭告天下的胜。入不入，有时候不在将令，而在整支军心被推到哪里。”

他停了停，抬眼望向远处，淡淡补了一句：“张仪在诸侯处替秦拆的是人心，我在丹阳替秦等的，是同一件东西落到战阵上的样子。”

次日一早，两军终于接战。

先是楚前锋压上，鼓声震野，旌旗如云。楚人确实盛，甲兵一开，整条战线都有一股要把前些日子所有屈辱一口撞回去的狠劲。秦军前垒应战不久，便渐次后撤，边退边守，既不乱，也不硬拼，只把楚军一点点往丹阳更深处引。

楚军初见秦退，果然大振。

前锋诸营本还记着“不可贪追”的将令，可阵势一顺、敌垒一松、人声一起，许多原本压在胸口的急躁便都一起翻了上来。有人高呼“秦败”，有人抢先争功，有将校见邻营推进，便也不肯落后。等到中军发觉前线已开始层层脱节时，战线已经比昭阳原先预想的更长、更散，也更难一把拽住。

昭阳立在后阵，脸色当场就沉了下来。

他知道，坏就坏在这里。

楚军不是无令而乱，而是在“看起来仍像进势”的时候，已经开始悄悄偏离了原先那条能自保的线。战场上最可怕的，往往不是突然溃，而是还觉得自己在赢时，阵脚已经先空了。

正午前后，秦军终于不再退。

丹阳两侧伏起烟尘，先前一再示弱的几支秦军忽然回身压出，与侧翼早伏的兵线一齐向中间合拢。楚前锋这才发现，自己追入的并非一块待取之地，而是一张早已留好出口、偏偏又不让你轻易退回去的网。

鼓声骤变，杀声四起。

屈匄亲领一军欲破秦中路，起初仍能撑住，可秦军这回与前几日全然不同，不再试，不再让，一旦迎上，便是按着楚军最急、最薄处猛压。两边绞在一处，土扬如雾，箭石交作，楚军先前那股借怒而起的猛劲，在这种真正需要阵形、调度、后应的硬仗里，渐渐显出不耐久战的亏处。

更坏的是后路。

昭阳原本留着的接应线，被前锋几支争功过深的部伍挤乱，等秦军两翼一收，后军再想补上去，已处处受阻。传令骑来回奔驰，尘土扑满面甲，喊得喉咙都哑了，也只是勉强止住局部崩散，根本无法再把整条战线重新拢回去。

到申时，丹阳战局已分。

屈匄所部先折，数员楚将死伤被围；其余各军见中路受压、左右不应，终于从“还能再冲”变成了“必须先退”。可退这个字，在阵还整的时候是号令，在阵已裂的时候便会瞬间变成传染。前队一退，后队便更慌；秦军又在这时压鼓齐出，逼得楚军越退越乱，最后竟成了一场从败局滑向崩局的乱撤。

昭阳亲自勒马断后，勉强收住一部分兵，却终究只能眼看着大片军阵在丹阳一线被切开、挤散、追杀。

那一日，楚军死伤极重。

军中宿将被俘者有之，甲士折损者更多。原本一路带着雪耻之气北来的楚师，到了傍晚时分，已经再找不回出营时那种要一鼓压秦的势头。

昭阳退至安全处时，天边残阳像被血抹过。

他下马站了很久，听着四处收拢败卒的号角，只觉得耳边每一声都比战鼓更沉。

这一仗最可怕的，不只是败。

而是它把楚军前些日子一直强撑着的那口气，当着天下的面，先打塌了一半。

消息入咸阳时，秦王嬴驷并未大喜，只问：“追得如何？”

回报之臣道：“已破其锋，楚军大败，俘获甚多。”

秦王点头，看向樗里疾：“你说楚还会不会再来？”

樗里疾道：“会。怀王此刻最不能承认的，便是自己怒而兴师，却换来丹阳之败。一个不肯认自己错在前头的君主，往往会把后头输掉的东西继续往里填。”

张仪站在一旁，接过话道：“而且他越要再来，诸侯便越不会替他来。因为到了这一步，楚若再举兵，在别人眼里就不只是受辱后的雪耻，而是自己不肯收手。”

秦王听罢，淡淡道：“如此，便趁此势取其更痛处。”

咸阳这边说得平静，郢都那边却已近乎凝滞。

丹阳败报入宫时，楚怀王最初竟没有说话。

他把军报来来回回看了三遍，像是总想从里面看出一句“只是小挫”之类的话。可战报写得很硬，硬到连一点可供自欺的空白都没有。屈匄被执，楚军大损，前线退却，军心震动，这几件事每一样都像一枚钉子，钉在他先前那句“先伐秦”上。

许久之后，他才抬头道：“昭阳何在？”

内臣低声回道：“令尹收拢败军，仍在前线。”

楚怀王的手缓缓握紧，青筋毕露。

群臣之中，有人面如土色，有人低头不语。主战者此刻反倒最沉默，因为先前鼓噪得最响的，也正是他们。只有屈原抬眼望着那份败报，心里一阵阵发冷。他知道，丹阳一败之后，楚国真正难的还不是前线，而是楚怀王接下来会不会允许朝廷承认这是一场必须收住的败。

果然，怀王下一句话便让殿中更冷了几分。

“胜负兵家常事，”他声音发涩，却仍硬撑着，“丹阳一战失利，不足定全局。传诏前线，收军整阵，再议后举。另遣使诸国，责秦背约，促其共起。”

屈原闭了闭眼。

他听得出，这道话里最重的部分，不是“收军整阵”，而是“仍不肯让天下觉得楚已输定”。

而天下这种东西，你越怕它看见什么，它往往就越快看见什么。

邯郸那边，苏秦在深夜收到丹阳急报。

烛火照着竹简上的字，一行一行都像压下来。门客在旁不敢多言，只等他开口。

苏秦看完之后，许久才道：“楚最怕的，不是打一场败仗。大国偶败，还能重整。它最怕的是在大家都还没来得及决定要不要扶它一把的时候，就先败得让别人更有理由不伸手。”

门客问：“相国还去赵、齐吗？”

苏秦把竹简慢慢卷起，道：“更要去。但从今往后，我每去一国，先要面对的都不是秦有多强，而是他们会说：楚自己打成这样，凭什么要我来替它扛。”

他说到这里，停了一停。

“张仪这一刀，终于从嘴里落到兵上了。”
