# 第二十四回 汉中再失楚廷争救策 诸侯愈冷苏子走危途

丹阳之后，楚军虽未全溃，气却已伤了。

前线各营忙着收败、补伍、固垒，郢都朝堂则忙着争一句更难的话：这一仗，到底该不该先停。

昭阳自前线回报的奏书写得极稳，通篇没有一句推罪，也没有一句逞强，只把当前兵势、粮道、士气、秦军守势与楚军短长一一列明，最后落到一句：“愿大王先固汉北，不与秦争一时意气。”

这一句从兵家看再正不过。

可从楚怀王的耳里听进去，却像是另一种认输。

因为丹阳一败之后，他最怕的不是前线真有多坏，而是朝臣、诸侯、天下人从此都把“楚王为张仪所欺，又为秦兵所挫”连成一句话。若先停手，固然可能保住更多兵；可在他看来，也等于亲口承认自己前后两步都走错了。

所以朝议一开，殿中便分得更厉害。

昭阳上书求稳，屈原则更进一步，请先修国内，再联外援。他道：“今之急，不在速胜，在止损。楚若再以怒争胜，只会让秦一胜之后再胜。若能守住汉北，修军政，明告齐、赵、魏、韩，使天下知秦今日可欺楚，明日亦可欺彼，则犹有合众之机。”

主战诸臣却都咬着另一层说辞不放。

“楚若此时自收，”有人高声道，“便是把丹阳之败坐实给天下看。秦兵未入郢都，汉中未尽失，何至于先自沮？”

“正因未尽失，才更不能再拿剩下的去赌。”屈原当殿回了一句。

这句话回得直，也回得太硬。楚怀王坐在上首，脸色顿时难看下来。

朝堂上最难的时候，从来不只在于意见对不对，而在于哪一种意见更像是在替君主保住颜面。屈原说的是实话，可实话到了这种时候，往往最先显得“不近人主之心”。

楚怀王沉声道：“汉中未尽失，前线尚可整，寡人岂能先说一个停字？传令昭阳，守则守，能战亦战。另遣使齐国，求其共压秦军。”

屈原抬头看了楚王一眼，终于没有再争。

他知道，朝堂走到这里，已经不是多说一句便能扳回来的局。一个君主若把“不能认错”当成眼下第一要务，旁人再讲兵势、国势，都会像在往他最不肯碰的地方再按一把。

而前线的坏消息并未就此停住。

秦国乘丹阳之胜，转而压汉中。

这一步，比单纯前线再胜更狠。因为汉中不是一块可有可无的边角，它是楚向西北撑开的骨头之一，也是楚人原本可以拿来与秦拉扯的一只手。丹阳若只是败军，汉中一动，便近乎败势了。

昭阳接到边报时，站在军图前许久不语。

左右将领都看得出，他此刻最恨的不是秦军能打，而是朝中到现在仍未肯让前线真正按“收、固、守”的思路重新布势。军心未稳，补给未齐，后方却还在等一个足以昭告天下的翻盘消息，这等于逼着前线在最虚的时候，还得装出自己并不虚。

有将领问：“若秦真取汉中，当如何报朝廷？”

昭阳道：“照实报。”

那人迟疑道：“大王若震怒……”

昭阳看着地图，声音极淡：“前线已经替朝廷受够了意气，再不能连实话也替它受掉。”

可实话送到郢都，依旧像火上浇油。

楚怀王得知秦军深入汉中后，整个人都坐不住了。他先前还可以把丹阳说成一战失手，如今汉中若再丢，便再不是“战场偶挫”四字能遮过去的。因为那意味着，张仪骗走的虽不是六百里，可秦人转手便要从战场上拿回比六百里更痛的一整块地方。

这比受骗更狠。

受骗丢的是脸；

失汉中，动的是筋骨。

郢都内外一时人心愈紧。坊间开始悄悄传说“张仪一言，胜过十万兵”，朝中虽没人敢当面这样说，可谁都知道，这句话之所以可怕，不在它说得夸，而在它已越来越像事实。

这时，齐国使者终于到了。

入殿那日，楚廷上下都盯着他。

许多人嘴上不说，心里却都存着同一个念头：丹阳既败，汉中又危，齐总不至于还把话说得太轻巧。可那使者冠带整齐，步子不急不缓，行礼、称王、问安，一切都周正得过分，周正得像故意不肯让自己沾上一点楚廷此刻的慌乱。

他先道：“齐王闻楚遭秦欺，甚以为憾。”

楚怀王盯着他：“既以为憾，齐可出兵乎？”

使者像是早料到这一问，仍旧从容答道：“出兵是大事，不敢以一语轻许。齐王以为，今日若秦楚相争至此，齐若要动，便不能只论愤，更要论后。”

楚怀王的手指在案边轻轻一顿：“何谓论后？”

使者道：“两国若并力，边议可否重定？互市可否重开？他日南北有警，谁先应、谁后应，可否先有明约？这些若不先明，齐虽有心，也难轻发大众。”

殿中立时起了一阵极低的骚动。

有楚臣忍不住斥道：“我楚方与秦交兵，齐使入殿先论的不是共抗强秦，倒是你齐国日后得什么利？”

使者转身看他，脸上仍带着那种不失礼的平静：“正因大军一动便是真金白银，所以更该先把后话说明。若只靠几句义气便让齐国发兵，岂不正是对盟事最轻的一种看法？”

这一回得极软，却把那层利字回得更硬。

楚怀王眼皮直跳，还是强压着问：“齐欲何待？”

使者这才微微一笑，把话往明处推了一寸：“楚若真欲并力，便须让齐国先看见，并力之后不是替楚独挡一阵，而是真有两国共利之局。”

殿中有人当场变色。

这便是最冷的地方。楚国这边血还没擦净，齐国那边已经先把秤砣摆出来了。它不是不肯帮，而是要你先承认：如今这一只手若想伸出去，先得看你能回过来托住它多少。

屈原在旁听着，胸中一点点发沉。

他知道齐国这一步其实并不奇怪。越是强国，越知道什么时候可以把“援手”变成“要价”。可放在此刻，对楚怀王而言，这比单纯的冷眼更难受。因为它等于逼他承认，楚国如今不仅要打败仗，还得为了求别人来帮自己，先吞下一层新的低头。

楚怀王没有当殿答应，也没有断然拒绝，只说“容后再议”。

可从齐使退下那一刻起，殿中许多人就都明白了：合纵到了这一步，已不再是苏秦当年那种“共惧秦而暂同向”的高势，而开始第一次被人当殿拆开来谈条件、谈后利、谈值不值得出兵。

邯郸那边，苏秦也终于切身体会到这张网有多难再拉住。

他先走赵，再试魏，末了又给齐发急书。可越奔走，越能听见同一种意思，只是各国包装得不同而已。

赵说：楚先失，赵不便先替其受兵。

魏说：河外太近，魏不能因一时义愤先动边境。

齐说：若真并力，当先把往后利益说清。

韩与燕更直接，一个忙于自固，一个干脆远叹。

苏秦在馆舍中把这些回书摊开，看了很久，忽然明白过来：从前合纵最难的时候，是人人不肯先站到一面旗下；如今更难的时候，是人人都肯嘴上站，却都要先问，自己若真伸手，能换回什么。

门客低声道：“相国，再这样下去，是否该先保赵、魏两国，暂不强求齐楚一体？”

苏秦沉默了一阵，才道：“这正是张仪要逼我走的路。若我也开始只顾先保哪几国，那便等于亲手认了合纵从此只是分段自保，再没有共局可言。”

门客问：“可若不这样，又如何救得住？”

苏秦望着满案回书，缓缓道：“救不住也要去救。因为一旦连‘还想把大家拽回一处’的人都先改成只算局部，那这局就真的散干净了。”

他仍要走，只是脚步已比先前更重。

而楚国前线，坏消息终于还是到了必须命名的程度。

汉中失守。

这四个字飞进郢都时，整个楚廷都像被当胸砸了一记。先前丹阳尚可说是兵败，汉中一失，便成了实地之痛。楚怀王坐在案前，整个人像一下子老了几岁，目光却反而更硬。

他久久不语，最后只说：“秦要寡人认输，寡人偏不认。”

屈原听见这句，心里一片冰凉。

他知道，从这一刻起，楚国最危险的已不单是秦兵西压，而是楚王已经开始把“不能认输”本身，当成新的国策。
