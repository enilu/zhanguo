# 第二十五回 蓝田道上楚兵又急 韩魏边前小手先伸

汉中再退之后，昭阳其实已看得很清。

楚军若再按“快打一场，把前头都补回来”的心思往上压，只会越压越乱。可前线看得清，不等于郢都也肯认清。朝中的急，军中的躁，天下盯着楚国看笑话的那股冷气，全都在逼着这支军继续往前找一场漂亮仗。

蓝田便在这样的逼迫里到了眼前。

那里比丹阳更坏。

坏在它离秦更近，地势也更利于秦守。你若求稳，它处处像铁；你若心急，它又偏偏总留一角看似可破。

昭阳下令时，比丹阳还严。

“不得分进，不得争先，不得见敌收垒便自作主张加兵。”

几名将校齐声应命。

可应命归应命，军中那股“此战必须见些体面”的心，却不是一道令就能压得住的。前头伤兵刚被抬下去，后头催报又从郢都连夜赶来。楚怀王在书里没有明写“速胜”，可句句都在催一个“快”字。

“勿久持，以沮众心。”

昭阳把那封书看完，半晌没有说话，只把它折起来塞进袖中。

次日两军接阵，秦军果然不与楚硬冲，只在主垒前一层层吃力、收锋、再起。楚军几番试压，都没能敲开它正面，军中那股焦躁便一点点从底下翻了上来。

最先乱的还是前头几营。

他们不是想违令，只是人人看着眼前那条沟、那道坡，都觉得只要自己这一营再往前半步，别营便能跟上，整场仗也许就能从“久持”翻成“破垒”。

于是半步接半步，便又把本该一起推进的数路压成了几股不齐的力。

昭阳在后军看见时，立刻命传骑叫停。

可蓝田这种地势，一处先急，旁边几处就都要跟着急。秦军等的也正是这一刻。中垒忽然起鼓，两侧伏着的兵线同时翻出来，卡在楚军最想合力、却还没合成的那一瞬间上。

楚兵不至大溃。

可不大溃，比大胜更难向后头交代。

因为它意味着，这仗既没有漂亮到能昭告天下，也没有稳妥到足以叫人安心止兵，只是一点点把楚军往更深的疲与乱里拖。

到暮色落下时，昭阳不得不先收军。

这一收，看上去还保着体面，可楚军上下都知道，蓝田也没打成。该开的口子没开成，该压回去的气也没压回去。最要命的是，这一回再退，天下便更会觉得楚已不是偶挫，而是真在一路往下滑。

消息传入郢都，殿上比前两回都安静。

因为连最爱叫喊的人，也开始不知道该怎么替这一仗圆场了。

楚怀王站在殿中，听完之后，一时间连怒都发不出来。

他最怕的是天下看见自己前后都错，如今偏偏是人人看得越来越清。

就在这时，东边边报又到。

韩、魏两处边境都开始异动。未必算大战，也不是真要趁机灭楚，可越是这种不深不浅的小动作，越叫人心里发冷。大国真滑下去的时候，常常不是众敌一齐扑上来，而是人人都先伸出一点小手，看你还有没有力气把这点小手也一并打开。

殿中有人当场怒骂韩魏无义。

屈原却只看着那两份边报，没有立刻说话。

过了一会儿，他才出班：“大王，事已至此，再不能只盯蓝田一线。楚若还想用下一仗洗尽前耻，四面只会越试越深。”

楚怀王看向他，目中满是疲色。

“那你要寡人如何？”

“先收兵，守边，定人心。”屈原道，“若再晚一步，别国看见的就不只是楚前线不利，而是楚已忙到连两边都顾不稳。”

这一次，连先前最爱主战的几人，也没有再立刻顶屈原。

不是因为他们忽然都懂了，而是因为眼前摆着的已不是一场两场的败，而是楚国开始被别人当成“可再试一试”的对象。

楚怀王缓缓坐回王座，像一下子被抽去了不少力。

他仍不肯说“错”，也不肯说“停”，可也终于没有像前几回那样立刻拍案再压前线。

殿中沉了许久，他才问：“齐那边呢？”

内臣答：“齐肯议援，只是条件又重了一层。”

楚怀王闭了闭眼。

秦从西边压，齐从东边量，韩魏在边上试。前线每退一步，朝堂便要多承一分重。他这时才真正尝到，张仪在郢都那一句“给地”的后面，原来藏着的是怎样一整条把楚往下拖的路。

邯郸那边，苏秦闻知蓝田再失，又见韩魏伸手，也是一夜未眠。

天亮之后，他照旧起身，更衣，上马。

门客忍不住道：“相国，众国都在先问价，再走下去，还能改什么？”

苏秦束紧衣带，只道：“至少要让他们知道，再这样问下去，终有一日也轮到自己被人这么问。”

说完，他翻身上马。

赵地清晨的风极冷，吹得马鬃往后一绺一绺倒过去。苏秦没有回头，只径直往齐的方向去。

他知道，自己眼下走的已不是当初佩印时那条把众人先拢过来的路。

而是一条在满地价码里，硬替“共局”留最后一点位置的路。
