# 第二十五回 蓝田再战楚王终难醒 两路乘虚诸国各自安

汉中失守之后，楚国本有一次收住的机会。

机会不在于立刻翻盘，而在于终于承认：这场仗若再按“必须尽快赢回来”的心思去打，只会越打越像秦人想看到的样子。

昭阳、屈原，甚至朝中一部分先前主战的臣子，到这时都已隐隐明白这一点。于是下一轮廷议时，求稳的声音第一次压过了鼓噪。有人请固边，有人请缓战，有人甚至请暂且低头修和，只求把楚国从“怒而两失”的势头里先拽出来。

这原本是最接近正确的一刻。

可也正因为最接近，它反而最难被楚怀王接受。

因为若在这一刻收手，就等于把前面所有硬撑都一并坐实了。他不只是得承认自己被张仪所欺、被秦军所败，还得承认自己在丹阳之后、汉中将失之时，仍一再押错。对一个志大而轻信的君主来说，这种承认比再打一仗更难。

所以楚怀王听完满殿劝言，最终只落下一句：“寡人若此时止兵，天下必以楚为怯。秦欺我、夺我，而我但知自守，这不是大国之道。”

屈原再拜，道：“大国之道，不在一时争强，而在知何时可止。”

楚怀王看着他，眼神冷了下来：“大夫近来劝寡人止得太多了。”

这一句话出口，殿中再无人敢正面顶回去。

于是楚国再整兵。

这一次，不只是前线续战，而是带着一种“非把前两步都补回来不可”的急。楚怀王要的，已不是普通意义上的战场胜负，他要的是一场足以洗去丹阳败报、压住汉中失地、堵住朝野流言的大战。

可战场最不替人遮羞。

你拿得越多，它反而越容易逼你多输。

秦国很快看出了楚国这股心气。

蓝田一线，山川形势与丹阳又异。此地更靠近秦人熟悉的腹地边缘，进退、粮运、据险都更便于秦。樗里疾得报楚军又将压前时，只说：“怀王终究还是不肯停。”

旁人问：“我军是否当先拒其锐？”

樗里疾摇头：“不急。楚如今最急的不是兵到哪里，而是心已经先到。心先到的人，脚下往往就更容易失空。”

秦王嬴驷亦不急。

他比谁都清楚，楚若真能在这时自己停住，那前两刀固然已伤其筋骨，却未必能彻底伤其气；可若怀王不肯停，非要再把一国意气压上蓝田，那这位楚王便会亲手把张仪的局往最深处送。

果然，楚军再至蓝田时，整支军的样子已与最初北上不同。

最初那股“雪耻”的气虽还在，却混进了更多更杂的东西：必须快胜的焦灼，已经两败不得再败的惶急，朝廷一次次催压下来的硬撑，以及诸将彼此之间谁都知道这回不能再出差错的紧绷。这样的军，表面上看更狠，骨子里却更脆。

昭阳一路压着节奏，不肯轻进。

可他越稳，来自后方的催令便越密。楚怀王不止一次遣使到军中，责其“持重太过，恐坐失鼓士之机”。军中也有人开始窃窃，说令尹是被丹阳打怕了。

昭阳听了只是沉默。

他知道，自己如今已不只是在同秦军打，也是在同朝廷的急、同军中的躁、同一场不能再败的气氛硬扛。若这时前线每个人都觉得必须尽快交出一个漂亮结果，那么稳字便会越来越像软，慎字便会越来越像怯。

蓝田终究开战。

这一次，秦军不似丹阳那般层层示退，而是守中带诱，诱中藏逼。楚军几番试压，都难撼其主垒。久攻不下之后，军中那股“再这样拖下去如何向郢都交代”的急火又烧了起来。几个方向几乎同时加力，想硬从一点打穿，再借一点成全局。

昭阳察觉时，已经迟了半步。

这半步，在平日里也许还可补；可在一支本就绷得过紧的军中，半步之迟往往便是全线开始乱动的口子。秦军等的恰恰就是这个口子。一处压、一处断、一处突然反切，楚军数路本想合成一拳，最后却被逼成了彼此难应的数截。

战至午后，蓝田尘起如墙。

楚军虽不至像丹阳那般一触即崩，却也始终找不到能真正把局面扳回来的那一下。越是想拿出一场“大胜”，越是被秦军按着往“小败积成大败”的方向拖。及至暮色将下，楚军后线已有动摇迹象，昭阳不得不下令全军回收。

这一退，虽还保留了几分体面，却终究仍是退。

而对楚怀王来说，最不能接受的，恰恰就是“还算体面地再退了一次”。

更坏的消息还在后面。

就在楚国把国中注意力几乎尽数压向西线时，韩、魏两国边境也开始异动。它们未必是大举灭楚，却都很懂得挑这样的时候伸一伸手，探一探楚国如今到底还能不能同时顾住两头。

这便是国势真正滑下去时最冷的一幕：

不是所有敌人一起扑上来，

而是人人都先伸出一点点手，看你是不是已经虚到连这一点点都挡得吃力。

楚廷接到东面边报时，朝中简直乱成一片。有人痛骂韩魏无义，有人怒斥诸侯坐视，有人这才后知后觉地明白，蓝田这一仗就算不算大崩，也已足够让天下觉得楚国可以被多处同时试探。

楚怀王站在殿中，半晌没说话。

他先前一直不肯认“停”，如今却第一次尝到了“不肯停”的后果不是只在一线，而是会把整个国境四周都一起拖松。

屈原出列，道：“事至于此，再不可只论一线胜负。楚必须先收兵、守边、定国内，再议外援与后策。若仍只想下一仗便洗尽前耻，则诸国只会继续趁我未定而来试我。”

这一次，连不少主战者都不再顶他。

因为蓝田之后，再说“只差一场胜就能全转回来”，连他们自己都听不太信了。

楚怀王终于缓缓坐下，整个人像一下子被抽去了几分硬撑的气。他没有立刻认错，也没有像从前那样再拍案而起，只是低声问了一句：“齐国那边，回话如何？”

内臣答道：“齐仍可议援，但条件更重。”

楚怀王闭上眼。

他终于走到了这样一步：仗打不赢，停又停得太晚，诸侯不来救，愿意谈的却先把价提得更高。张仪当初在郢都用六百里诱他的那一刻，他看见的是一块地；而走到今天，他才发现自己一步步被诱出来的，其实是楚国在天下棋盘上那份本来还算可观的主动。

邯郸那边，苏秦闻知蓝田再败，又接到韩魏乘隙而动的消息，沉默了整整一夜。

次日，他照旧起身更衣，照旧命人备马。

门客忍不住道：“相国，事到如今，众国都已各自起算，再奔走还有多少用？”

苏秦系好衣带，平静道：“有多少用，不是先问出来的，是走出来的。只要他们还没彻底与秦连成一线，我就不能先停。”

门客苦涩道：“可他们如今不是不听道理，是都先问价。”

苏秦点头：“我知道。可纵横到了这一步，也正该看谁还能在满地价码里，替天下留一句不只为自己算的话。”

他说完便出门。

风吹过庭前枯枝，声极细，却有一种越听越冷的空。

这一回里，苏秦没有赢什么。

可他仍在走。

因为当所有人都开始把合纵当成一张可以拆着卖、拆着算的旧网时，若连他也停下，那这网便真的只剩线头，再没人会记得它曾经是一面能叫东方暂时共向西方的大旗。
