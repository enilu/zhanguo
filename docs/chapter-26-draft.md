# 第二十六回 楚使四出先闻重价 咸阳一笑更看旧盟

蓝田之后，郢都最忙的已不是前营，而是使馆。

去齐的，去赵的，去魏的，去韩的，一拨拨出城，又一拨拨回来。回来的车马多半还带着尘，车里装的却不是兵书，而是一卷卷不肯说死、也不肯松口的回话。

齐说可议，只是边约、互市、战后得失都须先写清。

赵说并非不忧秦，只是近秦之国不可轻动。

魏说河外多警，难先受兵。

韩更直接，只说国小兵弱，动一步都得先看边口。

没有一封书写“我不救你”。

可每一封书都在告诉楚：若想叫别人替你出兵，旧盟、旧义、旧日声势都不够了，你得先拿得出新的东西。

楚怀王把这些书一卷卷翻完，最后全推到案边。

案边的木尺、玉镇被卷书撞得轻轻一响，殿中诸臣全低下头去。谁都知道，这些日子里最难受怀王的，已不只是秦兵，而是一个大国终于发现，旁人连拒绝它都可以拒绝得如此客气、如此体面。

屈原入殿时，正看见满案回书散着。

楚怀王抬头看他，目光里第一次少了许多先前那种只想压过别人话头的硬气。

“屈原，”他问，“寡人错得真有这么重？”

这话一出，殿里更静。

屈原停了停，方道：“大王起初错在轻信，后来错在不肯早止。前一错给了秦口，后一错让诸侯看见楚会自己把伤越按越深。”

楚怀王听完，半晌没说话。

他像终于不是在问别人该怎么替自己圆，而是真在问，自己前后到底差在何处。

可问出来，不等于就能轻易改。

“若再求齐，”他低声道，“齐必更重。”

屈原道：“会重。可若不求，楚眼下也没有第二只手能接得更稳。”

楚怀王笑了一下，那笑极淡，也极苦。

“那便还是要低头。”

屈原没有接这句。

有些话到了这时候，不接比接更重。

同一时日，临淄也在看楚使新报。

齐王把楚书放下，笑问田文：“楚肯再让几分？”

田文答：“急得越深，让得越多。”

齐王道：“那便等它再急一急？”

田文摇头：“拖得太狠，也未必好。真把它拖回秦那边，秦省的便是我们的力。”

“那你说该如何？”

“让它总觉得再让半步，齐便会动。”田文道，“半步接半步，楚便会自己把价抬到我们想要的位置。”

齐王笑了笑，没有反驳。

他知道田文说的是齐最熟的一套路子。大国最会的，往往不是当面回绝，而是把“还可谈”三个字留着，一边留，一边把秤越摆越清。

咸阳那头，消息也到了。

秦王把楚、齐往来新书递给张仪，问：“还撑得住么？”

张仪把几卷书都看完，才答：“楚国大，一时撑得住。可更要紧的是，诸侯现在一见楚，先想的不再是它当年多盛，而是值不值得扶。”

秦王道：“苏秦呢？”

“还在走。”张仪道。

“还有用？”

张仪停了一瞬，方道：“有。苏秦能把六国先拢到一处，这等本事本就少见。也正因他拢得高，如今楚一滑，诸侯心里那一下‘原来合从也能这样散’才会落得更重。”

秦王听后，淡淡一笑。

“你倒少见地夸起你师弟了。”

张仪低头道：“臣不是夸他，是看得清他立过什么局。立得越高，掉下来时，天下人记得越久。”

殿中风从门缝里进来，吹得图角轻轻一动。

秦王缓缓道：“如此看来，这局声上已分。”

张仪却道：“声已分，势未尽。”

“何意？”

“齐若真接住楚，楚便还能喘一口。赵虽冷，也未必永冷。合从这张网眼下是往下坠，不是已断净。”张仪说着，抬手在图上点了点临淄与郢都之间那条线，“要想让这张网再难收回去，还得叫齐楚之间也生出怕。”

秦王看着他的手指，没有出声。

片刻后，张仪又道：“楚现在最怕的，不只是败，而是为败后处处求人。若秦这时候肯让它看见另一条不必向齐低头太深的路，它自己心里便会先晃。”

秦王问：“什么路？”

张仪答：“归地。”

樗里疾站在一旁，眼神微微一动。

他一下便明白了。

齐眼下靠的是楚急。若秦此时也把一条“可归地、可缓兵、可再谈”的路亮给楚看，楚便会拿这条路去压齐；齐一被压，心里便也要疑楚究竟会不会真靠过来。两家还没抱实，便先要互相试。

秦王道：“你还要再入楚？”

张仪拱手：“臣愿去。”

殿上静了片刻。

秦王终于点头：“那便去。”

张仪退下后，樗里疾望着他的背影，道：“此人不是只会骗人。”

秦王道：“他是会叫别人先顺着自己的心骗自己。”

那夜，咸阳宫外风很冷。

张仪回到馆中，只命人备轻车，不带多旗，不惊驿路。随从问他此行几成把握，他整了整袖口，只说：

“不是我有把握，是楚王心里那条不肯低头太深的路，已经先开了半扇门。”
