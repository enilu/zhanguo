# 第二十六回 求援四出诸侯先论价 破盟至深纵横已分高

蓝田之后，楚国终于不敢再把“下一仗便翻身”挂在嘴上了。

可不敢再说，不等于局面就会因此变轻。

相反，真正沉的部分，到这时才彻底压下来。因为前线可以一败、两败，朝廷也总能为每一败各找说辞；但当你发现自己四面去求，回来听到的却大半不是“愿不愿救”，而是“准备拿什么来换”，那才说明国势已从战场上的输赢，滑到了天下人开始重新掂量你轻重的地步。

郢都这几个月，最忙的已不是军府，而是使馆。

往齐国的，往赵国的，往魏国的，往韩国的，一拨拨使者出城又归城，带回来的消息大都说不上绝，只是每一份都带着一种令人发冷的分寸。

齐说可以议，只是边议、后利、相报诸事都须先明白。

赵说并非不忧秦，只是赵近秦而重，轻动不得。

魏说河外多警，自保为先。

韩则更滑，只说国小兵弱，不敢妄议大局。

这些回话没有一封明着写“我不救你”，却又几乎每一封都在告诉楚国：你若想让我动，不是只靠一段义理、一份旧盟、几句共惧秦国的大话就够了。

楚怀王把回书一封封看完，终于有一天把它们全推到了案边，久久未语。

他到这时才真正尝到一种比战败更难受的东西：

原来一个大国一旦被人看出正在下坠，旁人连拒绝你都不必拒得太难看。

屈原入见时，正碰上这一幕。

地上散着几卷竹简，殿中无人敢出声。楚怀王听见脚步，抬起头来，眼里竟少了前些日子那种硬顶着不认的火，更多的是一种压在火底下、终于有些发哑的疲惫。

“屈原，”他问，“寡人从前真错得如此重么？”

屈原停了停，没有立刻答。

因为他知道，这句话若在丹阳前问，是怒；在汉中失后问，是恨；到如今再问，才第一次像是真的在问。

他终于道：“大王不是一开始就弱，也不是楚国本不可为。只是大王前面错在信人太轻，后面又错在认错太迟。前一个错让秦得了口，后一个错让诸侯看见楚会自己把伤越按越深。”

楚怀王苦笑了一下，那笑意极淡，也极短。

“寡人若此时再求齐，是否已晚？”

屈原道：“未必晚，只是再求，求来的便不会是旧日并力之盟，而更像一笔重价买来的援。大王若肯认这一点，还可保一时；若还想既不低头、又要别人出兵，那便是真晚了。”

楚怀王低头良久，最终只说了一句：“去议吧。”

这句话出得很轻。

可对楚国来说，分量极重。因为它意味着这位一再不肯停、不肯认、不肯先低头的君主，终究还是被局势逼到了要用让利来换援手的地步。

齐国那边果然没有放过这个时机。

临淄朝堂上，齐王听完楚使陈情，并不急着答应，只笑道：“楚昔日强，未必总看得起齐；今日急，倒知来问寡人了。”

楚使面上发热，却不得不强撑着道：“秦若坐大，齐亦终难独安。”

齐王点了点头：“这话不假。可秦坐大是后患，齐若此刻出兵，耗的却是眼前真金白银。后患与眼前，寡人总得先分一分轻重。”

田文在旁接道：“若楚真欲齐出力，便不能还按旧日强楚待人的法子来议。”

这便是把价彻底点明了。

齐不是不懂大势，也不是看不见秦的可怕。它只是比谁都清楚，当你明知对方急、又明知对方背后再无第二只手真能接住他时，你若还不先把条件抬高，那便对不起这份时机。

消息传回郢都后，朝中争议再起。

有人说再重的价也该付，总比独撑至死好；有人说若以重地重利求齐，那楚国纵保下一时，也等于从此承认自己已从“主局者”变成“求局者”。屈原心里清楚，这两句话都不假，正因都不假，才更显此局之痛。

而就在楚国为“求齐值不值”争得难分难解时，齐廷那边也没有闲着。

田文退朝之后，陪齐王缓步出殿。殿外霜气未散，石阶上风很硬。齐王忽然问他：“楚若真肯让，寡人当不当动？”

田文答得很快：“可动，但不必为楚动，只该为齐动。”

齐王看了他一眼。

田文继续道：“如今楚急，秦强，赵魏韩燕都在看。齐若全不动，日后秦势真重，齐也未必舒服；可齐若轻动，便成了白替楚挡这一阵。故最好是让楚先把话说尽，把价摆明，再看这一动究竟买来的是一时名，还是一段实利。”

齐王笑了笑：“你与张仪，倒像隔着千里说一样的话。”

田文道：“不是臣像张仪，是天下到了这个地步，谁都知道不能白出手。”

而就在这时，张仪已从另一个方向把局面看得更清楚了。

他回到咸阳后，再次入宫。

秦王问他：“楚还撑得住么？”

张仪道：“国大，一时还撑得住；可心势已经下来了。一个国家最怕的，不是打一场败仗，而是从此每逢有事，旁人第一反应都先问：它如今值不值得扶。”

秦王又问：“苏秦呢？”

张仪微微一顿，才答：“他还在诸侯之间走。”

“还走得动？”

“走得动，”张仪道，“只是他现在走的，已不是当年那条替六国立旗的路，而是替一张正在散开的网，一根根去捡线头的路。”

秦王看着他，似笑非笑：“你倒看得起你这位师弟。”

张仪拱手道：“苏秦能把六国一时拽到一处，天下原就少有人能及。也正因为他曾把局立得那么高，臣如今每拆下一寸，天下才会看得更清，记得更深。”

这番话里，竟少见地没有太多刻薄，只有一种同路之人最知道彼此斤两的明白。

秦王沉吟片刻，道：“如此看来，这一局胜负，声上已分。”

张仪道：“声已分，势未尽。臣还能再走，诸侯也还能再变。只是从今往后，谁再提合纵，众人心里都会先想起楚。”

这句话最狠。

因为它意味着楚国这几场败，不只伤了自己，还替“合纵”这个字也添上了一层新阴影。以后诸侯再想并力拒秦时，脑子里先浮出来的，未必再是佩六国相印的风光，反而可能是怀王为利所动、为怒所战、两败之后四处求援的狼狈。

张仪这一卷真正可怕处，正在这里。

他不仅是在拆一个联盟，

还是在改这个时代的人，今后如何去理解“联盟”本身。

邯郸，深冬。

苏秦又从一场无功而返的朝议中出来，衣上尽是风尘。

这些时日他走得极勤，话也说得极尽，可得到的多半还是那些老话的不同说法：赵不便先动，魏不敢先受，齐可以谈价，韩燕各顾其边。人人都还愿意敬他一声苏子，也都还承认秦国是最大的祸患，可真要他们往前迈一步时，每个人都先回到自己那张小算盘上。

门客跟在后头，忍不住道：“相国，若天底下连怕秦都不能把他们再拢起来，那还能拿什么拢？”

苏秦走在风里，许久才答：“拿将来。”

门客一怔。

苏秦道：“怕眼前，人就只会先顾眼前。只有真看见将来自己也会轮到这一步，才有可能再回头想今天该不该伸手。”

门客苦笑：“可他们眼下都不愿看那么远。”

苏秦停下脚步，望向远处灰白天色，轻声道：“我知道。所以我才更要替他们看着。”

这句话说得很轻。

可落在寒风里，却有一种比怒气更硬的执拗。

他没有在这几回里赢回局面。

可他也没有像许多人盼着那样，就此沉下去、消失掉、或者索性改成与众人一起先算自保。

因为苏秦若也只剩自保，那张仪赢的便不只是一步破盟，而是整整把这个时代里“还想把六国重新往一处拢”的那股心，也一并压下去了。

夜里，苏秦独坐灯下，提笔写给赵、齐两国的又一封书信。

他写得极慢，写一句，停一句。

灯火照着他清瘦的侧脸，也照着案边堆着的那些回书。那些竹简里装满了推托、迟疑、权衡、价码，可他仍旧一封封拆、一封封回，像明知洪水已来，却还不肯让那道堤就这样干脆垮掉。

窗外风声穿廊而过。

一名门客自外快步入内，低声禀道：“相国，齐国那边又有回书。不是回绝，是让楚国再明一层条件。另有消息，楚使已二次入临淄。”

苏秦握笔的手微微一停。

他看着那封尚未拆开的回书，沉默片刻，终于把笔重新落下：“备马。明日先去齐。”

窗外风声更紧。

他忽然想起当初佩印之时，六国旌旗并列、众使同在、天下侧目的那个日子。那时他知道这局难守；可直到如今，他才真正知道，“难守”二字，原来不是守一面旗，而是要在每个人都开始明码算账的时候，还硬把他们往将来那一处更大的祸上拽。
