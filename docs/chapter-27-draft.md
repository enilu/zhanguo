# 第二十七回 楚使再入临淄低头议价 齐王将动未改观风旧心

苏秦赶到临淄时，楚国第二拨使者已先入齐宫。

这一次，楚使来得比前回更急，也更低。

前回入齐，楚还带着几分“大国求援，不失其尊”的勉强撑持；到了这一次，丹阳、汉中、蓝田的几层败势都压在背后，连使者自己都知道，再若还摆着旧日强楚的架子，别说齐王不肯点头，连齐廷群臣都会先在心里发笑。

所以入殿时，他先把姿态放了下去。

“楚国知错失前机，今愿与齐重申旧好，共抗西秦。”

这句话说得极低。

齐王坐在上首，听完之后并不立刻表态，只把目光慢悠悠移向两侧群臣，像是故意要让这句“知错失前机”在殿中多停一会儿。

田文第一个出列，道：“楚若真知前失，便该明白今日议的已不是虚言相援，而是两国如何各取其安。齐若出兵，楚能给齐什么把握？”

楚使深吸了一口气，道：“若齐肯出兵，楚愿重修边约，开互市，后日秦若东压，楚亦当应齐之请。”

田文淡淡道：“后日之事，空得很。眼前之事却实得很。齐兵若南下，耗的是今日之粮与卒。楚若只拿一句后日相应来换，未免太轻。”

殿中几名齐臣都微微点头。

楚使额上已见细汗，却仍不得不往下说：“若齐有别议，亦请明示。楚王有命，只要两国真能并力，皆可商量。”

这话一出口，殿中气氛便变了。

因为到这一步，楚已经不是来问齐肯不肯援，而是来告诉齐：价，你可以开。

齐王这才缓缓开口：“楚使既如此说，寡人便也不绕了。齐若动，不为替楚争一口气，只为不使秦因楚倒而更坐大。故齐出兵可以，但楚也须让齐看见，今日这一动，不是白替旁人收残局。”

他说到这里，停了一停，方继续道：“边地旧议，须重定；若大战之后有所得失，亦须先有约。若连这些都不肯言，寡人便看不出齐为何要替楚先当这一阵。”

楚使听完，只觉后背发冷。

齐王并没有狮子大开口到不能谈，恰恰相反，他把每一句都落在“你若要我动，就得承认如今是你来求我”的那层分寸上。这样的条件最难回，因为它不算无理，却句句都在剥楚国的脸面。

使者只得俯身道：“外臣不敢专断，愿请回报我王。”

齐王点头：“可。”

殿议将散时，苏秦终于被召入偏殿。

齐王见他来，先笑了一声：“苏子这几个月走得很勤。”

苏秦拱手道：“局势逼人，不敢不勤。”

齐王问：“你也是来劝寡人援楚？”

苏秦道：“臣不是只为楚来，是为齐来。楚若真自此塌下去，齐今日或许还可从容；可再往后看，东方强国里能替齐先扛西压的，便又少了一层。”

田文在旁接口道：“苏子总爱说后势。可国与国相交，若只看后势而不看眼前，岂不正容易替别人先受苦？”

苏秦看向他，道：“眼前当然要看。可若眼前只看自己省不省一口气、损不损一笔粮，而不看将来自己是不是也会轮到今日楚国这个位置，那便不是会算，只是短算。”

田文笑了笑：“可短算往往先活。”

苏秦道：“先活不难。难的是不把自己活成下一次被单独拿出来算的那一个。”

这句话说完，偏殿里静了一瞬。

齐王端坐不语，像是在掂量。田文也没有立刻再逼，因为他知道苏秦这话并不虚。齐之所以还能如此从容，正是因为楚还没彻底倒、赵还在、山东诸侯的层层屏障还没全被秦一口口拆掉。若真拆得只剩齐一国独自面西，那今日所有“精明”都要重新算账。

可懂得这一点，不等于就愿意现在立刻白出兵。

齐王最终只道：“寡人会动，但怎么动、动到哪里，还要看楚能不能先把话说到让寡人放心的地步。”

这句话比前回更进了一步，却仍不肯彻底放口。

苏秦听完，心里既松了一线，又更沉了一层。

松的是齐终究不愿坐视秦势无穷坐大；沉的是齐直到此刻，想的仍是“如何以最小代价换最大后利”，而不是“该不该先扶楚一把”。

从偏殿出来时，楚使正候在廊下。

他见苏秦，脸上神色极复杂，像既盼这位昔日纵横诸侯的相国能替楚多说一句，又隐隐知道，事情走到今天，已不是苏秦一张嘴便能把旧局重新说回来。

“苏相国，”他低声道，“齐王肯动么？”

苏秦看了他一眼，道：“肯动，但不会白动。你回郢都后，劝大王少问齐是否还讲旧义，多想想楚如今拿什么让齐觉得这一动值得。”

楚使脸色一黯，终究还是点了点头。

郢都收到齐廷回话后，朝议又起。

这一次，殿中虽仍有不平之声，却已比前些日子少了许多。因为仗打到这一步，谁都知道“凭旧盟号召诸侯同起”的时候已经过去，眼下能不能把齐兵谈下来，关系到的不是脸面好不好看，而是楚国还能不能在继续下滑之前，先给自己找一只撑住的手。

屈原主张可谈，但须有底线。

他说：“让利可以，不能让到伤根。齐若要的只是名分与一段后利，尚可商议；若要得太深，则楚纵求来一时援手，也只是从西面失给秦，再从东面失给齐。”

楚怀王听得极慢。

这些时日里，他像终于学会了在别人说完之前，先不立刻拿怒气把话压回去。只是每听一句“让利”“低头”“可谈”，他脸上的肌肉仍会不由自主地紧一下，像那口撑了太久的气虽已被打裂，却还没彻底散。

最终，他低声道：“先议。能换得齐动，便先换。”

这句话说出口时，殿中没人再敢抬头去看楚王神色。

因为所有人都明白，楚国走到这里，已经不是在争“该不该求齐”，而是在学着接受：有些过去不肯给的东西，如今若还想要别人出力，便不得不拿出来。

而就在齐楚一步步快要把条件谈实之际，咸阳那边也得到了消息。

秦王把密报递给张仪，只说：“齐要动了。”

张仪接过来看完，神色却并不慌，只是微微笑了一下：“齐会动，但齐动得越像一桩买卖，就越说明这局还没真正重新拢起来。”

秦王问：“你以为还能拆？”

张仪道：“能。只要让楚王看见，跟齐绑得太深，未必比向秦回头更合算。”

秦王盯着他看了片刻：“你又想入楚？”

张仪把密报合上，缓缓道：“楚王先是为利背合纵，后又为怒失战机。如今若再让他看见一个‘不用向齐低头，也能从秦手里拿回些东西’的路，他未必忍得住不看第二眼。”

这句话一出，殿中几名近臣都抬了头。

他们知道，张仪这一回若真再入楚，便不只是重复“六里之欺”，而是要在楚国已经尝尽求援滋味之后，再递过去另一份更难拒绝的诱饵。

而这种诱饵，往往比第一次更危险。
