# 第二十七回 临淄殿里楚先压身价 郢都门前张又叩王门

苏秦赶到临淄时，楚国第二拨使者已在城中住了两日。

这一回来的使者，比上回更低，也更急。进齐宫前，他甚至先在馆中把旧日答辞删了半卷，只留下最要紧的两句：楚知前失，愿重议旧好；齐若肯动，诸事皆可再商。

这已经不是来讲旧盟了。

这是来告诉齐：价，你可以再开。

齐王听见这句话时，脸上并没有露出多少得色，只把目光往群臣身上一扫。那一扫，比直接发话更叫楚使难堪。因为人人都看得出来，楚已从过去那个不肯轻低头的大国，变成了殿上明说“诸事皆可再商”的求援者。

田文先出班。

“楚若真知前失，”他说，“便该知道今日议的不是漂亮话，而是齐兵动了之后，楚拿什么让齐觉得这一动不白。”

楚使额上已经起汗，却还是得往下答：“若齐肯出兵，边约可重修，互市可再开，他日秦东压，楚亦可应齐。”

田文摇了摇头。

“后日太远。齐今日动，耗的是今日的卒和粮。”

殿中几名齐臣都跟着点头。

这便是齐国最熟的路子。它不必把话说死，只要一层层把“今天”压在“明天”前头，便足以逼得对方自己一点点把价抬上去。

楚使咬了咬牙，只得道：“若齐另有深议，也请明示。外臣奉命而来，不敢藏。”

齐王这时才开口。

“楚既这样说，寡人也不绕。”他说，“齐若动，不是替楚争一口气，是不想让秦因此更坐大。既如此，边地旧议、战后得失、后应之次，都须先说。”

他说得不快。

每说一层，楚使心里便往下沉一层。

不是因为齐王狮子大开口，恰恰相反，是因为这些条件都算得过去，都像站得住。越站得住，楚越难翻脸。

这场议还没议完，苏秦便被召进偏殿。

齐王看见他，笑道：“苏子来得巧。”

苏秦一礼，道：“臣来得不巧。若再早些，也许还可少让大王多听几句价。”

齐王失笑：“你是来替楚省价，还是替齐算后祸？”

苏秦答：“臣是替齐算后祸。”

“说来听听。”

苏秦抬眼，看向齐王，也看向田文。

“楚若真倒下去，齐今天也许还能省一笔粮、一支军。可再往后看，东方能替齐先扛住西压的，便又少一层。”他说，“今日齐在殿上量楚，来日若只剩齐自己立在海东，那时再想叫别人分担，便不见得还有今日这份从容了。”

田文听完，淡淡道：“苏子总爱说后日。可国与国相交，若只看后日，不看今日，岂不成了白替别人吃眼前亏？”

苏秦道：“今日当然要看。可若今日只看自己省多少，便把后日轮到自己时别人也会怎么量你，一并忘了，那就不是会算，只是先把账算短了。”

屋里静了一瞬。

齐王没有接，田文也没有立刻逼。

因为他们都知道，苏秦这话并非全是空势。齐眼下能这样量楚，正是因为楚还没彻底倒，赵还在，山东诸国的层层屏障也还没全被秦拆净。可懂得这一层，不等于就愿意现在白替楚出兵。

齐王最终只道：“寡人会动，但怎么动，还要看楚能不能让寡人放心。”

这句话比之前又进了一寸，却仍不肯落死。

苏秦出了偏殿，在长廊里碰见楚使。

那使者连着几日低头议价，脸上倦色极重。见苏秦出来，忙迎上去：“相国，齐王可松了？”

苏秦看着他，道：“松了一线。可齐要的不是一句好听话，是楚自己承认如今已不能按从前待人。”

楚使嘴角一动，却终究还是低头应了。

同一时日，张仪也到了郢都城外。

郢都听说他再来，宫中先起的不是议，而是杀声。许多楚臣第一反应都是：此人既敢再踏楚门，便该立刻取头。可再往下一想，殿上那股杀意又不再那么整齐了。

为什么？

因为张仪若真只是来送死，秦不必遣他。

既敢来，便说明秦又递了一条路。

而楚此时最怕的，恰恰就是自己手里真只剩齐这一条路。

楚怀王听见“张仪又至”时，果然大怒，当场掀了案上一只玉杯。碎片散在阶下，谁也不敢先去捡。殿中主杀之声一时大作，昭阳却先一步出列。

“大王，张仪可恶，自不待言。”他说，“可正因如此，更该先听他来做什么。若只凭一怒便斩，楚今日失的也许不止一人之命。”

屈原随后也出班。

“臣请大王见其人，勿信其辞。”他说，“张仪最善处，不是无中生有，而是把人心里本已有的一点难处、贪念、不甘，说成一条极顺的路。大王若见，须先记住这一层。”

楚怀王坐在上首，脸色阴沉得像要滴下水来。

他当然恨张仪。

可他也无法不去想：若秦此时真肯归地、缓兵、或至少放出一点比齐更好看的意思，那么楚是不是就不必把“求齐”这条路走得那么低、那么深？

这念头一冒出来，他自己先惊了一下。

张仪人还没进殿，自己心里那杆秤竟已先晃了。

楚怀王沉默良久，最终还是道：“放他入城。”

旨意传出时，屈原心里更沉。

他知道，最难的不是楚王肯不肯杀张仪。

而是只要张仪一进郢都，齐使那边每一句条件，都会在楚王耳里变得更刺。
