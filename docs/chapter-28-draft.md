# 第二十八回 咸阳先许归地影 郢都再开旧辱门

张仪入郢那日，城门上风很硬。

守门甲士见过他的脸，神色都绷得紧。谁都知道，这人上次来楚，留下的是“六里”二字；如今楚军连败、齐援未定，他竟还敢再踏这座城，除非身后带着更大的话。

更大的话，确实带来了。

那是一封秦书，写得并不长。

书里没一句认错，也没一句低头，只说若楚愿与秦暂缓旧怨，则汉中旧地可议归还一部，边限亦可重定。句句都留着口，偏偏也正因留着口，才最叫人动心。因为这不像六百里那样大得露骨，反倒像一条人勉强还能说服自己去看看的小路。

楚怀王在正殿见张仪时，满殿先有的不是说话，是杀气。

张仪整衣、行礼、伏地，动作仍旧稳得很。

楚怀王看着他，眼神冷得吓人。

“寡人以为，你不敢再见寡人。”

张仪伏地答：“臣若不来，便是先把楚看轻了。”

这句话一出，殿中便有人低低倒吸了一口气。

不是因为它多高明，而是因为张仪一开口，先把自己放在了“仍重楚”的那一边。楚怀王最恨他，可楚怀王也最爱听别人承认楚仍重。

“你欺寡人至此，还敢言重楚？”

楚怀王声音不高，殿上的温度却像一瞬间都降了下去。

张仪仍旧没有抬头。

“六里之事，大王恨臣，是人情。可楚若今日只看臣该不该杀，便少看了一条更要紧的路。”

屈原立刻出班：“大王，张仪最善于给陷阱另起名字。所谓路，不过是要楚再看秦一眼。”

张仪这才抬头看他。

“大夫说得不差。”他道，“臣今日来，本就是请大王再看一眼。可大王眼下若只剩齐一路可走，齐价便只会越来越重。秦今日来这一眼，至少能叫齐不敢把话开到尽处。”

昭阳站在班中，一直没有说话。

他最懂这句话里的险处。它不是要楚立刻弃齐，而是要楚拿秦去压齐。可你只要肯拿秦去压齐，心里那杆秤就已经先动了。

楚怀王盯着张仪，良久才道：“你说秦愿归地？”

张仪答：“不是尽归，是可议归还一部。大王若肯缓兵，两国把旧限重定，楚至少不必在齐案前处处低头。”

一句“至少不必处处低头”，便把殿中许多人心里没说出口的难处都点破了。

楚这些日子最苦的，已不是一线战败，而是求援时那种处处被人量、被人开价的难堪。张仪没有先说国家大义，也没有先说修好之美，只把这层最难堪的地方挑了出来。

屈原急道：“大王，秦若真有诚意，为何不先归地，再言后事？”

张仪转头答他：“秦先归地，楚仍可握齐而去，世上哪有这样两国相交？秦今日放话，不是求楚即刻绝齐，只是让大王知道，手里不止一条绳。”

昭阳这时也开口了。

“张仪，你的话绕得再平，终究还是要楚松齐。”

张仪看着他，道：“令尹，臣今日若说‘绝齐’，楚王反倒不会信臣。臣只请大王想一想，齐之援值不值得楚把自己整个人都先压上去。”

这话一落，殿中便再无人敢轻易说“杀”了。

不是怒散了，而是人人都明白，张仪已经把楚廷最难看的那一层挑到了明处。你若立刻杀他，等于承认楚不敢算；你若留他，等于承认楚还得算。

楚怀王缓缓靠回王座。

“你说得倒像寡人是在用你。”

张仪立刻俯首：“大王若能把齐、秦、楚三边利害都摆平，再择其重，那自然不是受欺，是用人。”

“用人”二字，叫楚怀王眼神轻轻一动。

屈原看见这一动，心里便是一沉。

他最怕的不是怀王再被一句大利骗住，而是怀王会开始相信，自己如今不是在被路逼，而是在“用”两条路。

殿中静了很久。

最后，楚怀王没有杀张仪。

“先退馆舍。”他说，“寡人自会再议。”

张仪再拜而退。

走出大殿时，廊下檐铃被风轻轻撞了一下。随从低声问他，楚王是否松了。张仪整了整衣袖，只说：

“还没松，只是那只手不再只想掐人了。”

而大殿里，秦书与齐书已被一起摆上了案。

楚怀王盯着它们看。

一份重，一份轻。

可真正扎人的，反倒是那份看起来更轻的。
