# 第二十八回 欲绝齐欢秦廷筹归地 将回旧路张子再入郢

齐楚议价的消息传到咸阳后，秦廷并没有立刻喧哗。

因为所有人都明白，齐国若真出手，未必能立刻扭转战局，却足以让原本已经一路往下滑的楚国，重新多出一口喘息。对于秦而言，最怕的从来不是楚一国硬撑，而是楚在最该孤的时候，偏又被别人接上一手。

咸阳宫中，秦王嬴驷召见张仪、樗里疾等数人，共议后策。

樗里疾先道：“齐会援楚，但齐兵来得未必快，且齐心不纯。若我军此时趁势再压，未必不能先把楚逼得更深。”

这话不差。

可张仪听完，却没有立刻附和。

他立在那幅熟悉的天下舆图前，看了许久，手指最后却落在了汉中一带。

“大王，”他说，“若此时只再加兵，固然可以压楚；可压得越紧，楚就越只能死死抱住齐。臣以为，这一步不如暂缓刀锋，先给楚看一条岔路。”

秦王问：“什么岔路？”

张仪抬头道：“归地。”

殿中几人同时一动。

秦王却并未惊，只问：“归多少？”

张仪道：“不必多，只需归还楚一部分汉中之地，或者许之以归。重点不在地多少，而在让楚王看见：若与秦再通，则不必处处向齐低头；若与齐绑得太紧，则眼前要先付出去的，未必比从秦手里拿回来的少。”

樗里疾眯起眼，看了张仪一阵：“你是要用地去换楚绝齐？”

张仪道：“不是换，是诱。楚国如今最难受的，并不只是败，而是败后四处求人。若这时有一条路看起来像是既能少低头，又能拿回一点失地，怀王就算明知秦不可尽信，也会忍不住算一遍。”

秦王缓缓踱了两步。

他当然知道，真正的秦国利益并不在于多还楚一点地，而在于让楚和齐之间那点刚刚谈出轮廓的相靠，再次生出迟疑。只要楚王一迟疑，齐王便会起疑；齐王一起疑，这场尚未真成形的“援楚”就又会从里面松掉。

这便是张仪最狠的一处。

别人看见齐楚将近，先想的是如何硬压其合；

他看见齐楚将近，先想的却是如何在两家还没真正抱牢之前，让其中一方自己先觉得：也许不必抱这么紧。

秦王终于停下，道：“你愿再入楚？”

张仪一揖到地：“臣愿。”

“你不怕怀王见你便杀？”

张仪起身，神色平静：“大王若只看怀王之怒，臣自然该怕；可若看怀王这些时日一路走来的心，臣便知道，他如今最盛的未必还是怒，反而是那股不愿再向齐低头太深、又想快些从败势里抽身出来的急。臣去，正是去替他把这股急意说成一条路。”

秦王看着这个人，忽然笑了笑：“张仪，寡人越来越明白，为何同样学纵横，你与你师弟走出来的是两条路。”

张仪没有接这句，只低头听命。

秦王道：“去吧。若能让楚再回头，寡人便再记你一功。”

旨意一下，张仪再度整装。

咸阳冬色正深，他出城那日，渭水边风极冷。随行人马不多，旗号也不张扬，可一路西东往来的驿骑只要认得出前头那道身影，心里便都知道，这不是寻常使节北风里的一次往返，而是张仪又一次要去拿一个已经被他骗过、恨过、打过的楚王做文章。

这趟路，走得比前一次更险。

前一次他去，楚国尚在利心初动之时；这一次他去，楚国已尝过他的毒。可也正因为尝过，他才更知道该往哪一处旧伤上再按。

消息比他更快，先一步到了郢都。

“张仪又来？”

楚怀王听见这个名字时，脸上那层好不容易压住的阴沉一下子翻了起来。他猛地把案上杯盏掀落在地，厉声道：“此人欺寡人至此，竟还敢再来楚！”

殿中群臣一片静寂。

许多人第一反应都是：杀了他。

可下一层念头紧跟着就来了：若张仪敢来，秦那边是不是另有话？若真另有话，而楚此时又正与齐讨价还价，杀了张仪，是否等于连听都不听便先把一条可能的路彻底斩断？

这种念头一生出来，殿中的杀气便不再那么整齐。

昭阳率先开口：“张仪可恶，然其此时敢来，必有后着。臣请大王不妨先见其意，再议杀留。”

有主战臣子立刻道：“令尹莫非忘了六里之辱？此人入楚，不杀何待！”

昭阳冷冷道：“正因未忘，才更不能只靠一口气。楚今日不是在昭阳府中争一时面子，而是在国势下行时，先看秦究竟还想从我楚身上取什么。”

屈原随后亦出列，道：“臣与令尹同意一半。可见，可以；信，不可。张仪这等人，最善趁人急时把路说得比路本身更平。大王若见，须先定一条：只听其意，不为其言先动。”

楚怀王听着二人所言，胸中怒意翻涌不定。

他当然恨张仪，恨到恨不得立刻见血。

可他也确实正被齐国那边的一纸纸条件压得心头发堵。若秦此时真肯归地，或者哪怕只是肯放出归地之意，对他而言都不只是得地失地的问题，而是终于可以不必把“求齐”这一条路走得那么低、那么尽。

他想到这里，自己先是一惊。

因为这等于承认：张仪人还没进郢都，他心里竟已开始顺着对方可能递来的那句话往下算了。

楚怀王沉默良久，最终冷声道：“传令边吏，放他入郢。寡人要亲自看看，他还能说出什么鬼话。”

命令传出后，屈原退下时，心情比来时更沉。

他很清楚，真正危险的从来不在于楚王会不会当场又信张仪一次，而在于张仪太懂得怎么把一个人心里本就有的犹疑、难堪、好胜与贪利，重新拎出来排成一条看起来极顺的路。

若一个人原本心里毫无此念，张仪未必能凭空造出一念；

可若那念头本就藏在心底，只是自己还不肯明认，那张仪一来，往往便能替它取个极好听的名字。

数日之后，张仪终于抵郢。

城门外风尘未洗，他抬头望向这座自己曾经折辱别人、也几乎要折进去自己的楚都，神色竟仍平静得很。仿佛六里之欺、丹阳之败、蓝田之挫、满朝对他的杀意，都不过只是他再入此城时必须顺手一并算进去的几层风。

随行秦卒中有人低声问：“先生，此行真有把握？”

张仪整了整衣袖，只道：“不是我有把握，是楚王心里那条路已经半开了。我来，不过是把门再推一把。”

说完，他提步入城。

郢都冬空低沉，风从城楼掠过，吹得旌角作响。

这一回，张仪再入楚，不是来骗一个刚动利心的怀王。

他是来骗一个已经吃过亏、却仍不肯彻底死心的怀王。

而后者，往往比前者更难防。
