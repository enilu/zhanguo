# 第二十九回 屈原力言先防旧术 昭阳冷算只留一线

张仪退入馆舍之后，郢都当夜便又召群臣。

殿门合上，外头风声都像被关在了门外，只剩案上那两份文书摆得扎眼。一份是齐人先前开的条目，厚，重，句句都明；一份是秦书，新，薄，字不多，偏偏每个字后面都像还藏着一层没说出来的余地。

楚怀王没有先说话。

他把两卷书翻了又翻，像是在掂一轻一重之间，到底哪一样更压手。

屈原先出班。

“大王，秦书不可留心。”他说，“张仪今日最毒处，不在归地，而在把‘再听秦言’换成了‘手里多一条路’。大王若信了这名字，齐约便先松了一半。”

楚怀王抬眼看他：“那依你，寡人便只听齐？”

屈原答：“议齐可以，轻秦不得。齐要价虽重，却是明着要；秦说归地，却要大王先把更重的东西自己松出去。”

昭阳随后出列。

“臣也不信张仪。”他说，“但楚若今日把秦一路彻底堵死，齐明日只会更重。故臣请留张仪，不许他乱走，不许他回秦，也不许他再轻见大王。楚仍与齐议，让齐知道，我楚不是只剩这一条路。”

殿中立刻分成了两股。

主杀者只恨没在殿上直接取张仪首级，觉得今日多留一夜，便是多生一分祸；主张先留者却都知道，眼下最怕的已不是信不信秦，而是若楚自己先把“只有齐一路”写在脸上，齐那边就再不会客气。

楚怀王听着，始终没有立断。

这在这些月里已算少见。

他前些时日最怕听人把话说慢，怕一慢便像自己理亏。如今仗败、边试、齐量、秦诱，一样样压下来，反倒逼得他不得不把“慢一慢再看”摆上案头。

群臣争到夜深，楚怀王终于挥手让众人退去，只留屈原与昭阳。

三人对着那两份书，殿里更显得静。

楚怀王先问屈原：“你还是要寡人今夜就断秦？”

屈原点头：“至少不能让大王心里先靠过去。秦此时递书，不是怕楚强，是怕楚与齐终于靠实。它越来得急，越说明齐这一步不能轻给它拆。”

怀王没有接，转而看向昭阳：“你却要留张仪。”

昭阳道：“留，不是信；按着，不是放。臣要的是让齐看见，楚还有第二只手。只要它不敢把大王逼到尽处，这第二只手便有用。”

“若大王自己先被这第二只手拽过去呢？”屈原反问。

昭阳沉声道：“所以臣才请张仪留馆，不许大王多见。”

这句说得很实，也很险。

险在它等于当面承认，张仪多见一次，大王心里那杆秤就会多晃一次。

楚怀王果然脸色一沉。

“你们都把寡人看得这样轻？”

屈原立刻道：“臣不敢轻大王。臣只是知道，张仪所说，多半不是大王本来没有的念头，而是大王心里本就会生出来、只是不愿先认的念头。这样的话，最难防。”

殿里又静了。

楚怀王缓缓起身，走下阶来，把齐书、秦书各拿在手里。

齐书重得很，像一块压在胸口的石；秦书却轻，轻得像一根针。石压人，人知道疼；针扎人，往往是一阵一阵地磨。

他掂了片刻，终于道：“张仪留馆，不许擅出。齐使也留下，再议。”

昭阳领命。

屈原还想再说，楚怀王却已摆手：“今夜到此。”

次日一早，郢都内外便都听出了味道。

秦使没杀，说明楚未绝秦；齐使还留，说明楚未舍齐。街巷坊间都在猜大王究竟会把手伸向哪边，越猜，便越说明楚眼下最要紧的已不是前线某一垒得失，而是这两条路究竟会不会把它往不同的方向扯开。

临淄得报时，齐王正与田文对坐。

听完之后，齐王把茶盏轻轻一放：“楚在拿秦来压寡人。”

田文道：“它越急，越要多拿一路。大王若此刻松得太快，后头它便会一路这样压价。”

齐王笑道：“那便再晾它几日？”

田文摇头：“晾得太狠，反倒会真把它晾回秦那边。最好是让它总觉得，只差再让半步，齐就肯动。”

齐王点了点头。

他当然明白，齐要的不是把楚真逼死，而是逼到一个刚好肯把该吐出来的东西吐出来、又还没绝望到乱扑别路的地方。

馆舍那边，张仪听完探报，只问：“大王今日可再召我？”

探者答：“没有。”

张仪反而笑了。

“没有便好。”

随从不解：“先生不是正要他动心么？”

张仪把灯拨暗了一半，只留案前一点火。

“人一动心，最怕旁人催。”他说，“让他自己拿齐书和秦书去翻，比我多说十句都更磨。”

那一点灯火落在他眼里，细而冷，像一枚藏得极深的钉。
