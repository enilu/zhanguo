# 第三十回 楚廷两议都难尽握 怀王一念先求牌多

郢都又议了三日。

齐使住在东馆，张仪住在西馆，隔着几条街，各自不见面，城里却处处都在替他们说话。朝中每开一回议，便像把齐、秦两边的分量又拿出来重新称一遍。称来称去，谁都知道最难的不是选哪一边，而是两边都想先捏着一点。

这本就是怀王眼下最动心的地方。

他已尝过只信一边的苦，所以更想手里多捏一张牌。多一张牌，齐便不敢压到底；多一张牌，也许秦那边就真能放出一点地来。

可牌这东西，一旦只是想拿来压人，常常先压动的，反倒是自己的心。

这日再议，屈原还是先开口。

“大王，楚眼下最怕的不是齐价重，而是心先乱。”他说，“若今日既不肯把齐抱实，又去拿秦虚压齐，后头最容易成的，不是楚多一路，而是两边都觉大王不定。”

怀王皱眉：“那便叫寡人单押齐？”

“若楚真要齐动，便得先让齐知道，楚不是今日来议、明日便拿秦书又回去翻。”屈原道，“国与国之间，最怕的不是条件多，而是看不见你究竟站在哪里。”

昭阳接过话来：“屈大夫说的是一头。可臣要补另一头。齐若知楚已无别路，它开的便不只是条件，是刀。故臣仍请留张仪在馆，不必多见，只让齐知道秦门未死。”

屈原道：“可大王自己会去看那扇门。”

昭阳没有立刻回。

这一句话，正是他也最担心的地方。

楚怀王坐在上首，听得脸色愈沉。沉不是怒，而是被两边都说中了难处。殿上众臣这时候也少有人再喊“只战”或“只杀”了，因为事情走到今天，楚已被迫从一个只能在前线见输赢的大国，变成了一个在每一份来书里都得算自己还剩几张牌的国家。

议到午后，楚怀王忽然问：“若寡人现在就定齐，齐会立刻出兵么？”

殿中一时无人敢先答。

因为谁都知道，答案多半是不会。齐会接得更实，却不会因为楚今夜拍板，明日便把十万兵全推出去。它仍会算，仍会量，仍会把该收的先收一点。

屈原于是道：“未必立刻动，可总比楚这样两头晃更能逼它动。”

昭阳却道：“也未必。齐若看见大王已彻底背过身去，它反倒更有恃无恐。”

两边都不是假话。

也正因为都不是假话，怀王越听越难定。

散议之后，他没有立刻回后殿，只独自留在案前。齐书、秦书、前线军报、东边边报全堆在一起。风从殿门缝里灌进来，把最上头那卷秦书页角吹得轻轻一翘。

他伸手按住。

案上的东西，竟没有一样是让人舒心的。

齐书太重，重得像每让一步都要切下一块肉；秦书太轻，轻得像一捏便碎，却偏偏总在提醒他，也许还能拿它去换齐再退半步。

许久之后，怀王唤来近臣。

“齐使留下，再议。”他说。

“张仪也先留。”

“但不许他擅出。”

三道令发出，意思已很明。

楚不绝齐，也不绝秦；不肯全抱死哪一边，只想先把两边都按在自己案上。

消息一出，临淄与咸阳都明白了。

临淄那边，齐王笑道：“怀王还是想牌多。”

田文道：“人到这个份上，不求定，先求牌多，是常情。”

“那我们如何？”

“让他总觉得牌还在手里，”田文答，“却又不让他觉得齐非他不可。这样他才会一面捏着秦书，一面继续把该让给齐的慢慢让出来。”

同夜，张仪在馆中听完探报，也只笑了笑。

随从问：“先生，这算成了么？”

张仪摇头：“不算成，只算那杆秤已经离了原来那一点。”

“接下来还见大王？”

“不急。”张仪说，“眼下最有用的，不是我再去说，是让他自己看着齐书，想着秦书。”

他把案前那盏灯吹暗了半截。

屋里顿时阴下来许多，只剩一点火还亮着。

那点火照着桌上楚地边图，也照着张仪的手指。指尖没有落在汉中，也没有落在郢都，最后只是停在齐楚之间那条尚未连死的线上。
