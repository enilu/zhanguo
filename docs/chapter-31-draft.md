# 第三十一回 齐廷观望不肯先失价 楚王自用反把旧盟轻

楚王下令“齐使再议，张仪暂留”之后，楚齐秦三边局面忽然慢了下来。

可这种慢，不是缓。

它更像是三方都把刀暂时藏回袖中，先看别人会不会先露出更急的一面。

而一旦局走到这种时候，最先吃亏的往往还是最急的人。

楚国，偏偏就是最急的那个。

因为楚怀王表面上虽把两路都留着，心里却已经越来越不愿继续受齐国那股“我援你，所以你先得低头”的气。他如今每看一份齐国回书，最先跳进心里的已不再是“齐终究可能出兵”，而是“齐又要什么”。这份心气一变，整个人便会不自觉地顺着另一个方向去想：

若秦真肯归一些地、缓一些压，我何苦还要忍齐人如此开口？

这念头一起，便极难再压回去。

屈原很快便察觉了。

他发现楚怀王近来每逢议齐，问得越来越少；每逢听到“秦使仍在馆中安静候命”“秦边近日兵势略缓”之类的话，反倒会多停一会儿。那种停，不是信任，却是一种很危险的自我解释。人一旦开始主动替一条本不该轻信的路找根据，后面往往只需再轻轻推一把，就会自己往前走。

于是屈原再谏。

“大王如今最当防的，不是秦再欺，而是自己心里先嫌齐太甚。”

楚怀王本在看一份齐国回书，闻言抬头，脸色已带不悦：“齐不甚么？齐开口越来越深，寡人还不能嫌？”

屈原道：“能嫌。但大王若因齐索价太深，便忘了秦是为何急着在此时示好，那便是用一时憎齐，替秦开门。”

楚怀王冷笑：“依你看来，寡人只能受齐人拿捏？”

屈原沉声道：“臣不是叫大王受齐拿捏，是叫大王分清哪一种难堪更短，哪一种后患更长。齐是趁楚急而要利，难听，却在明处；秦是见齐楚将近而来拆，甜些，却在暗处。大王若只图眼前不再低头，后面怕是要把更多东西从更低处还回去。”

这话说得极重。

楚怀王脸色沉了半晌，终究还是没有当场发怒，只把那份回书往案上一丢：“退下。”

屈原退出后，只觉心里更沉。

因为他看得出来，怀王不是没听懂，而是正因为听懂了，才更不愿意承认自己心里已经开始偏向那条“看起来更不丢脸”的路。

另一边，齐廷也没闲着。

楚使再入临淄后，双方已把若干条件谈得更细。可越细，齐王越看得出楚国如今的虚。因为楚使每一次争，都不是真能转身就走的争，而更像一种“总不能让得太快太尽”的勉强支撑。齐王见得越明，心里便越不愿先松。

田文因此进言：“大王此时若轻降其价，楚只会觉得一压便有效。倒不如再稳两步，叫它自己更清楚：若无齐，眼前便更难。”

齐王道：“可若真稳得太久，张仪那边生出别样花样，又如何？”

田文笑了笑：“张仪当然在生花样。可大王不必怕他所有花样，只须记住一件事：楚如今最想要的，不只是援，而是不想太低头。谁能让它少低一点头，它心里便会向谁歪一点。故我齐要紧的，不是只说重话，也要适时给它一点‘似乎快要谈成’的样子。”

于是齐廷回书往郢都时，语气果然软了半分。

不是真软。

只是把原本全写在前面的利，略往后收了一层；把原本过硬的字眼，换成了“可再商议”“未必不可通融”这一类稍显活的说法。这样一来，楚若仍肯继续谈，便会觉得齐并非全无松口；可真要它以为齐已退让到位，却又还差得远。

这本是齐国最擅长的火候。

可这一回，火候再好，也架不住楚怀王自己的心先偏了。

他看完齐国新回书后，沉默了很久，最后竟说了一句：“他们到现在，还在教寡人如何来求。”

这句话轻得很。

可旁边侍立的内臣听见，心里都明白：大王对齐的不耐，已经不是一时火气，而是日日累出来的真厌了。

而张仪偏在这时，递了一封极短的书。

他没有再求见，只请馆吏转呈一句话：

“秦所重者，不在逼楚低头，而在与楚各释前隙。大王若愿再听一言，臣可明陈归地与修边之实数；若不愿，臣即刻请归，绝不再扰楚廷。”

这封书最厉害处，不在多说，而在“不多说”。

因为怀王眼下最怕的，就是看起来像自己又被张仪牵着走。张仪若日日求见、句句相逼，反倒会激起他的逆。可如今只递这样一封短书，倒像是在把选择权全放回怀王手里。

而一个本就已经在心里替自己找理由的人，最受不得的往往正是这种“你若要听，也不是我逼你，是你自己要听”的递法。

楚怀王看完之后，果然沉默许久。

他没有立刻召张仪，却也没有把那封书丢开，只慢慢放回案上，道：“明日再议。”

这一夜，昭阳再入宫时，正碰上怀王独坐灯下。

昭阳一看案上，便知张仪那封书已到了。他心里一紧，却仍先按礼行事，随后才道：“大王要再见张仪？”

楚怀王没有正面答，只道：“若他真肯先把归地与修边说到实处，寡人总不能连听都不听。”

昭阳沉声道：“听，可以。可大王一旦再单独见他，齐那边必闻风而疑；齐一疑，条件便未必还如今日这般可商。臣请大王若见，须先压住消息，且同时催齐定议，不可让楚先自己把路走成偏向。”

这已是极稳的补法。

可昭阳自己也知道，走到今天，补得再稳，也只是补。真正的主动，楚已经丢了太多。

楚怀王点了点头，像是听进去了。

可昭阳退出之后，心里却越发不安。因为他看得出来，怀王如今已不是在“防张仪”，而是开始在期待张仪真能说出一点让他不必再受齐气的话来。一个君主若到了这一步，旁人再怎么替他布防，终究还是隔着一层。

次日清晨，宫中再传令：

张仪可于次日复入，单陈秦意。

消息传到馆舍时，张仪只是轻轻抚平衣袖上的一道折痕，脸上连笑都不必有。

因为他知道，最难的从来不是第一次把门叫开，

而是叫开之后，让门里的人自己觉得：再开一次，也不过是我在做主。
