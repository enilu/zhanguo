# 第三十二回 再入楚宫张仪陈实数 一松齐约怀王失定心

次日再入楚宫时，张仪比上回更静。

上一次，他要做的是先把命留下，再把话塞进怀王耳里；这一次，他要做的便不是破门，而是顺着已经开出的那道缝，把“可听”推进成“可用”。

楚宫大殿里，杀气虽不如昨日那般逼人，冷意却更沉。

因为所有人都知道，第二次见面比第一次更险。第一次还能说是大王欲知秦意，不得不听；第二次若再听下去，便说明事情已开始往真正可议的方向滑了。

楚怀王照旧高坐其上，脸色阴沉，却已少了前一日那股几乎要立刻见血的暴怒。他盯着张仪，第一句话便道：“昨日你言归地与修边，今日寡人不听空话，只要实数。”

张仪一揖，道：“臣正为此来。”

他随即展开秦书，竟也不多绕，直把几处可议地界、边吏可撤之处、两国使节往返与暂息兵争的次第一条条说出来。没有一句说满，也没有一句空飘，像一个真正来谈边事的使臣，而不只是一个靠唇舌卖巧的纵横家。

这反倒比昨日更危险。

昨日他诛心，今日他落地。诛心之言，人还能以“此人口巧”自防；可一旦开始听见边界、城邑、使节、缓兵这些字眼，朝堂上的许多人便会不由自主地开始跟着算：这事若真谈成，对楚当下究竟能少几分难堪、争几分喘息。

屈原首先出班，道：“大王，秦书说得越细，越说明它此来不是为和，而是为拆。若真有诚，何以偏等楚与齐将成未成之时方来？”

张仪转头道：“正因臣不讳此事，故更说明今日楚国最该看的，是局而不是气。齐楚将近，秦来言和，难道大夫便因此断定秦意全假？若楚今日因此连一听都不听，那岂不是等于向齐明言：楚无论你开多少价，都只剩顺着走这一条路？”

屈原道：“楚若因此转念，齐便先疑。齐一疑，楚先失外援。秦未真让，楚已先松，这笔账只怕算得太早。”

张仪道：“故臣今日并未请大王绝齐。臣请的，只是让楚不必先把自己锁死在齐人案前。大王若仍与齐议，秦并不拦；可大王若手里明明还能多一条路，却偏要自己折去，那便不是谨慎，是自困。”

昭阳也在此时出言，问得更直接：“你言归地，可先归多少？若只是口头许诺，楚岂能因你一句话先松齐约？”

张仪看向昭阳，答道：“令尹要的，是能叫楚今日拿在手里的实物，这臣明白。臣不敢妄许尽归，只敢言：秦可先归楚边数城，以示非虚。余者再议。至于楚齐之约，楚可继续谈，秦亦不逼大王立断。臣只求大王别把自己逼成必须先向齐尽让的局面。”

这一句“先归数城”，像石子入水，殿中顿时起了极细的波。

楚怀王的手指也不易察觉地停了一下。

他这些日子最想要的，不是长久大安，那东西太远；他真正想要的，是眼前立刻能拿来压住朝堂、压住齐使、压住外间流言的一点“实得”。秦若真肯先归数城，那便不只是面子，而是能立刻摆到案上的东西。

这正中他心里最难压住的一处。

屈原自然也看得清，遂急道：“大王，正因它肯先给一点，才更当防后头拿得更深。张仪最擅者，从来不是一开始便取尽，而是先让人觉得‘这一点总不妨’，后头再叫人自己把更大的口子松开。”

张仪不与屈原缠，只转向怀王，道：“大夫忧后患，臣不敢说其无理。可后患未至，眼前之局却已逼人。大王如今对齐让一步，齐便进一步；若秦先归数城，齐国还敢如此压价么？大王到时再议齐，不是更从容些？”

楚怀王终于开口：“若寡人收了你秦数城，齐国怒去，又当如何？”

张仪道：“若齐因楚多一张牌便怒去，恰说明齐本就不是为楚而来，而是为价而来。如此之齐，大王纵今日尽让，明日便真能与楚同心么？”

这一句，正把怀王近来对齐的不满尽数挑了起来。

他想起齐使一次次在殿上讲后利、讲旧议、讲明约，想起自己每看一封齐书便觉得像有人在拿尺子量楚如今还能低到哪里。那股气闷本就日日积着，此刻被张仪一点，竟比昨日更容易翻上来。

怀王沉默许久，忽然道：“若寡人准你这条路，楚并未绝齐。你秦先归城，寡人再看其后。若秦敢虚，此番便不是六里之辱，而是你张仪自求死地。”

殿中顿时大震。

屈原上前一步，还欲再谏，楚怀王已抬手止住：“寡人并非尽信秦，只是先收秦之实，再观齐之心。此事寡人自有权衡。”

这句话一出，张仪心里便知道，最难的一道关又过去了。

因为怀王终于从“听”走到了“先收一点再看”。这一步看似不大，实则已经意味着楚廷的轴心从“防张仪”转成了“如何利用张仪给的东西去压齐”。

而一旦轴心转了，后头很多原本不会发生的让步，便会显得越来越像顺势。

退朝之后，屈原独自立在廊下，许久未动。

他听见殿内已有人在低声议“先归何城”“齐使闻之将如何变色”，只觉胸口像被一只手慢慢攥紧。最可怕的事终于还是发生了：朝堂讨论的重心不再是“该不该信秦”，而变成了“秦先给多少，齐会退几分”。

一旦到了这一步，合纵那条线就不是被人从外面一刀斩断，

而是会在一屋子自以为都还在算计别人的人手里，一寸一寸自己松开。

齐国那边，也很快闻得风声。

临淄朝上，齐王听说秦将先归楚数城，脸色当场便沉了几分：“楚竟真敢接？”

田文也皱起眉来。

他当然知道楚会拿秦压齐，可他原本料的是怀王还要再拖一拖、再试一试，不至于这么快就肯先把秦意往实里接。如今楚若真先收秦城，那齐手里原本那口压着楚的气，便要先散去一块。

田文沉声道：“大王，不可再稳了。再稳，楚便真要借秦把自己抬起来。”

齐王问：“那便降价？”

田文摇头：“可松，但不可叫楚看出我齐是被秦逼松。否则后面它更会来回挟两家取利。最好是我齐主动往前一步，仍占体面，却把它往我齐这边再拉一拉。”

齐王听完，长长吐出一口气。

他终于明白，这一局到了这里，已经不是谁先开价的问题，而是谁先露出自己开始怕对方真走向另一边。

而在这件事上，齐终究还是先露了一分。
