# 第三十二回 张仪再入先陈城数 怀王听实已乱定心

第二次见张仪，楚宫里反而更冷。

第一次还能说是“听听秦意”，杀气摆在明处；这一次若再开殿，便说明事情已从“可听”往“可议”上滑了一寸。许多人站在殿里，话还没出口，心里就先沉了一层。

楚怀王坐在高处，没让张仪多行虚礼。

“昨日你说归地与修边。”他说，“今日寡人不听空话，只要实数。”

张仪一揖到底，随即展开秦书。

这一次，他不讲大义，也不讲前嫌。开口便是边城、吏口、缓兵日限、使臣往返。哪几座城可先归，哪一段边限可先撤一层，哪一处不敢妄许，他都说得极清。这样一来，堂上的人反倒更难把他当成一个只靠口舌卖巧的说客。

落到实处的路，最容易叫人跟着算。

屈原第一个出班。

“大王，秦今日说得越细，越说明它来此不是为和，是为拆。”他说，“若秦真有诚，为何偏在齐楚将成未成时才来？”

张仪转头看他，神色仍平。

“正因齐楚将近，秦才更不愿见楚只剩一条路。”他说，“大夫若因此便叫大王连看都不看，岂不是亲手告诉齐王，楚无论他如何开口，终究还是只能顺着走？”

屈原厉声道：“楚若因此先松，齐必先疑。齐一疑，楚便先失外援。”

张仪道：“臣今日并未请大王绝齐。臣只请大王别把自己先锁死在齐人案前。”

昭阳这时也开口了。

“你说先归城，”他问，“先归多少？”

张仪看向他：“不敢尽许，只敢言先归边上数城，以示非虚。其余后议。楚齐之约，楚可继续谈，秦也不逼大王立断。”

一句“先归数城”，殿中便起了极轻的一层响动。

不是因为这几座城有多大，而是因为楚怀王眼下最缺的，正是这种立刻可以摆到案上的“实得”。前线败、边上试、齐在量，朝堂上下要看的，不是十年之安，而是今天能不能先拿一件东西压住这口气。

屈原看得分明，急忙再谏：

“大王，张仪最狠处，从来不是一开始便取尽，而是先给一点，让人觉得‘这一点总可先收’，后头再逼着人自己把更大的口子松出去。”

楚怀王没有立刻看屈原。

他还在看秦书上那几处城名。

张仪便在这时又补了一句：“大王若先收数城，再看齐书，不是更从容些？”

这话正中楚怀王心里那一处最难说出口的地方。

前些日子他最恨的，并不只是齐要利，而是齐每要一步，都像在提醒他如今楚有多急、多低。若秦真先给几座城，哪怕只是几座边城，那齐使明日再进殿时，楚也至少不至于还像如今这样只剩一个求字。

楚怀王终于问：“若寡人先收你秦数城，齐一怒而去，又如何？”

张仪答得极快：“若齐因楚多一张牌便怒去，恰说明它本不是为楚，只是为价。如此之齐，大王纵今日尽让，来日真能与楚同心么？”

这一句，把楚怀王近来对齐的不满一下都翻了上来。

殿里静得可闻风声。

过了很久，楚怀王缓缓道：“若寡人准你这条路，楚并未绝齐。你秦先归城，寡人再看其后。若再虚，便不是六里之辱，是你张仪自求死地。”

殿中顿时一震。

屈原上前一步，还欲再言，楚怀王已抬手止住：“寡人并非尽信秦，只是先收其实，再观齐心。”

张仪低头领命，脸上没有露喜色。

可他心里清楚，最难的一层已经过去了。楚廷议事的重心，已不再是“该不该信秦”，而是“秦先给多少、齐会退几分”。轴一旦偏了，后面许多看起来像顺手为之的动作，都会把这偏再往前推一寸。

退朝之后，屈原独自站在廊下，久久没有动。

他听见殿里已有人在低声议“先归哪几座城”“齐使闻之又当如何”。这些话听起来都像在替楚算，偏偏正因为都像在算，才更叫人发冷。

合从最坏的坏法，从来不只是有人翻盟。

而是满殿都还自认在为国谋利时，脚下那张网已开始自己松。

消息很快传到临淄。

齐王看完楚那边的密报，脸色立时沉下去：“怀王真敢接秦城？”

田文也皱了眉。

他原本料怀王还会再扛一扛，不至于这么快就把秦意往实里接。如今楚若真先收城，齐手里那把原本量楚的尺，便要先短一截。

“大王，”田文道，“不能再照旧压了。”

齐王反问：“降价？”

田文道：“可松，但不能叫楚看出我是被秦逼松。最好是齐自己先往前一寸，让它觉得这一寸是大国自定，不是被张仪赶出来的。”

齐王听完，缓缓吐出一口气。

他当然明白，这意味着齐先露了一点怕。

可若此时还强撑着不松，怕的便不止是一点脸面了。
