# 第三十三回 邯郸夜里苏秦更信 诸国案前众口先轻

邯郸那夜风极硬。

苏秦把楚地急报看完，又从头看了一遍。报上写得不长，只记秦再入楚廷、怀王准其“先收再观”、齐闻风后口气已变。字不多，却像一层层把这几个月辛苦拢起的那面旗又吹松了一寸。

门客站在一旁，许久才敢低声问：“相国，楚真要先收秦城？”

苏秦把书卷慢慢合上。

“收不收，已不是最重的。”

门客一怔。

苏秦望着灯火，道：“重的是它已肯这么想。”

这句话一出，屋里便更静了。

他起身走到窗前，隔着纸窗看外头沉沉夜色。那夜色里有邯郸城，也有他早年奔走赵地时一回回走过的道。那时他费尽力气，是要把六国都只顾自己门前那点安危的心，先往西边拽成一线；如今那条线还在，可绳股已经一根根自己松起来了。

“还去齐么？”门客问。

“去。”苏秦道，“还要去赵。”

“可他们如今多半都要说，楚自己都在另找路，我为何还守旧盟。”

苏秦点头：“他们当然会这样说。”

门客苦笑：“既知如此，还去？”

苏秦转过身，神色反倒更平。

“正因他们会这样说，我才更要去。若连我都不去问一遍，后头人人都能把退半步说得更轻巧。”

次日他先入赵。

赵王仍以旧礼待他，群臣也不敢轻慢这位佩印动天下的相国。可礼数归礼数，一说到兵和路，殿上的气立刻就凉了。

苏秦没有绕，只道：“臣今日不单为楚来，也为赵来。楚若先被秦拖出去单谈，后日被单拖出去的，未必不是赵。”

赵王听完，久久没有立刻应。

倒是一名近臣先道：“相国所忧，赵岂不知？可赵若此时先动，而楚转头又与秦缓和，赵岂不成了先把锋露出去的一国？”

苏秦看着那人，缓缓点头。

“正是难在这里。”他说，“诸君不是不懂，只是越懂，越不肯先担那一步。”

殿中一时无人接。

赵王轻叹，道：“苏子，赵近秦而重，走一步便是一大步。你要寡人此时先替楚、齐、秦这盘未定之局把赵压上，寡人不能不慎。”

赵王说得很实。

苏秦听完，只拱手一礼，没有再强逼。再逼下去，便真要像拿大义去压别人替旁人吃眼前的刀了。

出赵宫后，他又转去大梁。

魏王这回甚至连书卷都没放下，只淡淡道：“楚既能听秦，魏便更不敢先为楚受秦锋。”

这一句，比赵廷一殿的议论都更冷。

苏秦站了一会儿，才道：“若人人都这样想，秦后头便更容易一国一国去问。”

魏王抬头看他：“相国说得不错。可魏眼下能做的，先是顾魏。”

说到这里，已无可再说。

苏秦退出来时，大梁城里天已近暮。马车回馆一路不快，轮声碾过冻硬地面，响得很空。

夜里，他把赵、魏两处答话各记在一片竹简上，写到一半，笔忽然停住。

门客不敢出声。

良久，苏秦才低低说了一句：“师兄好狠。”

门客抬眼看他。

苏秦用笔尖在案上轻轻点了几下，一边点，一边道：

“楚可以说，我只是多留一路。”

“赵可以说，我不能先露锋。”

“魏可以说，楚既未定，我何必先担。”

“齐也可以说，它不过把账算细些。”

他说完，屋里只剩灯花一响。

门客过了半晌才道：“这些话……也不算全错。”

苏秦笑了一下，笑极淡。

“正因不全错，才最难。”

他随即提笔，给齐国写了一封极短的书。

书里没有铺陈，只一句：

“今日楚可被秦拖去单谈，明日齐亦可被秦另开一路。”

写完之后，他没有立刻封缄，而是又取一片空简放在旁边。

门客问：“这一片写给谁？”

苏秦道：“先空着。”

“为何空着？”

苏秦看着那片空简，像也在看后头还没到、却已能看见影子的日子。

“等再走一处，看他们怎么回，再写。”

窗外风声更紧，纸窗微微作响。

他一只手按着已写好的齐书，另一只手按着那片空简。

像是同时按着两样东西。

一样是还得去说的话。

一样是已经越来越难再说动的人心。
