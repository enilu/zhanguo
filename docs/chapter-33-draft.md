# 第三十三回 邯郸夜寒苏子知局坏 一封急书难挽诸侯心

邯郸那夜风硬，吹得纸窗一直轻响。

苏秦披衣坐在灯下，把楚地来的急报从头看到尾，又重新看了一遍。

书里写得并不长，只说秦使再入楚廷，楚王许其“先收再观”，齐国闻讯，口风随之微变。

门客等了许久，才低声问：“相国，楚真要收秦城？”

苏秦把书简放下，道：“收不收已不是最重的。”

门客一怔。

苏秦抬眼看着灯火，道：“重的是它已肯这么想。”

这句话落下，屋里便更静了。

风还在窗外吹。

苏秦起身走到窗边，隔着纸窗望向夜里沉沉的邯郸城。那一片暗影之中，他忽然想起自己最早在赵地奔走时的情形。那时他所费力气，是把各国都只顾自己门前的心，往西边拽成一线。

如今那条线还在。

只是绳股已经开始自己松了。

门客跟到身后，小心问道：“那相国还去齐么？”

苏秦道：“去。还要去赵。”

“如今他们多半会说，楚自己都在另找路，为何还要我守旧盟。”

苏秦点头：“他们自然会说。”

门客苦笑：“既知如此，还去？”

苏秦转过身来，神色反倒平了些：“正因他们会这样说，我才更要去。若连我也不去问一遍，后面人人都可以把话说得更轻巧。”

次日，苏秦先入赵。

赵王仍以旧礼待他，朝中也无人敢轻慢这位佩印震世的相国。可礼数归礼数，一论到兵，殿上的冷气立刻就出来了。

苏秦也不绕，开口便道：“臣今日不为楚而来，为赵而来。楚若先被秦拖出去单谈，后日被单拖出去的，未必不是赵。”

赵王没有立刻作答。

倒是座中一名近臣先应道：“相国之意，赵岂会不知？可赵若此刻先动，而楚转头又与秦缓和，赵岂不反成先把锋露出去的一国？”

苏秦看着那人，缓缓点头：“正是难在这里。诸君不是不懂，只是越懂，越不肯先担那一步。”

殿中一时无人接话。

赵王轻叹一声，道：“苏子，赵近秦而重，走一步便是一大步。你要寡人此时为楚、齐、秦这盘未定之局先把赵压上，寡人不能不慎。”

他说得已经很实。

苏秦听完，只拱手一礼，没再强逼。

因为再逼下去，便要像是拿大义去压人家替别人吃眼前的刀了。

从赵廷出来后，他又转去魏。

魏王听完来意，连案上的书都没放下，只淡淡道：“楚既能听秦，魏便更不敢先为楚受秦之锋。”

这一句，比赵廷里一殿的议论还冷。

苏秦站在那里，过了片刻才道：“若人人都这样想，秦后面便更容易一国一国去问。”

魏王抬头看了他一眼：“相国说得不错。可魏眼下能做的，先是顾魏。”

苏秦没有再说什么，行礼而退。

回到馆舍时，夜已深了。

他把赵、魏两处的答话各记在一片竹简上，写到一半，笔忽然停住。

门客在旁不敢出声。

良久，苏秦才低声道：“师兄好狠。”

门客抬头看他。

苏秦把笔搁在案边，慢慢道：“他最狠的，不是骗楚，也不只是折我。是把后面人人要替自己开脱的话，都先替他们备好了。”

他说一句，便在案上点一下。

“楚可以说，我只是多留一路。”

“赵可以说，我只是不能先露锋。”

“魏可以说，楚既未定，我何必先担。”

“齐也可以说，它只是把账算细些。”

屋里只听得见灯花轻响。

门客听完，半晌才道：“这些话……也不算全错。”

苏秦笑了一下，那笑薄得像刀背上一层霜：“正因不全错，才更难。”

他说罢，重新提笔，给齐国写了一封短书。

书里没有铺陈，只一句意思：

“今日楚可被秦拖去单谈，明日齐亦可被秦另开一路。”

写完之后，他没有立刻封缄，只把那封书压在案边，又另取一片空简。

门客问：“这是写给谁？”

苏秦道：“先空着。”

“空着作甚？”

苏秦看着那片空简，像也在看后面还没到、却已能望见几分影子的日子：“等再去一处，看他们怎么回，再写。”

门客没有再问。

窗外风过更急，灯火微微摇了一摇。

苏秦伸手把那封写好的齐书按住，另一只手却还压着那片空着的竹简。

像按着两样东西。

一样是还得去说的话。

一样是已经越来越难说动的人心。
