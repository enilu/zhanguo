# 第三十四回 齐使闻变终松先前口 楚王得城更觉张秦可借

秦先归楚边数城的消息，传到临淄时，齐廷上下都静了片刻。

不是因为这几座城本身有多大分量，而是因为它们来得太准。

齐国这些日子拿准了楚急，所以一路压着价往前谈；眼看楚虽难受，却还没真转身，齐廷心里原本已有七八分把握，觉得只要再稳上一阵，楚终究还是得回来把那一层更低的头低完。谁料秦竟在此时真先给了点东西。

这一下，齐国手里那把原本握得极稳的尺，便不由自主先短了一截。

田文入见齐王时，殿中正摆着楚、秦两边新到的文书。

齐王把秦归楚城的消息指给他看，道：“张仪终究比寡人更敢先下本。”

田文看完，沉声道：“不是他敢，是他知道此时最值。大王若再照旧压价，楚那边只会越觉得：齐只会要，秦反能先给。”

齐王冷笑：“给？秦今日给几座边城，明日未必不要更大的。”

田文道：“臣亦知秦不可信。可楚怀王眼下最先感到的，不是后日大患，而是今日谁让他没那么难堪。此处若输，后面再讲大势，怀王也未必听得进去。”

齐王久久不语。

他当然明白，自己真正输掉的不是几座城，而是张仪先一步拿住了楚怀王此刻最重的一层心气。到这一步，齐若还想把楚重新拉回自己这边，就不能再只是摆出“我也知大局，只是你先让我看够后利”的样子，而得让楚真切地感到：齐这只手也能给它一点立刻可握的东西。

于是齐廷终于松口。

不是大松。

只是把原本最伤楚颜面的几条收了一收，把若干必须先讲明的后约，改成了可以战后再议；又允诺一旦两国结兵，齐可先发一军，以示不是空言。这样一来，齐仍旧拿着自己的利，却已不再让楚国觉得自己是在一阶阶被人压着跪下去谈。

消息传回郢都，朝堂上的气氛立刻变了。

先前主张“齐压得太狠，不如借秦压齐”的那一派，此刻都微微一愣；而屈原等人则终于松了一线，因为他们看得出来，齐这一退，未必出于真心，却至少意味着楚还不至于立刻被彻底拖向秦那边。

可楚怀王看完齐书时，脸上的神情却比众人更复杂。

他当然知道齐这一松，是被秦逼出来的。

也正因知道，他心里反而更生出一股难言的意味：原来只要手里有秦这一张牌，齐便真会退。

这感觉极容易上瘾。

因为它像在向一个近来处处失势的君主证明：你还没有彻底失去主动，你只是先前手里少了东西。如今东西回来了，人便又肯看你脸色几分。

这种“主动似乎又回来了”的感觉，比归城本身更危险。

屈原见怀王神色不对，立刻进言：“大王，齐今日退，不是因我楚已稳，而是怕我楚真偏向秦。故越到此时，越不可因眼前一时得手而轻秦重齐、或轻齐重秦。楚要的是把局稳回，不是再靠一时机巧去左右横跳。”

楚怀王听到“左右横跳”这种近于刺耳的意思，脸色顿时一沉：“寡人何时在玩机巧？寡人是在救楚。”

屈原拜道：“臣知大王为楚。正因如此，臣才请大王别把今日之验看得太美。齐一退，大王便若觉得‘张仪果然可借’，那后面便更容易一步步借得自己也收不住。”

怀王没有接话。

可屈原知道，他最担心的事已经开始露头了。

因为怀王如今不再只是觉得“可以暂听秦”，而是开始尝到“借秦果然有效”的甜头。人一旦尝到一次有效，后面再要停手，往往比最初完全未动时难得多。

昭阳也看得清。

他当夜入见，只说了一句：“大王今日看见的是齐退了一步，臣看见的却是楚往后每退每进都更难。因为齐会记得是被逼退的，秦也会记得是它先给城逼退了齐。两边从今往后都要在楚身上拿回这一手。”

楚怀王听后沉默良久，才道：“寡人又何尝不知？可若不如此，寡人此前岂非真的只能任齐开口？”

昭阳一时竟无言。

因为怀王这一问，恰恰问在最难处。楚国走到今天，不是因为不知道谁危险，而是因为危险已四面都在。它只能在更短的难堪与更长的后患之间不断挪步，偏偏每一步都不干净。

而在馆舍中的张仪，得知齐果然退价后，只轻轻笑了一声。

随从道：“先生，齐既退，楚是否还会继续偏向秦？”

张仪道：“会，也不会。”

随从不解。

张仪道：“楚不会立刻弃齐，因为它还想把两边都握着；可也正因它还想都握着，齐与楚之间原本最要紧的那层‘先真心并力’便更回不去了。齐会疑楚，楚会嫌齐，双方越谈越像买卖。到那时，便是表面结兵，骨头也已不似从前。”

说完，他抬眼望向窗外夜色。

“对秦来说，”他慢慢道，“很多时候，不必真把他们全拆开。只要让他们抱在一起时都还各自想着：我得先防对方一手，便已够了。”

这句话，比“绝齐”更冷。

因为它意味着，真正成熟的破盟，并不总靠轰然断裂，更多时候靠的是把同盟变成一副人人看着还在、用起来却总觉得不敢尽信的空架子。
