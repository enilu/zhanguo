# 第三十四回 齐书终退先前重口 怀王得城更想借秦

秦先归楚边城的消息一到临淄，齐廷上下先静了一瞬。

不是因为那几座城有多大，而是因为它们来得太准。

齐这些日子一直拿准了楚急，所以一步步把价往前抬。眼看楚虽不舒服，却还没真转身，齐王心里原已有了七八分把握，觉得只要再稳上一阵，楚终究还是得回来把更低那一层头低完。谁料张仪偏在这时真先给了点东西。

这一下，齐手里那把量楚的尺，先短了一截。

田文入见时，齐王正把楚、秦两边新到的书并排摊在案上。

“张仪终究比寡人更敢先下本。”齐王道。

田文看完，沉声回道：“不是他敢，是他知道此时最值。大王若还照旧压，楚只会越觉得：齐只会要，秦反能先给。”

齐王冷笑：“给？秦今日给几城，明日便未必不要更大的。”

田文点头：“臣也知秦不可信。可怀王此刻最先感到的，不是后日大患，而是谁今日让他没那么难堪。”

这句话，齐王听进去了。

齐人最会的，本就是拿秤量人。可量人量到最后，也总有怕别人先不受量的时刻。眼下齐怕的，不是楚真倒向秦，而是楚一旦觉得拿秦压齐有用，后面每议一步，都要多生一层不甘。

于是齐廷终于退了一寸。

不是大退。

只是把先前最伤楚颜面的几条先收了收，把几项必须战前定死的后约，改成“可待后议”；又允楚一旦联兵成议，齐可先出一军，以示不是空话。

它仍在算。

但至少不再算得叫楚觉得自己像在一阶一阶往下跪。

这封书回到郢都时，殿中不少人都先松了一口气。

屈原也松了一线。

不是因为他信齐，而是因为齐既然肯退，说明楚至少不至于立刻被张仪那条路全拖过去。两边既还在拉，楚就还有把局往“共抗秦”那边扳一扳的余地。

可楚怀王看完齐书，脸上的神情却比众人更复杂。

因为他很清楚，齐这一步不是自己突然转善，而是被秦逼退的。也正因看得清，他心里反倒生出一种极难言的感觉：

原来只要手里有秦这一张牌，齐便真会退。

这感觉极易上瘾。

一个近来处处失势的君主，一旦尝到“我并非只能受人开价，我也能逼别人退一步”的滋味，后头再让他放下这张牌，便会比最初难得多。

屈原看见怀王神色有变，立刻进言：

“大王，齐今日退，不是因我楚已稳，而是怕我楚真偏向秦。越到此时，越不可把今日之验看得太美。”

楚怀王抬眼看他：“寡人何时说它美了？”

屈原道：“臣怕大王会觉得，张仪果然可借。”

这句话一出，殿中气便僵了。

楚怀王的脸色果然沉了下来：“寡人是在救楚，不是在玩机巧。”

屈原再拜：“臣知大王为楚。正因如此，臣才请大王记住，齐这一退、秦那一给，两边从今往后都要在楚身上把这一手讨回去。”

昭阳当夜入见，说的也是同一层意思，只是更冷。

“大王今日看见的是齐退一步，”他说，“臣看见的是齐会记得自己是被逼退的，秦也会记得是它先给城逼退了齐。两边后头都要在楚身上收这份账。”

楚怀王听后，许久没有作声。

过了一会儿，他才道：“寡人又何尝不知？可若不如此，寡人此前岂非真的只能任齐量着走？”

昭阳一时竟无言。

因为这问题正问在最难处。楚之所以走到今天，不是因为不知道谁险，而是因为险已四面都在，它只能在更短的难堪与更长的后患之间，一步步挪。

馆舍里的张仪，得知齐果然退价后，只笑了一声。

随从问：“先生，齐既退，楚是否更偏秦？”

张仪道：“会，也不会。”

“何意？”

“楚不会立刻弃齐，因为它还想把两边都握着。”张仪道，“可也正因它还想都握着，齐楚之间原本最要紧那层‘先真心并力’便更回不去了。齐疑楚，楚嫌齐，双方越谈越像一桩买卖。这样一来，便是表面成盟，骨头也不似从前。”

他说到这里，望了一眼窗外夜色，声音更淡。

“很多时候，不必真把他们拆开。只要让他们抱在一起时，也各自想着我得先防你一手，便已够了。”
