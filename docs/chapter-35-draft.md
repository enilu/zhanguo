# 第三十五回 合兵可成齐楚心已异 连横愈熟张仪势更深

齐国既已退了半步，楚廷中关于“是否还要再偏秦”的争论一时也缓了些。

表面看去，事情像是终于往一个较稳的方向走：齐愿出兵，楚不必再让得那么狠，秦也先归了城，西线可以稍缓，东边可借外援。若只看奏报文书，甚至会让人误以为这番兜转之后，楚竟像是把最难的一段都先熬过去了。

可真正敏锐的人都知道，熬过去的只是最难看的表层。

底下那层更要命的东西，反而才刚刚显出来：

齐楚还肯谈，还肯合，还肯一起对秦；

可它们已经很难再像苏秦当初最想要的那样，把“先并力挡秦”放在一切私算前头。

这一点，苏秦在齐、赵之间往返时看得尤其清楚。

他再入临淄时，齐王对他依旧有礼，甚至还笑着说：“苏子不必太忧，寡人终究还是要顾天下大势的。”

苏秦也笑，只是那笑极淡：“大王若真只顾天下大势，臣今日便不必来得这么勤了。”

齐王不以为忤，反倒道：“天下大势要顾，齐国利害也要顾。苏子总不能要寡人为了一个终究未必肯尽心守盟的楚，先把齐自己放空。”

苏秦听到这句，便知道最难的一层已经坐实。

齐王说得不算错。

可一旦连强国之君都能如此坦然地把“对方未必尽心守盟，所以我也不能先尽心”放在台面上说，合纵这东西便已从“大家都知道难，却还要先逼自己往同一个方向走”的大局，变成了“人人都拿对方的私心当自己先保留的理由”。

这已经不是某一国失信的问题。

而是守盟的道理本身，开始在一次次“我这样做也不能怪我”的解释里，被慢慢掏空。

苏秦没有当场再逼齐王，只拱手道：“臣只请大王记一句。今日人人都因别人未必尽心，故自己先留一手；明日秦若当真一路压来，人人也都会因今日这一手，而更难再信彼此。”

齐王听后，神色略沉，却终究没有应死，只道：“寡人知道。”

知道，不等于愿意改。

苏秦从临淄出来时，寒风正急。他在马车里坐了很久，忽然觉得自己这些日子做的事，越来越像是在向每个人证明：你们今日最聪明的做法，正是将来最贵的一笔账。可这种话，往往只有到真要付账那天，人才肯全信。

而楚廷那边，也开始尝到一种更隐蔽的变化。

齐既松口，秦又先归城，怀王心里那股最急的羞愤确实暂时缓了一缓。可也正因为缓了一缓，他现在再看齐、再看秦时，已不再像前些日子那样纯粹被逼，而更像真的在比较：谁更可借，谁更会拿楚当筹码。

这比较一旦起来，信便更难养。

齐使来时，怀王会先想：它是不是又要借援兵继续压我？

秦使有话递入馆舍时，怀王也会想：它此刻是不是又在顺着我心里最烦齐的那一点下钩？

这样的人心，表面上像更精了，实际上却更容易被反复牵动。

因为它永远在防别人，却很难停下来先问：我如今这一念，究竟是识破了别人，还是早被别人算进去了？

屈原对此最为忧心。

他在一次退朝后独对昭阳，道：“如今最坏的，已不是大王信秦，或大王亲齐。最坏的是大王自以为两边都看透、两边都在用，因而越发不觉得自己也会被人所用。”

昭阳沉声道：“这话我懂。可眼下楚至少先缓了一口气，你若此时还只言深忧，朝中许多人反倒会觉得你在故作危辞。”

屈原苦笑：“我也知不能日日只说最冷的话。可若此时大家都因为齐松了口、秦还了城，便以为局已回稳，那才是真坏。”

昭阳点了点头，却也无奈。

因为国势下行到这一段，很多人都只能先抓住眼前那一点稍能喘气的好处，不愿再多看一步。不是他们完全看不见后面，而是后面太重，重到人本能地想先把目光缩回来，先相信事情大概还没那么坏。

而张仪偏最懂这种人心。

他在楚馆中待得越久，越不急着日日求见。许多时候，只是一封短书、一句侧递，甚至只是故意让楚廷知道秦边近日又撤了一点兵、边吏又让了一步路，便足以让怀王心里的那杆秤再偏一点。

这种打法，比高声急攻更深。

因为它让楚王始终保有一种错觉：不是张仪在逼我，是局势本身使然，而我只是比旁人更懂得从中取最有利的一段。

一个君主若总这样想，旁人便很难再正面把他劝醒。

秦王嬴驷得到楚齐最新动向时，对张仪道：“齐楚终究还是没断。”

张仪答：“不断最好。”

秦王挑了挑眉。

张仪道：“真断，则诸侯反倒易惊。如今这般，表面仍盟，里面已异。齐疑楚，楚嫌齐，赵魏更乐得观望。苏秦若还想再把他们拢紧，便得付比从前更大的力，去补一种人人看着还像在、实则早不肯尽心的盟。这样的盟，最耗立盟之人。”

秦王听罢，缓缓点头。

这便是第二卷行到此处最冷的一层真相：

张仪最成功的时候，不一定是把人都说反了；

而是把那些本还能勉强聚在一起的人，说得即便继续站在一处，也再难真正把背后交给彼此。

夜里，楚宫之中，怀王又收到齐国新书。

书上所言，比前几次都更柔和。可怀王看完之后，却没有松快多少，反而把书卷起，放到一边，半晌未语。

内臣小心问道：“大王，齐国这回已让了不少。”

怀王淡淡道：“让，是让了。可它为何让，寡人也看得见。”

说完这句，他忽然又问：“秦馆那边，今日可有新书来？”

内臣低头应“有”。

怀王静了片刻，道：“拿来。”

这一声并不重。

可若让屈原听见，只怕心里会更凉一分。因为这说明怀王如今已经不再把“看秦书”当成一件需要自我辩解的大事了。它开始慢慢变成日常，变成一种习惯，甚至变成一种“我当然也该看一看另一边怎么说”的自然。

而很多真正危险的事，往往就是这样从“不该”一点点走进“也无妨”，再走进“本就如此”的。
