# 第三十六回 联兵纸定军前先涩 鼓角同吹号次已差

齐使真正把出军日期送到郢都时，楚宫里先松了一口气。

怀王把回书从头看到尾，终于点了点头：“齐终究还是来了。”

殿中许多人都跟着缓了一线。前些日子，朝上日日争的是齐会不会来、来了肯出几分力；如今纸上既写明兵数、将名与起行日期，至少在面上，这件事算是坐实了一半。

可昭阳看完文书，脸色并没有旁人那样松。

他把齐书翻到后页，指给怀王看：“大王，这里还有两句。”

“哪两句？”

“一是‘道路整束’，二是‘联军进止，临阵再议’。”昭阳道，“不算翻悔，却也没把最硬的那一步写死。”

怀王方才那点松气，立刻又淡下去。

“又是再议。”

屈原在旁接道：“来，总比不来强。只是来归来，后头谁先动、谁先担，若不先写死，到了军前便都能再往后让半步。”

楚怀王没有再多说，只命齐使入见，催问何日南下。

齐使答得极周整：“我王已命大将匡忌领军南出，只是春道泥重，后车稍慢，望楚王宽数日。”

这话字字都挑不出错。

也正因挑不出错，才最叫人憋闷。

楚使每日一催，齐廷每日一答，今天说前锋已过某邑，明天又说须候后车、整束粮道。文书一路来回，到了郢都，字字都像真，合在一起却只叫人更清楚一件事：

齐是来了。

可它不是按楚最急的时候来的。

昭阳先回到前线，把楚营沿河后挪半里，给齐军腾出扎营处，又命人先把接粮、通号、合鼓、夜哨的条目写成竹简，等齐将一到便当面议定。

齐军前锋终于出现在远处时，楚营里一阵躁动。

旗是齐的，甲也是齐的，沿河一列列扎下去，声势并不小。屈匄站在高处望了半晌，终于吐出一口气：“总算看见人了。”

昭阳却只道：“先看它把营怎么扎。”

果然，齐营一落地，头一件事不是议明日如何击秦，而是先问水源、问粮道、问楚军近来伤损、问前线旧路。问的都对，却没一句是楚将此刻最想听的。

次日军议，楚将先开口：“齐军既至，明日当先逼前一阵，让秦知道联兵已合。”

匡忌把军图看了又看，方道：“可以逼前，但不宜太急。秦与楚相持已久，未必不防联军初合先求一声势。依某看，先探地，再移垒，后议小击。”

屈匄当场便皱眉：“将军千里来此，难道就为一个稳字？”

匡忌抬眼看他：“贵军前几回吃亏，难道不是急出来的？”

帐中气一下绷紧。

昭阳立刻按住案边，先冲屈匄摇了摇头，又对匡忌道：“将军之虑也有理。只是联军初合，若一点声色都不出，前线士气也未免太冷。”

匡忌这才退了一步：“那便先试一阵，不贪小垒，不压深锋。”

话算议成了。

可帐中坐着的人心里都明白，成得并不痛快。楚将嫌齐将太慢，齐将嫌楚将太急；两边嘴上都说为全局，眼底却都各自藏着一层轻看。

这层轻看，到了阵前，比帐中更伤。

联军第一次小试，在河东一处低坡前。

楚军按约先出半阵，齐军随后压弩翼。号角吹起时，两边鼓声却先后错了一拍。外人未必听得出，站在阵里的兵卒却都听见了：楚这边已往前压，齐那边还在等；齐一等，楚这边便更往前催。

这一阵打得不大不小。

没吃亏，也没得利。

可收军回营后，话先在士卒嘴里长出来了。

楚卒说：“齐人走得像赶集。”

“人是来了，脚总慢半尺。”

齐军那边也在私下说：“楚人一见我军，便想拿我们去补它前头丢掉的气。”

这些话白日无人当面说，到了夜里，却比白日军议传得还快。

数日后，又有一场更小的争执。

楚将主张趁晨雾前推半里，把一处土坡先夺下来；匡忌却说雾重号令难通，秦若伏弩于后，联军初合便要吃闷亏。屈匄忍不住当帐发作：“齐若连这半里坡都不敢争，来此何为？”

匡忌脸色也沉了：“齐若只为壮贵国声势而来，便不必带这几万兵。”

昭阳重重拍案：“够了。明日不争坡，改探秦左垒。”

众人退下后，他独自站在帐中，看着案上满是记号的军图，半晌没有动。

他看得很清楚，最难的已不是齐来不来，而是来了之后，两边都不肯先吞下自己最不爱吞的那口气。一个要快，一个要稳；一个怕再失声势，一个怕先替别人担过头。都能说出理，偏偏正因都有理，才更难合成一只手。

军报很快回郢都。

怀王听完，先问的不是秦军如何，只问：“齐将又说慎了？”

来使低头：“说了。”

怀王冷笑，把书简丢到案上：“寡人便知道。齐人来是来了，心却没来。”

屈原在旁道：“齐将所虑也非全无道理。如今最怕的，是两边都各执一理，却无人肯先咽那半口气。”

怀王看向他：“楚已咽得够多了。齐如今连到军前，还总要楚再等它、再顺它，难道楚就该一直咽？”

屈原听完，心里一沉。

因为他知道，怀王心里那根线又往回绷错了方向。大王此刻更在意的，已不是“这盟怎样才能真用”，而是“齐又让楚多受了几分气”。

同一时刻，临淄也在收前线回报。

齐王看完，只问田文：“楚营又急了？”

田文答：“急。它急着先见个样子。可联军这种事，越急着见样子，越容易先见裂缝。”

齐王道：“那便叫前线多让几分？”

田文摇头：“让得太多，楚便会当成它该得。最好是既不退，也不全顺。让它始终觉得齐在，却总少那么半步。”

齐王默然。

他知道，这样最省齐国的力。

却最耗楚王的心。

馆舍中的张仪听完这些军报，只问了一句：“齐军可曾退去？”

随从答：“未退。”

张仪点头，把手边竹简放下：“那便够了。”

“先生不再写书推一推？”

“不必。”张仪道，“齐若全退，楚反会一怒而醒。如今齐兵在侧，事也在做，只是处处总差那半步，这才最磨人。”

他说完，起身走到窗边。

馆舍外夜风吹过，远处军中更鼓一下一下，隔得很远。

张仪听了片刻，淡淡道：

“人最怕的，不是别人不来，是别人来了，却总不替你先担最重的那一口。”
