# 第三十七回 陈轸回郢都先看秦齐来书 怀王问病时药已摆到案前

楚齐前线久合不顺，郢都终于把陈轸召了回来。

他入殿那日，怀王没有先说旧情，只把近来齐军军报和秦馆短书一并推到案前：“你看，楚如今病在何处？”

陈轸先不答，低头把两边书简都看完，才抬头道：“病不只在齐，也不只在秦。病在楚如今凡事都先看别人给了哪条路，却少看自己脚下站得稳不稳。”

怀王听了，眉头立刻沉下去：“你的意思，是寡人总在求人？”

陈轸俯身：“臣只见楚近来每一步都像在比，哪一条更不坏。比得久了，外头看着像人人都绕着楚转，实则楚自己先没有准绳。”

屈原站在一侧，听到这里，神色才微微一动。

怀王冷冷道：“那依你之见，楚该如何？”

陈轸把手按在两卷书上：“先定三件事。其一，齐军既至，前线节度不能再日日争；其二，秦馆来书不可一封封牵着朝议转；其三，对外之辞须先成一口，不可今朝借齐，明夕又借秦。”

昭阳当即应了一句：“这三条，先补的是骨。”

怀王却没有立刻点头。

他看着陈轸，隔了很久才道：“若寡人此刻少看秦书、多替齐前线说话，外面岂不都知楚被齐按住了？”

这一问一出，陈轸心里便是一沉。

他知道药在这里最难咽。于是只得再拜：“若只怕眼前难看，便不肯先把骨头定住，后头失的便不止一分脸面。”

怀王没有发作，只挥手散朝。

散朝后，屈原留住陈轸。两人走到廊下，外头风很硬，屈原低声道：“先生今日这药最切，只怕大王还是先嫌苦。”

陈轸苦笑：“最毒的不是旁人会说，是大王总能把难堪听成权衡，把被牵着走听成多留一路。”

两人正说着，内侍已从后头快步追来：“大王宣秦馆新信入内。”

陈轸回头看了一眼那卷尚未拆封的秦书，没有再说话。
