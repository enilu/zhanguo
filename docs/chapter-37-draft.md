# 第三十七回 陈轸入朝欲补将倾线 怀王听尽终难弃近利

联军既成而不畅，楚廷中终于有人忍不住提起了另一个名字。

陈轸。

此人本是楚之辩士，素来长于权变，也比寻常朝臣更懂诸侯之间那层既相用又相防的微妙气。他前些年奔走在外，未常久立朝端，如今闻楚局日下、张仪再入、齐楚表合而军前先起龃龉，终于也被召回郢都。

陈轸入殿那日，楚怀王看着他，神色颇复杂。

因为这时再召一位善辩之士回来，已不只是求谋，更像是怀王自己也隐约察觉到：事情走到如今，光靠怒、靠硬、靠一次次在齐秦之间转念，已不足以把这局整回去。他需要有人替他把这堆越来越乱的线重新捋一捋。

“陈轸，”怀王开门见山，“你来看，楚今日究竟病在何处？”

陈轸并未立刻答。

他先看完齐军前线的数份军报，又看了秦使馆舍近来递来的几封短书，最后才缓缓抬头道：“病不在齐，也不只在秦。病在楚如今凡事都先问‘谁更可借’，却已越来越少问‘自己能不能先稳住不被人借’。”

这第一句，便让殿中静了下来。

怀王眉头一沉：“你的意思，是寡人一味求人？”

陈轸一拜，道：“臣不敢言大王求人。臣只言，楚近来每一步都走在别人给的选项里。张仪给一条，齐使给一条，赵魏各自给一条，楚便总在比较哪条更不坏。这样走久了，外人看似都在围着楚转，实则楚最缺的，反而是自己先定一条不因别人一退一进便改的骨头。”

屈原闻言，终于第一次在这段时日里露出一点极淡的赞同神色。

因为陈轸这句话，正好补到了楚廷眼下最难自见的一层：他们人人都在说“为楚多留一张牌”，可牌留得越多，心越跟着外面走，自己反倒越难稳。

楚怀王听后却未必高兴。

“说得容易。”他冷冷道，“若楚不借齐、不借秦，难道只凭自己如今这局面，便能先稳住？”

陈轸答得更直：“正因不能全凭自己，才更要先定好借谁到什么分寸、绝不退到什么分寸。若无此度，则今日借齐，明日借秦，后日又想借赵魏。借到最后，人人看似都能替楚撑一把，实则谁都知道楚自己先没了准绳。”

怀王沉默不语。

陈轸继续道：“臣请大王做三件事。其一，齐军既已来，不可再让前线日日争谁先谁后，须由楚先明定联军节度之法；其二，对秦可留缓兵与归城之利，但不可再让馆中短书一封封牵着朝议转；其三，凡对外之辞，须重新一口。不可今朝说与齐共守，明夕又借秦压齐。这样说久了，别人未必先乱，自己朝中便先乱。”

这番话说得极整。

昭阳听后，当即点头道：“此三条，皆是补骨之法。”

屈原也道：“臣亦以为可采。”

满殿里，这是这些时日难得的一次几方意见明显相近。

可越是这样，越显出真正的难处不在朝臣会不会说，而在怀王肯不肯先忍一忍自己心里那口气。因为陈轸这三条若真要行，第一件要压住的，恰恰就是怀王近来最常被张仪与齐使轮番挑起来的那层“我不能太难堪”的急。

楚怀王果然听完良久，才道：“你说的，都不差。可若寡人此刻忽然少看秦书、多替齐前线说话，外间岂不立刻就知道，楚终究还是被齐按住了？”

陈轸听到这里，心里便是一沉。

他知道，病根还在这里。

怀王不是不懂这些补骨之法，也不是不知张仪在顺着他的难堪下刀。可每当真要他先吃一点看起来像“低了一分主位”的亏时，他又总忍不住先退。

这正是他最致命之处。

陈轸只得再拜，道：“大王，若只怕外间看见一时低了一分，便不肯先把骨头定住，那到后头失的便不止一分。臣所忧者，正是楚如今太怕立刻看起来输了，反把长久能不能不败的法一再往后压。”

怀王脸色极沉，却终究没有发作，只挥手散朝。

散朝之后，屈原留住陈轸，道：“先生今日这几句，已是眼下最切之药。只怕药虽切，病人未必肯先咽。”

陈轸苦笑：“我也看出来了。张仪最毒的，不只是能说，而是他总替大王把最不肯听的话换个名字。明明是难堪，叫作权衡；明明是被牵着走，叫作多留一路。叫得好听了，病人便觉得自己不是病，是更聪明了。”

屈原听后，只觉胸中更冷。

因为这正是他这些日子说不透、又最怕说透的一层。

而在另一边，楚怀王回到内殿后，并没有立刻传陈轸三策去行，反而先问了一句：“秦馆今日可有新信来？”

内臣低声答“有”。

怀王点了点头，道：“拿来。陈轸的话，寡人再想想。”

就这一句，便把方才殿上那点好不容易聚起来的“补骨”之意，又先吹散了半分。

因为陈轸要的是先定楚自己那条不随外人变的骨；

而怀王眼下最习惯的，却仍是先去看外面又递来了什么新路。

张仪拆到这里，真正可怕的已不只是外面齐秦怎么推，还是楚廷内部每次刚想把骨头正一正，怀王自己的心就又先往“再看看别家怎么说”那边歪过去一点。
