# 第三十八回 两军并列终难同一步 一夜看书始知旧骨伤

入春之后，楚齐联军的营盘总算真正连了起来。

从高处望去，旌旗相望，灶火相接，倒也像一副能压住秦边的样子。郢都、临淄两边收到前线奏报时，文书上写的也都好看：齐军既至，楚军亦整，诸将会于某地，议合击秦垒。

外人看着，多半都觉得这场拖了许久的联手，到底还是成了。

可真在军中待着的人，却没有几个敢把心放实。

楚营里，昭阳亲自看过次日要用的军图后，把竹简往案上一按，抬头问对面的齐将：“明日若秦右垒先露破绽，贵军当先压，还是我军先压？”

齐将姓匡，是临淄派来的宿将，年纪比昭阳略长，面色一向很平。他听罢没有立刻答，只把手指移到图上，点了点一处低洼地：“此地泥深，若昨夜又有雨，车不得疾。楚军近些，先起一步，更顺。”

昭阳道：“楚若先起，秦左垒援兵一到，谁来截？”

匡将军道：“齐军截。”

他说得不慢。

可昭阳还是听见了那句话里藏着的一寸空。

不是“我军与君同时起”，而是“你先起，我来截”。

这一寸空若放在寻常会兵之中，也算不得什么。可放在今日这种彼此都怕对方只想借自己立功、又都不肯先白担最重一口气的局面里，便重了。

昭阳沉默片刻，又问：“若秦军先放右垒，引我军前压，齐军见其后营动不动？”

匡将军答：“动。但须先见楚军压得实。”

帐中一时安静。

楚将屈匄站在昭阳身后，脸色已隐隐发沉。

他最烦的，便是这种一句句都像答应了、真拆开却总要先看别人一步的答法。可他也知道，若这时当面发作，只会把帐中这点勉强维住的面子立刻扯破。

昭阳最后只道：“好。明日各按军图行。”

匡将军点头告退。

待齐将走远，屈匄才低声道：“令尹，这仗还未打，我已先听出三分不像样。”

昭阳道：“不像样也得打。此时再退，郢都那边先乱。”

屈匄咬了咬牙，没再说话。

当夜果然有雨。

雨不大，却细而不断，把营前地皮泡得发黏。次日天还未亮，两营便已举火传令。楚军甲叶响得密，齐军鼓声却稍迟半拍。那半拍若只听一耳，几乎听不出来；可真站在阵前，便知道这是两路兵从心里就没有完全踏到同一个拍子上。

辰时前后，秦垒东侧忽然起烟。

探骑奔回，报说右垒似有调动，像是秦人要往中军抽兵。

昭阳闻报，当即下令楚军前锋先压，左翼徐进，中军随后。军令一层层传下去，楚军先动，战旗一线线顶了出去。前阵楚卒踏着湿土往前推，盾牌相摩，长戈如林，声势极足。

齐军也起了。

只是起得仍比楚军略慢。

这一慢，尚不至坏局；真正坏的是，楚军前锋已与秦军接上时，齐军那边还在等秦后垒是不是真空。

秦军却偏偏就等着这一刻。

右垒果然不是空，只是故意显得薄。楚军一压上去，垒后伏着的弓弩手便忽然起身，箭声密得像雨脚再落。前锋楚卒先是一滞，后队仍在推，阵势顿时紧了一下。

屈匄见状，立刻命亲兵传旗，催齐军按约截秦左援。

旗是打出去了。

齐军那边也看见了。

可匡将军仍多等了片刻。他不是故意见死不救，只是前方泥深，秦左垒又尚未尽出，他总怕自己这一压，反倒被秦人转头吃住。于是这一“再看一看”，便又把最要命的那一步生生往后拖了几息。

战场之上，几息已经够长。

秦左垒援兵先一步撞上楚军侧翼。楚军本以为齐军会来接，结果回头一看，只见齐旗在后，阵形仍整，却并未压到该压的位置。那一瞬间，前锋将校心里先凉了半截。

人一凉，刀便慢。

楚军侧翼很快被秦军扯开一道口子。屈匄怒得几乎要当场骂出声来，只得亲自领人往那缺处补。楚卒见主将上前，才勉强把那口气续住。齐军这时也终于前压，箭雨压过去，替楚军截住了一段秦援。

可战机已经过去了。

原本能咬住秦右垒的一口锋，变成了两边各自把自己该守的先守住。

到了午后，这场本该试出秦边虚实、顺势挤出一步的合击，只打成了两国各记一功、谁也没真把最重那步替对方担过去的平平一阵。

楚军伤得比预想重些。

齐军损失不大。

秦军退得不远，营垒也未失。

傍晚收军时，前线风里都是湿土和血腥气。楚营将士脸色很难看，不全是因为伤亡，而是因为人人都看见了那一步差在何处。

屈匄回营后，把头盔往案上一扔，砸得竹案震响：“齐人若早半刻压上去，今日秦右垒至少要吐一层血出来。”

昭阳没有接这句，只问：“军中传开没有？”

屈匄冷笑：“这种事还用传？前锋眼都看见了。”

过不多时，匡将军亲自来楚营致辞，说齐军并非不救，只因地湿阵滑，不敢轻进，以免全局更坏。

这话也不是全无道理。

可楚军上下听来，只觉每个字都像在替那半刻找台阶。

昭阳听完，倒没有当面翻脸，只淡淡道：“将军言之有理。只是下次若再有这等须先应一步的时候，还望贵军把‘怕更坏’四字稍稍往后放一放。”

匡将军眼皮微动，回道：“令尹这话，齐记下了。只是楚军今晨压得太急，也未尝不是让秦人有机可乘。”

一句话，又把帐中温度压下去了。

两边谁都没有再往前顶。

因为说到这一步，再说便真要把脸撕开。

可不再说，不等于心里就能过得去。

夜里，楚营上空风很硬。巡营士卒从齐营旁走过时，先前那点“友军”之意已淡了许多。有人低声说齐人怕死，有人反驳说楚人好功；骂来骂去，骂声都不高，却比白日争吵更伤。

次日，这份军报便到了郢都。

楚怀王展开来看，先看伤亡，再看过程。看到“齐军终至而稍后”几个字时，手指便慢慢收紧了。

殿中诸臣都在等他说话。

怀王把军报合上，半晌才冷冷道：“齐军倒是会拣时候。”

屈原听出话头不对，立刻道：“大王，前线仍是联兵，不可因一战之迟便先疑其尽心。”

怀王看了他一眼：“一战之迟？若今天迟半步、明天迟半步，楚还要替齐把这半步半步都圆成大局么？”

陈轸随即出班，道：“正因如此，才更要趁此时把军令、节度、先后之责再定死。今日之败不算大，却最怕被拿去养成两边各自认定‘对面终究不会先替我担’的心。”

怀王没有立刻答。

他当然知道陈轸说得对。

可他心里更先翻上来的，是另一股更直接的东西：齐兵明明到了，楚却还是在阵前先吃了那口苦。这说明什么？说明到头来，楚最重那一步，仍只有楚自己先咽。

这种感觉，比伤亡更坏。

因为它会把人前些时日靠着“齐终究已来”“局还可救一些”的那点勉强安稳，连根往下拔。

陈轸见怀王脸色阴沉，知道他那口气又要往旧路上走，便再进一步：“臣请大王此刻先别问齐军是不是可靠到十分，先问楚军今后要不要还把最重那一步总寄在别人身上。若要盟可用，便须把‘谁先应、谁后应、迟则何如’再写得更死；若不然，今日是一战之迟，后面便会是一战战都迟。”

昭阳的军报也随之送到，把前线失拍写得极细。末尾只一句：

“盟兵未散，而心已有参差。”

楚怀王把那八个字看了两遍，忽然觉得胸口发闷。

从前他总以为，只要秦肯退一点、齐肯松一点、楚再把阵面摆出来一点，自己便算一点点把局救回来了。可到了今天，秦退的城在手里，齐来的兵也在前线，为什么局却还是不像回来了，反倒更像四下都在，偏偏没一处真能用足？

他挥手退朝，独自回了内殿。

夜深之后，案上摊着三样东西。

一样是秦人归城的交割文书。

一样是齐军本次合击后的军报。

一样是陈轸白日里重新拟的联兵节度条陈。

怀王先看秦书，看完没说话；再看军报，眉头越皱越深；最后看陈轸的条陈，看到“前锋不得私进，援军不得后时，违者军前听劾”时，忽然把竹简放了下去。

他知道这些条文若都钉下去，楚以后的路会稳一点。

可他也清楚，一旦钉下去，楚自己也要先受拘束。君主、诸将、前线节度，谁都不能再像这些日子一样，靠着临机一转去给自己找补。

屋里静得只剩灯花轻爆。

过了很久，怀王才低声问身边内侍：“寡人这些日子，是不是把能看的脸面都先看住了，偏偏把最该钉住的东西越放越松？”

内侍扑通跪下，不敢答。

怀王也没再问。

他把那份联兵条陈拿起来，又放下；拿起来，又放下。这样反复几次，最后终究没有立刻盖印，只道：“明日再议。”

这四个字一出口，连他自己都觉得有些冷。

因为他忽然明白，最怕的事也许不是自己全然不懂，而是自己每次都懂一点，却总还想再留一点。

而国之大局，往往就是在这一点一点的“明日再议”里，被人慢慢抽空。

同一夜，邯郸也得了前线消息。

苏秦看完之后，没有像从前那样立刻提笔去催哪一国，也没有立刻再写长书去补哪一道缝。他只是把军报放在灯下，盯着那句“齐军终至而稍后”看了许久。

门客在旁轻声道：“相国，齐毕竟还是上去了。”

苏秦点头：“是。可最伤人的，也正在这个‘毕竟’。”

门客不解。

苏秦道：“若它全不来，天下反倒容易同声骂齐。如今它来了，也救了，也能把道理说圆。可楚军真正最重那一下，却仍是自己先吃。这样一来，齐可自言未负盟，楚也不能明着说它见死不救，可心里那根线便更难再像从前那样绷直了。”

他说完，低头取纸，只写了一句，便停了笔。

纸上只有六个字：

“盟可续，骨难完。”

这不是给诸侯的书。

也不是给后人的话。

倒更像他写给自己的一句认账。

窗外春夜尚寒，灯下那张纸被风吹得微微一颤。

苏秦伸手把它按住。

像按住一角还没有彻底碎裂、却已怎么也拼不回旧样的帛。
