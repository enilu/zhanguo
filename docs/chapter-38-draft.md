# 第三十八回 楚齐两营连火却不连步 东坡一迟前锋先知友军靠不住

入春之后，楚齐两营总算真正连到了一处。

从高处看去，旌旗相望，灶火相接，倒像一支真能向西压过去的大军。可真坐进军帐的人，心里都没有放实。

这一夜议战，昭阳先把军图按在案上，问对面的齐将：“若明日秦右垒露口，楚先压，还是齐先压？”

齐将看着图，道：“此地泥深，楚营更近，楚先一步更顺。齐来截秦左援。”

昭阳又问：“若秦故意让右垒薄出来，引楚军先压，齐见不见秦后营再动？”

齐将答：“见楚压得实，齐便动。”

屈匄站在昭阳身后，听到这里，脸色已经沉了。

次日天未亮，两营举火传令。夜里下过细雨，坡前泥黏得很。辰时前后，秦右垒果然起烟。昭阳当即命楚前锋先压，左翼徐进，中军随后。楚军一线线顶出去时，齐军也动了，却终究慢了那半拍。

楚军先与秦右垒接上。

垒后伏弩立起，箭声一下密得像雨脚再落。屈匄急催齐军按约截秦左援，旗已打出，齐军那边也看见了，却仍在等秦左垒是不是尽出。就这一等，秦左援已先撞上楚军侧翼。

楚前锋回头一看，只见齐旗在后，并未压到该到的位置，心里先凉了半截。

屈匄只得亲自领人去补。

齐军这时也终于前压，用箭阵截住了后半截秦援，可战机已过。原本该趁这一口气咬进秦右垒的攻势，最后只打成两营各自守住自己的那一步。

傍晚收军时，楚营将士脸色极难看。

屈匄回帐后，把头盔往案上一搁：“齐军若早半刻，右垒今日至少要吐一层血。”

没过多久，齐将亲自来致辞，说前头泥深，不敢轻进，以免全局更坏。话也不是全无道理，可楚营上下听来，只觉句句都是在替那半拍找台阶。

次日军报入郢都。

怀王先看伤亡，再看经过，看到“齐军终至而稍后”几字时，手指慢慢收紧。陈轸立刻进言：“今日不是大败，却最怕两边都从此认定，对面不会先替自己担那一步。”

怀王没有接话。

夜深后，案上摆着三样东西：秦人归城交割文书、齐楚合击军报、陈轸新拟的联兵节度条陈。怀王先看秦书，再看军报，最后看条陈，看到“前锋不得私进，援军不得后时，违者军前听劾”时，忽然把竹简放下。

屋里只剩灯花轻爆。

隔了很久，他才低声问身边内侍：“寡人这些日子，是不是把脸面看得太紧，把最该钉住的东西放松了？”

内侍扑通跪下，不敢答。
