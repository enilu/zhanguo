# 第三十九回 赵魏韩闻风各修自守策 合纵名在诸侯先顾门

楚齐前线的事越传越细，最先坐不住的，不是郢都，也不是临淄。

是邯郸。

赵近秦，凡事都比旁国更怕一个“迟”字。楚齐既已联兵，却仍处处失拍，这消息一到赵廷，满朝先静了好一阵，像谁都知道该说什么，却又谁都不愿先把那句话挑明。

最后还是一名老臣先开口：“臣请大王自今岁起，边仓、城修、练兵、骑探，都按最坏处作算。”

赵王看着他：“何谓最坏处？”

那老臣顿了一下，才道：“便是诸侯虽仍讲盟，急时却未必都能立至。”

话一出口，殿中竟没有多少人驳。

不是因为人人都想弃盟。

而是这句话，谁都觉得自己迟早会用得上。

苏秦恰在这日入朝。

他一进殿，便看见案上摊着的不是会盟文书，而是边仓、马政、城守、役粮的条目，心里先凉了半截。

赵王见他来到，仍起身相迎，礼数一点不减：“苏子来得正好，寡人与诸卿正在议边事。”

苏秦看了一眼案上竹简，道：“赵若只议边事，不议旁边诸国此后也会如何学赵，边事便只会越议越重。”

那老臣答道：“相国之意，臣等岂会不知？可赵若今日不先按最坏处作算，来日旁国一迟，先吃刀的还是赵。”

苏秦走到殿中，缓缓道：“赵当然该修边、练兵、储粮。可若连这些事都明明白白按‘旁国未必会来’去做，魏韩明日便也会这样做。赵到了最紧时，再叫它们来，它们凭什么信赵今日守的还是同一张盟？”

那老臣沉默了片刻，道：“相国说得长远。臣等如今先顾的是眼前。”

这一句不高，却把殿上气口一下压住了。

因为满殿上下，其实都正在这样想。

赵王低头看着案上那几份条目，半晌方道：“苏子之忧，寡人知之。可赵门在此，秦锋亦在此。盟约可以守，国门不能只靠人心去守。”

苏秦听完，没有立刻再逼。

因为他知道，这话若继续往下顶，便要像是逼赵先拿本国城门去替天下作赌。

从赵廷出来，他又转去魏。

魏地比赵更薄些，也更怕自己成了诸大国之间随手试边的一块软口。魏王听完来意，先命人把近来河外守备图取来，当着苏秦的面铺开。

“相国来看，”魏王指着图上几处渡口道，“若赵先自守，韩也先自守，魏却仍全按‘众国终会共应’去算，这几处空不空？”

苏秦低头看了一眼，道：“空。”

魏王点头：“故魏也只得先补。”

苏秦抬起头：“补是该补。可大王若把心也跟着一并先收回本国，后面就更难再叫别人为魏出锋。”

魏王笑得很淡：“相国，魏如今最怕的，正是自己先把锋露出去。”

这句话，说得比赵廷那一殿议论还更直。

苏秦听完，知道再说下去，也只是把同样的话换个壳子再争一遍，便只问：“那韩呢？魏若也如此，可曾想过韩会怎么做？”

魏王道：“韩自然更会如此。”

他答得太快。

快得连自己都像早已想过。

苏秦心里随之一沉，转身便去韩国。

韩廷里的话，比赵魏都更冷。

韩国近秦最深，刀锋也离得最近。苏秦刚坐下，韩相便亲手递给他一份新拟的边务条陈。上头写得极明：

“外仍修旧盟之辞，内则一切兵粮筹算，皆按外援未及时先办。”

苏秦把那行字看了两遍，才问：“这是韩廷近来定下的？”

韩相道：“还未定成，只是多数人都这样主张。”

“你们知道这意味着什么么？”

韩相苦笑了一下：“知道。可韩若不先这样想，便连活到后头去听诸国再议的资格都未必有。”

这话说得太实。

实得叫人连骂都骂不出口。

苏秦离开新郑时，天色已晚。

驿馆中，三国近来新下的边务条目都摆在案上。赵的是“独挡之备”，魏的是“相邻自护”，韩则干脆把“外守其名，内按最坏”写到了纸面。

这些条目若一件件看，都挑不出大错。

可合在一起，便像有人拿着细锥子，一下一下往合纵那张网上最吃重的几处捅。

苏秦提笔想写，又停下。

门客在旁低声道：“相国，三国都还没公开弃盟。”

苏秦看着灯下那几卷竹简，道：“正因没人公开弃盟，才更冷。”

门客不解。

苏秦把赵、魏、韩三份条目排成一列，点了点：“你看，赵没有说不守；魏也没有；韩嘴上还把旧盟挂着。可它们已经先按‘别人未必赶得及’去活了。如此一来，谁嘴上说什么反倒不那么要紧了。”

他说完，终于落笔，只写下八个字：

“人人守门，天下失门。”

写完之后，他盯着那八个字看了很久。

门客轻声问：“相国这是写给谁？”

苏秦摇头：“先不是写给谁。”

“那是写给什么？”

苏秦抬头，看向窗外黑沉沉的夜色：“写给后面这股风。风若再这样吹下去，每家门都会越关越紧。可天下那道更大的门一旦没人守，小门迟早也只是各自晚开几日罢了。”

窗外夜风掠过，灯焰微微一晃。

案上那张纸却被镇纸压得极稳。

像一句已经写出来、却还没来得及让太多人真的害怕的话。
