# 第三十九回 赵魏韩三国先改边务条目 苏秦一路看去只见小门都关紧了

楚齐前线失拍的消息传开后，最先坐不住的不是郢都，也不是临淄。

是赵、魏、韩三国。

赵廷这一日议边，案上摊开的不再是盟书，而是边仓、马政、城修、役粮。赵王问诸臣：“何谓最坏处？”一名老臣答得很直：“便是诸侯嘴上仍讲盟，急时却未必都能立刻来。”

殿里没人驳。

苏秦赶到时，正撞见这一场。他看了一眼案上条目，便知赵廷如今先想的已不是“如何共应”，而是“若别人迟了，赵这边怎么办”。

他从赵转魏。

魏王接见他时，直接把近边守具和渡口图铺开：“相国来看，若赵先自守，韩也先自守，魏若还全按众国终会来应去算，这几处空不空？”

苏秦低头看了片刻，只答一个字：“空。”

魏王便不再多说。

到韩国时，韩相更把一份新拟条目递到苏秦手里，上面写着：“外仍修旧盟之辞，内则一切兵粮筹算，皆按外援未及时先办。”

苏秦看完，久久没有说话。

三国都没说弃盟。

可三国都已先按“旁人未必赶得及”去活。

夜里，苏秦回到驿馆，把赵、魏、韩三份新条目排成一列。赵先修边，魏先补渡，韩先转粮。件件都挑不出错，摆在一处却让人心口发冷。

门客问：“相国，最坏的是哪一卷？”

苏秦摇头：“没有哪一卷最坏。坏的是它们排在一起，都在说同一件事。”

“何事？”

“人人先守自己的门。”

他说完，提笔写下八个字：

“人人守门，天下失门。”

写完之后，他把笔搁下，看着窗外黑沉沉的夜色，很久都没有动。
