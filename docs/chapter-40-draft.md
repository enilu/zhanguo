# 第四十回 齐廷算援总先看自己亏几分 苏秦出宫时只觉大国也先学会留手

若说赵魏韩的变化还有“近秦不得不然”的苦意，齐国的变化却更从容，也更冷。

临淄朝堂上，一提楚局，齐王总先问：“扶到什么分寸，齐才不至于把自己拖到最前？”

田文顺着这层心气往下铺路，说得最顺耳的一句便是：“不是弃楚，也不是尽楚，而是让楚离不得齐，又不能真把齐拖成前锋。”

朝臣听了，多半都点头。

苏秦再入齐廷时，没有先和田文争，先向齐王行了一礼，问道：“大王可知今日诸侯最像什么？”

齐王抬眼：“像什么？”

苏秦道：“像一群各自守火之人。人人都知西风大，也知若火尽灭，终有一日烧到自家屋下。可人人先护的，都是自己手里那一盏。”

齐王听完，轻轻一笑：“寡人若先把齐这盏分出去太多，最后连齐也暗了，谁来照天下？”

苏秦答：“臣从未劝大王尽舍本国。臣只怕大王连‘该分多少才不至让大火成势’这一层，都先被‘我不能先吃亏’压下去了。”

田文这时接了一句：“若楚自己都在齐秦之间留手，齐为何独要先尽心？”

这话一出，满殿便更静了。

苏秦看了田文很久，才道：“正因楚已如此，齐才更该想一想：若连齐也只拿这句话替自己留手，那后面还有谁肯把合纵从‘人人都说对’拉回到‘总得有人先做对’？”

齐王没有立刻回。

末了只说：“寡人再量。”

出宫时，门客问苏秦：“齐王可曾动心？”

苏秦望着长长的石阶，道：“动不动心未必最要紧。如今最要紧的是，连最有余力的大国，也越来越习惯先把‘我不能先吃亏’说成第一句。”

他说完，下了最后一级石阶。

宫门外风很长，把他衣袖吹得一直往后拉。
