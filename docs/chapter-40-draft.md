# 第四十回 齐国犹重后利难真尽力 六国皆言慎字苏子心冷

若说赵、魏、韩的变化还多少带着“近秦不得不然”的苦意，那么齐国的变化，则更显出强国自重时那种从容却冷的锋。

齐并不近秦。

正因不近，它更有资格把谨慎说得像高明，把观望做得像定力，把每一步都先算透的习惯，讲成“大国不轻动”的体面。

临淄朝堂之上，这层气慢慢越来越重。

齐王并不否认秦为大患，也从不在明面上说“楚之祸与齐无关”。可每逢臣下提起“是否当更真切地扶住楚，使其不至彻底寒心”时，他总会先问一句：“扶到什么分寸，齐才不至于反把自己拖进最前头？”

这句话本身没有错。

可一国之君若总先从这里发问，朝堂里的所有计议便会自然而然地围着它往外长：能不能援，先看损多少；该不该并力，先看齐会不会反倒因他人之乱而先耗本身。

田文对此看得最透，也最会顺着齐王这层心气往下铺路。

他常言：“强国之病，不在算得多，而在算得不够远；然若为远患便先把眼前全部压上，亦非善算。故齐今日最宜者，不是弃楚，也不是尽楚，而是让楚始终离不得齐，又不能真把齐拖成它的前锋。”

这话入耳极顺。

顺到许多齐臣听完，都会觉得这几乎就是强国在乱世里最该有的姿态。

可苏秦听见，只觉一阵寒意。

因为这便意味着，齐连“扶楚”都不再是从“天下大局须有人先托住”去想，而是从“怎样扶到让楚离不开我、却又不至让我先替它受最重之锋”去想。这样的援助未必无效，却注定不会生出真正可托背心的信。

苏秦再入齐廷时，并未立刻与田文争辩，反先问齐王：“大王可知，今日山东诸侯最像什么？”

齐王饶有兴致地看着他：“像什么？”

苏秦道：“像一群各自守火之人。人人都知西风愈烈，也都知若火尽灭，终有一日会轮到自己屋下。可人人手里护的，却还是先是自己的那点火，不肯先分一束去接旁人已快灭的那一盏。久而久之，看似人人都谨慎，最后却是大风来了，一盏一盏地灭。”

齐王听完，轻轻一笑：“苏子这比喻，自是好。可寡人若先把齐这一盏火分出去太多，最后连齐也暗了，谁来照天下？”

苏秦答道：“故臣从未劝大王尽舍本国。臣所忧者，是大王如今连‘该分多少才能不让大火成势’这一层，都先被‘我不能先亏’压下去了。”

田文在旁接道：“可若楚自己都在齐秦之间留手，齐为何独要先尽心？”

这句话正中如今人人最爱拿来自保的那一层理。

苏秦看了田文很久，才缓缓道：“正因楚已如此，齐才更该想一想：若连齐都只用这句话来替自己留手，那天下往后还有谁能把合纵从‘人人都说对’拉回到‘总得有人先做对’？”

田文不语，齐王也一时未答。

殿中竟短暂静了。

可苏秦心里清楚，这种静并不说明自己真把齐王说动了，只是因为他说到了最不好听也最难躲的一层：大家如今都越来越擅长用别人的私心，来证明自己也该有私心。可这层话，听懂归听懂，真要谁因此改变做法，却又是另一回事。

齐王最后只道：“寡人会再量。”

这一句，仍旧是齐王。

不拒，不尽，不死，不松。

出齐宫时，门客忍不住问苏秦：“齐王可曾动心？”

苏秦望着宫门外长长的石阶，道：“动不动心，未必最要紧。如今最要紧的是，连最有余力把众国重新拖住一点的大国，也越来越习惯先把‘我不能先吃亏’说成第一句。”

他顿了顿，又道：“而这句话，一旦成了各国君臣都觉得再正常不过的第一句，后头再讲什么天下共势，都要晚半拍。”

与此同时，燕国那边虽然仍远，却也受了这股风。

燕本不在当下刀锋最重处，故它既不会像赵魏韩那样被逼得日日看边，也不会像齐楚那样在两头大局中来回受挤。可它也正因远，最容易学会另一种精明：

既然诸强都在先顾自己，燕又何必急着把自己那点兵与粮先压出去？

于是燕廷对外之辞仍旧守礼，对内之策却更偏守土练兵、谨防北边，不轻许南下远援。

这看起来并不惊人。

可合在赵魏韩齐身上，便极惊人。

因为到这一步，天下已经不是单有几国各怀异心，而是几乎每一国都找到了一个足够堂皇、足够现实、也足够能自圆其说的理由，来把“我先少出一步”稳稳摆在自己面前。

苏秦从齐地往北行时，路上风沙扑面，驿道边偶有商旅议论，也都在说“各国近来更顾本国边备”。这些百姓未必知道什么叫合纵、什么叫连横，可他们能看见车马所向、粮草所屯、军令所改。天下的气，到这时其实已不必只从王侯口中判断，连道路与市井都在一点点替它作证。

夜宿驿亭时，苏秦独坐檐下，忽然想起自己当年初起合纵时，说的最多的一句话是“秦若次第困诸侯，则人人终不得安”。那时诸侯未必都真信，却还愿意被这句话往同一个方向拽一拽。

如今却像恰恰相反。

人人都知道“次第困诸侯”这事仍在眼前，

却更习惯先答自己一句：“正因如此，我更得先把本国这一层算足。”

一句话不变，时代里的听法却已变了。

苏秦想到这里，忽然有些明白，张仪到今天真正改掉的，或许已不止是一场盟，而是诸侯面对同一危险时，本能先往哪边想、先用什么句子替自己解释的那种习惯。
