# 第四十一回 秦廷坐观诸国先改法 苏子北望自知守势难

当山东诸侯都在各自改边务、改兵期、改回书时，咸阳反倒静了下来。

秦王这一段日子里，问得最勤的，不再是哪一国可立刻被说动，而是些看上去极碎的事：

“赵新修的是西北哪一段城？”

“魏韩近来使者往返，走的是河道还是陆道？”

“齐军虽出，实兵究竟到没到楚营正前？”

“燕廷南北两派，如今哪边声音更大？”

樗里疾把一卷卷新报摊在案上，越听越觉得，秦王如今看的，已不是哪一句话能压倒谁，而是诸侯日子到底换没换过来。

张仪站在一旁，也没有插话。

直到秦王把最后一卷报闻放下，才问了一句：“二位看，这些都是碎事么？”

樗里疾先答：“看起来碎，合起来却重。”

秦王抬眼：“重在何处？”

樗里疾把手指按在舆图上赵、魏、韩三处边线上，缓缓道：“重在诸侯如今做事，已不大按从前那套‘先合众，再分责’的走法了。它们先改的，都是各家自己门里的章法。”

张仪这时才接了一句：“臣先前拆的，是盟。如今他们自己改的，却是活法。”

殿中一时很静。

秦王听完，没有立刻作声，只把视线从舆图上慢慢移开，像在想另一件更远的事。过了片刻，他才道：“如此说来，秦下一步，倒更不必急了。”

张仪俯身：“越到此时，越不必急。诸侯如今表面仍能彼此招呼，心里却都先放着一句：若别人不来，我如何先保自己。”

秦王听后，竟笑了笑：“昔日苏秦，是把六国往一面旗下面拽；如今你，是叫六国各自先把门关上。你二人一立一拆，倒都替寡人把天下的筋骨试了一遍。”

张仪没有答。

因为这话虽像赞，细想却也并不全是赞。

立局者能立，说明天下原有可立之势；拆局者能拆，也说明那局从一开始便藏着够深的裂。

同一时刻，苏秦正从北地折返邯郸。

这一路，他见的不是谁忽然翻脸投秦，也不是哪一国把旧盟公然踩在脚下。恰恰相反，驿道上礼数仍在，馆舍里旧话也仍说着。可越是这样，他心里越冷。

因为这一路上，他每到一处，见得最多的都不是会盟之辞，而是各国近来新改的条目。

赵添城守。

魏补渡口。

韩先转粮。

齐重新核算援楚所费。

楚则把齐、秦两边来书分开归档，件件都留着余地。

这些事若一件件拆开看，都挑不出大错。

可它们一旦摆成一排，便像许多细钉子，把“先为天下共向秦”那块本来就不算厚的木板，一寸寸钉得更沉。

苏秦回到邯郸那日，正逢薄雪初下。

门客早替他把各国新报按国别排在案头。苏秦坐下，一卷卷看过去，看完之后却没有像从前那样立刻提笔，只把六卷竹简在案上排成了一列。

门客站在旁边，小声问：“相国，这些报闻里，最坏的是哪一卷？”

苏秦看着案上那一列竹简，道：“没有哪一卷最坏。坏的是它们排在一起，竟都在说同一件事。”

“何事？”

“各国都还认大患在秦。”苏秦说，“可做法已经不是先问‘如何共应’，而是先问‘若别人慢了，我这里怎么办’。”

门客低头去看，却仍看不出其中那层最冷的东西。

苏秦便把赵、魏、韩三卷抽出来，放在最前。

“你看赵，它修的是独挡之备。”

又抽出魏卷。

“魏补的是渡口和邻境互应。”

再把韩卷压在最上头。

“韩则干脆先按外援未及时去算粮。”

他说一句，手指便在竹简上点一下。

门客这才慢慢听明白了，低声道：“他们都还没说不守盟。”

苏秦点头：“正因没说，才更冷。”

他起身走到窗边，望着庭中那一层薄雪，良久才道：“名义有时也重。名还在，人便不敢彻底背着它去活。可若骨头已坏到人人都知道，只剩名还能挂一挂，那名也只会越来越轻。”

门客跟到后面，又问：“那还守么？”

苏秦转回身，神色在雪光里显得更静：“越到只剩名的时候，越得守。若连这点名也没人守了，后面便只剩一扇扇各关各的小门，再没有谁还肯抬头看一眼天下那面大旗。”

这话说完，屋里又静了。

门客当然明白相国说得对。

可他也明白，这种守法已经苦到近乎看不见成效了。

苏秦自己更知道这一层。

他重新坐回案前，提笔去回诸国的书。笔下不再是从前那种一口气压人同向的锐，反而更慢，更沉，也更绕。如今他要说动一国，先得碰它最现实的那根筋，再试着顺着那根筋，把它往更大的局上拽回去一点。

这比他当年初起纵横时难得多。

那时诸侯虽疑，至少心里还有热；如今这股热已被张仪拆冷，又被诸侯自己一层层现实算路磨薄。苏秦手里这支笔，写到后面，倒更像在风雪里替一面残旗多系一道绳。

夜深后，他写完一封，又抽出第二封。

手指已被窗缝里透进来的冷气冻得有些发僵。

门客劝道：“相国，明日再写吧。”

苏秦摇了摇头，没有停。

因为他心里很清楚，若连自己也停了，这一卷里便再没有人会去一遍遍提醒诸侯：

你们今日最自保的那一步，

正是在把明日更大的祸一点点放进门来。
