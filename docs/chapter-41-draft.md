# 第四十一回 咸阳静看六国先改活法 苏秦归邯郸后仍替残旗多系一道绳

当山东诸侯都在各改边务、兵期和回书时，咸阳反倒静了下来。

秦王这一段日子问得最勤的，不是哪一国可立刻说动，而是哪一国新修了哪段边城，魏韩使者近来走河道还是陆道，齐军到了楚营到底几成，燕廷南北两派哪边声音更大。

樗里疾把一卷卷新报摊在案上，答得极细。

秦王听完，才问：“这些都是碎事么？”

樗里疾道：“看着碎，合起来很重。”

张仪这时接了一句：“臣先前拆的，是盟。如今他们自己改的，是活法。”

秦王听完，没有立刻作声。

隔了好一阵，他才道：“如此说来，秦下一步倒更不必急了。”

张仪俯身：“越到此时，越不必急。诸侯如今表面仍能彼此招呼，心里却都先放着一句：若别人不来，我如何先保自己。”

同一时候，苏秦正从北地折返邯郸。

一路上他看见的，不是诸侯忽然翻脸投秦，反而是礼数仍在、回书仍来、旧盟的字眼也还挂在纸上。可越是如此，他心里越冷，因为每到一处，案上摆得更前的都不再是盟书，而是新改过的边务条目。

赵添城守。

魏补渡口。

韩先转粮。

楚把齐、秦两边来书分开归档。

苏秦回到邯郸那日，正逢薄雪初下。门客早把几国新报按国别排在案头。他一卷卷看过去，看完后却没有像从前那样立刻回书，只把六卷竹简在案上排成一列。

门客问：“相国，最坏的是哪一卷？”

苏秦看着那一列竹简，道：“没有哪一卷最坏。坏的是它们排在一起，竟都在说同一件事。”

“何事？”

“各国都还认大患在秦。”苏秦说，“可做法已经不是先问如何共应，而是先问若别人慢了，我这里怎么办。”

门客低头去看，还是看不全那层冷意。

苏秦便抽出赵、魏、韩三卷，一一按在最前。说完之后，他转身走到窗边，看着庭里那层薄雪，许久才道：“越到只剩名的时候，越得守。若连这点名也没人守了，后面便只剩一扇扇各关各的小门。”

夜深后，他又回到案前，提笔回书。

笔比从前慢了许多，像在风雪里替一面已经破了口的旗，再多系一道绳。
