# 第四十二回 连横成风天下皆学自保 合纵未亡却已难救人心

这一年入冬以后，天下并没有忽然大乱。

驿道仍通，使节仍往，赵王见苏秦时照旧称相国，齐王谈局势时也还会顺口说一句“终要顾天下”，楚怀王更常把“为楚多留一路”挂在嘴边。

若只看这些场面，不明底里的人甚至会觉得：张仪这几年虽折腾得厉害，终究也不过是把六国搅得乱了一阵，还谈不上真把它们怎样。

可真正坐在案前改字、看报、下令的人，都知道事情已经不一样了。

赵廷这一日议边，司籍把新拟好的兵役条目递上来。赵王拿起一看，前面三条写的都是修边、补仓、增骑，写到第四条，才见一句“若山东有急，当照旧盟先议”。

赵王看了片刻，抬头问：“为何‘先议’排到第四条？”

司籍伏地道：“臣只是照众卿这几日所议排序。”

赵王没有责他。

因为他自己也明白，如今朝臣谈起秦患，先跳出来的，已不再是“如何共起”，而是“赵这里先要补足几成”。

魏廷那边，则更露骨些。

魏王把近边守具、渡口粮车、邻境烽火次序一一问过之后，才像忽然想起什么似的，补问一句：“若秦再逼，告赵韩的急书该怎么走？”

座中老将听完，顺口答：“这个自当去办。”

话说得对。

可也正因为只是“顺口补一句”，才更显出次序已变。

韩相那边更干脆。

他批一份边粮条目时，本来写着“候诸国回书”，看到一半，提笔把“候”字涂去，改成了“先行”。改完之后，他把笔放下，盯着那个字看了许久，最后竟又命人把这份条目压在最底下，暂不外发。

幕僚问他：“相国既改了，为何不递？”

韩相道：“递了，便像韩先把话说穿。”

“不递又为何要改？”

“因为仗真压到边上，韩不能等。”

就这一句，已把韩国如今最深的心思全说出来了。

他们还不想先做那个把旧盟说死的人。

可做法上，已经越来越按“别人未必赶得及”去活。

苏秦此时仍在路上。

他看得比谁都清楚，却没有一国一国去骂，只是在各处朝堂里反复盯一件事：你们说的话，和你们排在第一行的事，还是不是同一样东西。

这一日他在赵馆夜坐，门客把近来几国回书放好。最上头那封来自赵，字里还带着旧日客气；中间那封来自魏，也仍说“西患共忧”；韩那边更添了几句“愿与诸侯共守名义”。

纸都写得不坏。

可苏秦把这些回书看完，反倒没有一点松气。

门客忍不住问：“相国，他们都还这样写，何以你反更冷？”

苏秦把那几封回书一一展开，又把旁边压着的边务条目一卷卷排开，这才道：“你只看信，自然觉得都还在。可你若把信和这些条目放在一起看，便会看见另一件事早已走在前头了。”

门客低头去看。

赵先修边。

魏先补渡。

韩先转粮。

楚先留路。

齐先算价。

件件都不荒唐。

可件件都先把‘本国’摆在最前。

苏秦把手按在这些条目上，半晌才轻声道：“盟还没废，可人心已经先疲了。”

门客一怔：“疲？”

“是。”苏秦道，“不是不肯守，是一提守，先想到的都是后面的亏、慢、迟、空。人心先被这些后账拖住，再大的名义也会越提越沉。”

他说到这里，忽然想起楚廷里那句已越来越顺口的话。

“总要多留一路。”

这句话本来只是怀王心病。

可如今越传越广，竟像成了乱局中人人都可以拿来替自己解释的好说法。君主可以用，朝臣可以用，诸侯彼此来书也可以用。

一句话若成了习惯，便最难拔。

这天夜里，苏秦在灯下写了一封长书，写到中段，只一句：

“今诸侯之病，不在不知秦强，而在知秦强之后，人人皆以先自完为上。”

写到这里，他停了很久。

因为他知道，这话送出去，诸侯未必不懂，甚至多半都懂。可懂归懂，要他们为更大的将来，先去做眼前那个最像吃亏的动作，依旧难。

他终究没有立刻把这封书封上。

而在咸阳，张仪也正向秦王回最后一轮话。

大殿里灯火很稳，樗里疾把楚、齐、赵、魏、韩、燕近来新动的边务和回书都简简说过一遍。说完之后，秦王并未先问“哪一国可立刻归秦”，反而看向张仪：“依你看，如今这股风可算立住了？”

张仪没有急着答。

过了片刻，才道：“楚已会拿秦压齐，齐已会拿价防楚，赵魏韩燕皆已不肯把自保押在众国尽心之上。到这一步，臣今日不再日日出声，诸侯自己也会顺着这条理往下活。”

秦王听完，许久没有说话。

殿外北风过檐，声音极细。

良久，他才缓缓道：“如此说来，连横成矣？”

张仪俯身：“成其势，未成其尽。盟还可名存，也还可偶尔借事复聚；只是人心已难复旧。”

秦王终于露出一丝很淡的笑：“这便够了。寡人本也不要他们今日尽归秦，只要他们从此越来越难真合。”

这一句出口，殿里更静。

因为谁都知道，这话说得并不响，却极重。它意味着秦所要的，从来就不只是眼前哪个诸侯低头，而是让六国从此越来越不敢真把背心交给彼此。

同一夜，苏秦终于把那封长书放下，另取一张小纸，只写了十个字：

“局未尽亡，心先各亡其半。”

写完之后，他看着那行字，很久都没有动。

屋外风更紧了。

这风已不再只是第一卷末尾那种林动前的细声，而是真的吹进了诸侯各自的门里，吹得人人都先去扶自己那扇门板，少有人还肯抬头去看整片林子会不会因此一起倒下。
