# 第四十二回 冬深之后诸侯仍通使节 案前排事却都把本国先摆到第一行

入冬之后，天下并没有忽然大乱。

驿道仍通，使节仍往，赵王见苏秦时照旧称相国，齐王谈天下时也还会顺口说一句“总要顾大局”。若只看这些场面，不明底里的人甚至会觉得六国不过是被张仪搅乱了一阵。

可真正坐在案前改字、看报、下令的人，都知道次序已经变了。

赵廷议边那日，司籍把新拟兵役条目递上来。赵王翻开一看，前头三条都是修边、补仓、增骑，写到第四条，才是“若山东有急，当照旧盟先议”。

魏廷那边更露骨些。

魏王把近边守具、渡口粮车、邻境烽火一项项问过之后，才像忽然想起似的补问一句：“若秦再逼，告赵韩的急书怎么走？”

韩相批一份边粮条目时，本来写着“候诸国回书”，看了一半，提笔把“候”字涂去，改成了“先行”。改完之后，他盯着那个字看了很久，最后又把这份条目压在最底下，暂时不发。

不是不想改。

是还不愿先把话说穿。

夜里，苏秦在馆中看完诸国回书，又把各国边务条目一卷卷排开。纸上写的仍是旧盟之辞，案上排的却已是赵先修边、魏先补渡、韩先转粮、楚先留路、齐先算价。

门客站在一旁，看了半晌才低声道：“他们都还没说不守盟。”

苏秦点头：“正因没说，才更冷。”

说完，他另取一张小纸，只写了十个字：

“局未尽亡，心先各亡其半。”

写完之后，他把纸压在几卷条目最上头，久久没有合卷。

同一夜，咸阳大殿上，秦王听完张仪和樗里疾回报，终于缓缓道：“如此说来，连横成其势矣。”

张仪俯身：“盟还可名存，人心却难复旧。”

殿里很静。

殿外北风过檐，声音极细，像正一点点吹进六国各自的门里去。
