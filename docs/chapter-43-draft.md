# 第四十三回 旧旗犹举诸侯已难同步 苏子再行只求不令尽坠

连横成风之后，天下表面上反倒安静了些。

诸侯依旧相见，旧礼还走，盟约也还提。若只从馆驿和朝会看过去，甚至会叫人生出一种错觉，仿佛前些年那股寒意已经过去一层。

可苏秦最清楚，这种安静并不是好。

更像一池被人从深处搅浑过的水，表面平下来了，底下却仍全是暗流。

他这一回去的，先不是楚，也不是齐。

而是赵。

因为赵还重。赵若还肯替那面旧旗多拽住一角，天下便还不至于立时全散；赵若也把“守盟”缩成一句场面话，后面诸侯再提合纵，就真只剩礼面了。

邯郸殿上，赵王仍照旧礼迎他。

可苏秦入殿之后，一开口便与当年不同。

他没有先画大势，也没有先说“合六国以制秦”的旧图，只把赵近来新下的边务条目往前轻轻一推：“臣今日先不说天下，只说赵。赵修边、储粮、固军，当然都该。臣只怕另一件事。”

赵王道：“何事？”

苏秦道：“怕赵把‘自守’慢慢写成‘自绝于共局’。”

殿中几名重臣都抬眼看他。

苏秦便指着条目上一行字，道：“赵可以加城、添骑、补仓，可若法令、军期、对外回书、朝中议论，处处第一句都写成‘我先自全’，后面诸侯也会照着抄。抄的人越多，这块牌越轻。”

赵王沉默片刻，道：“苏子今日是要寡人做圣人么？”

苏秦摇头：“臣不敢请赵做圣人。臣只请赵别做第一个把这块牌亲手改成空壳的人。”

这话说得不高。

却叫殿中一时都静了。

因为赵王当然知道，赵如今最怕的，正是自己在众国都往回缩的时候，还独自把脚伸出去太多。

苏秦见他不语，便再往下推一步：“臣今日只求三件。其一，边备可重，对外勿轻言‘各守各国’；其二，若齐楚后面再有合议，赵至少仍保一层应声；其三，凡近秦诸国来书只言自护者，赵不可全顺其意，总还要把‘若秦更进，赵可与诸侯共议共应’写在后头。”

赵王听完，许久才道：“寡人可留其名，也可留其势一线。可真到临锋之时，赵仍须先以赵为重。”

苏秦一礼：“臣求的，也正只是一线。”

这一回，他不是来求赵翻盘的。

他只是来求赵别先宣判。

从赵出来后，他又南下入魏。

魏王见他，比前些时日更谨慎。坐定之后，苏秦没有先劝，只问一句：“大王如今最怕何事？”

魏王答得极快：“怕别人嘴上谈局，最后却拿魏去受最前锋。也怕旁人一变，魏先独当其祸。”

苏秦点头：“故魏更不能把话说满。”

魏王看他：“为何？”

苏秦道：“魏若今日处处只讲自保，明日别人便也更容易说：魏既如此，我何必再顾魏？到那时，大王最怕之事，反倒更近。”

魏王没有立刻答。

过了一会儿，才道：“相国说得对。可要魏先信旁人，也难。”

苏秦苦笑了一下：“臣如今求的，也不是要魏立刻尽信。只是别先把回头的桥自己拆了。”

这一句，比许多大话都冷。

却也更像当下之实。

夜里宿在途中，门客终于忍不住问苏秦：“相国如今所争，已只是‘别把话说绝、别把名义全丢’了么？”

苏秦靠在车壁上，闭目良久，方道：“若连这点都守不住，后面便连回头的台阶都没了。”

门客一时无言。

因为这答法听上去太退。

可又退得极真。

守到今天，苏秦已不再奢望一步重聚，只能一层层替天下保住那些后面也许还能被重新拿来用的名、势、口子、台阶。

同一时刻，楚廷里也终于有人开始察觉另一重危险。

这危险不只在齐慢，也不只在秦狡，而在于楚上下越来越习惯用一句“总要多留一路”，替一切变化开脱。

屈原对此最不安。

这一日退朝后，他见怀王还在翻齐、秦两边的近报，终于直言：“大王，路留得越多，心越散。”

怀王抬头看他，脸色有些沉：“屈原，你也来劝寡人只走一条死路？”

屈原摇头：“臣不是劝大王只走一条。臣是怕大王今日总说自己多留一路，群臣明日便也学着各为自己多留一路。国若到这地步，不止是盟散，连朝中都要先软。”

怀王听完之后，竟一时没有反驳。

因为他自己近来上书看得多了，也已慢慢觉出一点不对。朝臣如今说话，不再只争“应当如何”，反倒越来越多地写成“若彼不应，我国当如何预留”“若秦、齐其一忽变，我楚先保哪一层”。

人人看似更周全。

可一屋子人若都先找退身处，到了真要压上一气的时候，又还能剩下多少硬骨？

想到这里，怀王第一次生出一丝近乎迟来的惊意。

张仪拆开的，也许不只是一场合纵。

他拆开的，更像是一种旧习惯：

大家还肯为了同一件更大的事，先约束自己半步。

而这一层一旦坏了，

后面再想把人一口气压到一处，便只会越来越难。
