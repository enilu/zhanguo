# 第四十三回 苏秦再入赵魏只求留桥 屈原看楚廷诸人已各替自己留路

连横成风后，天下表面反倒安静了些。

诸侯还相见，旧礼还走，盟书也还提。苏秦却知道，这种安静更像浑水沉下去之前那一下平。

他这一回先去赵。

邯郸殿上，赵王仍照旧礼迎他。苏秦却不再先讲“六国共起”，只把赵近来新下的边务条目往前一推：“赵修边、补仓、增骑，都该。臣只怕赵把‘自守’慢慢写成‘自绝于共局’。”

赵王看着那几卷条目，隔了很久才道：“赵近秦，国门总不能只靠别人替寡人守。”

苏秦点头：“臣求的，也不是叫赵先舍本国。只求赵别先把回头的桥拆了。”

从赵出来，他又去魏。

魏王见他，也没有绕，直接把渡口图铺开：“相国来看，若旁国迟了，魏这些口子空不空？”

苏秦低头看完，答：“空。”

魏王道：“故魏只得先补。”

苏秦没有与他争这句，只道：“魏若处处只讲自保，后头旁国便也更容易说：魏既如此，我何必再顾魏？”

魏王听完，没有立刻答，只把那张渡口图缓缓卷了起来。

这一圈走下来，苏秦心里越来越冷。

因为谁都还没说弃盟，谁也都能把自己的谨慎说得极有道理。可越是如此，越说明那张旧网不是被谁一刀砍断，而是在每个人都替自己多留半步时一点点松开的。

楚廷那边，屈原也在看同一件事。

退朝后，他见怀王还在翻齐、秦两边来书，终于直言：“大王，路留得越多，心越散。”

怀王抬头，脸色有些沉：“屈原，你也来劝寡人只走一条死路？”

屈原摇头：“臣怕的是大王今日总说自己多留一路，群臣明日便也学着各为自己留一路。到那时，不只是盟散，朝中也先软了。”

怀王听完，竟一时没有反驳。

因为这些日子上书看得越多，他越能感觉到另一层不对。众臣如今说话，不再只是争“应当如何”，而是越来越多地写成“若彼不应，我楚先保哪一层”。

人人看似更周全。

可一屋子人若都先找退身处，到了真要压上一气的时候，还能剩下多少硬骨？

这一夜，怀王坐在灯下，久久没有翻下一卷书。
