# 第四十四回 楚朝欲定分寸怀王复摇 近利在前陈轸终知难救

楚怀王那一点夜里的醒意，终究没能撑太久。

白日一到，朝会一开，案上的文书便一件件把人往旧路上拖。

这一日压在最上头的，是秦馆送来的一封新书。

张仪没有再写大话，只平平列了两件事：

其一，先前所许归城，可再明定交割次第；

其二，若楚愿于齐楚军议上暂缓一层，则秦边亦可相应缓一层。

字不多。

可每一行都像是顺着怀王这些时日最爱听的地方写的。

怀王看完之后，把书放在案上，指尖却没有立刻移开。

陈轸站在阶下，看见这一幕，心里便先沉了半分。

屈原也在一旁，脸色比平时更冷。

朝议一起，果然便有人顺着秦书开口。

“秦若肯先退城，我楚岂可弃实利不取？”

“齐军虽在，终究出力有限。秦边若愿缓一步，我楚何妨也在联军之议上缓一步？未必就是失盟。”

“眼下要紧的是先让楚缓过气来。至于后面大局，慢慢再看未迟。”

一殿说下来，没有一句明着讲“背盟”。

可一句句拆开来听，都是在替“先退半步”找路。

怀王坐在上首，起初还只听着，听到后来，目光便时不时往那封秦书上落。

陈轸终于出班。

他没有先驳众臣，只先看向怀王：“大王，臣请先分清两件事。秦给之利，可以收；但收利之后，楚该不该因此改军中节度，这是另一件事。”

怀王抬眼看他：“你是说，城可以收，兵却半步都不能动？”

陈轸道：“臣是说，收城与改局，不可绑在一处。”

殿中顿时安静了些。

陈轸又道：“若今日秦给一层，楚便退一层；明日秦再给一层，楚再退一层。退到后面，便不是楚借秦压齐，而是秦拿着一点点甜头，牵楚每一步。”

几名方才说话的臣子都不作声了。

因为这话太直，直得把那些本来听上去都能说成‘为楚着想’的话，一下子剥得只剩骨头。

怀王却缓缓皱起了眉。

他盯着陈轸：“照你说，寡人便该一边收秦之城，一边仍替齐死用其兵？”

陈轸一拜到底：“臣请大王收城，而不改节。”

怀王没有说话。

陈轸便再往前一步：“若还要再说得更短，便是八个字。”

他顿了顿，才道：

“收利，不改骨。”

这八个字落下，连昭阳都微微抬了抬眼。

因为这正是这些时日楚国最难做到、也最该做到的一层。

怀王听完，却先笑了一声。

那笑里并无喜意。

“好个不改骨。”他慢慢道，“你们一个个说得都像楚如今还有多少骨头可改。”

话说到这里，殿中气氛已很不对。

陈轸却知道，越是这种时候，越不能退。他硬着头皮再拜：“正因还未尽坏，才更要守。若今日连这点都守不住，后头便真只剩别人一退一进，我楚也跟着一退一进。”

怀王眼神忽然沉了下去。

因为这句话太近了。

近到几乎是在当着满殿之臣说：大王这些日子，正是在这样走。

他手中玉镇猛地一顿，截断陈轸：“够了。”

殿中众臣齐齐低头。

怀王盯着陈轸，声音已冷得发硬：“寡人自知楚该如何走，不必先生教寡人分辨自己是不是在随人摆布。”

陈轸伏地不动。

他知道，这一回药已经递到了嘴边。

可人不肯咽。

怀王挥手：“退朝。”

众臣鱼贯而出，谁都不敢多留。

屈原走得不快，出了殿门，正看见陈轸仍站在石阶下，脸色被天光照得有些发灰。

他走过去，低声道：“先生今日这几句，已是最直的一刀。”

陈轸苦笑了一下：“直刀能切病，也能先伤说刀的人。怀王如今最不肯受的，不是秦、齐来逼，是有人当面告诉他：你以为自己还在转圜，其实脚下已顺着人家的路在走。”

说完这话，他望着阶前空地，许久没有再动。

屈原站在一旁，也没有劝。

因为他知道，劝也无用。

殿上那封秦书，还摆在那里。

而大王刚才那一下断喝，也已把今日最紧要的那层话先压回去了。

午后，楚宫内殿又传出召见。

怀王没有再召陈轸，只命近臣把秦书中归城条目另誊一份，又问齐军前线近日报了几次“缓行”“待议”。

近臣一一答了。

怀王越听，脸色越沉。

他心里那股气，又慢慢朝另一个方向翻上来：

齐也迟，

秦却肯给。

这念头一起，便像草下火，压不住，只能越烧越闷。

同一时刻，馆舍里的张仪也得知朝会经过。

听完探者回报，他只轻轻点了点头。

随从在旁问：“先生，陈轸这回说得这样直，大王竟也压下去了？”

张仪道：“他说得越直，怀王越不肯咽。”

“为何？”

张仪把案上那封未写完的书翻过一页，淡淡道：“怀王不是不知道病。他只是最怕旁人当众说出，这些日子他最得意拿来护身的那点机变，原来也会害他。”

随从沉默了一下，又问：“那楚还救得回么？”

张仪笔尖停了停，才道：“人也许救得回。局却未必肯等。”

他说完，又继续往下写。

笔声细细，落在纸上，竟比白日朝堂里那些高声争辩还稳。

窗外风过，吹得馆舍门帘轻轻一动。

张仪却连头都没有抬。

像到了这一层，他已不必日日拿重话去推。

只要把每一次小退都写得像顺手，后面的路，便会有人自己去走。
