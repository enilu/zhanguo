# 第四十四回 秦馆新书又拿归城缓兵诱楚 陈轸当殿说收利可以不可改骨

怀王夜里那一点醒意，没撑太久。

次日朝会，压在最上头的又是一封秦馆新书。张仪这回没写大话，只平平列了两件事：其一，归城可再明定交割次第；其二，若楚愿在齐楚军议上暂缓一步，秦边亦可相应缓一步。

字不多，样样都写在怀王最爱看的地方。

朝议一起，果然便有人顺着往下说。

“秦若肯退城，我楚岂可弃实利不取？”

“齐军既慢，楚何妨也缓一步？”

“先让楚缓过气来，后面再看未迟。”

一句句拆开来，都不像在劝背盟。

陈轸终于出班。

他没有先驳众臣，只看向怀王：“收城与改局，不可绑在一处。”

怀王抬眼：“你的意思，是城可以收，兵却半步都不能动？”

陈轸再拜：“臣请大王收利，不改骨。”

这八个字一落，殿中一下静了。

昭阳抬眼，屈原也转过头来。怀王看着陈轸，脸色却一点点沉下去。因为这话太近了，近到像是在当众指出他这些日子每一次转念都在顺着人家的路走。

他手里的玉镇忽地一顿，打断陈轸：“够了。”

退朝后，屈原在阶下追上陈轸。陈轸苦笑道：“药已递到嘴边，人却嫌苦。”

午后，怀王没有再召陈轸，只命近臣把那封秦书中归城条目另誊一份，又问齐军前线近日报了几次“缓行”“待议”。

近臣一一答了。

怀王越听，脸色越沉。

齐也迟，秦却肯给。这念头一翻上来，便像火压在草下，闷得人更不肯回头。

同一时刻，馆舍里的张仪得知朝会经过，只轻轻点了点头。

随从问：“陈轸这回说得这样直，大王竟也压下去了？”

张仪道：“他说得越直，怀王越不肯咽。”

说完，他把那封未写完的新书翻过一页，继续落笔。
