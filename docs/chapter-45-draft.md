# 第四十五回 张仪笔短而诸侯自会往后退 苏秦北入燕地只求还留一句共势

秦馆里的灯近来常亮到很晚。

可张仪写的已不是从前那种一句句都带锋的长书，而是更短、更平的短简。许多事到了这一步，已不用他日日去推，诸侯自己便会沿着那条已经拆开的路往前走。

秦王见他时，问了一句：“若今日苏秦也学你这法，能不能把众国重新拽回去？”

张仪听后，沉默了片刻，才道：“能学其术，未必能学其势。臣拆局，是顺着人人都先不想吃亏的那层心去拆；苏秦若要重拢，却得逆着这层心往回推。”

这话不长，却已把苏秦如今的处境说透了。

他当然也懂诸侯的心，懂怎么因国设辞、因势用人。可到了今天，他要做的事比张仪当初难得多。张仪只需顺着人心说“你先别吃亏”，苏秦却得逆着人心一遍遍提醒：有时你得先吃一点眼前亏，后面才不至于吃更大的。

这一回他北入燕地。

不是因为燕最重，而是因为燕最远。远国若也稳稳站到“既然近国都先保自己，我又何必多担”这一边，说明这股风已不是刀口上的应激，而是真吹遍了山东。

燕王见他时，言辞依旧很恭，还先说了一串“旧盟不可轻”的话。

苏秦听完，只问一句：“大王今岁可曾为南面之事真多备一分兵、一分粮、一分期？”

燕王被问得一顿，苦笑道：“燕远而弱，不敢轻动国本。”

苏秦点头：“臣今日来也不是逼燕先压国本。臣只请大王记住，远国最怕的，不是暂不多出，而是把‘我离刀远’当成可以久安的理。”

燕王听后长叹，最后只答一句：“寡人谨记。”

从燕地出来，门客问苏秦：“相国如今每到一国，听见的都是同一句：我们懂大局，但不能先让自己吃亏。这样走，还有多少用？”

苏秦骑在马上，望着北地沉云，过了很久才道：“若我今日也因他们都这样说，便觉得不必再去，那后面就真的只剩这一种话了。”

他顿了顿，又道：“我如今不是去求他们立刻重回盛势。我是去逼他们至少还得承认，只顾自己，并不能解释尽天下所有利害。”

说完，他催马前行。

北地风大，吹得马鬃一直往后倒。
