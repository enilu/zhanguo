# 第四十五回 张仪不急而诸侯自变 苏秦再奔只为守一线名

秦馆里的灯，近来常亮到很晚。

可张仪写的已不是从前那种句句都带着锋刃、一定要在一朝一夕之间立见翻覆的长书。他现在写得更短，更平，也更少。

这并非怠。

恰恰相反，这是因为许多事已不用他再亲手推出去，诸侯自己便会沿着那条已经被他挑开的路往前走。赵会按最坏处修边，魏韩会先求相邻自护，齐会把“我不先吃亏”说得越来越像大道理，楚则会在每一次收利与缓兵之间更习惯拿“多留一路”替自己解释。

到了这一步，张仪最厉害的地方已不在于继续制造多少新的奇变，

而在于他成功把一种原本只属于纵横家手里的破局之术，慢慢变成了诸侯君臣自己都会本能使用的活法。

咸阳宫里，秦王有一次见他，问了一句很平常的话：“张仪，若今日苏秦也学你这法，是否还能把众国重新拽回去？”

张仪听后，难得沉默了片刻。

“能学其术，未必能学其势。”他说，“臣拆局，是顺着人人都先不想吃亏的那层心去拆；苏秦若想重拢，却得逆着这层心，把人再往共同处拽。顺水易，逆水难。何况今日之水，已比当年更急了。”

秦王点点头，未再多问。

这场对话极短。

可它已把苏秦此后在这卷里的处境说透了七八分。

他当然也懂诸侯的心，也懂如何因国设辞、因势用人。可到了今天，他要做的事，比张仪当初要难得多。因为张仪只需告诉众人：你先别吃亏；而苏秦却得逼众人承认：有时你得先吃一点眼前亏，后面才不至于吃更大的。

这话放在纸面上人人都懂。

真要落到一国国君和一份份边备文书上时，却最难让人点头。

苏秦对此比谁都清楚，仍旧没有停。

他这一回北上燕地，并不是因为燕最重，而是因为燕最远。远国之心一旦也开始稳稳站到“既然近国都先保自己，我又何必多担”这一边，便说明这股风已不是刀口上的应激，而是真正吹遍了山东诸国。

燕王见他时，言辞仍极恭，甚至先说了一串“旧盟不可轻”的话。

苏秦听完，只平静问道：“既不可轻，大王今岁可曾为南面之事真多备一分兵、一分粮、一分期？”

燕王被问得微微一顿，随即苦笑：“燕远而弱，岂能与诸大国并论？寡人不轻盟，却也不敢轻动国本。”

苏秦点头：“臣明白。臣今日来，也不是逼燕先压国本。臣只请大王记住：远国最怕的，不是暂不多出，而是明知天下之局在散，却还把‘我离刀远’当成可以久安之理。到那时，等刀真近，便已无人再有余心顾燕。”

燕王听后长叹，却终究只答一句：“寡人会谨记。”

这已算不错。

可苏秦心里知道，“谨记”与“真做”之间，隔着的正是今日六国最难跨过去的那层坎。

从燕地出来时，门客忍不住问他：“相国如今每到一国，听见的大多都是同一种话：我们懂大局，但不能先让自己吃亏。这样一圈圈走，还有多少用？”

苏秦骑在马上，望着北地沉云，许久才道：“若我今日也因他们都这样说，便觉得不用再去，那后面就真的只剩这一种话了。”

门客一时无言。

苏秦又道：“我如今不是去求他们立刻重回盛势。我是去逼他们至少还得承认：只顾自己，并不能真解释尽天下所有利害。只要这句还在，后面就还有人能借此再拢一回。若连这句都没人再提，便真只剩各国自己关门。”

这已不是开局时那个意气飞扬、言一出口便能叫众国侧目的苏秦。

却也正因此，更见其高。

因为越到后面，越能看出一个人的本事，不只在他顺势而起时多耀眼，也在他知道势已逆、路已薄、众人都越来越不愿听的时候，还肯不肯继续替那件更难也更慢的事往前扛一截。

而在楚廷那边，陈轸仍未死心。

他与屈原、昭阳偶有密议，尝试把对秦受利、对齐用兵、前线节度、朝中辞色一件件重新捋顺。成效不是没有，却总像刚理好一层线，外面便又有新风吹进来；刚压下一分躁气，怀王那边便又因齐慢一步、秦多让一点而重新转念。

陈轸后来对屈原叹道：“我们如今像是在扶一具已被风吹歪的架子。不是一点都扶不住，可扶住这一处，另一处又会自己偏。到最后，人心不先定，术便只能不断补漏，难成真势。”

屈原答道：“可补漏也要补。只要还没全塌，就不能先放手。”

陈轸点头，却也知道，这话里已带着一种很深的悲意。

因为越是到这种时候，越说明他们不是在改天换地，而是在替一个已开始下沉的大局尽量减一点它沉得太快的势。

这一夜，苏秦也在驿馆中对灯久坐。

他忽然想起自己少年时困顿家中，曾觉得最大的难是无人肯听；后来佩印六国，又曾觉得最大的难是众国虽听，却未必能久久照着去行。直到如今他才真正明白，原来更大的难还在后面：

不是无人听你，

也不是人人不懂你，

而是许多人越来越懂你说的是远理，却也越来越熟练地拿眼前之理，一层层把那远理往后压。

这才是张仪今日最深的胜处。

他不是让诸侯变笨，

而是让诸侯都变得更会替自己的眼前利益讲话。
