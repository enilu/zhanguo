# 第四十六回 冬尽之后诸国仍通使节 只是案前第一句都先改成了自保

冬尽春初，诸侯使者往来依旧。

盟书没有谁当众撕掉，旧礼也还在走。若只看驿道与馆舍，外人甚至会以为天下仍和前些年差不太多。

可真正坐在朝堂里的人，都已觉出味道变了。

赵廷议边，一名年轻臣子说“若秦逼急，仍当飞书诸国，共举旧约”。话音刚落，殿中竟没人像从前那样立刻接上。有人低头看案，有人轻咳，最后接出来的话却是“只是本国仓粮仍须先足”“边骑还得再添一拨”。

魏廷更直接。

魏王先问的不是齐楚近日如何，而是“近边几处渡口若各国回书慢三日，够不够守”。老将答“不够”，魏王便当场命人加拨粮车、增修鹿角。

韩相则在一份条陈上，把“待诸国报信而后定近边转粮”中的“待”字划去，改成了“先”。

这些都不算错。

也正因都不算错，合在一起才更冷。

苏秦这一夜坐在灯下，把赵、魏、韩三封回书与它们各自的新条目并排放在案上。门客看了半晌，低声道：“他们都还认旧盟。”

苏秦点头：“正因如此，才更显得次序已变。”

他说完，另取一张小纸，只写了十二个字：

“人皆言慎，盟便日轻；人皆求全，局便日碎。”

写完之后，他没有立刻把纸收起来，而是任它摆在案上，让灯火一点点把纸边照黄。

咸阳那边，秦王听完樗里疾与张仪回报，也只缓缓说了一句：“从今往后，再有人想把众国重新压成一线，先得跟这股已经吹进六国门里的活法斗一斗了。”

殿外北风过檐，声音极细。
