# 第四十六回 合纵未废而天下先疲于盟 连横既成战国渐换活法

冬尽春初，诸侯使者往来依旧。

盟书没有被谁当众撕掉，旧礼也还在走，若只看驿道车辙、馆舍迎送，外人甚至会以为天下仍和前些年差不太多。

可真正坐在朝堂里的人，都已经觉出味道不一样了。

邯郸先有一件小事。

赵廷议边，一名年轻臣子说了句“若秦逼急，仍当飞书诸国，共举旧约”。话音刚落，殿中竟没人像从前那样立刻接上“自当如此”。有人低头看案，有人轻咳，有人说“自然要书”，可下一句接的却是“只是本国仓粮仍须先足”“边骑还得再添一拨”。

赵王坐在上首，看着这一幕，心里先发冷。

因为他知道，众臣不是不认盟。

只是如今再说到盟，人人脑子里先跳出来的，已经不是“如何共起”，而是“若别人迟了，我这里怎么办”。

魏廷更明显些。

魏王这一日召近臣议河外守备，先问的不是“齐楚近日如何”，而是“近边几处渡口若各国回书慢三日，够不够守”。一名老将答“不够”，魏王便当场命人加拨粮车、增修鹿角。

座中有人提醒：“大王，如此大动守具，外间恐以为魏先疑盟。”

魏王看也不看，只道：“疑不疑是他们的话，守不守是魏的命。”

这一句出口，满座便都不再言语。

因为这话太短，也太实。

它把这些日子许多国家心里那句不好明说的话，直接说穿了。

新郑那边，韩相也正盯着一份边务条陈出神。

条陈上原本写着“待诸国报信而后定近边转粮”，韩相提笔把“待”字划去，改成“先”。改完之后，他停了停，又把这份改过的条陈翻过去压在最底下。

旁边幕僚问：“相国为何不直接递上去？”

韩相苦笑：“递上去便像韩先把话说穿了。”

幕僚又问：“那为何还改？”

韩相道：“因为仗若真压到边上，韩不能等。”

这一句，比条陈上的改字还重。

因为它说明韩廷心里其实已经有了定论，只是还不想先把那层窗纸捅破。

同一时刻，苏秦仍在路上。

他这一路走得比从前更勤，声势却比从前更小。昔日他入一国，满廷先看他要说什么；如今他入一国，旁人先看的是他来得再快，能不能替自己把眼前的兵、粮、边患一并料理了。

这不是轻慢。

恰恰相反，是天下正在换一种更硬的问法。

苏秦在魏馆夜坐时，门客替他把几国近来回书放好。最上头那封来自赵，写得仍是旧礼；中间那封来自魏，言辞也并未轻盟；最下面韩相那封，更还添了几句“西患共忧”。

字都好看。

可苏秦把三封信一并看完，半晌没说话。

门客忍不住问：“相国，他们都还认旧盟，何以神色这样冷？”

苏秦把那三封信排开，道：“你看字面，自然都在。可你若把它们跟各国新下的边务条目放在一处看，便知道另一件事早已走在前头了。”

门客低头去看案边那几卷条目。

赵修边。

魏增渡。

韩先转粮。

每一件都是为了防秦，也都不算错。

可每一件，先防的又都是“万一别人来不及怎么办”。

苏秦伸手按在那几卷条目上，轻声道：“盟还没废，可诸侯已经先疲了。”

“疲？”

“是。”苏秦道，“不是不想守，是一提到守，先想到的全是后面的亏、慢、迟、空。人心若先被这些后账拖住，再大的名义，也会越用越沉。”

门客沉默了一会儿，低声问：“那还守么？”

苏秦笑了笑，笑意极淡：“越到这时候，越得守。只是如今守的，已不是那口一下子就能聚起来的热气了。”

他没把后半句说完。

因为他自己也知道，如今再往下，要守的其实是另一件更苦的东西：在人人都更会算的时候，还要有人一遍遍去提醒他们，再这样各自算下去，后面会失掉什么。

咸阳那边，秦王也正在看同样的变化。

这日殿上铺开舆图，樗里疾把赵、魏、韩近来新动的边务一处处点给秦王看。秦王看完，没有先问“哪一国可立刻为秦所用”，反而问：“它们这些动作，是怕寡人，还是怕彼此？”

樗里疾答：“先怕寡人，后怕彼此。”

张仪在旁补了一句：“怕寡人，故人人都要守；怕彼此临时不至，故人人都要先把自家门关紧。”

秦王听完，忽然笑了笑：“如此说来，往后寡人未必事事都要逼得太狠。”

张仪俯身：“大王所见极是。如今诸侯最会伤彼此的，不必全是秦兵，也有他们自己那份越来越有理的谨慎。”

殿中一时无声。

这话说得平，却重。

因为它意味着连横走到这里，已不再只是张仪口中的术。

它开始变成诸侯自己也会下意识去用的活法。

遇事先问本国；

先看旁人是否尽心；

先留一路；

先防一步。

这几件事单拿出来，都像聪明。

合在一起，便把旧日那张还能一口气压过去的盟，慢慢拖成了人人都嫌沉、人人又不肯先扔的东西。

夜里，苏秦在灯下写完一页短记，写到最后，只留下十二个字：

“人皆言慎，盟便日轻；人皆求全，局便日碎。”

写完之后，他没有像往常那样立刻把这页纸收入简册。

而是任它摆在案上，任灯火一点点把纸边照黄。

因为他知道，这几句话已不只是写给楚、齐、秦这一段看了。

它们几乎可以拿去照后面很长一段天下路。

同一夜，咸阳宫中，张仪也正从殿中退下。

走到廊外时，随从低声问：“先生，如今可算大成了么？”

张仪停了停，道：“还没到能说大成的时候。”

随从不解。

张仪望着远处夜色里的宫墙，慢慢道：“只是从今往后，再有人想把众国重新压成一线，先得跟这股已经吹进六国门里的活法斗一斗了。”

说完，他沿廊而去。

廊外风不大，却把灯影吹得长了一截。

像一条已经走出来、且还会继续往前走很久的路。
