# 第四十七回 邯郸小议诸使都肯来听 却无人肯先把自己钉在纸上

苏秦没有再一国一国去追，这回先发数书，邀赵、魏、韩、齐、燕各遣一重臣，于邯郸小议。

书里没有大话，只问一件最实的事：

若秦再逼近国，各国究竟愿不愿把“何时应、如何应、谁若不应便如何处之”先写死一层。

诸使果然都来了。

馆驿里车马不绝，礼数也周正，外人若只从街上看过去，几乎会以为旧日合纵又回了半口气。苏秦站在廊下看诸使下车时，心里却一点也不热。来，可以；先把自己钉在纸上，难。

小议设在赵宫偏殿。

赵王不坐正中主断，只让苏秦先开口。殿内席位不多，案上酒未动，摆明不是热闹盟会。苏秦把草约摊开，只问一句：“若秦今年再逼近国，诸国愿不愿先把兵期与迟缓之罚写下？”

齐使第一个开口：“兵者大事，岂可先死于纸？”

魏使接着道：“近国与远国受势不同，若一概写死，反失轻重。”

韩使更直：“韩最近秦。若要韩先把兵期坐死，那援韩之期，谁又肯先坐死？”

燕使坐得最远，最后才道：“燕道里本远，若全依近国急势来定，燕怕最先落个失约之名。”

一殿人都不是说不援。

人人只是在把“现在先写死”往后推。

苏秦听完，便把草约往前又推了一寸：“兵期诸君不肯先写，那便退一步。只写一条：近国告急，众国不得观望为先，须先共议共动，可否？”

这一回，连赵廷陪坐几名重臣都暗暗皱眉。因为这句话一旦写下，赵自己也要被绑进去。

齐使沉吟半晌，道：“共议可写，共动二字太硬。”

魏使点头：“先议可以，先动难。”

韩使苦笑：“若只写议，韩不反对；若带一个动字，韩不敢诺。”

燕使亦如此。

最后，草约还是改了。

“共动”改作“共议”，“迟缓之罚”改作“后议”。最硬的几行字，一笔笔都被削成了留余地的样子。散席时，诸使仍旧守礼，话也都不难听。

夜里，苏秦独坐偏殿，把那卷改过的文书摊在灯下。

门客进来，轻声道：“至少他们还肯来，还肯认盟。”

苏秦点点头：“这自然比当场翻脸好。”

他顿了顿，把手指按在文书中段那两处被削去的字上：“可今日输掉的，不是理，也不是名。”

门客站住了。

“输掉的是一个敢字。”

说完，苏秦起身走到殿门前。门外夜风很冷，堂下灯焰被吹得斜斜一晃。他望着那一点火，忽然伸手护了一下。
