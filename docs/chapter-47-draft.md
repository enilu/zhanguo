# 第四十七回 再议诸侯无人肯先坐实 旧盟未散先输一个敢字

苏秦这一回没有再一国一国去追。

他先发数书，邀赵、魏、韩、齐、燕五国各遣一重臣，于邯郸小议。书里没有大话，只问一件最实的事：

若秦再逼近国，各国究竟愿不愿把“何时应、如何应、谁若不应便如何处之”先写死一层。

书发出去后，门客问他：“相国，这样问，会不会太硬？”

苏秦正封最后一封书，听完只道：“如今若还先说空话，人人都能陪我说。只有把刀口摆出来，才看得见谁真肯往前站半步。”

几日后，诸使果然都到了邯郸。

馆驿里车马不绝，礼数也周正，外人若只从街上看过去，几乎会以为旧日合纵的气又回来了些。

可苏秦站在廊下看那些使者下车时，心里却并不热。

因为他知道，这些人肯来，多半只是都不愿先做那个连“听一听”都不肯的人。

听，可以。

先把自己钉住，难。

小议设在赵宫偏殿。

赵王没有坐在正中主断，只让苏秦先开口。殿内席位不大，五国使臣围案而坐，案上酒未动，连乐工都未设，摆明了不是要做一场热闹盟会。

苏秦也没绕，只把一卷草约摊在案上：“诸君请看。若秦今年再逼近国，赵、魏、韩、齐、燕，各愿于几日内发兵；若有一国迟缓，诸国当如何处之。”

话一落，偏殿里先静了。

齐使第一个开口，却并不答几日之期，只道：“兵者大事，岂可先死于纸？局势一变，纸上期限反倒容易误国。”

魏使紧接着道：“近秦与远国受势不同，若今日一概写死，真到临边，只怕反失轻重。”

韩使更直，把手往案上一按：“韩最近秦。若要韩先把兵期坐死，那诸国援韩之期，谁肯先坐死？”

燕使坐得稍远，等前面三人都说完了，才缓缓道：“燕道里本远，若一切都依近国急势来定，燕怕最先落个失约之名。”

几句话说下来，人人都不是说不援。

人人都只是在把“现在先写死”往后推。

苏秦听完，反倒比众使更静。

他早料到会这样，只是亲耳听见，胸口还是像被什么压了一下。

“好。”他把那卷草约往前推了推，“兵期诸君不肯先写死，那便退一步。只写一条：若近国告急，众国不得以观望为先，须先共议共动，可否？”

这一回，连赵廷陪坐的几名重臣都暗暗皱眉。

因为这一条一旦写下，赵自己也要被捆进去。往后再遇急报，便不能总先把“我再看看别人动不动”挂在嘴边。

齐使沉吟片刻，道：“共议可写，共动二字太硬。”

魏使点头：“先议可以，先动难。”

韩使苦笑一声：“若只写议，韩不反对；若带一个动字，韩不敢诺。”

燕使更干脆：“燕亦如此。”

偏殿里一时只剩烛火轻响。

苏秦望着案上那卷草约，忽然觉得自己仿佛不是在看一张纸，而是在看一块始终钉不进去的木头。

他抬头，目光从诸使脸上一一扫过去：“诸君都认秦是大患，都认旧盟尚在。可真到要写下一句能先约束自己的话时，为何又都退了？”

韩使先低下头去，半晌才道：“相国，韩不是不想写。只是韩最怕的，正是纸上写得重，刀口来时却总等不到人。”

魏使也道：“魏若先把自己钉死，旁国明日若仍说‘再看一步’，魏岂不先成露锋之国？”

齐使没有辩，只用手指慢慢抹着杯沿。

那动作本身，便比言语更像回答。

赵王坐在上首，把这一切都看在眼里。

他也知道，若今日这场小议连一条“先共动”的约都定不下来，后面所谓盟仍在，便更像一张厅堂里挂着的旧画。可他自己同样张不开嘴。

因为只要他先张嘴，赵便也得先被绑上去。

而如今的赵，最怕的正是这个。

苏秦没有再逼。

他忽然发现，再往下逼，众使只会把眼前这些话说得更细、更像道理。那样一来，反倒连这场小议里剩下的那点羞意也要被磨掉了。

最终，案上的草约还是改了。

把“共动”改成了“共议”；

把“迟缓之罚”改成了“后议”；

把最硬的几行字，一笔笔都削成了能留余地的样子。

傍晚散席时，人人都还守着礼。

齐使说旧盟不可轻；

魏使说往后仍当互通急报；

韩使说韩虽薄，绝不敢忘西患；

燕使也说若真有大变，燕不会坐视。

话都不难听。

可苏秦捧着那卷最后议成的文书，心里却只剩一种极冷的明白：

这一回里，没有谁公开弃盟。

却也没有谁肯先把自己钉住。

夜里，诸使各归馆舍。

苏秦独坐偏殿，把那卷文书摊在灯下，看了很久都没卷起。

门客走进来，轻声道：“相国，至少他们还肯来，还肯认盟。”

苏秦点了点头：“这自然比当场翻脸好。”

他顿了顿，才又道：“可你看见没有？今日输掉的，不是理，也不是名。”

门客站住了。

苏秦把手指按在文书中段那两处被削去的字上，慢慢道：“输掉的是一个敢字。”

“敢字？”

“是。”苏秦道，“若总得有人先把自己约束死半步，众人才可能真聚。如今人人都还来，人人都还说，偏偏谁都不肯先把那半步钉在自己脚上。这样一来，后面就只剩可议，不剩可做。”

门客沉默了许久，才低声道：“那还再议么？”

苏秦把文书慢慢卷起，道：“议。”

“还议？”

“议。”苏秦答得不重，却很定，“他们今日不肯写死，我明日便再逼一次别的事。总得有人一遍遍去逼，天下才不会把这种退法退得像本来就该如此。”

说完，他起身走到殿门前。

门外夜风很冷，堂下灯焰被吹得斜斜一晃。

苏秦看着那一点火，忽然伸手去护了一下。

护的明明只是一盏灯。

门客看在眼里，却忽然觉得，他护的更像这场还没公开散掉、却已越来越轻的旧盟。
