# 第四十八回 小议既散边报又添冷色 前线无声兵气渐压纵横

邯郸小议散后不过数日，边上便又来了新报。

秦军并无大举东出，只是在几处边地添垒、换守、前逼一步。若只看兵数，这几封军报算不得惊人；可偏偏来得正是时候，像一根细针，正扎在那场小议没能写成的几行字上。

赵廷最先得报。

报书一到，赵王便召群臣入殿。苏秦也在。

众臣看完之后，第一句问的果然不是“依旧盟该如何共应”，而是“赵边备是否已足”“河外诸仓可否再补”“西陲骑军要不要再调一层”。

这些问题一个不差。

可也正因为都不差，合在一起便把那句本该同时被问起的话压到了后头。

赵王坐在上首，听着听着，心里竟生出一种莫名的疲。

不是因为群臣短视。

恰恰相反，正因为他们都不短视。他们知道秦这一回只试一步，后面便可能再试第二步；也知道山东若无共应，赵迟早独重。可事情到了如今，再叫他们先不从赵自己的兵粮边仓去想，反倒像是在逼人先闭眼。

苏秦见殿中气口已定，便上前一步：“臣请先问一句。若秦明日再进两步，赵愿不愿立刻飞书齐、魏、韩，而不先只议本国边筹？”

殿中微微一静。

一名近臣答道：“告，自可告。然告之外，总还得先把赵自己的兵粮算定。”

苏秦点头：“正是。可如今最坏处，便在于天下已越来越习惯把这句话放在第一句。第一句一变，后面所有同盟之辞都会慢半拍。”

他没有再多说。

因为这一句若再往下说，便要说到赵最不愿听的地方去了。

魏、韩那边的反应比赵更快，也更冷。

它们一得边报，先做的不是往远处发书，而是彼此先通消息。魏王命人重检河外渡具，韩相连夜催近边仓车。两边使者半路相遇，讲的也不是“天下共势”，而是“若秦在此处再逼一步，你我各能先顶几日”。

这当然也可算守秦。

可已不是昔日那种先有大局、再分远近的守法了。

而是先把离自己最近、最眼下能用得上的那一圈扣紧，其余后说。

齐国这边，看见边报，却仍没有把对楚的援兵再往前推得多快。

齐王问田文：“秦边这点动静，值得齐先更深压进去么？”

田文答：“尚不值得。秦这一回更像试。若齐一见试边便先重压，便是替别人把价抬高了。”

齐王点了点头，仍命前线维持旧数，不增不减。

这姿态最叫人无话可说：

不弃楚，

不尽楚，

不失价，

也不先受最重之锋。

楚廷对此却最敏。

怀王一闻诸国最新反应，脸色便更不好看。他从中听出来的，不只是“众国都在防秦”，而是“众国都在防自己先被别人拖进去太深”。

屈原因此再谏：“大王，越到此时，越不可让诸侯觉得楚自己也还在两边留手，却总盼别人先替楚多尽一步。否则边报越来，众国越会先守其门，不会先问我楚。”

怀王没有立刻反驳。

因为他也能感觉到，如今每一次秦边稍逼一步，诸侯的第一反应，已越来越不是“如何共顶这一逼”，而是“如何别叫我自己先露出最大的空门”。

昭阳在前线读到这些边报时，感受则更直。

他站在营外，远远看着秦军新添的那几处小垒，只对左右道：“从今往后，怕是要看谁手里真有兵，谁的话才更重。”

左右将领一时没听懂。

昭阳便抬手指了指远处：“昔日纵横家一句话，能先把众国往一处拽。如今众国心里都先各自有了退路，光靠一句话，越来越难压人。要紧的，还是谁能把兵、垒、粮、路先稳在前头。”

说完这话，他自己也沉默了一下。

因为这并不是在轻辩士。

恰恰相反，是最懂局势的人先看见：时代里更重的东西，正在慢慢换手。

同一夜，苏秦还在赵馆。

他把赵、魏、韩、齐四处新报一封封摊开，看完之后，只把它们按次序压在一处，没有立刻写回书。

门客问：“相国，这几处报闻里，最坏的是哪一封？”

苏秦道：“不是哪一封最坏，是它们都在说同一件事。”

“何事？”

“秦每逼一步，诸侯先看的都不是谁共应得快，而是谁别先把自己露空。”

门客听完，半晌无语。

苏秦这才慢慢提笔，在纸角写了四个字：

“兵气压言。”

写完之后，他并未立刻往下写，只把笔搁在砚边，望着案上那摞军报很久。

因为他知道，这四个字一旦真成了天下共识，后面再想单凭辩说把众国重新压到一线，便会比从前难上许多。
