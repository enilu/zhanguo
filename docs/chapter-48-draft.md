# 第四十八回 邯郸小议刚散秦边又试一步 昭阳望新垒时已知兵气开始压过言辞

邯郸小议散后不过数日，边上便又来了新报。

秦军并无大举东出，只在几处边地添垒、换守、前逼一步。若只看兵数，这几封军报都不惊人，可偏偏来得正是时候，像细针一样扎在那场小议没能写成的几行字上。

赵廷最先得报。

报书一到，赵王召群臣入殿。众臣看完后，第一句果然不是“依旧盟该如何共应”，而是“赵边备是否已足”“河外诸仓可否再补”“西陲骑军要不要再调一层”。

这些问题一个不差。

也正因为都不差，合在一起便把那句本该同时被问起的话压到了后头。

魏、韩那边的反应比赵更快，也更冷。魏王得报后，先命重检河外渡具；韩相连夜催近边仓车。两边使者半路相遇，讲的也不是天下大势，而是“若秦在此处再逼一步，你我各能先顶几日”。

齐国这边，看见边报，却仍没有把对楚的援兵再往前推得多快。

齐王问田文：“秦边这点动静，值得齐先更深压进去么？”

田文答：“尚不值得。秦这一回更像试。齐若一见试边便先重压，便是在替别人把价抬高。”

齐王点头，仍命前线维持旧数，不增不减。

楚廷对此最敏。

怀王一闻诸国最新反应，脸色便更不好看。屈原因此再谏：“越到此时，越不可让诸侯觉得楚自己也还在两边留手，却总盼别人先替楚多尽一步。”

昭阳在前线读到这些边报时，感受则更直。

他站在营外，远远看着秦军新添的那几处小垒，只对左右道：“从今往后，怕是要看谁手里真有兵，谁的话才更重。”

左右将领一时没听懂。

昭阳便抬手指向远处：“昔日一句话，能先把众国往一处拽。如今众国心里都先各有退路，光靠一句话，越来越难压人了。”

同夜，苏秦还在赵馆。

他把赵、魏、韩、齐四处新报一封封摊开，看完之后，只在纸角写了四个字：

“兵气压言。”

写完，他把笔搁在砚边，望着案上那摞军报，很久都没有再落下一句。
