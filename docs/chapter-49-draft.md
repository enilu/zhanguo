# 第四十九回 苏秦再缩一层只议近秦三国 兵期终空时连韩国也不敢再先信盟

邯郸小议既散，边报又到，苏秦仍不肯认输。

这一回，他不再碰齐，不再碰楚，只把事缩到赵、魏、韩三国之间，缩到最贴刀口的地方。

他亲自写了三封短书，只问三件事：

若秦压韩，则赵魏几日内应；

若秦逼魏，则赵韩几日内应；

若秦胁赵，则魏韩几日内应。

几日后，三国使者都到了。

这一场议比前番更冷。桌上不再摆旧盟之辞，只摊着一张草约，最中间空着三行兵期。赵使先看完，便道：“赵非不愿应。然秦若真专压韩魏，赵须先判其意是在试边，还是在引赵出锋。”

魏使皱眉：“若事事都先判清秦意，等判清时，近国怕已先受足了刀。”

韩使坐在最末，听到这里，忽然冷笑一声：“韩今日总算听见真话了。韩最怕的，从来不是秦太强，是人人都知韩最薄，口头却都能把援韩说得极重，真落到兵期时又总要再看一步。”

一句话，把席上又打回原处。

苏秦等三人都说完，才把草约往前推了一寸：“今日不谈大义。只谈一件最小、也最硬的事。韩若先受锋，赵魏肯不肯先在纸上把日子写死？”

赵使低头不语。

魏使则道：“写死容易，真临边时若旁国未至，写死的便成了先把自己送出去。”

韩使听完，反而笑意更冷：“也好。韩至少从今日起，知道不必再把盼头先压得那么死。”

说完，他是第一个起身出门的人。

那背影比来时还冷。

到散席时，草约上的三行兵期仍旧空着。

夜里，苏秦把那张未定成的草约摊在灯下，看了很久。门客轻声道：“相国，这一次，当真难起了。”

苏秦没有立刻答，只伸手按在那几处空白上，隔了很久才道：“不是今日才难起。是今日才肯承认，原来连退到这样小的一条线，也还是难起。”

说完，他另取一张空简，重新写下赵、魏、韩三国的名字，仍旧摆在案上，没有收。

同一夜，咸阳也得了简报。

秦王看完，只问张仪一句：“苏秦可还会走？”

张仪答：“会。”

“既如此，何以说成？”

“正因他还会走，才更见这一轮已成。”张仪道，“若他全然绝望，天下反倒容易惊醒；如今他仍在守，诸侯也仍肯听一听，可听来听去总定不成，后面他们便会越来越习惯：原来合纵可以一直挂着，却不必真先为它吃亏。”

秦王听后，低头把那份简报重新折起，压在了案边。
