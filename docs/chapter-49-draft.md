# 第四十九回 苏子再图会盟终于难起 张仪静坐咸阳更见其成

邯郸小议既败，边报又至，苏秦仍不肯认输。

他最后还想再试一次。

只是这一次，他不再去碰齐，不再去碰楚，也不再去谈天下一张大网，而是把事情缩到赵、魏、韩三国之间，缩到最贴刀口的地方。

他亲自写了三封短书，约三国再议，只问三件事：

若秦压韩，则赵魏几日内应；

若秦逼魏，则赵韩几日内应；

若秦胁赵，则魏韩几日内应。

门客看完那三句，轻声道：“相国，这是把合纵缩到骨头里了。”

苏秦封书时“嗯”了一声：“若连骨头都立不住，皮肉再多也无用。”

几日后，三国使者都到了。

这一场议，比前番邯郸小议更冷。

席上没有齐、燕，也没有那些还能在大势上多说几句漂亮话的大国余地，只剩近秦三国彼此对坐。桌上摊的不是盟书旧辞，而是一张草约，最中间空着三行兵期。

赵使先看了一遍，便道：“赵非不愿应。然秦若真专压韩魏，赵须先判其意是在试边，还是在引赵出锋。”

魏使听完就皱起眉：“若事事都先判清秦意，等判清时，近国怕已先受足了刀。”

韩使坐在最末，听到这里，忽然冷笑一声：“故韩才最难。诸君都知韩最薄、最先受，可一到真要写死谁几日之内必应，又都先有各自要判、要看、要缓的道理。”

一句话，把席上又打回原处。

苏秦坐在主位旁，一直没有插言。

直到三人都说完了，他才把那张草约往前推了一寸：“今日不谈大义。只谈一件最小、也最硬的事。韩若先受锋，赵魏肯不肯先在纸上把日子写死？”

赵使低头不语。

魏使则道：“写死容易，真临边时若旁国未至，写死的便成了先把自己送出去。”

韩使听了这句，反而笑意更冷：“相国，韩今日总算听见真话了。韩最怕的，从来不是秦太强，是人人都知韩最薄，口头却都能把援韩说得极重，真落到兵期时又总要再看一步。”

这话伤人。

却也真。

苏秦看着韩使，过了片刻，才道：“故我才仍坐在这里。不是要诸君再口头许我多少，是要把一件事当面问穿：若连最薄处都只等得到一句句‘再看一步’，那后面人人再说守盟，究竟是守给谁听？”

席上顿时无声。

赵使与魏使相顾一眼，都没有立刻接。

因为他们不是不知韩说得重，也不是不知苏秦这话刺心。他们只是太清楚，自己手里握着的是本国兵、粮、边与君命，谁也不愿率先把那一步钉成真会牵着自己往外走的死线。

于是这场比前番更小、更近、更实的共议，仍旧没能定成。

草约上的三行兵期，到散席时还是空着。

韩使是第一个起身的人。

临出门前，他回头看了一眼那张空着的草约，淡淡道：“也好。韩至少从今日起，知道不必再把盼头先压得那么死。”

说完便走。

那背影比来时还冷。

苏秦望着他走出殿门，胸口忽然也跟着凉了一下。

因为他到这一刻才真正看清：合纵坏到今天，最残酷处已不只是强国各算各的，而是连最薄、最痛、最该先盼着众国相援的国家，也开始不敢真信那句“众国终会相援”了。

夜里回到馆中，苏秦把那张未定成的草约摊在灯下。

约文只差最要紧的几行兵期与罚约，空在那里，像几处无论如何都填不满的缺口。

门客轻声道：“相国，这一次，当真难起了。”

苏秦没有立刻答。

他伸手按在那几处空白上，许久才道：“不是今日才难起。是今日才肯承认，原来连退到这样小的一条线，也还是难起。”

门客问：“那还守么？”

苏秦慢慢把草约卷起，道：“守。”

“守什么？”

苏秦看着手里那卷草约，道：“守这件事别被人忘得太快。后面若还有人想重提这局，至少要有人记得，它不是一下子碎的，是一行行兵期空出来，一句句‘再看一步’磨出来的。”

这话已冷到近乎自为后人作记。

可说完之后，苏秦却并未停手，而是另取一张空简，重新写下三国名字，仍旧摆在案上。

门客看了片刻，终究没再劝。

因为他知道，相国今日虽已认账，却还没认输。

同一夜，咸阳那边，张仪并未去搅这场小议。

他只在事后得了简报，看完之后，对秦王道：“连韩都已不敢真信‘众国终会相援’，这局便已坏到骨里了。”

秦王听后，问了一句：“苏秦可还会走？”

张仪答：“会。”

“既如此，何以说成？”

张仪把简报放下，声音很平：“正因他还会走，才更见这一轮已成。若他已全然绝望，天下反倒容易惊醒；如今他仍在守，诸侯也仍肯听一听，可听来听去总是定不成，后面他们便会越来越习惯：原来合纵可以一直挂着，却不必真先为它吃亏。”

秦王闻言，竟也沉默了一下。

因为这不是简单的胜负。

而是一面本来真曾被人拽起的大旗，如今被另一只手一点点拆到连最贴刀锋的几个人，都不敢再把命押在它上头。
