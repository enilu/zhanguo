# 第五十回 旧符在案苏子知时变 舆图无声秦廷见兵来

邯郸那场小议散后，苏秦在馆中留到很晚，才肯起身。

堂中几案上还摊着白日未收的文书。赵、魏、韩三国往返改过的草约堆在一处，删掉的字多，真定下来的字却少。案角烛泪流得极长，把几页未写死兵期的空白照得尤其刺眼。

门客上前替他卷简，卷到一半，忍不住道：“相国，夜深了。”

苏秦嗯了一声，却没有立刻动。

他目光还停在那几处空白上。

白日堂中诸使说的每一句话，他此刻都记得清楚。不是因为那些话多新，而是因为它们几乎没有一句说得不对。赵要看秦意，魏怕先露锋，韩不信别人真能及时来，燕嫌道里太远。每一句都合情，每一句也都能自保。

可合在一起，便什么也压不成。

门客低声道：“诸国至少还肯来。”

苏秦这才抬起头，笑了笑，笑意却极淡：“肯来，自然比不来好。只是从前人进这道门，心里先装着‘得把事议成’；如今人进这道门，先想的是‘别让我先显得不肯听’。你说这是同一回事么？”

门客一时无言。

苏秦把最后那卷草约慢慢收好，起身走到窗边。窗外邯郸夜色沉沉，坊市灯火已疏，只余远处更鼓断断续续传来。

他忽然想起自己当年初入赵廷时的样子。

那时他一身困顿，胸中却有一口极硬的气。六国各怀疑虑不假，可总还有人肯在“若秦东出，天下共危”这句话里先信一半。也正是靠着那一半，后来的会盟、佩印、共势，才一件件立了起来。

如今这口气还在不在？

也在。

只是已被拆得太散。

散到人人都还认得“大局”两个字，真到要把兵、粮、期限、罚约往纸上钉时，却总先想起本国眼前那一口更近的利害。

苏秦转回案前，打开箱匣，从里头取出一枚旧符。

那是当年佩六国相印时留下的一件旧物，铜色已暗，边角也磨旧了，拿在手里却仍沉。

门客见他把那旧符搁在案上，神色更不敢轻动。

苏秦用手指轻轻抹过那枚旧符，忽然问：“你说，当年我立的是不是假的？”

门客吃了一惊，忙道：“相国何出此言？若不是相国，当年六国哪里会真有那般声势。”

苏秦点头：“声势自然是真的。”

他说完，停了停，才又道：“只是那时我总想着，先把名拢起来，后面的兵和骨，慢慢总能再补。如今看，名可以先立，骨却未必肯等。”

这话落下，屋中更静。

门客站在一旁，连呼吸都放轻了。

苏秦没有再多说，只把旧符重新收回匣中，合上盖子，道：“明日把去赵王那封未发的书再誊一遍。”

门客道：“还发？”

苏秦道：“发。不是指望一纸能翻回来，只是总得有人把该说的话继续放在那里。往后谁还想再提这局，也好知道它不是一夕死的。”

同一时刻，郢都军中也在点灯。

昭阳披甲坐在军案前，面前摊着边图与粮册。白日里又有一支秦骑试边，人数不多，来得快，退得也快，像不是来战，倒像是专来量楚营反应快慢。

副将道：“秦人这些日子越发不肯正面扑了。”

昭阳道：“它不急。”

副将不解：“既知齐楚仍在，为何不急？”

昭阳把手里的木筹往图上一放，点在两营之间：“正因看见我们都在，它才更不急。秦怕的不是营盘挨着营盘，是两路兵真能踩成一个拍子。如今它已看见我们拍子不齐，自然愿多试、多磨。”

副将沉默片刻，低声道：“那往后天下与秦争，莫非越来越只看谁兵实、谁粮足、谁边固？”

昭阳抬头看了他一眼，没有立刻答。

风从帐门缝里灌进来，把案上一角军报吹得轻轻翻起。过了半晌，他才道：“说客当然还要紧。可再要紧，也得先有能受他一说的骨头。骨头若松了，再大的话，也只能拿来遮遮脸。”

副将听罢，心里一震。

昭阳却已不再往下说，只道：“把后营转粮时日再往前提三天，东垒再添拒马。秦如今试的，不止我们会不会应，还试我们能不能日日守住。”

这已不是纵横家的话。

是守边大将的话。

而楚宫之中，屈原与陈轸也正坐在一处。

桌上无酒，只有一盏快冷下去的茶。两人身前摆着白日朝会后留下来的几份奏简，写的却不是使秦、使齐，而是郡县赋调、军中补给、前后军令如何归一之类的琐细事务。

屈原看着那几份简，道：“这些事从前也有，只是从前朝中说得更响的，总是诸侯和盟。”

陈轸苦笑了一下：“从前人心还热，响话压得住。如今响话还在，底下却越来越要靠这些细事真顶着。”

屈原沉默片刻，道：“如此说来，我们这些人是不是越来越无用了？”

陈轸摇头：“不是无用，是不能只凭我们了。”

他伸手把一份军需条目推到屈原面前：“你看，前线若缺一旬粮草，任你在殿上说得再好，能替哪一军站住？君主若今日听齐、明日听秦、后日再听边报，日日改口，任你把大义讲得再正，能替哪一道军令长久？”

屈原听着，没有反驳。

他明白，陈轸不是在轻看辩说，只是这些日子越发看清：再响的道理，终究也要有人把粮、令、城、军一件件硬顶住。

良久，屈原才道：“可若连这层都被逼成了先看眼前、先圆今日，那国也就真只剩空壳了。”

陈轸点头：“所以才更难。”

两人相对而坐，谁都没再说话。屋外夜色浓沉，宫中远远传来更鼓，一声一声，都像敲在这些无人爱听、却越来越躲不过去的琐事上。

而在更西边的咸阳，秦王也尚未就寝。

咸阳宫中灯火亮得比别处都稳。大殿地上摊着一幅新补过的舆图，函谷以东诸国边线被重描过几处，楚、魏、韩边上还压着几枝细竹签。张仪站在图旁，樗里疾亦在，三人之间并无太多铺张辞色。

秦王先问的，也不是齐楚嘴上怎么说。

他问：“韩边今岁可受几压？”

樗里疾答：“若只试边，不求大取，可多做两回。若要真逼，则须再看魏赵反应。”

秦王又问：“楚军这些日子最怕什么？”

张仪道：“怕不是秦兵，先是自己日日要靠谁。”

秦王看了他一眼：“那寡人如今最该看什么？是诸侯还会不会再说合纵？”

张仪摇头：“说还是会说。”

他说这话时，声音很平。

“可大王往后更该看的，不是他们嘴上还怎么结线，而是谁说完之后，手里真有多少兵，国中真能撑多少日，边上一纸军令能不能稳稳落下去。使者还是要走的，只是再响的话，也得靠这些实东西托着。”

秦王没有立刻出声。

他低头看着舆图，手指缓缓从楚边移到韩魏之间，又从韩魏之间移回关中。那一下下移动很慢，像是在量地，也像是在量时。

过了一会儿，他才道：“如此说来，天下往后比的，便不只是会不会结盟了。”

樗里疾接道：“还要比谁练得住兵，守得住粮，熬得住年头，压得住朝局。”

秦王终于笑了一下，笑意极淡：“这倒正是秦这些年一直在做的事。”

张仪没有接笑，只道：“臣做的，不过是叫诸侯更难再把命押在那张盟上。后面一步步往前走，还得看秦国自己这副骨头。”

这句话说完，殿中便安静了。

安静里，反倒更显出一种极清楚的东西来。

从前诸侯论天下，最响的是谁合谁纵，谁离谁附，哪位说客今日又翻了哪家朝堂。如今这些事当然仍在，可它们背后站着的分量，已经慢慢换了。

往后谁还能扛，谁还能逼，谁还能久压不倦，越来越要看军、看吏、看粮、看法，看那副能不能年复一年把人和事都拧到一处去的国骨。

殿外夜色正深，风吹宫檐，铜铃细响。

秦王把一枝细竹签压在韩魏之间，终于道：“先把东边试得更勤些。”

樗里疾应命。

张仪亦俯身而退。

待二人退出大殿，秦王却没有立即离开。他仍站在那幅舆图前，看着函谷以东一片并未真正安静下来的天下。

他知道，张仪还会继续出手，诸侯也还会继续往来使节，苏秦更不会就此停步。

可他更知道，往后那条路，已不只在口舌之间。

同一夜，邯郸馆中，苏秦终于把写给赵王的那封书誊完。

誊完之后，他没有立刻封缄，只把它压在那只装旧符的匣旁，默默看了很久。门客已在外间睡下，屋里只余一灯。

他伸手碰了碰匣盖，忽然轻声道：“局是我起的，终局却未必还能等我。”

说完这句，他自己也怔了一下。

怔的不是失意，而是终于肯把这层话说出口。

窗外北风掠过屋角，灯火轻轻晃了一晃。

苏秦抬手护住火苗。

那姿势，像护着一点还没有灭尽的旧光。

可他心里已经明白，下一卷再站到天下前头的，未必还是自己这样的人了。
