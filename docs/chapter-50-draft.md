# 第五十回 旧符收匣时苏秦终知风变 咸阳舆图无声秦王已把目光移向兵粮

邯郸那场更冷的小议散后，苏秦在馆中留到很晚。

几案上还摊着白日未收的文书，赵、魏、韩三国改过的草约堆在一处，删掉的字多，真定下来的字却少。案角烛泪流得很长，把那几处空着兵期的地方照得尤其刺眼。

门客上前替他收简，卷到一半，低声道：“相国，夜深了。”

苏秦嗯了一声，却没有立刻动。

他从箱匣里取出一枚旧符，铜色已暗，边角也磨旧了。门客看见那枚旧物，便连气都放轻了。苏秦用手指慢慢抹过符面，忽然问：“你说，当年我立的是不是假的？”

门客吃了一惊：“相国何出此言？若不是相国，当年六国哪里会真有那般声势。”

苏秦点头：“声势自然是真的。”

他说完，把旧符放到案上，停了一会儿才又道：“只是那时我总想着，先把名拢起来，后面的兵和骨，慢慢总能再补。如今看，名可以先立，骨却未必肯等。”

他说这话时，手还压在那枚旧符上。

门客站在一旁，一时竟不知道该接什么。

苏秦最后把旧符收回匣中，合上匣盖，道：“明日把写给赵王那封未发的书再誊一遍。”

门客忍不住问：“还发？”

“发。”苏秦道，“不是指望一纸翻回来，只是总得有人把该说的话继续放在那里。往后谁还想再提这局，也好知道它不是一夕死的。”

同一时刻，楚军帐里也在点灯。

昭阳披甲坐在军案前，面前摊着边图与粮册。白日又有一支秦骑来试，人数不多，来得快，退得也快。副将道：“秦人这些日子越发不肯正面扑了。”

昭阳把木筹往图上一放，点在楚齐两营之间：“正因它看见我们都在，它才更不急。秦怕的不是营盘挨着营盘，是两路兵真能踩成一个拍子。”

屈原与陈轸那边，也正对着几份军需条目和朝议奏简。

屈原看着那些簿册，低声道：“从前朝中说得最响的，总是诸侯和盟。如今这些东西，倒压到前头了。”

陈轸苦笑：“不是辩说无用了，是再响的话，也得有人把粮、令、城、军一件件顶住。”

更西边的咸阳，大殿上铺着一幅新补过的舆图。

秦王站在图前，先问的已不是谁嘴上还会不会再说合纵，而是韩边今岁可受几压，楚军这些日子最怕什么，魏韩之间再试一步会露出哪道口子。樗里疾在旁答图，张仪只在最后补了一句：“说客还会走。只是往后再响的话，也得靠兵、粮、法、令这些实东西托着。”

秦王听完，没有立刻说话。

他低头看着舆图，手指从楚边移到韩魏之间，又从韩魏之间移回关中。隔了片刻，才道：“先把东边试得更勤些。”

张仪与樗里疾一齐俯身。

夜更深时，二人退出大殿，秦王却还留在图前没有离开。灯火照着函谷以东那一片边线，压得很稳，连一角都没有翻起来。
