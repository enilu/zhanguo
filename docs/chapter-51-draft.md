# 第五十一回 咸阳案上先换边图粮册 白起初入殿时开口只问道路

第二卷过去，诸侯使者还在路上来回。

可咸阳宫里先换了东西。

从前殿上最常摊开的，是盟书、回简、使报；这一日摆在案中央的，却是一张新补过的关东边图，旁边压着粮册、役籍和几卷道里簿。

秦王升殿后，没有先问诸侯近来又说了什么，只叫樗里疾上前，把韩、魏、赵边上新添的渡口、守具、粮垒一处处指出来。张仪也在殿中，却少见地没有先接话，只站在一旁看图。

秦王看完，先问了一句：“诸侯近来添得最多的，是使，还是守具？”

樗里疾答：“守具。”

殿里一下就静了。

这时门外军吏请见。

来人甲色很素，身量也不出奇，进殿行礼时却极稳。樗里疾道：“此人白起，近来在军中颇有见识，臣请大王一问。”

秦王问他：“听说你在军里常说，先问路，再问敌。何意？”

白起伏地答：“兵要到，粮要继，令要达，都先靠路。路不通，敌未至，军先死在脚下。”

话不漂亮，却很实。

秦王又问：“若韩魏近来都往边上添兵，你先看何处？”

白起起身走到图前，没有先指王都，也没有先点大城，只把手落在伊阙山口与几条道隘之间：“先看这里。”

“为何？”

“韩魏若并兵，最易会于此。”白起道，“若它们真能并，秦便先看地；若它们各有顾忌，也会在这里先露出来。”

张仪听到这里，目光微微动了一下。

秦王则一直看着白起，没有插话。等白起把伊阙几条路、几处谷和两国兵若会于此会生什么掣肘说完，秦王才缓缓道：“你退下后，不必回旧营，且留咸阳。”

白起领命而退。

他走后，秦王仍把手按在伊阙那一线，没有挪开。张仪终于开口：“臣拆的是诸侯嘴上的盟，此人想看的，是它们到了阵前还能不能把这盟撑成一股兵。”

秦王点了点头。

殿外风起，把图角吹得轻轻翻了一下，正露出伊阙那一小片山口来。
