# 第五十一回 咸阳先问粮道兵符事 辩士未退边将已登台

第二卷过去之后，天下朝堂上仍旧离不开使者。

可咸阳宫里最先变的，已经不是说话的人，而是案上的东西。

从前张仪在时，殿上铺得最多的是诸国回书、盟辞、使报；如今这些东西仍在，却慢慢退到了案角。摆在正中的，变成了边图、粮册、道里、军次、役籍。

这日清晨，秦王升殿，先叫人抬上的便不是书简，而是一张新补过的关东舆图。

樗里疾立在图侧，把韩、魏、赵三国近边新添的守具、粮垒和渡口一一指给秦王看。张仪仍在殿中，却少有地没有先说话，只立在一旁静听。

秦王听完后，先问：“诸侯还在不在相互致书？”

樗里疾答：“在。”

“礼还在不在？”

“也在。”

秦王点头，又问了第三句：“那它们近来先添的，是使节，还是守具？”

樗里疾这才笑了一下：“是守具。”

殿中众臣都明白，这一句问得比前两句更重。

因为它问的已经不是诸侯嘴上怎么说，而是它们心里如今先怕什么。

张仪这时才开口：“大王，诸侯如今还会彼此招呼，可每一国先动手补的，都是自己门前那几步。这样的天下，再要只凭纵横压住，便越来越难了。”

秦王听后，没有立刻答话，只把目光从舆图上移开，扫过殿中诸臣：“既如此，秦往后先该用谁？”

这一问，殿里气口顿时一变。

因为谁都听得出来，大王问的已不是该遣哪位使臣，而是该遣哪一路兵、用哪一类人了。

樗里疾上前一步，道：“若大王问臣，臣以为往后当多用知粮道、知兵势、知久战之人。如今诸侯心都先往回收，秦只要压得稳，后面许多路都会自己显出来。”

秦王点了点头，目光又落在舆图上韩、魏相接的一线，许久才道：“那便先看这条线。”

正说着，殿外有军吏请见。

来人不过中年，身形并不魁伟，甲衣也素，入殿后行礼极稳，不见半点张扬。樗里疾低声道：“此人姓白，名起，近来在西边军中颇有见识，臣前日已命他入咸阳候问。”

秦王抬眼看了白起一眼：“你便是白起？”

白起答：“臣是。”

秦王道：“听说你近来在军中最常说的一句话，是‘先问路，再问敌’。何意？”

白起伏地道：“敌在前，可路在脚下。兵要到、粮要继、令要达，前后都要靠路。路若不通，纵有十万之众，也只是把人推去挤死。路若通了，敌情便有许多事可慢慢算。”

这话并不漂亮。

却比许多漂亮话更像军中真言。

殿中几名旧臣都微微抬眼，多看了这名军吏一眼。

秦王又问：“那你再说一句。若如今诸侯虽未绝盟，却都先顾自己，秦该怎么打？”

白起答得更短：“不求一口吃尽。专找它们最难齐心的一线，逼它们先把各自那点不肯并力的病全露出来。”

张仪听到这里，眼神微微一动。

因为他听得出来，这人说的虽是兵，却与自己这些年所做之事，竟隐隐接成了同一条线。

自己是先把诸侯心里的裂缝挑出来；

这人则像是要把那裂缝带到阵上去打。

秦王沉默片刻，忽然问白起：“若寡人今日命你去看韩魏边上那几条道，你先看何处？”

白起起身，走到舆图前，没有先指大城，也没有先指韩魏王都，只把手落在伊阙一带的山口与道隘之间：“臣先看此处。”

樗里疾眼皮微动。

秦王道：“为何？”

白起答：“韩魏若要共守，兵便须会于此；若会于此，便有地利可借，也有彼此掣肘可看。此处若看得透，韩魏到底是两国同心，还是两支兵并肩而立、心各有算，便大半都能看见了。”

殿中一时无声。

因为这话已经不只是说路了。

它是在说人。

秦王听完之后，转头看向张仪：“你拆了他们的心，如今军中却有人想去看那心在阵上会怎么坏。寡人今日倒是看明白了一层。”

张仪俯身道：“臣拆的是口舌之盟。后面要把这天下往更冷处推去，还得靠大王手里的兵。”

秦王没有再多言，只命白起起身，又问了他几条极细的事：一日行程、山谷屯粮、车骑换守、两国并军时令出何门最易乱。

白起一一答了。

答法都不华丽，却极准，也极冷。

等他退下后，殿中一时没人先说话。

直到秦王把手按在舆图韩魏之间那一片地方，才缓缓道：“寡人今日算是真看见了。往后这天下，光有会说的人，还不够。”

樗里疾道：“说的人仍要有。”

秦王道：“自然要有。只是说完之后，还得有人把兵真正带过去，把路真正踩出来，把别人不肯并的一股气，逼到阵上去见个高下。”

这话一落，张仪也没有反驳。

因为他自己心里也清楚，第二卷走到今天，辩士并没有死，甚至以后还会继续左右人心。可时代里更重的那只手，已经开始慢慢换到另一批人那里去了。

同一时刻，邯郸那边也在议兵。

赵王这一日召近臣入殿，案上摆的全是边务与军伍，不见半卷盟书。苏秦亦在座中，看着这些东西，心里先是一沉，继而又慢慢平下来。

因为他知道，从今往后，自己再入朝堂，旁人先要听的，不会再只是“该不该合”，而是“若真打起来，谁的兵先到，谁的粮先够，谁的城先扛得住”。

门客夜里问他：“相国，往后是不是更难说动人了？”

苏秦坐在灯下，翻着赵地最新的兵册，过了很久才答：“更难。可也不是全无可说。只不过以后再说天下，先得把兵、粮、路、城这些更硬的东西一并说进去了。”

门客听后，半晌无语。

苏秦把兵册合上，轻声道：“旧旗还在。只是风已经变了。后面谁还能立得住，不只看谁说得高，还要看谁手里真有能顶住这阵风的东西。”

窗外夜色正沉。

远处军营传来换更之声，一下下敲得很稳。

苏秦坐在灯下听着那声音，第一次真切地觉出：从今往后，自己身边这卷卷书简之外，另有一批人，已经沿着城、路、谷、垒，悄悄走到天下前头去了。
