# 第五十二回 韩魏并兵伊阙先争主客 秦使白起到谷前只看两营落处

白起留在咸阳不过数日，秦王就把他遣去了伊阙前线。

与此同时，韩魏两国也不得不把兵往一处挪。

魏使到韩时，话说得并不好听：“秦若再一处处试，两国各守各门，终归要被它拆开。与其各坏，不如先会兵于伊阙。”

韩相听完，良久才点头。

道理谁都懂，真正难的却不是会兵，是会兵之后谁为主、谁为客、谁先受秦锋。几日后，两国前军到了伊阙，韩军先占谷口平地，魏军随后而至，营垒扎得更整，却故意离韩营隔了半里。

远远望去，两营并列，像一支大军。

真进了帐，争的第一件事就是主将。

韩将说此地近韩，道里山势韩更熟，自当以韩为主。魏将则说秦锋若来，多半先向魏压，魏兵也更厚，如何不可主兵。两边争到最后，只争出一条折中的军令：

“大令共署，小令各行。”

这令写出来时，苏秦正好赶到。

他看完后没有先劝，只问：“若秦三日内到，谁先迎，谁后应？”

韩将看了魏将一眼，魏将也看了韩将一眼，最后都只答：“临敌再议。”

苏秦听完，什么也没再说。

同一夜，白起已到秦军前部。

他不急着下令，先带数骑上高处看伊阙谷势。谷中两营灯火分得很清：韩营灯稀些，魏营灯密些；韩营巡骑贴着魏营外沿过时绕得更远，魏营炊烟却始终压着中路不肯太散。

副将问：“将军先打哪边？”

白起道：“先不问打哪边，先看谁最怕自己先替对方吃最重那一口。”

山风从谷里直穿过去，把两营的旗一齐压得往后倒。
