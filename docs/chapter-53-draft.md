# 第五十三回 伊阙谷口两军争主客 秦阵山前一将先算人

韩魏兵会于伊阙之后，谷中先热闹了几日。

两国鼓角并起，营垒连绵，往来斥候一日十数趟，外人若站得远些看，只会觉得这一线兵势厚得很，足够叫秦人先掂量一阵。

可真正坐在帐里的将校，心里却没有谁比来时更松。

因为兵虽会了，主客之争却还没完。

共署的大令才下第一道，问题便出来了。

韩军欲先夺谷口东侧一处高坡，借此压住秦军来路；魏军却说高坡虽好，却太近前，若秦故意让出再回扑，联军初合先折一阵，后面更难稳。两边各执一词，争到后头，竟又把那份共署军令推了回来，重新划了改，改了又划。

苏秦在旁看着，心中越发发冷。

他索性不开口劝和，只问一句：“若秦明日便到，这道令今晚定不定得下？”

韩将答：“自当定。”

魏将也道：“自然定得下。”

可说完之后，两边仍在争。

直到夜更已深，才勉强定成一份折中之令：两军同出，韩压东坡，魏护中路，彼此不得越节。

这令写出来，不算坏。

坏在它是折中的。

折中一回尚可，折中回回，到了真要快决的时候，便最易慢。

苏秦看着军令送出营去，忽然对门客道：“如今最怕的，不是他们各有道理。”

门客低声问：“那怕什么？”

苏秦道：“怕他们句句都有道理，偏偏合不成一口气。”

同一夜，秦营里却比韩魏安静。

白起把副将与斥候都叫到帐中，只问白日里看见的三件事：

“韩魏哪边巡营更勤？”

“哪边炊烟更稳？”

“哪边斥候从对方营前过时更绕？”

这些问题听起来都细。

副将却不敢轻忽，一一答了。

白起听完之后，便把木筹一支支排在图上韩魏两营之间，慢慢道：“韩怕自己先受锋，故更想借地形抢高；魏怕自己被引出，故更想守中路。两边都不是不想胜，只是都更怕自己先吃那口最重的亏。”

副将道：“既如此，将军先击韩？”

白起摇头：“先让他们自己再把这口气积深一点。”

“如何积？”

白起道：“明日小逼韩前，不深追；再放一支轻骑绕魏营外。韩会以为魏不肯尽出，魏会以为韩先乱了阵。它们本就没并稳，再添这一层，后面便更难真合。”

副将听罢，心里微微一寒。

因为这打法并不求眼前多斩几人，却更像是在往敌人胸口那层早已有的疑心上，再悄悄压一块石头。

次日天未亮，秦军果然先动。

并非大军压谷，只是一支前锋出东，一支轻骑绕西，看着都不算重。韩将一闻前方告急，立刻披甲上马，命东侧诸军先守高坡；与此同时，又急遣人去催魏军早出。

魏营那边接报，却先传话回来：“中路尚未见秦主力，魏军不可尽空。”

韩将看见回话，脸色当场就冷了。

他回头对左右道：“还未真撞，魏便先把后路看得比前锋还紧。”

左右谁都不敢接。

因为人人都知道，这话虽在怒里，却并不算冤。

魏将那边其实也难。

他站在营门前，远望东侧杀声渐起，心里同样着急。可秦军轻骑在西面绕着跑，尘土扬得很高，看着像随时会扑渡口。他不敢把中路真空出来，只能一边命前军缓出，一边反复探看秦军主力究竟压向何处。

这一缓，战机便又少了半口。

韩军独自先在东侧接住秦锋，高坡虽然未失，却也被打得阵形摇动。韩将急得连发三次旗，要魏军加压中路。

魏军终于动了。

可动得仍晚。

等它推到该到的位置时，秦前锋已经先收一步，韩军也已吃了一口硬撞。双方都不算大败，可这一下打完，韩军上下先起的不是同仇，而是那句更冷的话：

“魏又慢了。”

魏营里也并不好受。

将校回营之后，先听见的不是夸其持重，而是韩营那边传来的怨声。有人忍不住道：“韩军自己急着抢坡，倒怪魏不肯先替它把中路赌出去。”

这些话若放在平时，只是牢骚。

可放在今日联军初合、彼此本就未真信对方的时候，便像火星落在干草上。

午后又议军。

韩将入帐，先把剑往案上一按：“今日若魏军再早半刻，东坡便不会吃得这样苦。”

魏将也冷声回道：“若韩军不是急着先立首功，何至于一早便把阵脚压得那样前？”

两人越说越硬，帐中陪坐诸将连大气都不敢出。

苏秦在侧，看着这一幕，只觉胸口越发发闷。

他忽然明白，第二卷里张仪拆开的那些东西，到今天终于全都被带进帐里、带进阵前、带进了这群真正拿刀持令的人中间。

从前是朝臣彼此不肯先担；

如今连同营并坐的两国大将，也已如此。

他正出神时，帐外忽传急报，说秦军西侧轻骑又动，像是要再试魏营外沿。

魏将闻报，立刻回身欲出。

韩将冷笑一声：“将军果然先顾自己那边。”

魏将霍然回头：“贵军东坡若真稳得住，魏何至总防西面被捅？”

这一句出口，连帐中空气都像冻住了。

昭昭白日之下，两军主将竟已把心里那层最难听的话当面说出来。

苏秦听见，只觉得这一战就算后面还未大败，韩魏那一点原本就不厚的并力之心，也已先碎开了一道口。

同一刻，白起立在远处高地上，把伊阙谷中两营旗号、烟尘、兵走之势看得一清二楚。

副将在旁低声道：“将军，韩魏像是已生口角。”

白起道：“不是像，是已经生了。”

“那何时全压？”

白起目不转睛，只道：“再等它们各自都觉得，对面那一路未必真肯替自己先吃最重的那口气。”

副将听完，不由打了个寒噤。

因为他知道，将军等的已不是地，也不是时。

是在等敌人的心，自己先裂到底。
