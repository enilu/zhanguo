# 第五十三回 东坡初试韩急魏缓半拍 白起立在高处先等两营互生怨声

韩魏会兵后，谷中热闹了几日。

鼓角、斥候、巡骑一样不少，远看声势极厚，帐里的人却谁也没更松。第一道共署军令下去，争的就是东坡那处高地。韩军要先夺，魏军却说太近前，怕秦故意让出再回扑。

争到夜深，才勉强定成：

“韩压东坡，魏护中路。”

次日天未亮，秦军先动。

不是大举压谷，只是一支前锋逼东，一支轻骑绕西。韩营先闻急鼓，韩将披甲上马，开口第一句便问：“魏军动了没有？”

回报只一句：“尚在整队。”

韩将脸色当场就沉了。

他不得不先领本部压上东坡。那一边，魏将也站在营门前看尘头。秦轻骑绕着外沿跑，像随时要扑渡口，他不敢把中路真空出来，只能先探、再看、再缓半拍。

这一缓，韩军就成了独自先受锋的那一路。

高坡上杀声一起来，韩将连发三次旗要魏军加压中路。魏军终于动了，却仍晚了。等它推到谷中该到的位置时，韩军已吃了一口硬撞，阵形也被打得摇了起来。

午后再议军时，韩将把剑往案上一按：“今日若魏军早半刻，东坡不至吃得这样苦。”

魏将也冷声回他：“若韩军不是急着抢坡，又何至一早把阵压得那样前？”

帐里陪坐诸将谁也不敢抬头。

苏秦坐在一侧，听见这两句话，便知道这口并兵之气比军令上看着更薄了。

同一刻，白起站在远处高地，把伊阙谷中旗号、尘头和两营往来得一清二楚。副将低声道：“将军，韩魏已生口角。”

白起道：“再等一日。”

“等什么？”

“等它们都觉得，对面未必肯先替自己受最重那一口。”

说完，他把木鞭往东坡下轻轻一指，又收了回来。
