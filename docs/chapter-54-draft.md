# 第五十四回 白起压谷先切韩魏接缝 伊阙一日打得两营各自先退

第四日清晨，白起终于下了重令。

前军自东压韩，中路锐卒直切谷心，西侧轻骑再逼魏营外沿。三路兵都不起大声势，只求把韩魏之间那道本就未缝死的接缝狠狠干出来。

天色刚亮，韩营先闻急鼓。

东坡这一次来的不是试探。韩将出营时，又问了一遍魏军动了没有，回报依旧是“尚在整队”。他骂了一声，只得亲自领人去补东坡。

魏营那边也听见了谷中战鼓。

魏将站在营门前，看见韩营求援旗已先连发两次，心里仍还在算秦主力究竟压向何处。等他真正下令推进，韩军在东坡那条线已经被打得摇了。

白起等的正是这一摇。

他立刻命中路锐卒改直切为斜插，直往韩军与魏军本该接住的位置撞去。那一处本是两军并力之隙，此刻却因魏军迟缓成了最薄一层。秦兵一撞进去，韩军阵脚当场裂了半截。

韩将大怒，亲领亲兵补缝，一面又急遣人去催魏军。

魏军终于推到谷中时，迎面看见的已不是并力之机，而是韩军一边退一边骂“魏人误我”。这样的骂声一传过来，魏军前列心里也先凉了半口气。

白起这才下今日最重的一道令：“压谷口，不停。”

秦军于是顺着谷势一层层往前推。

韩军先退，魏军本还想撑，结果一见韩旗后移，前锋也跟着乱。命令、旗号、骂声全搅在一处，比战鼓还乱耳。

苏秦站在不远处高坡，看着早晨还并列在一处的两营旗号，到午后便各自东倒西歪，一时竟连叹气都叹不出来。

门客声音发颤：“相国，可还有法子收？”

苏秦望着谷中尘土，只道：“到这里，不是说话能收的了。”

傍晚时，伊阙胜负已定。

白起没有下令深追，只命各部先收谷口，把韩魏败兵往天下驿道上放出去。夜里咸阳接到捷报，秦王看完后把“韩魏阵乱”“彼此不相应”几句又看了一遍，才命人记下白起之名。

同夜，苏秦回馆，把一卷旧草约摊开到灯下。

那草约中间几处空白，至今还没填上。他提笔，只在纸边写下一句：“兵期不定，阵心先散。”写完便把笔搁下，任由窗外伊阙的风把灯吹得晃了两下。
