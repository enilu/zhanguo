# 第五十四回 伊阙一战白起初惊世 韩魏同营先败不在兵

伊阙山风紧了三日，韩魏两营里的火气也积了三日。

到第四日清晨，秦军终于不再只是小逼试探。

白起命前军自东压韩，又命一支锐卒直切谷中要路，另遣轻骑再逼魏营外沿。三路兵并不起声势，只求把韩魏心里那几道本就没缝死的线，一口气都绷出来。

天色微明时，韩营先闻急鼓。

东坡秦军压得比前几日都硬，阵势也更稳，显然不是来试一试便退。韩将立刻出营，上马时先问的还是一句：“魏军动了没有？”

回报的人低着头：“魏营尚在整队。”

韩将脸色一沉，几乎咬着牙道：“又是整队。”

他不得不先领本部顶上。

同一刻，魏营那边也已听见了谷中战鼓。魏将站在营门前，望见秦轻骑在外沿再起尘头，心里最怕的还是那一件事：秦今日主意，究竟是在真压韩，还是借压韩来引魏尽出？

正这一犹疑间，韩营求援旗已先连发两次。

副将急道：“将军，再不动，韩那边便真要怨死我军了。”

魏将沉着脸：“怨也得看清。”

可战场不给人太久看清。

韩军在东侧先接秦锋，本就地势略高，若魏能及时压上中路，韩军便还可借势守住坡线；偏偏魏军迟了那半拍，韩军便像独自先把全身架出去接了一记重撞。前阵士卒眼见魏旗未至，心里先凉，手上也跟着慢。

白起等的便是这一凉。

他立刻命中路锐卒改直切为斜插，直往韩军与魏军本该接住的位置撞去。那一处原是两军并力之隙，此刻却偏因魏军迟缓，成了最薄的一块。

秦军一撞进去，韩军阵脚当场就乱了半层。

韩将大怒，亲自领亲兵上前补缝，一面又急遣人去催魏：“再不压上，今日两军俱危！”

使者飞马冲到魏营时，魏军其实已在出。

可这一出，仍是晚。

魏军刚推到谷中，便看见韩军那一线已被秦兵斜插成两段。前方韩卒有人回头乱跑，有人边退边骂“魏人误我”，声浪一起来，连魏军前列都跟着先乱了心。

魏将这时才真正明白，自己先前那一点“再看一步”的迟，已不是谨慎，而是把最要命的一线让秦看见了。

他立刻命战车前压，想把中路夺回来。

白起却根本不与他硬撞，只命东侧压韩更紧，中路则专拿弩和短兵去削魏军前锋的气。魏军看着像在推进，实际每进一步，都像踩在一层滑土上，既接不住韩军已裂的那一边，又不敢把自己全压进谷中更深处。

一支并兵最怕的，原不是死伤。

而是前后相看，彼此都觉得：

对面那一路没真替自己尽。

到了午前，这一层终于彻底压垮了两军。

韩军先退。

魏军本还想撑，结果一见韩旗后移，前锋立刻乱了。有人骂韩先跑，有人骂魏不救，命令和骂声搅在一起，竟比秦军战鼓还乱耳。

白起立在高处，见两军心气已散，方下了今日最重的一道令：“全军压谷口，不求多喊，只求不停。”

这道令极冷。

冷在它不许军中先把胜势喊散，只许一层层往前压。

秦军于是像一道黑潮，顺着谷势一层层推下去。韩魏两军本还能各自结一阵自保，可一听见彼此营中都在乱，便谁也不敢真把后背交出去。结果越想自保，退得越快；越退得快，阵脚便越散。

苏秦此时还在不远处高坡。

他眼看着那两营早晨还并列在一处的旗号，到午后便开始各自东倒西歪，心里像被人硬生生压了一块冷铁。

门客在旁声音都发抖：“相国，可还有法子收？”

苏秦望着谷中滚起的尘土，许久才道：“到这里，已经不是谁一句话能收住的了。”

这话出口，他自己都觉得苦。

因为他比谁都清楚，韩魏今日先败的，未必只是兵。

更早败掉的，是那一口本该在最险时彼此替对方先咬住的心。

傍晚时，伊阙战事已分高下。

韩魏败报飞向各国。

秦营中，副将直到这时才敢稍稍松一口气，对白起道：“将军，今日大胜，可要立刻追得更深？”

白起摇头：“先收口，不贪远。”

副将一愣：“为何？”

白起道：“今日赢，不只赢在我军利，也赢在韩魏自乱。这样的胜，最要紧的是让天下都看见，它们是怎么败的。”

副将听完，背后竟又生出一层寒意。

因为他明白，将军要的已不止是这一仗的首尾。

他要的是让这一败，自己带着后劲往天下各国心里去走。

夜里，秦王在咸阳接到伊阙捷报。

他把战报从头看完，目光在“韩魏阵乱”“彼此不相应”“白起乘隙压谷”这几句上停了很久，最后才道：“白起。”

樗里疾在旁答：“在。”

“此人可重用。”

张仪也在。

他听完之后，没有立刻说话，只慢慢把那份战报又接过来看了一遍。

这一回，他心里升起的并不是单纯的喜。

而是一种更沉的确认。

自己这些年拆开的那些东西，到今天终于被另一个人带到了阵上，变成了能见血、能断兵、能压国势的东西。

秦王看了他一眼：“张仪，你在看什么？”

张仪合上战报，道：“臣在看，今日之后，天下再说合纵，不会只想到唇舌，也会先想到伊阙。”

秦王闻言，缓缓点头。

同一夜，赵廷、魏廷、韩廷、楚廷先后得报。

看过的人，反应各不相同。

有的先惊秦之胜，有的先痛韩魏之败，有的先忧本国后路。

可不论谁看，心里都绕不过同一层寒意：

原来如今打一场仗，败的不只是谁兵少、谁将弱。

还会败在众国虽会兵，却已很难真并成一股。

苏秦夜里回到馆中时，衣上还沾着伊阙谷外的尘。

他没有换衣，也没有先坐，只把那张早已卷起的三国草约重新摊开，盯着中间那三处空白看了很久。

门客站在一旁，不敢出声。

过了许久，苏秦才提笔，在空白旁另写了一行小字：

“兵期不定，阵心先散。”

写完之后，他把笔搁下，闭了闭眼。

等再睁开时，他把那卷草约慢慢合上。

窗外夜风挟着伊阙谷口的尘，从窗缝里细细吹进来，灯火一摇再摇。

这一回，他终于听见，远处那批更重的脚步声，已经沿着兵道走到天下前头去了。
