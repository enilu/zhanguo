# 第五十五回 伊阙败报先把赵廷照冷 白起一战得名咸阳却只继续问路

伊阙一败，诸国驿道上跑得最快的不是盟书，是败报。

韩魏两国把损折、散卒、失地一封封送出去，纸上的字还没到齐，赵、楚、齐几处朝堂里已先把“伊阙”二字说得发硬。

赵廷最先变色。

赵王一日连拆三封报，案上除了韩、魏两家败文，还有边吏自伊阙抄来的军情。看完后，他没有先问该不该再议合纵，只问苏秦：“此败最重在何处？”

苏秦答：“往后诸侯再会兵，人人先想的都会是伊阙。”

这话一出，殿里更静。

赵王把战报往案上一压，目光已移去边务条目与兵籍簿书。苏秦见了，也知道此后再说天下，不能只说盟辞了。

咸阳那边却更冷。

秦王召白起再入殿，第一句话不是问功，而是问：“若伊阙再打一回，你仍如此打？”

白起答：“仍如此打。”

“为何？”

“韩魏怕的不是秦兵多一倍，”白起道，“是怕自己先替对方吃最重那一口。病若还在，打法便不必大改。”

秦王听完，又问：“往后诸侯若真学会在阵前先把主客、粮路、退路写死，伊阙还好吃么？”

白起答：“便难吃得多。”

这句说完，殿中无人接话。

因为谁都听得出来，白起在答伊阙，其实已把后面数卷仗都看进去了。秦王盯着他看了许久，末了只道：“回军。寡人还要再看你几仗。”

白起领命退出，脚步仍和来时一样稳。

楚廷看见伊阙败报时，怀王也很久没有说话。屈原立在一旁，低声道：“伊阙败的不只是韩魏。”

怀王抬头看他一眼，自己把后话接了出来：“败的是天下如今都不敢先把背心交给别人。”

说完后，他自己也愣了一下。

夜里，苏秦独坐馆中，把旧草约和新战报一并摊在案上。战报写的是谷口、兵列和阵乱，草约空着的却仍是兵期、应日和罚约。他看了很久，最后只把那张写着“兵期不定，阵心先散”的纸轻轻压在草约正中。
