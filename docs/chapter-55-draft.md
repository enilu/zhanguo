# 第五十五回 伊阙败报诸侯先失色 白起一战咸阳始重名

伊阙战后一月之间，天下驿道上跑得最快的，不是盟书。

是败报。

韩魏两国先后把战后损折、失地、散卒、将校伤亡一一报出，纸上的字还没全送到，各国朝堂里却已先把“伊阙”二字传得发硬。因为谁都知道，这一战要紧的，不只是韩魏吃了一场败，而是韩魏在众目睽睽之下，把“会兵而不成一股”的病，整个打给天下看了。

赵廷最先变色。

赵王这一日召群臣入殿，案上摆的不是一封报，而是三封：韩报、魏报、边吏自伊阙抄来的军情。苏秦亦在殿中，看得比旁人更沉。

有臣先道：“韩魏兵不算少，伊阙地也不算全无可守，竟败成这样，秦军如今恐更难轻视。”

又有人低声接了一句：“怕的还不只是秦军。怕的是往后诸国再会兵，人人都先想到伊阙。”

赵王听到这里，抬头问苏秦：“苏子，你看这败，最重处在何处？”

苏秦答得很慢：“在天下从此更难轻信‘两军会于一处，便能成一股’。”

这话不高，却叫殿中一时都静了。

赵王沉默片刻，忽然看向案边那几份尚未批完的边务条目，缓缓道：“如此说来，赵往后更不能空守名了。”

苏秦没有驳。

因为到这一刻，再去单讲旧盟如何可贵，便像是逼赵拿本国城门去替别国补伊阙那道裂口。可他仍得把另一层话说出来：“赵自然更要重边。只是边若重，而心气不立，将再厚的守具，也只是替将来独受秦锋多垫几块木头。”

赵王点头，却没有就着这句话再往下谈盟，而是转头问兵：“西边如今能用之将，谁最合守？”

这一问，殿上气口又是一变。

因为伊阙之后，各国朝堂里最吃重的问题，正在慢慢从“该如何再说服诸侯”变成“谁能真把兵撑住”。

有人举廉颇。

有人举赵奢。

也有人说，此时不该急于议名将，先当重整兵籍、补城练骑。

赵王听完，没有立刻定，只命人把廉颇、赵奢近来军中簿书一并调来。

苏秦坐在席间，静静看着这场议论，心里第一次极清楚地感觉到，自己已站到了时代边上一寸。不是说辩士自此便全无用，而是往后再要谁在朝堂里一句话压下满殿，前头都得先有人拿兵、粮、城、路把那句话托住。

同一时刻，咸阳宫中却比旁处都更静。

秦王把伊阙捷报来回看了三遍，最后把那份战报放在案上，问樗里疾：“军中近来最常提谁？”

樗里疾答：“白起。”

“朝中呢？”

“也是白起。”

秦王这才笑了一下：“一战得名，倒快。”

张仪在侧，听到这里，目光却没落在战报上，而是落在舆图韩魏之间那一道细长的谷线上。过了一会儿，他才道：“大王，白起这一战打出来的不止是名。”

秦王抬眼：“还有什么？”

张仪答：“还有一种叫诸侯越看越冷的明白。往后再会兵，不只要防秦，还要先防自己会不会重演伊阙。”

秦王听后，没有立刻应声，只命内臣：“召白起。”

白起再入殿时，仍是那副不显山露水的模样。

秦王看着他，第一句话却不是问功，而是问：“伊阙若再打一回，你还会这样打么？”

白起答：“会。”

“为何？”

“韩魏当时最怕的，不是秦兵多一倍，而是自己先替对方吃最重那一口。”白起道，“臣只不过沿着它们最怕的地方去压。若再打一回，它们只要病还在，打法便不必大变。”

秦王又问：“那你眼里，伊阙后面该怕什么？”

白起答：“诸侯若从此学会，至少在阵上先把主客、粮路、退路、先后写死，伊阙这样的隙就不会再这样好吃。”

秦王盯着他看了许久，终于道：“白起，寡人今后不只让你看一路。你且先回军，寡人还要再看你几仗。”

白起伏地受命，没有半句自夸。

他退下后，樗里疾低声道：“大王，此人名起得虽快，性子却冷得像块铁。”

秦王道：“军中有时就要这等铁。”

张仪听完，没有说话。

因为他心里也已明白，自己这几年替秦国做的事，到今天算是真交给另一只手了。那只手比自己更少言，也更少让人看出喜怒；可一旦握住兵，天下后面许多话，便会被它逼得再改一层意思。

同样的败报到了楚廷，怀王看完之后，脸色极难看。

韩魏败，他并非不惊；可更叫他胸口发堵的，是自己竟从这场败里看见了一面镜子。伊阙里韩将与魏将互疑互怨、谁也不肯先吞最重那口气，这种败法，跟自己这些年在齐秦之间来回算路，又有几分不是同一层病？

屈原在旁见怀王久久不言，便只低声道：“大王，伊阙之败，败的不只是韩魏。”

怀王抬头看他。

屈原没把后半句说满。

因为他知道，再往下说，便要说到楚了。

怀王却自己把话接了下去：“败的是天下如今都不肯先把背心交给别人。”

说完之后，他竟自己愣了一愣。

殿中众臣谁也不敢接。

因为谁都听得出来，这话虽是看韩魏，其实也已照到了楚自己身上。

夜里，苏秦独坐馆中，把伊阙前后的几份旧草约和新战报全摊开在案上。战报写的是地、是兵、是谷口；草约空的是兵期、是应日、是罚约。他盯着它们看了很久，最后没有写长书，只在纸边添了一句：

“伊阙之后，天下先怕的不再只是秦强，且怕会兵而不成兵。”

写完之后，他把纸轻轻压在那张旧草约上。

像把一场战的后果，压回到那些战前便已空着、却一直没人肯填上的空白里。
