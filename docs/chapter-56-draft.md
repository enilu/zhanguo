# 第五十六回 赵王召廉颇先问如何不乱 边营夜里不追百骑只把烽候问到底

伊阙之后，赵廷议兵的次数一下多了。

殿上仍谈合纵，可每说几句，话头就会拐去西边：哪段城该补，哪支骑该添，谁领边军最稳。赵王连着数日看边图、兵籍和粮簿，最后案上只剩两个名字。

廉颇。

赵奢。

赵奢长于赋与军需，算事极细；廉颇则久在边军，压得住老卒宿将。赵王终于召廉颇入殿，开口便问：“赵眼下最先该做什么？”

廉颇先看了一眼案上的边图，才道：“先叫边军不乱。”

“如何叫它不乱？”

“整军令，定粮道，实巡骑，明赏罚。”廉颇答，“秦若来试，边上自己先浮，便不用等它真大举。边上先稳住，再谈压人。”

赵王听完，当即把西边诸营交给他整束。

廉颇到营第一天，不问远谋，也不问秦军多寡，只查三件事：粮、骑、夜巡。粮册少一石便问到执簿小吏，夜巡迟一刻便把当值烽候叫到跟前，骑棚里少几匹马，他亲自站在棚下数到最后一匹。

营中老卒起初都嫌新将太硬，既不先赏，也不先抚，倒像个查账的吏。

不到十日，营里气却先整了。

一夜，西边有秦骑百余来试。

旧例多半是鸣鼓聚众、追出去一阵，好叫对面看见赵也不弱。廉颇却只令前营不动，中营增炬，后营骑探出而不追。等秦骑退远了，他才把当夜巡骑与烽候全叫来，一个个问到末。

副将不解：“将军，百骑来试，我军不追，不怕边上笑赵怯？”

廉颇看了他一眼：“它百骑来试，你举营去追，谁怯？”

副将一愣。

廉颇又道：“它来，不是来送你立名，是来看看你乱不乱。你若一追，把自己营里最怕什么全露给它看了，才叫真亏。”

这事很快传回邯郸。

赵王听完，只说了一句：“廉颇把军当军，不当场面。”

同夜，苏秦出宫时正遇见赵奢自外入朝。赵奢刚从军需署来，袖口还沾着一点仓灰。苏秦问他去了何处，赵奢答得也短：“看西边添骑换粮，顺便重算了两条运道。”

苏秦点点头，没有多问。

因为他忽然觉得，赵这一卷真正能立住的，不只在谁善战，也在有人肯把战前这些最枯重的东西一件件钉实。

夜深时，廉颇站在边营土垒上望向西面。

远处秦境火色很淡，风却很硬，把赵旗绷得一丝不弯。副将立在他身后，低声问：“将军，秦迟早要大逼？”

廉颇道：“会。”

“那我们如今够不够？”

廉颇没有立刻答，只望着垒外黑夜，过了一会儿才道：“兵够不够，不看账上多几百少几百。看来时乱不乱，守时稳不稳，退时散不散。把这三样先练出来，秦就知道赵不是伊阙里的韩魏。”

说完，他仍站在原处，没有动。

风一直从西面撞过来，把旗面吹得更紧了。
