# 第五十六回 赵廷选将廉颇先受任 西边未交锋军气已成形

伊阙之后，赵廷议兵的次数一下子多了起来。

从前议的是合纵怎么维住，如今仍议这件事，却总是议着议着，案上的话就会拐到另一处去：西边哪段城该补，哪支骑该添，谁领边军更稳。

赵王连着数日召见军吏，把西陲几段防线来回问了个遍，最后案上只剩两个人名：

廉颇，

赵奢。

赵奢当时多在国中，善理赋与军需，名虽在，却更像能把军后那些硬事一件件算明白的人；廉颇则久在边军，兵里出身，性子硬，带兵也更整。

赵王看着这两个名字，许久之后，抬头问苏秦：“苏子若论西守，用谁？”

苏秦答得很谨慎：“若论眼下先把边上一口气撑住，廉颇更合。”

赵王点了点头，又看向几名老将。

其中一人道：“廉颇持军严，肯吃苦，也压得住边军那些宿将老卒。只是此人不喜虚声，真用了，怕未必合诸公口中‘立时扬名震敌’的期待。”

赵王听完，只问一句：“赵如今还该先求虚声么？”

殿中便没人再说话。

数日之后，廉颇入见。

他入殿的样子，比白起更像个军中人。身高肩阔，步子却并不猛，到了殿前也无多余姿态，只俯身行礼，一举一动都透着一种久在军中养出来的沉。

赵王先看了他一阵，才道：“廉颇，寡人今欲重整西边。你看，赵眼下最先该做的是什么？”

廉颇没有立刻答。

他先看了一眼案上的边图，随后才道：“先把军中那口气养回来。”

赵王道：“何为养回来？”

廉颇答：“伊阙之后，诸国都在怕会兵而不成兵。赵若此时边上也浮、军中也急，秦不必真大举来逼，只多试几回，军里自己便会先乱。故臣请先整军令、定粮道、实巡骑、明赏罚。边上先求不乱，再求压人。”

这话听着并不奇。

可赵王听完，却慢慢点了点头。

因为这正是他近来最想听见，又最难从旁人口中听得这样稳的一种话。

他问廉颇：“若秦来挑，你是先战，还是先守？”

廉颇答：“可战时战，不可战时守。能守住而叫敌不得便，是胜；为争一时气而空自出，是败。”

这句话一出口，苏秦心里便微微一动。

因为他听得出来，这不是一句怯战之词，而是一种与前些年辩士气口全然不同的硬。

不是把人往高处拽的硬。

是把军往地上扎的硬。

赵王当即下令，命廉颇往西边总领边军整束。

任命一下，边上气息也跟着变了。

廉颇到营第一天，先不议远谋，也不问秦军多寡，只做三件事：

查粮，

查骑，

查夜巡。

粮册里少一石，他便问是谁记的；

夜巡里迟一刻，他便问谁当值；

骑军里空出多少匹马，他便站在棚下数到最后一匹。

许多老卒最初还嫌这位新将来得太硬，既不先赏，也不先安抚，反倒像个来查账的吏。

可不到十日，营中气便先整起来了。

因为人人都看得出，这位将军不爱说空话，也不肯先拿豪语压人。他只把该补的补上，该堵的堵住，该罚的罚下去。许多往日靠资历混得过去的窟窿，到了他手里，竟一下都混不过去。

一次夜里，西边有秦骑来试。

不过百余骑，远远掠了一阵，想看赵营反应。旧例多半是鸣鼓聚众、追出一阵，叫彼此都先看个声势。廉颇却只令前营不动，中营增炬，后营骑探出而不追，待秦骑退远了，才把当夜巡骑与烽候一一叫来，当场问责。

副将不解：“将军，秦骑夜试而来，我军不追，不怕边上说赵怯？”

廉颇看了他一眼：“百骑来试，你便举营去追，谁怯？”

副将一愣。

廉颇又道：“它来，是看我乱不乱，不是送你立名。你一追，营中后线空，前后号令也乱，它今日百骑试你，明日便知你最怕什么。守得住阵，不中它试，才叫它白来。”

副将这才低头受教。

这件事很快传回邯郸。

赵王听完之后，只淡淡说了一句：“廉颇倒真把军当军，不当场面。”

苏秦坐在侧席，听到这话，心里竟比前几日稍稍安了半分。

不是因为赵就此可高枕。

而是因为在这个越来越多国家都先求自保、却又未必真懂怎么守得住的年代，赵边上终于多了一名不肯替自己争虚声、只肯替兵争实地的人。

夜里，苏秦走出宫门时，正碰见赵奢自外入朝。

两人在宫门阴影里对了一礼。

苏秦看着这位神色沉稳、衣冠整肃的赵臣，忽然生出一丝说不清的感触。赵奢此时还未在天下兵家中大显名声，可他眼里那股算事算到极细的劲，已与朝堂里许多只会空争的人全不相同。

苏秦便问：“君方自何处来？”

赵奢答：“自军需署来。西边骑军添换，粮道也要重理。”

苏秦点了点头，没有再多问。

因为他忽然觉得，赵这一国如今最可依之处，也许不只在谁善战，还在有人肯把战前那些最枯、最重、最不讨彩头的东西一件件稳住。

同一夜，廉颇站在边营垒上，望着远处秦境火色，一动不动。

副将走到他身后，道：“将军，秦近来试得勤，是否迟早要大逼？”

廉颇道：“会。”

“那我们如今够不够？”

廉颇没有立刻答。

过了片刻，才道：“兵这东西，够不够，不在账上多几百少几百。要看来时乱不乱，守时稳不稳，退时散不散。先把这三样练出来，秦便知道赵不是伊阙里的韩魏。”

这话说完，垒外夜风正从西面一阵阵吹来，吹得赵旗紧紧绷着。

廉颇抬头看了那旗一眼，仍旧没有动。

像一块先把自己钉稳了，再等大风真正来撞的石。
