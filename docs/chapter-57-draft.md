# 第五十七回 燕王压恨多年终问齐 乐毅入朝先把国势明

赵国在西边理兵的时候，东北的燕国也开始一寸寸收紧了。

只是赵之紧，紧在边上，看得见；

燕之紧，却更像宫里一口多年不曾真放下的气，平日听着不响，到了夜深时，方知它一直压在那里。

燕昭王这一日只召郭隗与数名近臣入小朝，殿上没有铺边报，只铺了一幅齐燕形势图。图上从燕南到济水，从临淄到即墨，连可通车几乘、可宿军几日的旧道都记得极细。

燕昭王看图许久，先问的却不是齐，而是燕。

“朕这些年养士、修政、恤民、实仓，所为者何？”

殿中无人敢轻答。

郭隗沉了一沉，才道：“为不再任人视燕为可欺之国。”

燕昭王点头，又问：“若只是不被人欺，够不够？”

这一下，连郭隗也没有立刻接。

因为人人都听得出来，大王今日问的，已不是守，而是更远一层的事。

终于，燕昭王自己把话点破：

“齐当年压我燕，非止一时兵锋之辱。朕这些年忍着，不是忘了，是要看燕哪一日能把这口旧气，从恨熬成国事。”

一名老臣忙道：“大王，齐终究强国。燕若妄动，只怕旧辱未雪，又添新祸。”

另一人却道：“齐如今四面用力，处处自矜，未必不是可乘之时。”

燕昭王听了半晌，不问谁主战、谁主忍，只问了一句：

“若燕真要动齐，谁能统这一仗？”

这一下，殿中更静。

因为这句问得太实。它说明燕昭王今日不是来借旧恨壮气，而是当真往“如何做成”上去了。

郭隗抬起头，道：“臣举一人。此人不喜空言，却看得清大局。若燕将来真要借天下怨齐之势起兵，他最合。”

燕昭王道：“谁？”

“乐毅。”

数日之后，乐毅入燕。

他到殿时衣冠整肃，神色平静，既无边臣初入中枢时的拘谨，也无游士逢时便要立刻显手段的锋外之气。燕昭王看了他很久，先问的不是“可伐齐否”，而是：

“燕这些年所积，你看够不够上阵？”

乐毅没有立刻答。他先走到图前，看了看燕南诸道，又看了看仓籍和兵册，才缓缓道：

“若只求守燕，已可自立。若要求伐齐，不够。”

燕昭王眉梢一动：“差在何处？”

乐毅道：“差在久与整。”

“说下去。”

“兵要久练，不是一时聚众；粮要整入，不是一时征发；将要可久任，不可今日一令、明日一改；南下诸道要先认死，不可临兵才算远近。”乐毅说着，把手落在图上一段自燕都南出的细线上，“燕若要动齐，不是先问恨有多深，而是先问自己能不能把一场大仗，从起兵、转粮、借道、会师，一口气撑到底。”

这番话一落，殿中不少人都听出分量来了。

乐毅不是来替燕昭王喊“齐可伐”的。

他是先把“燕是否真有资格去伐齐”这一层拆开来摆在案上。

燕昭王又问：“那齐呢？朕若只理燕自己，齐便由着它去盛？”

乐毅道：“也要看齐。只是看齐，不是去等它哪一日忽然虚弱，而是看它哪一日会强得太满。”

“何谓太满？”

乐毅答：“强国最怕的，不是人知其强，而是自己也开始觉得天下处处都可轻算。如今秦在西压，赵在自强，楚、魏、韩各有病，齐若仍自恃山东最富、最稳、最会算，便容易一处处去试人。试得越多，怨便积得越深。到那时，燕未必要先做最强的一国，却可以先做把诸国旧怨穿成一路的人。”

郭隗在旁低低应了一声。

因为这正是燕昭王这些年一直在等的一层：

不是等齐衰，

而是等齐太盛。

燕昭王望着乐毅，忽然笑了一下：“你不劝朕今日便起兵。”

乐毅道：“刀若未磨利，先出鞘，只会叫旧恨再添一层新败。”

燕昭王点头，道：“好。那便先把燕养成能久压一战的样子。”

说完，命人再把另一卷册子铺开。

那上头记的，是燕兵、燕仓、燕南道里与借赵诸路的缓急。

乐毅这一次看得更慢。他先问燕南可出兵多少，又问海道转粮几日可及，再问若借赵道，需提前多久定；若不借，燕军自走本境外道，哪几段最易拖死后粮。

几名旧臣起初只觉得他问得太碎。可问到后头，众人才慢慢听懂：

他不是在看一场仗怎么打。

他是在看，燕这口多年压着的恨，究竟能不能真被做成一场不靠血气、而靠国力与次第撑住的大举。

当夜散朝时，燕昭王没有立刻回寝宫，只在殿中又看了许久那幅齐燕图。

郭隗低声道：“大王今日像是宽了一层。”

燕昭王道：“朕等的，不只是会说齐可伐的人。朕等的是一个知道何时可伐、如何可伐、伐到何处不可贪的人。”

殿外北风一阵阵压来，把图角轻轻掀起。

燕昭王伸手按住。

那一下很轻。

却像按住了一张已经绷满，却还不肯轻发的弓。
