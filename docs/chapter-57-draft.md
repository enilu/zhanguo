# 第五十七回 燕王蓄锐多年终待时 乐毅入朝先论伐齐根

就在赵国忙着整西边时，东北的燕国也开始动得更密了。

只是燕国动得不像赵那样明。

赵是在边上添骑、补仓、练夜巡，人人看得见；燕却更像一张久压在匣底的弓，外面看着仍旧安静，里面的弦却一日日绷得更紧。

燕昭王这些年招士养客、恤民修政，做的事一直不响。天下诸侯提起燕，多半仍先想到它偏远、先想到它国力不如齐楚赵秦。可燕昭王自己心里清楚，燕最不能忘的，不是自己弱，而是齐当年那一场几乎压到燕国头上的旧辱。

这一日，燕宫设小朝，只留寥寥数臣。

案上摆的不是边报，而是一幅齐燕形势图。图上临淄、即墨、济水沿线都被细细标了红记，连某些旧道能通几乘车，都写得极详。

燕昭王看了那图很久，才对座下诸臣道：“燕这些年养士、修政，不是为叫天下夸朕勤俭。”

群臣低头不语。

因为谁都听得出来，大王今日要说的，是更久压在心底的事。

终于，一名老臣开口：“大王是要问齐？”

燕昭王点头：“问齐，也问燕自己。若今日天下已越来越乱，齐国又自恃强盛、四面树怨，燕是不是还要像旧年那样，只把恨压在袖里等死？”

这话一出，殿中气氛顿时就变了。

有臣主张再忍，说燕终究地狭人少，纵有旧怨，也不宜先挑大国；也有人道齐如今虽强，实则内外皆骄，正是多年未必再有的可乘之隙。

燕昭王听了半晌，只问一句：“若燕真动，谁可为将？”

这一问，比前头一切争论都更实。

因为它说明，今日这场小朝已不再只是泄恨，而是在往“如何真做”上走。

殿中一时没人敢先答。

不是无人可举。

而是此事太重，谁也不愿轻轻把一个名字抛出去。

最后，太傅郭隗抬头，道：“臣举一人。此人未必最会逢迎君上，却最知大局。若燕后面真要借乱势而动齐，他比旁人更合。”

燕昭王看向他：“谁？”

郭隗道：“乐毅。”

这名字一出，殿中几人神色各异。

有人点头。

也有人微皱眉。

因为乐毅虽早有贤名，终究不是燕本土旧贵那一路人物。他学识深，眼界广，待人也稳，可正因太稳，反叫一些只想立刻听见豪语的人嫌他不够快。

燕昭王却没有犹豫太久，只道：“召。”

数日后，乐毅入朝。

他到得并不张扬，衣冠整肃，神情也平，入殿后行礼毕，便立在阶下，不见半点急于自显。

燕昭王看了他一会儿，先问的却不是将兵，而是国事：“燕这些年修政养士，你以为所为者何？”

乐毅答：“为不再任人看作可欺之国。”

燕昭王又问：“那若只如此，够不够？”

乐毅抬眼：“不够。”

“为何？”

“可欺之国若只求不被欺，终究还在别人掌中。”乐毅道，“燕若真想翻旧势，后面便不能只守自己门内那一点安稳，还得等天下失序之时，择其一国之骄而重击之。”

殿中几人听到这里，气都微微一提。

因为这话已不是自守。

是图胜。

燕昭王却没有立刻露喜，只追问一句：“那你看，这‘一国之骄’，是指谁？”

乐毅答得极稳：“齐。”

这一字落下，殿里更静。

燕昭王缓缓坐直身子：“你何以如此看？”

乐毅上前半步，道：“齐国富强不假，兵也不弱。可富强最怕养出一个‘天下皆可轻’的心。今诸侯彼此失信，楚、赵、魏、韩都先忙自护，秦又一日日把兵往前推，齐便更易觉得自己居山东之上，可以处处算得更精、压得更稳。这样的国，表面最盛，底下也最容易积怨。”

郭隗在旁听得暗暗点头。

因为这正是燕这些年一直在等、却没人敢说得这样平而实的一层。

燕昭王又问：“若真伐齐，燕一国能为之？”

乐毅答：“不能。”

“那你方才所言，岂不成空？”

乐毅摇头：“燕不能独吞齐，却可以做那根先把众怨穿起来的针。今诸侯虽各自有病，怨齐者却未必少。若大王能先把燕自己国中政、兵、粮、将都理顺，再等齐骄势更满，后面未必不能合数国之力，一举压之。”

这话与第二卷的纵横又不同。

张仪是拆。

乐毅眼下说的，却是另一种聚。

只是这聚，不再是先拿大义压天下，而是拿众国早已有的怨与利，一根根去穿。

燕昭王听完之后，久久没有作声。

过了许久，才道：“你不劝朕立刻动手。”

乐毅答：“刀磨得不够利，先出鞘，只会叫旧恨再添新败。”

燕昭王听后，终于露出一点极淡的笑：“好。朕要的，也不是一个会在殿上喊杀的人。”

说完，他命左右展开另一卷册子。

那上头记的，正是燕这些年积下的兵籍、粮仓、将吏与道里。

乐毅低头看去，一页页翻得极慢。

他看得并不浮，只在几处停得久些：一处是燕南可出兵多少，一处是沿海转粮可及何地，一处是与赵通道若借、若不借，各差几日。

燕昭王看着他的手指在纸上移动，忽然觉得自己这些年压在心底的那点恨与耐，今日终于碰上了一个不只懂恨、更懂怎么把恨做成事的人。

夜里，乐毅退下之后，燕昭王独坐殿中，看着那幅齐燕形势图许久未动。

郭隗在侧，低声道：“大王今日似是宽了一口气。”

燕昭王没有否认，只道：“朕等的，不只是会说齐可伐之人。朕等的是一个能把‘何时可伐、如何可伐、伐到何处不可贪’都说得出的人。”

郭隗点头：“乐毅正是此类人。”

殿外夜风自北而来，吹得图角微微一翘。

燕昭王伸手将它按住。

那姿势极轻。

却像按住了一支已养了多年、如今终于开始听见弦响的箭。
