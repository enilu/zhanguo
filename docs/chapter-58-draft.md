# 第五十八回 乐毅不急谈兵先论国 燕廷定策只待齐心骄

乐毅入燕之后，燕国朝堂里最先变的，不是兵令。

是说话的次序。

从前诸臣谈齐，先说的是旧恨，次说的是可乘之隙，第三句才勉强落到兵、粮、路、将。可乐毅一入朝，头一件事便把这顺序反了过来。

这一日小朝，燕昭王仍命只留寥寥数臣。

燕昭王问：“若后面终有一战，燕先当做什么？”

有臣抢先道：“先结赵。”

有人道：“先联韩魏，使齐四顾受疑。”

也有人道：“先扬燕怨，激诸侯同恨。”

乐毅却道：“先看燕自己。”

殿中微微一静。

燕昭王看向他：“何解？”

乐毅答：“燕若国中不稳、兵粮不继、道路不明，今日就算赵肯借力、魏肯附议，也只是借来一阵风，吹过去便散。故臣请大王先做四件事：定兵籍，实粮仓，明南出诸道，择可久任之将。”

燕昭王点头：“这些朕都在做。”

乐毅道：“在做，和做到可用，是两回事。”

这句话说得很平，却把殿中几名旧臣都听得神色一凝。

因为它不是在说方向错，而是在说如今燕所缺的，正是把这些方向真拧成一股能上阵、能久撑、能不乱的硬力。

燕昭王并未动怒，只问：“那依你，燕离‘可用’还差什么？”

乐毅上前半步，道：“差两个字：久与整。”

“说。”

“兵要久练，不是一时聚众；粮要整入，不是一时征发；将要可久任，不可今日一令、明日一改。若后面真要压齐，燕先得把自己练成一支不怕久拖的国。”

这话一出，郭隗在旁也轻轻点头。

因为他听得明白，乐毅不是要把燕变成一个只会死守的国，而是先要把这张弓拉得真有筋力，再等放箭。

燕昭王又问：“那齐呢？燕既先理自身，齐那边便只看着？”

乐毅道：“看，也要做。”

“做什么？”

“做两层。”乐毅答，“一层在外，慢慢识齐所怨；一层在内，慢慢养燕所能。外头不必先声张，内里更不可先急。齐这几年最坏处，不只是强，而是强久了之后，容易把天下都看成可算之物。这样的国，最怕的是有人比它更肯久。”

燕昭王听到这里，忽然笑了笑：“你不急着叫朕报旧耻。”

乐毅答：“臣若只会劝大王报旧耻，今日便不会站在这里。”

殿中有臣忍不住道：“乐君说得固然稳，可若一直只做不发，岂不反叫齐越发坐大？”

乐毅转头看向那人：“齐今日坐大，最怕什么？”

那臣一时竟答不上来。

乐毅便自己接道：“最怕有人一边不惊动它，一边却把能动它的几样东西都先备足。齐若看见燕今日大张旗鼓结怨聚兵，自会先防；燕若安安静静地把该理的理顺，把该认的路认明，把可借之国一处处摸清，后面真正动手时，反倒更重。”

燕昭王沉默片刻，忽然命左右取来一卷燕南出兵道图。

图上自燕都南下，分陆道、海道、借赵道三线，连宿营可驻几日都记得极细。乐毅接过图来，先不看齐城，而是先把燕境内那几段最难转粮的地方一一问了清楚，又问赵道若借，需提前多久定；若不借，燕自走本境外道要多费多少日。

几名旧臣在旁听着，起初还觉得这位新来的乐君问得太细、太慢。可问到后头，众人也都慢慢听出味来：

他不是在看一仗怎么打。

是在看燕够不够资格去打一场大仗。

燕昭王听完之后，忽然道：“乐毅，朕今日更明白了。天下如今多半都在先讲自保，你却先教朕把国养成能久压的样子。如此一来，燕后面若动，便不是一时血气，而是真要去改一段局。”

乐毅低头：“臣所求，正是如此。”

与此同时，齐国那边却正好走到另一头。

齐湣王这一年气势更盛，临淄朝会也比往日更躁。楚、韩、魏、燕诸事一并来时，齐廷诸臣争的，多半不是齐该不该稍敛，而是如何在乱局中多取一分、多压一步。田文虽仍能劝住一部分，齐王却已越来越爱听那种“齐国地广兵强，诸侯终归还要看齐”的话。

消息传到燕廷，燕昭王只把那份密报看了两遍，便放到乐毅面前。

乐毅看完之后，没有立刻说齐可伐，只道：“齐心正往满处走。”

燕昭王抬眼：“这便是你要等的？”

乐毅点头：“国之将危，未必都先见其弱。许多时候，先见的是它越来越觉得自己处处都能算赢。”

这句话，叫殿中许多人都静了下来。

因为他们忽然明白，乐毅等的并不是齐某一日忽然兵弱粮空，而是等它那口太满的气，满到足以叫旁国都越来越容不下。

夜里散朝之后，郭隗陪燕昭王行至殿外。

宫墙外北风很硬，吹得灯火都斜。

燕昭王望着远处黑沉沉的夜色，低声道：“朕这些年养士修政，今日才真觉得，是在等对的人来把这些事拧成一路。”

郭隗道：“乐毅不喜虚名，却最能把虚的东西压成实的。”

燕昭王点了点头，没有再说话。

他心里已很清楚，燕后面要走的，不会是张仪那样一舌翻局的路，也不会是楚怀王那样多留一路的路。

它要走的，是另一条更慢、更稳、也更能久忍的路。

这一夜，乐毅回到馆中，亲手把案上三卷册子分开放好：

一卷记燕兵；

一卷记燕粮；

一卷记齐怨。

他把最后那卷压在最上。

却先翻开的，仍是第一卷。

窗外风过，灯影在册页上轻轻一动。

乐毅低头看着那些密密写着路、仓、兵数的小字，久久没有抬头。

因为他知道，后面若真要取齐，最先要赢的，仍不是齐。

而是先把燕自己，练成一把真能久握不折的刀。
