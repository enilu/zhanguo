# 第五十八回 乐毅定策先收兵粮路 燕廷观齐只待那口气满

乐毅入燕之后，燕廷里最先变的，不是兵令。

是说话的次序。

从前诸臣谈齐，多半先说旧恨，次说可乘之隙，第三句才勉强落到兵、粮、路、将。乐毅一到，却先把这顺序颠了过来。

这一日小朝，燕昭王仍只留寥寥数臣，开口便问：

“若后面终有一战，燕先当做什么？”

有人抢道：“先结赵。”

有人道：“先联韩魏。”

也有人道：“先扬燕怨，叫山东共恨齐。”

乐毅却道：“先把燕自己练到可用。”

殿中微微一静。

燕昭王看向他：“何谓可用？”

乐毅答：“兵籍要定，粮仓要实，南出诸道要先认清，将位要先坐稳。今日借得一阵外风，不算本事；后日自己能把一场大仗从起兵撑到收口，才算可用。”

燕昭王道：“这些年，朕一直在做。”

乐毅道：“在做，和做到足以用兵，是两回事。”

这句话说得很平，几名旧臣却都听得神色一紧。

因为他不是在说燕方向错了，而是在说如今燕欠的，是把这些零零散散的积累真正拧成一口能南下、能久持、能不乱的气。

燕昭王并不恼，只命人把燕南道图、仓册和兵籍全都取上来。

乐毅接过图册，先不看临淄，只先问燕南三道：

哪一道最稳，

哪一道最短，

哪一道若借赵可省几日，

哪一道若自走本境，最怕哪里断粮。

又问今岁新收粟有多少入常仓，

沿海道转粮可到何处，

边将中谁可久任、谁只堪一时锐进。

几名旧臣起初听得不耐，只觉得他问得太细、太慢。可问到后来，连他们也渐渐听出味来：

乐毅不是在问一仗怎么打。

是在问燕够不够资格去打一场会改天下格局的大仗。

燕昭王看着他一条条问下去，忽然道：“你还是没说齐。”

乐毅这才把手从燕境内移到更南那片齐地上，平静道：“齐不必急说。齐如今最坏处，不在它强，而在它越来越觉得自己处处都能算赢。这样的国，最怕别人一边不惊动它，一边把能动它的东西先都备足。”

殿中有人忍不住道：“若一直只备不发，岂不反叫齐越发坐大？”

乐毅转头问他：“齐今日最怕什么？”

那人一时答不上。

乐毅便自己接道：“怕的不是旁国明着骂它，怕的是旁国都开始在心里记它，记到最后，谁也不愿再让它往前多走一步。”

说完，他指了指案上一封新到的密报。

那报中写的，是齐湣王近日又在宋地与山东诸侯之间频频用力，临淄朝议中人人都在算如何再多压一步，几乎已少有人愿讲“该收一收了”。

乐毅看完，只说了一句：

“齐心正往满处走。”

燕昭王抬眼：“这便是你要等的？”

乐毅点头：“国之将危，未必先见其弱。多半是先见它越来越以为自己处处都能占先。”

郭隗在侧，低声道：“待它满到旁国都容不下时，燕便可为那一根穿众怨的针。”

燕昭王听完，没有立刻接话，只慢慢把那几卷册子重新压齐。

过了片刻，他才道：“如此说来，燕后面要走的，不是求快之路。”

乐毅答：“是求久之路。”

“也不是一时血气之路。”

“是先把兵、粮、路、将、怨与时，都理成一路，再动的路。”

殿中一时很静。

因为人人都明白，燕到这一步，已不再只是想着“有朝一日报旧耻”。它要做的，是把多年忍下来的那口气，慢慢养成一场真能压齐的大举。

夜里，乐毅回到馆中，把案上册子分开摆好：

一卷记燕兵，

一卷记燕粮，

一卷记南道，

最后一卷，才记齐这些年一处处积下来的怨。

他把“齐怨”压在最上，先翻开的却仍是“燕兵”。

灯影轻轻动了一下。

乐毅低头看着那些密密麻麻的小字，许久没有抬头。

因为他知道，后面若真要取齐，最先要赢的，从来不是齐。

而是先把燕自己，练成一把真能久握不折的刀。
