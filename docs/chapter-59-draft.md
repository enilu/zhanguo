# 第五十九回 秦廷乘胜不肯先贪远 赵国理兵赵奢始出锋

伊阙大捷之后，咸阳宫中喜气并没有闹得太满。

秦王连着几日召将议事，问得最多的不是“可趁胜夺几城”，而是“韩魏如今各自还剩几口气，哪一路最先会乱，哪一路反会因大败而死守”。

白起再入殿时，案上已不见前几日那些庆功酒器，只剩韩、魏新报与边图。

秦王问他：“伊阙之后，寡人若要你立刻再压韩魏，你请先取什么？”

白起答：“先取它们喘不过气的地方，不先取看着最肥的地方。”

秦王道：“说清。”

白起走到图前，把手落在韩境几段连着伊阙败军回路的要地上：“韩军伤得更急，魏军疑得更深。此时若贪远，诸侯反会因惊而暂并；若只一段段压它们刚露出的软处，它们便要一面补败，一面防彼此，一口气反更顺不过来。”

樗里疾在旁听得点头。

秦王又问：“你不求一口再立大功？”

白起答：“臣若只求立功，便会替诸侯省去它们自己犯错的工夫。”

殿中一时极静。

连张仪都抬眼多看了白起一下。

因为这话说得冷，却极准。伊阙之后，秦最可怕的并不是会不会趁势猛扑，而是它竟肯压住贪心，专等各国在败后自己露出第二层病。

秦王沉吟片刻，终于道：“准。你去，不必求声势，只求把这一战后劲，往韩魏骨里再推深一寸。”

白起受命而退。

张仪望着他背影，忽然道：“大王，白起之兵，和臣之舌，后面怕是要互相接着走了。”

秦王淡淡道：“你拆出路来，他踩进去。寡人正要如此。”

同一时刻，邯郸朝中也正因为伊阙之后的兵事忙得更紧。

廉颇已赴西边，赵王便把国中另一摊更枯、更容易得罪人的硬事，一并压到兵赋与军需上。

案上堆着各郡军簿、赋册、转粮路程与马政旧案，看得人头都发胀。几名旧臣争了半日，争的仍是旧话：今年边费已重，若再添军实，国中豪右必怨；若缓一缓，又怕边上赶不及。

赵王听到后头，脸色已沉。

他忽然问：“赵奢在不在？”

赵奢早候在殿外，闻召入内，行礼毕，站得极直。

赵王把一卷赋册抛到案前：“你近来在军需署看得多。寡人只问你一句，赵若要真把西边撑住，最怕的不是秦，先怕什么？”

赵奢几乎没有迟疑：“先怕国中有人嘴上说重兵，手里却不肯把该出的粮、马、税老老实实交出来。”

这话一出，几名大臣神色都变了。

因为赵奢这一句，已经不是在谈虚势，而是把刀口直直顶到了豪强、贵近与那些最会在国难时替自己找缓词的人身上。

赵王却听得精神一振：“你继续说。”

赵奢道：“廉颇在边上练的是兵。兵若要立住，后头便得有人把粮、马、甲、道路一件件钉死。国中若仍是谁都能欠粮、拖税、占马、挪役，边上就算有好将，也不过替这些空账去硬撑。”

一名老臣皱眉道：“赵奢，此言未免太峻。国中大族也并非不知国急，只是缓急有序，不可逼得太狠。”

赵奢转头看他，声音并不高：“边军若缺粮，秦骑会不会也讲缓急有序？”

那老臣一时竟被顶得无话。

赵王盯着赵奢看了许久，缓缓道：“那依你，该怎么办？”

赵奢道：“立一条死令。凡军赋军需，先从最会拖的地方查起；凡占官田、匿军马、借故缓输者，不论贵近，一体按军法旁比问治。先把国中这股‘都知该做、却总盼别人先做’的病压下去。”

苏秦坐在侧席，听到这里，心里忽然一动。

因为他听得明白，赵奢这条路，与自己从前在诸国说的“先共天下之患”并不一样。赵奢不是去求人人先想远处，而是先把一国之内那些最硬、最不讨喜、却也最不能再拖的东西逼实。

赵王没有立刻下令，只命赵奢把近三月拖欠军赋的郡县、贵族、署名人等，全部列出。

赵奢俯身受命，退下时步子仍稳，像刚接的不是一件最易得罪人的差事，而只是该有人去做的一道账。

数日之后，邯郸城中果然起了波澜。

赵奢查得极硬。

某贵戚家借修园之名扣下军马，他便亲自带吏入门验数；

某大族拖欠军赋，先让门客四出说情，他一句话不听，只把旧簿新册并列，当堂核问；

有郡守自恃资历，口称“民困不可急征”，暗地里却先保自家田租，赵奢当即把他名字记上，连夜上呈。

邯郸诸公一时间私下都在骂这位赵氏新臣手太黑，不给留面子。

可骂归骂，边上粮车却真的一批批动了起来。

夜里，苏秦出宫，恰见一列运甲车自军需署外缓缓驶过，车轮压着石道，声音沉而稳。

门客低声道：“相国，这赵奢倒像比廉颇还难惹。”

苏秦望着那些车，过了一会儿才道：“廉颇是在边上挡人。赵奢是在国内逼人。如今赵若想不做第二个韩魏，这两样人，恐怕都不能少。”

风从邯郸城头吹过，卷得旌旗轻响。

城外西路上，正有新一批粮车往边地去；

城内深门里，也正有更多人在算自己还能不能像从前那样拖过去。

赵奢站在军需署灯下，把最后一卷核过的赋册合上，没有抬头。

案上灯火照着他手边那几行新批的小字，笔画干硬，不留余地：

“边兵所食，不许欠。”
