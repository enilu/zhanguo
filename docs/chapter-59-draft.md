# 第五十九回 秦廷得胜仍先收其势 赵奢理赋直把深门开

伊阙大捷之后，咸阳宫里的喜气并没有闹得太满。

秦王连召数日军议，问得最多的不是“可趁势再夺几城”，而是“韩魏如今哪一处最喘不过气，哪一处又会因败而死守”。

白起再入殿时，案上已不见庆功酒器，只剩韩、魏新报与边图。

秦王问：“伊阙之后，若寡人命你再压韩魏，你先取何处？”

白起道：“先取它们最难同时顾到的地方，不先取看着最肥的地方。”

“为何？”

白起走到图前，把手落在韩境几处败军回路与魏韩交界的细线上：“韩军伤得更急，魏军疑得更深。此时若贪远，诸侯反会因惊先并；若只一段段压它们新露出来的软处，它们便要一面补败，一面防彼此，那口气便更顺不过来。”

秦王又问：“你不求再立大功？”

白起答：“臣若只求立功，便等于替诸侯省了自己再犯错的工夫。”

殿中一时极静。

张仪站在旁边，望着图上那几道细线，忽然明白自己这些年替秦拆开的东西，到了白起手里，已不是朝堂上的空隙，而是能在阵上继续往里踩的路。

秦王沉吟片刻，道：“准。你去，不必贪远，只把这一战的后劲往韩魏骨里再推深一寸。”

白起受命而退。

张仪望着他背影，低声道：“臣拆的是人心之盟，白起踩的是阵上之缝。秦后面这两只手，怕是要一路接着走了。”

秦王淡淡道：“寡人要的，正是如此。”

同一时刻，邯郸那边也忙得比往日更紧。

廉颇已赴西边，赵王便把国中另一摊最枯、最易得罪人的硬事压到了兵赋与军需上。案上堆着赋册、马政旧案、各郡转粮簿，看得几名旧臣头都发胀，争到最后，争的也还是旧话：

边费已重，

豪右必怨，

可若再缓一缓，又怕边上来不及。

赵王听得脸色渐沉，忽然道：“赵奢在不在？”

赵奢早候在殿外，闻召入内，行礼毕后站得笔直。赵王把一卷赋册丢到案前，只问一句：

“赵若要把西边撑住，最怕的不是秦，先怕什么？”

赵奢几乎没有迟疑：

“先怕国中人人嘴上说重兵，手里却不肯把该出的粮、马、役老老实实交出来。”

几名大臣神色同时一变。

因为赵奢这一句，已不是在谈空势，而是把刀口直直顶到了豪强、贵近和那些最会在国急时替自己找缓词的人身上。

赵王道：“说下去。”

赵奢答：“廉颇在边上练的是兵。兵若要立住，后头便得有人把粮、马、甲、车、道路一件件钉死。国中若仍是能拖便拖、能欠便欠，边上纵有好将，也不过替一堆空账去硬撑。”

一名老臣忍不住道：“话虽如此，也不可逼得太狠。大族若俱生怨，终究坏国中大局。”

赵奢转头看他，声音不高：“边军若缺粮，秦骑会不会也讲一句不可逼得太狠？”

那老臣顿时无话。

赵王盯着赵奢看了很久，道：“那依你，如何做？”

赵奢道：“先从最会拖的地方查起。凡军赋军需，先问旧簿，再开新验；凡占官马、匿军粮、借故缓输者，不论贵近，一体核实。先把一国之内这股‘都知该做、却总想别人先做’的病压下去。”

苏秦坐在侧席，听到这里，心里忽然一动。

因为他明白，赵奢走的不是自己那条“先把天下之患说到众人心里去”的路。赵奢做的，是把一国里最硬、最不讨喜、却最不能再拖的东西，硬生生逼实。

赵王没有再多犹疑，只命赵奢把近三月拖欠军赋的郡县、贵族、署名人等全部列出，当日就查。

数日之后，邯郸城中果然起了波澜。

赵奢查得极硬。

有贵戚借修园之名扣下官马，他便带吏入门，当场验数；

有大族拖欠军粮，门客四出说情，他一句不听，只并旧簿新册对问；

有郡守口口声声说“民困不可急征”，暗地里却先保自家租谷，赵奢便把名字记上，连夜上呈。

邯郸深门里一时骂声不少，都说赵奢不给人留台阶。可骂归骂，西去的粮车却真一批批动了起来。

夜里，苏秦出宫，正见一列运甲车自军需署外缓缓驶过。车轮压石而行，声音沉稳，不急不乱。

门客低声道：“相国，这赵奢像是比廉颇还难惹。”

苏秦看着那列车，过了一会儿才道：“廉颇是在边上挡人，赵奢是在国内逼人。赵若想不做第二个韩魏，这两样人，怕都少不得。”

风从邯郸城头卷过，城外西路上，新一批粮车正往边地去；城内深门里，也有更多人在算自己还能不能像从前那样继续拖。

赵奢站在军需署灯下，把最后一卷验过的赋册合上，没有抬头。

灯火照着他手边那几行新批的小字，笔画极硬：

“边兵所食，不许欠。”
