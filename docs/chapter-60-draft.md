# 第六十回 赵奢按赋贵戚先折气 廉颇守边秦骑不敢轻

赵奢查军赋的风，一起便收不回去了。

邯郸旧例里，最难碰的从来不是寻常郡县，而是那些与王室、权臣、豪族盘根错节的深门大户。平日朝中议到这里，总有人说一句“事关体面”，于是许多该补给边军的东西，便一点点烂在这些“体面”里。

赵奢偏先去碰的，就是这一层。

这一日清晨，他带着军需吏直入一处贵戚庄园。

庄门高大，门前车马未绝，门内家臣早听见风声，迎出来时脸上已带着笑：“赵君何必亲来？有事递个话，府上自会照办。”

赵奢把手里那卷册子展开，只道：“官簿记你家今岁应输军粮三千石，已缓两月。另有官马十七匹，至今未验。今日我来，不为递话，只为验数。”

那家臣笑意微僵，低声道：“此事主人已上达朝中，自有转圜。”

赵奢抬眼看他：“边军吃饭，也等你家转圜？”

说完，便命属吏入内开仓、点马。

家臣急了，连忙抬出府中名帖，又说主人与某重臣相善，若这样硬查，日后朝里不好相见。赵奢一句也不接，只让人当众把仓门启开。

仓门一开，里头粮数果然与申报不合。

围观的人群顿时起了窃语。

赵奢面色不改，当场命吏记下缺额，又把验出的官马一匹匹牵出，连毛色齿岁都核得清楚。最后他把名帖推回去，只说了一句：“有交情，去朝里说；有军赋，今日补齐。”

这句话不过十几个字，却比许多大声斥骂更叫人难堪。

消息很快传遍邯郸。

从前众人还只当赵奢是新得势，借查账立威；这一下之后，城中才真正明白，这人不是装硬，而是真不打算给谁留那层“先拖一拖”的台阶。

朝中立刻有人弹他。

说他苛急。

说他伤国体。

也有人说，边事虽重，如此得罪满城，终究要坏大局。

赵王把这些奏札一封封看完，没有先表态，只命赵奢入见。

赵奢到殿，依旧神情平定。

赵王问：“满朝都说你逼得太狠。你自己怎么说？”

赵奢答：“臣若今日给他们留脸，明日边军便要饿肚。”

赵王道：“就没有两全之法？”

赵奢抬头：“大王，兵者，国之重器。查到要处时，还想两全，便是两边都保不住。”

苏秦在侧席听着，忽然想起自己这些年在六国朝堂上见过的太多好话。那些话常常也都不是假的，只是每逢真正要割肉的时候，人人都盼着晚一日、轻一点、换个人先出。

赵奢却不同。

他做的，正是把那层最难听的话，当成政令去落。

赵王沉默良久，终究把弹劾奏札合上，只说：“继续查。谁若再拿空话阻边军，你把名字也一并送来。”

赵奢俯身受命。

这一日出宫之后，他没有回府，径直去了军需署后场。那里新到一批西送甲械，工匠正在验扣、整束。赵奢站在棚下，一件件看过去，连革带是否牢、车轴是否松都问得极细。

署中小吏私下都说，这位大人像是把一支军队拆回了粮、马、甲、绳、车，连最不起眼的一环都不肯放过。

可也正因为如此，西边送来的回报越来越实。

十余日后，廉颇自边上上书。

书不长，只写了几件事：

军粮已按期到；

骑军新换之马可用；

夜巡不再因辎重迟滞而误时；

近来秦骑又试边两次，皆未得便。

赵王把这份军书看完，递给苏秦。

苏秦看过之后，轻轻吐出一口气：“赵西边这口气，算是续上了。”

赵王又把军书给赵奢看。

赵奢只扫了一遍，便道：“还不够。如今不过是把该有的送到了。真正要紧的，是让边上三个月、半年后也不断。”

赵王听后，反倒笑了一下：“你这人，倒真不爱听好话。”

赵奢答：“臣爱听。只是边上的好话，得等秦骑白来几次之后，再听。”

与此同时，西边营中，廉颇也正立在垒上看秦骑远去。

副将道：“将军，近来秦人来试多次，终不敢深。是否已知赵边难啃？”

廉颇道：“它不是不敢，是在看我们是不是只硬一时。”

副将一怔。

廉颇又道：“守边最怕头一月整得像样，第二月粮车迟，第三月马乏、夜巡懈。秦若等的正是这个，我们便偏不叫它等着。”

他说完，抬手按了按垒上新添的木楯。

木楯后头，是一排排更整的甲兵；

木楯前头，是被夜风吹得忽明忽暗的秦境火色。

更深时，邯郸城里仍有人在骂赵奢不近人情。

可骂声传不过西边高垒。

那里只听得见军鼓换更、马鼻喷气，以及一支军队慢慢把筋骨接牢的声音。
