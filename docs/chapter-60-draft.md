# 第六十回 赵奢验赋先折深门气 廉颇坐边偏叫秦骑空来

赵奢查军赋的风，一起便再收不住了。

邯郸旧例里，最难碰的从来不是寻常郡县，而是那些与王室、权臣、豪族盘根错节的深门大户。平日朝中每议到这里，总有人先说一句“事关体面”，于是许多本该送往边军的粮、马、甲、车，便一点点烂在这些“体面”里。

赵奢偏先去碰的，就是这一层。

这一日清晨，他带军需吏直入一处贵戚庄园。庄门高阔，门前车马未绝，家臣早听见风声，迎出来时脸上还带着笑：“赵君何必亲来？有事递个话，府上自会照办。”

赵奢展开册子，只道：“官簿记你家今岁应输军粮三千石，已缓两月；另有官马十七匹，至今未验。今日我来，不为递话，只为验数。”

那家臣笑意微僵，低声道：“主人已向朝中说过，自有转圜。”

赵奢抬眼看他：“边军吃饭，也等你家转圜？”

说完，便命属吏开仓点马。

家臣急了，忙捧出名帖，又说主人与某重臣相善，如此硬查，日后朝里不好相见。赵奢一句也不接，只让人把仓门当众启开。

仓门一开，粮数果然与申报不合。

围观的人群顿时起了窃语。

赵奢面色不改，当场命吏记下缺额，又把验出的官马一匹匹牵出，连毛色齿岁都逐一核清。最后他把那名帖推回去，只说了一句：

“有交情，去朝里说；有军赋，今日补齐。”

这十几个字，比许多大声斥骂更叫人难堪。

消息很快传遍邯郸。

到这时，城中人才真正明白，赵奢不是借查账立威，而是真不打算再给谁留“先拖一拖”的台阶。朝中弹劾他的奏札也立刻多了起来，说他苛急，说他伤国体，说边事虽重，如此得罪满城，终究坏大局。

赵王把这些奏札一封封看完，只命赵奢入见。

赵奢到殿时仍神情平定。

赵王问：“满朝都说你逼得太狠。你自己怎么说？”

赵奢道：“臣若今日给他们留脸，明日边军便要饿肚。”

“就没有两全之法？”

赵奢抬头：“查到要处时还想两全，便是两边都保不住。”

苏秦在侧席听着，忽然想起这些年诸国朝堂里那些漂亮的好话。那些话未必全假，可每逢真要割肉时，人人都想晚一日、轻一点、换别人先出。赵奢做的，却正是把最难听的话直接当成政令去落。

赵王沉默良久，终究把那些弹劾奏札合上：“继续查。谁若再拿空话阻边军，你把名字也一并送来。”

赵奢俯身受命，出宫后没有回府，径直去了军需署后场。那日新到一批西送甲械，工匠正在验扣、整束。他站在棚下，一件件看过去，连革带是否牢、车轴是否松都问得极细。

署中小吏私下都说，这位大人像是把一支军队拆回了粮、马、甲、绳、车，连最不起眼的一环都不肯放过。

可也正因为如此，西边送回来的军书越来越实。

十余日后，廉颇自边上上书，所写不过几件事：

军粮已按期到，

新换官马可用，

夜巡不再因辎重迟滞，

近来秦骑又试边两次，皆未得便。

赵王把这封军书递给苏秦。苏秦看完，轻轻吐出一口气：“赵西边这口气，算是续上了。”

赵王又把军书给赵奢看。赵奢只扫一眼，便道：“还不够。如今不过是把该有的送到了。真正要紧的，是让边上三个月、半年后也不断。”

赵王听后反倒笑了一下：“你这人，倒真不爱听好话。”

赵奢道：“臣爱听。只是边上的好话，得等秦骑白来几次之后，再听。”

与此同时，西边营中，廉颇正立在高垒上看秦骑远去。副将道：“近来秦人来试多次，终不敢深，是否已知赵边难啃？”

廉颇道：“它不是不敢，是在看我们是不是只硬一时。”

副将一怔。

廉颇又道：“守边最怕头一月整得像样，第二月粮车迟，第三月马乏、夜巡懈。秦若等的正是这个，我们便偏不叫它等着。”

说完，他伸手按了按垒上新添的木楯。

木楯后头，是一排排更整的甲兵；

木楯前头，是夜色里忽明忽暗的秦境火光。

更深时，邯郸城里仍有人在骂赵奢不近人情。

可这些骂声传不过西边高垒。

那里只听得见军鼓换更、马鼻喷气，以及一支军队慢慢把筋骨一节节接牢的声音。
