# 第六十一回 乐毅入赵先问兵期路 田文望齐已知满处危

乐毅等的从来不是一日之怒。

可等，不等于不动。

他把燕国内的兵、粮、路大致摸清之后，向燕昭王请出的第一步，不是先去魏，也不是先去韩。

是先去赵。

燕昭王问：“为何先赵？”

乐毅答：“燕若后面真要南下，赵是背后之山，也是侧边之门。赵若肯听，诸国便知此事不只是燕一国泄恨；赵若不肯，燕也能早知最硬一处门槛，不至临到阵前才知路不通。”

燕昭王点头：“你欲遣谁？”

乐毅道：“臣愿亲去。”

数日之后，乐毅至邯郸。

赵王并未设大朝，只在偏殿留了苏秦、赵奢与数名近臣相见。这层安排本身，便透出赵廷如今的气：若放在前些年，燕使来议大事，多半先落到“合纵共势”的高话上；如今坐在殿中的，却是一半看兵、一半看账的人。

乐毅行礼毕，开口第一句也不大：

“燕此来，不为先谈伐齐，只为先问赵一句实话。”

赵王抬眼：“你且说。”

乐毅道：“齐近岁骄而自满，山东诸侯怨之者渐多。燕自知不能独举大兵，若后面真有大举，赵之向背极重。臣今来，只愿先听赵看齐，究竟是可借、可防，还是可击。”

殿中一时安静。

这个问题问得太实，不容拿场面话糊过去。

赵王没有立刻答，先看向苏秦。

苏秦沉吟片刻，道：“若论旧理，赵、燕、齐本不该自相削弱，以利西秦。可若论眼前实势，齐近来气太满，山东诸国也未必还吃得下它那套处处算尽的做法。”

这话仍留着一分旧纵横的分寸。

赵奢接得却更硬：“赵如今先重西边。东边若另起大局，赵不能替人打一场说不清的仗。兵期几日，粮路几线，主帅是谁，退路归谁守，若这些都写不死，赵便无话可谈。”

乐毅听完，眼底反倒微微一亮。

因为他怕的不是赵先谈条件，而是赵还在说那些谁都爱听、却落不到地上的漂亮话。

他当即道：“赵君之言，正合臣意。燕后面若再来议，不会只带怨齐之词，还会带来道里、粮期、会兵次第与诸国分任。燕不想再做第二个伊阙。”

赵奢这才第一次正眼看他：“若真如此，倒像是在议兵，不是在议怒。”

赵王缓缓点头：“赵今日可给燕一句话。赵不愿见齐独大，也不愿见山东更乱。燕若真有能落地的章法，赵可听。”

这句话不算承诺。

可对乐毅来说，已足够知道赵廷的门并未关死。

会后，苏秦单独送乐毅到宫门外。春寒尚硬，两人并行一段，都没有先说话。到了宫门影里，苏秦才低声道：

“乐君这次来赵，问得不像使者。”

乐毅道：“若还只像使者，许多事便永远只能停在纸上。”

苏秦望着远处邯郸城头，缓缓道：“我当年要的是天下共忧秦。如今你们要做的，却像是把各国已经各自长出的忧、怨、利，再重新拧成一路。”

乐毅答：“时势不同，只能如此。”

苏秦没有叹气，只点了点头。

因为他也已明白，后来的聚，不会再是自己那一套聚法了。

同一时刻，临淄那边却正热。

齐湣王近日接了数国使者辞色，越发觉得齐居山东之上，诸侯虽嘴上各有话，终究还得看齐脸色行事。朝中近臣顺着这股气，不断劝他再压一步：或争宋地，或逼韩魏，或借旧事再使燕低头。

田文越听，心里越沉，终于出班道：“大王，齐之强，贵在能使四邻有所忌，也有所依；不在使四邻尽生畏且怨。今秦方盛，山东若更因此相离，不是齐之福。”

齐湣王听罢，只淡淡笑了一声：“孟尝君总爱替别人多忧。”

这话一出口，殿中有人附和，有人低头，却都听得出味来：

齐王如今最不爱听的，正是这种不顺耳却偏有分量的话。

这句“总爱替别人多忧”，很快随密报追到了乐毅回燕的路上。

他在馆驿看完，只把那张纸慢慢折起，对随从道：

“齐这口气，又满了一层。”

随从问：“这不是好事么？”

乐毅望着北归的路，声音极平：“对想久成大事的人来说，是。”

马车继续北去，车轮碾过道上的碎石，声响细而硬。乐毅把赵廷那句“可听”，和临淄这句“总爱替别人多忧”，一并记进心里。

他知道，离那支真正该出鞘的联军，又近了一步。
