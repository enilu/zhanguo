# 第六十二回 燕使先把山东怨摸实 齐廷更盛天下已暗偏

乐毅回燕之后，没有立刻请战。

他先把邯郸之行所闻一条条写下：

赵可听，但先看实章；

赵奢最重兵期、主次与退路；

苏秦虽老，仍看得懂后来诸国之聚已与往日不同。

写完这些，他才入宫见燕昭王。

燕昭王听完，先问：“赵肯不肯借力？”

乐毅答：“肯听，未肯诺。”

“这算什么结果？”

“算门未关。”乐毅道，“赵今先重西边，不会替任何人打一场说不清的仗。也正因此，只要燕后面把章法带足，赵反而比那些满口好听话的朝堂更可信。”

燕昭王点了点头。

他不怕慢，怕的是看着众口热闹，真到用兵时却又各退半步。

乐毅随即请下第二步：

“臣请分遣数使，不先扬言合兵，只先去听山东诸侯对齐之怨究竟积到何处。”

燕昭王问：“为何不先说利？”

乐毅答：“怨是火种，利是柴。先不知火落在何处，柴再多，也未必点得起来。”

郭隗在旁含笑道：“乐毅这是要先替诸国把那些不肯明说的怨，一根根摸出来。”

燕昭王当即准了。

很快，燕使分向赵、魏、韩而去。带去的都不是大张旗鼓的盟书，而是一连串更细、更难听、却更见真心的问话：

若后面真有一举，各国最怕齐先压哪一线；

若会兵，谁愿主前锋，谁只肯为后应；

若借道，各国最忌哪一条兵路压过本境太深；

若事成之后，各自最想取回什么。

这一问，比空谈亲疏更能摸出真骨头。

魏那边回得最谨慎。伊阙之败刚过去不久，魏廷上下对“会兵”二字都先有一层寒意。可一听燕使说的不是谁先盟誓，而是兵路、退路、功地如何分，魏相反倒坐直了些，低声道：

“若真有人肯把这些丑话先说在前头，也未必不是件好事。”

韩那边更直。

韩王听完燕使转述，只问一句：

“若他日诸国兵并，谁来压齐锋最重处？谁又能担保前后不乱？”

燕使答不上太多，只把乐毅那句“先把主次写死，再谈义气”原样奉上。韩王听后，沉默了很久。因为伊阙之后，他对这句话比谁都敏。

同一时刻，临淄朝会却越发热闹。

齐湣王接连得报，说燕国仍在修政养士，赵国忙于西边，魏韩各顾各伤，便越发觉得山东诸国纵有小心思，也终归翻不起大浪。殿上近臣顺着这股气，不断劝他再压一步：或逼韩，或震魏，或借旧事再让燕低头。

田文听得越发发沉，终究再度入言：

“大王，齐之强，贵在使四邻有所依，而不在使四邻尽生惧。今若处处求压，便是自己替别人找并起来的理由。”

齐湣王这回连笑意都淡了，只道：“孟尝君近来常替诸侯说话，倒像不大替齐说话了。”

田文俯身答：“臣正是替齐说话，才不敢只顺大王耳边最爱听的话。”

殿里空气顿时紧了一层。

群臣多低头不语。因为谁都看得出来，大王如今最不爱听的，恰恰是这种不顺耳却偏有分量的话。

散朝之后，田文独自立在阶下，望着远处临淄宫阙，胸口像压了一层灰。门客趋前道：“君上何苦屡逆王意？”

田文道：“齐若只怕别人逆它，离真出事便不远了。”

同夜，燕廷密报一封封归来。

乐毅没有急着总看，只先按国分开，又按“怨”“利”“惧”“可谈之处”四样各自标记。

他看得极细。

魏怕再做第二个先败的前锋；

韩怕兵并而令不一；

赵最重兵期、借道与谁来镇后线。

这些东西若换成旧说客口中的大义，多少都不好听。

可在乐毅眼里，正因为它们难听，才是真话。

燕昭王夜里再召他入宫，问：“如何？”

乐毅把几份密报并排压在案上，手指缓缓点过：

“山东诸国未必已肯并力，却都越来越不愿再由齐这样高高压着。大王，怨已经不小。接下来只等齐自己再多走几步，替我们把最后那几根线绷紧。”

燕昭王看着那些密报，许久才道：“你仍不请战。”

乐毅答：“请战易，请成战难。再等等，等到诸侯提起齐时，不再只是心里有气，而是已各自开始算那条能真的出兵的路。”

宫外夜风很硬，吹得灯影在地图上轻轻发颤。

燕昭王抬手按住图角，没有催。

而在更南边的临淄，齐湣王正于盛宴中举杯，听殿上作乐，听臣下称颂齐威。杯中酒光一晃一晃，照着他眼里那层越来越亮、也越来越满的光。

燕都北夜与临淄华灯，隔着山河遥遥相对。

一边在等，一边在盛。

后来那场最整齐、也最危险的合兵之势，已在这两种全然不同的气里，慢慢长出骨来。
