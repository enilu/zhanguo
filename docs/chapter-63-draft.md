# 第六十三回 郢都旧臣争局终难定 楚国新士黄歇始入朝

山东诸国各自动起来的时候，楚国却像一条大船，船身仍阔，水也仍深，可舵下总像隔着几层各不相同的力，怎么拧都拧不成一股。

郢都近来朝会，最常见的不是谁拍案怒骂。

恰恰相反，是人人都各有道理。

有人说，秦兵既盛，楚当先保西北，不可再为旧盟空耗；

有人说，齐势渐满，楚若还总是前后摇摆，后面只会两头都不得信；

也有人说，国中连年兵费已重，先安民、先整赋，比再谈外局都急。

这些话拆开听，竟都不算错。

可也正因为都不算错，怀王坐在上首，反倒一日比一日更烦。

屈原仍在。

只是这两年下来，他说话更少，也更直。许多从前还能迂回着说的东西，到如今他已不大肯替谁遮掩。

这一日朝会又议到齐、秦两边辞色。怀王听了一阵，忽然把手中竹简往案上一搁：“寡人听了半晌，只听见人人都在说自己那一头难。那楚这条路，究竟还走不走得成？”

殿中顿时安静。

因为这是这些年最难答的一问。

昭阳正欲出班，殿外忽有近臣引一年轻楚士入内。那人衣冠整肃，神色却并不局促，入殿后先拜，动作极稳。

怀王皱眉：“此人是谁？”

近臣答：“兰台新选之士，姓黄，名歇。近来所上数条边务、赋议，颇为通达，臣等不敢自专，故请他今日随班入对。”

怀王听了，看了他一眼：“你便是黄歇？”

黄歇俯身：“臣是。”

怀王道：“方才寡人之问，你可听见了？”

黄歇答：“听见了。”

“那你说，楚这条路，还走不走得成？”

满殿目光一下都落在这年轻人身上。

黄歇没有急着答大，只先道：“若大王问的是还能不能像从前那样，一声号下，外有诸侯齐应，内有群臣同口，那恐怕难。若问楚还能不能自立其局、再择其可合者而用之，臣以为还走得成。”

这话一出，殿中许多人都抬了抬眼。

因为它既没顺着旧盟往高处说，也没顺着近年疲势往低处说。

怀王也微微坐直了些：“说下去。”

黄歇道：“楚地大、众多、财赋厚，这些都还在。可楚眼下最大的病，不在一处输赢，而在朝中与军中、对齐与对秦、外议与内理，总像分成几层，各自都想先把自己那一层说圆。如此一来，外人未必先压楚，楚自己那股气便先散。”

这话比起屈原和陈轸，少了几分刀口，更像是把散开的线一根根指出来给人看。

怀王沉默片刻，问：“那依你，当先理哪一层？”

黄歇答：“先理三件事。其一，朝中辞色要一口，不可今日借齐压秦，明日又借秦牵齐；其二，军中主次要定，不可逢事都盼外兵替楚先受最重一口；其三，国内赋与边储要分开归实，不可前线每急一次，国中便人人各寻缓词。”

屈原站在一旁，第一次认真看了这年轻人一眼。

因为这几句话虽不新奇，却稳，且是从楚自己身上先下手。

怀王又问：“你这些话，与屈原、陈轸说的，有何分别？”

黄歇抬头，答得平平：“屈大夫、陈大夫所见，比臣深。臣不过是觉得，药既已知，只靠说药理还不够，还得有人去把药一剂剂煎出来、端到人手边。”

这话一出口，连屈原眼里都闪过一点极淡的异色。

因为这年轻人没有借机高抬自己，也没有顺势踩前人。

他只是把自己的位置摆得很清：

不争第一句道理，只求后面真有人去做。

怀王听完之后，许久没说话。

过了半晌，才道：“黄歇，你先留在兰台，另把你方才说的三件事，各写一条可行之法上来。”

黄歇俯身受命。

散朝之后，屈原走得不快。

黄歇自后跟上，行了一礼：“屈大夫。”

屈原看着他，问：“你今日上殿，不怕说轻了被人当圆滑，说重了又触王怒？”

黄歇道：“怕。”

屈原微微一怔：“怕还说？”

黄歇答：“怕也得说。只是楚如今最缺的，未必是再多一句更响的话，而是有人把听得进去的话，先送到能做事的地方去。”

屈原听后，沉默了片刻，才道：“你这话，倒像不是士子口气。”

黄歇笑了笑：“臣若只想做士子，今日便不必入朝。”

这句话不锋利，却自有一股温厚里的硬。

屈原望着这年轻人远去背影，心里那股久压不散的冷意里，竟难得浮起一点极轻的松。

不是因为楚局便能自此回稳。

而是因为在这座越来越多人只会替自己那一层找说法的郢都朝堂上，终于又多了一个愿意先去托住整局的人。

夜里，黄歇回到舍中，案上没有摆酒，也没有摆诗。

只摆了三卷简：

一卷写朝辞；

一卷写边储；

一卷写军中主次。

他把那卷“边储”先摊开，提笔极稳。

窗外楚宫灯火层层，映得纸面一片暖黄。

这位方入朝的新士，没有先写最易显名的那卷。

他先写的，是最重、最慢、也最不讨人喜欢的那卷。
