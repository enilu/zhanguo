# 第六十三回 郢都旧臣各把一头说 楚廷新士只作后手看

山东诸国一处处都动起来的时候，楚国却仍像一条阔而重的大船。

船身未轻，

水势未浅，

可舵下总像隔着几层不同的力，怎么拧也拧不成一股。

郢都近来朝会，最常见的不是拍案怒骂，恰恰是人人都各有一番说得过去的道理。

有人说，秦兵既盛，楚当先保西北，不可再为空盟耗力；

有人说，齐势渐满，楚若仍两头摇摆，后面只会两头不得信；

也有人说，兵费已重，先安民、先整赋，比再论外局都急。

这些话拆开听，竟都不算错。

可也正因为都不算错，怀王坐在上首，反倒一日比一日更烦。

这一日朝会又议到齐、秦两边辞色。怀王听了半晌，忽然把手中竹简往案上一搁：

“寡人只听见人人都在说自己那一头难。那楚这条路，究竟还走不走得成？”

殿中顿时静了。

因为这是这些年最难答的一问。

屈原仍在，只是说话比往日更少，也更直。许多从前还能绕着说的地方，到如今他已不大肯替谁遮掩。昭阳正要出班，殿外却有近臣引一年轻楚士入内。

那人衣冠整肃，神色并不局促，入殿后先拜，动作极稳。怀王皱眉：“此人是谁？”

近臣答：“兰台新选之士，姓黄，名歇。近来所上几条边务、赋议还算通达，臣等不敢自专，故请他今日随班入对。”

怀王看了他一眼：“你也听见方才寡人之问了？”

黄歇俯身：“听见了。”

“那你说，楚这条路，还走不走得成？”

满殿目光一时都落到了这年轻人身上。

黄歇没有急着答大，只道：“若问还能不能像从前那样，一声号下，外有诸侯齐应，内有群臣同口，恐怕难。若问楚还能不能自立其局，再择其可合者而用之，臣以为还走得成。”

这话既没顺旧盟往高处说，也没顺近年疲势往低处说。怀王听完，微微坐直了些：“说下去。”

黄歇道：“楚地大、众多、财赋厚，这些都还在。楚眼下最大的病，不在一处输赢，而在朝中、军中、对齐、对秦、外议、内理，总像分成几层，各自都想先把自己那一层说圆。如此一来，外人未必先压楚，楚自己那股气便先散。”

屈原站在一旁，第一次认真看了这年轻人一眼。

因为这几句话不算新奇，却稳，且是从楚自己身上先下手。

怀王又问：“那依你，当先理哪一层？”

黄歇答：“先理朝辞与边储。前头说给外人听的话不能日日自打自脸，后头送往边上的粮与储也不能随朝议喜怒时多时少。至于军中主次，还得请军中宿将定得更实，臣不敢轻言。”

这一下，殿中几人都抬了抬眼。

因为这年轻人并没有顺势把什么都揽到自己头上。

他只挑自己此刻真正能碰的那几层说，也把自己不能轻说的一层让了出来。

怀王沉默片刻，才道：“你这些话，与屈原、陈轸所说，有何分别？”

黄歇答得很平：“屈大夫、陈大夫所见，比臣深。臣不过觉得，药理既已有人说过，后面总还要有人肯去把药煎出来、端到案前。”

这句话一出口，连屈原眼里都闪过一点极淡的异色。

因为这年轻人没有借机抬自己，更没有顺势踩前人。他只是把自己的位置摆得很清：

不争第一句道理，

只求后面真有人去做。

怀王听完之后，许久没说话，最后只道：“你先留兰台，把方才说的两件事，写成可行条陈送来。”

黄歇俯身受命。

散朝之后，屈原走得不快。黄歇自后跟上，行了一礼：“屈大夫。”

屈原看着他，问：“你今日上殿，不怕说轻了被人当圆滑，说重了又触王怒？”

黄歇道：“怕。”

屈原微微一怔：“怕还说？”

黄歇答：“怕也得说。只是楚如今最缺的，未必是再多一句更响的话，而是有人把听得进去的话，先送到能做事的地方去。”

屈原听后，沉默了片刻，才道：“你这话，倒不像寻常士子口气。”

黄歇笑了笑：“臣若只想做士子，今日便不必入朝。”

屈原望着这年轻人远去的背影，心里那股久压不散的冷意里，难得浮起一点极轻的松。

不是因为楚局便能自此回稳。

而是因为在这座越来越多人只会替自己那一层找说法的郢都朝堂上，总算又多了一个愿意去托住整局的人。
