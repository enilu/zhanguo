# 第六十四回 黄歇上书先定朝辞储 昭阳观简始知楚亦可补

黄歇上书很快。

不过三日，两卷条陈便先送入了兰台。

他没有一口气把所有大事都揽在纸上，只先写了自己在殿上应得起的两件：

《定朝辞》，

《实边储》。

近臣起初只当这年轻人头一回得了王问，难免急着求名，写出来多半还是好听话居多。可等把简一条条翻开，几人神色都慢慢变了。

《定朝辞》写得最短，只定两条：

凡对齐、对秦之外辞，先经兰台与令尹合议，不得前后自相打架；

凡军前未定之事，不得先在殿上放话示外。

字少，意思却硬。

《实边储》更重。黄歇把楚西北数郡近年调粮、转运、滞欠、借调之数一一列明，最后只落一句：

“边储不可再随朝议喜怒而时多时少。”

近臣看完，都不由低低吸了一口气。

因为这些东西没有一句是拿来在朝堂上争彩头的，却句句都冲着楚这些年最疼、也最没人肯先碰的地方去。

怀王这回没有把简先搁到一旁。

他从第一卷看到第二卷，眉头越看越紧，竟不是因为恼，更像是第一次发现，有人真把过去总在朝中高来高去、散来散去的话，压成了几样能问责、能执行、能见人的东西。

他看完之后，命左右道：“召屈原、昭阳。”

两人入内时，怀王已把那两卷条陈摆在案前。

屈原接过简，只看了片刻，便道：“黄歇此人，可留。”

怀王道：“你不嫌他年轻？”

屈原答：“年轻是年岁，不是病。楚如今真正的病，是知道该补哪几处的人不少，肯把补法写到这般不讨喜的人不多。”

昭阳看得更细。他把《实边储》那卷翻到中间，才抬头道：“此子不抢军中话，却先把后头要给军中垫底的那一层写实了。”

怀王听后，忽然哼了一声：“你们这些人，一个比一个爱叫寡人听难听话。”

屈原并不退：“若都只说好听话，大王今日案上也不会摆着这些简。”

怀王这回却没发作。因为他心里也清楚，黄歇写的未必都能立刻行下去，可至少不是拿来给他脸上添光的虚词。

午后，小朝再开。

怀王没有宣满殿群臣，只召昭阳、屈原、数名近臣与黄歇对坐。他把《实边储》压在最上，直接问：

“你说边储不可随朝议喜怒而多寡。寡人若问你，钱粮先从何处挤？”

黄歇答：“先从最会说缓词、又最不肯先出的一层挤。”

昭阳听了，眼皮微微一跳。

因为这句话说得并不狠，可刀口极准。

怀王又问：“若那些人反说民困、路远、仓虚呢？”

黄歇道：“那便把各郡仓数、行程、旧簿都摊开。真困者缓，假困者责。臣不怕他们说难，只怕谁都拿一个‘难’字，把该补的地方一并抹平。”

怀王沉吟片刻，又问《定朝辞》：“你说外辞要一口。可齐、秦本就两头牵扯，如何能一口？”

黄歇答：“不是只能说一句话，是不能今日说给齐听的，明日自己先忘；也不能今日借秦压齐，明日见秦让利，又把前日说过的话全翻过去。外辞若总这样自己打自己，外人便知楚心不定，楚自己的人也更不知该跟哪句走。”

这话说到后来，连怀王都没有再插断。

因为这些毛病，并不是没人早看见。只是从前看见的人，多半先把话顶到他脸上，逼得他先护自己的脸。黄歇却不同。他也说病，可说法更像是在理乱线，让人先看见“若不这样，后头哪里会坏”，而不是先听见“你哪里错了”。

散议之后，怀王破例留黄歇多说了一刻。等他退下，昭阳才低声道：“此子可先留兰台听用，不必急着压重任。”

怀王点了点头，没有多说。

这意思已经够明白：

黄歇可以留，

但还只是后手，

还不是眼下楚廷真正扛局的人。

夜里，屈原独坐舍中，把那两卷条陈又看了一遍。门客在旁问：“大夫觉得此人如何？”

屈原把《实边储》放下，缓缓道：“他不是最锋利的那种人。”

“那是好还是不好？”

屈原望着案上灯影，过了半晌才道：“楚这些年，锋利的人并不少。少的是这种能把锋藏在宽里、把重事一件件托起来的人。”

他说完，伸手按住案上一角微微翘起的竹简。

那一下很轻。

却像替楚国这条已经摇晃多年的路，先替后来人留住了一点不肯轻放的力。
