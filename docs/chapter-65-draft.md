# 第六十五回 赵奢离署亲看西路车仓 廉颇一试便知此人可算军后生死

赵奢终于离了军需署。

赵王召他入殿，案上摆着西路粮簿、马籍和几条新补过的运道图，只问一句：“账上说到了边上，便真到了边上么？”

赵奢答：“臣请亲去看。”

数日后，他轻车出邯郸，只带数十骑和几名掌簿小吏，沿西路一段段查过去。别的人看的是车多车少，他看的却是车辙深浅、封泥新旧、仓口地上有几层旧轮痕。

到第三处转运点时，账上写着“甲械已西发”，现场却还有十余车压着未动。

押车小吏想拿旧话搪塞，说等后一批齐发更稳。赵奢先看一眼地上的车辙，便问：“这几车在此停了几日？”

小吏脸色立刻变了。

赵奢没让他再说第二遍，命人把簿册、封泥、车轮痕一道对。对完便知，多拖了整整三日。赵奢当场换人，又改旧例：“边上吃紧时，满十车便发，不许再图省事压成一堆。”

同行诸吏到这时才知道，他查的不是几车快慢，是边上一道军令会不会被这三日活活拖死。

再往西，正逢廉颇遣人来接。

廉颇得知赵奢在路上，便命他顺道入营。赵奢到边营时已近黄昏，营门并不大开，廉颇只在中军帐外等着，见他下马便问：“一路查车、查仓、查道，不嫌烦？”

赵奢回礼：“将军守边都不嫌枯，臣怎敢嫌。”

两人入帐，案上铺着新补转粮道和近来秦骑试边的线。廉颇不先说边情，只问赵奢：“你一路看下来，赵眼下最大的问题在哪？”

赵奢答：“不是少粮，是总有人拿‘再等等’三个字替自己省事。”

廉颇抬眼：“若放你在军中，怎么治？”

“该等的按日等，不该等的按罪问。”赵奢道，“军前最坏的不是明着抗令，是人人嘴上都说无碍，只把要命那一步往后拖半刻。”

这话刚落，营外便传来急报，说秦骑又来试边。

人数不多，只两三百骑，在外沿扬尘绕行。副将正欲照旧只盯营门，赵奢看了一会儿那条尘头，忽然道：“它今日不是只来看营，是来看新补那段转粮道。”

副将愣住：“何以见得？”

“它不朝中垒走，专朝道口兜。”赵奢道，“若今夜只守营，不护道，明日它便知道哪一段车最难过。”

廉颇看着他：“那你说怎么做？”

赵奢道：“营不动，骑不远追。只在道口伏两层火，把它想看的那一截先照乱。”

廉颇点头：“依你。”

命令下去不久，秦骑果然朝那段道口多转了一圈。刚近前，暗伏火把一齐亮起，营后车影、甲影和木栅影层层错开，外头根本看不清道上究竟压了多少车、伏了多少人，只得很快退去。

更深时，廉颇和赵奢并立土垒上。

风从西面撞来，吹得人衣角发硬。廉颇看着那条刚被夜色压住的新转粮道，忽然道：“你这人，不只会跟账较劲。”

赵奢答：“账较到底，也是兵。”

廉颇听完，难得笑了一下。
