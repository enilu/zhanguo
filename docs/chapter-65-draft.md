# 第六十五回 赵奢出署先看边车道 廉颇营前一试识其才

邯郸那边，赵奢也终于不再只守着军需署那几间屋。

连月查赋、点马、催粮之后，他对国中哪一处最会拖、哪一处真有余、哪一段路最容易在关键时刻坏事，心里都已大致有了数。

赵王这一日召他入殿，只问：“你替寡人逼出了粮、马、甲。那这些东西究竟到不到得了边上？”

赵奢答：“账上说到了，不算到。臣请亲往西路一看。”

赵王点头：“去。回来告诉寡人，赵现在的边，到底是实厚，还是纸厚。”

数日后，赵奢轻车出邯郸，不带大张声势，只领数十骑与几名熟簿之吏，沿着近来运粮赴边的几条道一段段看过去。

他看得比查账还细。

哪一处车轴最易陷；

哪一段栈道逢雨就滑；

哪一座转仓看着满，实则新旧粮混压，一翻便乱；

哪一支押车吏最会嘴上说“已妥”，实则只把东西送到半路便算交差。

跟着他的人起初还觉得，赵君到底不是久带兵的大将，看这些零碎看得太过。

可走到第三处转运点时，众人便不敢再轻慢了。

那里账上写着“甲械已齐运西”，现场却还压着十余车没动。押车小吏见赵奢不过数骑而来，原想拿旧话支应过去，说是等后一批同发更稳。赵奢只看了一眼车辙深浅，便问：“这些车在这里停了几日？”

小吏脸色一变，支支吾吾。

赵奢没有再问第二遍，只命人把簿册、封泥、车轮痕迹全对。

一对之下，果然多拖了三日。

赵奢当场换了押车之人，又把那条“待后车同发”的旧例当场改掉：“边上吃紧时，满十车便发，不许再图省事压成一堆。”

同行之吏看得心里发紧。

因为他们这才明白，赵奢眼里看见的并不是几辆车快几辆车慢，而是边上一道军令，到底会不会被这三日活活拖死。

再往西，正遇上廉颇派来接应的人。

原来西边前线近来又有秦骑频频来试，廉颇得知赵奢在路上，索性命人请他顺道入营一看。

赵奢到边营时，已近黄昏。

营门不开大礼，廉颇只在中军帐外等着，见他来了，先看了他一眼，便道：“听说你一路都在查车、查仓、查道，不嫌枯？”

赵奢回了一礼：“将军守边都不嫌枯，臣查这些，哪有资格嫌。”

廉颇听了，嘴角极轻地动了一下，算是笑。

两人入帐，案上铺的也不是虚图，而是西路几段新补过的转粮道与秦骑近来试边的行迹。

廉颇不先讲边情，先问赵奢：“你一路看下来，赵现在最大的问题在哪？”

赵奢道：“不在缺粮，在粮到边上前，总有人拿‘可再等等’三个字替自己省事。”

廉颇抬眼：“若放你在军中，你怎么治？”

赵奢答：“该等的按日等，不该等的按罪问。军前最坏的，不是有人明着抗令，是人人都口称无碍，只把要命那一步往后拖半刻。”

廉颇听到这里，手指在案上轻轻点了点。

因为这句话，正碰在他心里。

夜里，秦骑果然又来试边。

人数不多，只两三百骑，在外沿掠阵扬尘，看着像寻常探试。营中副将正欲依旧按例观望，赵奢却盯着远处那一线尘头，看了一会儿，忽然道：“它今日不是只来试营，是在试新补那条转粮道。”

副将一愣：“何以见得？”

赵奢道：“它绕得不朝中垒，反朝道口。若今夜只守营不护道，明日它便知哪一段车最难过。”

廉颇闻言，没有立刻下令，只看着赵奢：“那你说该如何？”

赵奢答：“营不动，骑不远追。但道口伏两层火，把它想看的那一截，先让它看不清。”

廉颇听完，点头：“依你。”

命令下去不久，秦骑果然朝那段道口多转了一圈。可刚近前，暗伏火把一起，赵营后路灯影与甲声顿时错开层次，看得外头的人根本辨不清道上究竟车几何、兵几何，只得很快退去。

副将回营后，才后知后觉出了一身汗。

因为若真让秦骑把那段新补转道看穿，后面试边便不会再只来试营了。

更深之后，廉颇与赵奢并立垒上。

风从西面吹来，很硬。

廉颇望着远处黑处，忽然道：“你这人，不只会跟账较劲。”

赵奢道：“账若较到底，也是一仗。”

廉颇转头看了他一眼，眼里终于露出一丝真正的认同：“这话倒像兵家人了。”

垒下巡骑一队队过，甲叶相击，声响整齐。

赵奢看着那条被夜色压住的新转粮道，心里第一次极清楚地觉出，自己这些日子在邯郸查出来的那些数字、仓册、车痕，到了边上，全都会变成另一种东西：

变成军气，

变成生死，

也变成将来某一仗里，谁能快半步、稳半步的根。
