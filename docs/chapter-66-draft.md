# 第六十六回 乐毅夜成会兵冷约诸条 田文在廊下再谏齐王莫把自己看稳

燕国那边，乐毅等的回话终于渐渐齐了。

赵说可听，不先诺；魏说可议，却先问前军、后应、退路；韩更直，只问谁来压最重那口锋。燕昭王这一夜召乐毅入宫，案上点了三盏灯，照着一幅更细的山东图。

王问：“诸国回书，可合卿意？”

乐毅答：“它们不是不怨齐，是都怕自己成第二个伊阙里先受锋的那一路。”

燕昭王又问：“那你下一步做什么？”

乐毅把几份回书一一铺开：“先把最难听的话写死。”

于是这一夜，他写的不是檄文，是《会兵诸约草案》。竹简一卷卷展开，前头几条便很硬：

“主帅不得二出。”

“前后军次，不得临敌始争。”

“借道之国，不得尽空其守境之兵。”

“失期者，失其先议之利。”

宫中秉烛的小宦者站得极直，连眼也不敢多抬。

与此同时，临淄那边却还热着。

齐湣王近来连看数国辞色，听来的多是敷衍与拖字，心里愈发认定诸侯有怨也不敢真并。散朝后，田文追到廊下，请王留步。

齐湣王回头：“孟尝君又有何忧？”

田文道：“臣忧的不是一国强弱，是诸国近来都太安静。”

齐湣王失笑：“安静也成忧？”

“若是服，便不忧。”田文道，“怕只怕它们不是服，是在各自盘算，等谁先把别人的怨拢起来。”

齐湣王脸色便淡了些：“你总把旁人看得太重。”

田文俯身：“臣只怕大王把自己看得太稳。”

这句话落下，齐湣王没有再停，拂袖而去。

燕宫里，乐毅也正把最后一条草约写完。

燕昭王看着那卷新成的竹简，问：“如今还差什么？”

乐毅把简卷好，压到山东图上：“差齐自己再抬一抬头。”

殿外北风一过，把灯火吹得斜了一瞬。
