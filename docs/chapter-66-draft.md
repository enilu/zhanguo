# 第六十六回 乐毅会诸侯先写冷约 田文看齐已知众心偏

燕国那边，乐毅要的那些回话，终于一点点齐了。

赵说可听，不先诺；

魏说可议，但先要把前锋、后应、退路与得地写清；

韩更直，只问谁来压最重那口锋，谁又能保联军不乱。

这些话没有一句豪壮。

可乐毅把它们并排摆在案上，心里反倒更定。

因为他知道，真正能走到阵上的联盟，本就不该先从豪壮起。

燕昭王这一夜召乐毅入宫，案上点了三盏灯，照着一幅比前些日子更细的山东舆图。王问：

“诸国回话，与卿先前所料，相差几何？”

乐毅答：“大势不差。如今诸国不是不怨齐，是谁都怕自己再做第二个伊阙里的先受锋者。”

燕昭王点头：“那你下一步，欲先做什么？”

乐毅没有立刻答，只把几份回书一一铺开，才道：

“先把最难听的话写死。”

燕昭王眼里微微一动：“何谓难听话？”

乐毅道：“谁为前军，谁为后应；借道多深；若齐先击一处，谁不可先退；若事成，城地如何分；若有一国后时，罚至何处。这些都难听。可若不先写死，后面纵有五国之名，也仍不过是一群各怀后账的人并立阵前。”

郭隗在旁听得连连点头。

因为这正是乐毅和寻常说客最不一样的一处：

他不是怕把话说得不够好听，

而是怕不把最难听的话先说清，后头那些好听话便全都成祸。

燕昭王沉吟片刻，道：“你若把这些都写出去，诸国会不会先觉得燕太精？”

乐毅答：“若它们只爱听漂亮话，大王便更不该把兵交给它们。”

这话说得极平，宫中却一时极静。

过了片刻，燕昭王忽然笑了：“好。那便写。”

于是这一夜，乐毅没有写讨齐檄文。

他写的是《会兵诸约草案》。

竹简一卷卷铺开，头几条便重得发硬：

“主帅不得二出。”

“前后军次不得临敌始争。”

“借道之国得留守土之兵，不得尽空。”

“后时者失其先议之利。”

写到后来，连秉烛的内臣都不敢抬头。

因为这些字眼里没有一丝热血，只有一层层替后来阵前争命预先立下的冷。

与此同时，临淄那边却仍在热。

齐湣王近来又命人催问数国辞色，听来的多是敷衍之词，心里便越发认定诸侯纵有不平，也不敢真并。朝中有人趁势上言，说燕近年虽修政养士，终究地薄兵单，不足深虑；赵又被秦牵住，山东再无人足与齐争。

田文在侧听着，只觉背后发凉。

这一次，他没有在殿上立刻入言，直到散朝之后，才独自追到廊下，请齐湣王少留一步。

齐湣王回头看他：“孟尝君又有何忧？”

田文道：“臣所忧者，不在一国强弱，在诸国近来都太安静。”

齐湣王失笑：“安静也成忧了？”

田文道：“若人心已服，自然不忧。怕只怕它们不是服，而是在各自盘算，等谁先把别人的怨拢起来。”

齐湣王听完，脸色淡了下来：“你总爱把旁人看得太重。”

田文俯身：“臣只是怕大王把自己看得太稳。”

这一句落下，齐湣王没有再停，拂袖而去。

田文立在原地，看着廊下灯火一盏盏拉长，心里那股灰意越压越重。

同夜，燕宫里，乐毅把最后一条草约写完，提笔停了很久。

燕昭王问：“如何？”

乐毅把简卷起，道：“如今还差最后一步。”

“哪一步？”

“等齐自己再抬一抬头。”乐毅道，“头抬得越高，越看不见脚下已围过来的影。”

燕昭王望着他，缓缓点头。

宫外北风掠过殿角，吹得灯火一斜。乐毅把那卷草约压在图上，没有再多说一个“战”字。

可从这一夜起，燕廷手里已不再只是一股怨。

它开始有了兵次，

有了借道，

有了前后，

也有了后来真能把五国拧成一股的那层冷骨头。
