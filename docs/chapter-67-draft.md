# 第六十七回 齐廷外压先向宋地试 田文再谏满朝已顺声

田文那一句“怕大王把自己看得太稳”，并没有在临淄留下多久回响。

齐湣王近来最爱听的，终究还是另一种声音：

燕不足虑，

赵被秦牵，

韩魏新败，

楚自摇摆。

既然诸国都各有掣肘，齐若此时还不往前再压一步，倒像白白辜负了眼前这层势。

这一日大朝，齐廷议的便不再只是辞色。

案上铺开的，是宋地新报。

宋国这些年在齐楚之间反覆摇摆，国小而财不薄，地又夹在几路要道之间，本就最容易被大国拿来试手。齐廷中早有人主张，先拿宋地边事做一道外压，既可试诸侯反应，又可顺手再把山东几处关键口子往齐手里攥紧。

齐湣王看完边报，只问一句：“宋若再不顺，诸侯谁会真替它出头？”

殿中很快便有笑声低低起落。

有人道：“赵正顾西，不会为宋动。”

有人道：“魏韩如今自身难固，谁敢先替小国受锋？”

也有人道：“燕最会藏头，口里或有话，兵却必不先出。”

这几句一叠，满殿气便越发热了。

田文坐在席间，听得心里一点点发沉。

因为他知道，大王若只是自信，还未必立刻成祸；可若满殿臣子都开始替这份自信找一条条“已有根据”的证词，它便会越发像一件理所当然的事。

齐湣王终于开口：“既如此，先敕宋使，再问其边。若仍拖泥带水，便让东境诸军往前压一步，看它到底还想靠谁。”

这道令一下，殿中许多人脸上都露出喜色。

不是因为这一压必有大战。

而是因为这正合他们近来最顺手的那层心意：

先把旁人往后逼一逼，看看谁先退，齐便知自己还可再高出几分。

田文终究还是出班。

“大王，宋固可制。可今山东最忌的，不是齐强，而是齐强而愈爱四面试人。”

齐湣王抬眼，已带了几分不耐：“孟尝君又要说什么？”

田文道：“诸侯未必会为宋立刻合兵，可他们会记。今日记齐压宋，明日记齐逼韩，后日再记齐轻燕。记到后面，不等齐真有大失，它们心里先会把‘要不要让齐再高一步’这句话压住。”

这话一出，殿中反倒静了一瞬。

不是没人懂。

而是大王脸色已不好看。

果然，齐湣王冷冷道：“孟尝君这些年养士太多，倒把旁国心思都替它们养熟了。寡人问的是今日之事，你说的却总是日后之忧。若人人都照你这般先怕后患，齐还要不要做人主？”

田文俯身不退：“臣正是怕后患成了眼前之祸。”

齐湣王摆手：“够了。寡人不是叫你来替宋说话的。”

这一下，便连愿意替田文缓几句的臣子也都不敢动了。

朝议散后，命令很快发下。

齐东境一支军先行出境，不是大举，只是压近宋边数处要道，逼宋使日内入临淄答命。消息传出去，比兵本身更快。

魏先得报。

魏王看完，冷笑了一声：“宋离齐近，果然最先吃这一压。”

魏相在旁低声道：“齐这是试人。”

魏王把战报往案上一搁：“它每试一次，便替自己多添一层怨。”

说完之后，他却没有立刻再言。

因为他心里也清楚，魏如今不会先替宋出头。

能看明白，未必就肯先动。

同样的消息传到邯郸，赵王看完，先问的不是该不该理宋，而是：“乐毅那边，可知此事？”

近臣答：“当已知。”

赵王点头，没再多说。

可一旁的赵奢却道：“齐这一步，不在宋，在试谁会先当没看见。”

苏秦闻言，转头看了他一眼。

因为这句话，正点在如今诸国最隐的那层病上。

夜里，田文回府，没有召宾客。

他只把今日朝中发下的那道压宋令反复看了两遍，最后轻轻压在案角。

门客问：“君上为何不再争？”

田文答：“今日再争，只会让大王更信旁人，不会更信我。”

“那便看着？”

田文望着窗外临淄灯火，缓缓道：“我不是在看一纸军令。我是在看齐这口气，到底还要再满到什么地步，天下才会一起觉得，再不压它便晚了。”

风过窗棂，吹得灯火一摇。

那道压宋令在案上压得很平。

可田文知道，它落到各国案上的时候，已不只是一道令了。

它是齐替自己，又往高处走出的一步。
