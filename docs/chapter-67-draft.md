# 第六十七回 齐廷外压先向宋地试 田文再谏满朝已顺声

临淄这一日的大朝，比往常多摆了一张边图。

宋地几处城邑、渡口、仓口，都被朱笔圈了出来。图旁压着新到的边报，封泥未干，殿中已有人先笑了。

“宋使又在拖。”

“它还能拖到哪一步？”

“拖来拖去，不过还是想看齐、楚谁先让它靠一头。”

齐湣王坐在御座上，看完边报，手指在图上点了点，正落在宋地北口。

“宋若再不顺，诸侯谁会真替它出头？”

殿里静了一息，随即便是顺着这句话往下说的人。

“赵正顾秦，不会为宋动兵。”

“韩、魏新败，先顾自己还来不及。”

“燕便是有怨，也还要先藏着。”

齐湣王听着，眼里那层亮意慢慢起来。他近来最爱听的，便是这些各国各有顾忌的话。旁人每替他说出一条，他便觉得齐脚下又实了一分。

田文坐在席间，没有出声。

他看见的不是宋一国，而是满殿人说话时那股越来越轻的气。好像山东诸侯都已被摆在案上，一只手便可拨来拨去。

齐湣王把边报合上：“既如此，先敕宋使，限日入临淄。若仍支吾，东境诸军往前压一步，占它两处要道，教它先知轻重。”

殿中数人立刻应诺。

有人已顺着往下算：压宋能逼出多少岁币，能试出魏、赵、楚谁先开口，能把哪几处道口先攥到齐手里。

田文终于出班：“大王，宋固可制，但今日最忌的不是齐强，是齐强而不收。”

齐湣王看向他，面上笑意淡了：“孟尝君又忧什么？”

田文道：“诸侯未必即刻为宋出头，却会把这一笔记下。今日记齐压宋，明日记齐逼韩，后日记齐轻燕。记多了，它们未必是同怒，却会同怕。”

“怕什么？”

“怕齐再往前一步。”

殿里一下安静了。

齐湣王盯着田文，声音已冷：“寡人问的是宋，君却总替天下算心病。若照你这般，齐强也不可用，兵盛也不可试，难道要坐在临淄等旁国先来量齐深浅？”

田文俯身：“臣只怕大王把旁国都逼成一处时，齐自己还当它们各自散着。”

齐湣王没有再让他说下去，抬手便道：“够了。发令。”

殿中执事立刻趋前，捧起印匣。玺印按下去的时候，田文听见那一下闷声，竟像有人把一块重木楔进门缝里。

散朝后，东境军令飞出临淄。

先行的一支军并不多，只取宋边两处路口，一处逼近渡头，一处卡住商车往来的狭道。兵未至，檄已先到，催宋使限日入齐。

消息走得比骑兵还快。

大梁得报时，魏王正在看河上税簿。他把急报看完，丢在案边：“宋离齐近，先吃这一口，不冤。”

魏相低声道：“齐这是试山东诸侯。”

魏王抬手拨了拨案上的铜镇纸：“它每试一次，旁人心里便多一道刻痕。只是不知谁肯先认这道痕。”

他话说完，仍旧低头去看税簿，再没有下文。魏不会先替宋出头，这一点，他比谁都明白。

邯郸那边，赵王看完急报，先问：“乐毅可知？”

近臣答：“燕都当已得报。”

赵王点头，把简递给赵奢。赵奢只看了一遍，便道：“齐压的不是宋，是看谁先当没看见。”

苏秦听见这句，抬眼望了望殿外。

殿外天色很亮，邯郸风也不大，可那一刻，他却像看见山东诸国案头上都压了一封同样的急报，谁也不愿先动，谁也知道不能总这么不动。

夜里，田文回府，没有召门客议事。

他把那道压宋令摊在灯下，看了很久，只将一角压在铜兽镇下。门客在旁问：“君上，明日还入朝争么？”

田文摇了摇头：“明日再争，大王只会更信今日那些顺声。”

“那便由它去？”

田文没有立刻答话。他望着案上那方朱印，忽然抬手，把压宋令往外推了半寸。

“由它去，天下也会看见它去了哪一步。”

灯影落在简牍上，朱印还湿得发暗。
