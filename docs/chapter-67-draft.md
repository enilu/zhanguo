# 第六十七回 齐廷借宋先压两处道口 田文看满朝顺声只把新令压在灯下

临淄这一日的大朝，比往常多摆了一张宋地图。

宋地几处城邑、渡口、仓口都被朱笔圈了出来。齐湣王拆开边报，手指点在宋地北口：“宋若再不顺，诸侯谁会真替它出头？”

这句话一出，满殿顺着往下说的人立刻多了。

“赵正顾秦。”

“韩魏新败。”

“燕有怨，也还要先藏着。”

齐湣王听着，眼里那层亮意越发显出来。到后来，有人已经替他往下算起压宋能逼出多少岁币、能试出谁先开口、又能先把哪两处道口捏在齐手里。

田文坐在席间，一直没出声。

直到齐湣王发话：“先敕宋使限日入齐。若仍支吾，东境诸军先压一步，占它两处要道。”

这时田文才出班：“大王，宋固可制，最忌的却不是齐强，是齐强而不收。”

齐湣王看着他，笑意淡了：“孟尝君又忧什么？”

田文道：“今日记齐压宋，明日记齐逼韩，后日记齐轻燕。记得多了，它们未必一时同怒，却会同怕齐再往前一步。”

齐湣王没有让他再说下去，抬手便令用印。

玺印压下去时，殿里很静，那一声闷响却像格外重。

散朝后，东境军令飞出临淄。先行军并不算多，只取宋边两处路口，一处逼近渡头，一处卡住商车往来的狭道。兵还没到，催宋使限日入齐的檄文已先传出去。

大梁先得报。

魏王看完，把急简丢在案边：“宋离齐近，先吃这一口，不冤。”

魏相低声道：“齐压的不是宋，是试诸侯。”

魏王拨了拨案上的铜镇纸，没有再接话。

邯郸那边得报后，赵奢只看了一遍便道：“齐压的不是宋，是看谁先当没看见。”赵王听完，也没有立刻下令，只把那封急报又压回案上。

夜里，田文回府，把那道压宋令摊在灯下看了许久。

门客问：“明日还入朝争么？”

田文摇头：“明日再争，大王只会更信今日那些顺声。”

他说完，把那道新令往前推了半寸，朱印在灯下还湿得发暗。
