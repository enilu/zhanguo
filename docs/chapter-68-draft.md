# 第六十八回 乐毅持约先见魏韩主 诸侯肯议仍怕先吃亏

齐压宋边的急报到燕都时，乐毅正在改那卷《会兵诸约草案》。

他把急报展开，看了一遍，压在草约左侧。右手却没有停，仍在补最后一条：

“借道之国，不得因通军而空其守境之兵。”

燕昭王看着他落笔，问：“可到时候了？”

乐毅把笔搁下：“可送约，不可催兵。齐这一压，把各国心里的惧和怨都逼出来了，可谁也还不肯先认自己那一份责。”

数日后，燕使再出。乐毅先往大梁。

魏王没有设大朝，只在偏殿接见。殿内人少，案上却摆得极净，显见他是预备认真看这卷简，而不是拿几句场面话打发。

乐毅行礼毕，双手奉简。魏王展开，先见三条：

“前锋、后应，不得临敌始争。”

“后应不得先退。”

“失期者失其先议之利。”

魏王看完，抬起眼来：“乐君带来的，不像盟书，倒像要先拿一根绳，把诸国都捆上。”

乐毅道：“若不先捆住最易散的地方，后面便只有盟名，没有兵步。”

魏王没说话，把简递给魏相。魏相看得很慢，到“后应不得先退”那一条时，指尖在简边停了停。

“若前锋失利，后应仍不得退，那不是替别人把败局全吃下了？”

乐毅道：“若人人都只想做后应，这仗不必起兵，便已先散。”

魏相又问：“若魏为后应，前锋是谁？粮道谁护？退路在哪一段？”

乐毅从袖中再取一卷小简，放到案上：“臣已另列三案。前锋若在赵，魏当接哪一路；前锋若在燕，魏应守哪两段粮道；诸军若未能如期会于一处，各国各罚何等，皆已列明。”

魏王这才真正把身子坐直了些。

他听得出来，燕不是来喊一声“共伐齐”便算尽心，而是把最容易翻脸的地方先摊到了案上。

殿里静了一会儿，只剩竹简翻动的轻响。

魏王最终把草约合上：“此简，寡人留下。”

乐毅俯身：“留简，便是往前走了一步。”

离开大梁后，乐毅转道新郑。

韩王见他，不待寒暄，先问：“伊阙之后，乐君还敢来韩前讲‘后应不得先退’？”

乐毅没有避，仍把草约呈上：“正因伊阙之后，这句话才更不能省。”

韩王展开草约，越看脸色越沉。看至“主议之国若失期，罚重于诸国”时，才抬头道：“你这是把燕自己也先押上了。”

“若臣只来叫诸国认责，自己却留宽路，此约不必再议。”

韩王冷笑一声：“话说得直。可直话最容易伤人。”

乐毅道：“临阵再伤，总比此时先伤好。”

韩王没有接这句，只把草约翻到借道一段：“韩若借道，边上守军留多少？粮车过境，由谁给引？”

乐毅道：“韩若只借道，不借仓，是一案；借道兼借近边仓，是一案；若愿分一部兵守粮道，又是一案。臣不求韩此刻尽诺，但求韩把自己能给、不能给的界线先划明。”

韩王听完，沉默了很久。最终只说：“这卷约，寡人也留。”

出新郑时，天已将暮。道旁风硬，卷得马蹄边的灰土一层层贴着地走。

从人低声问：“魏、韩都未点头，君上这一趟，可算得手？”

乐毅把缰绳微微一勒，回望身后城门：“它们肯把简留下，便不是空跑。如今各国最怕的，不是齐太强，是自己若先应了，后面要不要先吃那口亏。只要这卷约留在它们案上，日后再议，争的就不只是愿不愿伐齐，而是哪一条敢认，哪一条不敢装听不见。”

他说完，抬手把那卷副本压回鞍旁皮囊里。

皮囊口只露出半截竹简，风一吹，简尾轻轻相击，发出细脆一声。
