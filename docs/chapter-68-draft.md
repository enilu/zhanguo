# 第六十八回 乐毅持约先见魏韩主 诸侯肯议仍怕先吃亏

齐压宋边的消息传到燕都时，乐毅没有说话，只把那份急报摆到《会兵诸约草案》旁边。

两卷简并排躺在案上，一卷是齐自己往高处走的脚印，一卷是燕准备拿来接这些脚印的绳。

燕昭王看完之后，问：“可到时候了？”

乐毅道：“可先送约，不可先催兵。”

于是数日之后，燕使再出。

这一次带去的，不再只是试问辞色的空函，而是那卷乐毅亲手定过的《会兵诸约草案》。

他先见魏。

魏王在偏殿接见，没有摆满朝。

因为他也知道，这种简一旦摆上大朝，人人先想的不会是如何成事，而是怎样先把自己最怕吃亏的那一面遮住。

乐毅行礼毕，先把简奉上，并不多说。

魏王展开一看，看到头几条，眉头便慢慢皱了起来。

“主帅不得二出。”

“后应不得先退。”

“失期者失先议之利。”

看完之后，魏王把简放下，第一句便问：“乐君，你这是来结盟，还是来替各国先立军法？”

乐毅答：“若无这些，后面纵有盟名，也不过是一群都怕自己先吃亏的人同站一阵。”

魏王没有立刻接。

因为这话说得太透。

透得叫人不好装没看见。

魏相在旁缓缓道：“乐君草约里，旁的都还可议，唯独这句‘后应不得先退’，最难。若前锋真失，而后应仍死顶，岂不成了替别人吃尽？”

乐毅看向他，答得极平：“正因这句话最难，才不能不先写。若人人都只愿做后应，战前便已输了。”

魏相被这一句堵得沉默了片刻，才道：“可魏若为后应，也需知前锋是谁，退路何在，粮道由谁护。”

乐毅点头：“正该如此。所以臣今日送来的不是盟誓，是草约。魏若有真难处，尽可逐条改。臣所怕的不是难，是把最难的那一句留到阵前再争。”

魏王听到这里，目光才第一次真正沉下来。

因为他听得出来，燕这回不是来借势喊一句“共伐齐”，而是真准备把那件最叫人头疼、也最能成事的事，一条条先做实。

魏王最终没有应允，只道：“此约，寡人要留。”

乐毅俯身：“留约便是好事。”

离魏之后，乐毅再往新郑。

韩王看这卷草约，比魏还快。

因为伊阙之败后，韩对“主次”“先后”“后应不得先退”这些字，比谁都敏。

他一条条看过去，脸色越来越沉，最后忽然道：“乐君，你这卷简里写的，全是败过一回的人才会真在意的东西。”

乐毅答：“正因败过，才知道哪些不先写死，后头连怨谁都怨不明白。”

韩王沉默许久，才道：“燕若后面真推此盟，谁主？”

乐毅没有闪：“若能成，燕愿主前议，不敢擅主诸国生死。真到用兵，仍须看诸国共定主帅。”

韩王冷笑了一声：“这话说得倒谨慎。可谨慎的人，多半也最会替自己留后手。”

乐毅听了，并不辩，只把草约翻到后头一条，指给韩王看：

“借道之国得留守土之兵，不得尽空；主议之国若先失期，罚重于诸国。”

韩王看完，微微一顿。

因为这条分明是在说，若燕起议而后时，罚要更重。

它至少把“你来劝别人，却先给自己留宽路”这层嫌疑，先压掉了半分。

韩王缓缓把简放下，道：“这卷约，寡人也留。”

乐毅离开新郑时，天已近暮。

随从低声问：“君上，魏韩都未应允，这算不算空跑？”

乐毅骑在马上，望着前路，声音极稳：“它们留约，不算空。如今诸国最怕的不是约太严，是自己若先点头，后面会不会先吃那口亏。只要这卷简留下，后面争的就不再只是愿不愿，而会慢慢变成哪一条该改、哪一条不敢不认。”

风自道旁吹来，卷得尘土细细贴地。

乐毅回头看了一眼身后。

魏、韩都没给痛快话。

可他心里却反倒比来时更定。

因为真正能走到兵前的事，本就不是一见面便人人拍案叫好。

它往往是从这些皱眉、沉默、留简不决里，一寸寸长出来的。
