# 第六十八回 乐毅先持冷约见魏韩之主 诸侯肯留简时却都先问谁吃重锋

齐压宋边的急报到燕都时，乐毅正在改那卷《会兵诸约草案》。

他把急报压在左手边，右手仍把最后一条补完：

“借道之国，不得因通军而尽空守境。”

燕昭王看着他落笔，只问一句：“可到时候了？”

乐毅答：“可送约，不可催兵。”

数日后，燕使再出，乐毅先往大梁。

魏王没有开大朝，只在偏殿接见。乐毅呈上草约，魏王展开，先看到的便是：

“前锋、后应，不得临敌始争。”

“后应不得先退。”

“失期者，失其先议之利。”

魏王看完，抬起眼来：“乐君带来的，不像盟书，倒像先拿绳索把诸国都捆上。”

乐毅道：“若不先捆住最易散的地方，后面便只剩盟名，没有兵步。”

魏相接过竹简，看得很慢，翻到“后应不得先退”时停了停：“若前锋失利，后应仍不得退，那不是替别人把败局全吃下了？”

乐毅从袖中再取一卷小简，平码在案上：“若前锋在赵，魏当接哪一路；若前锋在燕，魏该守哪两段粮道；若诸军失期，各国罚到何处，臣都列了三案。”

魏王这才把身子坐直了些，末了只说一句：“此简，寡人留下。”

离开大梁后，乐毅转去新郑。

韩王见他，不待寒暄，先问：“伊阙之后，乐君还敢拿‘后应不得先退’来见韩？”

乐毅没有避，双手奉简：“正因伊阙之后，这句话才不能省。”

韩王冷着脸往下看，看到“主议之国若失期，罚重于诸国”时，抬头道：“你这是把燕自己也先押上了。”

乐毅道：“若燕只叫诸国认责，自己却留宽路，此约不必再议。”

韩王听完没有再讥，只把竹简翻到借道一段：“韩若借道，守军留多少？粮车过境，由谁给引？”

乐毅答：“韩只借道，是一案；借道兼借仓，是一案；若再分兵守粮道，又是一案。臣不求韩此刻尽诺，只求韩先把能给和不能给的界线划明。”

韩王沉默良久，终于也说：“此简留下。”

出新郑时，天已将暮。

从人低声问：“魏、韩都未点头，君上这一趟，可算得手？”

乐毅把那卷副本压回鞍旁皮囊：“它们肯把简留在案上，后面再议时争的就不只是愿不愿伐齐，而是哪一条敢认，哪一条不敢装听不见。”

说完，马已出城。

风卷着道边灰土贴地走，皮囊里露出的半截竹简被吹得轻轻一响。
