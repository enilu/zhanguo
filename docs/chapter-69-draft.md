# 第六十九回 邯郸对约赵王先问实 赵奢论兵不肯吃空名

乐毅入邯郸时，齐压宋边的消息已经传开了。

街上议论虽多，城里却并不喧。仿佛人人都在等另一封更重的信来，看看山东这盘棋，是不是当真要换个下法。

赵王这次没有设偏宴，直接开了小朝。

在座者不过数人：苏秦、赵奢、廉颇，以及两名掌书记与一名司仓官。乐毅入殿时，先看见的不是酒器，而是地上铺开的赵东境道图。

他心里一沉，知道赵王今日要的不是辞色，是细数。

《会兵诸约草案》呈上后，赵王先看卷首三条，看到“失期者失其先议之利”时，停了片刻，方道：“乐君，你这卷约，比盟书更像军令。”

乐毅答：“若真要成事，最好先按军令看。”

赵王没有再说，把简先递给廉颇。

廉颇看得很慢，时而抬眼看乐毅，时而低头看简。到末了，才把竹简往案上一搁：“这卷东西，好在不好看。”

赵王问：“为何？”

廉颇道：“样样都好看的，临阵都能翻。它如今难看，说明里头写的，都是到阵前真会咬人的。”

殿中无人接这句。赵奢已经把草约拿过去，看得更细。他不看那些辞句起落，只盯借道、粮期、驻军、先后四段。

看至中段，他忽然问：“燕若主议，借赵道多深？兵车几日一转？粮由燕自继几程，又欲借赵近边几仓？”

乐毅没有迟疑，从袖中再取一卷小简：“臣另列了三案。赵若只借道，是一案；借道兼借两仓，是一案；若后面愿共前议，前锋、后应如何分配，又是一案。”

赵奢把小简展开，只看了几行，便伸手把赵东境道图扯近了些。

他按着图，一处处往下问：

“这里的坡道，辎车能不能过？”

“这段若遇秋雨，几日泥死？”

“燕骑若早于粮车半程，谁护后队？”

乐毅一条条答，既不夸，也不躲。

司仓官在旁听着，已经取笔记在空牍上。记到第三行时，抬头望了乐毅一眼，像是没想到燕使来赵前，竟把这些也先算到了。

苏秦从旁看着，没有插话。

他看着赵奢、乐毅两人一图一简地往下钉，忽然想起自己当年游说诸侯时，席上最重的是一句“可共”。到了今日，邯郸这殿里最重的，却成了“几日、几仓、几程”。

赵王等众人问得差不多了，才抬头道：“赵若听此约，得什么？”

乐毅道：“东边若真起兵，齐不能再把赵只当一国看；西边秦也会因此多顾几分。赵得的不只是名，也是一段可用的缓气。”

赵王仍不松口，只转问赵奢：“你看呢？”

赵奢把两卷简都合上，手掌按在上头：“臣只认一件事。赵不能为好名先把自己顶到最前，也不能嘴上说共兵，心里却处处留最宽的退路。若人人都这样，此约不必到阵前，路上便先烂了。”

廉颇也道：“可留，可再议。只是还得再看齐会不会继续往前压，也得看诸国谁敢把难看的那几条先认下来。”

赵王听罢，终于开口：“赵可留约。若后面诸国都只改宽己之条，不改重己之责，赵不进。若真有人肯把自己那一头也先写重，再来邯郸。”

乐毅俯身：“臣记下了。”

散朝时，天色已偏。乐毅出宫，苏秦随他并行到宫门外。

宫门外有风，吹得两人衣袂微微向一侧贴去。苏秦忽然道：“乐君今日这一卷简，倒叫我想起旧事。”

乐毅停步：“相国请讲。”

苏秦看着城头暮色，慢慢道：“从前最难的是叫人肯应。如今更难的，是叫人应了之后，还肯把自己最不愿先担的那一口，也一并认下来。”

乐毅点头：“所以臣才先写在简上。”

苏秦笑了笑，笑意很淡：“你们后来做事的人，比我们当年说话的人，更不讨人喜欢。”

乐毅也笑：“若只求讨喜欢，兵就走不到临淄。”

两人说完，便各自止步。乐毅上马前，低头看了看手中副本，指尖在“失期”二字上轻轻按了一下，才把竹简收回囊中。
