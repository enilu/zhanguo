# 第六十九回 乐毅持约入邯郸先过兵图 赵奢问到几仓几程不肯只听大义

乐毅到邯郸时，齐压宋边的消息早已传开。

赵王这次没设偏宴，直接开了小朝。殿上人不多，苏秦、廉颇、赵奢都在，地上铺着赵东境道图，案边压着仓簿和转运簿。

乐毅呈上《会兵诸约草案》，赵王翻到“失期者失其先议之利”时停了一下：“乐君这卷，倒不像盟书，更像军令。”

乐毅答：“若真要走到阵前，最好先按军令看。”

赵王把简递给廉颇。

廉颇看得很慢，翻到借道、前军、后应几段时，眼睛一直没离开竹简。看到末尾，他才把简放回案上：“这卷东西，好在不好看。”

赵王问：“为何？”

廉颇道：“越不好看，越像到阵前真会咬人的。”

这时赵奢已把后面附的小简抽了出来。

他不看辞句，只看数字与道线：“赵若借道多深？粮车几日一转？燕粮自继几程，又欲借赵近边几仓？”

乐毅又取一卷小简，平码在图边：“赵只借道，是一案；借道兼借两仓，是一案；若后面共前议，前军后应如何分，是第三案。”

赵奢把道图拖近，沿着几处险口一一往下问：“这里秋雨后车能不能过？这段泥死要几日？燕骑若早于粮车半程，谁护后队？”

乐毅一条条答，没有夸口。

掌书记已在旁边低头记了起来，司仓官也把几处仓名另写在一片空牍上。殿里没人说大话，只有竹简翻动和笔尖刮木的轻响。

问到最后，赵王才开口：“赵若听此约，得什么？”

乐毅道：“齐若真被东边牵住，赵西面便多一口缓气。得的不只是名，还有时。”

赵王不接，转问赵奢。

赵奢把两卷简都合上，手掌按在上头：“赵不能为好名先顶到最前，也不能嘴上说共兵，心里处处留最宽的退路。若人人都这样，这卷约走不到临淄，路上便先烂了。”

廉颇也道：“可留，可再议。只是还要看诸国谁先把自己最难看的那几条认下来。”

赵王这才点头：“赵可留约。若诸国只改宽己之条，不改重己之责，赵不进。若真有人肯先把自己那一头写重，再来邯郸。”

散朝时，天色已偏。

苏秦送乐毅到宫门外，两人并行到风口，衣袂都被吹得往一侧贴去。苏秦看着城头暮色，忽然道：“从前最难的是叫人肯应。如今更难的，是叫人应了以后，还肯把自己最不愿先担的那一口也认下来。”

乐毅点头：“所以臣先写在简上。”

说完，他低头把副本收回囊中，指尖正好压在“失期”二字上。
