# 第六十九回 邯郸对约赵王先问实 赵奢论兵不肯吃空名

乐毅入赵时，邯郸已知齐压宋边。

城中议论不少，却没有哪一处闹得很响。人人都像看见了什么，却又都还在等别人的下一步。

赵王这次没有把乐毅请到偏殿，而是直接召了小朝。

在座者不多：

苏秦，

赵奢，

几名近臣，

以及新自边上回都的廉颇。

乐毅把《会兵诸约草案》呈上时，赵王没有急着翻后面城地与借道的细条，反而先看最前面那几句：

“主帅不得二出。”

“前锋、后应不得临敌始争。”

“失期者失其先议之利。”

看完之后，赵王抬头：“乐君，你这卷约，比盟书更像军令。”

乐毅答：“若后面真要成事，最好它先像军令。”

赵王点了点头，没有立刻表态，只把那卷简递给廉颇。

廉颇看得很慢。

看完之后，只有一句：“这卷东西，好在不好看。”

殿中几人都微微一顿。

赵王问：“何意？”

廉颇道：“若样样都好看，便是样样都能在后面翻。如今它不好看，才说明这里头写的是阵前真会咬人的东西。”

赵奢接过草约，看得比廉颇还细。

他看到借道与粮期那几条时，眉头微皱，忽然问乐毅：“燕若主议，借赵道多深？粮由燕自继几程，又欲借赵仓几处？”

乐毅早有准备，直接从袖中取出另一卷小简，递上：“臣已另列三案。若赵只肯借道，不肯借仓，是一案；借道兼借近边两仓，是一案；若赵后面愿共前议，再是一案。”

赵奢把那卷小简展开，眼神终于真正沉了下来。

因为这里头写的，已不是泛泛之论，而是几日行程、几站转粮、几处可驻、几处不可过。

他看完之后，才道：“乐君这次倒真不是来讲义气的。”

乐毅答：“义气若无这些托着，走不到临淄城下。”

苏秦坐在一旁，从头听到这里，心里竟生出一种极复杂的感触。

自己当年合纵，也曾一国国去求一口可共的大义；

到了今日，乐毅在做的，却更像是把那些大义拆开，一寸寸换成各国都得先认的难听实话。

不是谁高谁低。

只是时代已真的换了问法。

赵王沉吟许久，问得更直：“赵若听此约，得什么？”

乐毅答：“得一件事：齐若继续高压，后面不至人人都只在自己案上冷看。”

赵王道：“寡人问的不是这层。”

乐毅抬眼，道：“若问更实处，赵得的是：东边若真起兵，齐不能再把赵只当一国看；西边秦也会因此多顾几分。赵可得缓，亦可得势。”

赵王没有说好，也没有说坏，只转头问赵奢：“你看呢？”

赵奢把两卷简都合上，道：“臣只看一条。若真走此约，赵不能为好听之名先把自己推到最前，却也不能只图后利、处处都留最宽的退路。若那样，此约不用到临敌，路上便先烂了。”

廉颇也道：“可听。但真动之前，仍得再看齐会不会继续压，诸国会不会真把最难看的那几条认下去。”

赵王听完，终于对乐毅道：“赵可留约，也可再议。若后面诸国都只改宽己之条，不改重己之责，赵便不进。若真有人肯把自己那头也写重，赵再往前走。”

乐毅俯身：“臣所求，正是这一句。”

散朝之后，苏秦与乐毅并行出宫。

风不大，邯郸城头日色正薄。

苏秦忽然道：“乐君今日这卷约，让我想起一件事。”

乐毅转头：“相国请讲。”

苏秦缓缓道：“天下从前最难的是叫人肯应。如今最难的，却是叫人应了之后，还肯把那些自己最不愿先担的东西也一并认下来。”

乐毅点头：“所以才更要先写。”

苏秦笑了笑，笑意很淡：“你们这些后来做事的人，比我们当年说话的人，更不讨人喜欢。”

乐毅也笑了一下：“若只求讨喜欢，后面就到不了兵前。”

两人并行到宫门外，便各自止步。

谁都没有再多说。

可邯郸这一回留下的，也已不再只是一卷“伐齐可不可”的空议。

它变成了另一个更沉的问题：

若真要并兵，赵到底肯先把哪一口难咽的东西，自己吞下去。
