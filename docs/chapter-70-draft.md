# 第七十回 楚廷闻齐压宋各自算 黄歇初试先劝莫冷看

齐压宋边的急报入郢都时，怀王正在看一卷东境旧报。

他把两卷简并在一处，看了片刻，忽然冷笑一声：“齐终究还是忍不住了。”

这话里竟带着一点说不清的快意。楚这些年看齐，最难受的一层，便是明知齐有骄气，却总等不到它真把那一步踏出来。如今齐自己先去压宋，怀王像是终于等着了这一记实锤。

朝议一开，殿里果然立刻分成两路。

一路说，齐既失德，楚正可借势责之，在东面收几分人心；

一路却说，齐既自高，楚更不必急动，只管居中观变，等燕、赵、魏、韩谁先去撞这一头。

怀王听到“居中观变”四字，手指在案上轻轻点了点，显然是听进去了。

屈原立在一侧，正要出班，昭阳却先向前半步：“大王，观可以，备不可后。东面兵储若仍旧簿不清、仓数不明，后头便是想选路，也无从下脚。”

怀王眉头一皱：“将军又要说粮？”

“兵未必即出，粮却不能临时问。”

殿里有人接声道：“齐压的是宋，不是楚。楚何苦先把自己绷紧？”

这时黄歇才出列，先向昭阳微一拱手，接着俯身道：“臣以为，楚可观，不可冷看。”

怀王抬眼：“如何叫不冷看？”

黄歇答：“先把该说的话说出去，先把该看的簿册看明白。若后头山东真变，楚至少不至于还坐在案前问自己能出多少兵、站哪一道线。”

殿中有人冷笑：“说到底，还是要楚替旁人操心。”

黄歇转身看去：“不是替旁人操心，是不叫旁人都看出楚只会等别人先动。”

这一句落下，殿中静了片刻。

怀王脸上那层方才还带着几分闲意的神色，也收了些。他望着黄歇：“若楚先责齐，齐反来讥楚这些年又何曾少算，如何应它？”

黄歇道：“由它去讥。楚先把自己这一回站住，比与齐争口舌更要紧。”

怀王没有立刻答。屈原看着案前几人，也不再出声。他知道昭阳方才先把“备”压上桌，黄歇再把“辞”接上去，这样的话，怀王比直接逼他选边更听得进去。

半晌，怀王终于抬手：“兰台拟书，责齐压宋。昭阳，会东境兵储。黄歇随看旧簿，把这些年拖着不定的地方一一圈出来。”

命令一下，殿中众人神色各异。有人觉得不过是做个样子，有人却知道，只要兵储簿册当真被翻出来，楚廷许多旧病便再藏不住了。

散议后，屈原与昭阳并肩出殿。走到廊下，屈原道：“今日你先出那一步，倒比我合适。”

昭阳道：“大王最怕人把话说成墙。先让他看簿，看完了，许多话自然无处可躲。”

屈原点了点头，没有再说。

夜里，昭阳在官署开了灯，命人把东面仓簿、转运簿、守军点册都搬来。黄歇坐在下首，只管把旧簿一卷卷展开，对着新报去核。

核到二更，案上已分出三摞：

可用的，

含混的，

必须重查的。

昭阳把其中一卷“必须重查”的册子拍在案上，脸色发沉：“这一处仓数，去年写八千石，今岁却还写八千石，连缺耗都没改。下面的人，是拿楚国当死簿养着么？”

黄歇没有接怒，只把另一卷推过去：“这里守军点册与转运簿对不上。兵数若按旧册，车不够；车若按新报，兵又少了三百。”

昭阳看完，伸手把两卷都压住，半天没说话。案头油灯燃得很稳，只把簿角照出一道发白的边。

过了许久，他才吩咐：“把这两卷另封，明日一早带入宫。”

黄歇应了一声，伸手去取封泥。封泥按上去的时候，发出轻轻一响，像是把楚廷这些年一直拖着不看的那道口子，先堵住了一半。
