# 第七十回 楚廷闻齐压宋各自算 黄歇初试先劝莫冷看

齐压宋边的消息入郢都时，怀王先是冷笑了一声。

“齐终究忍不住了。”

这一句里，竟还带着几分说不清的快意。

因为这些年楚国看齐，心里最难受的一层，便是明知齐也有骄、有躁、有急于多算的时候，却总苦于它还未真把那一步走出来。如今齐自己先压宋边，在怀王看来，像是总算有人替楚把这口憋着的闷气说穿了一点。

朝议很快开了。

果然，一开口便是两路声音。

一路说，齐既压宋，正是失德，楚可借势责之，也可顺手在东面多收几分人心；

另一路却说，齐既自高，楚更不必急着动，且看燕、赵、魏、韩谁先出头，楚正好居中观变。

怀王听到“居中观变”四字时，神色明显缓了一下。

因为这句话，正合他这些年最爱替自己留的一条路。

屈原在旁见状，心里先是一沉。

他正欲出班，黄歇却先一步走了出来。

这是他近来入朝后，第一次在外局大议上先发言。

怀王看了他一眼：“黄歇，你也有话？”

黄歇俯身：“臣有。”

“说。”

黄歇道：“臣以为，齐压宋，楚固可观；但不可冷看。”

殿中有人立刻皱眉：“观而不冷看，有何分别？”

黄歇转身看向那人，答得很稳：“分别在于，若楚只把此事当成别人先出头、自己可在旁省力的机会，那后面不论诸侯是合是散，楚都还会习惯性地等别人先把势做出来，再择利而入。这样做一次两次，看着都聪明；做久了，外人便知楚永远有后手，也永远不肯先把心摆出来。”

这几句话一落，殿中顿时安静。

因为它说的已不只是齐压宋，而是这些年楚最难看的老病。

怀王脸上那层方才才松下来的气，也一下收住了。

黄歇没有停，又道：“臣不是请大王立刻出兵。臣是请大王至少先做两件事：其一，对外之辞要明，齐此举不可不责；其二，对内之备要实，若后面山东真有大变，楚不可还是临到跟前才问粮几何、兵何从、自己到底站哪一线。”

昭阳在旁听着，缓缓点头。

因为这两句话，既没逼怀王立刻选死一条路，也没顺着怀王最舒服的那层“且看别人”往下滑。

怀王沉默片刻，问：“若楚先责齐，齐反问楚这些年何曾少算，又当如何？”

黄歇道：“那便由齐去问。楚先把自己这一回站稳，比与齐争口舌更要紧。大王若连该说的也不先说，只等看谁先替楚说，外人记的便不是齐错，而是楚冷。”

这话很平。

可怀王听完之后，指尖在案上轻轻敲了两下，竟没有立刻驳回。

屈原站在一旁，看着黄歇背影，心里第一次真正生出一点“此人后面能担事”的认定。

因为他看得出来，黄歇最有用的地方，不是看得比别人更深，而是他总能把最该说的那一句，放在怀王还听得进去的位置上。

午后，怀王果然命兰台拟书责齐，同时命昭阳与黄歇会看东面兵储，不许再拖。

命令不算多重。

却毕竟不是“只冷看”。

散议之后，屈原与黄歇并行出殿。

屈原道：“你今日替大王留了脸。”

黄歇摇头：“臣只是没把话先说成一堵墙。”

屈原看了他一眼：“你可知，有时墙才挡得住风。”

黄歇答：“臣知道。可怀王如今一见墙，先想的多半不是挡风，而是转身。”

屈原听后，没有再说什么。

因为他知道，这正是楚眼下最难、也最真的一层。

夜里，黄歇与昭阳同看东面兵储旧簿。

一卷卷翻过去，许多旧毛病仍在：

该补未补，

该明未明，

该定主次的地方仍写得含混。

昭阳看得脸色发沉，半晌才道：“楚这几年，果然不是只在外头摇。”

黄歇把其中一卷按住，道：“外头怎么摇，多半先是里头自己松。”

窗外夜色正深。

郢都宫灯之下，楚国终于也开始有人不再只把齐、秦、赵、燕、魏、韩怎么动挂在嘴上。

他们开始把目光重新压回楚自己案前那些最沉的简上。

这一层力虽还轻，

却毕竟已经落下去了。
