# 第七十回 郢都闻齐压宋先翻东境旧簿 黄歇第一次入局只劝楚莫冷看

齐压宋边的急报进郢都时，怀王正在看一卷东境旧报。

他把两卷简并到一起，看了片刻，冷笑了一声：“齐终究还是忍不住。”

朝议一开，殿里很快分成两路。有人说齐既失德，楚正可借势出言；也有人说齐既先高，楚更不必急动，只管居中看谁先去撞它。

怀王听到“居中观变”四字，手指在案上轻轻点了点。

昭阳先出列，没谈辞色，只道：“大王，观可以，备不可后。东面兵储若还是旧簿旧数，后头便是要选路，也无从下脚。”

殿里立刻有人接声：“齐压的是宋，不是楚。楚何苦先把自己绷紧？”

黄歇这时才出班，先向昭阳拱手，再向上道：“楚可观，不可冷看。”

怀王抬眼：“如何叫不冷看？”

“先把该说的话说出去，先把该看的簿册翻明白。”黄歇答，“若后头山东真变，楚至少不至于还坐在案前问自己能出多少兵、站哪一道线。”

怀王听完，脸上的闲意收了一些。他看着黄歇，又看一眼昭阳，末了才抬手：“兰台拟书，责齐压宋。昭阳，会东境兵储。黄歇随看旧簿，把含混不明的地方一一圈出来。”

散议后，当夜官署便点起了灯。

昭阳命人把东面仓簿、转运簿、守军点册全搬到案前。黄歇坐在下首，对着新报一卷卷核。核到二更，案上已分成三摞：可用的、含混的、须重查的。

昭阳翻到一卷仓簿，脸色当场沉下去：“这一处去岁写八千石，今岁还写八千石，连缺耗都没改。”

黄歇没接怒，只把另一卷推过去：“这里守军点册和转运簿对不上。兵数若按旧册，车不够；车若按新报，兵又少了三百。”

昭阳看完，把两卷都压在手下，隔了片刻才吩咐：“另封，明早带入宫。”

黄歇应声，伸手去取封泥。

封泥按上去时，发出很轻的一下闷响，案边油灯却仍烧得很稳。
