# 第七十一回 魏廷改约先争后应责 乐毅回燕知局已入深

乐毅送出的草约，很快开始回来了。

不是回以允诺。

是回以改字。

第一份送到燕都的，是魏改。

魏王没有把整卷都改得面目全非，只在最要命的几处落了笔：

“后应不得先退”，改成“前锋未溃，则后应不得先退”；

“失期者失先议之利”，改成“失期有故者，听诸国共议其责”；

又另添一条：

“前军若擅进而败，不得尽坐后应之责。”

这几笔一落，整卷简的气便立刻变了半分。

燕昭王看完，问乐毅：“这算退，还是进？”

乐毅答：“算真进。”

“何以见得？”

“因为魏已不再只问可不可，而是在问怎么不替别人白吃亏。”乐毅道，“只要它肯把这句话摆上简，后面便能再往下争。怕的是一句都不改，只说空听。”

郭隗在旁也道：“魏这几笔，看着像自护，其实比满口‘愿共大事’更重。”

乐毅点头，却没有立刻露喜。

他心里清楚，改约一开，局便更深了。

因为从这一刻起，各国要争的，不再只是站不站这一边，而是谁肯先把最难看的责任，认到什么地步。

同日，韩改也到了。

韩王的笔，比魏还直。

他把“主帅不得二出”旁边加了一行小注：

“主帅既定，前锋、后应、左次、右次之责，当日俱定，不得再议。”

又在“借道之国得留守土之兵，不得尽空”后面添道：

“借道若危本国要害，则先议代道之法，不得强之。”

这意思很明白：

韩肯议，

但绝不肯再做那个到阵前才知自己究竟该站哪一口锋的人。

乐毅把魏、韩两份改约并排压在图上，看了很久，最后才道：“局进了。”

燕昭王问：“可要再出使？”

乐毅道：“要。但这一次，不是再去劝，是去对改。”

说完，他亲自提笔，在魏改、韩改旁一条条作答。

有的可让，便让；

有的不可让，便在后面加得更细。

尤其是“前锋未溃，则后应不得先退”这一条，他在旁添了八个字：

“前锋若溃，退亦有次。”

郭隗看见，抬头问：“为何还要补这一句？”

乐毅道：“因为真到阵前，最易乱的不是能不能退，是谁先乱退。若这句不补，魏以为自己把后应这层保住了，后面却还是会在‘可退’二字上再争。”

这话说完，连燕昭王都沉默了片刻。

因为他听得出来，乐毅如今做的，已不是合纵家那种先聚一口势的事。

他是在替后来那场大军，把所有最容易出人命、也最容易出裂缝的地方，一刀刀先刻出来。

夜里更深时，赵回书也到了。

赵王没有在原简上多改，只另附一书，只有一句意思：

“赵观诸国先改其责，后论所出。”

赵奢则另列一份粮道小简，写得极细，问的全是几日转运、几处可驻、若齐先动南线则赵道该如何改避。

乐毅一份份看完，唇边才终于有了极淡的一点松。

随从在旁低声道：“君上，如今可算诸国真动了？”

乐毅把几份改约慢慢合上，道：“还不算动兵。”

“那算什么？”

“算它们终于肯把自己最怕的那口亏，拿出来摆在灯下讲了。”

他说到这里，抬眼望向殿外。

夜色正沉，燕宫灯火却比平日亮得更久。

因为案上这些改过的字，已不只是纸上斟酌。

它们每一笔后面，都连着一国到底肯不肯真往前走一步。
