# 第七十一回 魏廷改约先争后应责 乐毅回燕知局已入深

乐毅送出的草约，先后都回来了。

回来的不是允诺，是改字。

第一份送到燕都的，是魏改。魏王没有大改全卷，只在最紧的几处落了笔：

“后应不得先退”，改成“前锋未溃，则后应不得先退”；

“失期者失先议之利”，改成“失期有故者，听诸国共议其责”；

又在后面添了一句：

“前军若擅进而败，不得尽坐后应之责。”

燕昭王看完，把简搁在案上：“这是退，还是进？”

乐毅道：“是进。”

“为何？”

“因为魏已不再问可不可伐齐，只问自己怎么不替旁人白吃那口亏。”

郭隗在旁道：“肯把这话写进简里，便比满口愿共大事更真。”

乐毅点头，把魏改压在地图一角。没过多久，韩改也到了。

韩王的笔更直。他在“主帅不得二出”旁补了一行：

“主帅既定，前锋、后应、左次、右次之责，当日俱定，不得再议。”

又在借道一段后添道：

“借道若危本国要害，则先议代道之法，不得强之。”

燕昭王看见这里，抬头道：“韩也怕到阵前才知自己站哪一道口。”

乐毅道：“怕得越明，越能往下议。”

他说完，亲自提笔，在魏改、韩改旁逐条作答。可让的，便让；不可让的，便加细。

写到“前锋未溃，则后应不得先退”时，他停了一停，在后面添了八个字：

“前锋若溃，退亦有次。”

郭隗问：“为何还要补这句？”

乐毅答：“真到阵前，最易坏的不是能不能退，是谁先乱退。若不把退法先写清，后面还要争。”

燕昭王听着，没有作声，只看着乐毅一笔笔往下改。案上竹简渐多，灯影也压得更低，仿佛一支还未出门的大军，已先在这几卷简里把绊脚的地方露了出来。

夜深时，赵回书也到了。

赵王没有多改原简，只另附一书：“赵观诸国先改其责，后论所出。”

随书同来的，还有赵奢一卷小简。简上不谈气节，只问粮道：

几日一转，

几站一歇，

若齐先动南线，赵道该从哪一口改避。

乐毅把魏、韩、赵三份并在一处，看了很久，才把最上面那卷慢慢合起。

从人低声问：“君上，如今算是诸国真动了么？”

乐毅道：“还没动兵。”

“那算什么？”

“算它们肯把自己最怕吃的那口亏，摆到灯下来说了。”

他说完，把那几卷改约都移到灯前。火光落在竹简新添的字上，墨色有深有浅，像是各国都把最不愿轻易出口的话，终于写出了半截。
