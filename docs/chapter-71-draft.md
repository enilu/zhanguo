# 第七十一回 魏韩赵三国回简先改责条 乐毅连夜补字把退路也写成军次

乐毅送出去的草约，先后都回来了。

回来的不是允诺，是改字。

第一份到燕都的是魏改。

魏王没大改全卷，只在最紧的几处落了笔：“后应不得先退”，改作“前锋未溃，则后应不得先退”；“失期者失先议之利”，改作“失期有故者，听诸国共议其责”。后面还添了一句：“前军若擅进而败，不得尽坐后应之责。”

燕昭王看完，把简往案上一放：“这是退，还是进？”

乐毅答：“是进。”

“为何？”

“因为魏已不问可不可伐齐，只问自己怎么不替旁人白吃那口亏。”

没过多久，韩改也到了。

韩王在“主帅不得二出”旁补了一行：“主帅既定，前锋、后应、左次、右次之责，当日俱定，不得再议。”又在借道后添道：“借道若危本国要害，则先议代道之法。”

乐毅把魏改、韩改并排摊开，提笔逐条作答。可让的便让，不可让的便往细里补。写到魏改那句“前锋未溃，则后应不得先退”时，他停了停，在后面添了八个字：

“前锋若溃，退亦有次。”

郭隗在旁看见，问：“为何还补这一句？”

乐毅道：“真到阵前，最容易坏的不是能不能退，是谁先乱退。”

说话间，赵回书也到了。

赵王没有多改原简，只另附一书：“赵观诸国先改其责，后论所出。”随书同来的，还有赵奢一卷小简，简上只问几日一转、几站一歇、齐若先动南线时赵道该从哪一口改避。

乐毅看完三国回简，没有立刻卷起，只把灯往前拨亮了半寸，又摊开新竹片，继续往下添字。

从人站在一旁低声问：“君上，如今算是诸国真动了么？”

乐毅道：“还没动兵。”

“那算什么？”

“算它们肯把自己最怕吃的那口亏摆到灯下来说了。”

他说完，把三卷改约都移到灯前。

火光落在新添的字上，墨色有深有浅，一行行都还没干透。
