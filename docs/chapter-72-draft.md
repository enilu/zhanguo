# 第七十二回 临淄再压宋使田文冷 郭隗闻报知齐又高

宋使入临淄后，齐廷对它的那一压并没有停。

这一回，齐湣王没有先问辞色，先问边地；问完边地，又问贡数；最后索性当殿翻起宋近年在齐、楚之间摇摆的旧账，一桩桩点给满朝听。

宋使站在殿下，汗已透背，却还想拿“楚意未明”“魏亦有言”来缓。

齐湣王当场打断：“宋若真知所依，今日到寡人案前，便不该还拿他国之名作遮。”

殿中一片附和。

有人说宋素来反覆，本该重责；

有人说不如此，不足示山东诸侯；

还有人顺势道：“今诸国各自有病，大王若不趁此时定山东轻重，后面反教旁人借局自大。”

田文坐在席中，一句未发。

朝会走到这一步，他再开口，也不过是在满殿顺声里多添一个逆耳的人罢了。可他看着殿下那宋使发白的脸，又知道今日这一逼，逼的早不只是宋。

散朝后，宋使几乎是被逼着退出殿去的。

消息离开临淄，比人还快。

大梁得报时，魏王看完，只把急报往案上一扔：“齐如今不是问宋服不服，是问天下谁还敢拿旁人的名字挡它。”

魏相道：“如此一来，燕那卷约，怕要更紧了。”

燕都同样很快得报。

郭隗拿着急报去见乐毅时，乐毅仍在灯下对魏、韩、赵三份改约，一条条补字。郭隗把急报放到他案边：“齐又高了一步。”

乐毅展开，只见几行短报：

“宋使受迫于殿。”

“齐言不许再借他国之名自遮。”

“满朝附和。”

他看完，把急报压在宋地那一点上，才道：“好。”

郭隗一怔：“齐更压人，你反说好？”

乐毅伸手点了点图上宋边渡口：“齐越这样压，越是在替诸国选边。它不许宋居中，旁国便也更难总说此事与己无关。”

郭隗听完，没再作声。

当夜，燕昭王召乐毅入宫。

王问：“齐压宋使，你怎么看？”

乐毅答：“可借此再送一轮补约。”

“为何不是催兵？”

“诸国如今还怕先吃亏。臣若催兵，它们先疑燕急；臣若送补约，它们便得先把各自那份责认明。”

燕昭王点头：“去办。”

于是夜里车马再备。这一回带出去的，已不只是原约，还有魏、韩、赵改过、燕又补过的新简。

临淄这边，田文回府后，没有见宾客，也没有拆门客送来的策书。

他只把今日殿上录下的朝辞摊在案头，看了两遍。看到“借他国之名作遮”那一句时，手指停了许久，最后把竹简卷起，放进漆匣。

匣盖合上的那一下，声音很轻。

窗外临淄灯火仍盛，照得檐下铜钩都发亮。只是田文知道，这火若再往外逼一步，照到旁人眼里，便未必还是盛了。
