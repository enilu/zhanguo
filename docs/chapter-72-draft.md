# 第七十二回 临淄再压宋使田文冷 郭隗闻报知齐又高

齐廷那边，对宋的那一压并没有就此停住。

宋使入临淄后，原想着以辞色缓一缓，再看魏、楚之意。可齐湣王根本不给太多转圜，只先问边地、后问贡数，最后甚至当殿点出宋近年在齐楚之间左右逢源的几件旧事，语气越来越重。

宋使立在殿下，背后汗都透了。

因为这已不是寻常大国责小国失礼。

这是一种有意要让满殿、让外间诸国都看见的压。

齐湣王最后道：“宋若真知所依，便不该今日到寡人案前，还一味拿他国之名作遮。”

这句话出口，殿中近臣纷纷附和。

有人说宋素来反覆，本该重责；

有人说不如此不足示诸侯；

也有人顺势道：“今诸国皆各自有病，大王若不趁此时定山东轻重，后面反叫旁人各借局势做大。”

田文坐在席间，一句也没接。

因为他知道，朝会到了这里，再说“过满则折”，已不只是逆耳，而像是在当众扫大王此刻最得意的那层势。

可不说，他胸口那股灰意又越压越沉。

散朝之后，宋使几乎是被逼着退下去的。

消息很快出临淄。

先到魏。

魏王看完，只淡淡道：“齐如今不是在问宋服不服，是在问天下谁还敢拿别人名字挡它。”

魏相在旁，沉默良久，才道：“如此一来，乐毅那卷约，怕要更快了。”

同一刻，燕都也已得报。

郭隗先接消息，没有立刻入宫，只拿着那份急报去见乐毅。

乐毅此时仍坐在灯下对魏、韩、赵三份改约，一条条补答。

郭隗把急报递过去，道：“齐又高了一步。”

乐毅展开一看，只见几行：

“宋使受迫于殿。”

“齐言不许再借他国之名自遮。”

“临淄朝中附和者众。”

看完之后，他沉默了片刻，才道：“好。”

郭隗一怔：“齐又压人，你反说好？”

乐毅把急报压到图上宋地那一点，缓缓道：“齐越这样压，越不是只在压宋。它是在告诉山东诸国：谁若再想留中间，齐也要逼他先站。逼得越急，诸国便越难继续装作此事与己无关。”

郭隗听到这里，才缓缓点头。

因为他也明白，田文先前所忧，正在一步步变成事实：

齐不是一脚踩空。

而是一路都踩得太实、太顺，顺到把周边每一国都越逼越往同一个念头上靠。

当夜，燕昭王再召乐毅入宫。

王问：“齐压宋使之事，你怎么看？”

乐毅答：“可借此再送一轮补约。”

燕昭王道：“为何不是直接催兵？”

乐毅答：“因为诸国如今最要紧的，还不是热，而是冷里那层不肯先吃亏的心，已被齐逼得开始动。臣若此时只催兵，诸国会先疑燕急于借它们的手；臣若把补过的约再送去，它们便会明白，燕要的不是它们先热血，而是它们先把该认的东西认下来。”

燕昭王听后，只说了一个字：“准。”

夜色里，燕使再备车马。

这一次，他们带去的，不只是原约，也带上了魏、韩、赵已改过、燕又补过的新简。

另一边，田文回府之后，第一次没有去看门客献上的策书。

他只把今日殿上宋使那张发白的脸记在心里，久久没有说话。

门客低声问：“君上为何如此？”

田文道：“我先前总怕大王把自己看得太稳。如今看，朝中许多人也都已经跟着把这份稳，当成了天经地义。”

“那又如何？”

田文望着临淄夜色，声音很低：“那便不是一个人的高了。是一国上下，都在替这份高添柴。”

窗外鼓声远远传来，临淄仍是一城华灯。

可田文知道，这些灯照出去的时候，落在外人眼里，已越来越不像盛。

更像逼人抬头去看的火。
