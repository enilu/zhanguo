# 第七十二回 宋使入临淄当殿被逼认服 田文退朝后只把朝辞锁进漆匣

宋使入临淄后，齐廷对宋的那一压并没有停。

这一回，齐湣王没先问辞色，先问边地，又问贡数，最后索性把宋近年在齐、楚之间摇摆的旧账一桩桩翻出来点给满朝听。

宋使站在殿下，汗已透背，还想拿“楚意未明”“魏亦有言”来缓。

齐湣王当场打断：“宋若真知所依，今日便不该还拿他国之名作遮。”

殿中一片附和。

有人说宋素来反覆，本该重责；有人说不如此，不足示山东诸侯；更有人顺势道：“今诸国各自有病，大王若不趁此时定山东轻重，后面反教旁人借局自大。”

田文坐在席中，一句未发。

散朝时，宋使几乎是被逼着退出殿门的。

消息比人走得更快。大梁先得报，魏王看完，只把急简往案上一扔：“齐如今不是问宋服不服，是问天下谁还敢拿旁人的名字挡它。”

燕都也很快得了信。

郭隗拿着急报去见乐毅时，乐毅还在灯下对改约。郭隗把急报放到案边：“齐又高了一步。”

乐毅展开一看，只见几行短报：

“宋使受迫于殿。”

“齐言不许再借他国之名自遮。”

“满朝附和。”

他看完，把急报压在宋地那一点上，只道：“好。”

郭隗一怔。

乐毅没有再多解释，只叫人再备车马。燕昭王夜里召他入宫，听完这封急报，也只问一句：“这一回，是不是该催兵了？”

乐毅答：“还不催兵，再送补约。”

“为何？”

“齐越逼，诸国越难总说此事与己无关。”乐毅道，“臣若此刻催兵，它们先疑燕急；臣若再送补约，它们便得先把各自那份责认清。”

临淄这边，田文回府后没有见宾客。

他把今日殿上录下的朝辞摊在灯下，看了两遍，翻到“借他国之名作遮”那一句时，手指停了很久，最后把整卷朝辞缓缓卷起，放进漆匣。

匣盖合上时，声音很轻。
