# 第七十三回 诸侯对改最难在谁先担 苏秦看局知热已在冷中生

第二轮补约送出去后，回来的仍不是一句痛快话。

回来的，是一封封问题。

魏问：

“前锋若溃，而后应‘退亦有次’，这‘次’由谁定？”

韩问：

“若主帅已定，而借道之国中途边警突起，可否改次？”

赵问得最细：

“粮道三改，其主责落在借道国、主议国，还是前军之国？”

三国来书都不长，却都盯着最难躲的地方问。

燕昭王夜里与乐毅对案而坐，把那几封书信一一看过，问道：“它们问得这般紧，你不觉得烦？”

乐毅道：“烦，才是往前走了。若还是空赞空听，反倒无用。”

郭隗站在一旁，看着案上摊开的几封问书。每一封都不像豪言，倒像在拿小刀沿着竹简一寸寸往下刮，把藏在“共伐齐”下面那些最不好看的事，全刮了出来。

邯郸那边，苏秦也在看这些来往补约。

门客把魏、韩、赵、燕四方文字排开，忍不住道：“相国，它们还在争，燕那边为何反倒像越发有数了？”

苏秦没有立刻答，只拿起赵书，指着其中一句给他看：“你看，它们如今争的，已不是谁尊谁卑，而是粮断时谁补、路坏时谁修、前军溃时后应退到哪一步。这些字难看，却都是真字。”

门客听后，仍有些不解。

苏秦便把四卷简并到一处：“往常一说合兵，席上先热。可真能成事的，未必都在热处。也有时候，是这一封封冷简来回磨，把人心里那点不肯先认的东西，慢慢磨出来。”

同一刻，赵廷小朝也在议这几封问书。

赵奢把粮道图摊开，拿木筹点着几处险口：“若联军真起，最易坏的不在临淄城下，在到临淄前这一路。雨一来，车会陷；路一窄，粮会堵；谁都怕自己仓先空。若不把这些责任先钉死，兵还没见齐，心便先散了。”

廉颇抱臂站在图旁，只说一句：“兵未起，先守后路。”

赵王点头：“所以赵不急着问谁去攻城，先问谁肯守这一路苦差。”

夜里，乐毅回魏书，写到“退亦有次，由谁定”时，停笔片刻，最后落下：

“主帅先定其大次，各军预受其节；临机之权，归主帅，不归私军。”

写完之后，他又补了一句：

“若至乱退，先坏者非退法，乃军心。”

写毕，他把这卷答书晾在灯旁，火光照着新墨，微微泛亮。旁边还有韩书未答、赵书待覆，一卷卷都压在案角，没有一卷轻省。
