# 第七十三回 诸侯对改最难在谁先担 苏秦看局知热已在冷中生

第二轮补约送出去后，最先回来的，不是国书。

是问题。

魏问：

若前锋已溃，而后应“退亦有次”，这“次”由谁定？

韩问：

若主帅既定，而借道之国中途边警突起，可否改次？

赵问得更细：

若诸国共前议，粮道三改，其主责究竟落在借道国、主议国，还是前军之国？

这些问题一封封摆上来，看得外行都头大。

可乐毅看完，却比先前更安。

因为他知道，局到这里，才算真往兵前又走近了一截。

从前诸国谈伐齐，谈的是可不可；

如今问的，却是哪一条真落到道上、落到营里、落到败与不败之间时，谁先担责。

这已经不是看热闹的人会问的话了。

燕昭王夜里与乐毅对案而坐，看着这些问题，忽然问：“你不觉得它们问得烦么？”

乐毅答：“烦，便说明它们开始拿这件事当真。”

“若问到最后，人人都只想把责往外推呢？”

乐毅道：“那便推。能推到极处，谁最不肯担、谁又其实愿担，都会露出来。臣怕的从来不是争责，怕的是谁都不肯先把责摆出来。”

郭隗在旁静听，心里也慢慢服了。

因为他先前虽知乐毅稳，却也没想到他能稳到这个地步：

别人怕争，

他偏要先把最能争死人的地方，统统抬上来让人争。

而在邯郸，苏秦也正看着那几封来往补约。

门客替他把魏、韩、赵、燕几方文字一卷卷排开，忍不住道：“相国，这些简看得人眼花。诸国既都还在争，何以燕那边反倒像越发有数了？”

苏秦把其中一卷慢慢展开，道：“因为争的东西不一样了。”

门客不解。

苏秦便指给他看：

“你看，若还是往日空谈，争的会是先后名分、谁尊谁卑。如今它们争的，是前军败了后应退到哪一步，借道几程，粮改几站，失期由谁罚。这些都不好听，却是能把兵往一处拧的真东西。”

他说到这里，停了停，才又道：“天下有时并不是先在热里起势。也有时候，是在这一封封冷简里，慢慢把那点真要做事的火压出来。”

门客听后，半晌无语。

因为这话与他们这些年最熟的那种“振臂一呼，群情并起”全不一样。

可眼前这局，又偏偏正是这样在走。

同一时刻，赵廷小朝也在议那几处新问。

赵奢拿着粮道图，先把几处可能出事的点一一指出：“若联军真起，最易坏的不是临淄城下，而是到临淄前这一路。谁都怕后头空，谁都怕自己粮先断。若不把这几处责任事先钉实，后面再多胜心，也会被路上磨散。”

廉颇在旁听着，只道：“这便是兵未起，先打后路。”

赵王点头：“所以赵今日不急着问谁去攻城，先问谁肯先守这一路上的苦差。”

这句话说出来，连苏秦都微微抬了抬眼。

因为他知道，到了这一层，诸国其实已不再只是“被齐逼得不得不看”。

它们开始真往“若动，该怎么动才不再各败各的”上走了。

夜里，乐毅写答魏书，答到那句“退亦有次，由谁定”时，提笔停了一会儿，最后写下：

“主帅先定其大次，各军预受其节；临机之权，归主帅，不归私军。”

写完之后，他又在旁添了一句更短的：

“若事到乱退，便已不是退法之争，是军心先坏。”

灯火照着那几行字，映得竹简边缘微微发亮。

乐毅放下笔，轻轻揉了揉指节。

他知道，真正的难处还远没完。

可这几夜来回补答之间，他也越来越清楚地感觉到：

那些过去总被齐、秦两强压得各自自顾的国家，正在一种极冷、极不热闹的往来里，慢慢学着一件旧事之外的新事。

不是先喊自己愿意并。

而是先问，若真并，该怎样才不再先败在彼此心里。
