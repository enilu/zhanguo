# 第七十三回 诸侯第二轮回简先问退次 赵廷看粮道时苏秦知冷局已真起热

第二轮补约送出去后，回来的仍不是一句痛快话。

回来的，是一封封问题。

魏问：“前锋若溃，而后应‘退亦有次’，这‘次’由谁定？”

韩问：“若主帅已定，而借道之国中途边警突起，可否改次？”

赵问得最细：“粮道三改，其主责落在借道国、主议国，还是前军之国？”

燕昭王夜里与乐毅对案而坐，把三封问书一一看过。郭隗立在旁边，看着那一行行问句，只觉它们不像结盟，更像拿小刀沿着竹简往下刮，把“共伐齐”下面最不好看的地方全一点点刮出来。

邯郸那边，苏秦也在看这些往来补约。

门客把魏、韩、赵、燕四方文字排开，忍不住道：“相国，它们还在争，燕那边为何反倒像越发有数了？”

苏秦拿起赵书，指着那句问粮道主责的话给他看：“你看，它们争的，已经不是谁尊谁卑，是粮断时谁补、路坏时谁修、前军溃时后应退到哪一步。”

门客还是不甚懂。

苏秦便把四卷并到一处：“往常一说合兵，席上先热。如今这热，倒先是从冷简里磨出来的。”

同一夜，赵廷小朝也在议这几封问书。

赵奢把粮道图摊开，拿木筹点着几处险口：“若联军真起，最容易坏的不在临淄城下，在到临淄前这一路。雨一来，车会陷；路一窄，粮会堵；谁都怕自己仓先空。若不把这些责任先钉死，兵还没见齐，心就先散了。”

廉颇抱臂立在图旁，只说一句：“兵未起，先守后路。”

赵王点头：“所以赵不先问谁去攻城，先问谁肯守这一路苦差。”

燕宫里，乐毅此时正回魏书。

写到“退亦有次，由谁定”时，他停笔片刻，末了落下：“主帅先定其大次，各军预受其节；临机之权，归主帅，不归私军。”

写完之后，他又补了一句：“若至乱退，先坏者非退法，乃军心。”

新墨未干，韩书还摊在一侧，赵书也压在案角。

一卷卷都不轻省。
