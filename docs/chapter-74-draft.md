# 第七十四回 黄歇整储昭阳先服气 乐毅复出诸国议兵声

郢都那边，黄歇与昭阳接下东面兵储之后，头一件事便把旧簿全翻了出来。

翻得越深，两人脸色便越沉。

账上写着“可调”者，实则有些还压在郡仓未动；

写着“已明”者，实则谁前谁后、谁供谁护，条上仍糊着；

更有几处看似小额的转欠，拆开细看，恰恰都卡在最不该迟的几段路口上。

昭阳看完，冷冷道：“这些东西平日里不觉得，一到要用兵，便全是刀口。”

黄歇点头：“所以臣先前才说，楚若总等外头起势，再回头问自己粮几何、兵何从，便永远只赶得上最后一步。”

昭阳没有再接话，只命吏把那几处最紧的欠条、转仓、旧例全另抄一份，今夜便送来。

这是他这些年来第一次，对一个年轻后进的那套“先把最重最慢的东西补实”之法，生出真正的服气。

不是因为黄歇多有锋芒。

而是因为这年轻人看见的，恰恰都是那些平日最容易被朝中高声压过去、却最能在用时坏大事的地方。

另一边，燕使已再出赵、魏、韩。

这次去的，不再是单送一卷草约。

而是把魏、韩、赵三家已改、燕又补过的新简一并带上，逐条对议。

新郑最先开口的，反倒不是韩王，而是军中一名老将。

他把简往案上一拍，道：“若依此约，前军之苦最重，后应之责也不轻，谁来做这最不好看的差？”

殿中一时无人立刻答。

因为这正是一路改约改到今天，终于逼到台面上的那句话。

乐毅没有回避，只道：“谁兵近、谁势便、谁最不宜后退，谁便先做。若人人都争好看之处，后面便不用议了。”

那老将盯着他：“说得轻。若轮到燕呢？”

乐毅答：“若轮到燕，燕先担。”

这四个字落下，连韩王都抬起眼来。

因为谁都听得出来，这已不是场面话。

若燕真敢先把这句写进简里，许多国家后面便少了一层最顺手的疑：

你拉我们来，是不是先让我们吃苦、你自己居后？

乐毅随即真把那句话添上了简：

“若诸国共议而燕所当其前，燕不辞前军之责。”

韩王看着那行新字，半晌才道：“此条，寡人留。”

消息传到邯郸时，赵奢正与廉颇同看西边新报。

听完燕愿先担前军之责这句话，廉颇沉默了一会儿，才道：“这人是真肯把最难看的字先写在自己身上。”

赵奢也道：“若它真如此，赵后面便不好总拿‘再看一看’当宽路了。”

苏秦闻言，没有立刻说话。

过了许久，才缓缓道：“局要成，常不是人人都忽然热了。是先有一个人，肯把最难认的一条，先压到自己案前。”

这话说完，邯郸殿中一时极静。

赵王抬头看向东方，忽然道：“那便再等等齐。”

因为谁都知道，如今还差最后一层：

齐还得再往高处走一步，

诸国也还得再看清一步，

这卷已经改得越来越像军令的东西，才能真正从简上落进兵里。

夜深之后，郢都也静了。

黄歇把东面兵储新旧两册并列在案，一笔笔改，一条条重立。

昭阳站在一旁看着，没有催。

过了很久，他才忽然道：“黄歇，你这人不像在写条陈，倒像在替楚重新接骨。”

黄歇停笔，看着那两册简，低声答：“骨若松了，外头一阵风便能把人吹歪。臣不过是想让楚后面不至再总靠别人先替自己挡那阵风。”

昭阳听完，没有再说什么。

可他心里已明白，这年轻人后面未必只会在兰台写简。

因为一个真能把国中这些散骨慢慢接回去的人，迟早会被更重的事推到前头。

窗外夜风掠过宫檐，吹得灯焰微微向东一偏。

而在更北边，燕使车马也正沿着夜路向前。

一边在补骨，

一边在改约。

天下那场还未真正举旗的大合兵，已在这些最枯、最慢、最不好听的地方，越长越实了。
