# 第七十四回 昭阳黄歇夜整东境兵储册 乐毅复出时诸国已开始议谁先担前锋

郢都这边，昭阳与黄歇接下东面兵储后，先把旧簿全翻了出来。

不翻还好，一翻便处处见缝。

账上写“可调”的，有几处其实还压在郡仓未起；写“已明”的，细看却连谁前谁后、谁供谁护都还含着；还有几笔不大的转欠，偏都卡在最要命的两段路口上。

昭阳看完，脸色发沉：“这些平日看着像小病，到真要用兵时，全是刀口。”

黄歇没多说，只把最紧的几处圈出来，另立一册。昭阳立刻命吏重抄，当夜就封。

与此同时，燕使带着补过的新简再出赵、魏、韩。

这一回不单送约，是当面逐条对议。新郑殿上，最先发问的不是韩王，而是一名老将。他把竹简往案上一拍：“照你这卷约，前军最苦，后应也不轻。谁来做这最不好看的差？”

乐毅答得极平：“谁兵近，谁势便，谁最不宜后退，谁先担。”

那老将盯着他：“若轮到燕呢？”

乐毅道：“轮到燕，燕先担。”

这四个字一落，殿里静了静。

乐毅随即提笔，把这句话添进简里：“若诸国共议而燕所当前，燕不辞前军之责。”

韩王看着那行新字，隔了片刻，才把竹简缓缓合上：“此条，寡人留。”

消息传到邯郸时，赵奢正与廉颇同看西边新报。

听完燕愿先担前军之责，廉颇只道：“这人肯把难看的字先写在自己身上。”

赵奢也道：“若燕真如此，赵再总拿‘再看一看’当宽路，便不好看了。”

赵王坐在上首，听完很久，才把手里的简慢慢卷起：“那便再等等齐。”

邯郸无人接话。

郢都官署里，夜灯到更深还没灭。昭阳把新抄的东境兵储册翻到末页，提笔批了个“验”字，又把另一卷推给黄歇：“这一处路口，明日重查。”

黄歇接过，低头应了一声。

案上新旧两册并排，封泥与朱笔压得满满当当，卷末那枚新按的官印在灯下还泛着一层湿光。
