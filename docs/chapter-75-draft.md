# 第七十五回 齐兵逼宋再取一城口 诸侯闻报知其已不收

赵王那句“再等等齐”，并没有等太久。

临淄那边，很快便替诸侯把后面那一步走了出来。

宋使受迫退去之后，宋国朝中一面急遣使往楚、魏探意，一面仍想拖住齐军，不肯把边地与贡数立刻尽照齐意改定。齐湣王得报，脸上连一点意外都没有。

他只问了一句：“它还以为自己能等谁来替它说话？”

朝臣尽皆低首。

因为谁都听得出，大王这一问里，真正盯着的早已不只是宋。

两日之后，齐东境军再进一步。

这回不再只是压道。

宋边一处控着水陆转口的小城，被齐军骤然围住。齐军并未大举攻城，只先断外路、封渡口，再放话给城中守吏：三日不降，后果自取。

宋城不大。

可位置极要。

消息一出，山东诸国都立刻明白，齐这一手已经不是“试一试宋服不服”，而是在拿一处真口子告诉旁人：

齐若要往前，不会总停在言辞上。

临淄大朝之上，齐湣王听着前线报来的“已围”“已断”“守城摇动”几字，神色越发舒展。

有人当殿称贺：“宋小而反覆，大王先收其咽喉，诸侯自当更知轻重。”

又有人顺势道：“魏、赵、燕、韩纵有怨，也未必敢为宋出这一头兵。今一城若下，山东更无人敢轻逆齐意。”

满殿气热得几乎要浮起来。

田文坐在席间，看着那封军报，指节却慢慢收紧。

因为他知道，这已经不是“高”。

这是“已不收”。

他终究还是起身。

“大王，一城可围，一口气不可太尽。”

齐湣王抬眼，眉间已隐有怒意：“孟尝君，你又来扫寡人兴头？”

田文道：“臣不是扫兴。臣只怕齐今每下一步，都在替别人把‘再等等看’那层心，往‘不能再等’上逼。”

齐湣王冷笑：“宋一小国，逼得动谁？”

田文抬头，声音不高：“逼的从来不只是宋。逼的是旁人心里那句：若今日还只当与我无关，明日这城口会不会换到我眼前。”

这话一出口，殿中气立时紧了一下。

有人不敢看大王脸色，有人低头盯着案角。

齐湣王却已不再愿听，拂袖道：“寡人要的是山东知齐有轻重，不是叫诸侯看寡人手软。”

田文伏身不动。

可他心里已比先前更冷。

因为他知道，大王如今最怕的，已经不是做错。

而是被人当众说“该收一收了”。

同一时刻，围城之报先后到了邯郸、新郑、大梁、郢都、燕都。

赵王看完，只把军报放到乐毅草约旁边，许久没说话。

赵奢低声道：“齐这是在替我们把最后那句‘再看一看’烧掉。”

廉颇也道：“一旦这城真下，后头便不是谁还想不想议，是谁还敢不议。”

而在燕都，乐毅展开军报，只看了两遍，便把手落到图上宋地那一点，轻声道：“到了。”

燕昭王看向他：“到什么？”

乐毅答：“到诸侯不能再只改约，不认军次的时候了。”

夜里更深，围城火光映着宋境天边，远远看去，像一线烧着不灭的红。

而在各国案头，那点红也已从军报上的几行字，烧成了另一件更近的事：

再不把前后次第认下来，

齐再往前围的，就未必还只是宋。
