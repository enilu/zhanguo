# 第七十五回 齐兵逼宋再取一城口 诸侯闻报知其已不收

赵王那句“再等等齐”，没等太久。

宋使退去之后，宋国一面急遣使往楚、魏探意，一面还想拖住齐军，不肯把边地、贡数立刻照齐意改定。齐湣王得报，只问：“它还以为自己能等谁来替它说话？”

两日后，齐东境军再进一步。

这回不是压道，是围城。

宋边一处控着水陆转口的小城，天还未亮便被齐军围住。前锋先断外路，后队封渡口，商车、漕船一概不许入。午后，城外立起一面白底黑字的军牌：

“三日不降，自取后果。”

城不大，口却紧。消息一出，宋境南北都乱了。

临淄大朝上，前线快报被一卷卷送入殿中。齐湣王看到“已围”“已断”“城中摇动”几字，面上神色比前几日更松。

殿中随即一片称贺。

“宋小而反覆，正该先收其咽喉。”

“一城若下，山东自知齐轻重。”

“魏、赵、燕、韩纵有怨，也未必肯为宋先出兵。”

田文坐在席间，听着这些话，手指一点点扣紧在袖中。

他终于起身：“大王，一城可围，一口气不可太尽。”

齐湣王抬眼，眉间已沉：“孟尝君又来扫寡人兴头？”

田文道：“臣只怕齐每往前逼一步，旁人便少一分‘再等等看’的心。”

齐湣王冷笑：“宋一小国，逼得动谁？”

田文抬头：“逼的不是宋，是旁人心里那句‘今日无关，明日也无关’。”

殿中一静。

齐湣王没有再听，拂袖道：“寡人要的是山东知齐轻重，不是看寡人手软。”

田文只得伏身退下。他低头时，看见案上那卷围城急报被风吹起一角，很快又压回去，像是满殿人都已习惯了齐国这样再往前伸手。

围城之报很快传到邯郸、新郑、大梁、郢都、燕都。

邯郸殿里，赵王把军报搁到草约旁边，久久没说话。赵奢先开口：“齐这是替诸国把最后那句‘再看一看’烧掉。”

廉颇道：“一旦城真下，后面便不是想不想议，是还敢不敢不议。”

燕都那边，乐毅展开军报，只看了两遍，便把手按到宋地那一点上：“到了。”

燕昭王问：“到什么？”

乐毅道：“到诸国不能只改约、不认军次的时候了。”

夜深后，宋边围城的火把在远处连成一线。各国案头上的军报也都摊开了，几行短字压着同一个名字，谁都看得见，齐这回已不是虚逼。
