# 第七十六回 韩廷先认前军苦差名 乐毅收口知盟已近兵

齐围宋城的第三日，韩廷先动了。

不是发兵，是认次。

韩王召小朝时，案上只铺了两样东西：一卷宋城急报，一卷改了又改的会兵草约。韩王先看急报，再看草约，过了许久，才道：“再不答，后面便不是我们挑约，是齐替我们挑城。”

殿中无人接声。

那名先前拍简发问的老将先站出来：“若真要韩先认，便把话记死。韩可当前军，但须明三件事：后应不得后时，粮期不得空押，主帅若失节，韩不独坐前败之责。”

韩王看了他一眼，只道：“记。”

执简之吏立刻下笔。

竹简上墨还未干，韩王又转向乐毅：“燕先前说，若轮到燕，燕不辞前军之责。今日还算不算？”

乐毅俯身：“算。”

韩王点头：“那便记清。若诸国共兵而前路先当韩，韩可当前军。但后应、粮期、主帅之节，俱依今日新条。”

那老将听罢，长长出了一口气，像是把堵在喉口许久的一块硬骨，终于咽下去了。

乐毅当场命人把韩所认之条另誊一卷，加急送赵、魏、燕。他没有趁势再催韩多认，只把那卷新誊好的竹简亲自看了一遍，确认无误，才交给燕使。

散朝后，韩王独留乐毅。

“乐君，寡人今日认这口苦差，不是为给燕面子。”

乐毅道：“臣知道。”

韩王盯着他：“你知道什么？”

乐毅抬头：“大王是为韩自己。齐已围宋，若诸国还只争谁先吃苦，后面轮到自己时，连可争的局都没有了。”

韩王听完，半晌才道：“你这人最讨嫌处，就是总替人把不肯先讲透的话讲透。”

乐毅没有答，只深深一礼。

邯郸得报时，赵王把那卷“韩可当前军”的誊本看了两遍，递给赵奢：“你看。”

赵奢道：“韩既先认了这口苦差，赵再只拿借道粮期当宽路，便太滑了。”

廉颇也道：“还能再看，但已不能只看。”

苏秦坐在旁边，没有说话，只把那卷誊本慢慢卷起，放到自己案侧。竹简落下时，声音不大，却比前些日子那些漂亮回书都沉。
