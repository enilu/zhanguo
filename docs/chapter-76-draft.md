# 第七十六回 韩廷先认前军苦差名 乐毅收口知盟已近兵

齐围宋城的第三日，韩廷先动了。

不是发兵。

是认次。

韩王这一日召小朝，殿中除相国、老将与数名近臣外，只留乐毅在侧。

案上铺着两样东西：

一份是齐围宋城的最新急报；

一份是那卷改了又改的会兵草约。

韩王先看军报，再看草约，最后忽然道：“诸位争了这些日子，争的无非一句：谁先担最苦那口差。如今齐已替我们把后面的问题问到了门前。再不答，后面便不是我们挑约，是齐替我们挑城。”

殿中一时极静。

谁都知道，大王这句话一出，今日便不是再拖的时候了。

那名先前在殿上拍简问“谁来做这最不好看的差”的老将，先站了出来。

“若真要韩先认，臣请大王把话说死。韩可当前军，但须明三件事：后应不得后时；粮期不得空押；若主帅失节，韩不独坐前败之责。”

韩王看着他，目光沉而不动。

过了片刻，才缓缓点头：“记。”

一旁执简之吏立刻下笔。

那一刻，连乐毅都微微屏了一口气。

因为一路改约到今天，终于有一国肯把最难听的那句，从“若轮到我如何”变成“我可担，但你们也把你们的字认死”。

韩王又看向乐毅：“燕先前说，若轮到燕，燕不辞前军之责。此言，今日还算不算？”

乐毅俯身：“算。若诸国共议之后，前军归燕，燕不辞。”

韩王道：“好。那今日韩便先认一条：若诸国共兵而前路先当韩，韩可当前军。但后应、粮期、主帅之节，俱须依今日新记之条。”

这句话一落，殿中老将重重吐出一口气。

不是轻松。

是终于把那口一直卡在喉头最苦的东西，先咽下去了。

乐毅当场命吏把韩所认之条另誊一卷，加急送赵、魏、燕。

他没有多言，也没有趁势逼韩再认更多。

因为他知道，局走到这里，最怕的反而不是慢，而是见一国先认，便急着要旁国立刻跟到底，结果把刚刚咽下去的东西又呛出来。

散朝之后，韩王独留乐毅。

“乐君，寡人今日认了这口苦差，不是为给燕面子。”

乐毅答：“臣知道。”

“你知道什么？”

乐毅抬头：“大王不是为燕，是为韩自己。齐已围宋，若诸国还总争谁先吃苦，后面轮到自己时，连可争之局都没有了。”

韩王看着他，许久才道：“你这人，最讨嫌处，就是总把别人不肯先讲透的话，替人讲透。”

乐毅微微一笑，没有辩。

因为他知道，这已算是韩王此刻能给出的极高一句。

同日，邯郸得报。

赵王把那句“韩可当前军”看了两遍，抬头问赵奢：“你看如何？”

赵奢答：“韩既先咽了这口苦差，赵后面若还只拿借道粮期做宽路，便会显得太滑。”

廉颇也道：“可再看。但已不能只看。”

苏秦坐在席间，神色竟比前几日更沉，也更静。

门客夜里问他：“相国，韩先认次，这不是好事么？”

苏秦缓缓道：“是好事。也是更难的事。”

“为何？”

“因为从这一刻起，诸侯再往下走，便不是在纸上说愿不愿了。”苏秦道，“它们开始真要拿各自一国最疼的地方，去替这件事垫底。”

他说完之后，没有再多解释。

只是把那卷誊来的韩认次之简慢慢收好。

灯火照着竹面，映出一层沉沉的光。

他心里很清楚，这卷简比先前任何一封“愿共大事”的漂亮回书都更重。

因为它不是一句好听话。

它是一国第一次，真把自己放进了后面那场兵里。
