# 第七十七回 赵廷见韩认次终难缓 赵奢请担粮路第一苦

韩认前军之条送到邯郸之后，赵廷连着两日都在议。

起初仍有人主张再等等。

等魏如何改，

等燕是否真把“燕不辞前军之责”另誊成正式之条，

等齐围宋会不会就此止步。

这些等法，样样都像有理。

可如今再摆出来，殿中听着，味道已和前些日子不同了。

因为韩那卷简就压在案上。

谁都看得见，已经有人先把最苦的一口认了。

赵若还只在旁边算最滑的那条退路，便连赵自己也会觉得那份滑过了头。

赵王这一日先问廉颇：“边上若起联兵，赵最要紧的，不是打，而是什么？”

廉颇答：“不是打，是不断。”

赵王点了点头，又看向赵奢：“你说。”

赵奢几乎没有停：“是粮路。”

“为何？”

“韩今日先认前军，吃的是阵前第一口苦；赵若要在这局里立得住，便该把最不显名、却最怕出半刻差错的那一口担起来。”赵奢道，“臣请赵先认一路：联兵若起，赵担北线借道与转粮第一责。”

这话一出，殿中立刻有人变色。

因为这比说“赵可听盟”要重得多。

认了这条，便不是日后兵到哪里再看，而是先把最苦最慢、最容易被埋怨、也最难见功的一路压到自己肩上。

一名老臣忍不住道：“赵奢，你这不是替赵讨苦差？”

赵奢转头看他，道：“如今各国都怕吃苦差，才让齐一步步高到今日。赵若连这一路都不肯担，后面便只剩等别人先打、先败、先流血，再看赵要不要跟着收尾。”

那老臣一时无话。

赵王坐在上首，神色一点点沉了下来。

不是怒。

而是他已看明白，赵奢这句话说得太准。

赵这些年能撑到今天，靠的不是嘴上聪明，正是有人肯去把那些最不见彩头、却最关国本的苦事先揽下来。

廉颇也在这时开口：“赵奢此言可行。前军之苦，在刀口前；粮路之苦，在刀口后。两样都苦，只是后者更不显。赵若认这一条，后面才不像是在旁占便宜。”

苏秦听到这里，轻轻闭了一下眼。

因为他忽然觉得，自己这些年最盼的一件事，如今竟在完全不同的路数上，慢慢有了影。

不是靠一口大义把诸国先拽到一处。

而是靠这些人一国国去认最难看的差，把“可并”先一点点做出来。

赵王沉默了极久，终于道：“记。”

执简之吏抬起头，不敢确认：“大王是说……”

赵王声音不高，却极稳：“记下。若后面联兵成，赵担北线借道与转粮第一责。若误粮期，责先坐赵。”

这一句落下，连殿中空气都像重了一层。

因为它意味着，赵也认了一口真正会压死人的苦差。

赵奢俯身受命，脸上却并无喜色。

他太清楚，认下这句话，并不意味着后面就一定能成。

它只意味着，赵从今日起，也把自己最容易被人骂、却最不能不担的那部分，先放到了明处。

消息送出邯郸时，燕使尚在路上。

乐毅半途接到，展开只看了一眼，便把那卷简合上，久久没有出声。

随从低声道：“君上？”

乐毅缓缓道：“韩先认前军，赵再认粮路。局真近了。”

“那魏呢？”

“魏会更难。”乐毅道，“因为如今留给魏的，不再是能不能在旁观望，而是要不要也认一口别人能看见的苦差。”

风从道边吹来，卷起车前细尘。

乐毅抬眼望向大梁方向，心里已知道，最难的下一口，正在那里等着。
