# 第七十七回 赵廷见韩认次终难缓 赵奢请担粮路第一苦

韩认前军之条送到邯郸后，赵廷连议了两日。

起初仍有人主张再等等，等魏如何改，等燕那句“燕不辞前军之责”会不会另誊成正式之条，也等齐围宋会不会就此止步。

这些话单听都不算错。只是韩那卷誊本就压在案上，谁都看得见，已经有人先把最苦的一口认了。赵若还只挑最滑的退路，连殿中自己人都觉得难看。

第三日小朝，赵王先问廉颇：“若联兵起，赵最要紧的，不是打，而是什么？”

廉颇道：“不是打，是不断。”

赵王转问赵奢：“你说。”

赵奢几乎没有停：“是粮路。韩今日先认前军，吃的是阵前第一口苦；赵若要在这局里立得住，便该把最不显名、却最怕半刻差错的那一口担起来。臣请赵先认北线借道与转粮第一责。”

殿中立刻有人变色。

一名老臣忍不住道：“赵奢，你这是替赵讨苦差？”

赵奢看着他：“如今各国都怕苦差，才让齐一步步逼到今日。赵若连这一路都不肯担，后面便只能等别人先打、先败、先流血，再看赵要不要跟着收尾。”

那老臣一时无言。

廉颇也在这时开口：“前军之苦，在刀口前；粮路之苦，在刀口后。两样都苦，只是后者更不见功。赵若认这一条，后面才不像在旁占便宜。”

赵王听完，沉默很久，终于道：“记。”

执简之吏抬笔，赵王一字一顿：

“若后面联兵成，赵担北线借道与转粮第一责。若误粮期，责先坐赵。”

笔落下去，殿里安静得只剩书写声。

赵奢俯身受命，脸上并无喜色。他知道认下这句话，不是多了一层好听名色，是先把最容易被骂、也最不能出错的一段路，压到了赵国自己肩上。

消息送出邯郸时，乐毅正在路上。燕使快马追上，把赵书递到车前。乐毅展开，只看一眼，便把竹简合起。

从人低声问：“君上？”

乐毅道：“韩认前军，赵认粮路，魏更不能只守着自护之条了。”

他说完，抬眼望向大梁方向。道旁风起，细尘贴地而走，像是还有最后一卷最难写的简，正等着他去拿回来。
