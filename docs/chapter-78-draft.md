# 第七十八回 大梁对简魏王终难旁看 乐毅收三口苦差盟势成

韩认前军，赵认粮路，消息一前一后压到大梁时，魏王整整一日都没让人撤去案上的那几卷简。

魏是中州之国。

地势、处境、旧伤，都让它比赵韩更会算，也更难先把自己哪一口疼处亮给别人看。

可如今，韩已认第一口锋，赵已认第一道路。

魏若还只守着那些“后应不得先退”“失期有故可议”的自护之条，不是不对。

只是太轻。

轻到连魏自己都知道，再这样轻下去，便真成了只等别人先替它把局撑起来。

魏王这一日召相国、老将与乐毅共坐。

殿中没人先说话。

直到魏王自己把韩、赵两份新简先后推到众人面前，才缓缓开口：“韩吃了前军，赵吃了粮路。寡人若还只吃后应，天下会怎么看魏？”

这句话太直。

直得殿里一时连呼吸都轻了。

魏相最先道：“大王，魏处中州，向来最易受前后牵扯。若真要再认，臣以为不必去抢前军、粮路，却该认一条别人无法替魏代认的差。”

魏王抬眼：“说。”

魏相道：“认中军节应之责。”

乐毅眼神微微一凝。

因为这正是一路改约改到今天，最悬的一口。

前军苦在先撞，

粮路苦在不断，

而中军节应之责，苦在一旦前后失拍，人人都要先怨你应得不够、接得不住。

它同样不好看。

甚至更难留名。

魏王沉默了许久，才道：“若认此责，后面诸国如何看？”

乐毅这回没有抢着答。

他在等魏国自己把那句话说完。

魏相果然道：“会看见魏终于不再只替自己留一条最稳的路。中军节应，前要接韩，后要护赵，主帅令若有失，魏先吃骂。可也正因如此，若此责不认，前军与粮路便只是各自苦，不会真并成一股。”

殿中又是一静。

过了半晌，魏王忽然把手压在案上，声音极沉：“记。”

执简之吏立刻抬笔。

魏王一字一顿地道：

“若联兵成，魏认中军节应第一责。前军有进，魏当接；粮路有缺，魏当护；诸军有失拍，魏不得先辞其咎。”

写到最后几个字时，连那吏手都微微抖了一下。

因为这不是一句顺耳的话。

这是魏第一次，把自己最怕被骂、也最会吃累的一口差，真按在了简上。

乐毅直到这时，才深深一拜。

魏王看着他，道：“乐君，寡人今日认这一条，不是被燕逼的。”

乐毅答：“臣知道。是被齐逼的，也是魏自己不愿后面只做看客。”

魏王盯着他看了许久，最后竟淡淡笑了一下：“你这人，难怪张仪之后还能在山东走得动。你不拿话激人，你拿的是人最不愿承认、却又不得不承认的那点实处。”

乐毅没有接这层高话，只道：“魏今日这一认，后面便可排次。”

夜里，韩、赵、魏三份认责之简，与燕原先那句“不辞前军之责”，终于第一次被一并摆到燕昭王案前。

前军，

粮路，

中军节应。

四样最苦、最丑、最不讨好的差，已经有了三口半。

燕昭王看完之后，久久没有说话。

最后才抬头问乐毅：“如今可算什么？”

乐毅把那四卷简一一按平，缓缓道：“如今不再只是盟势。”

“那是什么？”

“是兵势。”乐毅道，“还未举旗，未出大军，可诸国已经把真正会把人压断、也能把兵并成一股的那几口苦差，先认下来了。到这一步，后面缺的已不再是空谈愿不愿。”

燕昭王听后，慢慢站起身。

殿外北风正急，吹得檐下灯焰几乎要横过去。

他却只道了一句：“那便等齐再走一步，寡人便看你把这股兵势，真正拉起来。”

乐毅俯身领命。

他走出殿外时，夜色极深。

可他心里已很清楚，自己这些日子一卷卷写出来、送出去、改回来、再补出去的简，到这一夜，终于不再只是竹上的字了。

它们已经变成了诸侯各自肩上真压住的一口苦差。

而一群国家，愿意各自先压住这样一口苦差的时候，

兵，离真正起，便只差最后那一下东风了。
