# 第七十八回 大梁对简魏王终难旁看 乐毅收三口苦差盟势成

韩认前军，赵认粮路，两卷新简一前一后送到大梁，魏王整整一日都没让人把它们撤下案去。

到了傍晚，他召相国、老将与乐毅同入偏殿。殿内没有闲人，案上只摆着韩、赵两份誊本和魏先前改过的草约。

魏王先把韩书推到前头，又把赵书压在旁边，开口便问：“韩吃了前军，赵吃了粮路。寡人若还只守后应，天下会怎么看魏？”

殿里一静。

魏相先道：“魏处中州，不必去抢前军、粮路，却该认一条旁人替不了魏的差。”

魏王看着他：“说。”

“认中军节应之责。”

乐毅抬眼，仍没有插话。

魏相继续道：“前军有进，魏当接；粮路有缺，魏当护；主帅令若有失拍，魏先吃骂。此责最不显功，也最难推给别人。可若此责不认，韩的前军与赵的粮路，便各是各的苦，不会真并成一路。”

魏王听着，手指一直压在案沿上。过了很久，才道：“记。”

执简之吏立刻抬笔。

魏王一字一顿：

“若联兵成，魏认中军节应第一责。前军有进，魏当接；粮路有缺，魏当护；诸军有失拍，魏不得先辞其咎。”

写到最后一句时，案前那吏手明显抖了一下，墨点差点落歪。魏王看见了，却没有责备，只等他把整条写完。

这时乐毅才深深一拜。

魏王看着他：“乐君，寡人今日认这一条，不是被燕逼的。”

乐毅道：“臣知道。是魏不愿只做看客。”

魏王听后，淡淡笑了一下：“你不拿好听话激人，你拿的是人最不愿认、又不得不认的那点实处。”

乐毅仍未多言，只请把新条另誊一卷，连夜送燕。

夜里，韩、赵、魏三份认责之简，连同燕那句“不辞前军之责”，第一次并排摆到燕昭王案前。

前军，

粮路，

中军节应。

燕昭王一卷卷看过去，最后把四卷都按平：“如今呢？”

乐毅道：“如今可以排次了。”

燕昭王抬头看他，过了片刻，只说一句：“那便等齐再走一步。”

殿外北风正急，吹得檐下灯焰几乎横过去。案上那几卷新简却压得很平，像几块已经落定的石头，先把诸国脚下那条将来要一起走的路，压出了形。
