# 第七十九回 宋城夜破齐势更逼人 乐毅开简先排诸军次

围宋之城，到第四夜便破了。

城并不大，守也不坚。齐军断路封渡三日，城中先乱了粮火，守吏又知外援无望，终于在夜半开了一角小门。齐军入城后不曾纵兵大掠，只先据渡口、封仓门、换旗号。

天亮时，宋城城头已尽是齐旌。

临淄宫中，捷报入殿，群臣称贺之声几乎压过了殿角铜漏。齐湣王看完军报，只说四个字：“宋亦如此。”

这句话一出，满殿更热。

有人请乘势再问宋余地；

有人请借此敕周边小国知轻重；

也有人已把话带到赵、燕头上，说山东诸国纵有不平，也终究要先看临淄颜色。

田文坐在席间，一句也没接。

散朝后，他独自立在阶下，看着报捷之吏一列列自殿前奔过，忽然想起许多年前齐国大胜之后，朝中还会先问一句“谁最可能因此生怨”。如今却只剩“还可再往前取哪一口”。

燕都这边，同样很快得报。

乐毅看完，没有先把急报压到图上去，只对燕昭王道：“可以排次了。”

燕昭王盯着他：“你等的，就是这一城？”

乐毅道：“臣等的不是一城，是诸侯心里那句‘还可再等等’。宋城一夜而破，这句话留不住了。”

他说完，命人把韩、赵、魏三家认责之简一并取来，另展开一卷空简，亲手写下四行：

“前军：韩。”

“北线粮路：赵。”

“中军节应：魏。”

“主议与总节：燕。”

写到最后一行时，他笔尖停了一瞬，又添上两个字：

“暂定。”

当天夜里，燕使再出。这一回带去的，不只是改约与认责之简，还带着乐毅亲排的第一版军次。

邯郸先得简。

赵王看到“北线粮路：赵”一行，并不意外；看到“主议与总节：燕”时，手指却停了一下。赵奢在旁道：“燕把自己压在总节上，往后前后若不应，先吃骂的也是它。”

廉颇也道：“这比它只说‘不辞前军’更重。”

赵王点了点头，把简递给苏秦。苏秦看罢，没说别的，只把竹简压平：“回燕。赵认原责，不争此排。但会期须明，粮站须先验。”

案旁执吏立刻记下。那卷回简写得很快，仿佛宋城夜破之后，许多原本还能慢慢计较的地方，一下都逼到了笔尖上。
