# 第七十九回 宋城夜破齐势更逼人 乐毅开简先排诸军次

围宋之城，到第四夜便破了。

城并不大，守也不算坚。齐军连着断路封渡三日，城中粮火先乱，守吏又知外援无望，终于在夜半开了一角小门。齐军入城并未纵兵大掠，只先据渡口、封仓门、换旗号。

天亮时，宋城城头已尽是齐旌。

这一城拿得不算血腥。

却比一场鏖战更叫山东发冷。

因为人人都看得见，齐如今最可怕的，不只是敢压人，而是压得又稳、又顺，仿佛世上本就该由它这样一寸寸往前取。

临淄宫中，捷报入殿，群臣称贺之声几乎压过了殿角铜漏。

齐湣王把那份军报看完，只说了四个字：“宋亦如此。”

这句话落下，满殿更热。

有人顺势请乘胜再问宋余地；

有人说当借此再敕周边小国知轻重；

也有人已经把话往赵、燕头上带，说齐既能一手按住宋，山东诸国纵有不平，也终究要先看临淄颜色。

田文坐在席间，胸口却越来越冷。

因为他知道，大王此刻最危险的地方，已不是一时得意。

而是这份得意已经被战报、朝声、顺势之言一层层托实，变成一种“再往前一步也无妨”的笃信。

散朝之后，他独自站在阶下，看着宫中一列列报捷之吏来回穿行，忽然想起很早以前齐国还会在大胜后先问一句“谁最可能因此生怨”。如今却已没人爱问这句话了。

同一时刻，燕都那边也接到了宋城夜破的急报。

乐毅看完之后，没有像往常那样先压在图上静看很久，而是直接对燕昭王道：“可以排次了。”

燕昭王盯着他：“你等的，就是这一城？”

乐毅答：“臣等的不是一城，是诸侯心里那句‘也许还来得及再等等’。宋城一夜而破，这句话已经留不住了。”

说完，他命人把此前四国认责之简一并取来，又另展开一卷空简，亲手写下四行：

“前军：韩。”

“北线粮路：赵。”

“中军节应：魏。”

“主议与总节：燕。”

写到最后一行时，他笔下微顿，又添了两个字：

“暂定。”

燕昭王看着那两个字，点了点头。

这两个字极要。

因为它既把局往前推了，又给诸国留了一口还能改、却不能再漫拖的气。

当天夜里，燕使再出。

这一次带去的，不只是改约、认责之简。

还带着乐毅亲自排下的第一版军次。

他先送邯郸。

赵王夜里接简，看见“北线粮路：赵”一行，并无意外；可当目光落到“主议与总节：燕”时，手指还是轻轻停了一下。

赵奢在旁道：“燕把自己压在总节上，后面若前后不应，先吃骂的便也是它。”

廉颇也道：“这比它只说‘燕不辞前军’更重。前军苦在刀口，总节苦在天下都盯着你失拍。”

赵王缓缓点头：“乐毅是把最不讨喜欢、也最不容失的一层，先压到了燕自己头上。”

苏秦接过那卷军次简，看了很久，才道：“到这里，天下终于不是只在议怎么并，已经是在试着并了。”

夜风吹进殿来，把简角吹得轻轻一动。

赵王伸手按住，忽然道：“回燕：赵认原责，不争此排。但会期须明，粮站须先验。”

邯郸这一下回得很快。

因为宋城一破，许多原本还能慢慢算的地方，忽然都变得太近了。
