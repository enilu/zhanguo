# 第八十回 韩魏对次各吞一口气 楚闻宋破怀王终难安

赵回简之后，新郑与大梁也很快都被那份“暂定军次”逼到了案前。

韩王接简时，先看的仍是“前军：韩”。

认责时还能说“若联兵成”，认次却已像兵在路上。韩王看完，许久没说话。那名老将仍在侧席，见君上不语，便道：“大王若此时再缩，前面咽下去那口苦，便成了笑话。”

韩王看了他一眼，把竹简推给韩相。

韩相看完，道：“韩可认前军，但臣请再添一条：后应军到期不得以‘路阻’‘讯误’为辞。若误，应坐军书。”

韩王点头：“加上。”

于是新郑回燕的，不再只是“韩认前军”，而是“韩认前军，后应到期不得误”。

大梁那边更难。

魏王看见“中军节应：魏”五字时，整个人都沉了下去。他把简看了两遍，才问乐毅：“燕把自己排在总节。若前军先失、中军未应、粮路又误，天下先骂谁？”

乐毅答：“先骂燕。”

魏王盯着他看了很久，终于把简合上：“好。魏不翻这行。回燕：魏认中军节应之次，但会兵之日、主帅之令、失拍之罚，须先明。”

乐毅俯身受命。

他出大梁宫门时，暮色已深。身后殿门一扇扇合上，里头誊简、盖印的声音却还没停。到这一刻，韩、赵、魏三国都已不只是认责，而是在自己那一行次第上开始落笔了。

郢都那边，宋城夜破之报也终于入宫。

怀王看完之后，没有像前些日子那样先冷笑。他把报文翻过来看了第二遍，问近臣：“宋竟破得这样快？”

近臣低头：“是。”

怀王把军报按在案上，过了许久，才低声道：“齐若如此，后面便不会只压宋了。”

屈原与黄歇很快奉召入见。

怀王没有多绕，把军报推过去：“你们看。”

屈原只看了一眼，便道：“再不把自己站定，后面便不是楚看齐如何走，是齐替楚把路越逼越窄。”

黄歇接道：“东面兵储、朝辞、与外间应对，都得再快一层。”

怀王这回没有立刻露出不耐，只盯着报文上“夜破”“换旗”几字，半晌才道：“去办。”

殿中执事立刻应声。宫灯照着那卷军报，纸上白得发亮，像是连郢都也被那一夜宋城的火光，照进了案前。
