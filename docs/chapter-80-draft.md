# 第八十回 韩魏对次各吞一口气 楚闻宋破怀王终难安

赵回简之后，新郑与大梁也很快都被那份“暂定军次”逼到了案前。

韩王接简时，先看的仍是“前军：韩”。

虽然这原本就是韩自己先认下的苦差，可一旦真被乐毅排进次第，落到白纸黑字上，那口苦便立刻比先前更沉了一层。

因为认责时还能说“若联兵成”；

到了认次，便已像是兵在路上了。

韩王看完之后，许久不语。

那名老将仍在侧席，见君上久久不言，终于先开口：“大王若此时再缩，前面咽下去那口苦，便成了笑话。”

韩王抬眼看他：“你倒直。”

老将道：“齐都打进宋城了，臣也学不会再绕。”

这一句竟叫韩王嘴角动了一下，像要笑，却又很快压了回去。

他把军次简推给相国：“你看。”

韩相看完，道：“韩可认前军，但臣请再添一条：前军既先当锋，后应军到期不得以‘路阻’‘讯误’为辞。若误，应坐军书。”

韩王点头：“加。”

于是新郑回燕的，便不只是“韩认前军”，而是“韩认前军，且要后应到期不得误”。

这不是示强。

是韩已知自己既先认了这口差，便再不能让旁国在自己背后留出那层最顺手的滑口。

大梁那边则更难。

魏王看见“中军节应：魏”五字时，整个人都沉了下去。

认责是一回事。

排次又是另一回事。

前者还可留在将来，后者却像已把魏真的摆到韩前赵后、左右都要接、上下都要看的一条线上。

魏相低声道：“大王，如今若再退，便会显得魏前面认责，只是空认。”

魏王冷冷道：“寡人岂不知？”

说完之后，他把那卷简又看了一遍，忽然问乐毅：“燕把自己排在总节。若前军先失、中军未应、粮路又误，天下先骂谁？”

乐毅答：“先骂燕。”

魏王盯着他看了许久。

这才缓缓把简合上：“好。那魏便不翻这行。回燕：魏认中军节应之次，但请先明会兵之日、主帅之令、失拍之罚。”

乐毅俯身受命。

他走出大梁宫门时，暮色已深，背后却像轻了一寸。

因为到这一刻，韩、赵、魏三国都已不只是认责，而是开始真在自己那一行次第上写字了。

而更南边的郢都，宋城夜破之报也终于入宫。

怀王看完之后，脸色并不像先前闻齐压宋那样带着一点快意。

他把军报翻过来看了第二遍，问近臣：“宋竟破得这样快？”

近臣低头：“是。”

怀王沉默了很久，忽然把报文往案上一按，低声道：“齐若如此，后面便不会只压宋了。”

这句话说出来，连他自己都怔了怔。

因为这是这些年里，他少有几次不先算“楚可借什么”，而先感觉到“若再不自定，外面那股势会走到哪里”的时候。

屈原与黄歇很快奉召入见。

怀王没有多绕，直接把宋破之报推过去：“你们看。”

屈原只看了一眼，便道：“齐此刻最坏处，不在一城，而在它会叫天下都明白，再不把自己站定，齐就会替众国一处处站。”

黄歇接得更稳：“故楚眼下更不可只停在责齐。东面兵储、朝辞一口、与外间应对，都得比前些日子再快一层。”

怀王坐在上首，听完之后，没有像往日那样立刻露出不耐。

他只是望着那份军报，许久没有动。

窗外宫灯映着报文上“夜破”“换旗”几字，照得格外白。

这一回，连怀王都终于感觉到，那股原本还只在赵、韩、魏、燕之间越逼越近的东西，已经把楚也真压到案前了。
