# 第八十一回 燕宫定期先压诸侯喘 苏秦看简知旧纵已远

韩、赵、魏三国先后认了军次之后，燕宫中的夜灯接连亮了三夜。

乐毅没有立刻宣“大事已定”。第四夜，他才把一卷空简摊开，只写四个字：

“会于何日。”

燕昭王站在案旁，看着这四字：“你欲定几时？”

乐毅把赵、韩、魏三家的回简一一排开，算了很久，才在空简上定下一日。那日不算近，也绝不算松。赵来得及验粮站，魏来得及定节应，韩来得及整前军，可若谁再慢，便是自己坏了前约。

燕昭王看完，只问：“你敢发？”

乐毅道：“到这里，不发，比发更险。”

于是燕使再出。这一回带去的，是定期简。

邯郸先得。

赵王看完日期，没有先问别的，只问赵奢：“来不来得及？”

赵奢心里一算：“若今日起便先验北线三站、腾两仓、改两段押运旧例，来得及。”

赵王点头：“那便不是日期太急，是赵手若再慢，自己坏约。”

新郑那边，韩王看完日期，也只是沉声道：“乐毅这是连诸侯喘气的余地都压着了。”

那老将却道：“压着才好。再让人喘久了，前军两个字就会越看越重。”

韩王没有反驳，只把定期简压在前军簿上：“回燕。韩按期整前军，但主帅之令须先行一份。”

大梁看见定期简时，魏王眉头皱得最深，可最终也只改了一句：“魏按期整中军节应，但诸军联络号次，当先定而后行。”

这一回，连魏都不再争“缓”。

夜里，苏秦在邯郸馆中看完诸国回书，久久没有说话。门客在旁低声道：“相国，眼下几国都已认责、认次、认期，这不正是你当年最想看到的局么？”

苏秦把那卷定期简慢慢展开，又慢慢卷起：“像，又不像。”

门客不解。

苏秦看着案上几卷新简：“像在于，天下又有了要往一处拧的气。不像在于，这口气不是靠歃血与大言先热起来，是被一步步逼到簿册、军次、会期里的。”

他说完，把定期简压到案角，指节久久没有松开。窗外邯郸夜风一阵阵钻进门缝，吹得灯影轻轻晃动，照得他手背的青筋更深了些。
