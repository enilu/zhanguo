# 第八十一回 燕宫定期先压诸侯喘 苏秦看简知旧纵已远

韩、赵、魏三国先后认了军次之后，燕宫中那盏夜灯便接连亮了三夜。

乐毅没有立刻宣“大事已定”。

他反而更慢了。

因为他知道，认责、认次之后，最容易坏人的，便是“等一等再定日”。日一拖，胆就散；胆一散，前面好不容易咽下去的那几口苦，也会慢慢从喉头里退回去。

第四夜，他终于把新的空简摊开，只写了一句：

“会于何日。”

燕昭王站在案旁，看着这四字，问：“你欲定几时？”

乐毅道：“不可太远。远则齐再多走几步，诸国又各自改主意；也不可太近。近则许多该备的粮、路、次第来不及实。”

说完，他把几国近来回简一一排开，最后把手按在中间，定了一个日期。

那日不算近，也绝不算松。

足够让赵验粮站，

让魏定节应，

让韩整前军，

也足够让齐在这段不长不短的空里，再露出一层更高的势。

燕昭王看完，只问一句：“你敢发？”

乐毅答：“到这里，不发，比发更险。”

于是燕使再出。

这一回带去的，是定期简。

邯郸先得。

赵王看完日期，没有先问别的，只问赵奢：“来不来得及？”

赵奢心里一算，道：“若今日起便先验北线三站、腾两仓、改两段押运旧例，来得及。”

赵王点头：“那便不是日期太急，是我赵手若再慢，便自己坏了前约。”

这话说得殿中众臣都不敢再拿“可否缓两日”作托辞。

新郑那边，韩王看完日期，也只是沉声道：“乐毅这是连诸侯喘气的余地都压着了。”

那老将却道：“压着才对。再让人喘久了，前军两个字就会越看越重，越看越不想认。”

韩王听完，竟没有驳。

他只把日期压在前军簿上，道：“回燕。韩按期整前军，但请主帅之令先行一份。”

大梁看见定期简时，魏王眉头皱得最深。

可他最终也只改了一句：“魏按期整中军节应，但诸军联络号次，当先定而后行。”

这回，连魏都已不再争“缓”。

夜里，苏秦在邯郸馆中看完诸国回书，久久没有说话。

门客低声道：“相国，眼下几国都已认责、认次、认期，这不正是你当年最想看到的局么？”

苏秦看着案上一卷卷新简，神情却极复杂。

过了很久，他才道：“像，又不像。”

“何意？”

“像在于，天下又有了要往一处拧的气。”苏秦道，“不像在于，这口气已不再是靠一篇篇大言、一次次歃血，把人先热起来。它是被齐一步步逼着，再被乐毅一条条压着，在责任、军次、会期里慢慢做出来的。”

他说到这里，忽然笑了一下。

那笑意很淡，也很苦。

“旧纵横，到我这里，其实已经很远了。”

门客一时不知如何接。

苏秦却并不伤怀太久。

他把那卷定期简慢慢卷起，轻声道：“远也无妨。若这条路真能让天下在齐之前先并成一股，便说明这个时代，总还有人能在旧法不灵之后，把新法做出来。”

灯影落在他手上，照见指节已比前些年更瘦，也更硬。

窗外邯郸夜风一起，门缝间便有细细凉意钻进来。

苏秦把那卷简压到案角，没有再看。

因为他知道，从这夜起，自己看到的已不再是一个靠口舌来回扳的大局。

而是另一种更硬、更慢、也更难回头的东西，终于真的往前走起来了。
