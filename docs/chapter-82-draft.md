# 第八十二回 齐王再问东边谁敢动 乐毅临会前夜兵气先成

燕使把会期定出去之后，山东的风忽然就变了。

不是人人看得见。

城中百姓照旧买卖，馆舍使者来往也照旧周旋。

可坐在几国案前的人，都已经开始按那日子往回数。

还剩几日，

哪一站粮该先到，

哪一支军该先起，

谁的号令若慢半刻，后面便要整段乱掉。

兵气往往不是起在战鼓响时。

它也会起在这种各国都忽然不再空谈、而开始按着日子去改自己手边旧例的时候。

临淄那边，齐湣王却还未真正觉出这股气已在外间慢慢结实。

他近来最在意的，仍是宋城既下之后，周边诸国辞色有没有再低一些。

这一日朝会，他问得很直：“赵、魏、韩、燕近来可有什么异动？”

边吏回道：“使者仍往，书仍通，并无大举。”

齐湣王听完，神色极淡，像这答案正合他意。

又有近臣顺势道：“大王，诸国纵有心，也不过纸上来回。宋城既破，谁还敢真先出头？”

田文站在阶下，听得指尖一点点发冷。

因为他比谁都清楚，真正危险的往往不是“已有大举”，而是外头已经在按日备粮、按次改令，而临淄里的人却还把“尚无大举”四字当成安心的凭据。

他终究还是出班：“大王，外间越静，越不可只拿‘尚无大举’宽自己。”

齐湣王皱眉：“孟尝君，你如今是见什么都疑了。”

田文俯身：“臣不是多疑，是怕诸国若真动，也不会先把鼓敲给临淄听。”

这句话说得不算重。

可齐湣王的脸色还是淡了下来：“寡人只问你一句。如今赵还顾西，楚还自整，魏韩未复，燕又最弱。谁敢真先动？”

田文抬头，沉默了片刻，才道：“若还是去年，臣也许会答无人敢。如今臣不敢。”

齐湣王冷笑一声，没有再理他。

因为在他眼里，田文如今几乎已成了满朝最爱替外人长志气的人。

散朝之后，田文并未立刻回府。

他独自走到宫外高台，远远望向东南天际，胸口像压着一团始终散不开的灰。

他忽然明白，自己如今最怕的，已不是大王做错一步。

而是大王做对了太多步，于是从上到下都开始信：

齐的路，只会越走越顺。

而在更北边，会期前夜终于到了。

燕都之外，军马尚未大举开动，可各国先行而来的校尉、主簿、押粮吏、号令官，已陆续入燕。

他们带来的不是彩旗。

是簿册、号牌、道图、改过的军书。

一间偏厅里，赵来的人先核北线粮站；

另一间里，魏来的人在对中军号次；

韩前军的副将则亲自盯着路次与先行之节；

燕这边，乐毅没有坐在高位等报，而是一屋一屋地过，一条一条地问。

“赵若第一站迟半日，后面改哪仓补？”

“魏中军接韩前军时，号角先后谁定？”

“韩前军若先见齐骑试探，不得擅逐，这条可都明了？”

他问得极细，也极冷。

可被问的人没有谁敢嫌烦。

因为到了这一夜，所有人都明白，这些问题若不先在灯下问死，后面便会在阵前用人命来答。

燕昭王夜里亲自来了一趟。

他没有进厅，只站在廊下，看见屋里灯一间接一间亮着，听见里头不是争吵声，便是压低了嗓子的应答声。

郭隗立在他身旁，低声道：“大王，这还未举兵，气却已先成了。”

燕昭王望着那一列列灯火，缓缓道：“从前只见人举杯结盟。今日才见，原来一股真兵气，也能先从这些最不见光彩的地方长出来。”

就在这时，乐毅从一间偏厅里走出，手里还拿着一卷新改过的号次简。

燕昭王看向他：“都定得住么？”

乐毅停了一下，才答：“还不能说全定住。可它们已经不是来问可不可，而是在问若明日便动，自己那一口该怎么顶。”

这句话说完，廊外夜风正起。

风穿过廊柱，也穿过一间间还亮着灯的屋子，吹得简角轻动，灯焰微斜。

而在那些灯下，一群原本各算各的国家，已第一次不再只在书上说“愿共”。

它们开始真把兵，往同一个时日、同一套次第、同一种不能再随手后退的规矩里，慢慢并过去了。
