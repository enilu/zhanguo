# 第八十二回 齐王再问东边谁敢动 乐毅临会前夜兵气先成

燕使把会期定出去之后，山东的风便悄悄变了。

城中百姓照旧买卖，馆舍使者照旧往来，可几国案前的人都已开始按着那日子往回数：哪一站粮先到，哪一支军先起，哪一道令若慢半刻，后头整段都要乱。

临淄那边，齐湣王却还未觉出这股气已在外间慢慢结实。

这一日朝会，他问得很直：“赵、魏、韩、燕近来可有什么异动？”

边吏回道：“使者仍往，书仍通，并无大举。”

齐湣王听完，神色极淡。近臣顺势道：“诸国纵有心，也不过纸上来回。宋城既破，谁还敢真先出头？”

田文站在阶下，只觉指尖一阵阵发冷。他终究还是出班：“大王，外间越静，越不可只拿‘尚无大举’宽自己。”

齐湣王皱眉：“孟尝君，你如今是见什么都疑了。”

田文俯身：“臣不是多疑，是怕诸国若真动，也不会先把鼓敲给临淄听。”

齐湣王冷笑：“如今赵还顾西，楚还自整，魏韩未复，燕又最弱。谁敢真先动？”

田文抬头，隔了片刻才道：“若还是去年，臣也许会答无人敢。如今臣不敢。”

齐湣王没有再理他。

会期前夜终于到了。

燕都之外，军马尚未大举开动，可各国先行而来的校尉、主簿、押粮吏、号令官，已陆续入燕。他们带来的不是彩旗，而是簿册、号牌、道图、改过的军书。

一间偏厅里，赵来的人先核北线粮站；

另一间里，魏来的人在对中军号次；

韩前军的副将则亲自盯着路次与先行之节。

乐毅没有坐在高位等报，而是一屋一屋地过，一条一条地问：

“赵若第一站迟半日，后面改哪仓补？”

“魏中军接韩前军时，号角先后谁定？”

“韩前军若先见齐骑试探，不得擅逐，这条谁签押？”

他问得很细，也很冷。被问的人却没有谁敢嫌烦。

夜里，燕昭王亲自来了一趟，没有进厅，只站在廊下。屋里灯一间接一间亮着，低低的应答声与翻简声不断传出来。郭隗立在他身旁，低声道：“大王，这还未举兵，气却已先成了。”

燕昭王望着那一列灯火，没有立刻答。就在这时，乐毅从一间偏厅里走出来，手里还拿着一卷新改过的号次简。

燕昭王看向他：“都定得住么？”

乐毅停了一下：“还不能说全定住。可它们已不是来问可不可，是在问若明日便动，自己那一口该怎么顶。”

廊外夜风正起，吹得简角轻轻相击。那些还亮着灯的屋子里，一卷卷簿册摊着，一枚枚印记新按上去。兵还没出门，规矩却已先压到了案上。
