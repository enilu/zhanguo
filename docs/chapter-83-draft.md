# 第八十三回 燕南会师诸军先见主 乐毅临众不言先整号

会期到了。

燕南大营外，天还未亮，地上先已有一层被马蹄踏得发黑的薄霜。

最先到的是赵车与粮骑。

他们不抢前头，只按此前定好的北线次序，把一车车号牌、一卷卷粮站簿书先送进营中。押运校尉下马之后，第一句话不是问哪国到了多少兵，而是问：“北线第三站新改那处仓，谁来对验？”

营门吏立刻引他入内。

不多时，韩前军也到了。

前列甲厚，兵不甚喧。那名先前在韩廷里一再问“谁来做最不好看的差”的老将，这回亲自压阵。他进营之后，先盯的不是迎接礼数，而是营中旗位与前军行列是不是照简上所定。

再后是魏中军节应的人马。

魏军到得最稳，车骑不显急，却一到便先对号次、对传令、对前后应答鼓。魏相未亲至，来的是军中最老成的一名节应将。他一眼看见韩、赵两边已各就各位，脸色虽还沉，却没有再说“可再看”。

因为到这时候，再看，已没有地方看了。

日色一点点上来，燕营中军鼓终于响了三通。

诸军主将、使臣、校尉、号令官、主簿一并入中军大帐。

帐极大。

却并不华。

案上铺着的，不是酒器，而是此前诸国认责、认次、定期、改号所积下的那一卷卷旧简与新图。

乐毅站在案后，没有立刻开口。

他先看了一遍帐中众人。

韩将眼神最硬；

赵奢与押粮校尉立在一处，先看的是案边那几道粮站标记；

魏节应将神色最稳，也最冷；

燕这边诸将则都压着一口不敢外泄的气。

这一眼看完，乐毅才道：“先对号。”

帐中诸人一时都没反应过来。

因为谁都以为，会师第一句话该是壮言、是盟誓、是晓谕天下的大词。

可乐毅偏偏只说了这三个字。

他说完，便命主簿把四国号次简逐卷摊开，一条条念：

“前军，韩。”

“北线粮路，赵。”

“中军节应，魏。”

“总节与会兵主议，燕。”

每念一条，对应主将便上前应一声。

声音不高。

却比满帐豪语更沉。

因为这不是在热血里彼此举杯。

这是在把先前各国咽下去的那些苦差，一条条当众认给所有人听。

念完之后，乐毅才抬头，道：“诸位今日来此，不是来学别人怎么好看。是来把自己那一口最不好看的差，当众再认一遍。认完之后，后面若有人乱次、误号、失期、先退，帐中人人都知道该先问谁。”

帐里极静。

静得能听见帐外风吹旌角的轻响。

韩那老将忽然先上前一步，道：“韩前军已到，不问别的，只问中军接不接得住。”

魏节应将冷冷看了他一眼：“魏既来了，便不是来看你韩独自冲阵。”

帐中气口一紧。

若在往常，这样一句话很容易就成了两国旧气再翻。

可乐毅没有让它往那边走。

他只把手落在案上前、中、后三处标记之间，道：“好。既都在问接不接得住，那便先试。”

说完，他当场命外营起一阵短号。

韩前军先出营三十步；

魏中军半刻后应；

赵押粮骑再按预定道口移一站；

燕本部只看不插。

这一下命令来得极快，连许多随行使臣都愣住了。

可帐中诸将反倒同时精神一紧。

因为他们明白，乐毅不是在立威。

他是在告诉众人：

今日会师，第一件事不是彼此把话说圆。

是先看这套已经在简上改了许久的东西，落到脚下究竟行不行。

帐外很快起了动静。

马蹄，号角，传令，车辙，短得像只是一阵试手。

可正因短，更叫人发寒。

因为这一试若乱，便说明前面所有好不容易认下去的字，仍只是纸上之字。

乐毅站在帐内，没有出去看。

他只低头听。

听号有没有误半拍；

听传令是乱成一团，还是各有节；

听帐中众人呼吸有没有跟着那阵外头的响动一起绷起来。

片刻之后，外头回报入帐：

“前军出而不乱。”

“中军应而不迟。”

“粮骑移站，无误。”

这一连三句报进来，帐中那股原本还横着的旧气，竟都微微松了一松。

不是尽散。

可至少第一口最易坏人的试，过了。

乐毅这时才第一次露出一点极淡的神色，道：“诸位，从今夜起，简上的字，便不只是字了。”

他说完，终于抬手，命左右奉酒。

酒很薄。

每人只一爵。

乐毅自己先举起，却没有说什么“共图大事”的热话，只道了一句：

“先敬诸位今日都肯把最难看的那一口，当众认下来。”

满帐无人失声。

可这一爵酒饮下去的时候，诸军心里都已清楚：

从这一夜起，他们不再只是各国来的人。

他们已是真正会在一面总节之下、往同一个方向去的人了。
