# 第八十四回 乐毅总节联军先南下 临淄未觉风已近城根

会师之后，乐毅没有给诸军太久缓气。

次日天刚发白，中军总节令便已发遍诸营：

前军先行，

中军半日后继，

粮骑按北线三站分批下移，

各军号旗、鼓次、夜宿之所，俱不得擅改。

韩前军得令最快。那韩老将看完，第一次没有再问“后应当不当真”，只把军书递给副将：“照行。”

赵奢这边则亲自去看北线第一站发车，盯的仍是封泥、车辙、押牌、补马。随行小吏低声道：“赵君，如今大军都已南下，还要这样一车车盯？”

赵奢头也不抬：“后头一车乱了，前头一营就要跟着乱。”

魏中军节应将领到军书之后，也没有多话，只命手下先把联络号次再演一遍。韩前军先动时，魏鼓何时起；赵粮站若报迟，魏讯骑先往哪一线接；若燕总节改路，魏中军如何转而不乱。

燕本部居中最静。乐毅自己并不骑在最前，他走得不快，却每到一处要道、每过一处合流点，都要停下来再问一次前后军次与粮路是否仍照简行。

联军自燕南下第一日，竟真没乱。

韩前军没有擅快，

魏中军没有误后，

赵粮车也没压成死串。

临淄那边，却还只当外间仍是照旧互致书简、互试辞色。

齐湣王这一日甚至还召近臣再议宋后之事，问的是要不要借宋新下之势，再敕周边诸国贡数有改。边吏回报，说赵、魏、韩、燕近来虽使节频仍，却仍未见大军压境。

齐湣王听后，眉头松了些：“寡人就知道，它们胆终究还是不够。”

田文站在下首，听得心口发冷，却没有再追着劝。散朝后，他独自回府，把近来零零散散收来的各国辞色一封封摊开。信都写得稳，稳得几乎没有破绽。

他把最后一封压下去，低声道：“不是没来，是已经在路上了。”

这一边，联军南下第一日的暮色也正压下来。

韩前军已过一处浅谷，

魏中军刚好接上，

赵粮骑在北线点起第一批夜火，

燕总节营则立在几路交会处。

乐毅站在高处，望着四面夜火，没有说话。那些火一盏盏沿着军路往南排下去，已不像各国旧营各照各的散灯，更像一条慢慢朝临淄压过去的长线。
