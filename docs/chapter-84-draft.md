# 第八十四回 乐毅总节联军先南下 临淄未觉风已近城根

会师之后，乐毅没有给诸军太久缓气。

次日天刚发白，中军总节令便已发遍诸营：

前军先行，

中军半日后继，

粮骑按北线三站分批下移，

各军号旗、鼓次、夜宿之所，俱不得擅改。

这道令不像讨齐檄文。

更像一把先把乱麻攥死的绳。

韩前军得令最快。

那老将看完，总算第一次没有再问“后应当不当真”，只把军书递给副将，道：“照行。今日若再慢半刻，后头连骂谁都不用骂了。”

赵奢这边则亲自去看北线第一站发车。

他盯得仍是最细的地方：封泥、车辙、押牌、补马。随行小吏低声道：“赵君，如今大军都已南下，还要这样一车车盯？”

赵奢头也不抬，只道：“兵在前头，气在后头。后头一车乱了，前头一营就会跟着乱。”

魏中军节应将领到军书之后，也没有多话，只命手下先把联络号次再演一遍。韩前军先动时，魏鼓何时起；赵粮站若报迟，魏讯骑先往哪一线接；若燕总节改路，魏中军如何转而不乱。

他一条条问得极细。

帐下副将听到后来，忍不住道：“将军，我们像不像还没上阵，便先把自己折磨死了？”

那节应将看了他一眼：“正因不想阵上死，才得现在先把这些问死。”

燕本部居中最静。

乐毅自己并不骑在最前。

他走得不快，却每到一段要道、每过一处合流点，都要停下来再问一次前后军次与粮路是否仍照简行。有人私下议论，说这位主将不像来领大胜，倒像个拿着竹简逐寸校路的工吏。

可议论归议论，谁都不敢轻慢。

因为联军自燕南下这第一日，竟真没乱。

前军没有擅快，

中军没有误后，

粮车也没有压成死串。

这在几国并兵之中，已算极难得的好兆。

而在临淄，齐湣王此刻还只当外间只是照旧互致书简、互试辞色。

这一日他甚至还召近臣再议宋后之事，问的是要不要借宋新下之势，再敕周边诸国贡数有改。边吏回报，说赵、魏、韩、燕近来虽使节频仍，却仍未见大军压境。

齐湣王听后，眉头松了些：“寡人就知道，它们胆终究还是不够。”

田文站在下首，听得心口发冷，却也发闷。

因为他知道，最糟的时候，往往不是临敌。

而是你已站在敌风之前，却还把“尚无大军到城下”当成一切都还来得及的凭据。

散朝之后，他没有再追大王。

因为说到这里，再追，也只会让自己显得像个日日惊风的乌鸦。

他只独自回府，把近来零零散散收来的各国辞色一张张摆开。

摆得越久，他心里越沉。

那些信都不算硬。

可硬处恰恰在这里：

太稳，

太静，

静得像人人都忽然学会了不再把最要紧的东西写给临淄看。

他把最后一封信压下去，喃喃道：“不是没来。是已经在路上了。”

同一时刻，联军南下第一日的暮色正压下来。

韩前军已过一处浅谷，

魏中军刚好接上，

赵粮骑在北线点起第一批夜火，

燕总节营则立在几路交会处，灯火一层层传开。

乐毅站在高处，望着四面夜火，脸上仍没有什么喜色。

随从低声道：“君上，第一日顺过来了。”

乐毅道：“第一日顺，不算什么。”

“那何时才算？”

乐毅望着远处南面的黑，缓缓道：“等这些火，真的照到临淄外头，而诸军还能像今日这样，不各自先乱。”

风从南面吹来，带着一点远处水气。

那是往齐去的风。

今夜它先吹过联军各营，把灯火吹得微微一晃。

可灯没有灭。

它们一盏盏沿着军路往南排下去，已不像零散夜灯。

更像一条终于从纸上走出来、正缓缓朝临淄方向压过去的长火。
