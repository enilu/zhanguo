# 第八十五回 郢都加整东储终见效 怀王临风忽知不可后

联军南下的消息，并没有立刻传满天下，可风比人快。邯郸、大梁、新郑几处驿道先急起来，郢都这边也慢慢觉出了不对。

黄歇近来一直守在东面兵储与朝辞两头之间，忙得整个人都瘦了一圈。这一日，东面新改的三处粮站第一次按新例同时点验，从前最爱拖字、最爱说“且缓一缓”的几道旧关口，这回竟都没出岔。

押运吏来报时，昭阳先是不信，亲自对了三遍封泥、车数、到时。对完之后，才把那卷新簿慢慢合上：“总算有一回，不是等到急时才知道哪处是空的。”

黄歇站在旁边，只道：“楚若总是临到跟前才补，外头风一转，便只能追着别人跑。”

同一日，屈原入兰台，看见新旧两套东储簿书终于第一次能一一对上，也停了片刻。案上簿书压得整整齐齐，看着仍枯，可至少不再是一推就散的旧样子。

夜里，怀王独坐宫中。近臣入报：“东面三站新验已齐，无误。”

怀王抬头，竟先愣了一下。他沉默片刻，忽然问：“黄歇在何处？”

黄歇奉召入内时，怀王没有先提齐、秦，也未先提山东，只把那份新验之簿推过去：“这是你近来做出来的？”

黄歇答：“臣不敢独居其功。”

怀王看着他，过了一会儿才问：“如今外头这风，楚还能不能站到后头再看？”

黄歇没有立刻答，最后只道：“能看。但不能后。”

怀王抬眼：“何解？”

黄歇道：“后看，是等别人先把局推到眼前，再择利而入。如此看久了，外人只会记住楚总有后手。”

怀王低头看着那份东储新簿，许久才缓缓道：“那便不能再只后了。”

窗外宫灯被风吹得一明一暗。怀王坐在灯下，又把簿书翻了一遍。这一回，他看的不再是谁先走到了哪一步，而是楚自己脚下这几处仓、这几道站，究竟有没有先站稳半寸。
