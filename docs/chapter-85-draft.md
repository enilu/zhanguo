# 第八十五回 郢都加整东储终见效 怀王临风忽知不可后

联军南下的消息，并没有立刻传满天下。

可风比人快。

先是邯郸、大梁、新郑几处驿道忽然都变得比往日更急，随后便连郢都这边，也慢慢觉出不对。

黄歇近来一直守在东面兵储与朝辞两头之间，忙得整个人都瘦了一圈。昭阳原先只服他会写、会理，到这时才真正看见，这年轻人最难得的地方，不是懂得说该补哪几处，而是能日复一日把那些最枯的窟窿真补上。

这一日，东面新改的三处粮站第一次按新例同时点验。

从前最爱拖字、最爱说“且缓一缓”的几道旧关口，这回竟都没再出岔。

押运吏来报时，昭阳先是不信，亲自对了三遍封泥、车数、到时。对完之后，他把那卷新簿慢慢合上，竟许久没说话。

黄歇站在旁边，也没有催。

过了很久，昭阳才吐出一句：“总算有一回，不是等到急时才知道哪处是空的。”

黄歇低声道：“这便是臣近来一直不敢松的缘故。楚若总是临到跟前才补，外头风一转，便永远只能追着别人跑。”

昭阳看了他一眼，忽然道：“你这句话，倒像不是说给我听。”

黄歇没有回避：“臣也说给自己听。”

同一日，屈原入兰台，看见新旧两套东储簿书终于第一次能一一对得上，心里也轻了一寸。

不是因为楚自此便强。

而是因为这至少说明，黄歇这一路“先补最慢最重之处”的做法，开始真能顶住外局往前压来的风了。

夜里，怀王独坐宫中，外头风正大。

他近来已经越来越少像从前那样一听外报先问“楚可借什么”。宋破之后、齐势更高、山东诸国辞色忽稳，这几件事像几块冷石，一块块压进他心里，把那些旧日最顺手的侥幸都压得薄了许多。

这时，近臣来报：“东面三站新验已齐，无误。”

怀王抬头，竟先愣了一下。

因为这类话，过去他听得极少。

不是没人做事。

是许多事总做到一半便被别的朝议、别的外书、别的心思给扯散了。

如今竟真有一条线，能不靠他一时起意，也一步步走到底。

他沉默片刻，忽然问：“黄歇在何处？”

黄歇奉召入内时，怀王并未先提齐、秦，也未先提山东，只把那份新验之簿推过去：“这是你近来做出来的？”

黄歇答：“臣不敢独居其功。令尹、昭阳与各署吏卒皆在其内。”

怀王看着他，问：“寡人若问你一句实话。如今外头这风，楚还能不能站到后头再看？”

黄歇没有立刻答。

他知道，这一句若说得太直，会像逼王；说得太轻，又会白白放过怀王此刻少有的一点真心。

最后他只道：“能看。但不能后。”

怀王抬眼：“何解？”

黄歇道：“后看，是把自己放在外势之后，等别人先把局推到眼前，再择利而入。如此看得久了，外人便先记住楚总有后手。大王若今日还这样，后面不论山东是合是散，楚都只会越来越像站在最后一层捡人余势的国。”

这话很平。

却比从前许多硬谏都更准地落在怀王心上。

怀王低头看着那份东储新簿，许久才缓缓道：“那便不能再只后了。”

黄歇没有立刻接话。

因为他知道，这一句虽还不是决断，却已比楚这些年许多朝会上的硬话都更重。

窗外宫灯被风吹得一明一暗。

怀王坐在灯下，把那卷簿书又翻了一遍。

这一次，他终于不是在看某一处可借的利，也不是在看别人先走到了哪里。

他是在看，楚自己脚下，终于有没有一块能站得更稳一点的地方。
