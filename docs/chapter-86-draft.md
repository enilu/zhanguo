# 第八十六回 联军夜宿诸营初见整 乐毅临图知大战将开

联军南下第二日，齐境的风便已隐隐能吹到前营脸上。

路上仍未见齐大军正面来撞。

可正因如此，乐毅反而更不敢松。

大军初合，最怕的从来不是前头忽见敌旗。

而是敌旗未见时，各军先拿“看来无事”四字慢慢把自己那根弦放软。

这夜宿营，乐毅特意没有让各军各扎各的旧例。

韩前军营位略前，却必须与魏中军号角相接；

赵粮骑夜火不得只照己营，还要照北线转道口；

燕总节营则设在几处号令交会之中，不高不远，恰好能听见四面应答。

韩那老将最初看见这营位，还皱了一下眉：“乐君，你这是不信韩能自守？”

乐毅答：“臣不是不信韩。臣是不信大军初合之夜，谁都能只靠自己一营的旧法守住全局。”

老将盯着他看了片刻，终于没再说什么，只回身去盯自家前营夜令。

魏中军那边，节应将也把原本魏军旧用的两道夜号先压掉一半，改用联军新号。帐下有人低声道：“将军，何不仍用本军旧号？自家人更熟。”

那节应将只冷冷道：“正因你熟，旁人不熟。你若夜里先把自己叫顺了，白天便会把联军叫乱。”

这话一出，再没人敢多嘴。

赵奢则带人绕着粮营、转道、补马点走了整整一圈，回来时靴上全是泥。押运校尉见他这般，忍不住道：“赵君，前头都未见齐兵，何至如此？”

赵奢把手里那根试地的短杖往地上一插，道：“前头若真见齐兵，便还有将军们盯着。后头若在未见齐兵时先松了，前头打得再好，也会自己饿回去。”

夜渐深时，几处营火次第亮起。

前营，

中军，

粮骑，

总节。

一层层排开，并不整齐得像一条线，却也再不是几国旧营各照各的散火。

燕昭王并不在军中。

可郭隗这夜随军来看，站在高坡上望着那些火，心里也慢慢起了一层说不清的重意。

因为这已不是纸上盟书，也不是殿中壮语。

而是一支真正会在夜里互相听号、白日彼此等节的大军，开始长出它自己的筋骨。

乐毅这时还未歇。

他独自回到中帐，帐中只点一灯，案上摊着齐境图、宋后新报、联军当下路次与各营今日应答记。

他把图上几处可能先遇齐骑、先逼齐边、先惹齐疑的地方一一看过，最后把手按在临淄北线外那几处最容易让齐错判联军轻重的道口上，久久未动。

随从轻声问：“君上，可要歇了？”

乐毅摇头：“还不能。”

“为何？”

乐毅没有立刻答。

过了片刻，才道：“会师、定期、排次、南下，到这里都还只是把我们自己拧成一股。后面真正要紧的，是让齐在第一眼看见我们时，先看错半寸。”

随从听得心里一紧。

因为他知道，这意味着主将心里已不再只想着如何把几国并住。

他开始真正对着齐排阵了。

帐外夜风掀起帐角一线，吹得灯火轻轻偏了一下。

乐毅伸手按住图角，目光却仍落在那几处道口上，没有移开。

因为他知道，第三卷这一大段由冷简、苦差、军次、会期一点点逼出来的兵势，到这一夜，终于要从“怎样不先自乱”，转到“怎样先让齐乱一眼”的时候了。

而那，才是真正的大战开口。
