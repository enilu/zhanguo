# 第八十六回 联军夜宿诸营初见整 乐毅临图知大战将开

联军南下第二日，齐境的风便已隐隐能吹到前营脸上。

路上仍未见齐大军正面来撞，可乐毅反而更不敢松。大军初合，最怕的不是前头忽见敌旗，是敌旗未见时，各军先拿“看来无事”四字把那根弦放软。

这一夜宿营，乐毅没有让各军各扎各的旧例。

韩前军营位略前，却必须与魏中军号角相接；

赵粮骑夜火不得只照己营，还要照北线转道口；

燕总节营则设在几处号令交会之中，不高不远，恰好能听见四面应答。

韩老将最初看见这营位，还皱了一下眉：“乐君，你这是不信韩能自守？”

乐毅答：“臣不是不信韩，是不信大军初合之夜，谁都能只靠一营旧法守住全局。”

魏中军那边，节应将也把原本魏军旧用的两道夜号压掉一半，改用联军新号。帐下有人低声道：“将军，何不仍用本军旧号？自家人更熟。”

那节应将冷冷道：“正因你熟，旁人不熟。你若夜里先把自己叫顺了，白天便会把联军叫乱。”

赵奢则带人绕着粮营、转道、补马点走了整整一圈，回来时靴上全是泥。押运校尉忍不住道：“赵君，前头都未见齐兵，何至如此？”

赵奢把手里那根试地短杖往地上一插：“前头若真见齐兵，还有将军们盯着。后头若在未见齐兵时先松了，前头打得再好，也会自己饿回去。”

夜渐深时，几处营火次第亮起：

前营，

中军，

粮骑，

总节。

一层层排开，已不再是几国旧营各照各的散火。

乐毅这时还未歇。他独自回到中帐，帐中只点一灯，案上摊着齐境图、宋后新报、联军当下路次与各营今日应答记。

他把图上几处可能先遇齐骑、先逼齐边、先惹齐疑的地方一一看过，最后把手按在临淄北线外几处最容易让齐错判联军轻重的道口上，久久未动。

随从轻声问：“君上，可要歇了？”

乐毅摇头：“还不能。”

“为何？”

乐毅道：“会师、定期、排次、南下，到这里都只是先把自己拧成一股。后面要紧的，是让齐第一眼看见我们时，先看错半寸。”

帐外夜风掀起帐角一线，吹得灯火轻轻偏了一下。乐毅伸手按住图角，没有再说别的。灯下那几处道口被指节压着，像是大战还未开门，门缝先被他看准了一道。
