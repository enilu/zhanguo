# 第八十七回 燕骑先放假影试齐眼 乐毅定谋只求敌先错

联军南下第三日，乐毅把大图卷起，只留北线外几处道口与浅谷的局部。

他没有急着把大军尽数压前，只先调了两支最轻的燕骑，又从韩前军里借出一队熟地形的斥候，分作两路：

一路故意露形，

一路专走浅谷。

韩老将听完，皱眉道：“大军已到这里，还要先做这等轻骑把戏？”

乐毅答：“越到齐境前，越不能叫它第一眼就看准我们全形。”

当日申时，燕骑先出。故意露形那一路，从最像主路的北坡慢慢现出旗角，不快，也不逼近，像是在替大军先量坡口；另一边，藏着韩、燕精骑的那一路，却从浅谷低处悄悄穿过去，只在几处最不该留痕的泥地上故意踩出浅浅马迹。

齐边果然很快得报。

北外一处边垒中，守将把两份斥候回报摊在案上：一份说北坡见燕骑与韩斥候并影，一份说浅谷有零散马痕，却不见旗号。

守将想也没想，便先指向北坡：“主路在此。联军若真来，必从这里露头。”

副将低声道：“那浅谷呢？”

守将冷笑：“车走不得，旗也展不开，不过几骑试探。先盯正路。”

这道判断很快飞往临淄。

齐湣王听完边报，只是微微一挑眉：“还是燕先来弄虚声。”

近臣立刻附和。田文站在下首，却听得心里发凉。他知道，大王如今不是没听见异动，是先替每一道异动都找到了最顺手的轻解释。

入夜后，联军营里也有回报到了：齐边主垒已把眼先压在北坡，对浅谷一线只留了寻常探骑。

乐毅听完，只把回报压在图上：“好。明日让它再看深一点。”

韩老将问：“主将，你要它看错到什么地步？”

乐毅指着浅谷后一线：“错到它觉得我们还只是几国并来的影，不是一口真能咬下来的兵。”

赵奢在旁接了一句：“若齐照这条错路去备，它的粮与救应也会先慢半拍。”

乐毅点头：“我要的便是这半拍。”
