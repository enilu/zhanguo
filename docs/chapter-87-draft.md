# 第八十七回 燕骑先放假影试齐眼 乐毅定谋只求敌先错

联军南下第三日，乐毅终于把那张反复看过的齐境图卷起，只留了北线外几处道口与浅谷的局部。

他没有急着把大军尽数压前。

而是先调了两支最轻的燕骑，又从韩前军里借出一队熟地形的斥候，命他们分作两路：

一路故意露形，

一路专走不该走的那道浅谷。

韩那老将听完，皱眉道：“主将，大军已到这里，还要先做这等轻骑把戏？”

乐毅答：“越到将近齐境，越不能叫它第一眼就看准我们全形。”

魏中军节应将也抬眼问：“若齐边骑顺势来咬呢？”

乐毅道：“便让它来咬。只要它第一口咬在错处，后面一日两日，便都是我们的气。”

这话说得极平。

可帐中诸将听完，心里都微微一紧。

因为他们明白，主将如今真正要抢的，已经不是哪一处路，而是齐第一次看见联军时心里那一下先偏去哪边。

当日申时，燕骑先出。

故意露形那一路，从一条最像主路的北坡慢慢现出旗角，不快，不远，也不肯逼太近，像是大军前锋试路；

而真正藏着韩、燕精骑的那一路，却自浅谷低处悄悄穿过去，只在最该叫齐以为“这里绝无大军会走”的地方，故意留下极浅的马迹。

齐边果然很快得报。

临淄北外一处边垒中，守将把两份斥候回报摊在案上，一份说北坡见燕骑与韩斥候并影，一份说浅谷有零散马痕，却不见旗号。

他想也没想，便先指向北坡：“主路在此，联军若真来，必从这里露头。”

副将迟疑了一下：“那浅谷……”

守将冷笑：“那地方车走不得、旗出不齐，不过几骑试探罢了。先盯正路。”

这道判断很快飞往临淄。

齐湣王听完边报，眉头却只微微一挑：“果然还是燕先来弄虚声。”

近臣立刻附和：“燕最会借势，不足为怪。”

田文站在下首，心里却一点点发凉。

因为他听得出来，大王如今已不只是轻外人。

而是在任何新来的异动前，都先替自己找到了一个最顺手、也最让人安心的旧解释。

“燕弱，故虚张。”

“诸侯散，故无大举。”

“宋既破，故人心更怯。”

这些话一遍遍说下来，到后来便会像一层极薄极顺的膜，先把所有真正不该被轻轻滑过去的东西都盖住。

而在联军营中，入夜后便有回报到了。

齐边主垒已把眼睛先压在北坡正路，对浅谷一线只留了寻常探骑。

乐毅听完，没有露喜，只道：“好。明日让它再看深一点。”

韩那老将忍不住问：“主将，你到底要它先看错到什么地步？”

乐毅把手落到图上一线，道：“错到它觉得我们还只是几国并来的影，不是一口真能咬下来的兵。”

赵奢在旁听着，忽然道：“若齐真照这条错路去备，后面它的粮与救应也会先慢半拍。”

乐毅点头：“我要的便是这半拍。”

帐外夜风很硬，吹得旗角猎猎作响。

乐毅抬头望了望北边天色，神色仍平。

可帐中众人已都明白，联军南下到这里，终于不只是自己如何不乱。

它开始真正伸手去拨齐的眼、齐的心、齐的第一口判断了。
