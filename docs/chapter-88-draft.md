# 第八十八回 临淄边报入殿仍半信 田文再阻只觉满城轻

第二日清晨，齐边又报。

这一次，北坡主路上的燕骑更像样了些。

旗号比前一日多，

往来次数也更密，

甚至在离边垒不远的一处高坡上，故意停了片刻，像在看地。

守将越发认定，这便是联军前锋露头之处。

而浅谷那一路，仍只见些零乱马迹，偶有探骑一晃便没。副将本想再提一句“会不会声东”，守将却已先摆手：“大军并来，粮车总得走正。它想拿几条马迹吓我，未免把齐边看得太浅。”

边报入临淄时，齐湣王正与近臣议宋后续处置。

听完之后，他竟笑了一声：“果然不过如此。”

田文心里一沉：“大王何以见得？”

齐湣王道：“真敢大举来的，不会先拿这种轻骑影子来回试人。燕、韩、赵、魏若真并，先要顾的事太多，哪里来得这么快。此刻不过是燕想借宋后之势，把自己抬高一点罢了。”

殿中近臣听完，也都跟着松下气来。

有人道：“大王所见极是。”

有人道：“若它们真有胆，早该旌旗蔽野，岂会如此遮遮掩掩。”

田文站在原地，只觉得那口冷意越来越深。

因为他听见的，不只是大王一句判断。

而是一城朝气都在顺着那句判断往最轻、最舒服的地方滑。

他终究还是出班。

“大王，臣只怕它们不是不敢来，而是知道临淄如今最爱听的，正是‘它们不敢来’。”

齐湣王抬眼，神色已极淡：“孟尝君如今连边报也不信了？”

田文俯身：“臣信边报。臣只是不敢只信边报第一眼看见的那层形。”

齐湣王冷冷道：“寡人看你是被几封外书、几次异动，自己先吓住了。”

田文没有再争。

因为他忽然明白，到了这一步，再把“别只信第一眼”说给满殿听，已几乎像是往一面极滑的墙上掷石。

石会碎。

墙却未必有痕。

散朝之后，他没有回府，反而转去见了自己一个向来不多言的老门客。

老人听完今日朝中诸语，只低声道：“大王若总爱把最顺耳的一层当成全局，后头就不是无人提醒，是提醒已经进不去。”

田文苦笑：“我早知。只是没想到齐会这么快，快到满城都跟着信，天底下真没人敢先动它。”

老人道：“城太盛时，最怕的不是外头没人说坏话。是城里人人都替盛作证。”

这句话很轻。

却像细针一样，直扎到田文心里。

而在更北边，乐毅已经接到第二拨齐边反应。

他听完之后，只把两份回报并在一处，看了片刻，忽然对韩那老将道：“今夜前军移半营。”

老将一怔：“移哪边？”

“移到浅谷后。”乐毅道，“不露旗，只动营。”

魏节应将听到这里，也立刻明白了：“主将是要让齐明日再看时，仍先盯正路。等真看见浅谷后有营火，它已来不及把头全转过去。”

乐毅点头：“正是。”

赵奢在旁忽然补了一句：“那北线粮骑也该陪着假路动一点，叫齐觉得它看见的仍是主路在吃重。”

乐毅看了他一眼：“依你。”

夜里，联军前营果然悄悄挪了一层。

动静不大。

可营火、号角、车痕、夜巡，偏偏都做得像原先那条北坡主路还吃着大军。

若非身在局中，几乎没人看得出，这一夜联军真正往前长出来的牙，已经慢慢压到了齐最轻看的那道浅谷后头。
