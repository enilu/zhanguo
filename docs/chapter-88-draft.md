# 第八十八回 临淄边报入殿仍半信 田文再阻只觉满城轻

第二日清晨，齐边又报。

这一回，北坡主路上的燕骑更像样了些。旗号比前一日多，往来次数也更密，甚至在离边垒不远的一处高坡上故意停了片刻，像是在看地。

守将越发认定，这便是联军前锋露头之处。浅谷那一路仍只见零乱马迹，偶有探骑一晃便没。副将本想再提一句“会不会声东”，守将却先摆手：“大军并来，粮车总得走正。几条马迹吓不住齐边。”

边报入临淄时，齐湣王正与近臣议宋后续处置。听完之后，他竟笑了一声：“果然不过如此。”

田文问：“大王何以见得？”

齐湣王道：“真敢大举来的，不会先拿这种轻骑影子来回试人。燕、韩、赵、魏若真并，先要顾的事太多，哪里来得这么快。”

殿中近臣听完，也都跟着松了气。田文却只觉那股冷意更深了。

他终究还是出班：“臣只怕它们不是不敢来，是知道临淄如今最爱听的，正是‘它们不敢来’。”

齐湣王神色一淡：“孟尝君如今连边报也不信了？”

田文俯身：“臣信边报。臣只是不敢只信第一眼看见的那层形。”

齐湣王没有再听。

散朝之后，田文没有回府，反倒转去见一位老门客。老人听完朝中诸语，只低声道：“城太盛时，最怕的不是外头没人说坏话，是城里人人都替这份盛作证。”

另一边，乐毅已接到第二拨齐边反应。

他把两份回报并在一处看了片刻，忽然对韩老将道：“今夜前军移半营。”

老将一怔：“移哪边？”

“移到浅谷后。不露旗，只动营。”

魏节应将立刻明白了：“主将是要让齐明日再看时，仍先盯正路。等它真看见浅谷后有营火，已经来不及把头全转过去。”

乐毅点头。

赵奢在旁又补了一句：“北线粮骑也该陪着假路动一点，叫齐觉得主路仍吃重。”

“依你。”

夜里，联军前营果然悄悄挪了一层。营火、号角、车痕、夜巡，都做得像原先北坡主路还吃着大军；真正往前长出的那一层，却已经慢慢压到了浅谷后头。
