# 第八十九回 浅谷后营一亮齐边惊 乐毅当机先夺第一势

第三日黎明，齐边探骑终于在浅谷后那片本不该有大营的地方，看见了新火。

火不多。

却整。

一线一线，正压在最容易遮住后军、也最能逼近齐边侧肋的地势后头。

探骑回报时，边垒守将脸色先是一僵，随即厉声道：“怎会在那里？”

副将低声道：“若联军前两日故意叫我们只盯北坡……”

守将猛地转头：“闭嘴。立刻再探！”

可这一回，再探已经晚了半拍。

因为就在齐边还想把眼重新转过去的时候，浅谷后那支一直压着的韩前军已先顺着谷口外推了一段，旗未尽出，阵却已摆开。与此同时，魏中军也照昨日预留的号次半刻后接上，赵粮骑则已在更后头亮起补站之火。

这一串动作一起，齐边垒中便有人先白了脸。

因为他们终于看清，自己这些日子看见的，并不是几国来回晃动的影。

是一口已经真拧到一处、而且第一眼就先叫自己看偏了的兵。

边报飞入临淄时，连齐湣王都第一次坐直了身子。

“浅谷后有联营？”

近臣答：“有。”

“非散骑？”

“非散骑。已见韩前军旗、魏中军号，赵北线粮火亦起。”

这一句句报出来，殿中方才还想顺着旧话说“不过虚张”的人，声音都先短了一截。

齐湣王脸色一点点沉下来，终于问了今天最重的一句：“乐毅亲在否？”

边吏低头：“暂未见其旗，但看诸军号次与应对，必有总节在中。”

田文站在席间，胸口那股压了许久的灰意，到这一刻反倒忽然散开了一点。

不是轻松。

是因为他终于知道，自己这些日子最怕的那件事，已经不再只是怕。

它来了。

可他还没来得及开口，齐湣王已先猛地站起：“传边将，不许再以轻骑探看了。守谷，固垒，先把北外几处要口锁住！”

命令一下，殿中顿时乱起细细脚步声。

直到这时，满城才第一次真的感觉到，那些前些日子还只像“外间互试辞色”的异动，已经压到了齐境脚边。

而在前线，乐毅并未贪快。

探得齐边已惊之后，他当即下令韩前军止于谷口外一线，不许深逐；魏中军前移半里，却不得抢过韩旗；赵粮骑补火亮站，不得因前面顺势便压乱次序。

韩那老将得令时，忍不住道：“主将，齐边既惊，为何不趁它乱，再咬深一点？”

乐毅道：“今日要的不是多吃几步地。”

“那要什么？”

“要齐第一眼看错之后，第二眼再看见我们不是一冲就散、也不是谁先抢功。”乐毅道，“它若今日只知我们能来，后面还会想试；它若今日连我们来后不乱都看见，后面心便先乱一层。”

这话说完，韩将怔了片刻，终于重重点头。

因为他忽然明白，主将眼里的第一势，从来不只是把敌边撞开一道口。

是先把敌人心里那份“也许还能慢慢试”的胆，狠狠干塌一层。

入夜时，齐边各垒都已添火、增岗、移车、堵道，忙得比先前围宋时还急。

而联军这边，营火却反而比白日更稳。

前军不躁，

中军不乱，

粮骑不误，

总节之令一层层往下走，像先前在燕都那些偏厅里反复问死过的问题，这时终于全变成了脚下真能踩住的东西。

乐毅站在高处，看着对面齐边那一片忽然多起来、也乱起来的火，终于第一次露出一点极轻的神色。

不算笑。

更像是心里一直压着的那口气，到了这里，终于顺了半寸。
