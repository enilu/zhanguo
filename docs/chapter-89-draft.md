# 第八十九回 浅谷后营一亮齐边惊 乐毅当机先夺第一势

第三日黎明，齐边探骑终于在浅谷后那片本不该有大营的地方，看见了新火。

火不多，却整，一线线压在最容易遮住后军、也最能逼近齐边侧肋的地势后头。

探骑回报时，边垒守将脸色先是一僵：“怎会在那里？”

副将低声道：“若联军前两日故意叫我们只盯北坡……”

守将立刻喝断：“再探！”

可这一回，再探已经晚了半拍。

就在齐边还想把眼重新转过去的时候，浅谷后那支一直压着的韩前军已先顺着谷口外推了一段，旗未尽出，阵却已摆开；与此同时，魏中军照昨日预留的号次半刻后接上，赵粮骑则在更后头亮起补站之火。

这一串动作一起，边垒中便有人先白了脸。

边报飞入临淄时，连齐湣王都第一次坐直了身子：“浅谷后有联营？”

边吏答：“有。”

“非散骑？”

“非散骑。已见韩前军旗、魏中军号，赵北线粮火亦起。”

这一句句报出来，殿中方才还想顺着旧话说“不过虚张”的人，声音都先短了一截。

齐湣王脸色一点点沉下来，终于喝道：“传边将，不许再以轻骑探看。守谷，固垒，先锁北外几处要口！”

而在前线，乐毅却并未贪快。

探得齐边已惊之后，他当即下令：

韩前军止于谷口外一线，不许深逐；

魏中军前移半里，不得抢过韩旗；

赵粮骑补火亮站，不得因顺势便压乱次序。

韩老将得令时，忍不住道：“齐边既惊，为何不趁它乱，再咬深一点？”

乐毅答：“今日要的不是多吃几步地。”

“那要什么？”

“要它第一眼看错之后，第二眼再看见我们不是一冲就散，也不是谁先抢功。”

入夜时，齐边各垒都已添火、增岗、移车、堵道，忙得比先前围宋时还急。联军这边，营火却反而比白日更稳：

前军不躁，

中军不乱，

粮骑不误。

乐毅站在高处，看着对面齐边那片忽然多起来、也乱起来的火，没有说话。对面的火越乱，这边的营火便显得越稳，像他在燕都那些夜里反复问死的问题，到这时才全落到了地上。
