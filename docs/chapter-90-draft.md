# 第九十回 临淄始惊田文知大势转 乐毅压兵不发先夺人心

齐边大惊的当夜，临淄城中也终于不再像前些日子那样安稳。

北门外的驿道上，来回奔走的不是贺捷吏，而是边急、军书、调令。

城中人最初还不知道究竟出了什么事，只看见宫门开得比平常更急，连夜送出的车骑一拨接一拨。到了后半夜，才有人在街角压低声音说：“燕、韩、魏、赵，真并来了。”

这句话一传开，整座临淄像被谁在暗处轻轻扯了一下。

不是立刻大乱。

而是那层先前一直罩在城上的“谁敢真动齐”的笃定，终于第一次裂出一道细口。

宫中小朝连夜再开。

齐湣王没有再坐得像白日那样稳，他立在图前，听边吏一条条报：

“浅谷后韩前军已列。”

“魏中军接应分明。”

“赵粮火不乱。”

“总节之令一线到底，尚未见其失拍。”

每一条报出来，殿中那股此前最顺手的“彼辈必散”之气，便又薄一层。

齐湣王终于咬着牙道：“乐毅。”

这名字到这一刻，才第一次真正在临淄殿上压出了重量。

田文站在下首，心里反倒生出一种极冷的清醒。

他知道，从这一刻起，自己这些日子所有“不可只看第一眼”“外间越静越险”的话，终于都不必再说了。

因为现实已替他说到了殿中央。

可他并没有因此松下一分。

恰恰相反，他更知道，最危险的时候，这才刚开始。

齐湣王能不能从“我没看错”那口气里退出来，转去认眼前这支联军已不是寻常拼凑之兵；

朝中群臣能不能从顺着大王心意说轻话，转去说那些真正难听却有用的话；

这一切，都还未见分晓。

果然，殿中很快就有人请战，要趁联军初到、根尚未深，先以齐锐骑出击其前。

也有人主张死守诸要口，不与联军轻战，先看其粮路是否真如报中所言那样稳。

齐湣王来回听了两路，脸色越来越硬，却迟迟未下最后一句。

因为他也已感觉到，这一次压到眼前的，不再是某一国来试锋。

而是一股先前自己最不愿信、如今又不得不看清的并兵之势。

而在前线，乐毅却没有趁着齐边大惊，立刻发大攻。

韩前军求战之声已起；

魏中军也知今日若再往前半里，齐边几处仓口便更慌；

赵后线则难得顺畅，正是乘势可进之时。

可乐毅看完一圈回报，只下了一道令：

“压兵，不发。”

帐中诸将都微微一怔。

韩那老将最先忍不住：“主将，齐边已惊，为何不打它一个立脚不稳？”

乐毅答：“惊可再惊，乱不可替它先乱过头。”

“何意？”

“齐今夜先乱的是眼，不是骨。”乐毅道，“若我们急着去求一场眼前之利，它反会被我们逼得立刻把骨先并住。我要它再看一夜，再猜一夜，再在自己殿中多争一回。让它先明白，我们不是一阵来风；也让它先怕，我们明日到底从哪一口下去。”

这几句话说完，帐中诸将竟都沉了下来。

因为他们都听懂了。

主将压着不发，不是怯。

恰恰是要把今日刚夺到手的那第一层人心之势，拿足、拿稳、拿到齐自己先在城中乱出第二层来。

夜更深时，联军诸营灯火并未骤盛。

它们照旧按次点着。

前军如前，

中军如中，

粮骑如后，

总节营火仍立在几路交会之间，不高，不炫，却始终不乱。

郭隗这夜也在军中，立在高坡上望着对面齐边多出来、又时急时慢的火，忽然低声道：“今日主将不打，竟比打一仗还叫人心里发寒。”

乐毅恰在他身后不远，听见了，却没有回头。

他只是望着南面那片火色，缓缓道：“仗后面自会有。今日先要的，是叫临淄城里所有人都睡不稳。”

风从齐境吹来，带着一点城郭与河水混在一起的夜气。

它吹过联军营前，吹过齐边乱火，也吹向更南面的临淄高城。

这一夜，真正先被攻下来的，还不是哪一道垒、哪一座城口。

而是齐国这些日子一直最稳、也最轻易替自己遮风的那层心。
