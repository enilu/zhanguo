# 第九十回 临淄始惊田文知大势转 乐毅压兵不发先夺人心

齐边大惊的当夜，临淄城中也终于不再像前些日子那样安稳。

北门外驿道上，来回奔走的不再是贺捷吏，而是边急、军书、调令。到了后半夜，街角终于有人压低声音说：“燕、韩、魏、赵，真并来了。”

宫中小朝连夜再开。

齐湣王立在图前，听边吏一条条报：

“浅谷后韩前军已列。”

“魏中军接应分明。”

“赵粮火不乱。”

“总节之令一线到底，尚未见其失拍。”

每一条报出来，殿中那股此前最顺手的“彼辈必散”之气，便又薄一层。

齐湣王终于咬着牙道：“乐毅。”

田文站在下首，没有再说什么。他知道，自己这些日子那些“不可只看第一眼”的话，到这里已经不必再说，现实已替他说到了殿中央。

殿里很快就分成两路。有人请战，要趁联军初到、根尚未深，先以齐锐骑出击其前；也有人主张先守诸要口，不与联军轻战，再看其粮路是否真如报中所言那样稳。

齐湣王来回听了两路，脸色越来越硬，却迟迟未下最后一句。

前线这边，乐毅却没有趁着齐边大惊立刻发大攻。

韩前军求战之声已起；

魏中军也知今日若再往前半里，齐边几处仓口便更慌；

赵后线难得顺畅，正是乘势可进的时候。

可乐毅看完一圈回报，只下了一道令：

“压兵，不发。”

韩老将最先忍不住：“齐边已惊，为何不打它一个立脚不稳？”

乐毅答：“齐今夜先乱的是眼，不是骨。我们若急着去求这一口眼前之利，它反会被逼得立刻把骨先并住。”

帐中一静。

乐毅又道：“让它再看一夜，再猜一夜，再在城中多争一回。”

夜更深时，联军诸营灯火并未骤盛，仍照旧按次点着：

前军如前，

中军如中，

粮骑如后，

总节营火仍立在几路交会之间。

郭隗这夜也在军中，立在高坡上望着对面齐边多出来、又时急时慢的火，低声道：“今日不打，竟比打一仗还叫人发寒。”

乐毅站在他身后不远处，只望着南面那片火色：“仗后面自会有。今日先要的，是叫临淄城里所有人都睡不稳。”
