# 第九十一回 齐骑初出先探联军骨 乐毅开战只许韩军半咬

临淄那一夜并没有让齐湣王睡稳。

天刚亮，小朝便再开。

争战与守的人，这一回都不敢再像昨夜那样说得太轻。因为齐边探来的军报已写得极明：

联军不是散股，

前军、中军、粮火、总节皆有次，

而且第一眼便先把齐边看偏了。

齐湣王听完，脸色比前夜更沉，却也更硬。

他终于下令：“先出骑，不出重兵。寡人要看它这股并兵，骨到底硬到什么地步。”

这命令很合齐眼下的心。

不肯先认自己已被逼到不得不守，

也不肯把第一口气就压成死守。

他还想先试。

于是当日下午，齐骑先出。

人数不算极多，却挑的都是齐边最快最利的骑卒。先头一队故意压近韩前军外沿，像要探它敢不敢真接；另一路则斜逼魏中军与前军之间那条本该最怕失拍的接应线上，想看联军是否真能一口应住。

韩那老将远远看见齐骑尘头，眼里立刻亮起一层硬光。

“终于来了。”

他回头便要命前军迎上，乐毅的军令却先一步到：

“韩军可迎三十步，不许过。”

老将接令，脸色一沉：“三十步？这岂不够人塞牙？”

传令吏只道：“总节令如此。”

老将咬了咬牙，终究还是压住那口往前猛撞的气，只命前列压出，至三十步便立阵。

齐骑见韩军真出，先是一喜，以为这前军到底还是压不住旧性，下一刻便要被自己拖着往前。谁知韩前军压到三十步整，竟真的停住，只以强弩、短突列成一口硬弧，不再多出半寸。

齐骑第一口气便先咬空了。

与此同时，斜逼魏中军那一路齐骑也已靠近。

他们最想看的，便是魏中军会不会因为韩前军那边一动，自己先忙乱着去补。若魏一乱，这股并兵之势便等于从中间先折了半口。

可魏节应将压根没往韩那边多看一眼，只盯着号次。

“前号不变。”

“中鼓不起。”

“左队只移，不并。”

几道令下得极快，也极冷。

副将低声道：“将军，韩前军已与齐骑接上了。”

节应将只道：“韩前未求我动，我先动便是乱它。”

这一句，竟把帐下几个原本急着想去应韩的人一下钉在原地。

齐骑第二口气，便也咬空。

赵这边更静。

齐骑有一小支想绕得更深些，看赵粮线能不能逼出慌来。赵奢却早命三处补火齐起，押粮车照旧按站不按慌，前站若动，后站不跟着乱。那支齐骑转了一圈，只看见粮火稳稳一层接一层，竟没咬出半点“后线将乱”的气。

三个口子一并试下来，齐骑先乱了一下。

不是阵乱。

是心里那层“联军虽并，终究有一口先松”的预断，一下没落着实处。

前线边将得报后，脸色已变。

他立刻命人再探，却再不敢像前两日那样拿“不过试影”四字去回临淄。

而在联军中帐，韩那老将已憋了一肚火。

“主将，韩军明明可以再咬深半口！”

乐毅看着回报，神色不动：“咬深之后呢？”

老将一怔：“自然是趁它乱，打它更退。”

乐毅抬眼：“今日齐骑出来，不是为与韩拼这三十步，是为看我们谁先忍不住。你若多追半口，魏便要被你拖着改次，赵后线也要跟着更火。我们今日赢的，不是这三十步，是让它三处都看不出哪一处先松。”

老将站在那里，脸色来回变了几次，最后重重吐出一口气：“臣明白了。”

其实他未必全服。

可他至少已知道，这位主将眼里的“赢”，比自己惯常认的那一口快胜，要冷得多，也深得多。

入夜时，齐边回报终于到临淄。

上头只有几句，却句句都像冷水：

“韩前军出而止，非躁兵。”

“魏中军应而不乱。”

“赵粮线稳，未见其慌。”

齐湣王看完，半晌没有说话。

田文站在下首，听见殿中竟一时无人敢先顺着大王心意开口，心里反倒比前几日更沉。

因为他知道，沉默本身，往往就说明很多人心里已经开始把最坏的那层真正算进去了。
