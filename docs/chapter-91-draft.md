# 第九十一回 齐骑初出先探联军骨 乐毅开战只许韩军半咬

临淄那一夜并没有让齐湣王睡稳。

天刚亮，小朝便再开。齐湣王听完前线诸报，终于下令：“先出骑，不出重兵。寡人要看它这股并兵，骨到底硬到什么地步。”

当日下午，齐骑先出。

人数不算极多，却尽挑齐边最快最利的骑卒。先头一队故意压近韩前军外沿，像要探它敢不敢真接；另一路则斜逼魏中军与前军之间那条最怕失拍的接应线。

韩老将远远看见齐骑尘头，眼里立刻亮起一层硬光。可乐毅的军令先一步到：

“韩军可迎三十步，不许过。”

老将接令，脸色一沉，却终究还是压住那口往前猛撞的气，只命前列压出，至三十步便立阵。

齐骑见韩军真出，先是一喜，以为这前军下一刻便要被拖着往前。谁知韩前军压到三十步整，竟真的停住，只以强弩、短突列成一口硬弧，不再多出半寸。

齐骑第一口，便先咬空了。

另一边，斜逼魏中军那一路也已靠近。魏节应将却没往韩那边多看一眼，只盯着号次：

“前号不变。”

“中鼓不起。”

“左队只移，不并。”

副将低声道：“韩前军已与齐骑接上了。”

节应将只道：“韩前未求我动，我先动便是乱它。”

齐骑第二口，也咬空了。

赵这边更静。齐骑有一小支想绕得更深些，看赵粮线能不能逼出慌来。赵奢却早命三处补火齐起，押粮车照旧按站不按慌。那支齐骑转了一圈，只看见粮火稳稳一层接一层。

三个口子一并试下来，齐骑先乱了一下。不是阵乱，是心里那层“联军终会有一口先松”的预断，一下没落着实处。

入夜时，齐边回报终于到临淄。上头只有几句：

“韩前军出而止，非躁兵。”

“魏中军应而不乱。”

“赵粮线稳，未见其慌。”

齐湣王看完，半晌没有说话。田文站在下首，只觉殿中那阵短短的沉默，比前几日满堂轻话更沉。
