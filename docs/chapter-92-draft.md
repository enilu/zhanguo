# 第九十二回 临淄议守始见轻敌裂 田文进言只求先认真

齐骑试过之后，临淄朝中那层“彼辈终究不成器”的气，终于裂开了一道真口。

这回再开朝，先前最爱顺着齐湣王心意说轻话的人，也都小心了许多。因为边报已经写得极明：

韩前军非躁，

魏中军非乱，

赵粮后非空。

齐湣王把那几行字看了三遍，忽然问：“若联军真这般稳，齐今当先打哪一口？”

殿里很快分成三路。有人主张仍打韩前军；有人主张专打赵后线；也有人说该先逼燕总节失拍。

田文这时出班，却没有先说打哪一口，只道：“大王，臣请先去一层旧气。”

齐湣王抬眼：“什么旧气？”

田文答：“去掉‘它们终究还是散兵’这层旧气。若这一层不先去，不论最后定打韩、打赵，还是打总节，底下行兵时心里都还会先想着：彼辈总会自己先乱。那样，先乱的反会是齐自己。”

殿中一静。

齐湣王盯着田文看了很久，最后才道：“寡人现在要的是法，不是教训。”

田文俯身：“臣所言，正是法之前的法。先认它是真并兵，再择其要口而击。若不先认，后面一切法，都只是拿旧眼看新局。”

这几句话，到底还是进了一寸。

齐湣王沉默片刻，终于转头对边将道：“传前线，自今日起，不许再以‘彼必自乱’立策。报我它前、中、后、总节四口真正轻重，再议下手。”

散朝之后，田文走下长阶，听见身后军书小吏快步而过，鞋底敲在石上，急而不乱。他没有回头，只觉那声音像是在提醒他，齐到今日才开始认真，已经是晚了半步。

可再晚，也得从这半步开始补。

田文心里很清楚，乐毅最容易利用的，正是齐朝上下那层“终究还是散兵”的旧看法。只要这层旧看法还在，后面不论定打韩、打赵还是打总节，底下的人真到临阵时，都会先替自己留一丝“也许它会自己先乱”的余地。留了这丝余地，便是把自己的刀口先磨钝了一寸。
