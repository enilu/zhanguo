# 第九十二回 临淄议守始见轻敌裂 田文进言只求先认真

齐骑试过之后，临淄朝中那层“彼辈终究不成器”的气，终于裂开了一道真口。

这回再开朝，先前最爱顺着齐湣王心意说轻话的人，也都小心了许多。

因为边报已经明明白白写着：

韩前军非躁，

魏中军非乱，

赵粮后非空。

这不是一股热起来便要各自散去的兵。

齐湣王把那几行字看了三遍，忽然问：“若联军真这般稳，齐今当先打哪一口？”

这句话一出，殿中气口便与前几日全不一样了。

因为它说明，大王终于不再只问“它们敢不敢来”，而是开始问“它们来后该怎么拆”。

有人主张仍打韩前军。

“前军先当锋，伤它最易见效。”

也有人道，不如专打赵后线。

“联军最怕不断，赵粮若乱，其势自衰。”

还有人说，真正该先下手的，是燕总节。

“几国并兵，最怕令不一。若能逼乐毅总节失拍，它不必大败，自会自乱。”

三路说法，样样都像有理。

齐湣王听到后来，脸色却并不见松。

因为越是人人都说得有理，越说明眼前这支联军，已不像从前那样一眼便能看出哪个是最软的一截。

田文这时终于出班。

他没有先说打哪一口，只先道：“大王，臣请先去一层旧气。”

齐湣王抬眼：“什么旧气？”

田文答：“去掉‘它们终究还是散兵’这层旧气。若这一层不先去，不论朝中最后定打韩、打赵、还是打总节，底下行兵时心里都还会先想着：彼辈总会自己先乱。那样，真正先乱的，反会是齐自己。”

殿中一下静了。

因为这话不再只是逆耳。

它几乎是在当众说：

齐此刻若再不把眼前这支联军当真兵来看，后面很多错，不会错在谋上，会先错在心上。

齐湣王没有立刻发作。

他只是盯着田文看了很久，最后才冷冷道：“寡人知道它们不是散兵。寡人现在要的是法，不是教训。”

田文俯身：“臣所言，正是法之前的法。”

“说。”

田文道：“先认它是真并兵，再择其要口而击。若不先认，后面一切‘法’，都只是拿旧眼看新局。”

这几句话，到底还是进了一寸。

齐湣王沉默片刻，终于转头对边将道：“传前线，自今日起，不许再以‘彼必自乱’立策。报我它前、中、后、总节四口真正轻重，再议下手。”

这道令虽短，却叫殿中许多人背后都微微起了一层寒。

因为这说明，大王总算开始把那层最顺耳的轻看，往下按了半分。

可田文心里并没有因此松下去。

他知道，认真只是一层门槛。

过了这门槛，齐能不能真改得过来，还远得很。

散朝之后，一名近臣低声对田文道：“君上今日总算劝进去一句。”

田文苦笑：“只怕还不够。”

“为何？”

田文望着宫中长阶，缓缓道：“因为乐毅如今最可怕的，不只是把兵并住了。他让我们这些日子所有晚了半步才开始认真的人，都像是在跟着他的节走。”

这句话说完，长阶尽头正有军书小吏快步而过。

鞋底敲在石上，急而不乱。

田文听着那声音，心里忽然更冷。

因为他明白，齐如今最难的，不是看不见敌。

而是敌已经把自己的快慢、轻重、先后，都先摆在了前头。

齐若再想改，便得拿整座朝堂、整条边线去追。
