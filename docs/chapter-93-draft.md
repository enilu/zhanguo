# 第九十三回 乐毅试锋半日不争地 齐军改眼一时仍难齐

齐廷新令到边之后，前线诸将果然不敢再像前几日那样先把“联军必自乱”挂在嘴边。

可嘴上改了，心里那口旧气却未必立刻改得过来。

第二日，齐边主将试着按新令重新布眼：

先看韩前军究竟是真硬还是只装硬；

再看魏中军会不会因韩受压而过应；

最后盯赵粮线，想找它是否仍有一站最怕急逼。

这一套看法，比前几日确实正了些。

可就在齐边刚把眼重新排过一遍时，乐毅已先动了。

他没有求大战。

只命韩前军半日试锋。

“只许压，不许夺。”

“只许逼，不许贪。”

“魏中军应其后，不许抢前。”

“赵后线火照旧，不因前锋动便乱改站。”

韩那老将接令时，几乎是从牙缝里把“得令”两个字挤出来的。

他最不耐烦这种“压而不夺”的打法。

可经过前两日几番见识，他再不服，也已知道主将不是无胆，而是另有更深一层的算。

于是这一日正午，韩前军真的只是压。

齐边前垒外一线浅坡，韩军盾列缓起，弩手不急着放满，只一层层逼近。齐军原以为联军既动，必求一口撕开口子，便早在坡后压了更重的弓与短突，准备等韩军一猛扑便反折它一回。

谁知韩军压到半坡，便只立弩压人，不再多进。

齐军第一口重箭，便又空等了半寸。

守将脸色立刻沉下去：“它又不抢？”

副将低声道：“主将，这几日它似乎处处都只取半步。”

“半步最烦。”守将咬着牙，“你想扑它，它又不全来；你想不理，它便一点点把坡口先压活。”

而在韩军背后，魏中军这回也没有像齐边旧想的那样，见前军已压便立刻大推，只把中线慢慢前移，既让韩前军心里知道“我在”，又不让自己变成一口抢前的兵。

赵后线则依旧像一条不肯乱的暗脉。

齐边派了一支轻骑想绕试它第二站，结果那一站粮火比前几日更早亮，押车、补马、夜伏之号一应俱全，竟像专为等它来看。

齐骑绕了一圈，什么也没咬着。

入夜时，齐边守将坐在垒中，看着白日回来的诸路报文，脸色越来越不好。

因为他终于发现，自己今日明明比前几日看得更认真，布得也更细，可偏偏还是处处晚半口。

不是大败。

却就是处处都没抢到自己原以为能抢到的那一下。

这种仗，比明刀明枪先折一阵更叫人烦。

而在联军营中，韩那老将也正憋着一肚气。

“主将，今日坡口明明可再往里咬一步！”

乐毅看着回报，头也不抬：“咬进去之后呢？”

老将一噎：“自然是试它垒里虚实。”

乐毅这才抬眼：“试出来，你会忍得住不再多抢一步么？”

老将沉默了。

因为他知道，自己多半忍不住。

乐毅又道：“齐今日已经开始改眼。我们此刻若猛贪一步，它反会最快把该并的地方并起来。今日半日之压，要的不是破垒，是叫它发现自己认真来看了，仍旧处处慢半拍。”

魏节应将在旁听到这里，忽然轻轻点了点头：“原来主将今日打的，还是它那口心。”

乐毅道：“兵未大开前，先争的便是这个。”

帐中一时都静了。

因为诸将此刻都慢慢看出来了：

主将这几日每一步都不大，

却步步都在逼齐把自己往更重、更急、更难回头的地方带。

而齐每改一步，反倒又更被这股节奏牵进去半寸。
