# 第九十三回 乐毅试锋半日不争地 齐军改眼一时仍难齐

齐廷新令到边之后，前线诸将果然不敢再像前几日那样先把“联军必自乱”挂在嘴边。

第二日，齐边主将试着按新令重布眼：先看韩前军究竟是真硬还是装硬，再看魏中军会不会因韩受压而过应，最后盯赵粮线，想找它是否仍有一站最怕急逼。

可就在齐边刚把眼重新排过一遍时，乐毅已先动了。

他没有求大战，只命韩前军半日试锋：

“只许压，不许夺。”

“只许逼，不许贪。”

“魏中军应其后，不许抢前。”

“赵后线火照旧，不因前锋动便乱改站。”

于是这一日正午，韩前军真的只是压。

齐边前垒外一线浅坡，韩军盾列缓起，弩手不急着放满，只一层层逼近。齐军原以为联军既动，必求一口撕开口子，便早在坡后压了更重的弓与短突，准备等韩军一猛扑便反折它一回。

谁知韩军压到半坡，便只立弩压人，不再多进。

齐军第一口重箭，又空等了半寸。

“它又不抢？”守将脸色立刻沉了。

而在韩军背后，魏中军也没有见前军已压便立刻大推，只把中线慢慢前移，既让韩前军知道“我在”，又不让自己变成一口抢前的兵。

赵后线仍旧像一条不肯乱的暗脉。齐边派轻骑去绕试它第二站，结果那一站粮火比前几日更早亮，押车、补马、夜伏之号一应俱全。

入夜时，齐边守将坐在垒中，看着白日回来的诸路报文，脸色越来越不好。他今日明明比前几日看得更认真、布得更细，可偏偏还是处处晚半口。

联军营中，韩老将也憋着一肚气：“主将，今日坡口明明可再往里咬一步！”

乐毅抬眼：“咬进去之后呢？”

老将一噎。

乐毅道：“齐今日已经开始改眼。我们此刻若猛贪一步，它反会最快把该并的地方并起来。今日半日之压，要的不是破垒，是叫它发现自己认真来看了，仍旧处处慢半拍。”

帐中一时都静了。诸将到这时才慢慢看出来，主将这几日每一步都不大，却步步都在逼齐把自己往更重、更急、更难回头的地方带。
