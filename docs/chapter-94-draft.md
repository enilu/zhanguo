# 第九十四回 临淄终定先守要口策 乐毅夜下明令大战在晨

齐边连着两日没抢到那口“主动”，临淄朝中终于也压不住了。

这回再开议，齐湣王已不再问“谁敢真动”，也不再问“联军会不会自乱”。他坐在图前，第一句话便是：“它到底要从哪一口真下？”

这句问得极重。

因为它说明，齐到了这一层，总算开始承认：

联军不是来示威，

而是来找一口真正能咬下去的地方。

有人仍主张先击韩前军。

“前军最外，先折其锋，余军自缩。”

也有人道，既然乐毅处处不贪，不如反守要口，逼它自己来撞。

“它要拿齐边心势，我们便先把心势锁回垒里。”

田文这回没有再绕。

他直接道：“臣请先守要口。”

齐湣王抬眼，神色有些冷：“为何不是先打它前军？”

田文答：“因为乐毅此刻最想要的，不是齐守，是齐先乱着出去抢。齐若按不住这口气，它便会最快看见我们哪一处仍旧最怕急、最怕乱、最怕谁先争功。守要口虽难看，却是眼下最不易送它势的一法。”

这话说到后来，满殿竟少有人再立刻反驳。

不是人人都服田文。

而是边报已经把许多人心里那层最顺手的轻敌，磨得差不多了。

齐湣王盯着图看了很久，终于道：“传前线，先固数口，不许轻逐。若联军来逼，再择其真口出重兵。”

这道令一下，田文心里并没有轻松。

因为他知道，齐只是终于走到“先守”这一步。

可这一步，原本本该早几日、甚至早几旬便走。

如今虽不算晚到不能用，却也已把很多该有的从容，先让给了对面。

而在联军前线，齐这一道新令也很快被探到。

入夜后，中军大帐再开。

这回帐里没有外使，也没有多余的文吏。

韩、魏、赵、燕几方真正掌兵与掌节的人都在。

乐毅把齐边今日新收的几份回报一一压在图上，最后道：“齐开始守口了。”

韩那老将眼里先亮了一下：“总算逼得它不敢乱出了。那明日是不是该狠狠干一口？”

乐毅没有立刻答。

他先看了看魏节应将，又看了看赵奢。

魏节应将道：“齐若守口不出，最怕我们逼一处而诸口不应。”

赵奢接得更实：“赵后线今夜可再前补半站。若明晨真打，一日内不断，臣敢保。”

乐毅听完，这才缓缓点头。

他伸手在图上齐边北外一处最容易牵动两垒、也最怕前后不及应的口子上轻轻一点：“明日，就试它这一口。”

帐中众人目光同时落过去。

那是一处不算最厚、却最连着边垒气的地方。

若咬开，齐不会立刻大崩；

可若守得不好，整条北外应答都会先乱一层。

韩那老将这回没有再先问“可追几步”，只问：“前军如何动？”

乐毅答：“前军先压，不求一口尽入。魏中军看齐第二垒号动再补。赵后线火明于旧，不许误。燕本部压住总节，不许诸军各自贪快。”

说完之后，他停了停，终于把这几日一直压着不肯轻吐的那句真正落了下来：

“明晨，可开大战。”

帐中一时无人出声。

不是不激动。

而是这句话到这里，反倒比任何壮语都更沉。

因为每个人都知道，这不是一场突然拍案起意的猛打。

它是从苦差、军次、会期、会师、试号、南下、误判、惊心，一层层逼到今夜，才终于逼出来的那一场真打。

韩那老将先一步抱拳：“前军得令。”

魏节应将随即应声：“中军得令。”

赵奢也道：“粮线得令。”

乐毅没有再多说。

他只命主簿把明晨各营先后号次再誊一遍，分送诸营，务必在二更前到。

散帐之后，诸将各自出营。

夜风比前几日都更硬。

韩前营里已在静静整甲；

魏中军那边压低了鼓，先对明晨应号；

赵粮骑灯下，一车车封泥再验最后一遍；

燕总节营中，则把那卷最终定下的明晨军次简，平平压在案中央。

乐毅独自留在帐内，看着图上那一处明日将开的口，久久未动。

帐外风声、甲声、马鼻声、远处偶有短号声，一层层都很轻。

可正因轻，才更叫人知道，大战真的近了。

他终于抬手，灭了帐中最后一盏偏灯，只留下图前那一盏。

灯火映着图上那一点将开的口，微微发亮。

像这一整卷第三卷后半一路积下来的兵气、火气、心气，到今夜终于都收进了这一点里。
