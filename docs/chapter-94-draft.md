# 第九十四回 临淄终定先守要口策 乐毅夜下明令大战在晨

齐边连着两日没抢到那口主动，临淄朝中终于也压不住了。

这回再开议，齐湣王已不再问“谁敢真动”，也不再问“联军会不会自乱”。他坐在图前，第一句话便是：“它到底要从哪一口真下？”

殿里有人仍主张先击韩前军，也有人道不如先守要口，逼它自己来撞。

田文这回没有再绕，直接道：“臣请先守要口。”

齐湣王抬眼：“为何不是先打它前军？”

田文答：“因为乐毅此刻最想要的，不是齐守，是齐先乱着出去抢。齐若按不住这口气，它便会最快看见我们哪一处仍旧最怕急、最怕乱。守要口虽难看，却是眼下最不易送它势的一法。”

齐湣王盯着图看了很久，终于道：“传前线，先固数口，不许轻逐。若联军来逼，再择其真口出重兵。”

齐这一道新令，很快也被前线探到。

入夜后，联军中军大帐再开。帐里没有外使，只有韩、魏、赵、燕几方真正掌兵与掌节的人。

乐毅把齐边今日新收的几份回报一一压在图上，最后道：“齐开始守口了。”

韩老将眼里先亮了一下：“那明日是不是该狠狠干一口？”

乐毅没有立刻答，先看了看魏节应将，又看了看赵奢。

魏节应将道：“齐若守口不出，最怕我们逼一处而诸口不应。”

赵奢接道：“赵后线今夜可再前补半站。若明晨真打，一日内不断，臣敢保。”

乐毅这才伸手，在图上齐边北外一处最容易牵动两垒、也最怕前后不及应的口子上轻轻一点：“明日，就试它这一口。”

韩老将这回没有再先问“可追几步”，只问：“前军如何动？”

乐毅答：“前军先压，不求一口尽入。魏中军看齐第二垒号动再补。赵后线火明于旧，不许误。燕本部压住总节，不许诸军各自贪快。”

他说完，终于把那句真正落了下来：

“明晨，可开大战。”

帐中一时无人出声。乐毅没有再多说，只命主簿把明晨各营先后号次再誊一遍，分送诸营，务必二更前到。

散帐之后，诸将各自出营。

韩前营里已在静静整甲；

魏中军那边压低了鼓，先对明晨应号；

赵粮骑灯下，一车车封泥再验最后一遍；

燕总节营中，则把那卷最终定下的明晨军次简，平平压在案中央。

乐毅独自留在帐内，看着图上那一点将开的口，久久未动。帐外风声、甲声、马鼻声都很轻，正因轻，才更叫人知道，大战真的近了。
