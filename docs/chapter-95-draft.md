# 第九十五回 乐毅开战先咬北外口 齐垒初折诸军始知真硬

天还未亮，齐边北外那几处要口上便已尽是湿重夜气。

齐军这一夜守得不敢懈，垒上火把多添了一倍，谷口外鹿角与拒马也比前几日更密。昨夜新到的军令说得很死：

不许轻逐，

不许见敌小动便自乱出击，

要先守住口。

因此这日黎明，齐边将士心里都压着同一口气：

守住第一口，

不让联军轻轻一撞便把齐这些日子迟来的认真再撞回去。

可乐毅的第一击，偏偏就不让它守得舒服。

韩前军先动时，并没有照着垒门正口去扑。

他们沿着昨夜便已摸准的那片坡线，先压齐垒外一截侧棱，盾列前逼，弩手跟进，脚下不快，节却极稳。齐军原先压在正口后那一层最厚的迎击之势，一下便先落空了半边。

守将立刻下令：“右队外补！”

令还未全传到，魏中军的号便已起。

不是大起。

只是一层低沉短号，把本来就最怕韩前军压侧、魏中军又不知何时补上的齐边右垒，先压得心里一紧。

副将低声道：“它魏要动了。”

守将咬牙：“看清再动！”

可战场最怕的便是这一句“看清再动”落在本就慢半拍的时候。

韩前军压的那一截侧棱，正卡在齐右垒与正垒之间最不好救的一段。齐右队一迟，韩前军便把那一段坡脚先压活。齐军几次想把人顶回去，都被韩那老将用最笨、也最硬的法子顶住：

不深进，

不争快，

只一层层把人往上拱。

帐中若有人看见，必会觉得这打法太慢。

可正因为慢，齐军那口本该用来“守口不乱”的劲，反而一点点被拖成了“哪里都像不该先松、哪里又都像快要先松”的烦乱。

就在这时，魏中军真补了。

不多。

只半里。

却恰好补在齐右垒刚想往前挤那一下上。

齐边守将脸色当场一变。

因为这说明，联军不是韩军自己蛮压，魏中军是真按昨日那套号次跟着动，而且动得刚好，不多不少，正卡在齐军最难受的时候。

他再顾不得昨夜那句“不许轻逐”，厉声喝道：“左垒再调一队！”

这一调，便乱了第二口。

左垒那边原是护着后头一条小转道的，一挪，赵粮骑照旧稳在那里的火线立刻更显得近。几名齐军将校回头一看，心便先虚了一层，竟不知这到底是联军后线，还是另一路埋着的应军。

齐军心里那一点“也许它们仍旧只是拼在一起”的侥幸，到这里终于开始真碎。

辰时刚过，齐右垒外那一段坡脚先失。

失得不算全垮。

只是鹿角被韩军一段段拆开，外列被逼退半重，垒中号令一时再压不齐。

可这半重一退，对齐边来说已够冷。

因为这不是齐军自己小退一步便能慢慢再收的那种失。

它是被联军前军、中军、后线、号次一层层卡着，把最不该先松的那口气先拗折了半寸。

韩那老将看见坡脚已活，眼里顿时又亮起那层旧硬。

“主将，该追！”

乐毅的令却仍比他的气快一步到：

“不追垒，稳坡脚，立外线。”

老将接令，几乎是从鼻子里哼出一口热气。

可这一回他没再像前几次那样先不服。

因为今日这一战打到这里，他已亲眼看明白，主将要的从来不只是“多咬进去一口”。

要的是这第一口真打，打完之后，齐边与临淄都再不能把这支联军看成一阵热闹的并兵。

于是韩前军真的止了。

不夺垒门，

不抢深壕，

只把刚拿下的那段坡脚、鹿角残口和外线一步步钉住。

魏中军随即压上半格，替韩稳住侧后。

赵后线火与粮车也照旧不乱，像今日这一仗不过是照着早已写死的简，在该动的时候动了那几笔而已。

这一幕，反比猛扑深夺更叫齐边发寒。

因为它说明，联军不是撞开一口气便会自己散。

它是拿得住，也收得住。

午后，败报与急报同时飞往临淄。

齐湣王看见“右垒外线失半”“韩军止而不贪”“魏应而有节”“赵火不乱”这些句子时，手指竟在报文上停了许久。

满殿无人先开口。

因为谁都知道，这不是一场大败。

可也正因不是大败，才更可怕。

齐第一口真正照着自己新定之法去守的边口，还是被人稳稳啃下了半寸。

而且，是被一支几国并成的兵，啃下来的。
