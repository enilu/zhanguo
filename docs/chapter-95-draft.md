# 第九十五回 乐毅开战先咬北外口 齐垒初折诸军始知真硬

天还未亮，齐边北外那几处要口上便已尽是湿重夜气。

齐军这一夜守得不敢懈，垒上火把多添了一倍，谷口外鹿角与拒马也比前几日更密。昨夜新到的军令说得很死：先守口，不许轻逐。

可乐毅的第一击，偏偏就不让它守得舒服。

韩前军先动时，并没有照着垒门正口去扑，而是沿着昨夜便已摸准的那片坡线，先压齐垒外一截侧棱。盾列前逼，弩手跟进，脚下不快，节却极稳。齐军原先压在正口后那一层最厚的迎击之势，一下便先落空了半边。

守将立刻下令：“右队外补！”

令还未传全，魏中军的号便已起。不是大起，只是一层低沉短号，把本就最怕韩前军压侧、魏中军又不知何时补上的齐右垒，先压得心里一紧。

韩前军压的那一截侧棱，正卡在齐右垒与正垒之间最不好救的一段。齐右队一迟，韩前军便把那一段坡脚先压活。齐军几次想把人顶回去，都被韩老将用最笨、也最硬的法子顶住：不深进，不争快，只一层层把人往上拱。

这时魏中军真补了。

不多，只半里，却恰好补在齐右垒刚想往前挤那一下上。齐边守将脸色当场一变，再顾不得昨夜那句“不许轻逐”，厉声喝道：“左垒再调一队！”

这一调，便乱了第二口。

左垒原是护着后头一条小转道的，一挪，赵粮骑照旧稳在那里的火线立刻更显得近。几名齐军将校回头一看，心便先虚了一层。

辰时刚过，齐右垒外那一段坡脚先失。鹿角被韩军一段段拆开，外列被逼退半重，垒中号令一时再压不齐。

韩老将看见坡脚已活，眼里顿时又亮起那层旧硬：“主将，该追！”

乐毅的令却仍比他的气快一步到：

“不追垒，稳坡脚，立外线。”

于是韩前军真的止了。不夺垒门，不抢深壕，只把刚拿下的那段坡脚、鹿角残口和外线一步步钉住。魏中军随即压上半格，替韩稳住侧后。赵后线火与粮车也照旧不乱。

这一幕，反比猛扑深夺更叫齐边发寒。

午后，败报与急报同时飞往临淄。齐湣王看见“右垒外线失半”“韩军止而不贪”“魏应而有节”“赵火不乱”这些句子时，手指在报文上停了许久。
