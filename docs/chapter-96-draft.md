# 第九十六回 临淄闻败先失一层胆 田文见势不敢言已验

那封败报入临淄时，天色还没全黑。

齐湣王坐在殿中，连看了三遍，仍像不信“失半重外线”这几字竟会写在北边第一口要垒上。不是大破，也不是奔溃，甚至连垒门都未尽失。可他心里最沉的，恰恰就在这里。

齐湣王终于把报文重重按在案上：“它为何止？”

边吏低头道：“报中言，韩军得坡脚后便止，魏中军只补外线，赵粮火如旧。似非力不能再进，像是有意只先夺这一口，叫我军看见它能拿、也能收。”

这句话一出，殿中空气都像沉了半层。

齐湣王又问：“前线如今如何？”

“已固余口，添车堵道，夜中增岗。”

“联军呢？”

“联军未深追。韩前军稳外线，魏中军护侧，赵后线无误，燕总节营仍居中。”

报到这里时，终于有人低声道：“这不是一战之利，是它整口气都没松。”

齐湣王猛地抬头，那人立刻伏下。可话既出口，便像一根刺，扎在满殿心里。

这一夜，临淄再开小朝。主战者仍有，也有人主张坚守诸口，待乐毅自己因并兵日久而露缝。

齐湣王听了许久，终于第一次自己把那句最难听的话说了出来：

“诸君今日莫再拿‘彼必自乱’宽寡人了。”

殿中一时死静。田文听见这一句，心里却没有半分轻松。他知道，承认对面不是散兵，只是第一步。

第一步之后，还得把这一句真正写进前线诸将的眼里、手里和鼓点里。若只是大王今夜口头认了，明日边将上阵时心里却仍照旧把对面看轻半寸，那这一句再难听，也不过只是宫里一声发狠。
