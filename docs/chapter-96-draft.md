# 第九十六回 临淄闻败先失一层胆 田文见势不敢言已验

那封败报入临淄时，天色还没全黑。

齐湣王坐在殿中，连看了三遍，仍像不信“失半重外线”这几字竟会写在自己北边第一口要垒上。

不是大破。

不是奔溃。

甚至连“垒门尽失”都没有。

可他心里最沉的，恰恰就在这里。

因为这说明，对面不是一股闯上来赌命的兵。

它是试得准、打得稳、拿得住，且最可怕的，是拿到手后还真肯停。

这比一时猛扑更像某种会越长越深的病。

殿中众臣一时都不敢先开口。

从前最会顺着大王心意说“不过小失”的那几人，这回也都把那句话压回去了。

因为谁都知道，今日若还只说“小失”，不是安君心。

是瞎。

齐湣王终于把报文重重按在案上：“它为何止？”

边吏低头道：“报中言，韩军得坡脚后便止，魏中军只补外线，赵粮火如旧。似非力不能再进，像是……”

他说到这里，竟有些不敢再往下说。

齐湣王冷冷道：“像是什么？”

边吏伏得更低：“像是有意只先夺这一口，叫我军看见它能拿、也能收。”

这句话一出，殿中空气都像沉了半层。

田文站在一旁，竟比先前更沉默。

因为他知道，到这一步，很多话已不必再靠自己说。

现实正在一寸寸替他说。

而现实最残的时候，往往也正在这里：

它不会先给你一个可以痛快拍案认错的大败。

它先给你的，常常是一种更冷的东西。

一种“你明明认真了、也守了、也改了，可还是先丢了半口，而且你知道对面只是先拿来试你”的感觉。

这比痛败更易伤胆。

齐湣王很快又问：“前线如今如何？”

“已固余口，添车堵道，夜中增岗。”

“联军呢？”

“联军未深追。韩前军稳外线，魏中军护侧，赵后线无误，燕总节营仍居中。”

报到这一句时，殿中终于有人忍不住低声道：“这不是一战之利，是它整口气都没松。”

齐湣王猛地抬头，看向说话那人。

那人立刻伏下，再不敢动。

可话既出口，便已像一根刺，扎在满殿心里。

这一夜，临淄再开小朝。

主战者仍有。

有人请夜袭坡脚，趁韩军立足未深，狠狠干回去；

也有人主张坚守诸口，待乐毅自己因并兵日久而露缝。

齐湣王听了许久，终于第一次自己把那句最难听的话说了出来：

“诸君今日莫再拿‘彼必自乱’宽寡人了。”

殿中一时死静。

因为这意味着，大王终于真把那层旧轻看，从自己口里亲手折掉了半截。

田文听见这一句，心里却没有半分轻松。

他只觉得，更大的险还在后头。

因为承认对面不是散兵，只是第一步。

后面齐若仍找不到真正能拆开它的那一下，这一口胆便会越失越深。

散朝之后，老门客在府中等他。

田文见了他，只道：“今日大王终于肯不再拿旧气压眼前局了。”

老人点头：“这是好事。”

田文却摇头：“也是迟事。好事若迟了，往往就会先变成苦药。”

他说完之后，回身望向临淄北面夜空。

城中灯火依旧盛。

可今夜这些灯照在他眼里，已不再是从前那种理所当然的稳。

更像一城人都在强撑着告诉自己：

齐还站得住。

问题只是，这句话如今要靠说几遍，才能像从前那样信了。
