# 第九十七回 乐毅不乘小胜先逼齐改 韩将渐服知大胜在忍

联军第一战得手之后，韩前营里一度都憋着一股要再往里咬的热。韩老将回营后连甲都没解，便在帐里来回踱步：“明明还能再试一口。”

夜半，中军再召议。

乐毅先看的，不是今日小胜，而是齐夜里如何添岗、如何换车、哪一垒增火最多、哪一垒反而更静。

韩老将忍不住道：“齐既已被我们啃下一口，它今夜忙成这样，不正是明日再逼的好时机？”

乐毅抬头：“将军以为，齐今夜最怕的是什么？”

老将一怔：“自然是怕我们再来。”

乐毅摇头：“不止。它更怕自己改得不够快，又怕自己改得太急而先乱。这样的敌，最容易在第二日把所有力都往昨天失的那口上压。”

说完，他把手移到图上另一处稍远的垒口旁：“明日，不先争今日之口。先逼它这一口动。”

第二日天未亮，联军果然不再照着昨日那口压。

韩前军只留外线稳住原得之地；

魏中军则暗把应节向侧略偏；

赵后线粮火照旧，却让一站更显、一站更暗。

齐边天明一看，心果然先乱了一下。忙了一夜，本以为联军今日必来夺回昨日未尽之势，谁知对面像根本不恋那一口，而是先把眼睛转向了别处。

午前，韩老将远远看见齐边果然把昨夜新压过去的一队车骑又匆匆往侧挪，终于明白了主将昨夜那句“更怕自己改得太急而先乱”的意思。

他站在坡上望了半晌，忽然不再觉得昨夜那份“明明还能再咬一口”的火有多可惜。

因为他亲眼看见了另一种更值钱的胜。不是今日多压下半垒，而是齐边明明忙了一夜，自以为把昨日失口补住了，今晨却又被联军逼得把刚压稳的车骑往旁边抽。抽这一回，齐自己便更会疑：我昨夜那样忙，到底补住了什么，又漏下了什么？
