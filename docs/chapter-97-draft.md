# 第九十七回 乐毅不乘小胜先逼齐改 韩将渐服知大胜在忍

联军那边，第一战得手之后，韩前营里一度都憋着一股要再往里咬的热。

尤其那老将，回营后连甲都没解，便几次在帐里来回踱步。

“明明还能再试一口。”

副将不敢多接，只低声道：“主将总节还未下新令。”

老将鼻中重重出气：“总节总节，他倒真沉得住。”

可嘴上虽硬，他心里却也明白，今日这一战若换自己统，十有八九已经追进去了。至于追进去之后，魏会不会被拖得改次，赵后线会不会被迫更站，齐会不会因此反倒得了“联军求快”的空子，他未必真算得清。

这也是他最不愿承认、却又不得不承认的一层：

今日这一口，主将看得比自己深。

夜半，中军再召议。

乐毅先看的，不是今日小胜，而是齐夜里如何添岗、如何换车、哪一垒增火最多、哪一垒反而更静。

韩老将终于忍不住道：“主将，齐既已被我们啃下一口，它今夜忙成这样，不正是明日再逼的好时机？”

乐毅抬头看他，问得却很平：“将军以为，齐今夜最怕的是什么？”

老将一怔：“自然是怕我们再来。”

乐毅摇头：“不止。它更怕自己改得不够快，又怕自己改得太急而先乱。这样的敌，最容易在第二日把所有力都往昨天失的那口上压。”

魏节应将在旁缓缓点头：“所以明日若仍照今日打法，齐便会最有备。”

乐毅这才道：“正是。”

说完，他把手移到图上另一处稍远的垒口旁，道：“明日，不先争今日之口。先逼它这一口动。”

帐中几人都微微一怔。

因为那处垒口不是今日主战处，却正连着齐边昨夜新调去的应车与补岗。若去逼那里，齐便会发现自己今夜辛苦改出来的那层防，又未必压对了地方。

韩老将看着那一点，许久没说话。

过了一会儿，才低声道：“原来主将今日拿下半口，不是为这一口。”

乐毅道：“拿半口，是为叫它改。它一改，我们便再看它改得哪里最重、哪里最急、哪里最露。”

这句话落下，帐中一时都静了。

因为到这一刻，几位主将都真正看清了：

乐毅不是在用一套已经定死的法子打齐。

他是在逼齐一步步改，然后专打它最新改出来、也最来不及养稳的那一层。

韩老将这回终于没有再争快。

他缓缓抱拳，道：“前军听令。”

乐毅看了他一眼，眼中第一次露出一点真正的认同：“将军今日能忍，后面便真能吃大胜。”

第二日天未亮，联军果然不再照着昨日那口压。

韩前军只留外线稳住原得之地，不躁；

魏中军则暗把应节向侧略偏；

赵后线粮火照旧，却让一站更显、一站更暗，像故意把齐夜里新改出来的那几处应路都照给它自己看。

齐边天明一看，心果然先乱了一下。

因为他们忙了一夜，本以为联军今日必来夺回昨日未尽之势。谁知对面竟像根本不恋那一口，而是先把眼睛转向了别处。

边将低声骂了一句，终于感觉到一种前所未有的烦。

不是因为敌强到不可挡。

而是因为你总像慢它一步，且你越改，它越像能立刻看出你刚改到了哪里。

午前，韩老将远远看见齐边果然把昨夜新压过去的一队车骑又匆匆往侧挪，忽然明白了主将昨夜那句“更怕自己改得太急而先乱”的意思。

他望着对面尘头，胸口那股一直不肯服的硬气，到这一刻竟慢慢落成了一种更沉的服。

不是服一句话。

是服一种层级。

他第一次极清楚地知道，自己擅咬阵、擅压锋、擅在刀口前硬顶，可若论怎么把敌人一层层逼得自己改、自己乱、自己越认真越迟半步，这位主将确实比自己高得多。
