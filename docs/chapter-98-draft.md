# 第九十八回 临淄北外齐军再换势 乐毅收线不贪将开大口

齐边第二日，果然比第一日更难受。

边将从早到午，来回调了两次车、三次岗、四次传令。每一回都不算大错，可每一回调完不久，联军那边便又像早知道似的，把新的轻重缓急压到了另一处。

到午后，齐边诸垒将校自己都已说不清，如今最该先护的是昨日失口、今日受逼的新口，还是再往后一线那些一旦再挪便可能全乱的中垒应道。

临淄午后又得边报。

齐湣王看见“敌不恋前口，转逼新应”“我军号车再移”“诸垒相顾”这些字时，终于不再立着看，而是慢慢坐回案后。

田文这回没有再绕，直接答道：“它要我齐自己先把每一层都挪动、挪薄、挪急，挪到最后，才真来咬那一口大处。”

齐湣王闭了闭眼，过了片刻才道：“传前线，先不争昨日今日之小口。收诸垒，锁中线，宁让半边，不许整线自乱。”

而在联军大帐，乐毅也很快得知齐边新令。

韩老将本以为主将会喜，谁知乐毅听完之后，只轻轻点头：“它总算学到一点了。”

赵奢问：“那我们是否该乘它收线时直逼？”

乐毅摇头：“不急。齐若现在只是殿中口头改了，前线未必真能一口改到底。我们此刻猛扑，大半会让它反而因外压而最快并住。我要再看半日，看它收线是不是收得整、收得齐、收得住。若收不住，那才是大口。”

当晚，联军果然仍只稳线，不抢。

韩前军立原口；

魏中军压节不躁；

赵后线照旧不乱。

乐毅站在图前，久久未动。帐外夜火轻轻照进来，映得图上那条将被齐自己重新收拢的中线，像一根刚拉紧、也刚最容易绷断的弦。
