# 第九十八回 临淄北外齐军再换势 乐毅收线不贪将开大口

齐边的第二日，果然比第一日更难受。

因为第一日丢的是半口坡脚；

第二日丢的，却是自己刚改出来的那点应势。

边将从早到午，来回调了两次车、三次岗、四次传令。

每一回都不算大错。

可每一回调完不久，联军那边便又像早知道似的，把新的轻重缓急压到了另一处。

到午后，齐边诸垒将校自己都已说不清，如今最该先护的是昨日失口、今日受逼的新口，还是再往后一线那些一旦再挪便可能全乱的中垒应道。

这种仗，比正面血拼更磨人。

因为你明明还没大败，

可心里那根原本想靠“先守、先改、先认真”重新并起的骨，却总被对面拿着你刚改出来的新法，一下一下敲得发空。

临淄午后又得边报。

齐湣王看见“敌不恋前口，转逼新应”“我军号车再移”“诸垒相顾”这些字时，终于不再立着看，而是慢慢坐回案后。

田文在旁，第一次从大王脸上看见了真正的疲。

不是败后的怒。

也不是轻敌被破后的惊。

而是一种更深的、来自“我已经在改了，可它比我改得更快”的疲。

这比前两层更险。

因为怒还可逼人拼，惊还可逼人收。

疲若先起，城中自上而下那口本来最容易撑住局面的气，便会先松。

齐湣王沉默许久，才问了一句：“它到底想要什么？”

田文这回没有再绕，直接答道：“它要我齐自己先把每一层都挪动、挪薄、挪急，挪到最后，才真来咬那一口大处。”

满殿无声。

因为这话太像真相。

而真相往往最叫人不想听。

齐湣王闭了闭眼，过了片刻才道：“传前线，先不争昨日今日之小口。收诸垒，锁中线，宁让半边，不许整线自乱。”

这一道新令，终于比前几日又退了一层。

不是怯。

是齐开始被逼着学会，先拿更大的线去护骨，不再处处都想把每一个已失或将失的小口立刻抢回来。

而在联军大帐，乐毅也很快得知齐边新令。

韩老将本以为主将会喜。

因为齐既再退一步，联军似乎正该乘势狠狠干上一口。

谁知乐毅听完之后，只轻轻点头，道：“它总算学到一点了。”

赵奢问：“那我们是否该乘它收线时直逼？”

乐毅却摇头：“不急。”

韩老将终于忍不住：“主将！齐已被我们逼得连线都要收了，再不打，还等什么？”

乐毅抬眼，看向他：“等它把这道新令真正传到底。”

老将一怔。

乐毅道：“齐若现在只是殿中口头改了，前线未必真能一口改到底。我们此刻猛扑，大半会让它反而因外压而最快并住。我要再看半日，看它收线是不是收得整、收得齐、收得住。若收不住，那才是大口。”

魏节应将在旁忽然笑了一下。

这人向来极少笑，这一笑，帐中几人反倒都看向了他。

他道：“韩将军，主将不是不打。主将是在等齐自己把门先关错。”

韩老将盯着图，看了很久，忽然也笑了一下。

笑意很短，却是这许多日来第一次真正带着服气的笑。

“好。那便再等它半日。”

当晚，联军果然仍只稳线，不抢。

韩前军立原口；

魏中军压节不躁；

赵后线照旧不乱；

燕总节营则把今日新得诸报一条条压在图上，静看齐边那道“收诸垒、锁中线”的新令，究竟会在黑夜里把哪些人先逼得急、逼得误、逼得自己先露口。

乐毅站在图前，久久未动。

帐外夜火轻轻照进来，映得图上那条将被齐自己重新收拢的中线，像一根刚拉紧、也刚最容易绷断的弦。

他心里很清楚，真正的大口，就在这根弦上。
