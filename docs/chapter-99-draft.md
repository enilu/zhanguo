# 第九十九回 齐边收线夜中先误令 乐毅乘半拍终于开大口

齐边那道“收诸垒、锁中线”的新令，是在午后从临淄发出的。

可边线太长，垒口太多，前两日又因韩、魏、赵、燕四路牵扯得处处都在挪。命令落到各处时，纸上虽一样，心里却未必一样。

有的将校听完，先想的是“总算可以不再去争昨日那半口坡脚”；

有的听完，却怕自己若退得太快，反倒成了先失职的一垒；

还有人一边收线，一边仍想着：若联军今夜又像前两日那样只逼不打，自己是否还可偷偷把某一队车再往前压半分，好叫明日不至太难看。

到夜里，齐边数口竟各自收成了不同样子。

有的真退；

有的半退半守；

有的表面锁中线，暗里还给自己留着一条“若见好可再顶出去”的旧手。

这种差半寸的乱，在平日里未必立刻要命。

可在乐毅眼里，却正是能开大口的缝。

二更将尽，探骑入帐。

“齐右中垒已收过头，左垒却仍留半队外压。”

“两处夜号先后不一。”

“中线车骑回拢时，留下了一截空。”

这三句一报完，帐中诸将几乎同时坐直。

乐毅却反而更静了。

他把三处回报一一压在图上，最后手指落到中线那一截空出来的地方，轻声道：“就是这里。”

韩老将眼里那层硬光几乎立刻跳起来：“前军请战！”

魏节应将也道：“中军可补。”

赵奢更短：“后线不断。”

乐毅点头，这回再没压。

“前军先起，不击其厚，直压其空。”

“中军随号，不得抢先，不得后误。”

“粮火今夜前明一站，给齐自己看。”

“诸军一旦得口，只许扩，不许贪。”

四道令一下，帐中气竟比前几日都沉。

因为人人都知道，这不是试了。

是真开。

韩前军最先出营。

夜色里不见大旗，只听得甲叶与马步在泥地上压出的沉声。前锋不去撞齐留得最厚的那两垒，而是直逼中线那一截夜中刚收出来、却还没来得及补死的空。

齐军那边直到听见前头短促而整的号，才猛然发觉对面今夜不是照旧试锋。

守将披甲冲出帐时，第一句便是：“它竟真从这里开！”

可这时再调，已晚半拍。

因为联军前几日最厉害的一点，恰恰就在这里：

不是哪一日多咬进去一点。

而是日日都逼得齐自己改，改到这一夜，终于有一处改得太急、太齐不住、太想两头都顾，结果真把口露出来了。

韩前军一压上去，齐那一截中线便先乱。

不是大崩。

是夜号先错了一拍，后队不知该往回并还是往前堵，留在那里的鹿角与车队也因本是临时收线，摆得既不深也不整，一撞便乱。

魏中军这时号起。

不像韩那样硬压。

只压在韩前军外侧与齐侧垒之间，既挡着齐想横补的一口气，又把韩最怕被侧击的那一层先托住。

这一托，韩前军心里那股原本最容易猛起来的火，反倒更能稳着往前走。

韩老将亲自在前，见齐中线那一口终于真开，胸口那股压了多日的战意几乎要炸开。

可他到底记住了主将那句“只许扩，不许贪”。

于是他没直扑更深处的第二道垒门，只先命左右把已开那口两侧撑大、钉稳，把后头该给魏中军立足的那一层地势先留出来。

这一忍，连他自己都觉得生疼。

可也正因这一忍，齐军本来还想靠“它一贪，我们便能并住”的那一点老路，当场又断了。

四更之前，齐边北外中线终于被真真切切撕开了一口。

不算深穿。

却已足够让几垒之间原本还勉强靠旧气硬连着的那条应路，彻底露在联军眼前。

联军营中并无大呼。

诸军照旧依号而动，前军止于新得之线，中军立于侧，粮火稳稳接上，燕总节营的令一层层传下去，竟比昨夜偏厅对号时还更静。

静得叫齐边更寒。

因为这说明，对面盼的本就不是一阵冲乱。

它盼的，是齐自己先把那一口开出来，然后它稳稳接过去。

天将亮时，新的急报已飞向临淄。

而在前线高处，乐毅望着那一口终于撕开的中线，脸上仍无大喜。

他只对身旁诸将道了一句：“口已开，齐后面便要改得更苦了。”
