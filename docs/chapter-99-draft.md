# 第九十九回 齐边收线夜中先误令 乐毅乘半拍终于开大口

齐边那道“收诸垒、锁中线”的新令，是在午后从临淄发出的。

可边线太长，垒口太多，前两日又因韩、魏、赵、燕四路牵扯得处处都在挪。命令落到各处时，纸上虽一样，心里却未必一样。到夜里，齐边数口竟各自收成了不同样子：有的真退，有的半退半守，有的表面锁中线，暗里还给自己留着一条旧手。

二更将尽，探骑入帐：

“齐右中垒已收过头，左垒却仍留半队外压。”

“两处夜号先后不一。”

“中线车骑回拢时，留下了一截空。”

这三句一报完，帐中诸将几乎同时坐直。乐毅把三处回报一一压在图上，最后手指落到中线那一截空出来的地方：“就是这里。”

韩老将眼里那层硬光几乎立刻跳起来：“前军请战！”

“前军先起，不击其厚，直压其空。”

“中军随号，不得抢先，不得后误。”

“粮火今夜前明一站，给齐自己看。”

“诸军一旦得口，只许扩，不许贪。”

韩前军最先出营。夜色里不见大旗，只听得甲叶与马步在泥地上压出的沉声。前锋不去撞齐留得最厚的那两垒，而是直逼中线那一截夜中刚收出来、却还没来得及补死的空。

齐军那边直到听见前头短促而整的号，才猛然发觉对面今夜不是照旧试锋。

韩前军一压上去，齐那一截中线便先乱。不是大崩，是夜号先错了一拍，后队不知该往回并还是往前堵，留在那里的鹿角与车队也因本是临时收线，摆得既不深也不整，一撞便乱。

魏中军这时号起，只压在韩前军外侧与齐侧垒之间，既挡着齐想横补的一口气，又把韩最怕被侧击的那一层先托住。

韩老将亲自在前，见齐中线那一口终于真开，胸口那股压了多日的战意几乎要炸开。可他到底记住了主将那句“只许扩，不许贪”，没直扑更深处的第二道垒门，只先命左右把已开那口两侧撑大、钉稳。

四更之前，齐边北外中线终于被真真切切撕开了一口。

不算深穿，却已足够让几垒之间原本还勉强靠旧气硬连着的那条应路，彻底露在联军眼前。
