# 战国小说核心人物表

这份文件以当前正文成稿为准，用于统一核心人物的写法、声口、功能与命运重心。

适用范围：

- `docs/chapter-XX-draft.md` 正文续写
- 章节回修与审读
- 后续补人物关系图、卷内场面索引、宣传文案时的人物统一口径

人物记录统一采用以下字段：

- 身份定位
- 能力标签
- 出场气质
- 正文中已形成的代表场面
- 致命短板
- 叙事功能
- 声口与写法提醒

## 一、纵横家与谋臣主轴组

### 1. 苏秦

- 身份定位：全书开局的第一位大主角，合纵路线的最高代表
- 能力标签：`合纵第一`
- 出场气质：困顿、锋锐、胸中有天下版图
- 正文中已形成的代表场面：困顿起身；六国游说；佩六国相印；眼看合纵名存实亡仍不肯轻弃那面残旗
- 致命短板：能聚势，难固势；能把天下暂时拧成一口气，却挡不住诸侯各自回头算本国账
- 叙事功能：为全书立下“天下尚可争衡”的起势，也让后文张仪破局、战争压倒辩说有了更重的反差
- 声口与写法提醒：宽、沉、有悲意；不要写成只会讲大道理的人，最好总系在路、书、简、未竟之事上

### 2. 张仪

- 身份定位：破盟主手，第二卷真正推动时代换气的人
- 能力标签：`破盟第一`
- 出场气质：沉稳、刻薄、短句如刀，极擅拿人心里那点“先顾自己”
- 正文中已形成的代表场面：入楚欺楚；反复入见楚怀王；把“再受骗”改说成“再权衡”；借诸侯私心一点点拆空合纵
- 致命短板：最会拆，不负责建；他的高明在于让旧局失效，而不是给天下新秩序
- 叙事功能：把“联盟可成”推进到“联盟虽在，已不可信”；让秦第一次真正显出比东方更懂利用裂缝
- 声口与写法提醒：短、稳、冷；不要长篇解释，往往一句平话比十句狠话更伤人

### 3. 范雎

- 身份定位：秦从强国走向吞并型大国时最关键的战略谋臣
- 能力标签：`远交近攻第一`
- 出场气质：阴沉、冷硬、极擅忍与反扑
- 正文中已形成的代表场面：把秦廷视线从一时胜负抬到长期吞并；看出廉颇不难打，难在让赵国自己不许他那样守
- 致命短板：目光太实，几无宽气；能为国家算尽利害，不会替旧义与旧情留多少地方
- 叙事功能：让秦不只会赢仗，更会把仗次第排进一张更大的国策图里
- 声口与写法提醒：冷，不多话；像在说一件迟早会发生的事，不像在抖机灵

### 4. 李斯

- 身份定位：帝国机器最核心的制度与政治执行者
- 能力标签：`帝国建构第一`
- 出场气质：清醒、冷静、善权衡，永远先看“这件事如何进入制度”
- 正文中已形成的代表场面：在咸阳把“胜到最后”说成一套法与器；与韩非形成同门异命；在统一后推动帝国秩序真正落地
- 致命短板：太能成事，也太知道如何自保；忠于秩序甚于忠于具体的人
- 叙事功能：把秦统一写成“国家机器如何完成”，而不是只靠一位君主或一群猛将
- 声口与写法提醒：句子要整，少热气；像把图纸一寸寸往案上铺开

### 5. 韩非

- 身份定位：法家思想最高锋刃，也是最典型的思想型悲剧人物
- 能力标签：`法术理论第一，现实生存最弱`
- 出场气质：冷峻、迟缓、孤高，像一柄太薄太利的刀
- 正文中已形成的代表场面：入秦论法；思想上压过殿中旧臣；最终被更冷的人世之用与帝国需要逼入死角
- 致命短板：能看透天下，不能保护自己；他说出的理越深，越容易在现实里容不下
- 叙事功能：把“法”从秦之术，抬到帝国命运之论；同时承担第五卷极重的一层悲剧寒意
- 声口与写法提醒：不要写成嘴炮型智者；他的锋利首先体现在思路之深和句义之寒

### 6. 陈轸

- 身份定位：楚国后段最清醒的诊病者
- 能力标签：`诊病开方第一`
- 出场气质：不怒而冷，像是在替一个将病未病的大国号脉
- 正文中已形成的代表场面：看清张仪破盟之后楚的病根；一再劝楚不要把面子当成骨头；在局已坏时仍试图给楚国开方
- 致命短板：看得早、说得对，但能用他的君王与时机都不够
- 叙事功能：让楚廷不只是“昏”，而是“明白人也在，只是药下得太晚”
- 声口与写法提醒：像诊病，不像空论；多用病、方、药、骨、气这类具体比喻

### 7. 屈原

- 身份定位：楚国最直的一根骨头
- 能力标签：`直言第一`
- 出场气质：硬、直、重是非，不替人找好听说法
- 正文中已形成的代表场面：在楚廷直撞怀王与群臣的侥幸、自欺与粉饰
- 致命短板：太直，难容于一个已习惯摇摆与护脸面的朝堂
- 叙事功能：让楚国的衰败不只是权谋败，也带着是非与人格之痛
- 声口与写法提醒：句子要硬，不绕，不替对方留台阶

### 8. 黄歇／春申君

- 身份定位：楚国后段少数还能“托局补骨”的大臣，后期转为权相
- 能力标签：`托局补骨第一`
- 出场气质：温厚而有韧，华贵里带大国相的气度
- 正文中已形成的代表场面：先被写成能托住楚廷残局的新士；后段逐渐具备春申君的相权规模
- 致命短板：能托一时之局，难逆楚之大势；权重之后亦易受旧病反噬
- 叙事功能：说明楚并非无人，只是再有能臣也只能让这具大国之躯慢些倒
- 声口与写法提醒：宽，不急躁；重“托”“稳”“补”，少尖刀式发言

## 二、统帅与名将主轴组

### 9. 白起

- 身份定位：全书最可怕的歼灭战统帅
- 能力标签：`歼灭战第一`
- 出场气质：寡言、冷定、像在看一张别人还没看见的死图
- 正文中已形成的代表场面：伊阙后逐步立威；在长平前看赵“还未自急”；合围长平，整整打断赵国元气
- 致命短板：太峻、太利，与长期政治终究难相容
- 叙事功能：把“秦能胜”推进成“秦能断一国元气”
- 声口与写法提醒：少说，多看图，多问细项；不要写成纯猛将

### 10. 廉颇

- 身份定位：赵国最懂如何“不败”的宿将
- 能力标签：`持重坚韧第一`
- 出场气质：沉、稳、硬，像一块不肯轻动的旧铁
- 正文中已形成的代表场面：长平前把赵线坐成铁壁；用粮、垒、月、线而不是靠一时热战扛住秦军
- 致命短板：守法正确，却太不合朝堂与年轻将校那口求快求名的气
- 叙事功能：让赵括之败不只是个人败，而是赵国先不许廉颇那种守法继续存在
- 声口与写法提醒：老，不絮叨；多落在灯下仓、垒上风、粮路月耗这些耐性物上

### 11. 赵括

- 身份定位：长平卷的核心悲剧人物
- 能力标签：`兵书志气极盛，实战转局最险`
- 出场气质：亮、热、自信，像一把刚出鞘便想立名的锋
- 正文中已形成的代表场面：受军接廉颇；以“争坡”把赵军从试拉入争；最终整口新锋压进白起设好的深处
- 致命短板：太想证明新法与自己，没能真正承受“不能急”的分量
- 叙事功能：让长平之败具备最大的人物悲剧性，而不是单纯战术失手
- 声口与写法提醒：不要写傻；他是热而急，不是蠢而空

### 12. 乐毅

- 身份定位：最会把联军打成秩序与势的跨国统帅
- 能力标签：`跨国远征第一`
- 出场气质：儒雅、整肃、深，看人心与军令都看得细
- 正文中已形成的代表场面：统五国伐齐；不急争门争桥，而是逼齐把“补漏法”一步步学进骨里
- 致命短板：功成之后终究难久，六国体系本身也容不住这份长期成果
- 叙事功能：证明六国并非不能打出高峰，但他们的高峰也常常不能久持
- 声口与写法提醒：多用图、小札、告示、秩序；少用空泛豪言

### 13. 赵奢

- 身份定位：赵国中段最亮的一位名将，也承担赵国内逼实、边上守稳的执行面
- 能力标签：`奇正兼具，名短而亮`
- 出场气质：干练、锐利、能把散的东西一件件逼实
- 正文中已形成的代表场面：把赵国内部军赋、粮马、贵戚窟窿一件件逼实；与廉颇形成赵国军政两层骨架
- 致命短板：名势不够长，未能成为长期扛局之人；其“后继”反倒转成赵国重创的一层反差
- 叙事功能：让赵国不是只有廉颇一种强，也解释赵括为何会带着更高的期待被推上去
- 声口与写法提醒：重后果、重执行，少飘

### 14. 李牧

- 身份定位：六国后期真正能让秦也不敢轻进的最后军魂
- 能力标签：`后期六国第一名将`
- 出场气质：沉静、坚忍、寡言，像最后一层还没断的硬骨
- 正文中已形成的代表场面：苦撑赵北；让秦不得不先用离间；死在朝局先坏而非阵前先败
- 致命短板：强在个人，败在国势与君臣疑心已不可逆
- 叙事功能：把“六国虽衰未屈”这层军魂写到最重
- 声口与写法提醒：越写他强，赵亡越痛；不要让他像另一个廉颇

### 15. 王翦

- 身份定位：帝国后段最稳的灭国统帅
- 能力标签：`灭国经营第一`
- 出场气质：深、缓、稳、老而极会先算君王之疑
- 正文中已形成的代表场面：索赏、要兵、先把疑心坐实；对楚用“更大、更稳、更能等”的打法
- 致命短板：传奇锋芒不如白起直刺人心，威压更多来自深与稳
- 叙事功能：把秦之胜从“歼灭名将”推进到“完成帝国最后工程”
- 声口与写法提醒：不喊，不躁；总像先把最难那一层人心算完

### 16. 项燕

- 身份定位：楚国最后一层硬骨与最后一支死战之兵的主帅
- 能力标签：`亡国死战第一`
- 出场气质：雄烈、悲壮、国气虽败而血仍热
- 正文中已形成的代表场面：率楚军以最沉的方式死撑到底；让灭楚成为全书最后最沉重的一场灭国大战
- 致命短板：力能战，势难回；败不在才，而在时代已只给楚留下死撑的空间
- 叙事功能：让楚亡不轻，让秦统一的代价真正压到最重
- 声口与写法提醒：写沉，不写浮；多用“撑”“压”“拖”“死战”

## 三、帝王与关键君主组

### 17. 嬴政

- 身份定位：全书后段真正意义上的时代主角，统一的最终执行者
- 能力标签：`帝王决断第一`
- 出场气质：锐、孤、冷、强，越到后面越像坐在图中央的人
- 正文中已形成的代表场面：亲手整肃嫪毐、吕不韦旧势；听韩非、用李斯；一路完成灭国与一统
- 致命短板：手段极重，统一之后也把新的裂纹一并埋下
- 叙事功能：把“秦之胜”具体化为一个人的意志与一套国家机器的合谋
- 声口与写法提醒：不要写成单纯明君或暴君；他必须像一个压时代的复杂巨影

### 18. 楚怀王

- 身份定位：第二卷最关键的失手型君主
- 能力标签：`志大而轻信`
- 出场气质：自负、犹疑、护脸面、总想两头都占
- 正文中已形成的代表场面：被张仪一步步牵着走；在“再信秦”与“再权衡”之间自己替自己找说辞
- 致命短板：耳根不稳，识人不明，总在关键一步先护面子后算大局
- 叙事功能：让楚之大败不是忽然庸，而是反复错在最要命处
- 声口与写法提醒：急气要足；他不是木头，是总想补面子的人

### 19. 赵武灵王

- 身份定位：全书背景层里赵国最具开创性的前代雄主
- 能力标签：`变法强兵第一`
- 出场气质：英锐、决绝、改革者锋芒极重
- 正文中已形成的代表场面：主要作为赵国国势底色与人物回望中的高峰坐标
- 致命短板：雄主太盛，晚年内患反噬
- 叙事功能：说明六国并非没有自强能力，也让后世赵之兴衰更具纵深
- 声口与写法提醒：目前更多用于背景提势，不宜滥用直接出场

## 四、烈士、公子与关键辅助组

### 20. 信陵君

- 身份定位：六国后段最后一位能把“愿替别人先赌命”写成现实的人
- 能力标签：`公子养士第一`
- 出场气质：风流、重义、能得士心，真正站出来时又极决
- 正文中已形成的代表场面：窃符救赵；在天下最暗处替六国还剩的一点人心狠狠干出一道光
- 致命短板：可成一时义举，难救魏国根本气运
- 叙事功能：让长平之后的时代不只剩下绝望，还留住一层悲中有烈的高光
- 声口与写法提醒：义气要真，不可写成空泛贵公子

### 21. 太子丹

- 身份定位：燕国最后反秦意志的承担者
- 能力标签：`孤注一掷第一`
- 出场气质：阴郁、急烈、亡国阴影逼出来的锋
- 正文中已形成的代表场面：押出荆轲这柄最后的险锋
- 致命短板：决绝有余，终究无力逆转总体实力差
- 叙事功能：说明弱国绝境中仍会生出最危险也最悲的最后反扑
- 声口与写法提醒：不要写成纯莽；他是知绝而赌

### 22. 荆轲

- 身份定位：全书个人悲壮色彩最强的人物之一
- 能力标签：`独胆赴死第一`
- 出场气质：沉静、决绝、隐士与烈士气相杂
- 正文中已形成的代表场面：图穷匕见；把一柄匕首真正送到帝王案前
- 致命短板：个人之烈，终究无法改变国家之势
- 叙事功能：把亡国者最后的不甘与尖意压缩到一人一夜一匕里
- 声口与写法提醒：重点永远在赴死前的气与赴死时的直，不在炫技

## 五、使用说明

### 5.1 写作时优先检查

每次准备写核心人物时，优先回看以下四项：

- 该人物在本书中的能力标签是否仍清楚
- 该人物是否已有至少一个正文里立住的代表场面
- 该人物的短板是否正与其败、挫、迟、误形成因果
- 该人物说话是否已和其他人分出声口

### 5.2 禁止事项

- 不把所有名将都写成同一种“冷静强者”
- 不把所有谋臣都写成同一种“善辩聪明人”
- 不让人物只剩标签，没有命运与代价
- 不为了强调秦强而把六国可敬人物写弱
- 不把烈士人物写成只会喊口号的薄角色
