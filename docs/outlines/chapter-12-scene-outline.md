# 第十二回正文细纲

对应回目：`强国争衡难忘旧算 小邦附义各有私谋`

本细纲用于把第十二回推进到可直接落笔的场景级结构。第十二回的任务，是把第十一回已经浮出来的摩擦进一步摊开，不再停留在事务层面，而是深入到六国各自对合纵的根本期待与根本限度。

如果第十一回解决的是：

`同盟一运转，难处便出来了。`

那么第十二回要解决的是：

`为什么这些难处不是偶然，而是六国从一开始就各有旧算，因此注定会不断冒出来。`

## 一、第十二回核心任务

1. 把强国与小国对合纵的不同立场写透
2. 把“共同惧秦”之外的旧算盘重新摊开
3. 让苏秦面对的困难从“处理事务”升级为“处理结构性分歧”
4. 让读者更清楚同盟为什么天然不牢
5. 在回末形成“局未坏，但每国都在按自己的方式使用这个同盟”的认识

## 二、场景拆分

### 场景一：盟中安坐，心中各算

- 功能：把上一回的事务摩擦推进为国家立场差异
- 写法：从数国代表的私议切入，不写大吵，而写各有保留
- 关键：表面都认同合纵，心里却各自另有账本

### 场景二：强国之疑，怕分其权

- 功能：写赵、齐、楚这类有分量国家的顾虑
- 核心：它们不怕合纵本身，怕的是合纵以后自己的主导权被稀释
- 关键：强国不愿白白替众国扛险

### 场景三：小邦之惧，怕为人盾

- 功能：写魏、韩、燕等更偏现实自保的一面
- 核心：它们想借盟保命，却又怕自己先被推到前面
- 关键：弱国不是没骨气，而是最清楚“名义的共同”未必能换来真正援手

### 场景四：苏秦再持衡，知其非一时口舌可尽解

- 功能：写苏秦开始明白，他面对的不只是意见不同，而是利益层次不同
- 写法：不必再长篇说理，而是以短促判断、次序安排、轻重分配来压局
- 关键：他开始更像织网者，而不是辩士

### 场景五：同盟未散，旧算未消

- 功能：收束本回
- 核心：众国仍会附在合纵之名下，但每一国都在按自己的方式解释这个同盟
- 回末感受：合纵像一张越拉越紧的网，没破，却已处处受力

## 三、本回重点

- 强化“同盟不是团结，而是暂时同向”
- 强化苏秦的吃力感，但不写其失控
- 为后面秦更精准观察裂缝铺路

## 四、必须避免

- 不要把强国写成纯傲慢
- 不要把小国写成纯怯弱
- 不要把分歧写得全靠几句争吵表现
- 不要让本回只剩抽象分析，没有人物气息

## 五、一句话概括

`合纵能成，是因为六国同惧秦；合纵难久，也是因为除了惧秦之外，六国没有哪一国真肯把自己的旧算轻易放下。`
