# 第十三回正文细纲

对应回目：`苏子持衡频调众口 秦人观衅暗记诸邦`

本细纲用于把第十三回推进到可直接落笔的场景级结构。第十三回第一次把苏秦的“热中持局”和秦的“冷中观衅”并列得更深。

如果第十二回解决的是：

`六国各有旧算，分歧出自结构。`

那么第十三回要解决的是：

`当苏秦不断修补这些结构裂缝时，秦已经开始按类别、按次序、按弱点去看这个联盟。`

## 一、第十三回核心任务

1. 写苏秦如何继续频繁调停众国
2. 写他不是无事可做，而是处处都要补缝
3. 写秦对六国不是笼统看，而是开始逐类理解
4. 建立“东方热修补，秦冷记账”的对照
5. 在回末留下秦已看出同盟骨弱于名的判断

## 二、场景拆分

### 场景一：苏秦奔走，日日补局

- 功能：把苏秦高位后的日常状态具象化
- 写法：多点短切，写他不断在不同阵营间转圜
- 关键：他不是偶尔救火，而是天天都在救火

### 场景二：一国一缝，处处不同

- 功能：具体化不同国家的问题类型
- 写法：赵有强国之矜，魏韩有近秦之惧，燕有附势之盼，齐楚有大国之算
- 关键：让读者明白苏秦不是重复工作，而是在处理不同种类的裂缝

### 场景三：秦廷得报，开始分类看局

- 功能：正式建立秦的“结构性观察”能力
- 写法：通过边臣、探报、议者口中，把六国弱点归类
- 关键：秦不急着破，只先把每一种弱点记清

### 场景四：热与冷的并行对照

- 功能：本回核心美感所在
- 写法：一边写苏秦忙于持衡，一边写秦冷静记账，交错推进
- 关键：同样是在处理这个联盟，东方靠人心与口舌，秦靠判断与耐心

### 场景五：名可借，骨未成

- 功能：收束本回
- 核心：秦已隐约看明白，这个联盟可以借名成势，却未必能经久成骨
- 回末感受：真正可怕的不是秦出手，而是秦已经知道以后该打哪里

## 三、本回重点

- 写苏秦的持局消耗继续加深
- 写秦的战略视野开始高出一层
- 为后续张仪式破局埋“分类打击”的逻辑基础

## 四、必须避免

- 不要让秦显得已经胸有成竹到轻松无敌
- 不要让苏秦显得只是疲于奔命、毫无作用
- 不要把对照写成机械切镜头

## 五、一句话概括

`苏秦在众国之间缝缝补补，尽力让合纵还像一个整体；而秦在关中所做的，却是把这个整体一层层看成几种不同的裂法。`
