## 标题优化原则

1. **统一风格**：人物 + 事件 + 悬念/结果
2. **避免模板化**：句式多变，杜绝雷同
3. **有记忆点**：朗朗上口，易于记忆
4. **有钩子**：每章标题都包含冲突或悬念
5. **字数控制**：尽量7-14字，符合阅读习惯

## 回数标注修正

- 确保回数顺序正确，从1开始递增
- 避免重复标注，如"第一百零一回"等错误
- 每卷10回，共50回，结构清晰
