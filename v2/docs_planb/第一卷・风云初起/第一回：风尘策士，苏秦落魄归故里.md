# 第一回：风尘策士，苏秦落魄归故里

## 归乡

夕阳西下，洛阳城的城门缓缓关闭。一个衣衫褴褛的年轻人背着破旧的书箱，站在城门外，望着熟悉的城楼，眼中泛起泪光。

他叫苏秦，今年二十七岁，离开家乡已经三年了。三年前，他满怀壮志，辞别父母，前往齐国求学，师从鬼谷子，学习纵横之术。如今，他学成归来，却一事无成，连路费都花光了。

"苏先生，进城吗？"守城的士兵看着他，语气中带着不屑。

苏秦苦笑一声，掏出怀里最后一枚铜板，递给士兵："烦请通融一下。"

士兵接过铜板，挥了挥手："进去吧。"

苏秦走进城门，熟悉的街道映入眼帘。他沿着青石板路前行，来到自己家的门口。

房门紧闭，他轻轻敲门："娘，我回来了。"

门内传来一个苍老的声音："谁啊？"

"娘，是我，苏秦。"

门吱呀一声开了，一个白发苍苍的老妇人出现在门口。她看着苏秦的模样，愣了愣，然后扑上去抱住他："秦儿，你怎么变成这样了？"

苏秦鼻子一酸，眼泪掉了下来："娘，儿子让您担心了。"

## 家人的态度

苏秦走进屋，发现家里还是和三年前一样，破旧不堪。父亲坐在炕上，抽着旱烟，看到他回来，只是哼了一声，没有说话。

妻子正在织布，看到他进来，停下手中的活计，脸上没有任何表情。

苏秦的嫂子端着一碗粥进来，看到他，把碗往桌子上一放，冷笑道："哟，我们的大学士回来了？怎么没骑高头大马，没带金银财宝啊？"

苏秦的脸涨得通红，他知道自己现在的样子很狼狈，但没想到家人会这么对待他。

"嫂子，我这次回来，是想休整一下，然后再出去谋事。"苏秦小声说。

"谋事？"嫂子冷笑一声，"就你这样，还能谋什么事？不如跟着你哥去种地，至少能混口饭吃。"

父亲把烟袋往桌子上一磕："苏秦，不是我说你，你读那些书有什么用？不如学门手艺，养家糊口。"

苏秦的妻子也开口了："相公，你要是实在没本事，就别出去丢人现眼了，在家好好过日子吧。"

苏秦听了这些话，心如刀绞。他想起自己在齐国求学时的雄心壮志，想起鬼谷子对他的期望，想起自己曾经立下的誓言：一定要出人头地，让家人过上好日子。

## 深夜苦读

当晚，苏秦躺在炕上，翻来覆去睡不着。他想起自己在齐国的学习，想起鬼谷子说过的话："纵横之术，非大才不能掌握。你有这个天赋，但需要更多的努力。"

他悄悄爬起来，点上油灯，打开书箱，拿出《阴符经》，开始研读。

夜深了，家人都睡了，只有苏秦还在读书。他越读越兴奋，越读越觉得自己之前的理解太浅了。

"苏先生，这么晚了还在读书？"一个细微的声音传来。

苏秦抬头一看，是家里的小丫鬟阿竹。她端着一碗热水，站在门口。

"阿竹，你怎么还没睡？"苏秦问。

"我看您房间还亮着灯，就给您端碗热水来。"阿竹把水放在桌子上，"先生，您别太辛苦了。"

苏秦看着阿竹，心里一暖。在这个家里，只有这个小丫鬟对他还有点善意。

"谢谢你，阿竹。"苏秦说。

阿竹笑了笑，转身出去了。

苏秦继续读书，不知不觉天已经亮了。他合上书，伸了个懒腰，感觉精神饱满，思路清晰。

## 下定决心

第二天早上，苏秦对家人说："我决定了，我要再次出去游说诸侯。这次，我一定要成功。"

家人都嘲笑他："就你这样，还想游说诸侯？别做梦了。"

苏秦没有反驳，他知道现在说什么都没用，只有成功才能证明自己。

他收拾好行李，再次辞别家人。临走前，他对阿竹说："阿竹，等我成功了，我会回来接你的。"

阿竹红着脸点了点头："先生，我相信你。"

苏秦走出家门，望着天空，暗暗发誓："这次，我一定要佩六国相印，让那些嘲笑我的人刮目相看。"

他不知道，他的命运即将改变，战国的历史也将因他而改写。
