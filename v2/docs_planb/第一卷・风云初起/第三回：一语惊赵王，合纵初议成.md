# 第三回：一语惊赵王，合纵初议成

## 邯郸城外

邯郸城，赵国都城。

苏秦和赵纵站在城门外，望着高耸的城墙。城门口车水马龙，商贾往来，一派繁华景象。

"先生，我们怎么进王宫？"赵纵问。

苏秦整理了一下衣衫："先找一家客栈住下，然后想办法见到赵王。"

两人进城，找了一家偏僻的客栈住下。苏秦让赵纵去打听消息，自己则在房间里研究对策。

傍晚，赵纵回来汇报："先生，我打听到了。赵王明天要在朝堂上商议边防之事，所有大臣都会参加。"

"好机会。"苏秦眼中闪过一丝光芒，"赵纵，你去准备一份名帖，就说我苏秦有安天下之策，求见赵王。"

"先生，您有把握吗？"赵纵有些担心。

"没有把握也要试。"苏秦说，"记住，游说之道，在于制造机会。明天朝堂上，必有我表现的机会。"

## 朝堂之上

第二天，赵王宫。

赵王坐在龙椅上，面色凝重。殿下站着文武百官，气氛紧张。

"诸位爱卿，"赵王开口，"秦军在边境集结，似有东出之意。诸位有何对策？"

大将军廉颇出列："大王，臣愿率大军前往边境，与秦军决一死战！"

"不可。"丞相蔺相如摇头，"秦军势大，我军贸然出战，恐有不测。"

"那怎么办？"赵王焦虑地问，"难道就坐以待毙？"

众臣议论纷纷，但都说不出个所以然来。

就在这时，殿外传来一个声音："草民苏秦，有安天下之策，求见大王！"

赵王皱眉："苏秦？是何人？"

侍卫禀报："大王，是一个书生，说有退秦之策。"

"让他进来。"赵王说。

## 一语惊人

苏秦走进大殿，向赵王行礼："草民苏秦，拜见大王。"

赵王打量着他："你有何策，可退秦军？"

苏秦直起身，目光扫过殿中文武，声音不高，却字字清晰："大王，秦军虽强，但六国合力，可胜秦。"

"六国合力？"赵王冷笑，"六国各自为战，如何能合力？"

"正因为各自为战，所以才被秦国各个击破。"苏秦说，"大王，秦国之心，早已不在一城一地，而在鲸吞天下。六国今日割五城、明日献十邑，看似苟安，实则一步步把刀递到秦人手里。"

殿中一片寂静。

苏秦上前一步，继续说道："但若六国合纵，以韩魏为屏障，以赵楚为骨干，齐燕为后援，缔结盟约，一国受敌，五国共伐——秦再强，也不敢踏出函谷关一步！"

赵王的手指轻轻叩击案几，眼中闪过一丝光芒："继续说。"

"合纵之策，于赵国有三利。"苏秦伸出三根手指，"其一，赵国不再孤军抗秦，有盟友相助；其二，赵国可借合纵之名，整合六国资源，增强国力；其三，若合纵成功，大王将成为六国盟主，名垂青史！"

赵王沉默了。

## 郦姜献策

就在这时，殿后传来一个女声："大王，臣女有话要说。"

众人回头，只见一个身着华服的年轻女子从殿后走出。她容貌秀丽，但眉宇间透着一股英气。

"郦姜，你怎么来了？"赵王问。

"臣女听说有人献策合纵，特来一听。"郦姜向赵王行礼，然后看向苏秦，"苏先生，你的合纵之策，听起来很好，但实施起来却难。六国各有私心，如何能同心协力？"

苏秦看着郦姜，微微一笑："姑娘问得好。六国确实有私心，但正因为有私心，才更容易被说服。"

"哦？"郦姜挑眉，"愿闻其详。"

"韩国最弱，怕秦怕赵，合纵可保其安全；魏国曾经强大，如今衰落，合纵可助其恢复荣光；齐国远离秦国，但担心赵国扩张，合纵可消除其疑虑；楚国地广人多，但内部不稳，合纵可获外部支持；燕国最北，与秦不接壤，但担心赵国，合纵可安其心。"

苏秦顿了顿，继续说道："六国各有诉求，只要对症下药，必能说服。"

郦姜眼中闪过一丝赞赏："先生果然深谋远虑。大王，臣女以为，苏先生之策可行。"

赵王看着苏秦，又看看郦姜，终于开口："苏秦，寡人任命你为纵约长，出使列国，缔结合纵盟约。郦姜，你协助苏秦，负责宫廷联络。"

"谢大王！"苏秦和郦姜同时行礼。

## 初议成

退朝后，苏秦、郦姜和赵纵在宫外相会。

"多谢姑娘相助。"苏秦向郦姜行礼。

"不必谢我。"郦姜淡淡地说，"我只是觉得你的策有道理。不过，我要提醒你，合纵之路困难重重，你要有心理准备。"

"我知道。"苏秦说，"但有姑娘相助，我相信一定能成功。"

郦姜看了他一眼，没有说话，转身离去。

赵纵凑过来："先生，这位郦姜姑娘是什么人？"

"赵国贵族之女，懂政治、懂人心。"苏秦说，"有她相助，我们在赵国的行动会顺利很多。"

"先生，接下来我们去哪里？"赵纵问。

"韩国。"苏秦说，"韩国最弱，也最怕秦，最容易说服。只要韩国加入，其他国家就会跟风。"

苏秦望着远方，心中充满了期待。合纵的大幕，即将拉开。
