# 第二回：刺骨苦读，胸中藏天下棋局

## 闭门苦读

苏秦离开家乡，没有再去齐国，而是来到了赵国的一处僻静山谷。这里远离尘嚣，适合静心读书。

他租了一间破旧的茅屋，开始了闭门苦读的日子。

白天，他研读《阴符经》《鬼谷子》等典籍；夜晚，他在油灯下反复揣摩纵横之术的精髓。

"苏先生，您已经三天没吃东西了。"房东老丈看着他憔悴的样子，忍不住劝道。

苏秦抬起头，眼中布满血丝："老丈，我不饿。您先放着，我看完这段就吃。"

老丈叹了口气，把饭菜放在桌上，转身出去了。

苏秦继续看书，完全沉浸在书里的世界中。他发现，自己以前对纵横之术的理解太肤浅了。纵横不仅仅是游说技巧，更是对人心、对大势的深刻洞察。

"合纵连横，关键在于势。"苏秦喃喃自语，"六国之势，在于合；秦国之势，在于分。若能合六国之力，秦国必不敢东出。"

## 刺股之痛

夜深了，苏秦的眼皮越来越重。他已经连续读书十几个时辰，身体极度疲惫。

"不能睡，不能睡……"苏秦用力掐自己的大腿，但困意依然袭来。

他环顾四周，看到墙上挂着一把锥子。那是房东老丈用来修补茅屋的工具。

苏秦拿起锥子，毫不犹豫地刺向自己的大腿。

"啊！"剧烈的疼痛让他瞬间清醒，鲜血顺着大腿流下。

苏秦咬紧牙关，继续读书。疼痛让他保持清醒，也让他更加坚定。

"苏先生，您这是干什么？"老丈听到叫声，冲进屋里，看到苏秦腿上的血迹，吓得脸色发白。

"老丈，我没事。"苏秦勉强笑了笑，"这是让我保持清醒的方法。"

老丈摇摇头，拿来布条帮他包扎："您这样读书，会把自己读死的。"

"读死也比碌碌无为强。"苏秦说，"老丈，您知道吗？我苏秦此生，要么佩六国相印，要么就死在这间茅屋里。"

老丈看着他坚定的眼神，不再劝说，只是默默地帮他包扎好伤口。

## 领悟棋局

日复一日，苏秦的学问越来越深。他不再满足于书本上的知识，开始思考天下大势。

他在地上画了一张地图，标注出七国的位置、兵力、经济状况。

"秦国最强，但六国合力，可胜秦。"苏秦指着地图，自言自语，"问题在于，如何让六国相信合纵对他们有利？"

他想起鬼谷子的教诲："游说之道，在于知彼知己。你要知道对方想要什么，害怕什么，然后对症下药。"

苏秦开始分析六国君主的心理：

- **赵王**：赵国紧邻秦国，最害怕秦国东出，最需要合纵
- **韩王**：韩国最弱，夹在秦赵之间，左右为难
- **魏王**：魏国曾经强大，如今衰落，渴望恢复昔日荣光
- **齐王**：齐国远离秦国，相对安全，但需要盟友制衡秦国
- **楚王**：楚国地广人多，但内部不稳，需要外部支持
- **燕王**：燕国最北，与秦国不接壤，但担心赵国扩张

"每个人都有自己的算盘，"苏秦眼中闪过一丝光芒，"但只要我找到他们的共同点，就能说服他们合纵。"

## 赵纵来访

这天，一个年轻人来到茅屋前，恭敬地敲门："请问是苏秦先生吗？"

苏秦打开门，看到一个二十出头的年轻人，衣衫朴素但整洁，眼神中透着聪慧。

"我是苏秦，你是？"

"学生赵纵，久闻先生大名，特来拜师求学。"年轻人跪地行礼。

苏秦打量着他："你为何想跟我学习？"

"学生出身寒门，但心怀天下。听说先生在研究合纵之术，学生也想为六国尽一份力。"赵纵诚恳地说。

苏秦点点头："进来吧。"

赵纵进屋，看到满地的地图和书籍，眼中露出敬佩之色："先生果然深谋远虑。"

"赵纵，我问你，"苏秦指着地图，"若让你去游说赵王，你会怎么说？"

赵纵想了想："学生会说，秦国强大，赵国危险，必须联合其他国家共同抗秦。"

苏秦摇摇头："太直白了。赵王不是傻子，他知道秦国危险。你要告诉他的是，合纵对他有什么好处，不合纵有什么坏处。"

"请先生指教。"

"你要说：'大王，秦国如猛虎，六国如羔羊。羔羊分散，必被虎噬；羔羊抱团，虎亦不敢近。合纵不是为别人，是为赵国自己。'"

赵纵恍然大悟："先生高见！"

## 出山

一年后，苏秦的学问大成。他不仅精通纵横之术，更对天下大势了如指掌。

"是时候出山了。"苏秦对赵纵说。

"先生要去哪里？"赵纵问。

"先去赵国。"苏秦说，"赵国紧邻秦国，最渴望合纵。只要说服赵王，其他五国就容易了。"

"学生愿随先生同行。"赵纵说。

苏秦看着他，点点头："好，你就做我的书童，负责联络和文书。"

两人收拾行装，离开茅屋。房东老丈送他们到村口："苏先生，祝您一路顺风，早日实现抱负。"

"多谢老丈这一年的照顾。"苏秦深深鞠躬，"他日若有所成，必当厚报。"

老丈摆摆手："不必了，能看到您成功，老朽就心满意足了。"

苏秦和赵纵踏上了前往赵国的道路。苏秦知道，这将是一条充满挑战的道路，但他已经准备好了。

"天下如棋局，六国如棋子。"苏秦望着远方，眼中闪烁着自信的光芒，"我苏秦，要做那个执棋的人。"
