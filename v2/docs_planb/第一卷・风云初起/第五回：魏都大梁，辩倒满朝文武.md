# 第五回：魏都大梁，辩倒满朝文武

## 大梁风云

大梁，魏国都城。

与冷清的新郑不同，大梁繁华依旧。街道上商贾云集，酒肆茶楼人声鼎沸，仿佛感受不到秦国的威胁。

苏秦三人进城，找了一家客栈住下。

"先生，魏国看起来比韩国强多了。"赵纵说。

"表面繁华，实则外强中干。"苏秦摇头，"魏国曾经拥有河西之地，兵强马壮。如今河西尽失，国力大不如前。"

"那魏王为何还如此傲慢？"郦姜问。

"正因为曾经强大，所以放不下身段。"苏秦说，"魏王渴望恢复昔日荣光，但又不愿意承认现实。这种人，最难说服，也最容易说服。"

"什么意思？"赵纵不解。

"只要让他看到希望，他就会拼命抓住。"苏秦眼中闪过一丝光芒。

## 朝堂之上

第二天，魏王宫。

魏惠王坐在龙椅上，目光傲慢。殿下站着文武百官，阵容比韩赵两国豪华得多。

"赵国的苏秦？"魏惠王打量着苏秦，"听说你在赵国、韩国鼓吹什么合纵？"

"正是。"苏秦不卑不亢，"大王，合纵是六国唯一的生路。"

"生路？"魏惠王冷笑，"寡人看是死路！六国各有私心，如何能同心协力？"

"大王说得对，六国确实有私心。"苏秦说，"但正因为有私心，才更容易被说服。"

"哦？"

"大王，魏国曾经拥有河西之地，兵强马壮，是天下霸主。如今河西尽失，国力衰退，大王甘心吗？"

魏惠王脸色一变："你！"

"草民不敢冒犯大王，但草民说的是事实。"苏秦继续说，"合纵，不仅是为了抗秦，更是为了恢复魏国的荣光！"

## 群臣围攻

这时，一位大臣出列："大王，此人妖言惑众，应当逐出大殿！"

苏秦看向那人，认出他是魏国大将军庞涓。

"庞将军，"苏秦不慌不忙，"您为何说我是妖言惑众？"

"合纵不过是纸上谈兵！"庞涓怒道，"六国军队互不统属，如何能与秦军抗衡？"

"将军说得对，六国军队确实互不统属。"苏秦说，"但合纵不是让六国军队合并，而是让六国在战略上协调。一国受敌，五国支援。秦军再强，能同时对抗六国吗？"

另一位大臣出列："苏秦，你说得轻巧。但六国之间矛盾重重，如何能协调？"

"大人问得好。"苏秦说，"六国之间的矛盾，源于利益冲突。但只要找到共同的敌人，就能暂时放下矛盾。秦国，就是六国共同的敌人！"

"荒谬！"又一位大臣站出来，"秦国与我国并无深仇大恨，为何要与之作对？"

苏秦看着那位大臣，冷冷地说："大人，秦国已经占领了河西之地，下一步就是函谷关。函谷关一失，大梁危矣。大人是想等到秦军兵临城下，才想起合纵吗？"

那位大臣哑口无言。

## 魏王心动

魏惠王沉默了。

苏秦上前一步，声音提高："大王，魏国曾经称霸天下，如今却沦为二流国家。大王甘心吗？合纵，是魏国恢复荣光的唯一机会！"

"恢复荣光……"魏惠王喃喃自语。

"若合纵成功，魏国可借六国之力，收复河西之地，重振霸业！"苏秦掷地有声，"大王，这是千载难逢的机会啊！"

"可是……"魏惠王犹豫，"若合纵失败呢？"

"若合纵失败，魏国也不过是维持现状。"苏秦说，"但若合纵成功，魏国就能重现辉煌！大王，这是无本万利的买卖，何乐而不为？"

魏惠王看向群臣："诸位爱卿，你们觉得呢？"

庞涓还想反对，但看到魏惠王眼中的光芒，只好闭嘴。

"大王，臣以为可以一试。"丞相公叔痤出列，"合纵若成，魏国得利；合纵若败，魏国也无损失。"

"好！"魏惠王拍案，"寡人加入合纵！"

## 三晋结盟

离开魏王宫，苏秦三人来到城外的驿站。

"先生，您太厉害了！"赵纵兴奋地说，"一个人辩倒了满朝文武！"

"不是我厉害，是我说到了魏王的心坎上。"苏秦说，"魏王最渴望的就是恢复魏国的荣光。只要给他希望，他就会拼命抓住。"

"先生，现在赵、韩、魏三国都加入了，接下来去哪里？"郦姜问。

"齐国。"苏秦说，"齐国远离秦国，是最难说服的国家。但只要齐国加入，合纵就大局已定。"

"齐国……"郦姜皱眉，"齐王贪利短视，恐怕不容易说服。"

"我知道。"苏秦眼中闪过一丝自信，"但正因为难，才更值得挑战。"

"先生，您已经说服了三晋，为何不休息一下？"赵纵担心地问。

"时间不等人。"苏秦翻身上马，"秦国不会给我们太多时间。必须在秦国反应过来之前，完成合纵。"

三人策马向北，向着齐国进发。
